# Legacy paper mirrors

本目录保留项目早期的两篇课程报告归档，不是站点内容的编辑入口。

站点唯一 canonical 正文位于：

- `docs/computing/papers/edge-computing-survey.md`
- `docs/intelligence/papers/jupiter.md`

由于现有学习正文按约束不可删除或重写，本目录文件暂时作为 `READ_ONLY_MIRROR` 保留。`tools/check_duplicates.py` 会检查 mirror 与 canonical 的 SHA-256 一致；需要修改内容时只改 `docs/` 下 canonical 文件，并在独立迁移批次决定 mirror 的归档方式。
