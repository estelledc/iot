# 6G 时代 Internet of Everything 的范式与使能技术

> 初读范围：本文基于 arXiv 元数据、摘要和公开条目信息建立阅读卡片；尚未完成 PDF 全文逐段复核、综述分类表抽取或引用链核验，因此保持 `UNVERIFIED / UNREVIEWED`。

## 为什么值得读

IoT 强调“物”联网，IoE 则把人、数据、流程和物一起放进智能生态。6G 语境下，IoE 不只追求连接更多设备，还会把 AI、空天地网络、边缘计算、感知通信一体化和新型服务编排放在一起。

这篇综述适合作为 Layer 8 的大图入口，帮助把 6G IoT、IoE、智能城市、医疗和工业应用放到一个框架里比较。

## 论文要回答的问题

1. IoE 相比 IoT 的边界和范式变化是什么。
2. 6G 的哪些能力支撑 IoE：AI、边缘、SAGIN、ISAC 或语义通信。
3. IoE 在智慧城市、医疗、工业和网络服务中有哪些潜力。
4. 当前还缺哪些标准、隐私、安全和部署条件。

## 初读结论

这篇论文适合做方向地图，但要避免把 IoE 当成单一技术。后续深读应抽取它的使能技术清单和应用分类，并逐项标注哪些已经工程可用，哪些还停留在 6G 愿景。
