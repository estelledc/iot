# 天线OTA测试与辐射性能评估
> **难度**: 高级 | **领域**: 天线测试 | **阅读时间**: 约 22 分钟

## 引言

假设你买了一个蓝牙音箱,规格表上写着"通信距离10米",但放在金属书架旁后隔3米声音就断断续续。这就是天线"纸面性能"和"实际性能"的差距。天线在仿真软件里可能增益完美、方向图对称,但一旦装进产品外壳,被电池、电路板、螺丝、金属支架包围,实际辐射性能可能大打折扣。

OTA(Over-The-Air)测试就是在真实条件下测量天线的"实战表现"。它不拆天线,不接电缆,而是直接测量整机在空间中辐射出去的能量。就像体检不只看某个器官的指标,而是看整个人的健康状况一样。本文将详细介绍OTA测试的原理、方法、标准和实践应用。

## 1 为什么需要OTA测试

### 1.1 仿真与现实的鸿沟

天线设计通常经历以下阶段:

```
[电磁仿真] --> [原型制作] --> [传导测试] --> [OTA测试]
   理想环境      接近现实      部分现实      完全真实

仿真假设:              实际情况:
- 无限大地平面         - PCB尺寸有限
- 理想介质             - FR4板材损耗
- 无周围金属           - 电池/屏蔽罩/螺丝
- 单频点优化           - 宽带工作需求
- 自由空间             - 外壳封装
```

每一步都更接近现实,OTA测试是最终的"实战验证"。

### 1.2 影响天线实际性能的因素

| 因素 | 典型影响程度 | 说明 |
|------|-------------|------|
| 塑料外壳 | -0.5到-2 dB | 介质加载效应,频率偏移 |
| 金属外壳 | -3到-15 dB | 严重屏蔽,可能完全失效 |
| 电池 | -1到-4 dB | 作为损耗介质吸收能量 |
| PCB走线 | -0.5到-2 dB | 耦合产生不需要的辐射 |
| 人手握持 | -3到-8 dB | 人体组织吸收(SAR效应) |
| LCD显示屏 | -1到-3 dB | 导电层屏蔽 |

### 1.3 传导测试的局限

传导测试(通过同轴电缆测量)只能反映天线端口的匹配情况(回波损耗/VSWR),无法体现: 天线在外壳内的实际辐射效率; 方向图是否被周围结构扭曲; 整机在各个方向上的信号覆盖能力; 用户使用姿态下的性能退化。OTA测试弥补了这些盲区。

## 2 OTA核心测试参数

### 2.1 TRP(Total Radiated Power)

总辐射功率是设备在所有方向上辐射功率的总和:

```
TRP定义(球面积分):
把设备想象成一个发光灯泡
TRP = 灯泡向所有方向发出的总光通量

与传导功率的关系:
TRP = 传导功率(dBm) + 天线增益(dBi) - 损耗(dB)

示例:
传导功率: +20 dBm (100mW)
天线效率: -3 dB (50%)
TRP = 20 + 0(全向天线) - 3 = +17 dBm (50mW)
```

TRP反映了设备的"发射能力",值越高意味着信号覆盖范围越大。

### 2.2 TIS(Total Isotropic Sensitivity)

总全向灵敏度是设备从所有方向接收信号的综合灵敏度。TIS值越低(负数越大)越好,表示能接收更弱的信号:

- 优秀BLE设备 TIS: -90 dBm
- 一般BLE设备 TIS: -82 dBm
- 较差BLE设备 TIS: -75 dBm

### 2.3 辐射方向图(Radiation Pattern)

描述天线在不同方向上辐射强度的三维分布:

```
理想全向天线(水平面): 圆形,所有方向等强度

实际PCB天线(水平面):
有明显的主瓣和零点
被PCB地平面遮挡的方向凹陷
整体呈不规则形状
```

方向图揭示了设备在哪些方向信号好,哪些方向是"盲区"。

### 2.4 天线效率(Antenna Efficiency)

天线将输入功率转换为辐射功率的比例:

```
总效率 = 反射效率 x 辐射效率
       = (1 - |S11|^2) x (辐射功率/接受功率)

典型值:
- PCB倒F天线(手机): 40-70%
- 外置鞭状天线: 70-90%
- 小型IoT芯片天线: 20-50%
- 金属外壳内天线: 5-30%

效率每降低3dB, 通信距离缩短约30%
```

## 3 OTA测试设施

### 3.1 全电波暗室(Fully Anechoic Chamber)

最标准的OTA测试环境:

```
暗室结构:
+=============================================+
||  吸波材料(锥形/平板)覆盖所有内壁          ||
||                                           ||
||     [测量天线]                             ||
||         |                                 ||
||         |  3m/5m/10m 测试距离             ||
||         |                                 ||
||     [转台 + 被测设备]                      ||
||                                           ||
+=============================================+
  外层: 金属屏蔽(隔离度>80dB)
  内层: 吸波材料(反射率<-40dB)

工作原理:
- 屏蔽层阻止外部信号进入
- 吸波材料消除内部反射
- 模拟自由空间条件
```

### 3.2 CATR(Compact Antenna Test Range)

紧凑型天线测试场,用反射面将球面波转换为平面波:

```
[馈源] --> [抛物面反射器] --> 平面波 --> [被测设备]
                                    静区(Quiet Zone)
优势: 暗室尺寸更小(2-3m距离实现等效远场)
局限: 反射面精度高(表面精度<lambda/50),成本百万美元级
```

### 3.3 近场测试系统

通过近场扫描计算远场方向图: 探头在被测设备附近扫描采集幅度和相位数据; 通过傅里叶变换得到远场方向图。暗室可以很小,但测试时间长且计算复杂。

### 3.4 成本对比

| 设施类型 | 建设成本 | 每次测试费 | 适用场景 |
|----------|----------|-----------|----------|
| 全电波暗室(3m) | 50-200万美元 | 500-2000美元/天 | 通用标准测试 |
| CATR | 100-500万美元 | 1000-3000美元/天 | 高频大设备 |
| 近场系统 | 20-100万美元 | 300-1500美元/天 | 研发调试 |
| 简易屏蔽室 | 5-20万美元 | 100-500美元/天 | 预研非正式 |

## 4 测量方法

### 4.1 三维方向图测量

标准球面扫描:

```
球面坐标系:
- theta: 从0到180度(仰角)
- phi: 从0到360度(方位角)

测量步骤:
1. 每个theta角phi旋转一圈记录信号
2. 区分Etheta和Ephi两个极化分量
3. 汇总所有(theta,phi)点的数据

测量点数:
- 粗测: 12x24 = 288个点(15度步进)
- 细测: 36x72 = 2592个点(5度步进)
- 高精度: 90x180 = 16200个点(2度步进)
```

### 4.2 大圆切面法(Great Circle Cuts)

简化版测量,只测三个正交切面(XY/XZ/YZ平面): 比全3D测量快5-10倍; 适合快速评估和A/B方案对比; 局限是无法精确计算TRP(需要全3D数据)。

### 4.3 TRP计算方法

```python
import numpy as np

def calculate_trp(eirp_data, theta_step, phi_step):
    """
    eirp_data: 二维数组[theta_index][phi_index], 单位mW
    theta_step/phi_step: 角度步进(弧度)
    返回: TRP值(dBm)
    """
    trp = 0.0
    for i in range(len(eirp_data)):
        theta = i * theta_step
        for j in range(len(eirp_data[0])):
            # 球面积分: EIRP * sin(theta) * d_theta * d_phi
            trp += eirp_data[i][j] * np.sin(theta) * theta_step * phi_step
    trp = trp / (4 * np.pi)
    return 10 * np.log10(trp)  # 转换为dBm
```

## 5 OTA测试标准

### 5.1 主要标准体系

| 标准 | 内容 | 适用设备 |
|------|------|----------|
| CTIA Test Plan | TRP/TIS测试方法 | 蜂窝设备 |
| 3GPP TS 37.544 | E-UTRA OTA | 4G LTE设备 |
| 3GPP TS 38.151 | NR OTA | 5G NR设备 |
| CTIA IoT OTA | IoT射频性能 | NB-IoT/Cat-M |

### 5.2 IoT特定情况

针对IoT设备的OTA测试还没有统一的强制标准,但以下规范被广泛参考: CTIA IoT OTA Test Plan(2019年发布,NB-IoT/Cat-M); LoRa Alliance射频性能要求; Bluetooth SIG RF PHY测试规范(传导为主,OTA可选)。

## 6 IoT设备OTA测试的特殊挑战

### 6.1 小型化设备

IoT设备通常很小,带来独特挑战:

```
天线效率与电尺寸的关系(Chu-Harrington极限):
- 天线尺寸 < 0.1波长 --> 效率极低(<10%)
- 天线尺寸 = 0.25波长 --> 效率中等(30-60%)
- 天线尺寸 > 0.5波长 --> 效率较高(>60%)

示例:
868MHz波长 = 34.5cm
典型IoT PCB: 5cm x 3cm
天线电尺寸: 约0.14波长
理论效率上限: 约40%
```

测试挑战: 设备小导致夹具对结果影响大; 供电线缆可能成为天线的一部分; 设备轻导致转台夹持方式影响结果。

### 6.2 多天线系统

现代IoT网关可能集成WiFi+BLE+Zigbee+LoRa+LTE等多种天线: 每种天线需单独测量方向图; 天线间互耦影响各自性能; 需测试同时工作时性能退化; 测试时间和成本成倍增加。

### 6.3 供电与触发问题

OTA测试需要设备自主发射信号,但IoT设备通常处于低功耗状态:

- 方案1: 测试固件(连续发射载波,简单但不反映真实模式)
- 方案2: 定时发射(每秒一包,无需额外触发通道)
- 方案3: USB/UART控制(精确但线缆需特殊处理避免影响方向图)
- 方案4: 远程触发(接近真实但触发信号可能干扰测试)

## 7 低成本测试方案

### 7.1 初创公司的分阶段策略

```
阶段1: 桌面级(成本<1000美元)
工具: RTL-SDR + 简易天线 + 开阔场地
能做: 粗略估计辐射范围,发现严重问题
不能做: 精确方向图,定量TRP

阶段2: 半专业(成本5000-20000美元)
工具: 频谱分析仪 + 简易转台 + 小型屏蔽室
能做: 相对对比(方案A vs B),粗略方向图
不能做: 绝对校准的TRP/TIS

阶段3: 专业外包(每次3000-10000美元)
工具: 认可实验室全电波暗室
能做: 完整合规测试,精确TRP/TIS
适合: 产品定型后正式测试
```

### 7.2 简易OTA对比方法

```python
# 简易OTA对比测试: 转台旋转记录各角度功率
def simple_ota_comparison():
    results = {}
    for design in ["design_A", "design_B", "design_C"]:
        place_dut_on_turntable(design)
        pattern = []
        for angle in range(0, 360, 10):  # 每10度一个点
            rotate_turntable(angle)
            power = read_spectrum_analyzer_peak()  # dBm
            pattern.append(power)
        
        avg_power = sum(pattern) / len(pattern)
        uniformity = max(pattern) - min(pattern)
        results[design] = {"avg": avg_power, "uniformity": uniformity}
    
    for d, r in results.items():
        print(f"{d}: avg={r['avg']:.1f}dBm, uniformity={r['uniformity']:.1f}dB")
```

### 7.3 利用设备内置RSSI

许多IoT芯片提供RSSI功能,可用于简易OTA评估: 固定位置放参考发射源; 被测设备在不同角度/距离读取RSSI; RSSI下降量反映该方向天线增益损失。局限是RSSI精度通常只有正负3dB,只能做相对测量。

## 8 实践案例: 金属外壳导致6dB增益损失的解决

### 8.1 问题发现

某公司开发了一款工业级LoRa传感器,采用铝合金外壳满足IP67防护等级。传导测试S11正常(<-15dB),但现场通信距离远低于预期:

```
设计目标: 3km(城区)
实际测试: 仅1.2km即丢包严重
链路预算计算: 损失约6dB
排除项: 天线匹配正常/发射功率正常 -> 怀疑天线效率!
```

### 8.2 OTA测试诊断

送专业暗室,对比有壳vs无壳:

| 指标 | 无外壳 | 有外壳 | 差值 |
|------|--------|--------|------|
| TRP | +12.5 dBm | +6.2 dBm | -6.3 dB |
| 峰值增益 | +2.1 dBi | -1.8 dBi | -3.9 dB |
| 天线效率 | 62% | 15% | -- |
| 方向图均匀度 | 8 dB | 14 dB | 恶化 |

6.3dB的TRP损失: 超过75%的发射能量被金属外壳吸收或屏蔽。

### 8.3 根因分析

```
金属外壳导致性能损失的三个机制:

1. 涡流损耗: 天线近场耦合金属壁产生涡流转为热量
   (当天线距金属壁<lambda/10时严重)

2. 屏蔽效应: 金属形成法拉第笼阻止电磁波辐射
   (完全封闭时隔离度>60dB)

3. 去谐效应: 金属改变天线周围电场分布导致频率偏移
   (原本匹配的天线变得失配)

本案例: 外壳与天线距离仅5mm(约0.014波长)
辐射只能从底部塑料密封区狭窄开口泄漏
```

### 8.4 解决方案迭代

```
方案A: 侧面开塑料窗(40x30mm)
  TRP恢复到+9.8dBm(改善3.6dB)
  问题: 防护降至IP54

方案B: 外置SMA天线
  TRP达+11.2dBm(接近无壳性能)
  问题: 成本高/外观差/可能被碰断

方案C(最终): 外壳集成天线
  天线设计在外壳外表面塑料区域
  通过弹片从PCB馈电到外壳天线
  天线与金属壳保持20mm以上距离
  TRP达+10.8dBm, 保持IP67
```

### 8.5 经验教训

金属外壳IoT设备天线设计原则:

- 天线与金属壁距离 > lambda/8(868MHz时约4.3cm)
- 预留射频窗口面积 > lambda/4 x lambda/4
- 设计阶段就做OTA测试(不要等量产前)
- 链路预算中金属壳预留4-6dB余量,塑料壳预留2dB

## 9 OTA测试报告解读

### 9.1 性能分级参考

```
TRP(以+20dBm传导功率为参考):
优秀: TRP > +17 dBm (效率>50%)
良好: TRP > +14 dBm (效率>25%)
及格: TRP > +11 dBm (效率>12%)
不及格: TRP < +11 dBm

方向图均匀度:
优秀: 峰值-谷值 < 8 dB
良好: 峰值-谷值 < 12 dB
及格: 峰值-谷值 < 18 dB
不及格: 峰值-谷值 > 18 dB (严重方向性)
```

### 9.2 报告关注要点

一份OTA测试报告中应关注: 测试配置是否反映实际使用条件; 测量不确定度(通常正负1-2dB); TRP与链路预算是否匹配; 方向图主瓣方向和零点位置是否影响实际部署。

## 总结

OTA测试是验证IoT设备天线真实性能的最可靠方法。它弥补了仿真和传导测试无法覆盖的实际使用条件差距,揭示了外壳、电池、PCB等周围结构对天线辐射的影响。

对于IoT产品开发,核心建议是: 尽早在开发流程中引入OTA测试(哪怕是简易对比测试),不要等到产品定型后才发现性能问题。金属外壳、紧凑布局、多天线共存等IoT设备的典型特征,都会显著影响天线的实际辐射性能。通过系统的OTA测试和迭代优化,可以在不增加太多成本的前提下最大化设备的无线覆盖能力。
