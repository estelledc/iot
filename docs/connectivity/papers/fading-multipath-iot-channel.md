# 衰落与多径效应对IoT无线信道的影响
> **难度**：🔴 高级 | **领域**：无线信道 | **阅读时间**：约 22 分钟

## 引言

想象你站在一个空旷的操场上和朋友说话,声音清晰稳定。但如果你们走进一个四面都是玻璃墙的大厅,你会发现有些位置听得很清楚,挪动半步却突然听不清了——这就是多径效应的日常体验。声波从墙壁反射后与直达声叠加,有时增强有时抵消。

无线电信号在城市环境中面临同样的问题。对于IoT设备来说,一个传感器可能固定安装在某个位置,如果恰好处于信号死角,就会持续丢包。理解衰落和多径效应,是设计可靠IoT无线系统的基础。

## 1. 什么是衰落

### 1.1 衰落的基本定义

衰落(Fading)是指无线信号强度随时间、空间或频率发生随机波动的现象。与路径损耗(Path Loss)不同,路径损耗是可预测的平均信号衰减,而衰落是围绕平均值的随机变化。

关键区别: 路径损耗是确定性的(距离加倍衰减6dB),衰落是随机的(同一距离上信号可能瞬间下降20-40dB)。

### 1.2 衰落对IoT的特殊影响

IoT设备的特殊性使衰落问题更加突出: 设备通常固定安装无法移动避开衰落点; 发射功率极低没有多余余量; 要求长期可靠运行; 天线通常很小且简单。

### 1.3 衰落的时间尺度

衰落发生在不同的时间尺度上,理解这些尺度有助于选择合适的缓解策略:

```
衰落时间尺度:
- 快衰落: 毫秒级变化, 由多径干涉引起
  对策: 分集技术(频率/空间), 交织编码

- 慢衰落: 秒到分钟级变化, 由障碍物遮挡引起
  对策: 衰落余量, 功率控制

- 超慢衰落: 小时到天级变化, 由天气和环境变化引起
  对策: 长期监测, ADR自适应
```

快衰落对每个数据包都有影响,而慢衰落影响整个通信会话。IoT设备因为发射间隔长(可能每小时一次),每次发射时信道状态可能完全不同,这使得信道预测变得困难。

## 2. 大尺度衰落与小尺度衰落

### 2.1 大尺度衰落(阴影衰落)

由信号传播路径上的大型障碍物引起:

```
特征:
- 成因: 建筑物、山丘、大型车辆遮挡
- 变化尺度: 数十米距离上变化
- 统计分布: 对数正态分布
- 标准差: 城市8-12dB, 郊区6-8dB, 室内4-6dB
```

### 2.2 小尺度衰落(多径衰落)

由多径传播引起,信号在极短距离内剧烈变化:

```
特征:
- 成因: 多条反射路径信号在接收端叠加
- 变化尺度: 半个波长(Sub-GHz约15-17cm)
- 深衰落: 可导致信号瞬间下降20-40dB
- 发生频率: 城市环境中非常频繁
```

### 2.3 两种衰落的叠加

```
接收功率(dB) = 发射功率 - 路径损耗 - 阴影衰落 - 多径衰落

其中:
- 路径损耗: 确定性, 可计算
- 阴影衰落: 随机, 对数正态分布, 变化缓慢
- 多径衰落: 随机, 瑞利/莱斯分布, 变化快速
```

## 3. 多径传播机制

### 3.1 多径产生的原因

无线信号从发射端到接收端可能经过多条路径: 直射路径(Line-of-Sight)、反射(遇到大于波长的光滑表面)、绕射(遇到尖锐边缘)、散射(遇到小于波长的粗糙表面)。

在IoT典型环境中,多径来源非常丰富: 城市街道中建筑物外墙和地面产生强反射; 工厂中金属设备和管道是高效的反射体; 室内环境中墙壁、家具和人体都会产生散射。每条路径的信号具有不同的传播延迟、到达角度、衰减程度和相位。

### 3.2 多径叠加的结果

```
两条路径叠加:
r(t) = A1*cos(2*pi*f*t + phi1) + A2*cos(2*pi*f*t + phi2)

合成幅度: A = sqrt(A1^2 + A2^2 + 2*A1*A2*cos(phi1-phi2))

相位差180度: A_min = |A1 - A2| (深衰落)
相位差0度:   A_max = A1 + A2 (增强)
```

## 4. 瑞利衰落模型

### 4.1 瑞利衰落的条件

当没有主导直射路径(NLOS)、存在大量反射路径、各路径幅度和相位独立随机时,接收信号幅度服从瑞利分布。这在城市IoT部署中非常常见,因为设备通常安装在建筑密集区域,与网关之间没有直射路径。

```
瑞利分布概率密度函数:
p(r) = (r/sigma^2) * exp(-r^2/(2*sigma^2)),  r >= 0

关键统计量:
- 均值: 1.253 * sigma
- 中值: 1.177 * sigma
- 方差: (2 - pi/2) * sigma^2
```

### 4.2 深衰落概率

| 衰落深度(相对中值) | 发生概率 |
|---|---|
| -10dB | 约10% |
| -20dB | 约1% |
| -30dB | 约0.1% |

### 4.3 IoT中的瑞利衰落场景

典型场景: 城市街道中被建筑物包围的IoT设备、室内经过多次墙壁反射的传感器、地下停车场完全依赖反射路径的设备。

## 5. 莱斯衰落模型

### 5.1 K因子

当存在主导直射路径(LOS)加上反射路径时,信号服从莱斯分布。K因子是关键参数:

```
K = 直射路径功率 / 散射路径总功率

K = 0 (负无穷dB): 退化为瑞利衰落
K = 10 (10dB): 直射路径主导, 衰落较轻
K -> 无穷: 接近无衰落信道
```

### 5.2 典型K因子值

| 环境 | 典型K因子 |
|---|---|
| 室内无LOS | 0dB (瑞利) |
| 室内有LOS | 3-6dB |
| 城市微蜂窝 | 4-8dB |
| 郊区 | 6-12dB |
| 农村开阔地 | 10-20dB |

## 6. 时延扩展与相干带宽

### 6.1 时延扩展

多径传播导致信号在时间上展宽。RMS时延扩展是关键参数:

```
典型RMS时延扩展:
- 室内: 20-50ns
- 城市微蜂窝: 100-300ns
- 城市宏蜂窝: 1-3us
```

### 6.2 相干带宽

```
相干带宽 Bc 约等于 1/(5 * RMS时延扩展)

示例:
- 室内(50ns): Bc = 4MHz
- 城市(200ns): Bc = 1MHz
```

### 6.3 平坦衰落与频率选择性衰落

- 信号带宽 < 相干带宽: 平坦衰落, 整个信号同时衰落
- 信号带宽 > 相干带宽: 频率选择性衰落, 不同频率分量独立衰落

## 7. 对IoT系统的具体影响

### 7.1 窄带IoT系统

LoRa(125kHz)和Sigfox(100Hz)带宽远小于相干带宽,经历平坦衰落。整个信号同时衰落或增强,要么全收到要么全丢失。深衰落期间整个数据包丢失,衰落持续时间可能覆盖多个数据包,需要时间分集(重传)来恢复。

```
窄带IoT的衰落影响:
- LoRa SF12数据包持续时间可达1秒以上
- 深衰落持续时间为相干时间量级
- 静态设备: 相干时间长, 衰落持续久
- 可能连续丢失多个重传包
- 需要: 跨频率重传 + 跨时间重传
```

### 7.2 宽带IoT系统

WiFi(20-160MHz)带宽远大于相干带宽,经历频率选择性衰落。部分子载波衰落,其他正常,OFDM可利用这一特性。通过频域均衡和信道编码,即使部分子载波深衰落也能恢复数据,因此宽带系统对平坦衰落的抵抗力更强。

### 7.3 对数据包的影响总结

窄带系统中深衰落意味着整个数据包丢失,需要重传或分集来恢复。宽带系统中部分数据受损而部分正常,可通过频域均衡和编码恢复。这解释了为什么LoRa需要多网关宏分集来达到高可靠性,而WiFi主要靠OFDM本身的频率分集。

## 8. 多普勒效应

### 8.1 多普勒频移

```
多普勒频移: fd = (v/c) * f * cos(theta)

示例(868MHz):
- 步行(1.5m/s): fd = 4.3Hz
- 自行车(5m/s): fd = 14.5Hz
- 汽车(30m/s): fd = 87Hz
```

### 8.2 相干时间

```
相干时间 Tc 约等于 1/(4 * fd_max)

- 静止设备: Tc 非常长(秒级)
- 步行速度: Tc = 58ms
- 车速: Tc = 2.9ms
```

大多数IoT设备是静态的,多普勒效应不严重。但物流追踪、共享单车、无人机等移动场景需要考虑。

## 9. 衰落缓解技术概述

### 9.1 四种基本分集技术

- 时间分集: 在不同时间重传,间隔大于相干时间,简单重传或交织编码均可
- 频率分集: 在不同频率发送,间隔大于相干带宽,跳频或多载波实现
- 空间分集: 多天线接收,间距大于半波长,选择合并或MRC合并
- 编码分集: FEC纠错码和扩频技术,提供3-6dB编码增益

LPWAN协议通常组合使用多种分集: LoRaWAN同时利用频率跳变(频率分集)、多网关(空间分集)和重传(时间分集); NB-IoT通过重复传输(时间分集)、多子载波(频率分集)和基站多天线(空间分集)实现覆盖增强。

### 9.2 衰落余量

```
链路预算中预留额外余量:
- 90%可靠性: 约10dB余量
- 95%可靠性: 约13dB余量
- 99%可靠性: 约20dB余量

LPWAN高灵敏度有助于吸收衰落余量:
LoRa SF12灵敏度-137dBm, 即使预留15dB余量
仍有足够链路余量实现数公里覆盖
```

## 10. LoRaWAN中的实际抗衰落设计

### 10.1 LoRa的内在抗衰落能力

- CSS扩频提供处理增益
- 恒包络调制对幅度衰落有容忍度
- 频率扫描对窄带干扰有天然抵抗力

### 10.2 LoRaWAN协议层设计

- 频率分集: 8个信道随机选择
- 时间分集: 确认帧在不同时间和频率重传
- 空间分集: 多网关部署,任一网关收到即有效
- ADR: 根据信号质量调整SF

### 10.3 多网关宏分集

```
单网关丢包率: 10%
假设各网关独立衰落:
- 双网关同时丢包: 1%
- 三网关同时丢包: 0.1%

从90%可靠性提升到99.9%, 只需3个网关
```

### 10.4 深度室内覆盖的挑战

即使有上述机制,深度室内仍具挑战: 建筑穿透损耗10-30dB,室内多径增加衰落深度。解决方案包括室内中继器、更高SF、室内网关部署。

```
深度室内覆盖的额外损耗:
- 木墙穿透: 4-6dB
- 砖墙穿透: 10-15dB
- 混凝土墙: 15-30dB
- 金属屏蔽: 30-50dB

应对策略:
- 提高SF(SF12比SF7多14dB链路余量)
- 部署室内或窗边网关
- 使用LoRa中继器扩展覆盖
- NB-IoT的CE Level 2(最多128次重复)
```

### 10.5 实际部署经验

在城市LoRaWAN部署中,3个网关间距500m可覆盖95%以上的户外设备。但对于地下室和深层室内设备,即使3个网关也可能不够,需要额外的室内网关或中继器。经验表明,每增加一个网关,整体可靠性提升约一个数量级(从90%到99%到99.9%),但成本也线性增加。

## 总结

衰落和多径效应是IoT无线通信中不可避免的物理现象。大尺度衰落由障碍物遮挡引起,变化缓慢但幅度可达10dB以上; 小尺度衰落由多径干涉引起,变化快速且可能导致20-40dB的深衰落。

对于IoT系统设计者来说,关键要点包括: 理解部署环境的衰落特征(瑞利还是莱斯),在链路预算中预留足够的衰落余量,利用LPWAN协议内建的分集机制,以及认识到需要多种技术的组合才能有效对抗衰落。
