# Sub-GHz频段穿透力与传输距离优势分析
> **难度**：🟢 初级 | **领域**：无线传播基础 | **阅读时间**：约 18 分钟

## 引言

想象你站在一栋公寓楼外面喊话。低沉的男低音能穿过好几堵墙传到里面,而尖锐的口哨声在第一堵墙就被挡住了。无线电波的传播也遵循类似规律: 频率越低(声音越"低沉"),穿透障碍物和绕过拐角的能力就越强。

在IoT无线通信中,Sub-GHz频段(通常指433MHz、868MHz、915MHz等低于1GHz的频率)相比2.4GHz频段,就像那个能穿墙的男低音。这篇文章将从物理原理出发,系统分析Sub-GHz频段在穿透力和传输距离上的优势,帮助你理解为什么很多远距离IoT应用选择Sub-GHz。

## 1. 频率与传播的物理关系

### 1.1 波长与绕射能力

无线电波遇到障碍物时,能否绕过去取决于波长与障碍物尺寸的关系:

- 868MHz波长约34.5cm
- 433MHz波长约69.2cm
- 2.4GHz波长约12.5cm

当障碍物尺寸与波长相当或更小时,电磁波可以有效绕射(衍射)。Sub-GHz的长波长意味着它能绕过更大的障碍物,比如建筑拐角、家具、管道等。

### 1.2 穿透与吸收

电磁波穿过材料时,部分能量被吸收转化为热量。吸收程度与频率正相关: 频率越高,材料中的分子振动越剧烈,吸收越多。这就是为什么微波炉用2.4GHz加热食物,而收音机的AM信号(几百kHz)能穿过整栋建筑。

### 1.3 基本公式

自由空间路径损耗(FSPL)公式清楚地展示了频率的影响:

```
FSPL(dB) = 20*log10(d) + 20*log10(f) + 32.44

其中:
  d = 距离(km)
  f = 频率(MHz)
```

从公式可以看出,频率每翻倍,路径损耗增加6dB。这意味着在相同距离下,2.4GHz比868MHz多损耗约8.8dB。

## 2. 自由空间路径损耗对比

### 2.1 计算实例

在1km距离下的FSPL对比:

| 频率 | FSPL (1km) | 相对868MHz的额外损耗 |
|------|-----------|-------------------|
| 433 MHz | 84.2 dB | -6.0 dB (更少) |
| 868 MHz | 90.2 dB | 基准 |
| 915 MHz | 90.7 dB | +0.5 dB |
| 2.4 GHz | 99.0 dB | +8.8 dB |

### 2.2 距离优势换算

8.8dB的损耗差异意味着什么?

假设发射功率和接收灵敏度相同,868MHz相比2.4GHz在自由空间中的传输距离优势约为2.7倍。计算过程:

```
距离比 = 10^(8.8/20) = 2.75

即: 如果2.4GHz能传1km, 868MHz在相同条件下能传2.75km
```

### 2.3 实际意义

这个2.7倍的距离优势是纯物理定律决定的,不依赖任何调制技术或协议优化。它是Sub-GHz频段的"先天优势",任何工作在这些频段的协议(LoRa、Sigfox、Z-Wave、Wi-SUN等)都自动享有。

## 3. 材料穿透损耗对比

### 3.1 常见建筑材料

不同材料对不同频率的衰减差异显著:

| 材料 | 868MHz损耗 | 2.4GHz损耗 | Sub-GHz优势 |
|------|-----------|-----------|------------|
| 混凝土墙(15cm) | 5-10 dB | 10-20 dB | 5-10 dB |
| 木质墙体 | 2-3 dB | 3-5 dB | 1-2 dB |
| 普通玻璃 | 1-2 dB | 2-4 dB | 1-2 dB |
| 砖墙(单层) | 4-8 dB | 8-15 dB | 4-7 dB |
| 人体 | 3-5 dB | 5-10 dB | 2-5 dB |
| 金属板 | 20-40 dB | 20-40 dB | 接近(均高) |

### 3.2 关键观察

混凝土和砖墙的差异最为显著。在工业环境和老式建筑中,这些材料大量存在,Sub-GHz的穿透优势在这些场景中最为明显。

金属是例外: 无论什么频率,金属都会造成极大衰减。这是因为金属中的自由电子会反射电磁波,与频率关系不大。

### 3.3 湿度影响

潮湿材料的衰减更大,且频率越高影响越明显。水分子在2.4GHz附近有强吸收(这正是微波炉的工作原理)。因此在潮湿环境(地下室、浴室、雨天户外)中,Sub-GHz的优势更加突出。

## 4. 室内传播模型

### 4.1 多墙模型

室内传播常用多墙模型估算:

```
PL(dB) = FSPL(d) + sum(L_wall_i) + sum(L_floor_j)

其中:
  FSPL(d) = 自由空间损耗
  L_wall_i = 第i面墙的穿透损耗
  L_floor_j = 第j层楼板的穿透损耗
```

### 4.2 实际穿墙能力对比

假设每面混凝土墙的衰减:
- 868MHz: 约7dB/墙
- 2.4GHz: 约15dB/墙

假设接收灵敏度为-100dBm,发射功率为+14dBm(25mW):

| 穿过墙数 | 868MHz剩余链路预算 | 2.4GHz剩余链路预算 |
|---------|------------------|------------------|
| 1面墙 | 充裕 | 充裕 |
| 2面墙 | 充裕 | 勉强 |
| 3面墙 | 可用 | 不可靠 |
| 4面墙 | 勉强 | 断连 |
| 5面墙 | 边缘 | 断连 |

### 4.3 实际覆盖范围

在典型住宅环境中:
- Sub-GHz(如Z-Wave 868MHz): 从发射点可覆盖3-5个房间,包括隔了多面墙的区域
- 2.4GHz(如Zigbee/WiFi): 从发射点通常覆盖1-3个房间,远端房间信号弱

这解释了为什么Z-Wave(868/908MHz)在智能家居中常常不需要中继器就能覆盖整栋房子,而Zigbee(2.4GHz)在大户型中经常需要添加中继节点。

## 5. 户外传输距离

### 5.1 典型范围对比

| 技术 | 频段 | 城市环境 | 郊区环境 | 开阔地带 |
|------|------|---------|---------|---------|
| LoRa | 868/915MHz | 2-5 km | 5-10 km | 10-15 km |
| Sigfox | 868MHz | 3-5 km | 5-10 km | 30-50 km |
| Z-Wave LR | 868/908MHz | 0.5-1 km | 1-2 km | 2-4 km |
| WiFi | 2.4GHz | 50-100 m | 100-200 m | 200-500 m |
| BLE 5.0 | 2.4GHz | 50-100 m | 100-300 m | 400-800 m |
| Zigbee | 2.4GHz | 30-75 m | 75-150 m | 100-300 m |

### 5.2 为什么户外差距更大

户外环境中,距离本身就是主要损耗来源。FSPL公式中的频率项在长距离时影响被放大:

```
在10km距离:
  868MHz FSPL = 110.2 dB
  2.4GHz FSPL = 119.0 dB
  差异仍为8.8dB, 但此时链路预算已经很紧张
  这8.8dB可能就是"能通"和"不能通"的区别
```

### 5.3 地形影响

Sub-GHz的长波长在遇到地形起伏时,绕射能力更强。小山丘、树林、建筑群等障碍物对Sub-GHz的阻挡效果弱于2.4GHz。这使得Sub-GHz在非视距(NLOS)户外场景中优势更加明显。

## 6. 天线尺寸权衡

### 6.1 天线长度计算

高效天线的长度与波长直接相关。四分之一波长天线:

| 频率 | 四分之一波长 | 天线形式 |
|------|------------|---------|
| 433 MHz | 17.3 cm | 弹簧/鞭状天线 |
| 868 MHz | 8.6 cm | PCB走线/芯片天线 |
| 915 MHz | 8.2 cm | PCB走线/芯片天线 |
| 2.4 GHz | 3.1 cm | 微型芯片天线 |

### 6.2 实际设计考量

868MHz的8.6cm天线长度对大多数IoT设备来说是可接受的:
- 智能家居传感器: 外壳通常大于10cm,可容纳PCB天线
- 工业传感器: 尺寸不敏感,甚至可用外置天线
- 可穿戴设备: 偏大,可能需要折叠或螺旋天线设计

433MHz的17.3cm天线确实较大,但通过螺旋天线或弯折设计,仍可集成到中等尺寸设备中。

### 6.3 PCB天线方案

现代Sub-GHz设备广泛使用PCB走线天线:
- 倒F天线(IFA): 紧凑,适合868MHz
- 蛇形天线(Meander): 通过弯折缩短物理长度
- 芯片天线: 最小尺寸,但效率略低

这些方案使得Sub-GHz天线在实际产品中并不构成严重限制。

## 7. 干扰环境对比

### 7.1 2.4GHz的拥挤现状

2.4GHz ISM频段是全球最拥挤的无线频段:
- WiFi (802.11b/g/n/ax)
- Bluetooth/BLE
- Zigbee/Thread
- 微波炉泄漏
- 无线鼠标/键盘
- 婴儿监视器
- 无人机图传

在一个典型办公室中,可能同时有50-100个2.4GHz设备在工作。

### 7.2 Sub-GHz的清净环境

Sub-GHz频段的使用者相对较少:
- LoRa/LoRaWAN
- Sigfox
- Z-Wave
- 部分遥控器和无线门铃
- 一些工业遥测设备

设备密度通常比2.4GHz低1-2个数量级。

### 7.3 干扰对可靠性的影响

在拥挤频段中:
- 数据包碰撞概率增加
- 需要更多重传
- 有效吞吐量下降
- 延迟增加且不可预测

Sub-GHz的清净环境意味着:
- 首次传输成功率更高
- 更少重传,节省电量
- 延迟更可预测
- 通信更可靠

## 8. 各地区法规差异

### 8.1 Sub-GHz频段分配

| 地区 | 频段 | 发射功率限制 | 特殊要求 |
|------|------|------------|---------|
| 欧洲 | 868 MHz | 25mW (14dBm) | 占空比限制(0.1%-10%) |
| 北美 | 915 MHz | 1W (30dBm) | 需要FHSS或数字调制 |
| 中国 | 470-510 MHz | 50mW | 需要型号核准 |
| 日本 | 920 MHz | 20mW | 特定信道规划 |
| 澳洲 | 915-928 MHz | 1W (30dBm) | 类似北美规则 |

### 8.2 2.4GHz的全球统一优势

2.4GHz ISM频段在全球几乎所有国家都可用,且规则相似(通常100mW/20dBm)。这对需要全球销售的产品是巨大优势: 一个硬件设计可以卖到全世界。

### 8.3 占空比限制(欧洲)

欧洲868MHz频段的占空比限制是重要约束:
- 部分子频段: 每小时只能发射总时间的0.1%(3.6秒)
- 部分子频段: 每小时可发射1%(36秒)
- 部分子频段: 每小时可发射10%(360秒)

这意味着Sub-GHz设备不能持续发送数据,适合低占空比的传感器应用,不适合流媒体或频繁交互。

## 9. 实际测试案例

### 9.1 智能家居场景

Z-Wave(868MHz)与Zigbee(2.4GHz)在同一栋三层住宅中的对比测试:

```
测试条件:
  - 三层独栋住宅, 混凝土楼板, 砖墙
  - 网关位于一楼客厅
  - 测试点: 各楼层各房间

结果:
  Z-Wave (868MHz):
    - 一楼所有房间: 信号良好
    - 二楼所有房间: 信号良好
    - 三楼大部分房间: 信号可用
    - 地下室: 信号可用
    - 需要中继器: 0个

  Zigbee (2.4GHz):
    - 一楼所有房间: 信号良好
    - 二楼近端房间: 信号良好
    - 二楼远端房间: 信号弱
    - 三楼: 大部分断连
    - 地下室: 断连
    - 需要中继器: 2-3个
```

### 9.2 LoRa户外测试

LoRa(868MHz)城市环境传输测试:

```
测试条件:
  - 网关: 建筑屋顶, 高度约20m
  - 终端: 地面手持, 天线高度1.5m
  - 发射功率: 14dBm
  - 扩频因子: SF12 (最大范围)
  - 环境: 城市密集区

结果:
  - 500m: 丢包率 < 1%
  - 1km: 丢包率 约 5%
  - 2km: 丢包率 约 15%
  - 3km: 丢包率 约 30%
  - 5km: 偶尔可通(视距方向)
```

### 9.3 对比总结

同等条件下(相同发射功率、相同天线增益),Sub-GHz的实测覆盖范围通常是2.4GHz的3-5倍,在有障碍物的环境中优势更加明显。

## 10. 选型决策指南

### 10.1 选择Sub-GHz的场景

- 传输距离是首要需求(户外部署、农业、城市基础设施)
- 需要穿透多面墙壁或楼板(大型建筑、地下室)
- 数据量小、发送频率低(传感器周期上报)
- 部署环境2.4GHz干扰严重(工厂、办公楼)
- 电池寿命要求极长(10年以上)

### 10.2 选择2.4GHz的场景

- 需要较高数据速率(音频、图像、OTA升级)
- 产品需要全球统一频率(简化硬件设计)
- 天线尺寸严格受限(微型可穿戴)
- 需要与现有生态兼容(BLE手机直连、WiFi基础设施)
- 需要频繁双向通信(无占空比限制)

### 10.3 双频组合方案

越来越多的IoT设备采用双频设计:
- Sub-GHz用于长距离传感器数据上报(主要通信)
- 2.4GHz BLE用于本地配置和调试(辅助通信)

典型例子:
- LoRa传感器 + BLE配网: 日常数据走LoRa远传,初始配置用手机BLE近场完成
- Z-Wave设备 + BLE: 智能家居控制走Z-Wave,固件更新走BLE
- Wirepas双频: 同时支持Sub-GHz远距离和2.4GHz高速率

这种组合方案兼顾了两个频段的优势,是当前IoT产品设计的趋势之一。

## 总结

Sub-GHz频段在穿透力和传输距离上的优势源于基本物理定律,是不可改变的"先天优势"。核心要点包括: 频率越低波长越长,绕射和穿透能力越强; 相同距离下868MHz比2.4GHz少8.8dB路径损耗,等效2.7倍距离优势; 穿过混凝土墙时Sub-GHz每面墙少损耗5-10dB; Sub-GHz频段更清净,干扰更少,通信更可靠。

选择频段时需要综合考虑: 距离需求、数据速率、天线尺寸、法规限制、生态系统等因素。没有"最好"的频段,只有"最适合"的频段。对于大多数远距离、低速率、需要穿墙的IoT应用,Sub-GHz是更优选择。
