# 可重构智能表面RIS辅助IoT通信
> **难度**：🔴 高级 | **领域**：新型通信技术 | **阅读时间**：约 22 分钟

## 引言

想象你在一条光线昏暗的走廊里走路，虽然天花板上的灯在正前方，但走廊拐角后面一片漆黑。现在如果有人在拐角处放一面镜子，调整角度把灯光反射到黑暗区域，你就能看清脚下的路了。这面镜子不需要自己发电，只是改变了光的传播方向。

可重构智能表面 (Reconfigurable Intelligent Surface, RIS) 就是无线世界里的这面"智能镜子"。它是一块平面阵列，由大量可编程的反射单元组成，每个单元都能独立控制反射信号的相位。通过协调所有单元的相位，RIS 可以将信号"聚焦"到期望的方向，在非视距 (NLOS) 场景中创造出虚拟的视距路径。

对于 IoT 而言，RIS 提供了一种全新的覆盖增强手段：不增加基站、不消耗额外射频功率，仅靠一块被动反射板就能显著改善信号覆盖。本文将系统介绍 RIS 的原理、硬件结构、在 IoT 中的应用及面临的挑战。

## 一、什么是 RIS

### 定义与核心概念

可重构智能表面是一种由大量亚波长人工单元 (meta-atom) 构成的平面阵列，每个单元可以独立调节入射电磁波的反射相位 (在某些设计中还可调节幅度)。通过对所有单元的反射相位进行协同控制，RIS 能够实现波束成形，将信号能量集中到目标方向或抑制干扰方向。

RIS 代表了一种范式转变：传统通信系统通过调整发射机和接收机来适应固定的无线信道；RIS 则直接改变无线传播环境本身，使信道成为通信系统可调控的一部分。

```
传统通信范式:
  发射机 --(固定的无线信道)--> 接收机
  策略: 适应信道 (波束成形, 编码, 自适应调制)

RIS 通信范式:
  发射机 --(RIS 可控信道)--> 接收机
  策略: 同时调整收发机和信道
  RIS 重新配置传播路径, 创造有利信道条件
```

### 为什么 IoT 需要 RIS

IoT 场景对 RIS 有特殊的需求：

覆盖扩展是首要需求。许多 IoT 设备 (如 LoRa、NB-IoT 终端) 处于深度覆盖场景——地下室、管道、仓库角落。RIS 可以在不增加基站的情况下延伸信号覆盖。

非视距场景是 IoT 的常见挑战。工业 IoT 中大量金属设备遮挡视线，城市 IoT 中建筑物阻挡信号。RIS 可以创造虚拟视距路径，绕过障碍物。

能效是 IoT 的核心约束。RIS 本身是被动反射器件 (或近被动)，不包含射频链路，功耗极低，这使其比传统中继更适合能量受限的 IoT 场景。

## 二、RIS 硬件结构

### 基本组成

RIS 的硬件由以下几个核心部分组成：

```
RIS 硬件架构:

+------------------------------------------+
|          RIS 表面 (平面阵列)              |
|                                          |
|  [单元1] [单元2] [单元3] ... [单元N]     |  <- 反射单元阵列
|  [单元1] [单元2] [单元3] ... [单元N]     |
|  [单元1] [单元2] [单元3] ... [单元N]     |
|                                          |
|  每个单元包含:                            |
|    - 金属贴片 (亚波长尺寸)                |
|    - 可调元件 (PIN 二极管或变容二极管)    |
|    - 介质基板                            |
|                                          |
|  控制电路:                               |
|    - FPGA/MCU 控制每个单元的状态          |
|    - 通过偏置线调节二极管电压             |
+------------------------------------------+
```

### 反射单元设计

每个反射单元 (也称超原子 meta-atom 或单元细胞 unit cell) 的尺寸通常为亚波长级别——即小于工作频率对应波长的一半。

以 868MHz (LoRa 常用频段) 为例：波长约为 34.5cm，单元尺寸约为 17cm。一块 1 平方米的 RIS 表面大约可以容纳 36 个单元 (6x6 阵列)。

在更高的频率 (如 2.4GHz 或毫米波)，波长更短，相同面积可以容纳更多单元，获得更高的波束成形增益。这也是 RIS 在 5G 毫米波和 6G 太赫兹频段更受关注的原因之一。

### 可调元件

反射相位的调节通过可调元件实现。两种主流方案是：

PIN 二极管方案：通过控制二极管的导通/关断状态，在两种反射相位之间切换。通常提供 1-bit 相位控制 (0 或 180 度)，结构简单、功耗低。多 bit 控制需要多个二极管，增加复杂度。

变容二极管方案：通过调节反向偏置电压连续改变电容值，实现反射相位的连续调节。提供更高精度的相位控制，但功耗略高且控制电路更复杂。

## 三、RIS 工作原理

### 相位协同与波束成形

当电磁波到达 RIS 表面时，每个反射单元以预设的相位反射信号。通过精心设计所有单元的相位配置，可以使各单元的反射信号在目标方向上同相叠加 (相长干涉)，在其他方向上反相抵消 (相消干涉)。

```
RIS 波束成形原理:

入射波到达 RIS 表面
     |
     v
+--+--+--+--+--+--+
|P1|P2|P3|P4|P5|P6|   <- 各单元施加不同的相位偏移
+--+--+--+--+--+--+
     |
     v
各单元反射波在目标方向叠加:
  目标方向: 相长干涉 -> 信号增强 (波束指向)
  其他方向: 相消干涉 -> 信号减弱
     |
     v
等效效果: 信号被"聚焦"到目标接收机
```

这类似于阵列天线的波束成形，但关键区别在于：RIS 不产生信号，只反射已有信号；RIS 不需要射频链路，功耗仅来自控制电路。

### 波束成形增益

RIS 的波束成形增益与单元数量密切相关。理论分析表明，当 RIS 具有 N 个反射单元时，接收端的信号功率增益正比于 N 的平方：

```
RIS 增益估算:

信号功率增益 ~ N^2 (N = 反射单元数)

示例:
  N = 36  (1m2, 868MHz):  增益 ~ 36^2 = 1296 倍 = 31 dB
  N = 100 (2m2, 900MHz):  增益 ~ 100^2 = 10000 倍 = 40 dB
  N = 400 (0.5m2, 5GHz):  增益 ~ 400^2 = 160000 倍 = 52 dB

注意: 实际增益低于理论值, 受以下因素影响:
  - 单元损耗
  - 相位量化误差 (1-bit 控制)
  - 通道估计误差
  - 表面效率
  实际增益通常为理论值的 50-70%
```

## 四、RIS 与传统中继对比

### 功能差异

RIS 和传统中继 (relay) 都可以扩展覆盖范围，但工作方式截然不同：

| 特性 | 传统中继 (Relay) | RIS |
|------|-----------------|-----|
| 工作方式 | 接收 -> 放大/解码 -> 重新发射 | 纯反射, 相位调控 |
| 射频链路 | 需要 (接收机 + 发射机) | 不需要 |
| 信号放大 | 有 (AF) 或无 (DF) | 无 |
| 噪声引入 | 放大转发会引入噪声 | 不引入额外噪声 |
| 功耗 | 高 (射频收发 + 处理) | 极低 (仅控制电路) |
| 复杂度 | 高 | 低 |
| 成本 | 高 | 低 (潜在) |
| 延迟 | 有 (处理 + 转发) | 极低 (瞬时反射) |

### RIS 的优势

RIS 的核心优势在于其被动性。传统中继需要接收信号、进行处理 (放大或解码) 然后重新发射，这个过程需要完整的射频链路，功耗和成本都很高。放大转发 (AF) 中继还会放大噪声，降低信噪比。

RIS 只反射信号并调整相位，不涉及接收、放大或解码，因此不引入额外噪声，功耗仅为控制电路所需的毫瓦级或更低。此外，RIS 的瞬时反射几乎不引入延迟，对延迟敏感的 IoT 应用更为友好。

## 五、信道估计挑战

### 核心难题

RIS 的波束成形需要知道信道状态信息 (CSI)，包括发射机到 RIS 的信道和 RIS 到接收机的信道。然而，RIS 通常没有射频链路，无法自行测量信道——这是 RIS 面临的最核心技术挑战。

```
信道估计难题:

发射机 -----> [RIS] -----> 接收机
  信道 h1        信道 h2

需要估计: h1 和 h2 (级联信道 h1 * h2)

挑战:
  - RIS 无射频链路: 不能自行测量
  - 单元数量多: N 个单元 -> N 个未知参数
  - 需要导频开销: 培训时间随 N 增长
  - IoT 设备资源有限: 难以承担复杂估计
```

### 解决方案

目前主要的信道估计方案包括：

基于导频的估计：发射机发送已知的训练序列，接收机通过接收到的信号反推信道参数。对于 N 个单元的 RIS，需要 N 次以上的训练才能估计所有参数，开销巨大。

基于码本的波束扫描：预先定义一组相位配置 (码本)，依次尝试每种配置，选择信号最强的配置。这种方法不需要显式信道估计，但扫描时间随码本大小增长。

机器学习预测：利用历史信道数据和环境特征，通过深度学习模型预测当前信道状态。这种方法有望减少实时估计开销，但需要训练数据和足够的计算资源。

信道估计仍然是 RIS 技术走向实用的主要瓶颈之一，特别是对于资源受限的 IoT 场景。

## 六、RIS 辅助 LPWAN 覆盖扩展

### LoRaWAN 覆盖增强

LoRaWAN 设备通常部署在远距离、深度覆盖场景。RIS 可以在以下情况提供帮助：

```
LoRaWAN + RIS 部署场景:

场景 1: 城市峡谷
  设备在两栋建筑之间, 被建筑遮挡
  RIS 安装在其中一栋建筑外墙
  -> 将设备信号反射到网关方向
  -> 创造虚拟视距路径

场景 2: 室内到室外
  设备在室内, 网关在室外
  RIS 安装在窗户外侧
  -> 将室内信号反射到室外网关
  -> 改善室内到室外的穿透

场景 3: 深度阴影区
  设备在山丘或建筑物后方 (阴影区)
  RIS 安装在制高点
  -> 将信号越过障碍物反射到网关
  -> 恢复被遮挡的通信链路
```

在 LoRaWAN 场景中，RIS 的一个重要优势是可以采用预配置的固定反射模式。如果设备和网关的位置相对固定，RIS 不需要实时调整相位，只需要一次性配置最优反射方向，后续无需控制能量输入，实现真正的零功耗覆盖增强。

### NB-IoT 覆盖增强

NB-IoT 已经通过重复传输实现了极强的覆盖增强 (MCL 达 164dB)。RIS 可以作为补充手段，在特定盲区提供额外增益，减少所需的重复次数，从而降低传输延迟和能耗。

## 七、相位优化

### 优化问题

RIS 的相位优化目标是找到一组单元相位配置，使得接收端的信号功率 (或信干噪比) 最大化：

```python
# RIS 相位优化问题 (简化)
# 目标: 最大化接收信号功率

# 已知: 级联信道 h = h1 * diag(theta) * h2
#   h1: 发射机到 RIS 的信道 (1 x N)
#   h2: RIS 到接收机的信道 (N x 1)
#   theta: RIS 相位向量 [theta_1, ..., theta_N]

# 优化问题:
#   max   |h1 * diag(exp(1j*theta)) * h2|^2
#   s.t.  theta_n in {0, 2*pi}  (1-bit) 或 [0, 2*pi) (连续)

# 理想解 (连续相位):
#   theta_n = -angle(h1_n) - angle(h2_n) + constant
#   -> 使所有路径同相叠加

# 1-bit 量化约束:
#   theta_n in {0, pi}
#   -> 只能近似最优, 损失约 4 dB
```

### 算法选择

相位优化是一个非凸问题，特别是当考虑离散相位约束时。主要的求解算法包括：

交替优化：将多变量优化分解为一系列单变量子问题，交替求解。收敛性有保证但可能陷入局部最优。

半定松弛 (SDR)：将非凸约束松弛为半定约束，通过凸优化求解，然后从松弛解中恢复原始解。性能较好但计算复杂度高。

深度学习方法：训练神经网络直接从信道特征预测相位配置。推理速度快，适合实时应用，但训练阶段需要大量数据。

对于 IoT 场景，简化的方法可能更为实用。当设备位置固定时，可以离线计算最优相位并存储；当设备移动时，可以使用粗粒度码本扫描代替精确优化。

## 八、部署场景

### 覆盖盲区

在城市环境中，建筑物遮挡经常造成覆盖盲区。在建筑外墙上安装 RIS，可以将基站的信号反射到盲区，为 IoT 设备提供覆盖。这种方案不需要新的基站或中继，成本远低于传统方案。

### 室内覆盖

室外基站的信号穿透建筑外墙时损耗严重 (10-30dB)，导致室内覆盖不佳。RIS 可以安装在窗户或外墙上，将室外信号引导到室内，改善室内 IoT 设备的通信质量。

### 智能工厂

工业环境中大量金属设备和结构造成严重的多径和非视距问题。在工厂天花板或墙壁上部署 RIS，可以将信号定向反射到被遮挡的区域，确保工业传感器的可靠通信。

### 智慧城市

在建筑物外立面部署 RIS，可以增强街道层面的信号覆盖。这对于部署在路灯杆、地下管道、公交站等位置的 IoT 传感器尤为有用。

## 九、当前局限性

### 技术挑战

信道估计开销仍然是最大的瓶颈。对于大规模 RIS (数百个单元)，完整的信道估计需要大量导频开销，这在 IoT 的低速率、低功耗约束下尤其困难。

相位优化的计算复杂度也很高。最优解需要迭代优化或矩阵分解，超出了大多数 IoT 设备的处理能力。虽然可以采用简化方法，但性能会有所损失。

实际增益低于理论值。单元损耗、相位量化误差、通道估计不精确、表面安装损耗等因素都会降低实际增益，通常只有理论值的 50-70%。

### 标准化和商用

RIS 尚未纳入 3GPP 或 IEEE 的正式标准。虽然 3GPP 在 Release 19 中开始研究 RIS 相关课题，但距离标准化还有相当距离。没有标准意味着缺乏互操作性，不同厂商的 RIS 产品无法通用。

成本方面，虽然 RIS 的理论成本低于传统中继，但目前仍处于研发阶段，大规模制造尚未实现，单位成本仍然较高。低成本的印刷电子技术有望在未来降低 RIS 制造成本。

技术成熟度方面，RIS 目前主要处于实验室验证和早期现场试验阶段，大规模商用部署还需要解决工程化、可靠性、维护等实际问题。

## 十、研究方向

### 主动 RIS

传统 RIS 是完全被动的，只反射不放大。主动 RIS 在每个单元中引入微型放大器，在反射的同时对信号进行放大。这可以弥补 RIS 的双重路径损耗，显著提升覆盖增强效果，代价是增加了功耗和复杂度。

### STAR-RIS

同时透射和反射 RIS (Simultaneous Transmitting And Reflecting RIS, STAR-RIS) 是一种新型 RIS 架构，支持信号同时透射和反射。部署在窗户上的 STAR-RIS 可以同时服务室内和室外的设备，消除传统 RIS 只能服务反射侧的限制。

### RIS 辅助感知

RIS 不仅可以用於通信，还可以辅助雷达感知。通过控制反射波束，RIS 可以增强对特定区域的探测能力，实现通信与感知的一体化。这对于需要同时进行数据传输和环境感知的 IoT 场景非常有价值。

### 与 6G 融合

在 6G 愿景中，RIS 被视为网络的基础元素之一。未来的无线网络可能将 RIS 集成到建筑外墙、室内装饰、街道设施中，形成"智能无线电环境"，使无线传播本身变得可控和可编程。6G 频段 (太赫兹) 波长极短，RIS 单元可以做得非常小，在相同的面积上集成更多单元，获得更高的增益。

## 十一、实际案例：智慧校园 IoT 覆盖增强

考虑一个大学校园的 NB-IoT 网络覆盖增强场景：

```
场景设定:
  - NB-IoT 基站部署在主教学楼楼顶, 覆盖半径约 1km
  - 图书馆地下层 (B1) 有环境监测传感器: 温湿度, CO2, 人员计数
  - 地下层信号被土壤和混凝土遮挡, 处于覆盖盲区
  - 传感器数量: 20 个, 分布在 500m2 的地下空间

问题分析:
  - 基站到地下层直射路径被完全遮挡
  - 地下层入口处有微弱信号 (-120 dBm), 不足以支持 NB-IoT 通信
  - 需要至少 -108 dBm 才能进行 CE Level 2 通信

RIS 部署方案:
  - 在图书馆外墙上安装 2m2 RIS 面板
  - 工作频率: 900MHz (NB-IoT Band 8)
  - 单元尺寸: 约 16.7cm (半波长)
  - 单元数量: 约 100 个 (10x10 阵列)
  - 反射方向: 预配置为将基站信号反射向地下层入口

  - 控制电路: 小型太阳能板 + 超级电容供电
  - 相位配置: 离线优化后固化, 无需实时调整
  (因为基站和图书馆位置固定, 最优反射方向不变)

性能评估:
  理论增益: N^2 = 100^2 = 40 dB
  实际效率: 约 60%
  实际增益: 40 * 0.6 = 24 dB (保守估计 15 dB)

  地下层入口处信号:
    原始: -120 dBm (不可用)
    RIS 增强后: -120 + 15 = -105 dBm (勉强可用)
    配合 NB-IoT 重复传输 (CE Level 2): 通信成功率 > 95%

  地下层深处:
    入口处信号通过室内传播衰减约 10-15 dB
    深处信号: -105 - 15 = -120 dBm
    需要更多重复 -> CE Level 3, 128 次重复
    仍可通信, 但数据速率极低, 功耗较高

方案效果:
  - 地下层传感器实现覆盖, 无需新增基站
  - RIS 成本: 约 500-1000 USD (面板 + 安装)
  - 对比: 新建微基站约 5000-10000 USD
  - RIS 功耗: < 1W (仅控制电路, 太阳能供电)
  - 维护: 几乎零维护 (无活动部件, 无射频链路)

限制:
  - 增益有限, 深层区域仍需大量重复传输
  - 预配置方案不适应环境变化 (如新建筑遮挡)
  - 实际增益受墙面安装角度和极化匹配影响
```

## 总结

可重构智能表面 (RIS) 是一种将无线传播环境从不可控变为可编程的革命性技术。本文系统介绍了以下内容：

RIS 的核心概念是通过大量可编程反射单元协同控制电磁波的反射相位，实现波束成形。这代表了一种范式转变——从适应信道到控制信道本身。

RIS 对 IoT 的价值在于被动覆盖增强：不增加基站功率、不引入额外噪声、功耗极低，适合能量受限的 IoT 场景，特别是非视距和深度覆盖场景。

硬件方面，RIS 由亚波长反射单元阵列组成，使用 PIN 二极管或变容二极管实现相位控制。单元数量越多，波束成形增益越大 (理论正比于 N 的平方)。

与传统中继相比，RIS 不需要射频链路、不放大噪声、功耗和成本更低，但增益也受限于被动反射的特性。

信道估计是 RIS 面临的核心技术挑战，特别是对于资源受限的 IoT 设备。基于码本的扫描和机器学习预测是有前途的解决方向。

在 LPWAN 场景中，RIS 可以通过预配置的固定反射模式提供零功耗覆盖增强，非常适合设备位置固定的 IoT 部署。

RIS 目前仍处于早期阶段，面临标准化缺失、成本较高、实际增益低于理论值等挑战。主动 RIS、STAR-RIS、RIS 辅助感知等研究方向正在拓展其能力边界。

在 6G 愿景中，RIS 有望成为无线网络的基础元素，使整个物理环境变成可编程的智能无线电空间。
