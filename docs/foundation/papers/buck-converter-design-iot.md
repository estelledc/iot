# Buck降压转换器在IoT设备中的设计要点
> **难度**: 黄 中级 | **领域**: 开关电源设计 | **阅读时间**: 约 20 分钟

## 引言

想象一个水龙头和一个水桶的关系: 你需要把高位水塔的高压水,变成水桶里平稳的低压水。如果直接开大水龙头,水花四溅(纹波大);如果频繁快速地开关水龙头,每次接一点水,水桶里就能保持平稳的水位。开关频率越快,水桶里的水位越平稳,但水龙头的磨损也越大(开关损耗)。

Buck降压转换器就是这个"快速开关的水龙头"。它通过高频开关将较高输入电压转换为较低输出电压,是IoT设备中最常用的电源拓扑。锂电池3.7V给MCU 3.3V供电,5V USB给3.3V系统供电,都离不开Buck。但设计一个高效、低噪声、小体积的Buck并不容易,需要在效率、尺寸、纹波和EMI之间做出权衡。

本文系统讲解Buck转换器在IoT设备中的设计要点,从原理到元器件选型,从轻载效率到PCB布局,帮助你做出正确的设计决策。

## 1. Buck转换器工作原理

### 1.1 基本拓扑

Buck转换器由四个核心元件组成:

```
       Q1(高侧开关)
Vin ---+---[ MOSFET ]---+---[ 电感L ]---+--- Vout
       |                |               |
       |           Q2(低侧开关)      Cout
       |           或续流二极管         |
       |                |               |
      Cin              GND             GND
```

两个工作阶段:
1. Q1导通,Q2关断: 输入通过电感向负载供电,电感电流线性上升,电感储能
2. Q1关断,Q2导通(或二极管续流): 电感释放储能,电流通过Q2/二极管续流,电感电流线性下降

### 1.2 伏秒平衡

稳态下电感满足伏秒平衡:

```
(Vin - Vout) x ton = Vout x toff

占空比 D = ton / (ton + toff) = Vout / Vin
```

例如: Vin=3.7V, Vout=3.3V, D = 3.3/3.7 = 89.2%

注意: 这是理想占空比,实际因开关损耗和压降,D会略高。

### 1.3 电感电流波形

电感电流呈三角波,围绕平均值(IL_avg = Iload)上下波动:

```
      ton          toff
   /|            \
  / |             \
 /  |              \
/   |               \
    Delta_IL/2  Delta_IL/2
```

纹波电流: Delta_IL = (Vin - Vout) x D / (fsw x L)

## 2. 连续导通模式与断续导通模式

### 2.1 CCM(连续导通模式)

电感电流始终大于零,每个开关周期都完整经历储能和释能阶段。

- 适用于: 中大负载电流
- 输出纹波: 较小且可预测
- 效率: 在中高负载时最佳
- 缺点: 低负载时效率下降(开关损耗占比增大)

### 2.2 DCM(断续导通模式)

电感电流在toff期间降到零,存在一段电感电流为零的死区时间。

- 适用于: 轻负载
- 输出纹波: 较大,与负载有关
- 效率: 轻载时比CCM高(减少了不必要的开关)
- 缺点: 右半平面零点,环路补偿更复杂

### 2.3 临界电流

CCM与DCM的分界点:

```
Icrit = Delta_IL / 2 = (Vin - Vout) x D / (2 x fsw x L)
```

当Iload > Icrit时为CCM,反之为DCM。增大电感值L可降低临界电流,扩大CCM范围。

### 2.4 BCM(临界导通模式)

电感电流恰好降到零时立即开通Q1,常用于反激和LED驱动,IoT的Buck中较少使用。

## 3. 元器件选型

### 3.1 电感选型

电感是Buck转换器中最关键的元件,选型需关注四个参数:

1. **电感值(L)**: 决定纹波电流大小
   - 典型取值使纹波电流为最大负载电流的20-40%
   - L_min = (Vin_max - Vout) x D_min / (fsw x Delta_IL_max)

2. **饱和电流(Isat)**: 电感值下降30%时的电流
   - 必须大于峰值电感电流: Isat > Iload_max + Delta_IL/2
   - 留30%裕量

3. **额定电流(Irsm)**: 温升40C时的电流
   - 必须大于最大负载电流

4. **直流电阻(DCR)**: 影响铜损
   - DCR越低效率越高,但体积越大
   - IoT建议DCR < 100mOhm

电感类型选择:
| 类型 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 绕线屏蔽 | 低DCR,高Isat | 体积大 | 大电流Buck |
| 多层片式(MLCI) | 体积小 | Isat较低 | 小电流IoT |
| 合金电感 | 低DCR,抗饱和 | 成本高 | 大电流/高效率 |

### 3.2 输出电容选型

输出电容平滑电感纹波电流,提供瞬态负载电流。

关键参数:
1. **电容值**: 满足纹波和瞬态要求
   - 纹波电压: Delta_Vout = Delta_IL / (8 x fsw x Cout)
   - 瞬态压降: Delta_Vtr = Istep / (2 x pi x fBW x Cout)

2. **ESR(等效串联电阻)**: 影响纹波和损耗
   - 纹波分量: Delta_Vout_esr = Delta_IL x ESR
   - 陶瓷电容ESR极低(1-10mOhm),是首选

3. **电压降额**: 陶瓷电容在直流偏置下实际容值下降
   - X5R/X7R材质在50%额定电压时容值可能下降30-50%
   - 建议: 选择额定电压为工作电压2倍以上的电容

### 3.3 输入电容选型

输入电容提供开关瞬态电流,防止输入电压跌落。

- 必须低ESR,优先陶瓷电容
- 容值通常4.7-22uF
- 布局: 必须紧贴IC的Vin引脚

## 4. 开关频率权衡

### 4.1 频率对设计的影响

| 参数 | 低频(100-500kHz) | 高频(1-3MHz) |
|------|------------------|--------------|
| 电感尺寸 | 大 | 小 |
| 电容尺寸 | 大 | 小 |
| 开关损耗 | 低 | 高 |
| 效率(满载) | 高 | 稍低 |
| EMI | 低频段集中 | 高频段集中 |
| 瞬态响应 | 慢 | 快 |

### 4.2 IoT中的频率选择

IoT设备通常优先选择1-2MHz:
- 电感和电容可以做到较小封装(0402/0603)
- 效率损失可以通过低Iq设计在睡眠模式补偿
- PCB面积显著减小,利于小型化

但若效率是首要目标,500kHz-1MHz是更优选择,尤其是输入电压较高时。

## 5. 反馈环路补偿基础

### 5.1 环路稳定性要求

Buck转换器的反馈环路必须稳定,否则输出会振荡。稳定性判据:
- 相位裕度 > 45度(建议60度)
- 增益裕度 > 6dB

### 5.2 Type II补偿网络

电流模式Buck常用Type II补偿(一个零点,一个极点):

```
       R1
FB ---[___]---+--- COMP
               |
          [___]C1
               |
          [___]C2
               |
              GND
```

- R1和C1产生零点,提升相位裕度
- C2产生高频极点,滤除开关噪声
- 许多现代Buck IC内置补偿,无需外部元件

### 5.3 环路测量

- 网络分析仪注入扰动信号到反馈环路
- 测量开环传递函数
- 确保穿越频率低于1/10开关频率

IoT项目中,若使用内置补偿的IC,通常不需要手动调环路。

## 6. 轻载效率与工作模式

### 6.1 PWM模式

固定频率,通过调节占空比控制输出。轻载时开关损耗占比大,效率下降明显。

### 6.2 PFM模式(脉冲频率调制)

轻载时降低开关频率,甚至跳过一些脉冲。每次开关都传递一包能量,然后等待输出电压跌落到阈值再发下一包。

- 优势: 轻载效率显著提高,尤其是uA级负载
- 劣势: 输出纹波增大,频率可变导致EMI难以预测

### 6.3 PFM/PWM自动切换

大多数IoT Buck IC支持自动切换:
- Iload > 阈值(通常50-100mA): PWM模式,低纹波
- Iload < 阈值: PFM模式,高效率

这是IoT应用的最佳选择,兼顾满载和轻载效率。

### 6.4 超低Iq Buck

TPS62840是超低Iq Buck的典型代表:

```
Iq = 60nA(仅PFM模式)
Vin = 1.8-6.5V
Vout = 0.8-5.0V(可调)
Iload_max = 750mA
效率: >90%(10uA-1mA范围)
封装: 2.0 x 1.5mm SOT-563
```

睡眠状态下,TPS62840仅消耗60nA,对CR2032纽扣电池的寿命几乎无影响。

## 7. EMI与输入滤波

### 7.1 EMI来源

Buck的EMI主要来自:
1. 开关边沿的高频振荡(数十至数百MHz)
2. 输入侧的脉冲电流(非连续,高di/dt)
3. 功率环路寄生电感引起的振铃

### 7.2 输入滤波器设计

```
Vin ---[ Lf ]---+--- Buck IC Vin
                 |
                Cf
                 |
                GND
```

- Lf: 1-10uH电感,抑制输入电流纹波
- Cf: 0.1-1uF陶瓷电容,提供高频去耦
- 注意: 输入滤波器不能与Buck输入阻抗形成谐振

### 7.3 缓冲电路(RCD Snubber)

开关节点振铃可用RCD缓冲器抑制:

```
SW ---[ R ]---+---
              |
             [C]
              |
             GND
```

- R: 10-100Ohm,消耗振铃能量
- C: 100pF-1nF,与R一起衰减高频振铃
- 代价: 缓冲器自身有损耗,效率略降

## 8. 热设计

### 8.1 功耗计算

Buck的功耗 = Pin - Pout = Vin x Iin - Vout x Iload

近似效率法: Pdiss = Pout x (1/eff - 1)

例如: Vout=3.3V, Iload=300mA, eff=92%:
- Pdiss = 3.3 x 0.3 x (1/0.92 - 1) = 86mW

### 8.2 结温估算

```
Tj = Ta + Pdiss x Rth_ja
```

例如: Ta=60C, Pdiss=86mW, Rth_ja=80C/W(DFN封装):
- Tj = 60 + 0.086 x 80 = 66.9C(安全)

### 8.3 散热优化

- 选择热阻低的封装(DFN/QFN优于SOT-23)
- PCB铜箔作为散热面: 增加IC下方和周围的铜箔面积
- 热过孔: IC底部裸焊盘通过热过孔连接内层或底层地平面
- 避免将热敏感元件(如温度传感器)放在Buck IC附近

## 9. 实战设计示例: 锂电池3.7V到MCU 3.3V

### 9.1 需求定义

- 输入: 单节锂电池,3.0-4.2V
- 输出: 3.3V, +/-3%
- 最大负载: 200mA(MCU+RF发射峰值)
- 平均负载: 20mA(MCU运行+传感器采样)
- 睡眠电流: 5uA(MCU深度睡眠)
- 效率目标: >85%(平均负载)

### 9.2 IC选择

选择TPS62840:
- Vin范围1.8-6.5V,覆盖锂电池全电压
- Iq=60nA,睡眠功耗几乎为零
- 内置补偿,外部元件少
- 自动PFM/PWM切换

### 9.3 外围元件计算

```
fsw = 1.5MHz(TPS62840默认)
D_max = 3.3/3.0 = 110% -- 不可能!
// Vin_min=3.0V时需要Buck-Boost,但3.0V只是截止电压
// 实际工作最低Vin=3.3V+压差,约3.4V
// 大部分时间Vin在3.5-4.0V范围

// 电感选择
Delta_IL = 0.3 x Iload_max = 0.3 x 200mA = 60mA
L = (Vin_nom - Vout) x D / (fsw x Delta_IL)
L = (3.7 - 3.3) x 0.89 / (1.5M x 0.06) = 3.96uH
// 选择4.7uH标准值

// 输出电容
Delta_Vout = 10mV(目标纹波)
Cout = Delta_IL / (8 x fsw x Delta_Vout)
Cout = 0.06 / (8 x 1.5M x 0.01) = 0.5uF
// 考虑直流偏置降额,选择10uF/6.3V X5R 0402
```

### 9.4 最终BOM

| 元件 | 型号 | 封装 | 参数 |
|------|------|------|------|
| U1 | TPS62840 | SOT-563 | Buck, 60nA Iq |
| L1 | MLD2012-470K | 0805 | 4.7uH, Isat=350mA |
| Cin | GRM155R60J475ME | 0402 | 4.7uF/6.3V X5R |
| Cout | GRM155R60J106M | 0402 | 10uF/6.3V X5R |

### 9.5 效率预估

| 工作模式 | 负载 | TPS62840效率 |
|----------|------|-------------|
| 睡眠 | 5uA | ~50%(PFM,但Iq极低) |
| 传感器采样 | 20mA | ~88% |
| RF发射 | 200mA | ~91% |
| 平均 | - | ~87% |

## 10. PCB布局指南

### 10.1 功率环路最小化

Buck的EMI和效率高度依赖PCB布局。最关键的原则: **功率环路面积最小化**。

两个关键环路:
1. 高侧导通环路: Cin -> Q1 -> L -> Cin(通过GND)
2. 低侧续流环路: L -> Q2/二极管 -> Cin(通过GND)

这两个环路应尽可能短而宽,减少寄生电感。

### 10.2 布局规则

1. Cin紧贴IC的Vin和GND引脚,距离<2mm
2. SW节点铜箔面积最小化(减少辐射),但要满足载流
3. 电感尽量靠近SW引脚
4. Cout紧贴电感输出端
5. 反馈分压电阻靠近FB引脚,走线远离开关节点
6. 模拟地(AGND)和功率地(PGND)单点连接
7. IC底部裸焊盘(Exposed Pad)通过热过孔连接地平面

### 10.3 常见布局错误

```
错误1: Cin离IC太远
  Vin ---[长走线]--- IC Vin
  长走线寄生电感导致输入振铃

错误2: SW走线大面积铺铜
  SW节点是高频方波,大面积铺铜相当于天线,辐射EMI

错误3: 反馈走线经过SW节点
  SW的高频噪声耦合到反馈走线,导致输出不稳定
```

## 总结

Buck降压转换器是IoT设备电源设计的核心。理解CCM/DCM模式对轻载效率的影响、正确选择电感和电容、合理设置开关频率、优化PCB布局,是实现高效率低噪声Buck的关键。对IoT而言,超低Iq Buck(如TPS62840)在睡眠模式下的功耗几乎为零,是实现长续航的利器。记住: Buck设计不只是选一个IC,外围元件和PCB布局同样决定了最终性能。
