# PM2.5激光散射粒子传感器精度分析

> **难度**：🟡 中级 | **领域**：颗粒物检测、光散射、空气质量 | **阅读时间**：约 25 分钟

## 日常类比

想象你站在一间暗房里，手里拿着一支激光笔。空气看起来很干净，但按下激光笔后一道光柱出现，里面密密麻麻全是细小的亮点——那就是悬浮颗粒物。平时看不见，但在强光照射下它们把光散射到各个方向，才变得可见。

PM2.5 传感器的工作原理一样：微型激光源照亮空气样本，颗粒物散射光线，旁边的光电探测器接收散射光，光越强说明颗粒越多越大。然后芯片通过 Mie 散射理论换算出 PM1.0、PM2.5、PM10 浓度。

听起来简单，但"散射光强"和"空气里到底有多少灰"之间隔着一座叫精度的桥。湿度高了颗粒吸水变胖、不同成分折射率不同、出厂标定用标准粉尘而非真实空气——每一步都可能引入偏差。这篇文章从原理到工程，把精度问题讲透。

## 1. 颗粒物尺寸分类

### 1.1 什么叫 PM1 / PM2.5 / PM10

PM 是 Particulate Matter（颗粒物）的缩写，后面的数字代表空气动力学当量直径的上限，单位是微米（μm）：

- **PM10**：直径 ≤ 10 μm 的颗粒，也叫"可吸入颗粒物"，能进入上呼吸道
- **PM2.5**：直径 ≤ 2.5 μm 的颗粒，也叫"细颗粒物"，能深入肺泡甚至进入血液循环
- **PM1.0**：直径 ≤ 1.0 μm 的颗粒，也叫"超细颗粒物"，穿透力最强

### 1.2 为什么要分级测量

日常类比：查户口。"这条街住了 1000 人"什么也判断不了，但"其中 200 老人、300 小孩、500 青壮年"信息量就大多了。颗粒物同理——PM10 主要沉积在上呼吸道引发咳嗽，PM2.5 能深入肺部引发心血管疾病，PM1.0 甚至能穿过肺泡进入血液。不同粒径对健康影响差异巨大，所以必须分级。

### 1.3 粒径分布特征

| 分类 | 粒径范围 | 主要来源 | 健康影响 | 沉积区域 |
|------|---------|---------|---------|---------|
| PM10 | 2.5–10 μm | 扬尘、花粉、矿山 | 上呼吸道刺激 | 鼻咽、气管 |
| PM2.5 | 1.0–2.5 μm | 燃煤、机动车、工业 | 心血管与呼吸系统 | 支气管、肺泡 |
| PM1.0 | < 1.0 μm | 二次气溶胶、燃烧核 | 全身性炎症反应 | 肺泡、血液循环 |

真实大气中的颗粒物粒径分布通常是多模态的——有一个粗模态峰值在 5–10 μm，一个积聚模态峰值在 0.3–0.5 μm。传感器需要在这两个模态之间正确"切分"出 PM2.5 的贡献。

## 2. 激光散射原理

### 2.1 光散射的物理图像

当一束光照射到颗粒上时，颗粒表面的电子被激发产生振荡，向四周重新辐射电磁波——这就是散射。散射光的强度和角度分布取决于两个关键参数：

1. **颗粒尺寸参数** $x = \frac{\pi d}{\lambda}$，其中 $d$ 是颗粒直径，$\lambda$ 是入射光波长
2. **相对折射率** $m = \frac{n_p}{n_m}$，其中 $n_p$ 是颗粒折射率，$n_m$ 是介质折射率

### 2.2 三种散射区间

| 区间 | 条件 | 散射规律 | 适用传感器 |
|------|------|---------|-----------|
| Rayleigh 散射 | $x \ll 1$（颗粒远小于波长） | 散射强度 $\propto d^6/\lambda^4$，前后对称 | 超细颗粒检测（受限） |
| Mie 散射 | $x \approx 1$（颗粒与波长相近） | 散射强度随 $d$ 非线性变化，前向散射增强 | **激光散射 PM 传感器的主战场** |
| 几何光学 | $x \gg 1$（颗粒远大于波长） | 折射 + 衍射 + 反射，散射光与截面成正比 | 大颗粒检测 |

对于 650 nm 红色激光，2.5 μm 颗粒的尺寸参数 $x \approx 12$，正处于 Mie 散射区间——前向散射远强于后向散射，且散射光强度与粒径呈复杂的非线性关系。

### 2.3 Mie 散射理论要点

Mie 理论是麦克斯韦方程组在均匀球形颗粒上的精确解析解。核心结论：

- 散射光强度不是粒径的单调函数——存在共振峰和谷
- 前向散射（与入射光同侧）通常远强于后向散射
- 散射光的角分布随粒径变化剧烈：小颗粒近似对称，大颗粒高度前向集中
- 非球形颗粒的实际散射模式会偏离 Mie 理论预测

这最后一条是精度问题的根源之一：真实颗粒不是完美球体，但理论假设它们是。

### 2.4 从散射信号到质量浓度

传感器实际测量的是**数浓度**（单位体积内颗粒个数），但国标要求报告的是**质量浓度**（μg/m³）。转换需要假设颗粒的密度和形状：

$$C_{mass} = \frac{\pi}{6} \cdot \rho \cdot \sum_i d_i^3 \cdot N_i$$

其中 $\rho$ 是颗粒密度（通常假设为 1.5–1.65 g/cm³），$d_i$ 是第 $i$ 个粒径段的代表直径，$N_i$ 是该粒径段的数浓度。这个假设本身就引入了不确定性——如果实际颗粒密度偏离假设值，质量浓度就会系统性地偏高或偏低。

## 3. 传感器内部结构

### 3.1 核心组件

一个典型的激光散射 PM 传感器由以下四部分组成：

```
          ┌──────────────────────────────────┐
          │         传感器外壳                │
          │                                  │
          │   ┌─────┐                        │
          │   │激光管│───►───┐                │
          │   └─────┘       │ 光束            │
          │                 ▼                │
  空气流入 ──►  [检测腔]  ··· ● ···  ──► 空气流出
          │                 ▲ 散射光         │
          │                 │                │
          │            ┌────────┐            │
          │            │光电探测器│           │
          │            └────────┘            │
          │                                  │
          │   ┌─────┐                        │
          │   │ 风扇 │  ← 驱动气流            │
          │   └─────┘                        │
          └──────────────────────────────────┘
```

### 3.2 激光源与光路

大多数低成本 PM 传感器使用 650–680 nm 红光激光二极管（1–5 mW），单色性和方向性好，能在检测腔内形成高亮窄带照射区。SPS30 等高端型号使用蓝光激光（450 nm 附近），更短波长让散射信号更强，但对 Mie 区间的改善有限。

激光通过准直透镜形成窄光束，光束与气流交叉的区域即**敏感体积**——体积越小统计波动越大。光路设计的关键细节是**光阱（light trap）**：光束穿过检测腔后必须被吸收而非反射，否则反射光成为强背景噪声。

### 3.3 光电探测器

通常采用光电二极管或 APD，安装在检测腔侧面约 60°–90° 位置，前面加窄带滤光片抑制环境光。探测器输出经跨阻放大器（TIA）转为电压再 ADC 数字化。每个颗粒通过敏感体积产生一个脉冲——脉冲高度对应粒径，脉冲计数对应数量，即**单粒子计数模式**。

### 3.5 气流驱动

传感器需要一个稳定的气流让空气样本穿过检测腔。两种主流方案：

- **风扇驱动**（如 PMS5003、SDS011）：风速高、采样量大，但功耗高（约 30–80 mA），有运动部件寿命问题
- **热敏电阻驱动**（如 SPS30）：利用加热元件产生热对流，无运动部件、寿命长，但采样量小、响应慢

## 5. 精度影响因素

### 5.1 湿度——最大的精度杀手

这是精度问题的头号元凶。日常类比：干面粉上喷水雾——面粉颗粒吸水后表面形成水膜，体积膨胀、折射率改变。传感器看到的不再是"面粉"，而是"裹了水膜的面粉"。

具体影响机制：

1. **吸湿增长**：硫酸盐、硝酸盐等二次气溶胶具有强吸湿性。RH > 60% 后颗粒表面形成水膜粒径增大，RH > 80% 增长加速，RH > 95% 时粒径甚至增大 2–3 倍
2. **凝露干扰**：RH 接近 100% 或温度骤降时，检测腔内凝结微小水滴被误计为颗粒物
3. **光学串扰**：水膜改变折射率，使 Mie 散射角分布变化，原有标定曲线不再适用

实验数据表明，RH 从 40% 升到 90% 时，PMS5003 读数可偏高 30%–100%。

### 5.2 颗粒物化学成分

Mie 散射的强度不仅取决于颗粒尺寸，还取决于折射率。不同化学成分的颗粒折射率差异显著：

| 颗粒类型 | 实部折射率 | 虚部折射率（吸收） | 散射效率 |
|---------|-----------|------------------|---------|
| 硫酸铵 | 1.53 | ≈ 0 | 高 |
| 黑碳 | 1.95 | 0.66 | 低（强吸收） |
| 矿物粉尘 | 1.52 | 0.008 | 中 |
| 海盐 | 1.50 | ≈ 0 | 高 |
| 有机气溶胶 | 1.45–1.55 | 0.001–0.02 | 中–高 |

传感器出厂标定通常使用亚利桑那标准粉尘（ISO 12103-1 A2）或类似的标准粒子，其折射率约为 1.54。但真实大气中是混合气溶胶，黑碳含量高时传感器会低估（黑碳吸收入射光，散射弱），而海盐/硫酸盐含量高时可能高估。

### 5.3 粒径分布偏移

传感器的粒径分级能力有限。它不是物理筛分（像冲击式采样器那样），而是通过散射光脉冲高度来"推算"颗粒粒径。这种间接推算有几个问题：

- 同样粒径的非球形颗粒，不同取向的散射截面差异可达 2–3 倍
- 粒径接近分界线（2.5 μm）的颗粒容易被误分
- 高浓度时多个颗粒同时进入敏感体积，脉冲重叠导致漏计或误计

### 5.4 温度、海拔与老化

温度变化通过两个渠道影响精度：激光二极管输出功率随温度漂移（约 -0.2 mW/°C），以及空气密度变化改变颗粒运动特性。高海拔气压低、空气密度低，同样质量的颗粒物散射截面非线性变化。此外，激光输出随使用时间缓慢衰减，光学元件积累灰尘，风扇轴承磨损导致气流不稳——PMS5003 典型寿命约 3 年，之后精度可能下降 20%–30%。SPS30 无运动部件标称 10 年寿命，但激光衰减同样存在。

## 6. 校正算法

### 6.1 湿度校正

最常用的校正方法是引入湿度补偿因子：

$$C_{corrected} = \frac{C_{raw}}{1 + \alpha \cdot f(RH)}$$

其中 $f(RH)$ 是湿度的某种增长函数，常用形式有：

- **线性补偿**：$f(RH) = (RH - 40) / 100$，当 RH > 40% 时启用
- **指数补偿**：$f(RH) = \exp(\beta \cdot RH) - 1$，更贴合吸湿增长曲线
- **分段补偿**：RH < 60% 不补偿，60%–80% 线性补偿，> 80% 指数补偿

$\alpha$ 和 $\beta$ 参数需要针对具体传感器型号和当地气候条件进行标定。很多开源项目（如 AirGradient、Sensor.Community）提供了针对 PMS5003 的经验参数。

### 6.2 线性回归校正

在传感器附近有参比仪器（如 BAM-1020 或 TEOM-1405）的情况下，可以建立线性回归模型：

$$C_{ref} = a \cdot C_{sensor} + b$$

这种方法简单直接，但只在训练数据的浓度范围内有效。外推到高浓度或极端天气条件时可能失效。

### 6.3 多元回归与机器学习

更高级的校正方法同时利用温度、湿度、气压、季节、时间等多维特征：

```python
# 多特征校正示例（scikit-learn）
from sklearn.ensemble import GradientBoostingRegressor
from sklearn.model_selection import train_test_split
import numpy as np

# 特征: [传感器PM2.5, 温度, 湿度, 气压, 小时]
features = np.column_stack([
    sensor_pm25, temperature, humidity, pressure, hour_of_day
])

X_train, X_test, y_train, y_test = train_test_split(
    features, reference_pm25, test_size=0.2, random_state=42
)

model = GradientBoostingRegressor(
    n_estimators=200,
    max_depth=5,
    learning_rate=0.1,
    random_state=42
)
model.fit(X_train, y_train)

# 校正后 RMSE 通常可从 15-25 μg/m³ 降到 5-10 μg/m³
print(f"R² = {model.score(X_test, y_test):.3f}")
```

### 6.4 校正效果对比

| 校正方法 | 所需数据 | PM2.5 RMSE 典型值 | 优点 | 局限 |
|---------|---------|------------------|------|------|
| 无校正 | 无 | 20–40 μg/m³ | 零成本 | 高湿度时严重失真 |
| 湿度补偿 | RH | 10–20 μg/m³ | 简单易实施 | 无法修正成分偏差 |

## 7. 代码示例：UART 读取 PMS5003

### 7.1 PMS5003 协议简介

PMS5003 通过 UART（9600 baud, 8N1）输出 32 字节的数据帧，格式如下：

| 字节偏移 | 长度 | 内容 |
|---------|------|------|
| 0–1 | 2 | 帧头 0x42 0x4D（"BM"） |
| 2–3 | 2 | 帧长度 = 0x001C（28 字节） |
| 4–9 | 6 | CF=1 大气浓度（PM1.0, PM2.5, PM10） |
| 10–15 | 6 | CF=1 工厂浓度（用于校准） |
| 16–21 | 6 | 粒径段颗粒数（>0.3μm, >0.5μm, >1.0μm, >2.5μm, >5.0μm, >10μm） |
| 22–23 | 2 | 甲醛浓度（部分固件） |
| 28–29 | 2 | 校验和 |

实际使用中，偏移 4–9 的"大气浓度"就是校准后的输出值。

### 7.2 ESP32 读取代码

```python
# MicroPython: ESP32 + PMS5003 读取示例
from machine import UART
import struct
import time

class PMS5003:
    def __init__(self, uart_id=1, tx=17, rx=16):
        # 初始化 UART，9600 baud, 8N1
        self.uart = UART(uart_id, baudrate=9600, tx=tx, rx=rx)
        self.frame_len = 32  # PMS5003 每帧固定 32 字节

    def read(self):
        """读取一帧数据，返回 PM 浓度字典"""
        buf = b''
        start = time.ticks_ms()

        # 等待帧头 0x42 0x4D，超时 2 秒
        while time.ticks_diff(time.ticks_ms(), start) < 2000:
            if self.uart.any() > 0:
                b = self.uart.read(1)
                if b == b'\x42':
                    b2 = self.uart.read(1)
                    if b2 == b'\x4d':
                        # 找到帧头，读取剩余 30 字节
                        buf = b'\x42\x4d' + self.uart.read(30)
                        break
            time.sleep_ms(10)

        if len(buf) != self.frame_len:
            return None  # 超时或帧不完整

        # 校验和验证：前 28 字节累加应等于校验和
        checksum = struct.unpack('>H', buf[28:30])[0]
        if sum(buf[:28]) != checksum:
            return None

        # 解析各字段（大端序，无符号短整型）
        data = struct.unpack('>HHHHHH', buf[4:16])
        return {
            'pm1_0_atm':   data[0],  # 大气 PM1.0 (μg/m³)
            'pm2_5_atm':   data[1],  # 大气 PM2.5 (μg/m³)
            'pm10_atm':    data[2],  # 大气 PM10 (μg/m³)
            'pm1_0_fact':  data[3],  # 工厂 PM1.0
            'pm2_5_fact':  data[4],  # 工厂 PM2.5
            'pm10_fact':   data[5],  # 工厂 PM10
        }

    def passive_read(self):
        """主动模式：发送读取命令再接收"""
        # 主动读取命令: 42 4D E2 00 00 01 71
        cmd = b'\x42\x4d\xe2\x00\x00\x01\x71'
        self.uart.write(cmd)
        time.sleep_ms(200)
        return self.read()


# 使用示例
if __name__ == '__main__':
    sensor = PMS5003(uart_id=1, tx=17, rx=16)

    while True:
        result = sensor.read()
        if result:
            print(f"PM2.5 = {result['pm2_5_atm']} μg/m³")
        else:
            print("读取失败，检查接线")
        time.sleep(5)
```

### 7.3 注意事项

- **主动模式 vs 被动模式**：PMS5003 默认每秒自动上报（被动模式），上电即开始输出；主动模式需要先发命令再读取，功耗更低
- **风扇预热**：上电后风扇需要 30 秒预热，前几帧数据不可靠，建议丢弃
- **串口缓冲区**：ESP32 的 UART 缓冲区有限，长时间不读取会丢帧，建议用定时器中断读取
- **3.3V vs 5V**：PMS5003 供电 5V，但 UART 逻辑电平 3.3V，可以直接连 ESP32，无需电平转换

## 8. IoT 应用场景

### 8.1 户外空气质量监测站

传统国控站设备（BAM/TEOM）单台 10–30 万元，而基于 PMS5003 的节点成本不到 200 元。单个节点精度不及国控站，但高密度部署 + 机器学习校正可获得更好的空间分辨率。典型架构：`[PMS5003 + BME280] → [ESP32] → [Wi-Fi/MQTT] → [云端校正] → [可视化/API]`。关键设计要点：防晒防雨外壳（工作温度 -10–60°C）、防虫网、定期清洁光路窗口。

### 8.2 室内空气净化器

空气净化器是 PM2.5 传感器最大的商业市场。传感器安装在进风口附近，实时监测空气质量并自动调节风速档位。PMSA003I 因其小巧体积和 I2C 接口，成为空气净化器设计的热门选择。

工程挑战：净化器内部气流复杂、电机产生的电磁干扰可能影响传感器信号、用户期望即时响应但传感器需要数秒稳定——这些都需要在固件中做滤波和延迟处理。

### 8.3 可穿戴空气质量监测器

随着低功耗传感器和 BLE 的发展，可穿戴 PM2.5 监测器开始出现。这类设备需要在功耗、体积和精度之间做艰难的取舍——不可能塞进一个风扇进去，所以只能用热敏电阻驱动或纯扩散式采样，采样量小、统计噪声大。

SPS30 是目前最适合可穿戴场景的传感器，14 mm 高度、30 mA 工作电流、10 年寿命。但它仍然不是真正意义上的"可穿戴级"——未来的柔性传感器和 MEMS 超微型化可能改变这一局面。

### 8.4 智能建筑与 HVAC

商业楼宇 HVAC 系统根据 PM2.5 读数按需调节新风量：室外差时减少引入，好时增加自然通风，兼顾空气质量和能耗。此类应用对长期稳定性要求极高，通常需定期用参比仪器校准。

## 9. 总结与展望

### 9.1 核心要点回顾

1. PM2.5 传感器基于 Mie 散射原理，通过测量颗粒物散射光的强度推算浓度，但"散射光强"到"质量浓度"之间有多重假设
2. 湿度是最大的精度杀手——RH > 80% 时读数偏差可达 30%–100%，必须做湿度校正
3. 颗粒物化学成分影响折射率，传感器出厂标定用标准粉尘，面对真实混合气溶胶会产生系统性偏差
4. 低成本传感器（PMS5003、SDS011）适合高密度网络部署，但单个节点精度有限，需配合校正算法
5. SPS30 在寿命和精度上领先，但成本是国产方案的 2–3 倍

### 9.2 技术趋势

- **多波长检测**：用两种或以上波长的激光同时照射，可以反演颗粒的折射率虚部，区分黑碳与散射性气溶胶
- **机器学习校正即服务**：云端校正模型利用全国传感器网络 + 国控站数据持续训练，传感器节点只需上传原始读数
- **MEMS 微型化**：基于硅光工艺的片上散射检测，将激光源、波导、探测器集成在一颗芯片上，体积和功耗降低一个数量级
- **标准与认证**：EPA 的 Air Sensor Performance Targets 和中国的 HJ 1090-2020 正在为低成本传感器建立统一的性能评价标准
- **融合感知**：PM2.5 传感器与 CO₂、VOC、温湿度传感器融合，利用多变量约束提升整体估计精度

低成本 PM2.5 传感器不会取代精密仪器，但正在填补精密仪器永远无法覆盖的空白——每一个街角、每一间教室、每一个呼吸的瞬间。精度是问题，也是方向。
