# 去耦电容选型与PCB放置策略
> **难度**：🟢 初级 | **领域**：PCB电源完整性 | **阅读时间**：约 18 分钟

## 引言

想象一栋大楼的自来水系统。主管道源源不断地供水,但当某一层突然打开多个水龙头时,局部水压会瞬间下降。去耦电容就像安装在每个楼层的小型蓄水罐:平时蓄满水,用水高峰时迅速补水,保持水压稳定。在PCB上,IC芯片就是那个突然打开水龙头的"住户",去耦电容就是为它就地供能的"蓄水罐"。本文系统讲解去耦电容的原理、选型、放置和常见陷阱。

## 1 为什么需要去耦

### 1.1 IC开关产生电流瞬变

数字IC在输出信号翻转时,内部MOSFET开关动作会在极短时间内抽取大量电流:

```
电流瞬变示意:

  I
  ^      ___
  |     /   \
  |    /     \
  |   /       \
  |  /         \
  | /           \
  |/             \_______
  +----------------------> t
       几纳秒内
       di/dt极大
```

对于STM32这类MCU,单个IO翻转时的电流瞬变可达数十mA,上升时间在1-5ns范围。多个IO同时翻转时,瞬变电流更大。

### 1.2 电源分配网络(PDN)阻抗

PDN是从电源到IC之间所有阻抗的统称,包括:

- 电源模块输出阻抗
- PCB走线阻抗(典型值:1mm宽走线约5mOhm/mm + 0.7nH/mm)
- 过孔阻抗(约0.5nH/过孔)
- 电源平面感抗

如果PDN阻抗在某个频率上过高,IC在该频率抽取电流时,电源电压就会出现明显跌落(即电压纹波增大)。

### 1.3 目标:低阻抗PDN

去耦电容的核心目标是在尽可能宽的频率范围内,使PDN阻抗保持在目标值以下:

```
|Z|          无去耦电容
  ^          /\
  |         /  \
  |        /    \___   有去耦电容
  |       /         \___________
  |      /                      \_____
  |_____/______________________________> f
       1kHz   10kHz  1MHz  100MHz  1GHz

  目标: |Z| < Ztarget (例如 50mOhm) 在全频段内
```

## 2 电容的频率特性

### 2.1 理想电容与实际电容

理想电容的阻抗随频率线性下降:Z = 1/(2*pi*f*C)。但实际电容存在寄生参数:

```
实际电容的等效电路:

       ESL(等效串联电感)
    +---LLLL---+
    |          |
    |  C(标称电容)
    |          |
    |  ESR(等效串联电阻)
    |    RRRR  |
    +----------+
    |
   === GND
```

实际阻抗:Z = sqrt(ESR^2 + (2*pi*f*ESL - 1/(2*pi*f*C))^2)

### 2.2 自谐振频率

当容性阻抗与感性阻抗相等时,阻抗最小,这就是自谐振频率(SRF):

```
|Z|
 ^  \
 |   \        容性区(下降)    感性区(上升)
 |    \                     /
 |     \                   /
 |      \       SRP      /
 |       \_____|_______/
 |         min(Z)=ESR
 +----------------------------------------> f
              SRF
```

SRF计算: f_SR = 1 / (2*pi*sqrt(ESL*C))

| 封装 | 典型ESL | 100nF的SRF | 10nF的SRF | 1nF的SRF |
|------|---------|-----------|-----------|----------|
| 0402 | 0.5nH | 约22MHz | 约70MHz | 约220MHz |
| 0603 | 0.8nH | 约18MHz | 约56MHz | 约178MHz |
| 0805 | 1.2nH | 约15MHz | 约46MHz | 约145MHz |

> 关键结论:封装越小,ESL越低,SRF越高,高频去耦效果越好。去耦电容优先选0402封装。

## 3 电容类型与角色分工

### 3.1 三级去耦体系

| 级别 | 容值范围 | 典型器件 | 作用频段 | 放置位置 |
|------|----------|----------|----------|----------|
| 体电容(Bulk) | 10-100uF | 电解/钽电容 | <1MHz | 电源入口附近 |
| 中频去耦(Mid) | 1-10uF | X5R/X7R MLCC | 1-10MHz | IC周围1-2cm |
| 局部去耦(Local) | 100nF | X5R/X7R MLCC | 10-100MHz | 紧贴IC电源引脚 |
| 高频去耦(可选) | 1-10nF | X5R/X7R MLCC | >100MHz | 紧贴IC电源引脚 |

三级体系的工作方式:

```
频率覆盖示意:

  Bulk (10uF)  _____________
  Mid (1uF)          _____________
  Local (100nF)             _____________
  Hi-freq (1nF)                   _____________
  ------------------------------------------------> f
  10kHz  100kHz  1MHz  10MHz  100MHz  1GHz

  各级电容在不同频段主导,共同覆盖全频段
```

### 3.2 MLCC特性

多层陶瓷电容(MLCC)是去耦的主力器件。关键特性:

- **介电类型**:
  - X7R: -55~125degC, 容值变化低于15%,首选
  - X5R: -55~85degC, 容值变化低于15%,常用
  - Y5V: -30~85degC, 容值变化可达-82%,慎用

- **直流偏压降额**(DC Bias Derating):这是MLCC最容易被忽视的特性。施加直流电压后,有效容值会大幅下降:

| 施加电压/额定电压 | X5R 10uF 0402 剩余容值 |
|-------------------|------------------------|
| 0% | 100% |
| 25% | 约85% |
| 50% | 约65% |
| 75% | 约45% |
| 100% | 约30% |

> 实际选型法则:工作电压不超过MLCC额定电压的50%,以保证容值降额在可接受范围内。例如3.3V电源选用6.3V或10V额定电压的MLCC。

### 3.3 温度稳定性

X7R在-55~125degC范围内容值变化不超过15%;Y5V在-30~85degC范围内可能下降82%。IoT设备在户外场景中工作温度范围宽,应避免使用Y5V。

## 4 去耦电容放置规则

### 4.1 尽可能靠近IC电源引脚

这是去耦电容布局的第一法则。原因:走线电感会与电容串联,降低电容的高频去耦效果。

```
好: 电容紧贴IC                  差: 电容远离IC

  +-------+                      +-------+
  |       |                      |       |
  |  IC   |VDD                   |  IC   |VDD
  |       |--+---C---GND         |       |-----+------C---GND
  +-------+  |                   +-------+     |
             |                                 | 长走线
            GND                               GND = 高电感
```

### 4.2 最小化回路面积

电流从IC的VDD引脚流出,经过去耦电容,从GND引脚返回。这个回路的面积决定了它产生的电磁辐射和接收干扰的敏感度:

- 回路面积越大 -> 辐射发射越高 -> EMI问题越严重
- 回路面积越大 -> 更容易拾取其他信号的噪声

最小化回路面积的方法:

1. 电容放在IC的VDD引脚旁边
2. 电容的GND焊盘紧邻IC的GND焊盘
3. 两者通过短走线或直接铜皮连接

### 4.3 过孔放置与地回路

```
电容接地的过孔放置策略(俯视图):

  最佳:过孔紧贴焊盘        次佳:过孔在焊盘旁        差:过孔远离焊盘

  +---+  [过孔]            +---+                     +---+
  | C |                     | C |                     | C |
  +---+                    +---+                     +---+
    |                                                 |
   [过孔]                             [过孔]         走线
                                                       |
                                                     [过孔]

  电感: 最低                   中等                      最高
```

- 过孔直接放在焊盘上(Via-in-pad)电感最低,但工艺成本略高
- 过孔紧贴焊盘放1-2个是性价比最高的方案
- 绝对不要让过孔远离焊盘,用长走线连接

## 5 多电容并联策略

### 5.1 不同容值并联

用不同容值的电容并联,可以覆盖更宽的频率范围:

- 10uF覆盖低频(<5MHz)
- 100nF覆盖中频(5-50MHz)
- 1nF覆盖高频(>50MHz)

每个电容在其SRF附近提供最低阻抗,三个电容接力覆盖全频段。

### 5.2 反谐振问题

并联电容时,大电容的感性区与小电容的容性区可能形成并联谐振,在某个频率上产生阻抗峰值,这就是反谐振:

```
|Z|
 ^                  反谐振峰
 |        \         /\
 |         \       /  \
 |  10uF    \_____/    \_____  100nF
 |                  \          ___  1nF
 |                   \________/
 +--------------------------------------------> f
              f_anti-resonance
```

反谐振峰出现的位置:在两个电容的SRF之间。峰值高度取决于ESR。

### 5.3 减小反谐振的方法

1. **选容值相近的电容**:例如100nF + 47nF,而不是100nF + 1nF。容值比不超过10:1时反谐振峰较低
2. **选择ESR稍高的电容**:ESR可以抑制谐振峰,但会增加最低阻抗
3. **利用PCB平面电容**:电源/地平面对的分布电容可以平滑反谐振
4. **实测PDN阻抗**:用网络分析仪或VNA实测,确认反谐振在可接受范围

> 实用经验:对于大多数IoT应用,100nF + 10uF的搭配反谐振峰可控,不需要额外处理。只在高速数字(>200MHz时钟)场景才需要特别关注。

## 6 PCB叠层对去耦的影响

### 6.1 电源/地平面对的分布电容

相邻的电源平面和地平面形成一个平板电容:

```
C_plane = epsilon_0 * epsilon_r * A / d

epsilon_0 = 8.854 pF/m (真空介电常数)
epsilon_r = 约4.5 (FR-4)
A = 平面面积
d = 平面间距
```

典型值:10cm x 10cm板子,平面间距0.2mm,平面电容约200pF。虽然容值不大,但因为几乎没有ESL(平面电感极低),在1GHz以上频段非常有效。

### 6.2 叠层设计建议

```
4层板推荐叠层:

  Layer 1: 信号层(元器件面)
  Layer 2: GND平面        <-- 紧邻Layer 1,提供最短地回路
  Layer 3: 电源平面        <-- 与Layer 2形成分布电容
  Layer 4: 信号层(底层)

  Layer2-Layer3间距建议: 0.1-0.2mm(薄芯板),增大平面电容
```

6层板可以将电源/地平面对进一步优化:

```
6层板推荐叠层:

  Layer 1: 信号层
  Layer 2: GND平面
  Layer 3: 信号层(内层走线)
  Layer 4: 电源平面
  Layer 5: GND平面
  Layer 6: 信号层

  Layer4-Layer5间距0.1mm,形成高质量分布电容
```

## 7 实际设计案例

### 7.1 STM32去耦方案

以STM32F103为例(48引脚LQFP封装):

| 电源引脚 | 去耦电容 | 放置要求 |
|----------|----------|----------|
| 每个VDD | 100nF X5R 0402 | 紧贴引脚,回路面积最小 |
| VDDA(模拟电源) | 1uF X5R 0402 + 100nF | 放在VDDA引脚旁,远离数字噪声 |
| VDD总入口 | 4.7uF X5R 0603 | 靠近MCU,但不必紧贴引脚 |

关键细节:

- VDDA的1uF电容必须放在VDDA引脚旁,不要放在板边
- VDDA的去耦电容GND应连接到模拟地(AGND),不要直接连到数字地平面
- 每个VDD引脚的100nF电容应紧贴该引脚,不要几个VDD共用一个电容

### 7.2 RF模块去耦方案

以nRF52832蓝牙SoC为例:

| 电源域 | 去耦电容 | 注意事项 |
|--------|----------|----------|
| VDD(数字) | 100nF x4 + 4.7uF x1 | 分散放置在4个VDD引脚旁 |
| DEC1(内部1.3V) | 100nF NPO 0402 | 必须NPO(C0G)类型,不可用X5R |
| DEC2(内部1.3V) | 47nF NPO 0402 | 同上 |
| VDDH(高压域) | 1uF X5R 0402 | 靠近VDDH引脚 |

> RF芯片对去耦电容的介质类型有严格要求。内部LDO的输出电容如果使用X5R(直流偏压下容值下降),会导致LDO不稳定。NPO(C0G)电容不存在直流偏压降额问题。

### 8.1 常用工具

| 工具 | 类型 | 特点 |
|------|------|------|
| LTspice | SPICE仿真 | 免费,适合单节点PDN分析 |
| Keysight ADS PIPro | 专业PDN分析 | 支持全板PDN仿真,含平面谐振 |

| Cadence Sigrity PowerDC/PowerSI | 专业PDN分析 | 与Allegro无缝集成 |
| KiCad + ngspice | 开源方案 | 适合简单PDN分析 |

### 8.2 仿真流程

1. 建立PDN模型:电源源 -> 走线/平面 -> 过孔 -> 电容 -> IC引脚
2. 设置电容模型:含ESR, ESL, C标称值
3. 扫频分析:10kHz - 1GHz
4. 检查|Z|是否在全频段低于目标阻抗
5. 如有超标,调整电容容值/数量/位置

### 8.3 目标阻抗计算

```
Ztarget = (容许电压纹波) / (最大瞬态电流)

例如: 3.3V电源,容许纹波5%,最大瞬态电流2A
Ztarget = (3.3V * 5%) / 2A = 82.5mOhm
```

## 9 常见错误

### 9.1 电容离IC太远

```
错误:电容放在板边,距离IC 3cm

  +-------+                        C
  |       |                       ---  
  |  IC   |VDD-----长走线---------| |---GND
  |       |                       ---
  +-------+

  长走线(约3cm)增加约2nH电感,100nF电容的有效频率上限从22MHz降到约11MHz
```

> 错误认知:电容放在板边也行,反正都在同一电源网上。
> 正确理解:去耦电容必须紧贴IC电源引脚,距离每增加1cm,有效频率上限约降低30%。

### 9.2 长走线到过孔

电容焊盘到地过孔之间有长走线,增加了电感:

```
差:  [电容焊盘]---走线5mm---[过孔]---GND平面
好:  [电容焊盘] [过孔]---GND平面 (过孔紧贴焊盘)
```

5mm走线增加约3nH电感,在高频下使去耦效果大打折扣。

### 9.3 用错电容类型

- 在RF芯片LDO输出用X5R代替NPO:直流偏压下容值下降,LDO可能振荡
- 在高温场景用Y5V:容值下降80%以上,去耦几乎失效
- 用0805封装代替0402:ESL增加60%以上,高频去耦效果变差

### 9.4 多个VDD引脚共用一个电容

```
错误: 4个VDD引脚共用1个100nF电容

  VDD1 ---+            正确:每个VDD引脚各有1个100nF电容
  VDD2 ---+
  VDD3 ---+---100nF---GND     VDD1---100nF---GND
  VDD4 ---+                    VDD2---100nF---GND
                                 VDD3---100nF---GND
                                 VDD4---100nF---GND
```

共用电容时,不同引脚之间的开关噪声会相互耦合,且每个引脚到电容的走线电感不同。

## 总结

去耦电容设计看似简单,实则是电源完整性的核心。要点回顾:

1. **原理**:IC开关产生电流瞬变 -> PDN阻抗导致电压波动 -> 去耦电容提供本地低阻抗通路
2. **频率特性**:实际电容有ESR和ESL -> SRF以上表现为感性 -> 多容值并联覆盖宽频
3. **选型**:X7R/X5R为主力,避免Y5V;注意直流偏压降额,额定电压留50%余量
4. **放置**:紧贴IC电源引脚,最小化回路面积,过孔紧贴焊盘
5. **陷阱**:反谐振、电容太远、长走线到过孔、选错介质类型

去耦电容的设计是"细节决定成败"的典型场景。一个100nF电容放在正确位置与放在错误位置,在100MHz频率上的阻抗差异可以达10倍以上。
