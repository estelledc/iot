# 光耦与数字隔离器在IoT隔离通信中的对比

> **难度**：🟡 中级 | **领域**：信号隔离技术 | **阅读时间**：约 20 分钟

## 引言

想象你在医院里做核酸检测:护士隔着一面玻璃窗递给你棉签,你采完样再从窗口递回来。你们之间没有直接接触,但信息(样本)成功传递了。电气隔离也是这个道理:两侧电路之间没有电流通路(没有直接的"接触"),但信号可以安全传递。这面"玻璃窗"在电子系统里就是隔离器件。

在IoT系统中,隔离器件承担三重使命:安全(高压侧不伤到低压逻辑侧)、抗干扰(切断地环路和共模噪声)、兼容(两侧可以有不同的地电位)。传统方案用光耦(光电耦合器),新兴方案用数字隔离器(电容或磁耦合)。两者各有优劣,本文从原理、性能、应用场景到安全标准,帮你做出正确的选择。

## 1 为什么需要电气隔离

### 1.1 安全需求

工业IoT中,强电(220V/380V)与弱电(3.3V逻辑)共存。如果电机驱动侧的高压因故障窜到MCU侧,轻则烧芯片,重则伤人。隔离器件在两侧之间提供绝缘屏障,即使一侧故障,另一侧也安全。

### 1.2 噪声抗扰度

长距离通信中,两端设备的接地点电位可能不同(地电位差可达几十伏)。不隔离时,地电位差形成地环路,共模噪声叠加在信号上。隔离切断地环路,消除共模干扰。

### 1.3 不同地电位兼容

分布式IoT系统中,各节点独立供电,地电位不共。隔离允许两侧以不同地为参考,信号正常传递,无需强制共地。

### 1.4 隔离的核心指标

| 指标 | 含义 | 典型值 |
|------|------|--------|
| 隔离电压 | 屏障能承受的最大电压 | 2.5-7kV |
| 工作电压 | 长期安全承受的电压 | 100-1000V |
| 爬电距离 | 沿绝缘表面的最短距离 | 3-8mm |
| 电气间隙 | 通过空气的最短距离 | 3-8mm |

## 2 光耦原理与特性

### 2.1 工作原理

光耦由发光二极管(LED)和光敏三极管(Phototransistor)组成,两者之间隔着透明绝缘介质:

```
              隔离屏障
                |
LED侧:    LED --> | --> 光敏三极管 : 输出侧
                |
(输入电流)  光信号    (输出电流)
```

输入电流驱动LED发光,光穿过绝缘层照射光敏三极管,三极管导通产生输出电流。信号以光为媒介传递,两侧无电气连接。

### 2.2 电流传输比(CTR)

CTR(Current Transfer Ratio)是光耦最关键的参数:

```
CTR = Ic(输出电流) / If(输入电流) x 100%
```

典型CTR值:
- 普通光耦(PC817): 50-300%
- 高速光耦(6N137): 典型值用CTR不适用(逻辑输出)
- 线性光耦: CTR受温度影响大,需补偿

CTR随温度升高而下降(约-0.5%/C),随使用时间增加而衰减(LED光输出逐渐减弱)。

### 2.3 光耦的局限性

| 问题 | 原因 | 影响 |
|------|------|------|
| CTR退化 | LED光输出随时间衰减 | 寿命末期可能无法正常驱动 |
| 速度慢 | 光敏三极管存储效应 | 普通光耦带宽<100kHz |
| 功耗高 | LED需要mA级驱动电流 | 不适合低功耗系统 |
| 体积大 | LED+三极管结构 | 封装大,通道密度低 |
| CTR温漂 | LED效率和三极管增益随温度变化 | 输出电流不稳定 |

## 3 数字隔离器原理与特性

### 3.1 电容耦合型

用片上二氧化硅(SiO2)电容器作为隔离屏障,信号通过高频调制跨越电容传递:

```
输入侧 --> [调制器] --> [SiO2电容] --> [解调器] --> 输出侧
                        (隔离屏障)
```

SiO2是极好的绝缘材料(击穿场强约500V/um),薄膜电容提供kV级隔离,同时寄生电容极小(约1pF),共模瞬态抗扰度(CMTI)好。

代表产品: Si864x(Silabs)、ISO774x(TI)。

### 3.2 磁耦合型

用片上微型变压器,通过磁场传递信号:

```
输入侧 --> [初级线圈] --磁通--> [次级线圈] --> 输出侧
                    (隔离屏障)
```

变压器通过交变磁场传递能量和信号,天然隔离。采用高频载波(数百MHz)实现高速传输。

代表产品: ADuM12xx/24xx(Analog Devices)。

### 3.3 数字隔离器的优势

| 优势 | 原因 |
|------|------|
| 速度快 | 无光-电-光转换延迟,最高150Mbps |
| 功耗低 | 无LED驱动电流,每通道<2mA |
| 寿命长 | 无LED光衰问题 |
| 体积小 | 单芯片集成多通道,SOIC-16封装4通道 |
| CMTI高 | 寄生电容小,共模抑制>25kV/us |

## 4 光耦与数字隔离器全面对比

### 4.1 性能对比表

| 参数 | 光耦(PC817) | 高速光耦(6N137) | 数字隔离器(Si8641) |
|------|------------|----------------|-------------------|
| 最高速率 | 100 kHz | 10 Mbps | 150 Mbps |
| 传播延迟 | 3-5 us | 75 ns | 10 ns |
| 功耗/通道 | 10-50 mW | 30-60 mW | 1-5 mW |
| 隔离电压 | 5 kV | 5 kV | 5 kV |
| CMTI | 5-10 kV/us | 10-25 kV/us | 25-50 kV/us |
| 使用寿命 | 5-10年(CTR退化) | 5-10年 | 20年+(无光衰) |
| 通道数/封装 | 1-2 | 1 | 4 |
| 工作温度 | -40~+100C | -40~+85C | -40~+125C |
| 成本(单通道) | ~$0.1 | ~$1.0 | ~$0.5 |

### 4.2 关键差异解读

**速度**: 数字隔离器快1-3个数量级。光耦的LED响应和光敏三极管存储效应是速度瓶颈。

**功耗**: 数字隔离器低10-50倍。光耦的LED需要持续mA级驱动,数字隔离器输入端是CMOS电平。

**寿命**: 数字隔离器无光衰问题,寿命远超光耦。CTR退化是光耦的阿喀琉斯之踵,高温环境下尤其严重。

**成本**: 单通道光耦(PC817)最便宜;高速光耦与数字隔离器价格相当;多通道场景数字隔离器性价比更高。

## 5 主流器件介绍

### 5.1 常用光耦

**PC817(Sharp)**: 最经典的通用光耦,CTR 50-300%,带宽约80kHz,SOP-4封装,价格不到$0.1。适用: 低速数字信号隔离、开关量传输、电源反馈。

**6N137(Vishay)**: 高速逻辑光耦,内置施密特触发器,10Mbps,传播延迟75ns。适用: SPI隔离、高速数字通信。

**HCPL-0611(Broadcom)**: 逻辑输出高速光耦,10Mbps,3.3V/5V兼容,推挽输出。适用: 电机驱动隔离、工业总线。

### 5.2 常用数字隔离器

**Si8641(Silabs)**: 4通道电容耦合数字隔离器,150Mbps,2.5-5.5V供电,5kV隔离,CMTI 35kV/us。适用: SPI/I2C/UART多线隔离,工业IoT通信。

**ISO7741(TI)**: 4通道电容耦合数字隔离器,100Mbps,2.76-5.5V供电,5kV隔离,CMTI 50kV/us。适用: RS485隔离、多通道数字隔离。

**ADuM1201(ADI)**: 2通道磁耦合数字隔离器,25Mbps,2.7-5.5V供电,2.5kV隔离,CMTI 25kV/us。适用: 精密ADC隔离SPI、医疗设备。

### 5.3 选型对比

| 器件 | 类型 | 通道 | 速率 | 隔离电压 | 价格 |
|------|------|------|------|---------|------|
| PC817 | 光耦 | 1 | 80kHz | 5kV | ~$0.1 |
| 6N137 | 高速光耦 | 1 | 10Mbps | 5kV | ~$1.0 |
| HCPL-0611 | 逻辑光耦 | 1 | 10Mbps | 5kV | ~$2.0 |
| Si8641 | 电容数字 | 4 | 150Mbps | 5kV | ~$2.5 |
| ISO7741 | 电容数字 | 4 | 100Mbps | 5kV | ~$2.0 |
| ADuM1201 | 磁数字 | 2 | 25Mbps | 2.5kV | ~$2.5 |

多通道场景下,数字隔离器单通道成本低于高速光耦。

## 6 隔离电压等级与安全标准

### 6.1 基本隔离与增强隔离

| 等级 | 含义 | 测试电压 | 适用 |
|------|------|---------|------|
| 基本隔离(Basic) | 单重绝缘保护 | 2.5-3kV | 辅助保护(另有接地等手段) |
| 增强隔离(Reinforced) | 双重绝缘等效保护 | 5-7kV | 直接安全保护(无需额外手段) |

人可接触的场合必须用增强隔离。

### 6.2 工作电压与测试电压

- **工作电压(VIOWM)**: 隔离屏障长期安全承受的最高电压,如400Vrms或1000Vpeak
- **测试电压(VISO)**: 型式试验时的1分钟耐压值,如5kVrms

测试电压远高于工作电压,是安全裕度的体现。选型时工作电压必须大于系统最大差分电压。

### 6.3 爬电与电气间隙

| 参数 | 定义 | 影响因素 |
|------|------|---------|
| 爬电距离(Creepage) | 沿绝缘表面的最短路径 | 污染等级、材料CTI |
| 电气间隙(Clearance) | 通过空气的最短路径 | 海拔、污染等级 |

PCB设计时,隔离器件下方不能铺铜跨越隔离带,必须留出足够的爬电和间隙距离。

### 6.4 安全标准

| 标准 | 覆盖范围 | 要点 |
|------|---------|------|
| UL 1577 | 光耦安全认证 | 美国市场必备 |
| IEC 60747-5-5 | 光耦安全标准 | 国际通用 |
| VDE 0884-11 | 数字隔离器安全标准 | 欧洲市场,数字隔离器认证依据 |

出口设备必须通过对应市场的安全认证。

## 7 应用实例

### 7.1 RS485隔离收发器(工业IoT)

工业RS485总线可能存在地电位差和共模干扰,必须隔离:

```
MCU UART TX ---[Si8641 CH1]--- RS485 DI
MCU UART RX ---[Si8641 CH2]--- RS485 RO
MCU DE/RE   ---[Si8641 CH3]--- RS485 DE/RE

Si8641隔离侧由DC-DC或隔离电源供电。
```

4通道Si8641隔离3条信号线,剩余1通道可做使能控制。5kV增强隔离满足工业安全要求。

### 7.2 隔离SPI(高压传感器)

高压侧ADC的SPI信号需要隔离才能传到低压MCU:

```
高压侧ADC ---[ISO7741]--- MCU SPI
  SCLK  ---[CH1]---> SCLK
  MISO  ---[CH2]<--- MISO
  MOSI  ---[CH3]---> MOSI
  CS    ---[CH4]---> CS
```

ISO7741的4通道正好覆盖SPI的4条信号线。100Mbps速率满足大多数ADC的SPI时钟需求。

### 7.3 隔离栅极驱动(电机控制)

电机驱动的IGBT/SiC栅极需要隔离驱动,既是安全需求也是功能需求:

```
MCU PWM ---[ADuM3223]--- IGBT栅极
             (隔离栅极驱动器)
```

ADuM3223集成了2通道隔离+栅极驱动,4A输出电流,5kV隔离。比"光耦+驱动IC"方案更简单可靠。

## 8 选型指南

### 8.1 按数据速率选择

| 速率范围 | 推荐 |
|---------|------|
| < 100kHz | 光耦(PC817) |
| 100kHz-1Mbps | 高速光耦或数字隔离器 |
| 1-10Mbps | 数字隔离器 |
| 10-150Mbps | 高速数字隔离器(Si8641/ISO7741) |

### 8.2 按通道数选择

| 通道需求 | 推荐 |
|---------|------|
| 1通道,低速 | 光耦(PC817),最便宜 |
| 1通道,高速 | 6N137或单通道数字隔离器 |
| 2-4通道 | 多通道数字隔离器(Si8641/ISO7741) |
| > 4通道 | 多片数字隔离器级联 |

### 8.3 按隔离等级选择

| 场景 | 隔离等级 | 推荐 |
|------|---------|------|
| 信号隔离(无安全需求) | 基本隔离(2.5-3kV) | 通用数字隔离器 |
| 人可接触(安全需求) | 增强隔离(5kV) | 增强型数字隔离器 |
| 医疗设备 | 增强隔离(5kV+医疗认证) | 医疗级数字隔离器 |

### 8.4 按寿命需求选择

| 预期寿命 | 推荐 |
|---------|------|
| < 3年(消费电子) | 光耦可满足 |
| 3-10年(工业设备) | 数字隔离器优先 |
| > 10年(基础设施) | 必须数字隔离器 |

### 8.5 功耗预算考虑

| 功耗预算 | 推荐 |
|---------|------|
| 不敏感(市电供电) | 光耦或数字隔离器均可 |
| 中等(电池+太阳能) | 数字隔离器 |
| 极低(纽扣电池) | 纳功耗数字隔离器或间歇供电 |

## 9 光耦CTR退化与寿命评估

### 9.1 CTR退化机制

光耦中LED的光输出随工作时间和温度逐渐衰减,导致CTR下降:

- LED光输出每年衰减约1-3%(25C)
- 温度每升高25C,退化速率约加倍
- 驱动电流越大,退化越快

10年后CTR可能降至初始值的50-70%。如果设计时CTR余量不足,系统可能在寿命末期失效。

### 9.2 设计余量

为确保寿命末期仍能正常工作,设计时必须预留足够的CTR余量:

```
If_min(寿命末期) = Ic_required / CTR_min(寿命末期)
CTR_min(寿命末期) = CTR_initial x 退化系数
退化系数(10年, 50C) 约 0.5-0.7
```

例如:输出需要1mA,初始CTR=100%,10年后降至50%:
If_min = 1mA / 50% = 2mA(寿命末期驱动电流)
If_design = 2mA x 1.5(安全系数) = 3mA

### 9.3 高温加速退化

工业环境温度可达85C,CTR退化加速。经验公式:

```
退化率(T) = 退化率(25C) x 2^((T-25)/25)
```

85C时退化率是25C的约16倍。高温环境下光耦寿命可能只有2-3年,这是工业系统转向数字隔离器的重要原因。

### 9.4 CTR退化监测

关键系统可在软件层面监测CTR:

- 定期测量已知输入电流下的输出响应
- 与初始值比较,推算CTR退化程度
- CTR低于阈值时发出维护告警

这种方法增加了软件复杂度,但可预防寿命末期的突发失效。

## 10 隔离电源方案

### 10.1 隔离侧供电的必要性

隔离器件的两侧需要独立供电。如果数字隔离器的输出侧没有独立电源,信号传过来也没能量驱动负载。

### 10.2 常见方案

| 方案 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 隔离DC-DC | 小功率变压器隔离 | 稳定输出 | 成本高、体积大 |
| 隔离IC内置电源 | 数字隔离器集成iso-power | 简单 | 输出功率小(约50mW) |
| 自举二极管 | 高侧驱动自举 | 低成本 | 仅适合栅极驱动,不可连续 |

Si8641+Si80xx(内置隔离电源)是IoT常用组合,单芯片解决信号+电源隔离。

### 10.3 功率预算

隔离侧功耗 = 数字隔离器输出侧静态电流 + 负载电流。例如ISO7741输出侧静态电流2mA/通道,4通道共8mA,5V供电约40mW。需确保隔离电源能提供足够功率。

## 11 PCB设计注意事项

### 11.1 隔离带(Barrier)

隔离器件下方的PCB区域为隔离带,规则:
- 两侧铜皮不能跨越隔离带
- 隔离带宽度 >= 器件要求的爬电距离(通常3-8mm)
- 隔离带内不开过孔
- 隔离带区域可开槽增加爬电距离

### 11.2 布局对称性

隔离器件两侧的布局尽量对称:
- 两侧去耦电容位置对称
- 两侧地平面面积相近
- 高速信号走线长度匹配

### 11.3 共模瞬态抗扰度(CMTI)

高速开关(如SiC MOSFET)产生的共模瞬态可达50kV/us,可能穿透隔离屏障:
- 选择CMTI > 25kV/us的数字隔离器
- 隔离器件远离开关管
- 两侧地平面各自完整,不交叉

### 11.4 EMI考虑

数字隔离器的高频调制(数百MHz)可能产生EMI:
- 隔离器件紧邻连接器放置
- 输入输出走线短而直
- 两侧各放0.1uF去耦电容

## 总结

光耦和数字隔离器各有定位,核心选择依据:

1. **低速低成本用光耦**: PC817不到$0.1,隔离开关量和低速信号足够。别为省钱把光耦用在高速场景——CTR退化会让你后悔。
2. **高速多通道用数字隔离器**: Si8641/ISO7741四通道150Mbps,单通道成本远低于高速光耦,且无寿命之忧。
3. **寿命是关键分水岭**: 光耦CTR随时间和温度退化,5年以上应用必须选数字隔离器。
4. **安全标准必须遵守**: 出口设备需UL/IEC/VDE认证,选型时确认器件已通过对应标准。
5. **隔离侧需要独立电源**: 信号隔离+电源隔离缺一不可,内置iso-power的数字隔离器可简化设计。
6. **PCB隔离带不可马虎**: 爬电距离和电气间隙是硬性安全指标,隔离带内不铺铜、不开过孔。

回到开头的类比:隔离器件是两侧电路之间的"玻璃窗",既要让信号透过(低延迟、低损耗),又要挡住危险(耐高压、抗干扰)。光耦是早期的"单层玻璃",数字隔离器是现代的"钢化夹层玻璃"——更薄、更透、更安全。
