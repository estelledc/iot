# PCB板载天线设计：倒F/蛇形/环形天线
> **难度**：🔴 高级 | **领域**：天线设计 | **阅读时间**：约 22 分钟

## 引言

想象你在搭建一座广播站。如果买一根专业天线，效果好但要花钱;如果直接用站房屋顶的钢梁，稍加弯折也能辐射信号——这就是PCB板载天线的思路。天线不一定是单独的金属棒，PCB铜皮本身就是导体，只要几何形状设计得当，就能成为有效的辐射器。对IoT产品而言，省掉一颗BOM器件、少一次贴片工序，意义远不止省几毛钱——更少的器件意味着更简单的供应链、更高的可靠性。

本文将系统介绍三种最常见的PCB板载天线——倒F天线(IFA/PIFA)、蛇形天线和环形天线，从原理到设计到实测调校，覆盖2.4GHz BLE和Sub-GHz两大IoT主力频段。

## 1 PCB板载天线的优势

### 1.1 零BOM成本

PCB天线是铜皮走线的一部分，不增加任何元器件成本。对于年出货百万级的IoT产品，省一颗天线器件加上贴片费，总节省可达数十万元。

### 1.2 高度集成

天线与电路在同一块PCB上，无需额外连接器或馈线，消除了同轴电缆的插入损耗(典型0.3-0.5 dB/cm at 2.4GHz)。

### 1.3 缺点与代价

- 性能上限低于外置天线：典型增益-2 ~ 0 dBi，而chip天线可达1-3 dBi
- 对PCB布局高度敏感：地平面形状、附近金属都会显著改变谐振频率
- 调试周期长：一次改版需重新制板验证

## 2 倒F天线(IFA)与平面倒F天线(PIFA)

### 2.1 从1/4波长单极子到倒F

1/4波长单极子是最基本的天线形式。在PCB上实现1/4波长单极子的问题是：(a)需要较大的净空区;(b)馈点阻抗约36.5 ohm，与50 ohm系统不匹配。

倒F天线的思路：把单极子顶端向下折叠成与地平面平行的"帽"，同时在馈点附近引入一条短路支节。折叠降低高度，短路支节引入并联电感来抵消天线的容性电抗，从而实现阻抗匹配。

### 2.2 IFA几何参数

IFA有三个关键尺寸：

| 参数 | 含义 | 对谐振频率的影响 |
|------|------|------------------|
| L (辐射臂长) | 从馈点到开路端的总长 | L增大 -> 频率降低 |
| h (辐射臂高度) | 辐射臂到地平面的距离 | h增大 -> 带宽增加 |
| S (短路支节宽度) | 短路条宽度 | S增大 -> 阻抗降低 |

经验公式(2.4GHz FR4基板)：

```
L_total = lambda_g / 4
lambda_g = c / (f * sqrt(Ee))
其中 Ee 是有效介电常数, FR4约4.4, Ee约3.0-3.5
```

### 2.3 PIFA：IFA的平面化

PIFA将辐射臂展宽为平面，等效增大了h和电容，带来两个好处：

1. 带宽更宽：PIFA可达到5-10%相对带宽，而IFA通常只有2-3%
2. 更容易实现双频：在辐射面上开槽可以激励第二个谐振模

PIFA是手机和BLE模块中最常用的PCB天线形式。

### 2.4 IFA/PIFA设计步骤

1. 确定目标频率和PCB可用面积
2. 用经验公式估算初始L、h、S
3. 在仿真工具(HFSS/CST)中建模，扫描S11
4. 微调L使谐振频率对准目标
5. 微调S使阻抗接近50 ohm
6. 确认辐射方向图和增益满足要求

## 3 蛇形天线(Meandered Antenna)

### 3.1 为什么需要蛇形走线

1/4波长在2.4GHz自由空间约31mm。Sub-GHz(如868MHz)则达到86mm——对小型IoT设备而言太长。蛇形走线通过"折叠"直线段为来回折返的折线，在保持总电长度不变的前提下大幅缩短物理长度。

### 3.2 蛇形天线设计要点

- 折弯次数越多，尺寸越紧凑，但辐射效率越低(相邻段电流方向相反，辐射部分抵消)
- 折弯间距至少3倍线宽，过近会导致强耦合、Q值飙升、带宽极窄
- 起始段(馈点附近)电流最大，应尽量保持直线以减少效率损失

### 3.3 蛇形天线效率估算

```
eta_meander = eta_straight * (1 - k * N_bends)
其中:
  eta_straight = 同长度直线单极子效率
  k = 0.03 ~ 0.08, 取决于折弯间距与线宽比
  N_bends = 折弯次数
```

一个4折弯868MHz蛇形天线，典型效率约30-45%，而同等长度的直线单极子约60-70%。

## 4 环形天线(Loop Antenna)

### 4.1 小环形天线

当环的周长远小于波长(小于lambda/10)时，天线等效为磁偶极子，辐射特性与电偶极子互补：

- 输入阻抗很低(几ohm)，需要匹配网络
- 辐射方向图在环面法线方向呈最大值
- 对附近介质和人体不太敏感(磁近场衰减快)

### 4.2 大环形天线

周长接近一个波长时，环形天线在环周上形成驻波电流分布，输入阻抗可达约120 ohm，带宽和效率显著优于小环。

### 4.3 IoT中的环形天线应用

- NFC/RFID：13.56MHz环形天线是标准选择
- 小型BLE标签：方环/圆环天线配合匹配网络
- 穿戴设备：环形天线可沿外壳轮廓走线

## 5 地平面要求与影响

### 5.1 地平面是天线的一部分

单极子/IFA/PIFA都需要地平面作为镜像面。地平面尺寸决定了天线的等效电气性能。经验法则：地平面至少lambda/4 x lambda/4，越小则谐振频率偏移越大、效率越低。

### 5.2 地平面对IFA的影响

| 地平面尺寸 | 谐振频率偏移 | 效率下降 | 增益 |
|-----------|-------------|---------|------|
| 1 x lambda/2 | <2% | <1 dB | ~0 dBi |
| 0.5 x lambda/4 | 5-10% | 2-4 dB | -2~-4 dBi |
| < 0.3 x lambda/4 | >15% | >5 dB | <-5 dBi |

### 5.3 地平面中断与狭缝

PCB上切割地平面的走线(如电源线、信号线)会在地平面上形成狭缝，破坏回流路径，导致：

- 天线谐振频率偏移
- 辐射方向图畸变
- EMI风险增加

设计规则：天线正下方的地平面保持完整，不允许跨区域走线。

## 6 天线仿真基础

### 6.1 HFSS与CST对比

| 特性 | HFSS (Ansys) | CST Studio |
|------|-------------|-----------|
| 核心算法 | 有限元法(FEM) | 有限积分法(FIT) |
| 适合场景 | 复杂结构、高精度 | 宽频扫描、时域分析 |
| 学习曲线 | 较陡 | 相对平缓 |
| 授权费用 | 高 | 高 |

两者都有免费的学生版(功能受限)。

### 6.2 仿真建模流程

1. 建立PCB几何(铜皮、介质层、地平面)
2. 设置馈电端口(lumped port或wave port)
3. 添加辐射边界(距天线至少lambda/4)
4. 设置求解频率和扫频范围
5. 运行求解，查看S11、远场方向图
6. 参数化扫描关键尺寸，优化谐振频率

### 6.3 仿真与实测的差距

仿真基于理想化模型，实测还受以下因素影响：

- PCB介电常数偏差(FR4的Er在4.2-4.8间波动)
- 铜皮厚度和表面粗糙度
- 焊盘、过孔的寄生效应
- 外壳和人体的影响

典型仿真-实测频率偏差3-8%，需要实物调校。

## 7 回波损耗(S11)与VSWR目标

### 7.1 指标定义

- **回波损耗(Return Loss, S11)**：反射功率与入射功率之比，单位dB。S11 = -10 dB意味着10%功率被反射。
- **电压驻波比(VSWR)**：传输线上电压最大值与最小值之比。VSWR = (1+|Gamma|)/(1-|Gamma|)

| S11 | VSWR | 反射功率比 | 评价 |
|-----|------|-----------|------|
| -3 dB | 5.83 | 50% | 不可用 |
| -6 dB | 3.00 | 25% | 边缘可用 |
| -10 dB | 1.92 | 10% | 基本可用 |
| -15 dB | 1.43 | 3.2% | 良好 |
| -20 dB | 1.22 | 1% | 优秀 |

### 7.2 IoT产品的目标

- BLE/Zigbee：S11 < -10 dB在2.4-2.4835GHz全频段
- Sub-GHz(868/915MHz)：S11 < -10 dB在目标频点+/-5MHz
- 认证要求：FCC/CE对天线效率有间接要求，S11 < -10 dB是惯例基线

## 8 辐射方向图、增益与效率

### 8.1 辐射方向图

PCB板载天线的辐射方向图受地平面限制，通常呈半球形(地平面上方辐射强，下方被遮挡)。PIFA的方向图相对全向，蛇形天线方向图随折弯方向有差异。

### 8.2 增益与效率关系

```
Gain = Efficiency * Directivity
Efficiency = Radiation_Power / Input_Power
```

PCB天线典型值：

- IFA/PIFA在2.4GHz：-1 ~ +2 dBi增益，50-80%效率
- 蛇形天线在Sub-GHz：-3 ~ 0 dBi增益，20-50%效率
- 小环形天线：-5 ~ -2 dBi增益，10-30%效率

## 9 频率调谐方法

### 9.1 走线长度调谐

最直接的方法：改变天线走线长度。增长降低频率，缩短升高频率。每次改版调整0.5-2mm，观察S11曲线移动。

### 9.2 匹配网络调谐

在天线馈点与射频IC输出之间加PI/L型匹配网络：

- 串联电容 -> 频率升高(减小等效电感)
- 并联电感 -> 频率降低(增加等效电容)
- 匹配网络同时调谐频率和阻抗

### 9.3 短路支节调谐(仅IFA/PIFA)

改变短路支节的位置和宽度，可以在小范围内调谐谐振频率(典型范围+/-5%)。

## 10 多频天线设计

### 10.1 双频PIFA

在PIFA辐射面上开一条缝隙，将辐射面分为两个子区域，分别谐振于两个频率。缝隙长度和位置决定两个谐振频率的间距。

### 10.2 双频IFA

在IFA辐射臂上并联一条支路(分支)，支路长度对应第二谐振频率。这种方法结构简单但调试困难，两个频率间存在耦合。

### 10.3 BLE + Sub-GHz双频方案

由于频比过大(2.4GHz/868MHz约2.76)，单一天线难以覆盖。常见方案：

- 两个独立天线，用射频开关切换
- 一个宽频PIFA(牺牲效率换取带宽)

## 11 Keep-out区域规则

### 11.1 定义

Keep-out区域是PCB上天线周围不允许铺铜、走线、放置器件的区域。目的是保证天线辐射不受干扰。

### 11.2 规则

| 规则 | 说明 |
|------|------|
| 天线正下方不铺地铜 | 避免电容加载使频率偏移 |
| 净空区半径至少lambda/20 | 2.4GHz约6mm, 868MHz约17mm |
| 附近无金属元件 | 电池、连接器等距天线至少lambda/10 |
| 天线区域不走过孔 | 过孔会扰动电流分布 |

## 12 设计实例：2.4GHz BLE IFA

### 12.1 规格目标

- 频率：2.4-2.4835GHz
- S11：全频段 < -10 dB
- 增益：> -2 dBi
- PCB：4层FR4, 1.6mm厚

### 12.2 初始尺寸

```
辐射臂长L = 24mm (lambda_g/4 at 2.44GHz, Ee=3.2)
辐射臂高h = 4mm
短路支节宽S = 1.5mm
馈点距短路支节d = 4mm
```

### 12.3 仿真优化

仿真发现谐振频率偏低至2.36GHz。将L缩短至22mm，S增大至2mm，S11在2.44GHz达到-18dB，2.4-2.4835GHz全频段< -10dB。

### 12.4 实测结果

制板后用VNA实测，谐振频率2.41GHz(仿真2.44GHz，偏差1.2%)。在馈点处增加0.3pF串联电容将频率拉回2.44GHz。最终全频段S11 < -12 dB，增益-0.5 dBi。

## 13 设计实例：868MHz Sub-GHz蛇形天线

### 13.1 规格目标

- 频率：868MHz +/- 5MHz
- S11：< -10 dB
- 可用长度：仅35mm(lambda/4约86mm)

### 13.2 蛇形走线设计

```
总电长度：86mm
折弯次数：4
折弯间距：3mm
线宽：1mm
物理长度：约32mm
```

### 13.3 匹配设计

蛇形天线馈点阻抗约15-j30 ohm，与50 ohm差距大。采用L型匹配网络(2.7nH串联+3.3pF并联)，S11在868MHz达到-16dB。

### 13.4 性能

实测效率约35%，增益-3 dBi。对于受限尺寸，性能可接受。

## 14 制造公差的影响

### 14.1 主要公差来源

- 铜皮蚀刻公差：+/-0.05mm(外层), +/-0.1mm(内层)
- 介质厚度公差：+/-10%(FR4)
- 介电常数公差：+/-5%(FR4)

### 14.2 对谐振频率的影响

以2.4GHz IFA为例：

| 公差来源 | 典型值 | 频率偏移 |
|---------|-------|---------|
| 蚀刻公差(0.05mm) | L变化0.3% | +/-7MHz |
| 介质厚度(10%) | Ce变化8% | +/-50MHz |
| 介电常数(5%) | Ee变化5% | +/-60MHz |

最坏情况综合偏移可达+/-120MHz，但各公差并非完全相关，统计后RMS约+/-80MHz。这解释了为什么仿真与实测总有一定差距，匹配网络需要预留调谐裕量。

### 14.3 应对策略

1. 设计阶段预留匹配网络焊盘(0402/0603)
2. 首板测试后根据实测数据焊接匹配元件
3. 量产时通过统计控制减少偏差
4. 关键应用考虑高精度基板(Rogers, Er公差+/-1%)

## 总结

PCB板载天线是IoT产品中最经济的天线方案，三种主流形式各有适用场景：

- **IFA/PIFA**：2.4GHz BLE首选，性能与尺寸平衡最佳
- **蛇形天线**：Sub-GHz或空间受限场景，牺牲效率换尺寸
- **环形天线**：NFC/穿戴设备，磁偶极子特性对环境不敏感

设计要点：地平面完整、keep-out区域充分、预留匹配网络、首板必测。仿真提供起点，实测决定终点。制造公差不可避免，匹配网络是你的安全网。
