# MEMS压电能量采集器微型化设计

> **难度**：🔴 高级 | **领域**：MEMS能量采集 | **阅读时间**：约 22 分钟

## 引言

想象一块手表，你戴上它之后，每次挥臂走路，手表里的一根微小悬臂梁就在振动——这根梁上的压电薄膜把机械能变成电，给手表的传感器供电。你永远不用换电池，也永远不用充电。这就是 MEMS 压电能量采集器要实现的目标：从环境中"捡"振动能量，给 IoT 传感器节点自供电。

传统 IoT 节点依赖电池，但电池寿命有限、更换成本高(尤其在植入医疗、结构健康监测等不可达场景)。MEMS 压电能量采集器 (Piezoelectric Energy Harvester, PEH) 利用压电效应将环境振动转化为电能，体积可小至 mm 级别，与 CMOS 工艺兼容，是 IoT 自供电传感节点的关键技术路线之一。

本文系统梳理 MEMS 压电能量采集器的核心原理、材料选择、结构设计、频率调谐、功率调理电路、工艺流程及典型应用，并与电磁式、静电式采集器做横向对比。

## 1. 压电效应基础

### 1.1 正压电效应与逆压电效应

压电效应是某些晶体材料在受力变形时表面产生电荷的物理现象。正压电效应 (Direct Piezoelectric Effect) 是"力生电"：材料受机械应力时，晶胞中正负电荷中心相对位移，宏观表现为电极化。逆压电效应 (Converse Piezoelectric Effect) 是"电生力"：施加电场时材料产生应变。

对能量采集而言，我们利用正压电效应：环境振动使压电材料周期性变形，上下表面产生交变电压，通过外电路提取电能。

### 1.2 d31 模式与 d33 模式

压电常数 d 下标表示应力方向与极化方向的关系，两种工作模式是 MEMS 采集器设计的核心选择：

| 对比项 | d31 模式 | d33 模式 |
|--------|----------|----------|
| 应力方向 | 沿长度方向(x), 极化沿厚度(z) | 应力与极化同沿厚度(z) |
| 电极位置 | 上下表面(垂直于极化方向) | 上下表面(叉指电极也可) |
| 压电常数 | d31 (较小, PZT 约 -170 pC/N) | d33 (较大, PZT 约 400 pC/N) |
| 开路电压 | 较低 (几十 mV ~ 几 V) | 较高 (几 V ~ 几十 V) |
| MEMS 工艺兼容性 | 高 (平面电极, 标准工艺) | 需叉指电极或厚膜极化 |
| 典型结构 | 悬臂梁上下电极 | 叠层或叉指电极结构 |

d31 模式是 MEMS 悬臂梁采集器的主流选择，因为平面电极工艺简单、与标准薄膜沉积流程兼容。d33 模式虽然压电常数更大、输出电压更高，但叉指电极 (Interdigitated Electrodes, IDE) 加工复杂，且薄膜厚度受限时增益有限。

压电本构方程 (d31 模式简化)：

```
D3 = d31 * T1 + eps_33 * E3
S1 = s11 * T1 + d31 * E3

D3: 电位移 (C/m2), T1: 轴向应力 (Pa), E3: 电场 (V/m)
S1: 轴向应变, s11: 弹性柔度 (m2/N), eps_33: 介电常数 (F/m)
d31: 压电常数 (C/N 或 m/V)
```

## 2. 压电材料对比

### 2.1 三大材料体系

MEMS 压电采集器可选的薄膜材料主要有三种，各自在压电性能、工艺兼容性和生物相容性上差异显著：

| 参数 | PZT (锆钛酸铅) | AlN (氮化铝) | PVDF (聚偏氟乙烯) |
|------|----------------|-------------|-------------------|
| d31 (pC/N) | -170 ~ -274 | -1.0 ~ -2.5 | 12 ~ 23 |
| d33 (pC/N) | 300 ~ 590 | 3.0 ~ 5.5 | -20 ~ -33 |
| eps_r (相对介电常数) | 300 ~ 1700 | 8.5 ~ 10.5 | 10 ~ 12 |
| 居里温度 (度C) | 250 ~ 380 | > 1000 | ~100 |
| 薄膜沉积方式 | Sputter / Sol-gel / MOCVD | Sputter (反应溅射) | Spin-coating / 打印 |
| CMOS 兼容性 | 含铅, 不兼容后端工艺 | 完全兼容 | 兼容 (低温工艺) |
| 柔性 | 否 (陶瓷) | 否 (陶瓷) | 是 (聚合物) |
| 典型薄膜厚度 | 0.5 ~ 5 um | 0.5 ~ 2 um | 5 ~ 50 um |
| 疲劳寿命 | 中等 (极化退化) | 高 (无极化退化) | 高 |

### 2.2 材料选择策略

- **PZT**：压电常数最高，功率输出最大，是追求高能量密度的首选。但含铅(欧盟 RoHS 限制)，与标准 CMOS 后端不兼容，需单独工艺线。Sol-gel 法沉积 PZT 薄膜工艺成熟，但厚度超过 2 um 时需多层沉积加退火，容易开裂。
- **AlN**：压电常数小两个数量级，但 CMOS 完全兼容——可在标准 CMOS 晶圆上直接沉积 AlN 薄膜，实现"先 CMOS 后 MEMS"的单片集成。无需极化工艺，不存在极化退化问题，长期稳定性优异。对低功耗 IoT 传感节点 (nW ~ uW 级) 足够。
- **PVDF**：柔性基底可选，适合可穿戴和生物植入场景。但压电常数较低，且居里温度仅约 100 度C，高温环境受限。优势在于可印刷加工，适合大面积、低成本制备。

实际选择取决于应用场景：工业振动监测选 PZT(高功率)，CMOS 单片集成选 AlN(兼容性)，可穿戴选 PVDF(柔性)。

## 3. 悬臂梁谐振采集器设计

### 3.1 悬臂梁结构原理

MEMS 压电采集器最常用的结构是压电复合悬臂梁 (Unimorph Cantilever)：一层弹性基底(Si 或金属)上沉积一层压电薄膜，末端可加质量块 (Proof Mass) 以降低谐振频率、增大振幅。

悬臂梁一端固定、一端自由，受到环境振动激励时，在谐振频率处振幅最大，压电薄膜应变最大，输出功率最高。这就像秋千——你在正确的节奏推，秋千越荡越高；频率不对，就荡不起来。

### 3.2 谐振频率计算

均匀悬臂梁一阶谐振频率：

```
f1 = (lambda1^2 / 2*pi) * sqrt(E*I / (rho*A*L^4))

lambda1 = 1.875 (一阶模态系数)
E: 等效弹性模量 (Pa), I: 截面惯性矩 (m4)
rho: 等效密度 (kg/m3), A: 截面积 (m2), L: 梁长 (m)
```

末端加质量块后，谐振频率显著降低：

```
f_eff = f1 / sqrt(1 + M_tip / (0.24 * m_beam))

M_tip: 末端质量块质量, m_beam: 梁自身等效质量
0.24: 等效质量系数(一阶模态)
```

典型设计参数：梁长 L=2000 um, 宽 W=500 um, Si 厚 10 um, PZT 厚 1 um，不加质量块时 f1 约 1200 Hz(远高于环境振动主频段)；加 0.5 mg 质量块后 f_eff 约 50~200 Hz，落入机械设备(50~200 Hz)、车辆(10~100 Hz)的振动频段。

### 3.3 功率输出模型

谐振状态下压电采集器输出到匹配负载的最大功率近似为：

```
P_max ~ (m_total * a^2) / (8 * zeta * omega)

m_total: 总等效质量 (kg), a: 激励加速度幅值 (m/s2)
zeta: 阻尼比 (典型 0.01 ~ 0.05), omega: 谐振角频率 (rad/s)
```

典型数值示例：

| 设计参数 | 值 |
|----------|-----|
| 梁尺寸 | 2000 x 500 um |
| Si 基底厚度 | 10 um |
| PZT 薄膜厚度 | 2 um |
| 末端质量块 | 0.5 mg |
| 激励加速度 | 1g (9.8 m/s2) |
| 谐振频率 | ~100 Hz |
| 阻尼比 | 0.02 |
| 输出功率 | 1 ~ 10 uW |

## 4. 频率匹配与带宽扩展

### 4.1 频率失配问题

环境振动频率往往不稳定——电机转速变化、风载波动、人体步频变化都导致激励频率漂移。而线性谐振采集器的 -3dB 带宽极窄 (典型仅 1 ~ 5 Hz)，频率偏移几个 Hz 功率就可能下降 50% 以上。这就像收音机调台——稍微偏离一点，信号就模糊了。

### 4.2 宽带化技术

**振动阵列法**：在同一芯片上集成多个不同长度的悬臂梁，各梁谐振频率错开覆盖一定频带(如 5 根梁覆盖 80~120 Hz)。代价是芯片面积增大 N 倍，但优势是线性系统、无阈值、功率叠加。

**非线性弹簧法**：引入 Duffing 非线性刚度 (m*x'' + c*x' + k1*x + k3*x^3 = F*cos(wt))，使谐振曲线出现滞回，带宽可展宽至 20~50 Hz。实现方式包括磁铁耦合、机械止挡、预应力梁。

**双稳态系统**：设计双势阱结构(如磁铁对悬臂梁施加吸引力)，使系统在两个稳态间跳跃，即使低频激励也能产生大幅度跨越运动，有效采集低频宽频振动。

**自调谐机构**：通过静电梳齿驱动或热执行器实时调节悬臂梁刚度追踪频率变化。系统复杂度较高，但带宽理论上无限制。

### 4.3 宽带技术对比

| 技术 | 带宽提升 | 面积开销 | 额外功耗 | 复杂度 |
|------|---------|---------|---------|--------|
| 振动阵列 | 5 ~ 20x | 高 (Nx) | 无 | 低 |
| 非线性弹簧 | 3 ~ 10x | 低 | 无 | 中 |
| 双稳态跳跃 | 5 ~ 15x | 中 | 无 (被动) | 中 |
| 自调谐 | 理论无限 | 中 | 高 (驱动) | 高 |

## 5. 功率调理电路

### 5.1 信号特征与整流

压电采集器输出交变电压，而 IoT 传感器需要稳定直流。功率调理链路：压电元件 -> 整流器 -> 滤波电容 -> DC-DC 变换 -> 储能/负载。

标准全桥整流器 (FBR) 最简单，但两个二极管压降 (每个 0.3~0.7 V) 在低压输出时损耗显著。PZT 采集器(开路电压几 V)FBR 可行；AlN 采集器(开路电压可能低于 1 V)需用低阈值方案。

### 5.2 同步开关技术

同步开关阻尼 (SSHI) 是提升压电采集功率的经典技术：压电电压达极值时闭合开关，等效电容 Cp 与电感 L 形成 LC 谐振，电压在半个 LC 周期内快速翻转。翻转后电压与后续应变同向叠加，提高输出。

```
电压翻转比 gamma = exp(-pi / (2*Q_LC))
功率提升倍数 P_SSHI / P_std ~ (1 + gamma) / (1 - gamma)
gamma = 0.5 ~ 0.9 时, 提升约 3 ~ 20 倍
```

代价是需要精确的峰值检测与开关控制电路，增加了系统复杂度和静态功耗。

### 5.3 DC-DC 变换与阻抗匹配

压电采集器存在最优负载电阻 R_opt = 1 / (2*pi*f*Cp)。PZT (Cp=10 nF, f=100 Hz) 时 R_opt 约 159 kOhm；AlN (Cp=50 pF, f=100 Hz) 时约 31.8 MOhm。

DC-DC 变换器控制方法：(1) 电阻匹配模式——固定占空比使输入阻抗=R_opt；(2) MPPT——扰动观察法调整占空比；(3) 开路电压比例法——V_opt = V_oc/2。拓扑选择上，Boost 适用于压电电压不够高时，Buck-Boost 适用于宽输入范围。

### 5.4 冷启动问题

储能电容完全放电时，控制电路没有供电。解决方案：(1) 并联纯被动整流通路，冷启动阶段被动充电至阈值后切换主动控制；(2) 利用振动驱动机械开关，无需电控；(3) 亚阈值工作的峰值检测电路，静态功耗 < 100 nW。

## 6. MEMS 制造工艺

### 6.1 PZT 薄膜悬臂梁工艺流程

典型 PZT 压电悬臂梁 MEMS 工艺流程 (正面沉积 + 背面 DRIE 释放)：

```
1. SOI 晶圆准备 (器件层 Si 10um / 埋氧 1um / 衬底 400um)
2. 底电极: 溅射 Ti/Pt (5/100 nm), 光刻+刻蚀图形化
3. PZT 沉积 (Sol-gel): 旋涂前驱体 -> 300度C预烘焙 -> 650度C退火
   重复 5~10 次至目标厚度 1~2 um (单层 < 100nm 防开裂)
4. 顶电极: 溅射 Ti/Au 或 Pt (100~200 nm), 剥离图形化
5. PZT 极化: 150度C + 100~150 kV/cm 电场, 保持 15~30 min
6. 背腔释放: 背面 DRIE 至埋氧层 -> HF 气相刻蚀去除埋氧层
7. 质量块: 电铸 Ni / 粘接 W / 保留局部 Si 块
```

### 6.2 AlN 薄膜工艺特点

AlN 的关键优势在于 CMOS 工艺兼容：反应溅射沉积(Al 靶在 N2/Ar 气氛中)，无需高温退火；c轴(0002)取向度决定压电性(XRD 摇摆曲线 FWHM < 2 deg 为高质量)；沉积过程中自发形成 c轴取向，无需极化；沉积温度 < 400度C，可在 CMOS BEOL 之后进行。

### 6.3 工艺难点与对策

| 难点 | 原因 | 对策 |
|------|------|------|
| PZT 开裂 | Sol-gel 多层沉积热应力积累 | 单层 < 100 nm, 阶梯式退火 |
| 极化退化 | PZT 疲劳/老化 | 掺杂改性 (Mn, Nb), 优化电极 |
| AlN 取向度不足 | 溅射参数偏离 | 种子层(Pt), 优化功率/气压比 |
| 释放粘连 (Stiction) | 湿法释放时毛细力 | 干法释放(DRIE+HF气相), CPD |
| 质量块对准偏差 | 双面光刻套刻误差 | 单面工艺, 埋氧自停止 |

## 8. 与电磁式、静电式采集器对比

### 8.1 三种机制原理对比

| 对比项 | 压电式 | 电磁式 | 静电式 |
|--------|--------|--------|--------|
| 转换机制 | 应变 -> 极化电荷 | 线圈切割磁力线 -> 感应电动势 | 电容变化 -> 电压变化 |
| 典型结构 | 压电悬臂梁 | 线圈 + 永磁体 | 可变电容 + 偏置电压 |
| 输出电压 | 高 (V ~ 几十V) | 低 (mV ~ 几百mV) | 高 (V ~ 几十V) |
| 输出电流 | 低 (uA) | 高 (mA) | 低 (uA) |
| MEMS 兼容性 | 好 (薄膜工艺) | 差 (集成磁体困难) | 好 (纯硅工艺) |
| 需要偏置 | 否 | 否 | 是 (初始偏压) |

### 8.2 尺寸缩放效应

微型化对不同机制的影响截然不同：压电式 P ~ L^2(与电极面积成正比，应变不变)，缩放友好，mm 级仍有效；电磁式 P ~ L^4(磁通~L^3，匝数~L，速度~L)，微型化后功率急剧下降，mm 级以下几乎不可用；静电式 P ~ L^2，缩放友好但需要初始偏置电压(鸡生蛋问题)。

结论：在 MEMS 尺度(mm 级及以下)，压电式是最优的能量采集机制。近年有研究将两种机制集成(如压电-电磁混合)，同时利用高电压和高电流输出，但双机制器件工艺复杂、面积开销大，仍处于实验室阶段。

## 9. IoT 自供电传感器节点应用

### 9.1 典型系统架构

```
[振动源] -> [MEMS PEH] -> [整流+SSHI] -> [DC-DC] -> [储能电容]
                                                      |
                                                 [PMU + LDO]
                                                      |
                                  +-------------------+-------------------+
                                  |                   |                   |
                             [MCU/传感器]        [ADC/FIFO]          [无线电]
                             (温度/加速度)      (数据缓冲)          (BLE/LoRa)

间歇工作模式 (Intermittent Computing):
1. 储能电容充电至 V_high -> 2. MCU 上电采样 -> 3. 累积数据后无线发送
-> 4. 电压降至 V_low, MCU 断电 -> 5. 重复
```

### 9.2 应用场景

| 场景 | 振动源 | 频率范围 (Hz) | 可用功率 (uW/cm3) |
|------|--------|-------------|-------------------|
| 工业设备监测 | 电机/泵/压缩机 | 50 ~ 500 | 10 ~ 500 |
| 桥梁/建筑 SHM | 风/交通/地震 | 1 ~ 30 | 0.1 ~ 10 |
| 人体健康监测 | 行走/心跳/呼吸 | 1 ~ 10 | 0.5 ~ 5 |
| 航空发动机 | 叶片/转子 | 100 ~ 2000 | 50 ~ 1000 |
| 铁路轨道监测 | 列车通过 | 5 ~ 50 | 1 ~ 50 |
| 水管管网监测 | 水流脉动 | 10 ~ 100 | 0.5 ~ 20 |

### 9.3 间歇计算挑战

储能电容小、供电不稳定导致 MCU 可能随时断电，带来独特软件挑战：

1. **状态保持**：断电前将寄存器和 RAM 关键数据写入非易失存储 (FRAM/Flash)。
2. **任务原子性**：一个采样-处理-发送周期必须能在一次供电窗口内完成，或设计为可中断-恢复。
3. **时间感知**：断电期间 RTC 停止，需要外部事件或近似时序估计。
4. **能量感知调度**：根据当前储能电容电压动态决定执行哪些任务(采样 vs 计算 vs 发送)。

## 10. 设计优化方向

### 10.1 材料层面

- **PZT 掺杂改性**：Mn 掺杂提高品质因子 Q，Nb 掺杂提高 d31，兼顾高输出与低损耗。
- **Relaxor-PT 单晶**：PMN-PT、PZN-PT 的 d33 可达 2000+ pC/N，远超多晶 PZT，但薄膜制备仍在研究中。
- **AlN 掺钪 (ScAlN)**：Sc 掺杂可将 d33 提升至 25~30 pC/N(比纯 AlN 高 5 倍以上)，是近年最热门的 CMOS 兼容压电材料。

### 10.2 结构层面

- **梯形/三角形梁**：等强度梁设计，应变沿梁均匀分布，材料利用率最高。
- **折叠梁/S形梁**：在有限面积内增加有效梁长，降低谐振频率。
- **多模态采集**：同一器件同时采集振动和热能(压电+热电复合)。

### 10.3 电路层面

- **自供电 SSHI**：无需外部供电的同步开关电路，用压电电压本身驱动开关。
- **超低功耗 MPPT**：亚阈值 CMOS 实现纳瓦级最大功率点追踪。
- **多采集器能量合并**：阵列中多个梁的输出通过 OR-ing 整流合并。

## 总结

MEMS 压电能量采集器的微型化设计是一个材料-结构-电路-工艺四维耦合的优化问题。核心要点：

1. **d31 模式悬臂梁是 MEMS PEH 的主流结构**，工艺简单、兼容性好；d33 模式在叉指电极工艺突破后功率密度更高。
2. **PZT 功率密度最高但不环保且 CMOS 不兼容**；AlN 是 CMOS 单片集成的最佳选择，ScAlN 将进一步缩小差距。
3. **频率匹配是性能关键**，窄带谐振在真实环境中功率衰减严重；非线性/双稳态/阵列是主要的宽带化手段。
4. **uW 级功率可支撑间歇采样的 IoT 节点**，但无线传输需储能+低占空比策略。
5. **压电式在微型化维度上优于电磁式和静电式**，是 MEMS 能量采集的首选机制。
6. **间歇计算是自供电节点的独特软件范式**，需要硬件-软件协同设计。

MEMS 压电能量采集要走向大规模商用，仍需在降低谐振频率(适配环境振动)、提高可靠性(疲劳/老化)、降低系统级成本三方面持续突破。
