# LDO与DC-DC电源转换器效率对比与选型
> **难度**: 绿 初级 | **领域**: 电源设计基础 | **阅读时间**: 约 18 分钟

## 引言

想象你正在给手机充电,手机电池是3.7V,充电器输出5V。这多出来的1.3V去哪了?如果用"电阻降压"的方式,多出的电压全部变成热量白白浪费;但如果用一个智能变压器,只取需要的能量,效率就高得多。

这个比喻恰好对应了电源设计中两个最基础的架构:线性稳压器(LDO)就像电阻降压,简单但效率受限于压差;开关稳压器(DC-DC)就像智能变压器,效率高但电路更复杂。IoT设备通常由电池供电,对效率和体积都有苛刻要求,选对电源架构直接影响续航和成本。

本文从原理出发,系统对比LDO与DC-DC的效率、噪声、静态电流等关键指标,给出IoT项目的选型决策树和实战建议。

## 1. LDO线性稳压器原理

### 1.1 基本工作方式

LDO(Low Dropout Regulator)的核心是一个串联调整管,它串联在输入和输出之间,像一个可变电阻,根据输出电压的反馈自动调节自身的导通程度,使输出电压保持稳定。

```
Vin ---[ 调整管(PASS) ]--- Vout
              |
           误差放大器
              |
          基准电压(Vref)
```

关键参数:
- 压差(Dropout Voltage): 输出电压能保持稳定所需的最小Vin-Vout差值
- 典型LDO压差: 50-500mV,超低压差型号可低至30mV

### 1.2 效率计算

LDO的效率公式非常直观:

```
Efficiency = Vout / Vin x 100%
```

例如: Vin=3.7V(锂电池), Vout=3.3V:
- 效率 = 3.3 / 3.7 = 89.2%
- 损耗功率 = (Vin - Vout) x Iload = 0.4V x 100mA = 40mW

但如果Vin=5V, Vout=3.3V:
- 效率 = 3.3 / 5.0 = 66%
- 损耗功率 = 1.7V x 100mA = 170mW

压差越大,效率越低,这是LDO的根本限制。

### 1.3 静态电流

静态电流(Iq)是LDO自身消耗的电流,不流向负载。对IoT设备至关重要,因为传感器节点90%以上时间处于睡眠状态,此时负载电流可能仅1uA,如果LDO的Iq是50uA,那稳压器自身功耗就是负载的50倍。

典型值:
- 普通LDO: Iq = 50-100uA
- 低Iq LDO: Iq = 1-5uA
- 超低Iq LDO: Iq < 1uA (如TPS7A02: 25nA)

## 2. DC-DC开关稳压器原理

### 2.1 基本工作方式

DC-DC转换器通过高频开关(数百kHz至数MHz)对输入能量进行"切割",然后通过电感和电容的储能滤波,输出不同的电压。核心思想是:开关管导通时存储能量,关断时释放能量,理论上没有纯电阻损耗。

```
Vin ---[ 开关管 ]---[ 电感 ]--- Vout
          |               |
       续流二极管      输出电容
          |               |
         GND             GND
```

### 2.2 效率特性

DC-DC的效率通常在85-95%之间,且与Vin-Vout的差值关系不大(不像LDO那样线性恶化)。

| 条件 | LDO效率 | DC-DC效率 |
|------|---------|-----------|
| 3.7V->3.3V, 100mA | 89% | 90-93% |
| 5.0V->3.3V, 100mA | 66% | 88-92% |
| 12V->3.3V, 100mA | 28% | 85-90% |

DC-DC的损耗来源:
- 开关损耗: 开关管开关过程中的交叠损耗,与频率和电压成正比
- 导通损耗: 开关管导通电阻(Rds_on)产生的I^2R损耗
- 电感DCR损耗: 电感直流电阻
- 驱动损耗: 驱动开关管栅极的损耗
- 死区时间损耗: 同步Buck中体二极管导通损耗

### 2.3 三种基本拓扑

| 拓扑 | 功能 | 典型应用 |
|------|------|----------|
| Buck(降压) | Vin > Vout | 锂电池->MCU 3.3V |
| Boost(升压) | Vin < Vout | 单节电池->5V |
| Buck-Boost(升降压) | Vin可大于或小于Vout | 锂电池全电压范围->3.3V |

## 3. 噪声与纹波对比

### 3.1 LDO的噪声特性

LDO输出几乎没有开关纹波,输出噪声主要来源于:
- 基准电压噪声: 通常20-100uV_rms
- 误差放大器噪声
- 输入电源纹波抑制比(PSRR): 好的LDO在1kHz处PSRR可达60-90dB

这使得LDO特别适合给噪声敏感的模拟电路供电,如ADC参考电压、RF前端、传感器等。

### 3.2 DC-DC的噪声特性

DC-DC输出含有开关纹波,典型值10-50mV_pp,频率等于开关频率。此外还有高频尖峰(开关边沿引起),可达100mV_pp。

降低DC-DC纹波的措施:
- 增大输出电容和降低ESR
- 使用LC后级滤波器
- 优化PCB布局减小寄生参数
- 选择更高的开关频率(纹波更易滤除)

### 3.3 噪声对比总结

| 指标 | LDO | DC-DC |
|------|-----|-------|
| 输出纹波 | <10uV | 10-50mV |
| PSRR(1kHz) | 60-90dB | 20-40dB |
| 高频噪声 | 极低 | 显著(尖峰) |
| 适合负载 | 模拟/RF | 数字/电机 |

## 4. 压差与热设计

### 4.1 压差(Dropout)

LDO的压差决定了输入电压最低能到多少仍保持稳压输出。对锂电池供电的IoT设备:
- 锂电池满电: 4.2V
- 锂电池截止: 3.0V
- MCU工作电压: 3.3V
- 若LDO压差200mV,则输入低于3.5V时输出开始跌落

因此选择超低压差LDO(压差<100mV)可以更充分利用电池电量。

### 4.2 热设计

LDO的功耗全部转化为热量,封装的散热能力决定了最大允许功耗:

```
Pd_max = (Tj_max - Ta) / Rth_ja
```

例如: Tj_max=125C, Ta=85C(工业级), SOT-23封装Rth_ja约250C/W:
- Pd_max = (125-85)/250 = 160mW
- 若Vin=5V, Vout=3.3V, 最大电流 = 160mW / 1.7V = 94mA

SOT-23封装的LDO在5V->3.3V场景下仅能输出不到100mA! DC-DC则几乎没有这个限制。

## 5. IoT场景下的选型决策

### 5.1 何时选择LDO

1. Vin-Vout压差小(<1V): LDO效率可接受,且更简单
2. 噪声敏感的模拟电路: ADC参考、传感器激励、RF前端
3. 负载电流小(<50mA): 损耗的绝对值很小
4. PCB空间和成本极度受限: LDO只需1uF输入输出电容
5. 瞬态响应要求高: LDO没有控制环路延迟

### 5.2 何时选择DC-DC

1. Vin-Vout压差大(>2V): LDO效率太低,热量无法散出
2. 电池供电且需要长续航: 85-95%效率可节省可观电量
3. 负载电流大(>200mA): LDO损耗和散热成为瓶颈
4. 输入电压范围宽: 如USB 5V或太阳能等波动源

### 5.3 混合拓扑: DC-DC + LDO后级稳压

这是IoT中最常见的折中方案: 先用DC-DC高效降压到略高于目标电压(如3.6V),再用LDO稳压到3.3V。

```
锂电池4.2V --[DC-DC Buck]--> 3.6V --[LDO]--> 3.3V(低噪声)
```

优势:
- DC-DC承担大部分压差,保证整体效率
- LDO压差仅300mV,效率 = 3.3/3.6 = 91.7%
- LDO滤除DC-DC的开关纹波,输出干净
- 总效率 = DC-DC效率 x LDO效率 = 90% x 91.7% = 82.5%

相比纯LDO: 3.3/4.2 = 78.6%,混合方案效率更高且噪声更低。

### 6.3 选型代码示例

根据应用场景快速筛选的伪代码:

```
function select_regulator(vin, vout, iload, noise_sensitive):
    vdiff = vin - vout

    if vdiff < 1.0 and iload < 200mA:
        return "LDO"  // 压差小,电流小,LDO足够

    if noise_sensitive == true and vdiff < 2.0:
        return "LDO or DC-DC + LDO"

    if vdiff > 2.0 or iload > 300mA:
        if noise_sensitive:
            return "DC-DC + LDO"  // 混合方案
        else:
            return "DC-DC"

    if battery_powered and duty_cycle < 1%:
        return "Ultra-low-Iq LDO (Iq < 100nA)"

    return "DC-DC (default for IoT)"
```

## 7. IoT电源选型决策树

```
开始
 |
 +-- Vin-Vout压差 < 1V?
 |    +-- 是 --> 负载电流 < 200mA?
 |    |         +-- 是 --> LDO
 |    |         +-- 否 --> DC-DC + LDO
 |    +-- 否 --> 噪声敏感?
 |              +-- 是 --> DC-DC + LDO后级
 |              +-- 否 --> DC-DC
 |
 +-- 电池供电?
      +-- 是 --> 睡眠电流占比 > 99%?
      |         +-- 是 --> 超低Iq LDO (Iq < 100nA)
      |         +-- 否 --> 超低Iq DC-DC (Iq < 1uA)
      +-- 否 --> 看压差和电流决定
```

实际决策要点:
1. 计算工作模式下的总效率,不能只看峰值效率
2. 计算睡眠模式下的总功耗,Iq是关键
3. 评估热设计是否可行,封装散热能力是否足够
4. 考虑BOM成本和PCB面积: LDO更简单但可能效率不够
5. 预留裕量: 输入电压波动、负载瞬态、温度范围

## 8. 实战设计要点

### 8.1 LDO使用注意事项

- 输入输出电容选择: 输入1uF陶瓷电容,输出1-4.7uF低ESR陶瓷电容
- 压差计算要留裕量: 至少比标称值多留50mV
- 热设计: 计算最恶劣工况下的功耗和温升
- PCB布局: 输入输出电容尽量靠近IC引脚

### 8.2 DC-DC使用注意事项

- 电感选择: 饱和电流大于峰值电流+30%, DCR越低效率越高
- 输入电容: 必须低ESR,防止输入电压跌落和振铃
- 布局关键: 功率环路面积最小化,减小EMI
- 续流二极管: 用肖特基二极管,正向压降低,反向恢复快
- 使能控制: 利用EN引脚在睡眠时关闭DC-DC节省电量
- 软启动: 多数DC-DC内置软启动,防止上电浪涌电流过大;外接SS引脚电容可调节启动时间
- 频率折中: 1-2MHz适合IoT,兼顾效率和尺寸;高频需注意EMI,低频需注意电感体积

### 8.3 常见踩坑

1. LDO输入电压瞬态尖峰超过绝对最大额定值 --> 加TVS保护
2. DC-DC电感饱和导致效率骤降和过热 --> 选电感时看饱和电流,不只看额定电流
3. LDO输出电容ESR过高导致振荡 --> 使用陶瓷电容(ESR<100mOhm)
4. DC-DC布局不合理导致EMI不过关 --> 功率环路走线短粗,信号地与功率地分开
5. 忽略Iq导致电池待机时间远低于预期 --> 选型时Iq和效率同等重要
6. DC-DC使能引脚浮空导致输出不确定 --> EN引脚必须明确上拉或下拉
7. 陶瓷电容直流偏置导致实际容值远低于标称值 --> 选2倍额定电压以上的规格

### 8.4 IoT电源设计清单

在开始PCB设计前,建议逐项确认:

- [ ] 列出所有电压轨及其电流需求(峰值和平均)
- [ ] 计算每路的效率,确认总功耗在预算内
- [ ] 计算睡眠模式总功耗(Iq x Vbat),确认满足续航目标
- [ ] 热设计验证: 最恶劣工况下结温低于Tj_max - 20C
- [ ] 噪声敏感电路是否有独立LDO或LC滤波
- [ ] 电池放电曲线末端是否仍能保持输出稳定
- [ ] 所有DC-DC的EN引脚有明确上拉/下拉
- [ ] 陶瓷电容选型考虑直流偏置降额

## 总结

LDO和DC-DC各有不可替代的优势。LDO简单、安静、便宜,适合小压差和噪声敏感场景;DC-DC高效、灵活,适合大压差和大电流场景。IoT设备最常用的策略是混合拓扑: DC-DC提供高效率的主电源转换,LDO提供低噪声的后级稳压。选型的核心思路是: 先算效率,再看噪声,最后看热设计。不要忘记,IoT设备的睡眠功耗往往决定了实际续航,因此Iq参数在IoT选型中具有一票否决权。
