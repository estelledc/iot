# 电桥电路传感器激励与零点调整

> **难度**：🟡 中级 | **领域**：传感器接口电路 | **阅读时间**：约 20 分钟

## 引言

想象一个老式的杆秤：你在一端挂上货物，另一端移动秤砣，直到秤杆水平——这就叫"平衡"。惠斯通电桥的原理一模一样：四个电阻像四个"秤砣"，当对角乘积相等时，中间的电压差为零，秤杆"水平"。但现实中传感器不是完美对称的——制造公差、温度漂移、引线电阻都像在秤杆上偷偷加了小砝码，让秤杆歪了。零点调整就是把秤杆重新调水平的过程，而激励方式则决定了你的秤有多灵敏、多稳定。

再打个比方：电桥就像一个跷跷板，传感器信号让一端微微下沉，激励电源是跷跷板的底座——底座不稳，整个跷跷板的读数都会晃。激励方式的选择（电压还是电流）和零点调整的精度，直接决定了你能测多小的信号、读数能有多准。

## 1. 惠斯通电桥基本原理

### 1.1 电路结构与平衡条件

惠斯通电桥由四个电阻臂组成，对角施加激励，另一对角读取差分输出：

```
    Vex (+)
     |
    [R1]        [R2]
     |            |
     +--- Vo+ ---+
     |            |
    [R3]        [R4]
     |            |
    GND          GND
```

输出电压：

```
Vo = Vex * (R1/(R1+R3) - R2/(R2+R4))
```

当 R1*R4 = R2*R3 时 Vo = 0，电桥平衡。实际传感器四臂电阻初始值难以完全匹配，零点偏移几乎是必然的。

### 1.2 灵敏度定义

电桥灵敏度指单位被测物理量变化引起的输出电压变化：

```
S = dVo / d(delta_R/R)
```

灵敏度越高，对后级放大和 ADC 的要求越低。

## 2. 电桥配置：四分之一桥、半桥、全桥

### 2.1 四分之一桥(1/4 Bridge)

只有一个桥臂是传感元件，其余三个为固定电阻。输出近似：

```
Vo = Vex * (delta_R / R) / 4    (当 delta_R << R 时)
```

灵敏度最低，布线最简单，温度漂移无法自补偿。典型应用：低成本应变测试、简单测温。

### 2.2 半桥(1/2 Bridge)

两个桥臂为传感元件，通常一个受拉、一个受压，输出叠加：

```
Vo = Vex * (delta_R / R) / 2
```

灵敏度是 1/4 桥的 2 倍，同温度场下温漂可部分抵消。典型应用：弯曲梁上下表面贴片。

### 2.3 全桥(Full Bridge)

四个桥臂全部为传感元件，两片受拉、两片受压：

```
Vo = Vex * (delta_R / R)
```

灵敏度最高，是 1/4 桥的 4 倍；同材质同温度场时温度自补偿。典型应用：称重传感器、高精度压力传感器。

### 2.4 配置对比

| 配置 | 传感臂数 | 灵敏度 | 温度自补偿 | 布线复杂度 | 典型场景 |
|------|:---:|:---:|:---:|:---:|----------|
| 1/4 桥 | 1 | Vex/4 * dR/R | 无 | 低 | 简单测温、低成本测试 |
| 1/2 桥 | 2 | Vex/2 * dR/R | 部分 | 中 | 弯曲梁、差温测量 |
| 全桥 | 4 | Vex * dR/R | 完全 | 高 | 称重、精密压力 |

## 3. 电压激励与电流激励

### 3.1 电压激励

在电桥两端施加恒定电压 Vex。

- 优点：实现简单，ADC 参考引脚可直接使用同一电压源
- 缺点：输出与 Vex 成正比，Vex 波动直接影响精度
- 输出：Vo = Vex * f(dR/R)

### 3.2 电流激励

在电桥两端施加恒定电流 Iex。

- 优点：对引线电阻不敏感，适合远距离传感器
- 缺点：电桥等效电阻随温度变化，自热控制更复杂
- 输出：Vo = Iex * R * f(dR/R)

### 3.3 选择决策

| 考量因素 | 电压激励优先 | 电流激励优先 |
|----------|:---:|:---:|
| 引线长度 | < 5m | > 5m |
| ADC 参考方式 | 比例测量 | 独立参考 |
| RTD 测温 | - | 优先 |
| 应变片/称重 | 优先 | - |
| 多传感器切换 | - | 优先 |

## 4. 比例测量的优势

### 4.1 原理

让 ADC 的参考电压 Vref 与电桥激励 Vex 来自同一电压源。ADC 输出码：

```
D = (Vo / Vref) * 2^n = (Vo / Vex) * 2^n
```

Vo 和 Vref 都与 Vex 成正比，Vex 在分子分母中约掉：D = f(dR/R) * 2^n。Vex 波动不再影响结果。

### 4.2 对比

| 项目 | 非比例测量 | 比例测量 |
|------|-----------|----------|
| Vex 波动影响 | 直接引入误差 | 完全消除 |
| 基准电压要求 | 精密基准源 | 与 Vex 共用即可 |
| 长期稳定性 | 依赖 Vex 漂移 | 不依赖 |
| 典型芯片 | ADS1248 | HX711 |

### 4.3 实现要点

1. Vex 和 Vref 必须来自同一节点，不能各自稳压
2. 走线尽量短且对称
3. Vex 经过开关时 Vref 切换要同步
4. Vex 驱动能力要同时满足电桥和 ADC 参考的电流需求

## 5. 电桥偏移与零点调整

### 5.1 偏移来源

零点偏移(Offset)指无被测量时输出不为零，主要来源：

1. **制造公差**：四臂电阻初始值不等，典型偏移 0.5% ~ 2% FS
2. **温度系数不匹配**：温漂典型 0.05% FS/C
3. **安装应力**：粘贴或安装引入的预应力
4. **引线电阻**：长导线引入的不对称电阻
5. **时效漂移**：长期使用后材料蠕变

### 5.2 电位器法

在电桥输出端并联电位器，滑臂接 Vex 或 GND，调节滑臂注入小电流抵消偏移：

```
          Vex (+)
           |
      [R1]      [R2]
       |          |
       +-- Vo+ --+
       |   |      |
       | [Radj]   |
       |   |      |
       +-- Vo- --+
       |          |
      [R3]      [R4]
       |          |
      GND        GND
```

简单低成本，但电位器有机械磨损和温漂，长期稳定性差。适用：消费级产品、一次性校准。

### 5.3 DAC 注入法

用 DAC 输出直流电压，通过大电阻(R_inject，通常为桥臂电阻 100 倍以上)注入到 Vo+ 或 Vo-：

```
DAC_out ---[R_inject]--- Vo+
```

可编程、可远程校准、无机械磨损；但需额外 DAC，DAC 噪声和漂移会耦合到信号路径。适用：工业仪表、定期自动校准系统。

### 5.4 软件零点调整

MCU 存储零点偏移值，读数后减去：V_corrected = V_measured - V_offset。

零硬件成本、灵活性最高；但不释放 ADC 动态范围（偏移仍占用量程）。适用：几乎所有现代系统，作为硬件补偿的补充。

### 5.5 三种方法对比

| 方法 | 精度 | 稳定性 | 成本 | 自动化 | 动态范围影响 |
|------|:---:|:---:|:---:|:---:|:---:|
| 电位器 | 中 | 差 | 低 | 无 | 消除偏移 |
| DAC 注入 | 高 | 良 | 中 | 有 | 消除偏移 |
| 软件补偿 | 高 | 优 | 零 | 有 | 占用 ADC 量程 |

工程实践：硬件粗调释放 ADC 量程 + 软件精调补偿残余和温漂。

## 6. 温度补偿技术

### 6.1 温度漂移机制

应变片温漂有两个独立效应：

1. **电阻温度系数(TCR)**：即使无应变也产生信号，康铜约 10-20 ppm/C
2. **灵敏度系数温度系数(TCGF)**：GF 值随温度变化，康铜约 -0.01%/C

### 6.2 全桥自补偿

四臂同材质、同温度场时，温度变化使四臂同比例增减，差分输出不变。最优雅的补偿方式，无需额外电路。前提：四臂必须同温度，温度梯度大时自补偿失效。

### 6.3 串联/并联电阻补偿

在适当桥臂串联温度系数相反的温补电阻 Rs(T)，抵消应变片温漂。选值通过实验标定确定。

### 6.4 软件温度补偿

在传感器附近安装温度传感器，MCU 测温后通过多项式修正：

```
V_compensated = V_measured * (1 + k1*(T - T0) + k2*(T - T0)^2)
```

k1、k2 通过多点标定回归得到。

| 补偿方式 | 精度 | 额外硬件 | 标定工作量 | 适用场景 |
|----------|:---:|:---:|:---:|----------|
| 全桥自补偿 | 良 | 无 | 低 | 四臂同温场 |
| 串并联电阻 | 高 | 温补电阻 | 中 | 全桥不够时 |
| 软件补偿 | 最高 | 温度传感器 | 高 | 高精度系统 |

## 7. 引线电阻补偿

### 7.1 问题严重性

10m 长 24AWG 铜导线电阻约 0.84 ohm，对 350 ohm 电桥占比 0.24% FS——在 0.05% 精度要求的工业秤中不可接受。

### 7.2 三线制补偿

在传感元件一端引出第三根线，利用两根引线的对称性抵消。前提：三根引线材质、线径、长度、温度必须一致。三线制通常能将引线误差降低一个数量级。

### 7.3 四线制(Kelvin 接法)

两根线通激励电流，另两根线测量电压。电压测量回路电流趋近于零，引线电阻压降可忽略，误差完全消除。代价是多一根线。适用：高精度 RTD 测温、远距离传感器。

### 7.4 方式选择

| 引线方式 | 引线数 | 引线误差消除 | 成本 | 适用场景 |
|----------|:---:|:---:|:---:|----------|
| 二线制 | 2 | 无 | 低 | 短距离、低精度 |
| 三线制 | 3 | 大部分消除 | 中 | 工业主流 |
| 四线制 | 4 | 完全消除 | 高 | 高精度、长距离 |

## 8. 电桥放大器电路设计

### 8.1 INA + 参考电压拓扑

电桥输出为差分小信号，需仪表放大器(INA)进行差分放大和电平转换。INA 的 REF 引脚设定输出零点，通常接 Vex/2 使输出对称摆动，最大化 ADC 动态范围。

### 8.2 增益选择

```
G = V_ADC_range / (Vex * sensitivity)
```

示例：5V 激励、2 mV/V 传感器、ADC 量程 0~3.3V：

```
Vo_max = 5V * 0.002 = 10 mV
G = 3.3V / (2 * 10mV) = 165    (取 150 或 200，留裕量)
```

### 8.3 常用 INA 芯片

| 芯片 | 增益范围 | CMRR(dB) | 偏移(uV) | 供电 | 典型应用 |
|------|----------|:---:|:---:|----------|----------|
| INA128 | 1~10000 | 120 | 50 | +/-2.25~18V | 通用精密 |
| AD620 | 1~10000 | 110 | 50 | +/-2.3~18V | 工业过程 |
| INA333 | 1~1000 | 115 | 10 | 1.8~5.5V | 低功耗便携 |
| HX711 | 128/64 | - | - | 2.7~5.5V | 称重专用 |

### 8.4 REF 引脚驱动

REF 引脚必须低阻抗驱动。直接用电阻分压阻抗过高，会导致偏移和 CMRR 恶化。正确做法：电阻分压后接运放跟随器，再连到 REF。这是初学者最常见的错误之一。

## 9. 实际应用示例

### 9.1 应变片称重系统

```
传感器：350 ohm 全桥，2 mV/V
激励：5V 电压激励（比例测量）
ADC：HX711（24 bit，PGA G=128）

Vo_max = 5V * 2mV/V = 10 mV
HX711 输入范围 = 5V/128 = 39 mV
理论有效位数约 6.5M counts，实际受噪声限制约 17~19 bit

零点：软件去皮    引线：四线制
```

### 9.2 RTD 温度测量

```
传感器：PT100    激励：1 mA 恒流
量程：-50C ~ +200C

-50C: R=80.31 ohm, V=80.31 mV
+200C: R=175.86 ohm, V=175.86 mV
满量程输出 = 95.55 mV

选用 INA333 (G=51)，REF 接 4.096V 基准
引线：三线制/四线制    温补：软件多项式
```

### 9.3 压力传感器接口

```
传感器：压阻全桥，1.5 mV/V，5V 激励
量程：0~1 MPa，工作温度 -40C~+125C

Vo_max = 7.5 mV，选 AD620 (G=200)，Vout_max = 1.5V
温补：片上温敏电阻 + 软件查表
零点：DAC 注入 + 软件补偿    引线：四线制
```

## 10. 噪声与干扰考虑

### 10.1 主要噪声源

| 噪声源 | 频率特征 | 幅度 | 抑制方法 |
|--------|----------|------|----------|
| 电阻热噪声 | 白噪声 | nV/sqrt(Hz) | 低通滤波 |
| 工频干扰 | 50/60 Hz 及谐波 | uV~mV | 屏蔽 + 陷波 |
| 开关电源纹波 | 数十 kHz~MHz | uV | LDO 或 LC 滤波 |
| 电磁耦合 | 宽带 | uV~mV | 双绞线 + 屏蔽 |

### 10.2 布线与屏蔽

1. Vo+ 和 Vo- 用双绞线走线，INA 可抑制共模干扰
2. 屏蔽层单点接地（通常在 ADC 端），避免地环路
3. Vex 入口加 LC 滤波，INA 电源引脚加 0.1 uF 陶瓷电容
4. 模拟地和数字地在 ADC 处单点连接

### 10.3 滤波策略

```
电桥 ---[RC: fc=10Hz]--- INA ---[RC: fc=100Hz]--- ADC
```

第一级在 INA 前端滤除高频噪声并保护输入，第二级在 INA 输出端进一步抑制。工频干扰严重时可加数字陷波（50/60 Hz IIR 陷波器）或同步采样。

## 11. 软件自动校准

### 11.1 零点自动校准

```
1. 断开被测量（去皮操作）
2. 采集 N 个样本（N >= 64，过采样降噪）
3. V_offset = mean(V_samples)
4. 存入非易失存储器
5. 后续：V_corrected = V_measured - V_offset
```

### 11.2 增益自动校准

施加已知标准量 Fs（如砝码），采集 V_fs，则增益系数 K = Fs / (V_fs - V_offset)。后续测量：Value = (V_measured - V_offset) * K。

### 11.3 温度补偿查表法

在多个温度点校准零点和增益，存入查找表：

```
T = -20C: offset=-0.15mV, K=999.85
T =   0C: offset=-0.02mV, K=1000.02
T = +25C: offset= 0.00mV, K=1000.00
T = +50C: offset= 0.08mV, K=999.92
T = +85C: offset= 0.21mV, K=999.79
```

运行时根据实时温度线性插值(offset, K)。

### 11.4 校准数据管理要点

- 存储介质选 EEPROM(>100K 擦写)或 Flash 分区
- 用 CRC16/CRC32 保护数据完整性
- Flash 有限擦写次数(10K~100K)，避免频繁校准
- 出厂校准一次写入，运行时只更新温漂参数

## 总结

1. **电桥配置**决定灵敏度上限：全桥 4 倍于 1/4 桥，且温度自补偿；1/4 桥最简单但需外部补偿。
2. **激励方式**影响系统架构：电压激励适合比例测量，Vex 波动被消除；电流激励适合远距离 RTD 测温。
3. **零点调整**三层防线：硬件粗调释放 ADC 量程，软件精调补偿残余和温漂，两者结合是工程标准做法。
4. **温度补偿**分层处理：全桥自补偿优先，不够加温补电阻，最终软件查表收尾。
5. **引线补偿**按精度选择：三线制是工业主流，四线制是高精度标配。
6. **噪声控制**贯穿信号链：布线屏蔽、滤波网络、数字后处理每一环都要做对。
7. **自动校准**保障长期可靠：零点和增益周期性校准配合温度查表，实现工业级精度。
