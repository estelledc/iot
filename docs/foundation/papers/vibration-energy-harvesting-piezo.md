# 振动能量采集压电换能器匹配电路

> **难度**：🔴 高级 | **领域**：振动能量采集 | **阅读时间**：约 22 分钟

## 引言

想象你住在铁路边，每趟列车经过时窗户都在嗡嗡振动——这些能量平时全散掉了。如果把压电片贴在窗框上，振动让压电片反复弯曲，两端就产生交变电压。但这个电压又小又乱，怎么变成 IoT 传感器能用的稳定直流电？这就需要匹配电路——就像变速箱把发动机的高转速低扭矩变成轮子需要的低转速高扭矩，匹配电路把压电片的高阻抗交变输出变换成负载需要的低阻抗直流供电。

同一片压电片，匹配电路不同，输出功率可以差 4-10 倍。压电换能器本身只完成了"力生电"这一步，从压电片两端到 IoT 节点的 VCC 引脚之间，隔着整流、滤波、阻抗匹配、稳压好几道关卡。本文系统梳理机电建模、标准整流、SSHI、SECE、商用芯片、宽带技术及部署案例。

## 1. IoT 环境中的振动源

### 1.1 振动源分类

IoT 节点可利用的振动源遍布工业与生活场景，频率和加速度差异极大：

| 振动源 | 主频范围(Hz) | 加速度幅值(g) | 典型场景 |
|--------|-------------|---------------|----------|
| 工业电机 | 50-200 | 0.5-5 | 旋转机械、泵站 |
| HVAC 风机 | 30-120 | 0.1-1 | 楼宇暖通 |
| 桥梁/铁路 | 1-30 | 0.01-0.5 | 结构健康监测 |
| 汽车发动机 | 50-300 | 0.5-10 | 车载传感器 |
| 人体运动 | 1-10 | 0.1-2 | 可穿戴设备 |
| 航空发动机 | 200-2000 | 5-50 | 航空健康监测 |

### 1.2 振动特性对采集的影响

三个关键参数决定可采集功率的大小：

1. **频率**：压电采集器须在谐振频率附近工作，偏离 5% 功率即下降 50% 以上
2. **加速度幅值**：功率与加速度平方近似正比(小振幅范围内)
3. **频谱纯度**：窄带振动(如旋转机械)适合窄带谐振采集器；宽带振动(如人体运动)需要宽带方案

### 1.3 可用功率密度

单位体积压电材料的理论可采集功率密度：

```
P/V = (d * eps * Y)^2 * a^2 / (4 * omega * rho)

d: 压电常数, eps: 介电常数, Y: 杨氏模量
a: 加速度, omega: 角频率, rho: 密度
```

典型 PZT-5A 悬臂梁在 0.5g/100Hz 条件下功率密度约 0.1-1 mW/cm3，足以支撑低功耗 IoT 节点间歇工作。

## 2. 压电换能器机电建模

### 2.1 机电耦合机制

压电换能器是机电耦合系统：机械端受振动激励产生变形，正压电效应在电端产生电荷；电端负载通过逆压电效应影响机械响应。这种双向耦合是匹配电路设计的核心难点。

一维 d31 模式本构方程：

```
D3 = d31 * T1 + eps_33^T * E3
S1 = s11^E * T1 + d31 * E3

D3: 电位移(C/m2), T1: 轴向应力(Pa)
E3: 电场(V/m), S1: 轴向应变
s11^E: 恒电场弹性柔度, eps_33^T: 恒应力介电常数
```

机电耦合系数 k31 衡量转换效率：PZT-5A k31~0.34, PZT-5H k31~0.39, PVDF k31~0.11。k 值越大，电能与机械能之间的转化比例越高。

### 2.2 Van Dyke 等效电路

将分布参数的压电结构用集总参数电路表示，是匹配电路分析的基础：

```
     Lm    Cm    Rm
 +--[LLLL]--[C]--[RRR]--+
 |                        |
 |        Cp              |
 +--------||--------------+
 +----(压电两端子)-------+

Lm: 等效质量(动质量), Cm: 等效柔度(1/刚度)
Rm: 等效机械阻尼, Cp: 夹持电容(Clamped Capacitance)
```

机械侧与电气侧的类比关系：

| 机械量 | 电气类比 | 关系 |
|--------|----------|------|
| 力 F | 电压 V | F = alpha * V |
| 速度 v | 电流 I | I = alpha * v |
| 质量 m | 电感 Lm | Lm = m / alpha^2 |
| 柔度 1/k | 电容 Cm | Cm = alpha^2 / k |
| 阻尼 c | 电阻 Rm | Rm = alpha^2 / c |

力因子 alpha(N/V 或 A/(m/s))是连接机械侧与电气侧的桥梁参数。

### 2.3 夹持电容与功率极限

Cp 是理解匹配电路的关键参数。压电片开路时，Cp 上积累的电荷随振动周期性充放电。理论最大功率(阻抗匹配条件下)：

```
Pmax = alpha^2 * Vm^2 / (8 * Rm)

Vm: 振动速度幅值, alpha: 力因子
```

Cp 的存在使简单电阻负载无法实现最佳匹配——匹配电路存在的根本原因。

## 3. 谐振频率匹配

### 3.1 谐振条件与阻抗特性

压电悬臂梁在谐振频率处机械振幅最大，输出功率也最大：

```
谐振频率 fs = 1 / (2*pi*sqrt(Lm*Cm))
反谐振频率 fp = fs * sqrt(1 + Cm/Cp)

fs < fp, 间距取决于 keff^2 = Cm/(Cm+Cp)
```

在 fs 处机械支路串联谐振，阻抗最小，电流(对应速度)最大；在 fp 处 Cp 与机械支路并联谐振，阻抗最大，电压最高。

### 3.2 阻抗匹配策略

"阻抗匹配"有两种等价含义：1) 电端匹配——负载阻抗等于压电源内阻的共轭，最大化传输功率; 2) 机电匹配——电端负载映射到机械侧的阻尼等于机械阻尼，实现最优能量提取。

最优负载电阻近似：`Ropt = 1 / (2*pi*fs*Cp*keff^2)`，典型值数百 kOhm。高阻抗特性决定了后续整流和调理电路的设计约束。

### 3.3 调谐方法

| 方法 | 原理 | 调谐范围 | 局限 |
|------|------|----------|------|
| 质量块调节 | 改变端部质量改变谐振频率 | +/-10% | 需停机 |
| 阵列多频 | 多个不同频率采集器 | 离散覆盖 | 面积大 |
| 非线性宽带 | 双稳态/单稳态结构 | 连续覆盖 | 建模复杂 |
| 主动调谐 | 闭环反馈+执行器 | +/-20% | 额外功耗 |

## 4. 标准 AC-DC 整流电路

### 4.1 全桥整流

最基本的匹配电路是全桥整流器(Full-Bridge Rectifier, FBR)：

```
压电片--+--D1-->--+--D3-->--+
        |         |         |
        o        ===Cp     GND
        |         |         |
        +--D2-->--+--D4-->--+
                 | |
                ===Cfilt
                [RLoad]
```

正半周 D1/D3 导通，负半周 D2/D4 导通，Cfilt 滤波后为负载提供近似直流。

### 4.2 整流损耗分析

全桥整流有两个主要损耗来源：

**1) 二极管压降损耗**：每个导通半周两个二极管串联，压电片输出须超过 2*Vd 才能导通。以肖特基二极管 Vd=0.3V 计：

```
Voc = 3V 时: 整流输出上限 = Voc - 2*Vd = 2.4V
损耗比例 = 2*Vd / Voc = 40% (Voc 较低时非常严重)
```

**2) Cp 充放电损耗**：当 |Vp| < Vrect+2Vd 时整流桥截止，Cp 上的电荷通过阻尼释放而非馈入负载。Cp 每个周期有一段时间在"空转"。

### 4.3 全桥整流的功率输出

全桥整流在最优负载下的输出功率(忽略二极管压降)：

```
P_FBR = (alpha^2 * Vm^2) / (2*Rm) * (1 / (1 + pi/(2*keff^2)))

对比 Pmax: P_FBR/Pmax = 4 / (pi*(1 + pi/(2*keff^2)))
keff^2 = 0.09 时: P_FBR/Pmax ~ 18%
```

全桥整流仅能提取理论最大功率的 15%-25%，大量能量因 Cp 充放电损耗而浪费。

## 5. 同步开关采集技术 (SSHI)

### 5.1 核心思想

SSHI(Synchronized Switch Harvesting on Inductive load)的核心创新：在压电电压过零时，用电感与 Cp 形成 LC 振荡，瞬间翻转 Cp 上的电压极性，减少 Cp 充放电的净能量损耗。

类比：钟摆到最高点时推一把，让它更快摆向另一侧——SSHI 就是在电压过零时"推一把"，让 Cp 上的电荷不白白浪费，而是更快翻转方向去给负载充电。

翻转后电压：`V_after = -gamma * V_before + (1+gamma) * Vrect`

gamma 为翻转效率因子(0.5-0.9)，取决于 LC 回路 Q 值：`gamma = exp(-pi/(2*Q))`。

### 5.2 串联 SSHI (Series-SSHI)

电感与压电片串联，翻转电流通过整流桥：

```
压电片--[开关S]--[电感L]--+--D1-->--+
                           |         |
                           |        GND
                           |         |
                           +--D2-->--+
                           | |
                          ===Cfilt
                          [RLoad]
```

### 5.3 并联 SSHI (Parallel-SSHI)

开关-电感支路与 Cp 并联，翻转电流不经过整流桥：

| 对比项 | Series-SSHI | Parallel-SSHI |
|--------|-------------|---------------|
| 电感位置 | 与压电片串联 | 与 Cp 并联 |
| 翻转效率 gamma | 0.5-0.7 | 0.6-0.9 |
| 最优负载范围 | 较窄 | 较宽 |
| 低耦合时增益 | 更显著 | 更显著 |
| 实现复杂度 | 开关时序更严格 | 略简单 |
| 典型功率提升(相对FBR) | 2-4x | 2-5x |

### 5.4 开关时序控制

自驱动方案：比较器检测 Vp 降到 Vrect 的时刻，输出脉冲驱动 MOSFET。电路从压电片取电，无需外部供电。外部控制方案用 MCU 定时器，精度更高但需额外功耗。

### 5.5 功率输出

```
P_SSHI / P_FBR = (1+pi/(2*keff^2)) / (1+pi/(4*keff^2))
keff^2=0.09: ~2.8x,  keff^2=0.04: ~3.5x
```

耦合系数越低，SSHI 增益越大，对 PVDF 等低耦合材料特别有价值。

## 6. 同步电荷提取 (SECE)

### 6.1 原理

SECE(Synchronous Electric Charge Extraction)在电压达到峰值时将 Cp 全部电荷提取到储能电容。类比：等水桶装满后一次性倒空——SECE 不在电压过零时翻转，而是在电压最高时全量提取。

### 6.2 关键特性

**负载无关性**：电荷提取在极短时间内完成(通常 < 1% 周期)，提取能量仅取决于 Cp 电压峰值，与负载阻抗无关。这意味着 SECE 天然实现了最优阻抗匹配——无需调负载电阻。

**功率-耦合关系**：

```
keff^2 < 0.05: SECE 优于 SSHI (4-5x FBR)
keff^2 > 0.10: SSHI 优于 SECE (SSHI 2-3x FBR, SECE 1.5-2x)
```

### 6.3 SECE vs SSHI

| 对比项 | SSHI | SECE |
|--------|------|------|
| 核心机制 | 电压翻转减少损耗 | 峰值时全量提取 |
| 负载匹配 | 需调最优负载 | 负载无关 |
| 电感需求 | 中等(翻转用) | 中等(转移用) |
| 低耦合优势 | 3-4x FBR | 4-5x FBR |
| 高耦合优势 | 2-3x FBR | 1.5-2x FBR |
| 输出电压范围 | 受 Vrect 限制 | 可升压输出 |
| 自驱动实现 | 较容易 | 较困难(峰值检测) |

## 7. 接口电路效率综合对比

### 7.1 归一化功率(P/Pmax)

| keff^2 | FBR | S-SSHI | P-SSHI | SECE |
|--------|-----|--------|--------|------|
| 0.05 | 12% | 35% | 40% | 42% |
| 0.10 | 22% | 55% | 62% | 50% |
| 0.20 | 35% | 72% | 78% | 58% |

### 7.2 实际考量因素

| 因素 | 说明 |
|------|------|
| 电感体积 | SSHI/SECE 需 mH 级电感，对微型化不利 |
| 开关损耗 | 自驱动检测电路静态功耗(通常 1-10 uW) |
| 二极管选择 | 肖特基 vs 有源整流，影响低压启动 |
| 负载变化 | SECE 对负载变化鲁棒，SSHI 需要阻抗跟踪 |
| 启动时间 | SECE 需 Cstore 预充电，启动较慢 |

### 7.3 选型决策树

```
开路电压 > 5V 且 keff^2 > 0.08?
  是 --> 负载变化大? 是-->SECE, 否-->P-SSHI
  否 --> 功率需求 < 50uW? 是-->FBR, 否-->S-SSHI
```

## 8. LTC3588 压电能量收集 IC

### 8.1 芯片概述

LTC3588-1(Analog Devices)集成全桥整流、低损耗采样、降压 DC-DC 和输出稳压，是最成熟的商用方案。

| 参数 | 数值 |
|------|------|
| 输入电压范围 | 2.7V-20V(峰值) |
| 静态电流 | 450 nA(欠压锁定模式) |
| 输出电压选项 | 1.8V/2.5V/3.3V/3.6V |
| 输出电流(3.3V) | 最高 ~100 mA(脉冲) |
| DC-DC 效率 | 最高 85% |
| 封装 | QFN-16 (3mm x 3mm) |

### 8.2 工作原理

两阶段间歇工作：

1. **充电期**：压电片经内部全桥给 Cin 充电，DC-DC 休眠(功耗仅 450nA)
2. **转移期**：Cin 电压升至 Vhigh(5.05V)时 DC-DC 启动，将能量转移至 Cout；Cin 降至 Vlow(3.67V)时休眠

```
Vhigh ─────┐
            │  DC-DC ON (能量转移)
Vlow  ─────┘

每次转移能量: E = 0.5 * Cin * (Vhigh^2 - Vlow^2)
```

### 8.3 局限

采用标准全桥，未集成 SSHI/SECE；输入须达 2.7V 才启动，对微弱振动不友好；充放电窗口固定不自适应。适用于振幅大、频率稳定的工业场景。

## 9. 宽带能量采集技术

### 9.1 窄带困境

传统压电采集器带宽仅谐振频率的 1%-3%(Q=50, fs=100Hz 时 BW=2Hz)，频率漂移 3Hz 功率下降 >70%。环境振动频率往往不稳定，窄带谐振器难以持续高效工作。

### 9.2 非线性振荡器

**双稳态振荡器**：磁铁排斥或弯曲梁预压产生两个稳态位置。小幅振动时在单稳态附近振荡(阱内运动)，振幅足够大时在两稳态间跳跃(阱间运动)，呈现宽带响应。

```
势能函数: U(x) = -a*x^2/2 + b*x^4/4
两稳态: x = +/-sqrt(a/b), 势垒: Delta_U = a^2/(4*b)
```

**单稳态硬化/软化振荡器**：非线性弹簧使谐振峰向高频(硬化)或低频(软化)倾斜，有效带宽增大 2-3 倍。

```
硬化型: F = k*x + k3*x^3 (k3>0), 谐振峰右倾
软化型: F = k*x + k3*x^3 (k3<0), 谐振峰左倾
```

### 9.3 频率上转换

低频振动触发高频谐振器自由振动——像弹吉他，拨一次弦振动多次衰减。三种实现方式：

1. **碰撞式**：低频梁撞击高频梁，高频梁自由振动衰减
2. **磁耦合式**：低频运动磁铁激励高频线圈/压电片
3. **棘轮式**：低频运动驱动棘轮旋转触发高频振子

可采集 <10Hz 的低频振动(人体运动)，但碰撞损耗和寿命是挑战。

### 9.4 技术对比

| 技术 | 带宽提升 | 功率密度 | 复杂度 | 可靠性 |
|------|----------|----------|--------|--------|
| 线性阵列 | 3-5x | 中 | 低 | 高 |
| 双稳态 | 5-10x | 中高 | 中 | 中 |
| 硬化非线性 | 2-3x | 高 | 中 | 高 |
| 频率上转换 | 频段变换 | 高 | 高 | 中低 |
| 主动调谐 | 自适应 | 中 | 高 | 中 |

## 10. 功率输出与振动幅度关系

### 10.1 线性区与小振幅近似

小振幅(<1g)时功率与加速度平方成正比：P = K * a^2，K 为与材料、结构、匹配电路相关的常数。

### 10.2 非线性效应

大振幅(>2g)出现三种非线性：

1. **机械非线性**：大振幅下梁应变超出线性范围，有效刚度变化，谐振频率漂移
2. **电端饱和**：输出电压升高导致逆压电效应增强，形成额外电学阻尼
3. **整流饱和**：全桥整流在高压时二极管非线性加剧

### 10.3 各电路的功率-振幅趋势

SSHI/SECE 在所有振幅下都优于 FBR，但优势幅度随振幅增大递减——大振幅时 Cp 电压更高，FBR 充放电损耗占比相对减小。在极低振幅(<0.1g)场景下，SSHI 的优势可达 5-8 倍。

## 11. 实际部署案例

### 11.1 工业电机监测

某工厂 750kW 电机外壳部署压电自供电温度/振动传感器。振动源为轴承振动(50Hz, 0.5-2g)。PZT-5A 悬臂梁(fs=52Hz, Cp=20nF) + P-SSHI(电感 10mH)，输出 1.8-12mW。驱动 BLE 传感器节点(每 10s 发送，均耗 0.5mW)。储能电容 2g 时 3 分钟充满，可维持 30 分钟无振动传输。

### 11.2 桥梁健康监测

某公路桥跨中安装压电自供电应变监测节点。车辆通过引起桥面振动(2-8Hz, 0.02-0.1g)。PZT-5A 放大式悬臂梁 + 频率上转换结构(Cp=45nF) + S-SSHI + LTC3588 后级稳压，输出 30-200uW。驱动 LoRa 节点(每 5 分钟发送，均耗 20uW)。挑战：低频需大质量块(50g)，频率上转换结构需防尘。

### 11.3 可穿戴步态采集

鞋底嵌入式压电采集器驱动步态监测传感器。步行脚跟着地冲击(1-5Hz, 1-3g)。PVDF 柔性薄膜(Cp=10nF)多片串联 + SECE(负载无关特性适配可变负载)，输出 50-500uW。挑战：PVDF 耦合系数低，人体运动频率和幅度极不稳定，鞋内空间受限。

### 11.4 HVAC 管道监测

商用楼宇空调风管振动驱动温湿度传感器。风机振动传导至风管壁(30-60Hz, 0.1-0.5g)。PZT-5A 贴片式(Cp=15nF, fs=45Hz) + LTC3588 标准方案(振动稳定无需复杂匹配)，输出 0.2-1.5mW。驱动 Zigbee 温湿度节点(每分钟上报)。特点：LTC3588 集成度高，BOM 最简。

## 总结

1. **建模先行**：Van Dyke 等效电路和 Cp 是分析所有匹配电路的基础，理解 k 的物理含义比背诵公式更重要
2. **标准整流局限**：全桥仅提取理论功率 15%-25%，Cp 充放电损耗是瓶颈
3. **SSHI 以翻转破局**：功率提升 2-5 倍，P-SSHI 通常优于 S-SSHI；耦合越低增益越大
4. **SECE 以提取取胜**：天然负载无关匹配，低耦合场景优势更大；高耦合时不如 SSHI
5. **商用方案折衷**：LTC3588 集成度高但效率受限，适用于大振幅稳定振动场景
6. **宽带是未来**：非线性振荡器和频率上转换突破窄带瓶颈，是走向实际部署的关键
7. **系统思维**：匹配电路不是孤立的，必须与压电片参数、振动源特性、负载功耗预算协同设计
