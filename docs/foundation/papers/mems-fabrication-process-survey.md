# MEMS传感器制造工艺：体硅与表面微加工

> **难度**：🔴 高级 | **领域**：MEMS制造工艺 | **阅读时间**：约 25 分钟

## 引言

想象你是一位木雕匠人。你有两种做立体雕刻的方式：一种是从一块厚木料上凿掉不要的部分，留下凸起的形状——这叫"减法"；另一种是在一块平板上一层一层地往上贴薄木片，贴完再把中间某层抽掉，让上层悬空——这叫"加法再拆"。

MEMS制造工艺的核心逻辑与此完全对应：体硅微加工 (Bulk Micromachining) 是"从整块硅往下挖"，表面微加工 (Surface Micromachining) 是"在表面一层一层搭再抽掉牺牲层"。两种思路决定了你能做出什么样的结构、多大的尺寸、多高的良率，以及最终成本落在哪个量级。

本文系统梳理这两种主流工艺路线及其衍生技术，包括湿法/干法刻蚀、DRIE (Bosch工艺)、SOI晶圆、LIGA工艺、晶圆键合等，并通过工艺能力对比表和典型传感器实例，帮助读者建立完整的MEMS制造工艺认知框架。

## 1. 体硅微加工 (Bulk Micromachining)

### 1.1 基本原理

体硅微加工通过选择性地去除硅衬底的大部分材料来形成三维结构。其本质是对整块硅晶圆进行"深度雕刻"，最终结构是硅衬底本身的一部分，而非后期沉积的薄膜。

核心特征：
- 结构深度大 (数十至数百微米)
- 结构材料即为衬底硅，力学性能优异
- 工艺步骤相对简单，适合早期MEMS产品

典型应用场景：压力传感器膜片、微通道流体器件、红外测辐射热计的隔热结构。

### 1.2 湿法刻蚀 (Wet Etching)

湿法刻蚀是将晶圆浸入化学溶液，利用化学反应溶解硅。根据刻蚀方向性与晶面的关系，分为各向同性和各向异性两大类。

**各向同性湿法刻蚀**

常用腐蚀液：HNA体系 (HF + HNO3 + CH3COOH)

特点：
- 刻蚀速率在所有方向近似相同
- 速率受组分配比和温度强烈影响
- 典型速率范围：1-80 um/min
- 无法形成尖锐的几何棱角，侧壁呈圆弧形
- 适用于大致去除大量硅材料，不适合精密结构

**各向异性湿法刻蚀**

常用腐蚀液及参数：

| 腐蚀液 | 刻蚀速率 (um/min) | (100)/(111) 选择比 | 温度 (deg C) | 氮化硅掩膜刻蚀速率 (nm/min) |
|--------|-------------------|-------------------|-------------|---------------------------|
| KOH | 0.5-2.0 | 200-600:1 | 60-90 | < 0.01 |
| TMAH (25%) | 0.3-1.0 | 30-60:1 | 70-90 | < 0.01 |
| EDP (乙二胺邻苯二酚) | 0.5-1.5 | 35-100:1 | 80-115 | 极低 |

关键概念：硅晶体的 {111} 晶面原子密度最高、键合最强，因此刻蚀速率最慢。在 (100) 硅片上进行各向异性刻蚀时，侧壁自然形成 54.74 度的倾角——这是 (100) 面与 (111) 面之间的几何夹角。

实际工艺要点：
- KOH对CMOS工艺中的铝引线有强腐蚀性，与IC集成时需严格保护
- TMAH与CMOS兼容性更好，且对硅-氧化硅选择比高
- EDP有毒且致癌，实验室使用受限，工业上已大量被TMAH替代
- 角补偿：凸角处需要设计补偿图形，防止 {111} 面过刻蚀导致结构变形

### 1.3 干法刻蚀 (Dry Etching)

干法刻蚀利用等离子体中的活性离子或自由基与硅反应，生成挥发性产物被抽走。

**反应离子刻蚀 (RIE)**

典型工艺气体：SF6 / C4F8 / CHF3 / O2

RIE特点：
- 兼具化学刻蚀 (各向同性) 和物理溅射 (各向异性)
- 通过调节气体配比和偏压实现刻蚀形貌控制
- 典型刻蚀速率：0.5-3 um/min
- 侧壁角度可控，但深宽比受限 (约 5:1)

**深反应离子刻蚀 (DRIE) 与 Bosch 工艺**

DRIE是目前体硅微加工中最关键的高深宽比刻蚀技术。Bosch工艺通过交替进行刻蚀和钝化两个步骤，实现近乎垂直的侧壁：

```
Bosch 工艺循环 (每个周期约 5-10 秒):
1. 刻蚀阶段: 通入 SF6, 等离子体产生 F* 自由基
   - F* 与 Si 反应: Si + 4F* -> SiF4 (挥发)
   - 刻蚀深度: 约 1-2 um/周期
   - 侧壁也受刻蚀 (各向同性分量)

2. 钝化阶段: 通入 C4F8, 等离子体产生 CFx 自由基
   - CFx 在所有表面沉积氟碳聚合物 (~10 nm)
   - 形成保护层防止侧壁继续被刻蚀

3. 下一刻蚀阶段: SF6 等离子体的离子轰击
   - 离子垂直轰击底部, 去除底部钝化层
   - 底部硅重新暴露, F* 继续纵向刻蚀
   - 侧壁钝化层因无离子轰击而保留

结果: 纵向刻蚀持续推进, 侧壁被保护
```

Bosch工艺关键参数与性能：

| 参数 | 典型值 | 影响 |
|------|--------|------|
| 刻蚀速率 | 5-30 um/min | 由RF功率和气体流量决定 |
| 侧壁垂直度 | 89-90 度 (+/-0.5 度) | 刻蚀/钝化时间比是关键 |
| 深宽比 | 最高 30:1-50:1 | 随深度增加刻蚀速率下降 (ARDE效应) |
| 侧壁粗糙度 | 100-500 nm (扇形纹) | Bosch工艺固有特征,可后处理改善 |
| 单晶硅/氧化硅选择比 | > 100:1 | 氧化硅掩膜消耗极慢 |

扇形纹 (Scalloping) 是Bosch工艺最显著的侧壁特征，由刻蚀-钝化交替产生。每个周期在侧壁留下约100-200 nm的波纹。对光学器件和精密流体通道，扇形纹可能影响性能，需要通过优化工艺参数 (缩短周期、降低刻蚀分量) 或使用非Bosch的低温DRIE来改善。

### 1.4 SOI晶圆在体硅微加工中的应用

SOI (Silicon-On-Insulator) 晶圆由三层构成：器件硅层 / 埋氧层 (BOX) / 衬底硅层。

SOI晶圆为体硅微加工带来的核心优势：
- 埋氧层作为天然刻蚀停止层，深度控制精度可达 +/-0.5 um
- 器件层厚度均匀 (典型值 5-100 um)，直接定义结构厚度
- 避免了湿法刻蚀中由晶面决定的几何约束 (54.74度限制)
- 绝缘隔离彻底消除寄生效应，适合高频/高精度器件

SOI晶圆类型与选择：

| SOI类型 | 器件层厚度 | BOX层厚度 | 制造方式 | 适用场景 |
|---------|-----------|----------|----------|----------|
| 薄膜SOI | < 5 um | 0.1-1 um | SIMOX/Smart-Cut | 集成电路, 逻辑器件 |
| 标准SOI | 5-30 um | 0.5-5 um | Smart-Cut/Bonded | 加速度计, 陀螺仪 |
| 厚膜SOI | 30-200+ um | 1-5 um | Bonded | 压力传感器, 微镜阵列 |

工艺示例：基于SOI的电容式加速度计
1. DRIE从正面刻蚀器件层，形成质量块和梳齿结构
2. BOX层自然隔离各结构单元
3. 背面DRIE释放质量块下方的衬底硅，形成悬空质量块
4. BOX层作为刻蚀停止层和机械支撑

## 2. 表面微加工 (Surface Micromachining)

### 2.1 基本原理

表面微加工的核心思路：在衬底表面逐层沉积结构层和牺牲层，最后选择性地去除牺牲层 (释放)，使结构层悬空形成可动结构。

与体硅微加工的关键区别：
- 结构层是沉积的薄膜 (通常1-5 um)，而非衬底本身
- 垂直方向尺寸小，但平面内设计自由度高
- 可制作复杂的多层互锁结构
- 工艺与CMOS后端工艺兼容性更好

### 2.2 牺牲层技术

牺牲层 (Sacrificial Layer) 是表面微加工中最核心的概念——它是一个临时存在的材料层，在结构完成后被去除，留下悬空的可动结构。

**常见牺牲层/结构层材料组合：**

| 牺牲层 | 结构层 | 牺牲层去除液 | 结构层损伤风险 | 典型应用 |
|--------|--------|-------------|--------------|----------|
| PSG (磷硅玻璃) | 多晶硅 | HF (49%) | 低 (poly-Si耐HF) | 加速度计, 陀螺仪 |
| 氧化硅 (SiO2) | 氮化硅 (Si3N4) | HF | 低 | 谐振器, 滤波器 |
| 光刻胶 (PR) | 金属 (Al/Au) | 丙酮/O2等离子体 | 低 | 射频开关 |
| 聚酰亚胺 | 金属/氮化硅 | O2等离子体 | 需评估 | 微镜, 阀门 |
| 非晶硅 | 氮化硅 | XeF2气相刻蚀 | 极低 | 谐振器 |
| Cu (铜) | 聚合物/氮化物 | FeCl3或NH4OH/H2O2 | 中等 | RF MEMS |

**释放工艺的关键挑战：粘附 (Stiction)**

粘附是表面微加工中最普遍且最致命的失效模式。当牺牲层被去除后，结构在干燥过程中，毛细力将悬空结构拉向衬底，一旦接触，范德华力和氢键合力可能使其永久粘住。

防粘附策略：

1. **临界点干燥 (CPD)**
   - 用CO2在超临界状态下替换结构内的液体，避免气液界面产生毛细力
   - 设备成本中等，实验室常用
   - 步骤：脱水 -> 乙醇替换 -> CO2替换 -> 升温至超临界 -> 缓慢排气

2. **疏水化表面处理**
   - 自组装单分子层 (SAM)：如OTS (十八烷基三氯硅烷)、FDTS
   - 在硅表面形成疏水涂层，接触角 > 100 度
   - 同时降低静摩擦系数，减少工作过程中的粘附

3. **结构设计防粘附**
   - 增加支撑点密度，减小悬臂长度
   - 在接触面上设计凸点 (dimple)，减少接触面积
   - 刚度设计：确保弹性恢复力 > 粘附力

4. **干法释放**
   - 使用气相刻蚀剂 (如XeF2刻蚀硅牺牲层、HF蒸汽刻蚀氧化硅)
   - 全程无液相，从根本上消除毛细力
   - XeF2对硅/氧化硅选择比 > 1000:1，但刻蚀各向同性，控制难度大

### 2.3 多层表面微加工工艺流程

以经典的三层多晶硅工艺 (如Sandia SUMMiT工艺) 为例：

```
SUMMiT-V 工艺流程 (5层多晶硅, 4层牺牲层):
1. 衬底准备: 硅片热氧化 -> 氮化硅绝缘层
2. 牺牲层1: 沉积PSG (2 um) -> 光刻/刻蚀 -> 锚点开孔
3. 结构层1 (Poly0): 沉积多晶硅 (0.3 um) -> 掺杂 -> 光刻/刻蚀 -> 地平面/电极
4. 牺牲层2: 沉积PSG (2 um) -> 光刻/刻蚀 -> 锚点到Poly0的开孔
5. 结构层2 (Poly1): 沉积多晶硅 (2 um) -> 掺杂 -> 光刻/刻蚀 -> 可动结构下层
6. 牺牲层3: 沉积PSG (1.5 um) -> 光刻/刻蚀
7. 结构层3 (Poly2): 沉积多晶硅 (1.5 um) -> 掺杂 -> 光刻/刻蚀 -> 可动结构上层
8. 牺牲层4: 沉积PSG (1.5 um) -> 光刻/刻蚀
9. 结构层4 (Poly3): 沉积多晶硅 (2 um) -> 掺杂 -> 光刻/刻蚀 -> 外壳/互连
10. 释放: HF浸泡去除所有PSG牺牲层 -> 临界点干燥
```

多层工艺的工程难点：
- 层间对准精度随层数累积 (典型 +/-0.25 um/层)
- 牺牲层去除时间随厚度增加，可达数十分钟至数小时
- 大面积悬空结构的释放均匀性难以保证
- 残余应力梯度导致结构弯曲 (多晶硅沉积条件的函数)

### 2.4 残余应力与应力梯度

薄膜沉积过程中不可避免地引入残余应力，是表面微加工中最棘手的工艺问题之一。

残余应力来源：
- 热膨胀系数失配 (薄膜与衬底不同材料)
- 沉积过程中原子的非平衡排列
- 掺杂引起的晶格畸变
- 多晶硅退火条件对内应力的强烈影响

应力梯度导致的效果：
- 均匀压应力 -> 欧拉屈曲 (Euler Buckling)，结构在平面内弯折
- 均匀拉应力 -> 结构刚度增加，谐振频率偏移
- 应力梯度 (沿厚度方向应力变化) -> 悬臂结构翘曲 (Curling)

工艺控制手段：
- 多晶硅高温退火 (> 1000 deg C) 消除压应力
- PSG回流退火改善层间平坦化
- 快速热退火 (RTA) 减少热预算的同时释放应力
- 通过离子注入精确调控应力大小甚至符号

## 3. 体硅与表面微加工的系统性对比

### 3.1 工艺能力对比

| 对比维度 | 体硅微加工 | 表面微加工 |
|---------|-----------|-----------|
| 结构厚度 | 50-500 um | 1-10 um (典型) |
| 侧壁形貌 | 取决于刻蚀方式; 湿法受晶面约束, DRIE近垂直 | 由薄膜沉积和刻蚀定义, 侧壁垂直度好 |
| 深宽比 | DRIE可达30:1-50:1 | 通常 < 5:1 |
| 平面内尺寸 | 受晶面限制 (各向异性湿法) | 自由设计 |
| 结构材料 | 单晶硅 (力学性能最优) | 多晶硅/氮化硅 (有残余应力) |
| 传感器质量块 | 大 (利于惯性传感器灵敏度) | 小 (限制了加速度计和陀螺仪性能) |
| 密封腔体 | 需晶圆键合实现 | 可原位制作 (外壳层+密封) |
| IC兼容性 | 较差 (大刻蚀量、与CMOS热预算冲突) | 较好 (低温沉积、后CMOS工艺可行) |
| 典型工艺层数 | 1-3层 (结构简单) | 3-7层 (多层互锁) |
| 主要失效模式 | 过刻蚀、侧壁粗糙度 | 粘附 (Stiction)、残余应力变形 |
| 封装前可靠性 | 高 (单晶硅结构) | 中 (受粘附和应力影响) |

### 3.2 典型传感器结构与工艺映射

| 传感器类型 | 推荐工艺 | 关键结构特征 | 代表产品/工艺 |
|-----------|---------|-------------|-------------|
| 压阻式压力传感器 | 体硅湿法刻蚀 | 5-50 um薄膜 + 54.74度V型槽 | Bosch BMA系列, Freescale MPX |
| 电容式加速度计 | 表面微加工 | 梳齿驱动结构 + 悬臂梁 | Analog Devices ADXL系列 |
| 陀螺仪 | 体硅DRIE + SOI | 深槽梳齿 + 大质量块 | InvenSense ITG系列 |
| MEMS麦克风 | 体硅DRIE | 穿孔背板 + 薄膜 | Knowles SPH系列 |
| 微镜阵列 | 表面微加工 | 多层铰链 + 平板反射面 | TI DLP (基于SUMMiT) |
| 谐振器 | 表面微加工/体硅 | 高Q值梁/盘结构 | SiTime SiMEMS谐振器 |
| 微流体通道 | 体硅湿法/干法 | 封闭管道网络 | BioMEMS LOC器件 |
| 红外测辐射热计 | 体硅湿法刻蚀 | 超薄隔热桥结构 | FLIR Lepton核心 |

## 4. LIGA工艺

### 4.1 原理与流程

LIGA是德语Lithographie (光刻)、Galvanoformung (电铸)、Abformung (模铸) 的缩写，是一种高深宽比微结构制造技术，材料范围远超硅基工艺。

LIGA工艺流程：

```
1. X射线光刻:
   - 同步辐射X射线光源 (波长0.1-1 nm)
   - PMMA厚光刻胶 (100 um - 3 mm)
   - X射线掩膜版 (金吸收体 + 铍/硅支撑膜)
   - 曝光后显影获得高深宽比胶模

2. 电铸 (Galvanoformung):
   - 在胶模中电铸金属 (Ni, Cu, Au等)
   - 电铸完成后去除残余胶
   - 得到金属模具 (初级模具)

3. 模铸 (Abformung):
   - 用金属模具批量复制塑料/陶瓷零件
   - 注塑、热压或浇铸
   - 单个模具可生产数万至数十万件
```

### 4.2 LIGA与硅基工艺对比

| 维度 | LIGA | 硅基体/表面微加工 |
|------|------|-----------------|
| 深宽比 | 最高100:1 (远超DRIE) | DRIE最高约50:1 |
| 结构厚度 | 100 um - 3 mm | 体硅可达500 um; 表面通常 < 10 um |
| 材料选择 | 金属 (Ni, Cu, Au)、聚合物、陶瓷 | 硅、多晶硅、氮化硅、氧化硅 |
| 侧壁粗糙度 | 极低 (X射线光刻精度) | DRIE有扇形纹; 湿法受晶面限制 |
| 设备成本 | 极高 (需同步辐射光源) | 中等 (标准半导体设备) |
| 批量生产 | 模铸成本低 (一旦模具制成) | 晶圆级批量生产 |
| IC集成 | 不兼容 | 可集成 (尤其表面微加工) |
| 典型应用 | 微电机、微泵、光纤连接器 | 传感器、谐振器、执行器 |

UV-LIGA是LIGA的低成本变体，使用紫外光替代X射线，设备成本大幅降低，但深宽比受限 (约10:1)，适用于精度要求不太极端的场景。

## 5. 晶圆键合 (Wafer Bonding)

### 5.1 键合技术分类

晶圆键合是将两片或多片晶圆永久结合的技术，在MEMS中用于形成密封腔体、保护可动结构、制作三维复合结构。

| 键合方式 | 温度 (deg C) | 对准精度 | 是否需要中间层 | 键合强度 | 典型应用 |
|---------|-------------|---------|-------------|---------|---------|
| 阳极键合 | 300-450 | 1-5 um | 否 (玻璃-硅直接) | 高 | 压力传感器封装 |
| 直接键合 (Fusion) | 800-1100 (或室温) | 0.5-2 um | 否 | 极高 | SOI制备, 光学器件 |
| 共晶键合 | 200-400 | 1-3 um | 是 (Au-Si等) | 高 | 真空封装, 3D集成 |
| 玻璃料键合 | 400-500 | 5-20 um | 是 (玻璃焊料) | 中 | MEMS密封腔体 |
| 聚合物键合 | 100-250 | 2-10 um | 是 (BCB, SU-8等) | 中低 | 微流体, BioMEMS |
| 金属扩散键合 | 300-450 | 1-3 um | 是 (Cu-Cu, Au-Au) | 高 | TSV, 3D IC |

### 5.2 阳极键合详解

阳极键合 (Anodic Bonding) 是MEMS中最常用的键合方式之一，特别适用于硅-玻璃封装。

原理：
1. 硅片与含钠玻璃 (如Pyrex 7740) 对准贴合
2. 加热至300-450 deg C (玻璃中Na+离子活化)
3. 施加直流电压 (500-1000 V，硅接正极)
4. Na+向阴极迁移，在硅-玻璃界面留下空间电荷区
5. 空间电荷区产生强电场，将两表面紧密拉合
6. 界面发生化学反应形成Si-O-Si共价键

关键参数控制：
- 温度和电压的乘积决定键合质量
- 表面粗糙度需 < 10 nm (Ra)
- 翘曲和颗粒会导致局部未键合 (空洞)
- 热膨胀系数匹配至关重要 (Pyrex 7740与硅匹配)

### 5.3 键合在MEMS封装中的作用

晶圆级封装 (Wafer-Level Packaging, WLP) 是MEMS制造的关键环节：

- **气密封装**：保护可动结构免受颗粒和湿气侵害
- **真空封装**：为谐振器和陀螺仪提供高Q值工作环境
- **应力隔离**：减少封装应力对传感器性能的影响
- **预封装测试**：键合前可测试，良率损失降低

## 6. 工艺选择决策框架

### 6.1 决策流程

选择MEMS制造工艺时，需综合考虑以下因素：

```
Step 1: 确定结构需求
  |- 需要大质量块? -> 体硅 (DRIE/SOI)
  |- 需要复杂多层互锁? -> 表面微加工
  |- 需要非硅材料 (金属/聚合物)? -> LIGA/UV-LIGA

Step 2: 评估性能约束
  |- 高深宽比 (> 20:1)? -> DRIE或LIGA
  |- 精密侧壁? -> LIGA或低温DRIE
  |- 低残余应力? -> 体硅 (单晶硅无应力问题)

Step 3: 制造可行性
  |- 是否需要IC集成? -> 表面微加工优先
  |- CMOS后端兼容? -> 低温表面微加工
  |- 封装需求? -> 阳极键合/共晶键合

Step 4: 成本与量产
  |- 小批量/原型? -> 体硅湿法刻蚀 (设备简单)
  |- 大批量消费电子? -> 表面微加工或SOI+DRIE
  |- 超大批量非硅零件? -> LIGA (模铸)
```

### 6.2 良率与成本考量

| 因素 | 体硅微加工 | 表面微加工 |
|------|-----------|-----------|
| 关键良率限制 | DRIE均匀性、过刻蚀 | 粘附、残余应力变形 |
| 光刻层数 | 少 (3-6层) | 多 (8-15层) |
| 每层对准误差累积 | 较小 | 较大 |
| 晶圆利用率 | 中 (结构间距受刻蚀约束) | 高 (结构紧凑) |
| 热预算 | 湿法低; DRIE中等 | 退火需高温 (> 1000 deg C) |
| 典型MPW成本 | 5-15K USD/次 | 10-30K USD/次 |
| 量产单颗成本 (消费级) | 0.05-0.30 USD | 0.03-0.20 USD |

成本驱动因素分析：
- 晶圆尺寸：8寸线单颗成本约为6寸线的40-50%
- 光刻次数：每增加一次光刻，成本增加10-15%
- DRIE时间：深结构刻蚀时间可达数小时，是主要瓶颈
- 封装占比：MEMS封装成本通常占器件总成本的40-70%
- 测试成本：MEMS传感器需要机械激励测试，比纯IC测试复杂得多

## 7. 工艺发展趋势

### 7.1 先进刻蚀技术

- **原子层刻蚀 (ALE)**：原子级精度控制刻蚀深度，对标ALD的沉积精度
- **低温DRIE**：-100 deg C以下刻蚀，侧壁无扇形纹，适合光学器件
- **混合刻蚀**：先DRIE快速刻蚀，再湿法修整侧壁粗糙度

### 7.2 异构集成

- **3D集成**：TSV + 晶圆键合实现传感器与ASIC三维堆叠
- **CMOS-MEMS单片集成**：MEMS后处理直接在CMOS晶圆上完成
- **Chiplet架构**：MEMS chiplet + 信号处理chiplet + 通信chiplet

### 7.3 新材料与新结构

- **压电薄膜 (AlN, PZN-PT)**：替代静电驱动，降低驱动电压
- **2D材料传感器**：石墨烯、MoS2等用于超高灵敏度检测
- **3D打印微结构**：双光子聚合实现亚微米分辨率3D结构
- **柔性衬底MEMS**：聚酰亚胺/PDMS上的可穿戴传感器

## 总结

MEMS制造工艺的核心分野在于体硅微加工与表面微加工两种基本思路：

1. **体硅微加工**适合需要大质量块、高深宽比、优异力学性能的场景，DRIE和SOI晶圆是其中的关键技术。其主要局限在于工艺与CMOS集成困难、湿法刻蚀受晶面几何约束。

2. **表面微加工**适合复杂多层结构、与IC集成要求高的场景，牺牲层技术是其核心。粘附和残余应力是两大主要失效模式，临界点干燥、疏水化处理和干法释放是关键的防粘附手段。

3. **LIGA工艺**在高深宽比和非硅材料方面不可替代，但同步辐射光源的高成本限制了其应用范围。

4. **晶圆键合**是MEMS封装和三维结构制造的关键使能技术，阳极键合和共晶键合是最常用的两种方式。

5. 工艺选择没有"最优解"，只有"最适解"——需根据结构需求、性能约束、制造可行性和成本预算综合决策。

理解这些工艺的原理、能力和限制，是设计高质量MEMS传感器的前提。在实际工程中，往往需要组合使用多种工艺，例如体硅DRIE加工质量块 + 表面微加工制作梳齿 + 阳极键合真空封装，才能实现高性能MEMS器件的完整制造。
