# IoT硬件环境试验：温度/湿度/振动/盐雾
> **难度**：🟡 中级 | **领域**：环境可靠性测试 | **阅读时间**：约 20 分钟

## 引言

想象你把一部手机放在赤道沙漠的正午太阳下暴晒, 再扔进撒哈拉的沙尘暴里, 最后丢到海边让海风吹一周 -- 手机还能正常工作吗? 大部分消费电子产品扛不住, 但很多IoT设备恰恰就部署在这样的恶劣环境中: 户外气象站、工业车间传感器、车载追踪器、海洋浮标。环境试验就是在实验室里模拟这些极端条件, 在产品出厂前暴露设计缺陷, 确保设备在真实部署场景中长期可靠运行。

## 1 为什么需要环境试验

### 1.1 IoT部署环境的特殊性

与传统消费电子不同, IoT设备往往部署在无人值守的恶劣环境中:

- **户外**: 日晒雨淋、四季温差、紫外线辐射
- **工业**: 高温、粉尘、腐蚀性气体、机械振动
- **车载**: 温度循环剧烈、振动冲击、潮湿
- **海洋/沿海**: 盐雾腐蚀、高湿度、台风
- **地下/管道**: 潮湿、腐蚀、积水

这些环境下, 电子元件加速老化、密封失效、接触腐蚀, 没有经过环境试验验证的产品, 在现场出问题的概率极高。

### 1.2 环境试验的目的

- **验证设计裕度**: 产品在规格条件外还有多少余量
- **暴露薄弱环节**: 在实验室中找到现场才会出现的问题
- **提供改进依据**: 通过失效分析指导设计优化
- **满足认证要求**: 部分行业认证(汽车、军工)要求通过特定环境试验

### 1.3 试验类型概览

| 试验类型 | 模拟环境 | 典型标准 |
|----------|----------|----------|
| 温度试验 | 高低温、温度循环 | IEC 60068-2-1/2/14 |
| 湿度试验 | 潮湿、凝露 | IEC 60068-2-3/30/78 |
| 振动试验 | 运输、运行振动 | IEC 60068-2-6/27/64 |
| 盐雾试验 | 海洋/沿海腐蚀 | IEC 60068-2-11/52 |
| 防尘防水 | 沙尘、浸水 | IEC 60529 (IP等级) |
| 低气压 | 高海拔 | IEC 60068-2-13 |
| 太阳辐射 | 紫外线老化 | IEC 60068-2-5/9 |

## 2 温度试验

### 2.1 工作温度范围验证

验证设备在规格温度范围内能正常工作:

- 低温工作: 将设备降温至规格下限(如-40 C), 保持至温度稳定后, 进行功能测试
- 高温工作: 将设备升温至规格上限(如+85 C), 保持后进行功能测试
- 典型持续条件: 每个温度点保持2-4小时, 或按标准规定

IoT设备常见温度等级:

| 等级 | 工作温度范围 | 典型应用 |
|------|-------------|----------|
| 商业级 | 0 C 到 +70 C | 室内IoT |
| 工业级 | -40 C 到 +85 C | 工厂、户外 |
| 汽车级 | -40 C 到 +125 C | 发动机舱附近 |
| 军品级 | -55 C 到 +125 C | 特殊应用 |

### 2.2 温度循环 (IEC 60068-2-14)

模拟昼夜温差和季节变化的累积效应:

```
温度剖面:
  +85 C ─────┐         ┌─────────┐
              │         │         │
  -40 C ─────┘         └─────────┘
       保持   升温  保持  降温  保持
       2h    5C/min 2h   5C/min 2h
```

关键参数:
- 高低温极值: 根据部署环境确定
- 温度变化速率: 1-5 C/min(常规), 10-15 C/min(加速)
- 循环次数: 100-1000次(取决于寿命要求)
- 保持时间: 1-4小时

温度循环的典型失效:

- 焊点热疲劳裂纹(不同CTE材料的膨胀/收缩差异)
- 密封失效(O型圈、灌封胶老化)
- PCB过孔断裂(镀铜与基材CTE不匹配)
- 连接器接触不良(金属与塑料CTE差异)

### 2.3 温度冲击

比温度循环更严酷, 温变速率极快:

- 两个温箱(高温箱和低温箱), 样品在两者之间快速转移
- 转移时间 < 10秒
- 温差可达100度以上
- 循环次数通常少于温度循环: 50-200次

主要用于验证焊接、粘接、密封在极端温变下的可靠性。

### 2.4 高低温储存

验证设备在非工作状态下经过极端温度后仍能正常工作: 高温储存+85/+125 C (48-168h), 低温储存-40/-55 C (48-168h), 储存后恢复至室温进行功能测试。关注点: 电池低温储存后容量是否恢复, 电解电容高温储存后ESR是否增大。

## 3 湿度试验

### 3.1 恒定湿热 (Steady-State)

最基础的湿度试验, 也称85/85测试:

- 条件: 85 C / 85% RH
- 持续时间: 240小时(10天)至2000小时(83天)
- 标准: IEC 60068-2-78

85/85是半导体行业标准加速条件, 用于评估: 塑封IC潮湿敏感性(吸湿分层)、PCB绝缘电阻下降(表面凝露吸湿)、连接器接触腐蚀、显示屏起雾脱胶。

### 3.2 循环湿热 (Cyclic)

模拟昼夜湿度变化, 更接近真实环境: 高温高湿(85% RH, 12h)与低温低湿(25% RH, 12h)交替循环。循环湿热的特点: 降温阶段产生凝露比恒定湿热更严酷, 凝露在PCB表面形成水膜导致绝缘下降甚至短路, 适合评估户外设备的抗潮湿能力。

### 3.3 凝露对电子设备的影响

| 现象 | 机制 | 后果 |
|------|------|------|
| 绝缘下降 | 水膜降低表面绝缘电阻 | 漏电流增大、误触发 |
| 电化学迁移 | 水膜 + 偏压下金属离子迁移 | 枝晶生长导致短路 |
| 腐蚀 | 水膜 + 污染物(手汗残留) | 走线开路、焊点失效 |
| 参数漂移 | 元件吸湿(尤其MLCC) | 容值变化、Q值下降 |

防护措施: 三防漆(Conformal Coating)、密封外壳、吸湿剂。

## 4 振动试验

### 4.1 随机振动

模拟运输和运行中的随机振动环境:

- 频率范围: 10-2000 Hz(常用)
- 加速度谱密度(ASD): 根据应用确定
- 总RMS加速度: 典型0.5-5 Grms
- 持续时间: 每轴10分钟到2小时

随机振动剖面示例(车载IoT):

```
频率(Hz)    ASD(g^2/Hz)
10          0.01
50          0.02
200         0.01
500         0.005
2000        0.001

总RMS: 约 2.0 Grms
```

### 4.2 正弦扫频振动

用于寻找共振频率:

- 频率扫描: 5-500 Hz 或 10-2000 Hz
- 扫频速率: 1 oct/min (对数扫频)
- 加速度: 0.5-5 g
- 方向: X/Y/Z三轴

扫频试验中发现共振点后, 可在共振频率上做定频耐久试验:

- 在共振频率上持续振动10-30分钟
- 验证设备结构不会因共振而破坏

### 4.3 机械冲击

模拟跌落、碰撞等瞬间冲击:

- 波形: 半正弦波最常用
- 峰值加速度: 15-100 g
- 脉冲宽度: 6-20 ms
- 次数: 每方向3次

| 应用场景 | 峰值加速度 | 脉冲宽度 |
|----------|-----------|----------|
| 日常搬运 | 15-30 g | 11 ms |
| 车载安装 | 30-50 g | 6-11 ms |
| 军工/航天 | 50-100 g | 6 ms |

### 4.4 运输模拟

模拟产品包装后在运输过程中的振动:

- 标准: ASTM D4169, ISTA系列
- 包含随机振动 + 冲击 + 跌落组合
- 目的: 验证包装防护是否足够, 产品是否在运输中损坏

振动试验的典型失效:

- 螺丝松动(需用螺纹胶或弹垫)
- BGA焊点断裂(需底部填充)
- 连接器脱落(需锁扣机构)
- 电池连接松动(需焊接或弹片)

## 5 盐雾试验

### 5.1 试验方法 (IEC 60068-2-11)

模拟海洋/沿海环境的盐雾腐蚀:

- 溶液: 5% NaCl水溶液(pH 6.5-7.2)
- 温度: 35 C
- 喷雾方式: 连续喷雾
- 试验时间: 24/48/96/240/480/960小时
- 盐雾沉降量: 1-2 ml/h / 80 cm^2

### 5.2 加速腐蚀与实际环境的对应

盐雾试验是加速试验, 试验时间与实际服役时间的对应关系:

| 试验时间 | 大致等效实际环境 |
|----------|------------------|
| 24小时 | 温和室内环境数月 |
| 96小时 | 轻度海洋环境1-2年 |
| 240小时 | 海洋环境2-5年 |
| 960小时 | 严酷海洋环境5-10年 |

注意: 这种对应关系是粗略估计, 实际腐蚀受温度、湿度、污染物等多种因素影响。

### 5.3 盐雾试验的评估

| 评估项目 | 方法 | 判据 |
|----------|------|------|
| 外观 | 目视检查 | 无明显腐蚀、起泡、脱落 |
| 功能 | 电性能测试 | 功能正常 |
| PCB走线 | 微电阻测量 | 阻值变化 < 规定值 |
| 连接器 | 接触电阻 | 阻值增量 < 规定值 |
| 外壳 | 机械检查 | 无结构损伤 |

### 5.4 盐雾防护措施

| 措施 | 适用场景 | 效果 |
|------|----------|------|
| 三防漆 | PCB表面 | 防止走线腐蚀, 但连接器需遮蔽 |
| 灌封 | 整个PCB | 完全密封, 但散热差、维修难 |
| 密封外壳 | 整机 | IP67以上, 配合密封圈 |
| 防腐镀层 | 外壳金属件 | 阳极氧化、达克罗、锌镍合金 |
| 耐盐雾材料选择 | 结构件 | 不锈钢316L代替304, 塑料代替金属 |

## 6 其他环境试验

### 6.1 防尘防水 (IP等级)

IEC 60529定义的防护等级:

```
IP 6 7
   |  |
   |  +-- 第二位: 防水等级 (0-9K)
   +-- 第一位: 防尘等级 (0-6)
```

常用等级:

| IP等级 | 防尘 | 防水 | 典型应用 |
|--------|------|------|----------|
| IP54 | 防尘(不完全) | 防溅水 | 室内工业 |
| IP65 | 防尘 | 防喷水 | 户外设备 |
| IP67 | 防尘 | 短时浸水 | 戶外/车载 |
| IP68 | 防尘 | 持续浸水 | 水下设备 |

### 6.2 太阳辐射

模拟户外紫外线老化: UV-A(315-400 nm)导致塑料变黄变脆, UV-B(280-315 nm)加速老化用于加速试验。标准参考IEC 60068-2-5, 关注外壳塑料老化开裂、LCD偏光片退化、密封胶老化。

### 6.3 低气压与霉菌

- **低气压**: 模拟高海拔(3000-15000 m), 空气绝缘强度下降需增加爬电距离, 散热效率因空气稀薄而下降
- **霉菌**: 28-30 C / 95% RH以上, 28天试验, 霉菌在PCB表面生长导致绝缘下降

## 7 试验标准体系

主要标准: IEC 60068系列(通用电子电气设备), MIL-STD-810H(军工), AEC-Q100/Q200(汽车电子), ISO 16750(道路车辆), GB/T 2423系列(中国国标, 等同IEC 60068)。

IEC 60068-2常用分标准: -1低温, -2高温, -6正弦振动, -11盐雾, -14温度变化, -27冲击, -30循环湿热, -64宽频随机振动, -78恒定湿热。

## 8 试验规划

### 8.1 定义环境剖面

试验规划的第一步是根据部署场景定义环境剖面:

```
示例: 户外气象站(华东沿海)

温度: -20 C (冬季) 到 +60 C (夏季暴晒)
湿度: 20% RH 到 98% RH
振动: 安装在2m杆上, 风致振动 < 0.5 Grms
盐雾: 距海岸5 km内, 中等盐雾
紫外线: 全年户外暴露
雨水: IP65防护即可
```

### 8.2 选择试验等级

根据环境剖面选择合适的试验严酷度:

| 环境应力 | 轻度 | 中度 | 严酷 |
|----------|------|------|------|
| 低温 | -10 C | -25 C | -40 C |
| 高温 | +55 C | +70 C | +85 C |
| 温循次数 | 50 | 200 | 500 |
| 湿热(85/85) | 240h | 500h | 1000h |
| 随机振动 | 0.5 Grms | 2 Grms | 5 Grms |
| 盐雾 | 48h | 96h | 240h |

### 8.3 试验顺序

试验顺序影响结果, 一般原则:

1. 先低温后高温(低温可能产生微裂纹, 高温可加速其发展)
2. 先温度循环后湿热(温循产生微裂纹, 湿热让水汽侵入)
3. 先振动后温度(振动可能松动连接, 温度加速失效)
4. 盐雾放最后(腐蚀性试验不可逆)

## 9 判定准则

### 9.1 功能测试

试验过程中和试验后进行功能测试:

- **试验中**: 连续监测或间隔监测(如每小时一次)
- **试验后**: 全面功能测试, 覆盖所有关键参数
- 判据: 功能正常, 无误触发、无数据丢失、无通信中断

### 9.2 外观与电参数检查

- 外观: 目视/显微检查腐蚀、变色、起泡、开裂、焊点裂纹
- 绝缘电阻: 测试引线间施加500 VDC, > 10 MOhm
- 接触电阻: 连接器四线测量, 变化 < 10 mOhm
- 射频性能: 发射功率/灵敏度变化 < 3 dB


## 10 实战案例: 户外IoT气象站

### 10.1 设计要求

- 工作温度: -40 C 到 +85 C
- 防护等级: IP67
- 盐雾: 96小时
- 安装: 2m立杆, 沿海地区
- 设计寿命: 5年

### 10.2 试验方案

| 试验项目 | 条件 | 持续时间 | 判据 |
|----------|------|----------|------|
| 低温工作 | -40 C | 4h | 功能正常 |
| 高温工作 | +85 C | 4h | 功能正常 |
| 温度循环 | -40/+85 C, 5 C/min | 100次 | 功能正常, 焊点无裂纹 |
| 恒定湿热 | 85 C/85%RH | 240h | 绝缘电阻 > 10 MOhm |
| 随机振动 | 10-500 Hz, 2 Grms | 每轴30min | 功能正常, 无松动 |
| 盐雾 | 5% NaCl, 35 C | 96h | 功能正常, 外壳无穿透腐蚀 |
| IP67 | 1m浸水 | 30min | 无渗水, 功能正常 |

### 10.3 试验结果与改进

| 发现问题 | 根因 | 改进措施 |
|----------|------|----------|
| 85/85后绝缘电阻降至2 MOhm | 天线馈线入口吸湿 | 加灌封胶密封 |
| 盐雾后USB接口腐蚀 | USB连接器未密封 | 改用防水USB或密封盖 |
| 温循后BGA焊点裂纹 | 大尺寸BGA无底部填充 | 加底部填充胶 |
| 振动后电池松动 | 电池仓卡扣力度不足 | 增加弹片固定 |

## 11 从试验失败中改进设计

### 11.1 三防漆 (Conformal Coating)

三防漆是应对潮湿和腐蚀最常用的措施:

| 类型 | 优点 | 缺点 | 厚度 |
|------|------|------|------|
| 丙烯酸(AR) | 易返修 | 耐溶剂性差 | 25-75 um |
| 环氧(ER) | 耐化学性好 | 难返修 | 25-75 um |
| 有机硅(SR) | 耐高温、柔韧 | 耐溶剂性一般 | 50-125 um |
| 聚氨酯(UR) | 耐化学性、耐磨 | 难返修 | 25-75 um |

涂覆区域: 所有PCB表面, 需遮蔽: 连接器触点、测试点、天线区域、散热焊盘。

### 11.2 密封设计改进

- O型圈密封: 选择合适硬度和压缩率(15-30%)
- 超声波焊接外壳: 不可拆但密封性好
- 灌封: 整体灌封防护最彻底, 但散热和维修困难
- 呼吸阀: 平衡内外气压, 防止密封失效, 需选防水透气型

### 11.3 元件升级

从试验失败中发现薄弱元件:

- 商业级IC换成工业级(更宽温度、更严筛选)
- 普通电解电容换成固态电容(无电解液干涸)
- 通孔连接器换成带锁扣的工业连接器
- 0402电容换成0603(机械强度更高, 抗振动)

## 12 HALT: 发现设计裕度

### 12.1 HALT方法

HALT (Highly Accelerated Life Test) 与常规环境试验不同, 目标是找到产品的设计极限:

- 温度步进: 从20 C开始, 每步升/降10度, 保持10分钟后功能测试
- 继续直到发现工作极限(功能异常但恢复)和破坏极限(不可恢复)
- 振动步进: 从5 Grms开始, 每步增加5 Grms

### 12.2 HALT与常规试验的区别

| 维度 | 常规环境试验 | HALT |
|------|-------------|------|
| 目的 | 验证是否满足规格 | 发现设计裕度和薄弱环节 |
| 应力水平 | 规格范围内 | 远超规格, 直到破坏 |
| 结果 | 通过/不通过 | 工作极限和破坏极限的量化 |
| 用途 | 认证和验收 | 设计改进 |

### 12.3 HALT的价值

- 在设计早期发现薄弱环节, 改进成本远低于量产后发现
- 提供设计裕度的量化数据: 如规格-40 C, HALT发现工作极限-65 C, 则有25度裕度
- HALT发现的每一个失效都值得改进, 即使不在规格范围内

## 总结

环境试验是IoT硬件从设计走向可靠部署的关键验证环节。温度试验验证工作范围和热循环耐久, 湿度试验暴露凝露和吸湿问题, 振动试验确保机械可靠性, 盐雾试验评估腐蚀防护。试验规划的核心是从部署场景出发定义环境剖面, 选择合适的试验严酷度, 设计合理的试验顺序。试验发现的问题应通过三防漆、密封改进、元件升级等措施解决, 并通过复试验证改进效果。HALT作为设计验证的补充, 可以在规格范围外发现设计裕度和薄弱环节。环境试验不是走过场, 而是产品质量保证的重要一环。
