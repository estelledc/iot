# 光电二极管与光电晶体管选型对比

> **难度**：🟢 初级 | **领域**：光电检测、光通信、环境感知 | **阅读时间**：约 15 分钟

## 日常类比

想象你站在雨里接水。手里拿一个敞口杯子，雨滴落入杯中，水面上升的速度和雨的大小严格成正比——雨大水涨得快，雨小水涨得慢，一滴就是一滴，不多不少。这就像光电二极管：一个光子打进来，产生一个电子-空穴对，输出电流和光照强度保持精确的线性关系，没有放大，没有"添油加醋"。

现在换一种接法：你在杯口下面接了一个漏斗，漏斗连着一根细管，细管里水流被水泵加速——雨水滴进来一小股，管口喷出来一大股。虽然输出水量比实际雨量大得多，但你很难从管口的流量精确推算出到底下了多少雨，因为水泵的加速比不是完全稳定的。这就像光电晶体管：入射光只是"触发信号"，晶体管内部的电流放大机制会把光生电流放大几十到几百倍，灵敏度高了，但线性度也打了折扣。

最后一个场景：你要测量教室里有没有人开灯。如果只是判断"亮/暗"，用放大后的信号非常方便——一点光就能触发高电平，电路简单、反应明确。但如果要精确知道灯光有多少勒克斯，就得回到那个一滴一滴滴水的杯子，老老实实量水位。选光电二极管还是光电晶体管，本质上就是在"精确量水"和"灵敏报警"之间做取舍。

## 1. 光电效应基础

### 1.1 外光电效应与内光电效应

光电效应分为两大类：

- **外光电效应**：光子能量足够大时，电子从材料表面逸出，成为自由电子。光电倍增管（PMT）就基于此原理。需要真空环境，体积大、成本高。
- **内光电效应**：光子能量被半导体吸收，在材料内部产生电子-空穴对，但电子不逸出材料。光电二极管和光电晶体管都属于此类。

### 1.2 光子能量与带隙

半导体吸收光子的前提是光子能量大于材料的禁带宽度（带隙）$E_g$：

```
E_photon = h · ν = h · c / λ ≥ E_g
```

其中 $h$ 为普朗克常数，$c$ 为光速，$\lambda$ 为波长。由此可推出截止波长：

```
λ_c = h · c / E_g ≈ 1240 / E_g(eV)  (nm)
```

硅的带隙约 1.12 eV，对应截止波长约 1100 nm，因此硅基光电二极管可覆盖可见光到近红外波段。

### 1.3 光生载流子的产生与收集

光子进入半导体后，在耗尽区或其附近产生电子-空穴对。耗尽区内建电场将电子和空穴分别拉向 N 区和 P 区，形成光生电流。这是光电二极管和光电晶体管共同的工作基础——区别在于后续如何处理这个光生电流。

## 2. 光电二极管结构与工作模式

### 2.1 基本结构

光电二极管本质上是一个 PN 结或 PIN 结，带有透明窗口让光线射入。PIN 结构在 P 区和 N 区之间插入了一层本征半导体（I 层），增大了耗尽区宽度，从而提高量子效率、降低结电容、加快响应速度。

```
    光 ↓↓↓↓↓↓↓
  ┌──────────────┐
  │  P+  │  I  │  N  │   ← PIN 结构截面示意
  └──────────────┘
         耗尽区（I 层）
```

### 2.2 光伏模式（Photovoltaic Mode）

零偏压工作。PN 结受光照后产生开路电压或短路电流，不需要外部电源。

- **优点**：零暗电流，噪声最低，线性好
- **缺点**：结电容大导致响应慢（kHz 量级），输出电流小
- **典型应用**：太阳能电池、精密光功率计、环境光传感器

### 2.3 光电导模式（Photoconductive Mode）

反向偏压工作。外加反向电压扩大耗尽区、减小结电容，光生载流子在更强电场下更快漂移。

- **优点**：响应快（MHz ~ GHz），线性好，带宽大
- **缺点**：存在暗电流（反向漏电流），噪声比光伏模式略高
- **典型应用**：光纤通信接收、光学编码器、高速脉冲检测

### 2.4 两种模式对比

| 维度 | 光伏模式 | 光电导模式 |
|------|---------|-----------|
| 偏置 | 零偏压 | 反向偏压 |
| 响应速度 | kHz | MHz ~ GHz |
| 暗电流 | 近零 | nA ~ μA 级 |
| 线性度 | 优 | 优 |
| 噪声 | 最低 | 略高（暗电流贡献） |
| 电路复杂度 | 简单 | 需偏压源 |
| 典型应用 | 光功率计、太阳能 | 光纤通信、编码器 |

## 3. 光电晶体管结构与增益

### 3.1 基本结构

光电晶体管可以理解为一个基极悬空的 NPN（或 PNP）晶体管。光线照射到基区-集电结（BC 结），产生光生电流 $I_{ph}$，这个电流充当基极电流，经晶体管电流放大后输出：

```
I_C = β · I_ph
```

其中 $\beta$（或 $h_{FE}$）是晶体管的直流电流增益，典型值 50 ~ 500。

```
         光 ↓↓↓↓↓
       ┌──────────┐
  C ───┤  N  │ P │ N  ├─── E
       │ 集电区│基区│发射区│
       └──────────┘
        BC 结受光照
     （等效"光基极电流"）
```

### 3.2 达林顿光电晶体管

将两个晶体管级联形成达林顿结构，总增益为两个晶体管增益的乘积（$\beta_{total} = \beta_1 \times \beta_2$），可达数千甚至上万。

- **优点**：极高灵敏度，可检测极弱光信号
- **缺点**：响应更慢（开关时间 μs ~ ms 级），线性更差，暗电流更大

### 3.3 光电晶体管的局限性

光电晶体管有几个不可忽视的缺陷：

1. **增益不稳定**：$\beta$ 随温度、集电极电流、器件个体差异变化，批次间差异可达 2:1
2. **线性差**：$I_C$ 与光照强度只在很窄范围内近似线性，大信号时明显饱和
3. **频率响应差**：米勒效应导致输入电容被放大，带宽通常只有几十 kHz
4. **暗电流被放大**：集电结的反向漏电流同样被 $\beta$ 放大，温度升高时尤其显著

## 4. 详细对比

### 4.1 核心参数对比表

| 参数 | 光电二极管 | 光电晶体管 |
|------|-----------|-----------|
| 响应速度 | ns ~ μs（光电导模式可达 GHz 带宽） | μs ~ ms |
| 灵敏度 | 低（无内增益） | 高（β 倍增益） |
| 线性度 | 优（> 5 个数量级） | 差（窄范围内近似线性） |
| 暗电流 | pA ~ nA（光伏模式近零） | nA ~ μA（被 β 放大） |
| 温度稳定性 | 好 | 较差（β 和漏电流均随温度变化） |
| 噪声特性 | 低 | 较高（散粒噪声 + 热噪声） |
| 成本 | 中（需配 TIA） | 低（可直接驱动逻辑电平） |
| 电路复杂度 | 高（需跨阻放大器） | 低（上拉电阻即可） |
| 一致性 | 好 | 差（β 分散性大） |

### 4.2 灵敏度 vs 线性度的权衡

这是两者选型的核心矛盾：

- **光电二极管**：输出电流 = 光子数 × 量子效率，严格线性，但电流极小（μA 级甚至 nA 级），必须外接放大器
- **光电晶体管**：输出电流 = 光子数 × 量子效率 × β，灵敏度高出数十倍，但 β 非恒定导致线性劣化

### 4.3 选型决策流程

```
是否需要精确测量光强？
├── 是 → 光电二极管 + TIA
│   ├── 需要高速响应？ → 光电导模式 + 低噪声 TIA
│   └── 低速即可？ → 光伏模式 + 精密运放
└── 否 → 只需检测有光/无光？
    ├── 是 → 光电晶体管 + 上拉电阻（最简方案）
    └── 否 → 需要中等灵敏度且允许一定非线性？
        └── 光电晶体管（折中方案）
```

## 5. 光谱响应特性

### 5.1 硅基器件的光谱响应

硅的带隙决定了其光谱响应范围约为 300 ~ 1100 nm，峰值响应在 850 ~ 950 nm 附近。这与常用红外 LED（如 850 nm、940 nm）的发射波长高度匹配，因此硅基光电器件在红外遥控、光断续器等场景中广泛应用。

### 5.2 不同材料的光谱覆盖

| 材料 | 光谱范围 | 峰值波长 | 典型应用 |
|------|---------|---------|---------|
| 硅 (Si) | 300 ~ 1100 nm | ~900 nm | 可见光 + 近红外检测 |
| 锗 (Ge) | 700 ~ 1800 nm | ~1550 nm | 光纤通信（已少用） |
| InGaAs | 900 ~ 1700 nm | ~1550 nm | 光纤通信接收端 |
| GaAsP | 300 ~ 680 nm | ~560 nm | 可见光色传感器 |
| PbS | 1000 ~ 3500 nm | ~2500 nm | 中红外气体分析 |

### 5.3 滤光片的应用

在实际 IoT 应用中，常在光电器件前方加装滤光片以排除干扰光：

- **红外截止滤光片**：只透可见光，用于环境光传感器（排除红外干扰）
- **窄带滤光片**：只透特定波长，用于脉冲血氧仪（只测 660 nm 和 940 nm）
- **紫外带通滤光片**：用于 UV 指数监测

## 6. 光电二极管的跨阻放大器电路

### 6.1 为什么需要 TIA

光电二极管在光电导模式下输出电流极小（nA ~ μA 级），不能直接驱动 ADC 或逻辑电路。跨阻放大器（Transimpedance Amplifier, TIA）将电流信号转换为电压信号，同时提供低输入阻抗以保持光电二极管的工作点稳定。

### 6.2 基本 TIA 电路

```
              R_f
    ┌───────/\/\/───────┐
    │                   │
    │   ┌───────┐       │
    │   │       │       │
 PD ├───│-      │       ├─── V_out
    │   │  OpAmp│       │
    │───│+      │       │
 GND│   └───────┘       │
    │                   │
   GND                GND

PD: 光电二极管（反向偏置接法）
R_f: 反馈电阻，决定跨阻增益
V_out = -I_ph × R_f
```

### 6.3 关键设计参数

| 参数 | 公式 | 说明 |
|------|------|------|
| 跨阻增益 | $V_{out} = -I_{ph} \times R_f$ | $R_f$ 越大灵敏度越高 |
| 带宽 | $f_{-3dB} \approx \frac{1}{2\pi \cdot R_f \cdot C_f}$ | $C_f$ 为补偿电容 |
| 噪声 | 热噪声 $\sqrt{4kTR_f \cdot BW}$ + 运放电流噪声 | $R_f$ 增大时热噪声也增大 |
| 稳定性 | 需满足 $C_f \geq \sqrt{\frac{C_j + C_{in}}{2\pi \cdot R_f \cdot GBW}}$ | 防止振荡 |

**实用建议**：

1. 运放选择：低输入偏置电流型（如 JFET 输入 OPA657、CMOS 输入 LMP7721）
2. $R_f$ 取值：先确定最大光电流和 ADC 满量程电压，再反算 $R_f = V_{FS} / I_{max}$
3. 补偿电容 $C_f$：从稳定性公式估算，实验微调（过大降带宽，过小易振荡）
4. PCB 布局：反馈电阻和电容尽量靠近运放反相输入端，减少寄生电容

## 7. 代码示例：Arduino 环境光检测

### 7.1 光电二极管方案（TIA + ADC）

```cpp
// 光电二极管 + TIA 环境光检测
// 硬件连接：PD → TIA (OPA333 + Rf=1MΩ + Cf=10pF) → Arduino A0
// TIA 输出电压范围: 0 ~ 3.3V (对应 0 ~ 3.3μA 光电流)

const int ANALOG_PIN = A0;
const float V_REF    = 3.3;     // 参考电压
const int ADC_BITS   = 10;      // Arduino UNO 为 10 位 ADC
const float R_FEEDBACK = 1.0e6; // TIA 反馈电阻 1MΩ
const int NUM_SAMPLES = 16;     // 过采样次数

void setup() {
  Serial.begin(9600);
  analogReference(EXTERNAL); // 使用 AREF 引脚外部参考电压
}

void loop() {
  // 多次采样取平均，降低量化噪声
  long adc_sum = 0;
  for (int i = 0; i < NUM_SAMPLES; i++) {
    adc_sum += analogRead(ANALOG_PIN);
  }
  float adc_avg = (float)adc_sum / NUM_SAMPLES;

  // 将 ADC 值转换为电压
  float voltage = adc_avg * V_REF / (1 << ADC_BITS);

  // 计算光电流 (I = V / Rf)
  float current_uA = (voltage / R_FEEDBACK) * 1e6;

  // 粗略估算照度（需根据实际 PD 灵敏度标定）
  // 假设 PD 灵敏度 0.5 A/W @ 550nm，辐照度 → 照度换算系数约 683 lx/(W/m²)
  float irradiance = current_uA * 1e-6 / 0.5;  // W/m²
  float lux = irradiance * 683.0;               // lx

  Serial.print("电压: "); Serial.print(voltage, 3); Serial.print(" V  ");
  Serial.print("光电流: "); Serial.print(current_uA, 3); Serial.print(" μA  ");
  Serial.print("照度: "); Serial.print(lux, 0); Serial.println(" lx");

  delay(500);
}
```

### 7.2 光电晶体管方案（分压 + ADC）

```cpp
// 光电晶体管环境光检测（简易方案）
// 硬件连接：光电晶体管 C 极 → 5V，E 极 → 10kΩ 下拉电阻 → GND
//           E 极同时接 Arduino A0
// 输出: E 极电压随光照增大而升高

const int ANALOG_PIN = A0;
const float V_CC     = 5.0;
const int R_PULLDOWN = 10000;  // 下拉电阻 10kΩ
const int NUM_SAMPLES = 8;

void setup() {
  Serial.begin(9600);
}

void loop() {
  long adc_sum = 0;
  for (int i = 0; i < NUM_SAMPLES; i++) {
    adc_sum += analogRead(ANALOG_PIN);
  }
  float adc_avg = (float)adc_sum / NUM_SAMPLES;

  float voltage = adc_avg * V_CC / 1023.0;

  // 计算通过光电晶体管的集电极电流
  float iC_mA = (voltage / R_PULLDOWN) * 1000.0;

  // 判断光照等级（非线性，仅做定性参考）
  const char* level;
  if (voltage < 0.5)       level = "很暗";
  else if (voltage < 1.5)  level = "较暗";
  else if (voltage < 3.0)  level = "正常";
  else if (voltage < 4.2)  level = "较亮";
  else                     level = "很亮";

  Serial.print("电压: "); Serial.print(voltage, 2); Serial.print(" V  ");
  Serial.print("电流: "); Serial.print(iC_mA, 2); Serial.print(" mA  ");
  Serial.print("等级: "); Serial.println(level);

  delay(500);
}
```

**两个方案的关键差异**：

| 对比项 | 光电二极管方案 | 光电晶体管方案 |
|--------|-------------|-------------|
| 额外器件 | 运放 + 4 个阻容 | 1 个下拉电阻 |
| 输出定量性 | 可标定到 lx | 仅定性分级 |
| 成本 | 较高 | 极低 |
| 开发难度 | 中 | 低 |

## 8. IoT 应用场景

### 8.1 光学编码器

旋转编码器使用光电二极管检测码盘上刻线的明暗变化，输出两路正交脉冲（A/B 相）用于判断旋转方向和角度。要求高速响应（> 100 kHz），必须使用光电二极管。光电晶体管的带宽不足以应对高速旋转场景。

### 8.2 接近传感器

红外接近传感器发射 940 nm 红外光，检测反射光判断物体距离。典型方案使用光电晶体管作为接收端，因为只需要判断"有无反射"或"强弱变化"，不需要精确测量光强。成本低、电路简单是核心优势。

### 8.3 光隔离器（光耦）

光耦内部将一颗红外 LED 和一颗光电晶体管封装在一起，实现输入输出之间的电气隔离。光电晶体管的增益使光耦可以直接驱动后级逻辑，无需额外放大器。这是光电晶体管最常见的应用之一——2024 年全球光耦出货量超过 50 亿颗。

### 8.4 脉冲血氧仪

脉搏血氧仪使用两个波长的光（660 nm 红光 + 940 nm 红外）交替照射手指，通过检测透射光强度计算血氧饱和度（SpO₂）。需要精确测量两种波长下的吸光度比值，必须使用光电二极管 + TIA 方案。光电晶体管的非线性误差会直接影响 SpO₂ 计算精度。

### 8.5 应用选型速查

| 应用 | 推荐器件 | 关键原因 |
|------|---------|---------|
| 光学编码器 | 光电二极管 | 需要高频响应 |
| 接近传感器 | 光电晶体管 | 只需有无判断，成本低 |
| 光耦隔离 | 光电晶体管 | 内增益直接驱动逻辑 |
| 脉搏血氧 | 光电二极管 | 需要精确光强测量 |
| 环境光传感器 | 光电二极管 | 线性度要求高 |
| 红外遥控接收 | 光电晶体管 | 信号强，只需解码 |
| 条码扫描 | 光电二极管 | 高速 + 高线性 |
| 自动门感应 | 光电晶体管 | 成本敏感，只需遮挡检测 |

## 总结与展望

光电二极管和光电晶体管虽然共享"内光电效应"这一物理基础，但在工程特性上走出了两条截然不同的路线：光电二极管追求"忠实"，一个光子换一个电子，线性与速度俱佳，但需要外部放大电路助力；光电晶体管追求"敏锐"，内部增益把微弱光信号放大到可用电平，电路极简，但代价是线性、速度和一致性的牺牲。

选型的核心判断只有两个问题：**需不需要精确测光强？需不需要高速响应？** 任一答案为"是"，就应选择光电二极管；两个都是"否"，光电晶体管的简单和低成本就是优势。

展望未来，几个趋势值得关注：

1. **集成化**：将光电二极管和 TIA 集成在同一芯片内（如 OPT101），降低设计门槛
2. **光谱扩展**：基于 InGaAs 等材料的光电二极管向短波红外（SWIR）延伸，为气体传感和食品安全检测开辟新应用
3. **智能光电晶体管**：集成温度补偿和自动增益控制的光电晶体管 IC 正在出现，缩小与光电二极管的线性差距
4. **硅光子集成**：在 SOI 平台上集成光电二极管和波导，为片上光谱分析提供可能
