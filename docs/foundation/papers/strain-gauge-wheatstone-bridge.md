# 应变片与惠斯通电桥测量电路

> **难度**：🟢 初级 | **领域**：力学传感、模拟电路、结构监测 | **阅读时间**：约 15 分钟

## 日常类比

想象一根橡皮筋，你用力拉它，它会变长变细。松手后它又恢复原状。应变片的工作原理和这个几乎一样——只不过它不是橡皮筋，而是一根极细的金属丝或箔栅，紧紧贴在你要测量的结构表面。结构被拉伸或压缩时，贴在上面的应变片也跟着变形，金属丝变长则电阻增大，变短则电阻减小。

但问题来了：这个电阻变化极其微小，通常只有零点几欧姆。如果你直接用万用表去量，根本分辨不出变化。这时候就需要惠斯通电桥出场了。惠斯通电桥就像一个精密的天平：天平两端各放砝码，只有当两边完全平衡时指针才指零；哪怕一端只多了一粒灰尘，天平也会偏转。惠斯通电桥把电阻的微小变化转换成了电压信号，让哪怕纳米级的形变也能被"看见"。

再打个比方：你站在体重秤上，秤内部的金属梁会微微弯曲，贴在梁上的应变片跟着变形，惠斯通电桥把变形转成电压，再放大显示成数字。每一次称重背后，都是应变片和电桥在默默工作。

## 1. 应变片基础

### 1.1 工作原理

应变片（Strain Gauge）是基于**电阻应变效应**工作的：导体在机械变形时，其电阻值会发生变化。电阻变化率与应变之间的关系由灵敏度系数（Gauge Factor, GF）描述：

$$
GF = \frac{\Delta R / R}{\varepsilon}
$$

其中：

- \(\Delta R / R\) —— 电阻相对变化率
- \(\varepsilon\) —— 轴向应变（无量纲，通常用 \(\mu\varepsilon\) 微应变表示）

### 1.2 常见材料与灵敏度系数

| 材料类型 | 灵敏度系数 (GF) | 电阻温度系数 | 典型应用 |
|----------|:---:|:---:|----------|
| 康铜 (Constantan) | 2.0 ~ 2.1 | 低 | 通用测量、称重传感器 |
| 卡玛合金 (Karma) | 2.0 ~ 2.4 | 极低 | 高精度、温度变化大的场景 |
| 半导体 (Si) | 100 ~ 175 | 高 | 微小应变检测、MEMS |
| 铂钨合金 | 3.0 ~ 4.5 | 中 | 高温环境 |

金属应变片 GF 约为 2，这意味着 1000 微应变（0.1% 变形）只产生约 0.2% 的电阻变化。对 120Ω 应变片来说，电阻只变化 0.24Ω——直接测量几乎不可能，必须借助电桥。

### 1.3 粘贴工艺要点

应变片的粘贴质量直接决定测量精度和长期稳定性：

1. **表面处理**：打磨至 Ra 3.2 以下，丙酮去油污
2. **选胶**：氰基丙烯酸酯（快速）、环氧树脂（长期稳定）
3. **定位**：沿主应力方向对齐，偏差 < 2°
4. **加压固化**：均匀加压 0.1~0.3 MPa，按胶水要求控制温度和时间
5. **防护**：涂覆硅橡胶或防护蜡，防潮绝缘

> 粘贴是应变片测量中最容易出问题的环节，被称为"三分器件、七分工艺"。

## 2. 惠斯通电桥电路理论

### 2.1 基本结构

惠斯通电桥由四个电阻臂组成，供电端施加激励电压 \(V_{ex}\)，输出端测量差分电压 \(V_o\)：

```
    Vex (+)
     |
    [R1]        [R2]
     |            |
     +---- Vo+ ---+
     |            |
    [R3]        [R4]
     |            |
    GND          GND
```

当电桥平衡时（\(V_o = 0\)），满足：

$$
R_1 \cdot R_4 = R_2 \cdot R_3
$$

### 2.2 输出电压推导

当某个桥臂电阻发生 \(\Delta R\) 变化时，输出电压近似为：

$$
V_o \approx \frac{V_{ex}}{4} \cdot \frac{\Delta R}{R}
$$

代入灵敏度系数公式 \(GF = \frac{\Delta R / R}{\varepsilon}\)，得到：

$$
V_o \approx \frac{V_{ex} \cdot GF \cdot \varepsilon}{4}
$$

**数值示例**：\(V_{ex} = 5V\)，\(GF = 2\)，\(\varepsilon = 1000\,\mu\varepsilon\)：

$$
V_o \approx \frac{5 \times 2 \times 0.001}{4} = 2.5\,\text{mV}
$$

仅 2.5 mV 的信号！这就是为什么后级必须放大。

## 3. 电桥配置：1/4 桥、半桥与全桥

### 3.1 四分之一桥（Quarter Bridge）

只有一个桥臂为工作应变片，其余三个为固定电阻。

- **优点**：布线最简单，只需一片应变片
- **缺点**：灵敏度最低，无法自补偿温度漂移
- **输出**：\(V_o = \frac{V_{ex}}{4} \cdot GF \cdot \varepsilon\)

适用场景：简单测量、温度稳定环境、成本敏感应用。

### 3.2 半桥（Half Bridge）

两个桥臂使用工作应变片，通常一片受拉、一片受压（如弯曲梁上下表面）。

- **优点**：灵敏度翻倍，自动温度补偿
- **缺点**：需要两片应变片，布线略复杂
- **输出**：\(V_o = \frac{V_{ex}}{2} \cdot GF \cdot \varepsilon\)

适用场景：弯曲梁测量、精度要求中等。

### 3.3 全桥（Full Bridge）

四个桥臂全部使用工作应变片，两片受拉、两片受压。

- **优点**：灵敏度最高（4 倍于 1/4 桥），最佳温度补偿，非线性最小
- **缺点**：成本最高，需要四片应变片，布线最复杂
- **输出**：\(V_o = V_{ex} \cdot GF \cdot \varepsilon\)

适用场景：称重传感器、高精度力测量。

### 3.4 配置对比

| 配置 | 工作应变片数 | 灵敏度倍数 | 温度补偿 | 非线性误差 | 典型应用 |
|------|:---:|:---:|:---:|:---:|----------|
| 1/4 桥 | 1 | 1× | 无 | 较大 | 简单应变测量 |
| 半桥 | 2 | 2× | 自补偿 | 中等 | 弯曲梁、轴力测量 |
| 全桥 | 4 | 4× | 自补偿 | 最小 | 称重传感器、精密测量 |

> 称重传感器几乎全部采用全桥配置，这是精度与稳定性要求共同决定的。

## 4. 信号放大：仪表放大器

### 4.1 为什么不能用普通运放？

电桥输出是**差分信号**且内阻较高，需要放大器同时满足：

- **高输入阻抗**：不从电桥取电流，避免加载误差
- **高共模抑制比（CMRR）**：抑制共模噪声（通常需 > 80 dB）
- **低输入偏置电流**：减少直流偏移
- **增益可调**：通过一个外接电阻设定增益

仪表放大器（Instrumentation Amplifier, INA）是专门为此设计的三运放结构，典型芯片包括：

| 芯片 | 增益范围 | CMRR | 供电 | 特点 |
|------|----------|------|------|------|
| INA125 | 1 ~ 10000 | > 80 dB | 单 5V | 内置基准电压、电桥激励 |
| AD620 | 1 ~ 10000 | > 100 dB | ±15V | 经典高精度 |
| HX711 | 固定 128× | — | 5V | 24位 ADC 集成，专为称重设计 |
| NAU7802 | 固定 128× | — | 3.3V | 低功耗，I2C 接口 |

### 4.2 放大链路设计

典型信号链路：

```
应变片 → 惠斯通电桥 → 仪表放大器 → 低通滤波 → ADC → 数字处理
  mV级        mV级          V级         V级      数字量
```

关键参数计算：

- 目标分辨率：1 \(\mu\varepsilon\)
- 全桥输出：\(5 \times 2 \times 10^{-6} = 10\,\mu\text{V}\)
- ADC 满量程 5V / 24位：LSB = 0.3 \(\mu\text{V}\)
- 所需增益：约 1000×（考虑噪声裕量）

## 5. 温度补偿方法

温度变化是应变片测量的头号敌人，它通过两条路径引入误差：

1. **应变片自身**：电阻温度系数导致阻值漂移
2. **被测材料**：热膨胀系数差异产生"视在应变"

### 5.1 自补偿应变片

选用与被测材料热膨胀系数匹配的应变片材料。例如康铜通过调整合金成分，使其对特定材料（如钢、铝）的温度漂移最小。

### 5.2 补偿片法（Dummy Gauge）

在电桥中增加一个相同型号的应变片，贴在同一材料上但不承受应变（补偿片）。温度变化时，工作片和补偿片阻值漂移相同，在电桥中相互抵消。

- 1/4 桥 + 补偿片 = 等效半桥温度补偿
- 半桥（上下表面）= 天然温度补偿

### 5.3 三线制与四线制接法

长导线电阻也会随温度变化。三线制接法将导线电阻分配到相邻桥臂，使温度漂移相互抵消；四线制则完全消除导线电阻影响。

| 接法 | 导线数 | 导线电阻影响 | 适用距离 |
|------|:---:|:---:|:---:|
| 二线制 | 2 | 不补偿 | < 3 m |
| 三线制 | 3 | 部分补偿 | 3 ~ 30 m |
| 四线制 | 4 | 完全消除 | > 30 m |

### 5.4 软件补偿

通过同时测量温度（如 PT100 或热敏电阻），在软件中建立温度-零漂曲线，实时修正。这是现代 IoT 传感节点最常用的补充手段。

## 6. 代码示例：Arduino + HX711 称重模块

### 6.1 基本接线

| HX711 引脚 | Arduino 引脚 | 说明 |
|------------|:---:|------|
| DT | D3 | 数据线 |
| SCK | D2 | 时钟线 |
| VCC | 5V | 供电 |
| GND | GND | 地 |
| E+ / E- | — | 电桥激励（接传感器） |
| A+ / A- | — | 通道 A 差分输入（接传感器） |

### 6.2 完整代码

```cpp
/*
 * HX711 称重传感器读取示例
 * 适用：全桥称重传感器 + HX711 模块 + Arduino Uno
 * 依赖库：HX711 by bogde（库管理器搜索安装）
 */

#include <HX711.h>

// 引脚定义
const int HX711_DT = 3;   // 数据引脚
const int HX711_SCK = 2;  // 时钟引脚

// 创建 HX711 对象
HX711 scale;

// 校准参数（需实际校准后填入）
const float CALIBRATION_FACTOR = 2280.0;  // 校准系数，因传感器而异
const float OFFSET = 0.0;                  // 零点偏移

void setup() {
  Serial.begin(57600);
  Serial.println("=== HX711 称重传感器测试 ===");

  // 初始化 HX711
  scale.begin(HX711_DT, HX711_SCK);

  // 等待传感器就绪
  Serial.print("等待传感器就绪...");
  while (!scale.is_ready()) {
    delay(500);
    Serial.print(".");
  }
  Serial.println(" 就绪!");

  // 读取原始值，确认通信正常
  long raw = scale.read();
  Serial.print("原始读数: ");
  Serial.println(raw);

  // 设置校准参数
  scale.set_scale(CALIBRATION_FACTOR);  // 设置校准系数
  scale.tare();                          // 去皮（零点校准）

  Serial.println("校准完成，开始测量...");
  Serial.println("发送 't' 去皮，'r' 重置校准系数");
}

void loop() {
  // 检查串口命令
  if (Serial.available()) {
    char cmd = Serial.read();
    if (cmd == 't') {
      scale.tare();  // 去皮
      Serial.println(">>> 已去皮");
    } else if (cmd == 'r') {
      scale.set_scale();  // 重置校准系数为 1
      Serial.println(">>> 校准系数已重置");
    }
  }

  // 读取重量（单位取决于校准时使用的标准砝码）
  if (scale.is_ready()) {
    float weight = scale.get_units(10);  // 10 次采样平均
    Serial.print("重量: ");
    Serial.print(weight, 2);
    Serial.println(" g");

    // 同时输出原始值，便于调试
    Serial.print("  原始: ");
    Serial.print(scale.read());
    Serial.print("  均值: ");
    Serial.println(scale.get_units(10), 5);
  }

  delay(500);
}
```

### 6.3 校准步骤

1. 上电，空载状态下发送 `t` 去皮
2. 放上已知重量砝码（如 100g）
3. 读取原始值，计算校准系数：`校准系数 = (原始值 - 空载值) / 已知重量`
4. 用 `scale.set_scale(校准系数)` 写入
5. 验证：放上不同砝码，误差应在 ±0.1% 以内

## 7. IoT 应用场景

### 7.1 结构健康监测（SHM）

在桥梁、大坝、高层建筑的关键部位粘贴应变片，通过 LoRa/NB-IoT 长期采集应变数据，实现：

- 超载预警：应变超过阈值触发报警
- 疲劳累积评估：统计应变循环次数和幅值
- 损伤定位：多测点数据对比识别异常区域

典型架构：

```
应变片 → 电桥 → INA → MCU(STM32L4) → LoRa → 网关 → 云平台
 [感知]   [调理]  [放大]   [采集/低功耗]   [传输]  [汇聚]  [分析]
```

### 7.2 智能称重

电子秤、物流分拣、料仓料位监测，全桥配置 + HX711 是最经典组合：

- 分辨率：24 位 ADC，可达 0.1g 级别
- 动态称重：结合滤波算法，在振动环境中稳定读数
- 多通道：一个 MCU 可挂多个 HX711，实现多点称重

### 7.3 工业过程控制

- **张力控制**：轧钢、造纸、拉丝生产线的张力实时监测
- **扭矩测量**：旋转轴上的应变片 + 无线滑环，替代传统滑环传输
- **压力容器监测**：高温高压容器的应变在线监测，保障安全

## 8. 常见问题排查

### 8.1 输出漂移严重

| 可能原因 | 排查方法 | 解决方案 |
|----------|----------|----------|
| 温度漂移 | 观察漂移是否与环境温度相关 | 采用补偿片或三线制接法 |
| 粘贴不良 | 检查应变片边缘是否翘起 | 重新粘贴，确保加压固化 |
| 导线电阻变化 | 测量导线电阻温度系数 | 改用三线制或四线制 |
| 绝缘下降 | 测量应变片与结构间绝缘电阻 | 重新防护密封，应 > 50 MΩ |

### 8.2 读数噪声大

- **检查电源**：电桥激励电源纹波 < 1 mV，建议使用基准电压源
- **增加采样平均**：HX711 内部已有采样，软件再平均 10~50 次
- **硬件滤波**：在 ADC 输入端加 RC 低通滤波（截止频率约 10 Hz）
- **屏蔽布线**：使用双绞线或屏蔽电缆，远离动力线

### 8.3 线性度差

- 确认应变范围不超过应变片量程（金属片通常 < 5000 \(\mu\varepsilon\)）
- 检查 1/4 桥的非线性误差：\(\delta \approx \frac{\Delta R}{2R}\)，全桥可消除
- 排除机械间隙或蠕变：加载后等待稳定再读数

### 8.4 HX711 常见通信问题

```cpp
// HX711 调试：手动读取原始数据，排除库的问题
long readRaw() {
  // 等待数据就绪（D0 变低）
  while (digitalRead(HX711_DT)) {
    delayMicroseconds(10);
  }

  long value = 0;
  // 读取 24 位数据
  for (int i = 0; i < 24; i++) {
    digitalWrite(HX711_SCK, HIGH);
    delayMicroseconds(1);
    value = (value << 1) | digitalRead(HX711_DT);
    digitalWrite(HX711_SCK, LOW);
    delayMicroseconds(1);
  }

  // 第 25 个脉冲设置通道和增益：通道 A，增益 128
  digitalWrite(HX711_SCK, HIGH);
  delayMicroseconds(1);
  digitalWrite(HX711_SCK, LOW);
  delayMicroseconds(1);

  // 转换为有符号数
  if (value & 0x800000) {
    value |= 0xFF000000;  // 补码扩展
  }

  return value;
}
```

如果手动读取正常但库读取异常，检查：

- SCK 和 DT 引脚是否接反
- 供电是否稳定（HX711 对电压波动敏感）
- 是否与其他 SPI 设备冲突

## 总结与展望

应变片和惠斯通电桥是力学传感的基石组合。核心要点回顾：

1. 应变片将微小形变转为微小电阻变化（GF ≈ 2），直接测量不可行
2. 惠斯通电桥将电阻变化转为差分电压，灵敏度取决于桥臂配置
3. 全桥配置提供最高灵敏度和最佳温度补偿，是称重和精密测量的首选
4. 仪表放大器（或 HX711 等集成方案）完成 mV → V 的放大，是信号链关键一环
5. 温度补偿是长期稳定的保障，需从材料、接法、软件多层面综合施策

未来趋势：

- **MEMS 应变片**：与信号调理电路单片集成，体积更小、成本更低
- **光纤光栅（FBG）**：替代传统电阻应变片，免疫电磁干扰，单根光纤可串联数十个测点
- **无线应变传感器**：能量收集 + LoRa/蓝牙，真正免布线部署
- **AI 辅助校准**：利用机器学习模型补偿非线性与温度漂移，减少人工标定工作量
