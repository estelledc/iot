# 开关电容电路在传感器接口中的应用
> **难度**：🔴 高级 | **领域**：模拟集成电路 | **阅读时间**：约 22 分钟

## 引言

想象你在用天平称重。如果砝码的绝对重量不准,称量就不准;但如果两个砝码之间的比例是精确的,你就可以用比例关系做相对测量。开关电容电路(SC)的核心思想正是如此:不依赖电阻的绝对值,而是靠电容的比值来设定增益和频率特性。由于集成电路中电容比值可以做到 0.1% 精度,而电阻绝对值偏差高达 20%,SC 技术成为传感器接口电路的基石。

## 1 开关电容的基本原理

### 1.1 用开关和电容模拟电阻

开关电容电路的核心操作:在两个时钟相之间,电容 C 在两个节点之间转移电荷。当开关以频率 f_clk 切换时,等效电阻为:

```
R_eq = 1 / (f_clk * C)
```

例如: C = 1 pF, f_clk = 1 MHz, 则 R_eq = 1 M(omega)。

这个等效电阻不依赖任何物理电阻,只由时钟频率和电容值决定。时钟来自片上振荡器,精度极高;电容比值由光刻决定,精度同样极高。这就是 SC 电路可全集成的基础。

### 1.2 基本工作过程

典型的两相不重叠时钟控制开关:

```
相位 1 (phi1): 开关 S1 闭合, S2 断开 -- 电容充电到 V1
相位 2 (phi2): 开关 S1 断开, S2 闭合 -- 电容放电到 V2
```

每个时钟周期转移电荷量 Q = C * (V1 - V2),等效电流 I = Q * f_clk = C * f_clk * (V1 - V2),即 I = (V1 - V2) / R_eq。

### 1.3 为什么不用真实电阻

| 特性 | 片上电阻 | 开关电容等效电阻 |
|------|----------|------------------|
| 绝对精度 | 15-20% | 由 f_clk 和 C 决定 |
| 温度系数 | 0.1%/C | 几乎为零 |
| 面积 | 大(高阻值) | 小(小电容+开关) |
| 可编程性 | 无 | 改变 f_clk 即可 |
| 与数字工艺兼容 | 差 | 好 |

## 2 SC 基本积木

### 2.1 SC 积分器

SC 积分器是 Delta-Sigma ADC 的核心模块:

```
         phi1      phi2
          |         |
Vi o--S1--+--S2--+--||--+--||--+--Vo
                C1     C2   |\|
                             | \|
                             |  |--- GND
                             | /
                             |/
                             Op-Amp
```

phi1 期间 C1 采样输入电压,phi2 期间 C1 向 C2 转移电荷。传递函数:

```
H(z) = -(C1/C2) * z^(-1) / (1 - z^(-1))
```

增益由 C1/C2 比值决定,而非绝对值。这正是 SC 电路的核心优势。

### 2.2 SC 放大器

将 SC 积分器增加复位相,即可实现放大器:

- phi1: C1 采样输入, C2 复位
- phi2: C1 向 C2 转移电荷, 输出电压 Vout = -(C1/C2) * Vin

增益精度仅取决于 C1/C2 比值,与运放开环增益无关(在合理范围内)。

### 2.3 SC 采样保持电路

ADC 前端必备,消除采样瞬间的输入变化:

- phi1: 采样相,电容跟踪输入
- phi2: 保持相,电容保持采样值供 ADC 转换

## 3 SC 在传感器接口中的优势

### 3.1 增益由电容比值决定

传感器信号通常很小(毫伏甚至微伏级),需要精确放大。SC 放大器增益 = C1/C2:

- 片上电容比值精度可达 0.1%
- 不受温度和工艺漂移影响
- 不同批次一致性极好

### 3.2 低功耗

SC 电路功耗主要来自:

- 运放的静态电流
- 开关的动态功耗(C * V^2 * f)

通过降低时钟频率可以同时降低两者。现代 SC 传感器接口功耗可低至微瓦量级。

### 3.3 与数字电路同工艺集成

SC 只需要:开关(MOS 管)、电容(MIM/MOM)、运放。这些在标准数字 CMOS 工艺中都有。不需要做高精度电阻、不需要特殊工艺选项。传感器模拟前端和数字信号处理可以放在同一颗芯片上。

### 3.4 可编程性

改变时钟频率即可改变等效电阻,从而改变滤波器截止频率或放大器增益。这使 SC 接口可以通过软件配置适应不同传感器。

## 4 SC 在 Sigma-Delta ADC 中的应用

### 4.1 SC 积分器作为环路滤波器

Delta-Sigma ADC 的核心是环路滤波器,通常用 SC 积分器级联实现:

```
+----+     +----+     +----+
| SC |     | SC |     | SC |
| INT |--->| INT |--->| INT |---> Comparator
| 1  |     | 2  |     | 3  |
+----+     +----+     +----+
  ^                       |
  |------- DAC <---------+
```

1 阶调制器用 1 个 SC 积分器,2 阶用 2 个,以此类推。阶数越高,量化噪声整形越好,但稳定性越差。

### 4.2 内在抗混叠

SC 电路的采样操作本身就具有低通特性。SC 积分器在采样频率整数倍处有陷波,自然抑制高频信号折叠。这减少了对前端模拟抗混叠滤波器的要求。

## 5 电容式传感器读出: CDC

### 5.1 电容式传感器

MEMS 湿度、接近度、压力等传感器常以电容变化为测量量:

- 湿度传感器: 介电常数随湿度变化, C = C0 * (1 + alpha * RH)
- 接近度传感器: 目标物靠近改变电极间电容
- MEMS 加速度计: 质量块位移改变差分电容

### 5.2 SC 电荷转移法

SC 电容-数字转换器(CDC)的基本原理:

```
phi1: Cs 充电到 Vref
phi2: Cs 向 Cf 放电
重复 N 次直到输出达到阈值
N 正比于 Cf/Cs, 即正比于被测电容
```

这种电荷平衡法不需要参考电容,精度由时钟和参考电压决定。

### 5.3 代表性 CDC 芯片

| 型号 | 制造商 | 分辨率 | 量程 | 接口 |
|------|--------|--------|------|------|
| AD7745 | ADI | 24 bit | 0-4 pF | I2C |
| AD7747 | ADI | 24 bit | 0-4 pF | I2C |
| FDC1004 | TI | 24 bit | -15~+115 pF | I2C |
| PCap01 | ACAM | 20 bit | 0-2 nF | SPI |

AD7745/7747 使用 SC 电荷平衡 CDC,20 位有效分辨率,适用于高精度电容测量。

## 6 SC 可编程增益放大器

### 6.1 工作原理

SC PGA 通过切换不同的输入/反馈电容组合实现不同增益:

```
增益选择: G1 = C1/Cf,  G2 = C2/Cf,  G4 = C4/Cf
C1, C2, C4 是二进制加权的输入电容
Cf 是固定反馈电容
```

用数字信号选择哪个输入电容接入即可改变增益。增益切换速度取决于时钟频率,可实现逐通道自动增益。

### 6.2 在 ADC 前端的应用

传感器信号幅度范围差异大(温度传感器 mV 级,加速度计 V 级)。SC PGA 在 ADC 前端自动调整增益,使信号充分利用 ADC 满量程,最大化信噪比。

## 7 相关双采样 CDS

### 7.1 CDS 原理

运放失调和 1/f 噪声是低频精密测量的两大敌人。CDS 通过连续两次采样相减消除它们:

```
相位 1 (采样相): 采集 [信号 + 失调 + 1/f噪声]
相位 2 (参考相): 采集 [失调 + 1/f噪声]
相减结果 = 信号
```

两次采样间隔只有一个时钟周期,失调和 1/f 噪声几乎不变,相减后被消除。

### 7.2 CDS 在传感器中的应用

- 图像传感器: 每个像素读出都做 CDS,消除列放大器失调
- 精密称重: 消除运放失调的温漂
- 传感器桥路接口: 消除桥路共模和运放失调

CDS 的代价是需要两个采样相,带宽减半。但在低频传感器应用中这不是问题。

## 8 噪声考量

### 8.1 kT/C 噪声

开关打开时,电容上采样的热噪声均方值:

```
Vn^2 = kT / C
```

k = 1.38e-23 J/K, T = 绝对温度, C = 采样电容。

| C | Vn (rms) at 300K |
|---|-------------------|
| 1 pF | 64.5 uV |
| 10 pF | 20.4 uV |
| 100 pF | 6.45 uV |

增大 C 可降低噪声,但:

- 面积增大
- 充放电时间增大,限制速度
- 功耗增大

设计时在噪声、面积、速度三者间权衡。

### 8.2 时钟馈通和电荷注入

MOS 开关关断时,沟道中残留电荷注入到采样电容:

- 电荷量: delta_Q = C_ox * W * L * (Vgs - Vth) / 2
- 引入失调电压: delta_V = delta_Q / C_s

缓解方法:

1. 互补开关(PMOS + NMOS): 电荷部分抵消
2. 底板采样(bottom-plate sampling): 先断开底板开关,将电荷注入固定到已知节点
3. 虚拟开关(dummy switch): 引入等量反号电荷抵消

## 9 实际案例: MEMS 压力传感器读出

### 9.1 系统架构

```
MEMS 压力传感器(差分电容)
        |
   SC CDC (电荷平衡)
        |
   20-bit Delta-Sigma ADC
        |
   数字滤波器 (sinc3)
        |
   温度补偿 (片上温度传感器)
        |
   校准系数 (OTP 存储器)
        |
   I2C/SPI 输出
```

### 9.2 SC 电荷平衡 CDC 工作流程

1. phi1: 传感器电容 Cs 充电到 Vref
2. phi2: Cs 向积分电容 Cin 转移电荷
3. 积分器输出与阈值比较
4. 反馈 DAC 根据比较结果增减参考电荷
5. 计数器累计 DAC 输出,得到数字码

### 9.3 温度补偿

MEMS 电容既随压力变化,也随温度变化。补偿方案:

- 片上温度传感器测量温度
- OTP 中存储的校准系数(多点校准)
- 数字域做温度补偿: C_compensated = C_measured - f(T)

### 9.4 性能指标

| 参数 | 典型值 |
|------|--------|
| 分辨率 | 20 bit |
| 量程 | 300-1100 hPa |
| 精度 | +/-1 hPa |
| 噪声 | 0.02 hPa (rms) |
| 功耗 | 5 uA @ 1 Hz |
| 采样率 | 1-200 Hz 可编程 |

## 总结

开关电容电路是传感器接口的核心使能技术:

1. 增益和频率特性由电容比值决定,与工艺和温度无关
2. 全集成于标准 CMOS 工艺,无需高精度电阻
3. 通过改变时钟频率实现可编程性
4. CDS 技术消除失调和低频噪声
5. kT/C 噪声是基本限制,需在电容大小和性能间权衡

从 MEMS 压力传感器到电容式湿度计,从图像传感器到称重仪,SC 技术无处不在。理解其原理,是设计高精度传感器接口的第一步。
