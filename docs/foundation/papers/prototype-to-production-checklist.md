# IoT硬件原型到量产的设计检查清单
> **难度**：🟡 中级 | **领域**：产品工程 | **阅读时间**：约 20 分钟

## 引言

想象你做了一道菜,自己和家人尝了都说好吃。但要把这道菜放到餐厅菜单上,就需要考虑:食材能否稳定采购?每份份量一致吗?厨房忙的时候还能做出来吗?冷藏运输后口感会变吗?原型到量产的距离就像"家里做菜"到"餐厅出品"的距离:不仅功能要工作,还要在大规模制造和现场使用中可靠工作。

## 1 原型与量产的差距

### 1.1 原型的特点

原型阶段的特点和隐藏问题:

- 手工焊接、飞线补丁 -- 量产时无法复制
- 元器件来自淘宝/Adafruit -- 不是正规供应渠道
- 室温 25C 下测试 -- 没有覆盖全温度范围
- 一个人使用 -- 没有用户误操作的考验
- 功能可用 -- 没有经过 EMC/ESD 考验

### 1.2 量产的要求

量产必须满足:

- 每台设备一致性: 生产线上的第 1 台和第 10000 台性能相同
- 制造可行性: 贴片机、回流焊、ICT 测试都能处理
- 环境适应性: -40 到 85C, 湿度, 振动, ESD
- 合规性: FCC/CE/安全认证
- 可维护性: 固件升级、现场维修
- 可追溯性: 序列号、批次追踪

### 1.3 差距来源

```
原型阶段                     量产阶段
-------------------------------------------
"能用"                     "每台都能用"
实验室环境                  现场恶劣环境
手工装配                  自动化产线装配
随便买器件                正规供应链
无认证                    全套认证
无测试                    100% 产线测试
```

## 2 电气检查清单

### 2.1 电源轨验证

- [ ] 所有电源轨在满载下纹波在规格内
- [ ] 电源轨在最大负载跳变时瞬态响应满足要求
- [ ] 各电源轨上电顺序符合器件要求
- [ ] 电池供电: 最低电压下系统仍能正常工作
- [ ] 电池放电曲线末端,看门狗复位后能可靠重启

检查方法: 示波器测量各轨在空载/满载/动态负载下的波形。

### 2.2 热分析

- [ ] 每个器件的结温/壳温在最高环境温度下不超过额定值
- [ ] 热源附近器件没有受到影响
- [ ] 散热方案(散热片/导热垫/铜皮)已验证

```
T_junction = T_ambient + P_dissipation * R_theta_ja
R_theta_ja = 结到环境的热阻(从数据手册获取)
```

### 2.3 ESD 保护

- [ ] 所有外部接口(I2C/SPI/UART/USB/天线)有 ESD 保护
- [ ] ESD 器件选型: 钳位电压低于被保护器件的耐压
- [ ] ESD 器件放置: 尽可能靠近连接器入口
- [ ] 人手可触及的按钮/接口有 ESD 保护

常见错误: ESD 器件放得离连接器太远,ESD 放电在 PCB 走线上辐射到其他信号。

### 2.4 EMC 预扫

- [ ] 传导发射(CE): 电源线上的噪声在限制线以下
- [ ] 辐射发射(RE): 空间辐射在限制线以下
- [ ] 快速瞬变脉冲群(EFT): 电源和信号端口通过
- [ ] 浪涌(Surge): 电源端口通过

建议在正式认证前做预扫描,及早发现问题。

### 2.5 电压降额

- [ ] 电容耐压至少为工作电压的 1.5-2 倍
- [ ] 电阻功率至少为实际功耗的 2 倍
- [ ] MOS 管Vds 至少为工作电压的 1.5 倍

### 2.6 测试点

- [ ] 所有电源轨有测试点(便于产线测试和调试)
- [ ] 关键信号(SWD/I2C/UART)有测试点
- [ ] 测试点位置: 贴片机可达、探针可接触
- [ ] 测试点间距足够(探针阵列的最小间距要求)

## 3 机械检查清单

### 3.1 外壳与 PCB 配合

- [ ] PCB 在外壳内不晃动(固定点数量和位置)
- [ ] 连接器与外壳开孔对齐(3D 模型验证)
- [ ] PCB 与外壳壁之间有足够间隙(考虑公差)
- [ ] 拆装方便(无需特殊工具)

### 3.2 连接器对齐

- [ ] 连接器伸出外壳的开孔居中
- [ ] USB/RJ45 等标准连接器的弹片与开孔匹配
- [ ] 天线连接器的位置不影响射频性能

### 3.3 热路径

- [ ] 发热器件有导热路径到外壳
- [ ] 导热垫/导热硅脂的厚度和面积已确定
- [ ] 外壳散热孔的位置和面积已验证

### 3.4 振动和跌落

- [ ] 大体积器件(电解电容、变压器)有固定措施
- [ ] 跌落测试: 1m 高度, 6 面, 每面 1 次
- [ ] 振动测试: 根据应用场景选择标准(如 IEC 60068)

### 3.5 防护等级

- [ ] IP 等级已确定并验证(如 IP65: 防尘防喷水)
- [ ] 密封方式: 密封圈/灌封/超声波焊接
- [ ] 透气阀(如需要,平衡内外气压差)

### 3.6 标签和标记

- [ ] 外壳上有: 型号、序列号位置、认证标记
- [ ] PCB 上有: 版本号、日期码、测试点标记
- [ ] 标签空间: 二维码/条码的尺寸和位置已预留

## 4 制造检查清单

### 4.1 BOM 定稿

- [ ] BOM 中无原型阶段专用的元器件
- [ ] 每个器件有制造商型号和替代型号
- [ ] 所有器件的封装与 PCB 焊盘匹配
- [ ] BOM 版本号已锁定

### 4.2 器件可用性

- [ ] 所有器件不是 EOL(End of Life)状态
- [ ] 所有器件不是 NRND(Not Recommended for New Design)
- [ ] 关键器件有第二供应商
- [ ] 交货周期可接受(长交期器件需提前备货)

### 4.3 DFM 审查

- [ ] 组装厂已完成 DFM(Design for Manufacturing)审查
- [ ] 所有器件适合机器贴装(无手焊件,或手焊件已标注)
- [ ] 器件间距满足贴装要求(最小间距已确认)
- [ ] PCB 拼板设计完成(工艺边、定位孔、Mark 点)

### 4.4 钢网和贴装

- [ ] 钢网已订购(厚度根据器件类型选择)
- [ ] 贴装程序已生成(坐标文件从 PCB 软件导出)
- [ ] 回流焊温度曲线已确认(无铅工艺峰值温度 245-255C)

## 5 测试检查清单

### 5.1 产线测试方案

- [ ] 测试步骤文档已编写
- [ ] 测试夹具(Jig)已设计和制作
- [ ] Pass/Fail 标准已定义(每个测试项的具体数值)
- [ ] 校准步骤已编写(如有需要)

### 5.2 测试覆盖

| 测试类型 | 检查内容 | 典型耗时 |
|----------|----------|----------|
| ICT/飞针 | 开短路、器件值 | 30-60s |
| 功能测试 | 电源轨、通信、传感器 | 1-3 min |
| 射频测试 | 发射功率、频率误差 | 1-2 min |
| 老化测试 | 高温运行 4-24h | 4-24 h |

### 5.3 校准

- [ ] 哪些参数需要校准已确定(ADC 增益、RTC、传感器偏移)
- [ ] 校准数据存储方案已实现(Flash/EEPROM)
- [ ] 校准夹具和参考源已准备

## 6 固件检查清单

### 6.1 生产固件

- [ ] 固件版本已锁定(不使用 debug 版本)
- [ ] 固件版本号可从设备读取(AT 命令或串口)
- [ ] 看门狗已启用(防止死机)
- [ ] 断电保护: Flash 写入中途断电不损坏固件

### 6.2 工厂测试模式

- [ ] 产线可进入测试模式(按键组合或测试点)
- [ ] 测试模式可逐项执行: LED 亮灭、传感器读数、通信
- [ ] 测试结果可读取(串口输出或 Pass/Fail 指示灯)

### 6.3 预烧录(Provisioning)

- [ ] 每台设备的唯一序列号写入方案
- [ ] 安全密钥/证书的写入和验证(安全存储区)
- [ ] WiFi/BLE MAC 地址写入(如未预烧)
- [ ] 预烧录过程有校验步骤(写后读回比对)

### 6.4 OTA 更新

- [ ] OTA 更新机制已验证(完整性和回滚)
- [ ] 双 Bank 或 A/B 分区方案已实现
- [ ] OTA 失败后设备可自动回滚到旧版本
- [ ] OTA 过程中有进度指示(防止用户断电)

## 7 认证检查清单

### 7.1 射频和 EMC

- [ ] FCC Part 15(美国)或 CE/RED(欧洲)已获得
- [ ] 其他区域认证: IC(加拿大), RCM(澳洲), TELEC(日本)
- [ ] 认证报告和证书已存档

### 7.2 安全认证

- [ ] UL/IEC 62368(音视频和 IT 设备)如需要
- [ ] IEC 61010(测量设备)如需要
- [ ] ATEX/IECEx(防爆)如需要

### 7.3 射频模块认证

- [ ] 射频模块的认证覆盖了最终产品形态
- [ ] 天线增益和类型与认证一致
- [ ] 模块认证的 SDR(Specific Derivative Rules)已遵守

## 8 供应链检查清单

### 8.1 双源策略

- [ ] 关键器件(MCU、电源IC、射频模块)有第二供应商
- [ ] 替代器件已经过验证(不是纸上替代)
- [ ] 替代器件的 PCB 封装兼容

### 8.2 交货周期

- [ ] 所有器件的交货周期可接受
- [ ] 长交期器件(>16 周)已提前备货
- [ ] 有安全库存策略应对缺货

### 8.3 MOQ 和量产计划

- [ ] 器件 MOQ 与生产计划匹配
- [ ] 价格梯度与预计用量匹配
- [ ] 供应商已确认产能可满足需求

### 8.4 器件生命周期

- [ ] 所有器件不在 EOL 状态
- [ ] 已查询 PCN(Product Change Notification)历史
- [ ] 关键器件的生命周期预测: 至少 5 年供货

## 10 常见错误

### 10.1 最后一刻器件替换

问题: BOM 中的某个器件缺货,临时替换了一个"兼容"的。

- "兼容"可能只是引脚兼容,电气特性有差异
- 不同厂商的运放输入偏置电流可能差 10 倍
- 不同厂商的 LDO 噪声特性可能不同

建议: 替换后必须做完整的验证测试,不能想当然。

### 10.2 跳过试产

问题: 从原型直接跳到大批量生产。

- 试产(Pilot Run)是验证制造流程的关键步骤
- 通常做 50-200 台,验证:
  - 贴装和回流焊是否正常
  - 测试夹具是否好用
  - 良率是否达标
  - 组装是否顺利

### 10.3 BOM/PCB 无版本控制

问题: 修改了 BOM 或 PCB,但没有更新版本号。

- 产线可能用了旧版 BOM 配新版 PCB
- 固件可能基于旧版硬件调试
- 维修时无法确认具体版本

建议: 任何修改必须更新版本号,并通知所有相关方。

## 总结

从原型到量产是一条系统化的工程路径,每个环节都不能跳过:

1. 电气: 电源、热、ESD、EMC 全面验证
2. 机械: 外壳配合、连接器对齐、防护等级
3. 制造: DFM 审查、拼板设计、钢网准备
4. 测试: 测试方案、夹具、校准
5. 固件: 生产版本锁定、预烧录、OTA
6. 认证: FCC/CE/安全
7. 供应链: 双源、交期、生命周期
8. 文档: 设计、测试、用户文档全部就位

最关键的原则: 不要跳过试产。50 台试产发现的问题,比 5000 台量产发现的问题便宜 100 倍。
