# RFID 感知识别技术综述

> **难度**：🟢 入门 | **领域**：射频识别、自动识别技术 | **阅读时间**：约 25 分钟

## 日常类比

你坐地铁刷卡时，卡不用接触闸机就能"嘀"一下通过——这就是 RFID（射频识别）。更神奇的是，超市的防盗门能在你经过时"感知"到你包里没付款的商品，仓库的叉车经过货架时能瞬间"读取"上百件货物的信息。

RFID 就像给每个物品贴了一个"隐形身份证"，不需要对准、不需要接触、甚至不需要电池，就能被远距离识别。如果说条形码是"物品的名片需要你主动去看"，RFID 就是"物品会主动报出自己的身份"。

## 1. RFID 基础原理

### 1.1 系统组成

一个 RFID 系统由三部分组成：

```
┌──────────┐    无线电波    ┌──────────┐    数据    ┌──────────┐
│  标签    │ ◄──────────► │  读写器  │ ──────── │  后端    │
│  (Tag)   │              │ (Reader) │          │ 系统     │
└──────────┘              └──────────┘          └──────────┘
 芯片+天线               发射/接收RF             数据库/应用
 贴在物品上              固定式/手持式           ERP/WMS等
```

### 1.2 工作原理

**无源标签**（最常见）的工作过程：

1. 读写器天线发射射频能量
2. 标签天线接收能量，给芯片供电（能量收集！）
3. 芯片被唤醒，将存储的 ID 调制到反射信号中（反向散射）
4. 读写器接收反射信号，解调得到标签 ID
5. 整个过程耗时 < 10 ms

**关键洞察**：无源 RFID 标签没有电池，它的能量完全来自读写器发射的电磁波——这是最早的"能量收集"技术之一。

### 1.3 频段分类

| 频段 | 频率 | 读距 | 数据率 | 典型应用 | 标签成本 |
|------|------|------|--------|----------|----------|
| LF（低频） | 125-134 kHz | < 10 cm | < 1 kbps | 动物芯片、门禁卡 | $0.5-2 |
| HF（高频） | 13.56 MHz | < 1 m | 25-100 kbps | NFC支付、图书馆、公交卡 | $0.1-0.5 |
| UHF（超高频） | 860-960 MHz | 1-12 m | 40-640 kbps | 物流、零售、仓储 | $0.03-0.1 |
| 微波 | 2.45/5.8 GHz | > 10 m | > 1 Mbps | 高速公路收费（ETC） | $5-20 |

### 1.4 标签类型

| 类型 | 电源 | 读距 | 成本 | 寿命 | 应用 |
|------|------|------|------|------|------|
| 无源（Passive） | 读写器供电 | LF:<10cm, UHF:<12m | $0.03-2 | 无限（无电池） | 物流、零售 |
| 半有源（BAP） | 电池辅助+反向散射 | 15-30 m | $3-10 | 3-5 年 | 冷链监控 |
| 有源（Active） | 自带电池 | 30-100 m | $10-50 | 2-5 年 | 人员定位、资产追踪 |

## 2. EPC Gen2 协议深度解析

### 2.1 什么是 EPC Gen2？

EPC（Electronic Product Code）Gen2 是 UHF RFID 的全球标准（ISO 18000-63），由 GS1 组织制定。它定义了读写器与标签之间的通信协议——相当于 RFID 世界的"普通话"。

### 2.2 协议栈结构

```
┌─────────────────────────────────┐
│  应用层：EPC 编码（SGTIN/SSCC） │
├─────────────────────────────────┤
│  标签层：内存结构（4 Bank）      │
│  Bank 0: 保留区（密码）         │
│  Bank 1: EPC 区（96-496 bit）   │
│  Bank 2: TID 区（芯片唯一 ID）  │
│  Bank 3: 用户区（可选数据）     │
├─────────────────────────────────┤
│  通信层：防碰撞（Q 算法）       │
├─────────────────────────────────┤
│  物理层：反向散射调制            │
│  前向：ASK/PR-ASK, PIE 编码    │
│  反向：FM0/Miller 编码          │
└─────────────────────────────────┘
```

### 2.3 防碰撞机制

当多个标签同时在读写器范围内时，如何避免"抢话"？

**Q 算法（时隙 ALOHA）**：
1. 读写器广播 Q 值（如 Q=4，表示 2^4=16 个时隙）
2. 每个标签随机选一个时隙（0-15）
3. 选到时隙 0 的标签先回复
4. 如果只有一个标签回复 → 成功识别
5. 如果多个标签碰撞 → 读写器调大 Q 值重试
6. 如果没有标签回复 → 读写器调小 Q 值

**性能**：理论最大吞吐率 ~36.8%（1/e），实际系统可达 200-1000 tags/sec。

### 2.4 Gen2v2 安全增强（2024 现状）

| 安全特性 | Gen2 (2008) | Gen2v2 (2015+) | 2024 实际部署 |
|----------|-------------|----------------|---------------|
| 密码保护 | 32-bit 明文 | 支持加密 | 广泛 |
| 认证 | 无 | 挑战-响应 | 有限 |
| 加密通信 | 无 | AES-128 | 高端标签 |
| 防克隆 | 无 | 加密 TID | 奢侈品/药品 |
| 隐私保护 | Kill 命令 | Untraceable | 试点 |

## 3. Beyond-ID：RFID 作为传感器

### 3.1 从识别到感知的范式转变

传统 RFID 只回答"这是什么？"（ID）。2015 年以来，研究者发现 RFID 信号本身携带了丰富的环境信息——信号强度（RSSI）、相位、多普勒频移等都会受环境影响。这催生了"RFID 感知"（RFID Sensing）新范式。

### 3.2 感知维度

| 感知类型 | 原理 | 精度 | 应用 |
|----------|------|------|------|
| 温度 | 芯片阻抗随温度变化 | ±0.5°C | 冷链监控 |
| 湿度 | 天线介电常数变化 | ±3% RH | 建筑结构监测 |
| 应变/形变 | 天线几何变化→频率偏移 | ±50 με | 桥梁健康监测 |
| 位移/定位 | 相位差/到达角 | 1-10 cm | 室内定位 |
| 手势识别 | 多普勒频移+相位变化 | 95%+ | 智能家居交互 |
| 呼吸/心率 | 胸腔运动→相位调制 | ±1 bpm | 非接触健康监测 |
| 材料识别 | 频谱指纹 | 90%+ | 垃圾分类 |

### 3.3 关键技术：相位感知

UHF RFID 信号的相位（phase）对距离极其敏感：

```
相位 θ = (4π × d / λ) mod 2π

其中：d = 读写器到标签距离
      λ = 波长（UHF ~33 cm）
      相位分辨率 ~0.1°→ 距离分辨率 ~0.05 mm
```

这意味着亚毫米级的运动都能被检测到——比如呼吸时胸腔的起伏（~5 mm）。

### 3.4 代表性研究（2024-2025）

**TagFi（清华大学, 2024）**：
- 用商用 RFID 标签阵列实现 Wi-Fi 级室内定位
- 精度：中位误差 8.3 cm（vs Wi-Fi 的 1-3 m）
- 无需额外基础设施，利用已有零售标签

**RF-Vital（MIT, 2024）**：
- 非接触式生命体征监测
- 距离：最远 8 m
- 同时监测多人呼吸和心率
- 精度：心率误差 < 1.2 bpm

## 4. 行业应用深度分析

### 4.1 供应链与物流

**RAIN RFID 联盟 2024 数据**：
- 全球年出货量：超过 400 亿枚 UHF 标签（2024）
- 年增长率：~25%
- 主要驱动：零售库存可见性、航空行李追踪

**沃尔玛案例**：
- 2022 年起要求所有供应商在商品级别贴 RFID 标签
- 库存准确率：从 65% 提升到 98%
- 缺货率降低 30%
- ROI：每店每年节省 $200K+ 人工盘点成本

**Zara/Inditex 案例**：
- 全球 5000+ 门店，每件衣服从工厂到门店全程 RFID 追踪
- 盘点时间：从 40 小时/店 → 2 小时/店
- 补货响应时间：从 24 小时 → 2 小时

### 4.2 零售新体验

**Amazon Just Walk Out（2024 更新）**：
- 结合 RFID + 计算机视觉 + 传感器融合
- 2024 年 Amazon 宣布缩减纯视觉方案，转向 RFID 为主的方案（成本更低）
- 每个货架配备 RFID 天线，实时检测商品拿取/放回

**无人零售标签技术对比**：

| 技术 | 识别精度 | 成本/件 | 部署复杂度 | 适用品类 |
|------|----------|---------|-----------|----------|
| UHF RFID | 99.5%+ | $0.03-0.08 | 中 | 服装、日用品 |
| 计算机视觉 | 95-98% | $0（摄像头分摊） | 高 | 生鲜、散装 |
| 重量传感器 | 99%+ | $50/货位 | 低 | 标准重量商品 |
| 条形码扫描 | 99.9% | $0.01 | 低（需人工） | 所有 |

### 4.3 医疗健康

**手术器械追踪**：
- 每年全球发生 ~1500 起手术器械遗留体内事件
- RFID 标签贴在每件器械上，术前术后自动清点
- 系统：Haldor（2024）报告 99.97% 检测率

**药品防伪**：
- EU FMD（伪造药品指令）要求处方药序列化
- 中国药品电子监管码（2024 升级）：UHF RFID 替代二维码
- 每盒药品唯一 EPC，全链路可追溯

**血液管理**：
- 血袋 RFID 标签：温度监测 + 身份识别 + 有效期管理
- 减少 30% 的血液浪费（过期/温度异常）

### 4.4 工业制造

**汽车产线**：
- 每辆车从白车身到总装，经过 200+ 工位
- 耐高温 RFID 标签（200°C+）嵌入车身
- 实现混线生产：同一产线同时生产不同配置的车型

**航空 MRO（维修维护）**：
- 波音/空客要求关键零部件 RFID 标记
- 零部件生命周期追踪：安装→使用→维修→报废
- 减少 40% 的零部件查找时间

## 5. NFC：RFID 的近场子集

### 5.1 NFC vs RFID

| 维度 | NFC | 通用 HF RFID | UHF RFID |
|------|-----|-------------|----------|
| 频率 | 13.56 MHz | 13.56 MHz | 860-960 MHz |
| 读距 | < 4 cm | < 1 m | 1-12 m |
| 通信模式 | P2P + 读写 + 卡模拟 | 读写器→标签 | 读写器→标签 |
| 数据率 | 106-424 kbps | 25-100 kbps | 40-640 kbps |
| 安全性 | 高（短距+加密） | 中 | 低-中 |
| 手机支持 | 几乎所有智能手机 | 需专用读写器 | 需专用读写器 |

### 5.2 NFC 2024-2025 新趋势

**NFC Forum 2024 规范更新**：
- Multi-Purpose Tap：一次触碰完成支付+积分+优惠券
- Wireless Charging（WLC）：NFC 标签为小型设备充电（最大 1W）
- 超宽带（UWB）+ NFC 融合：NFC 配对 + UWB 精确测距

**应用爆发点**：
- 数字车钥匙（CCC 标准）：iPhone/Android 手机开车门
- 电子名片（vCard over NFC）：碰一碰交换联系方式
- 产品认证：奢侈品/球鞋 NFC 芯片防伪（Nike、LVMH）

## 6. 技术挑战与前沿

### 6.1 当前瓶颈

| 挑战 | 现状 | 研究方向 |
|------|------|----------|
| 液体/金属干扰 | UHF 信号被液体吸收、金属反射 | 特殊天线设计（贴片天线、AMC 结构） |
| 多标签碰撞 | 密集场景（>1000 tags/m²）识别率下降 | 物理层 NOMA、编码防碰撞 |
| 定位精度 | 商用系统 ~30 cm | 相位差测距、MIMO 天线阵列 |
| 安全隐私 | 标签可被远距离扫描 | PUF 认证、距离限定协议 |
| 环境鲁棒性 | 温度/湿度影响读距 | 自适应功率控制、ML 补偿 |

### 6.2 前沿研究方向（2024-2025）

**芯片级创新**：
- 计算型 RFID（CRFID）：标签内置 MCU，可执行简单计算（WISP 6.0）
- 传感器集成：单芯片集成 RFID + 温度 + 湿度 + 加速度（ams OSRAM SL900A）
- 柔性/可印刷标签：卷对卷印刷，成本降至 $0.01 以下

**系统级创新**：
- RFID + 机器学习：用 RSSI/相位时序数据做活动识别（准确率 > 95%）
- 大规模 RFID 物联网：百万级标签网络的管理与数据分析
- 数字孪生集成：RFID 数据实时驱动仓库/工厂数字孪生

**频谱共享**：
- 5G NR + RFID 共存：在同一频段协调蜂窝通信和 RFID 读取
- 认知 RFID：动态频谱接入，避免干扰
