# BLE模组硬件设计：天线匹配与射频前端
> **难度**：🟡 中级 | **领域**：BLE硬件设计 | **阅读时间**：约 20 分钟

## 引言

想象你要开一家广播电台:电台的功率再大，如果天线没调好，信号也传不出去，听众收到的只是噪音。BLE模组的硬件设计也是同样的道理--2.4GHz的射频信号对走线长度、阻抗匹配、地平面完整性极度敏感。一段看似普通的PCB走线，在2.4GHz下就是一根传输线，稍有不慎就会导致信号反射、驻波比恶化，通信距离从30米缩到3米。

## 1. BLE射频架构

### 1.1 2.4GHz ISM频段

BLE工作在全球通用的2.4GHz ISM频段:

- 频率范围: 2.400GHz - 2.4835GHz，40个信道，间隔2MHz
- 调制方式: GFSK(高斯频移键控)
- 数据速率: 1Mbps(BLE 4.x) / 2Mbps(BLE 5.0)

| 信道类型 | 信道编号 | 用途 |
|----------|----------|------|
| 广播信道 | 37, 38, 39 | 广播、扫描、连接发起 |
| 数据信道 | 0-36 | 连接后数据传输(含自适应跳频) |

### 1.2 射频收发链路

```
发送: 数字基带 --> DAC --> 低通滤波 --> 调制器 --> PA --> 匹配网络 --> 天线
接收: 天线 --> 匹配网络 --> LNA --> 解调器 --> ADC --> 数字基带
```

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 发射功率 | 0 ~ +4 dBm | Class 2 (10m) |
| 接收灵敏度 | -90 ~ -95 dBm | 取决于芯片 |
| 频率误差 | < +/-20 ppm | 由晶振决定 |

## 2. 天线类型与选型

### 2.1 PCB走线天线

最常见的BLE天线方案，直接在PCB上蚀刻出天线形状(如倒F天线PIFA):

优势: 零成本、易于生产一致性、带宽较宽
劣势: 占用PCB面积(约30mm x 8mm)、受周围元件影响大

### 2.2 芯片天线

小型化陶瓷芯片天线，适合空间受限场景:

| 尺寸 | 增益 | 典型型号 |
|------|------|----------|
| 2.0x1.2mm | -5 ~ 0 dBi | Johanson 2450AT18A100 |
| 3.2x1.6mm | -2 ~ +1 dBi | Yageo ANT016008LCD24 |

优势: 体积小、有成熟参考设计
劣势: 增益通常低于PCB天线、需额外BOM成本

### 2.3 外接天线与选型建议

通过IPEX/U.FL连接器外接天线: 蘑菇头(2-5dBi)、FPC(薄型柔性)、外置偶极子(增益最高)。

```
空间充足 + 成本敏感 --> PCB走线天线
空间受限 + 中等性能 --> 芯片天线
距离优先 + 有空间   --> 外接天线
```

## 3. 天线匹配网络

### 3.1 为什么需要匹配

天线阻抗通常不是纯50欧姆，需要匹配网络变换:

- 不匹配的后果: 信号反射 --> VSWR增大 --> 辐射效率下降
- 目标: VSWR < 2.0，回波损耗(S11) < -10dB

### 3.2 Pi型与L型匹配网络

**Pi型**(BLE最常用):

```
        C1
   +---||---+
   |        |
--+-+     +-+--
  C2      L1     --> 天线
--+-+     +-+--
   |        |
  GND      GND
```

Pi网络可同时调整实部和虚部，兼具高通滤波特性抑制低频干扰。

**L型**(最简单，仅两个元件):

| 拓扑 | 串联 | 并联 | 适用场景 |
|------|------|------|----------|
| L型(上) | 电感 | 电容 | 天线阻抗 > 50欧姆 |
| L型(下) | 电容 | 电感 | 天线阻抗 < 50欧姆 |

### 3.3 阻抗计算实例

假设天线阻抗 Z_ant = 30 + j25 欧姆，需匹配到50欧姆:

```
1. 计算导纳: Y_ant = 1 / (30 + j25) = 0.0197 - j0.0164 S
2. 并联元件抵消虚部，串联元件调整实部
3. 选值: C1 = 1.2pF(并联), L1 = 2.7nH(串联)
4. 验证: S11 @ 2.44GHz < -15dB 则匹配良好
```

## 4. Smith圆图基础

Smith圆图是射频工程师的"地图":

- 圆图中心 = 完美匹配(50欧姆)
- 实部: 水平方向，左小右大
- 虚部: 上半圆为感性，下半圆为容性

匹配操作在圆图上的移动:

| 操作 | 圆图轨迹 |
|------|----------|
| 串联电感 | 沿等电阻圆顺时针 |
| 串联电容 | 沿等电阻圆逆时针 |
| 并联电感 | 沿等电导圆逆时针 |
| 并联电容 | 沿等电导圆顺时针 |

工具推荐: SimSmith(免费交互式)、ADS/Microwave Office(专业仿真)、NanoVNA(低成本实测必备)。

```
NanoVNA测量S11:
1. 校准(开路/短路/负载)
2. 连接天线到CH0
3. 设置频率 2.4GHz - 2.5GHz
4. 读取S11曲线，找到谐振点
5. 调整匹配元件直到S11 < -10dB
```

## 5. RF布局规范

### 5.1 地平面与禁布区

- 天线下方必须有完整地平面，不要有长缝隙
- 天线周围设置禁布区: 天线上方无覆铜(顶层)，天线下方地铜镂空(底层，至少5mm)，禁布区内无走线无过孔

### 5.2 过孔缝合与走线规则

射频地平面通过密集过孔连接各层(推荐间距2-3mm，2.4GHz波长1/20约6.25mm)。

| 规则 | 参数 | 原因 |
|------|------|------|
| 阻抗 | 50欧姆 | 匹配传输线 |
| 走线最短 | < 10mm | 减少损耗和寄生 |
| 避免直角 | 圆弧或45度拐角 | 减少反射 |
| 远离数字信号 | > 3mm间距 | 避免耦合干扰 |

50欧姆走线宽度(FR4, 4.4): 1.6mm板厚约3.0mm，0.8mm板厚约1.5mm，共面波导约0.5mm。

## 6. 晶振选择

BLE芯片通常需要两个晶振:

| 晶振 | 频率 | 用途 | 精度要求 |
|------|------|------|----------|
| 高频晶振 | 32MHz | 射频合成器参考频率 | < +/-20 ppm |
| 低频晶振 | 32.768kHz | 睡眠定时、协议时序 | < +/-20 ppm |

```c
// 32MHz晶振选型参数
// 负载电容计算:
// C_load = (C1 * C2) / (C1 + C2) + C_stray
// C_stray 约 2-5 pF (PCB寄生)
// 例: C1 = C2 = 12pF, C_stray = 3pF
//     C_load = (12*12)/(12+12) + 3 = 6 + 3 = 9pF
```

晶振布局: 尽量靠近芯片引脚，负载电容接地端直连地平面，走线远离射频和高速数字信号，晶振下方不走其他信号线。

## 7. 电源滤波设计

BLE射频部分对电源纹波极度敏感，纹波会直接调制到射频载波上:

```
VDD_RF ---[10uH电感]---+---[100pF]---GND
                        +---[10nF]---GND
                        +--- VDD_RF (芯片射频供电引脚)

要点: 电感抑制低频纹波, 100pF去耦2.4GHz, 10nF去耦低频
      电感SRF(自谐振频率) > 2.4GHz
```

| 供电引脚 | 滤波要求 | 典型器件 |
|----------|----------|----------|
| VDD_RF | 最严格 | 10uH + 100pF + 10nF |
| VDD_LNA | 严格 | 10uH + 100pF |
| VDD_DIG | 中等 | 10nF + 100nF |

## 8. 模组与SoC设计选择

| 特性 | BLE模组 | BLE SoC |
|------|---------|---------|
| 射频设计难度 | 无(已集成) | 高(需自行设计) |
| 认证 | 模组已通过 | 需自行认证 |
| BOM成本 | 较高 | 较低 |
| 灵活性 | 受限 | 完全自由 |
| 适合产量 | < 100K | > 100K |

```
选型决策:
  需要FCC/CE认证?
    |-- 否 --> SoC(省钱)
    |-- 是 --> 预算充足?
                |-- 是 --> 模组(快速上市)
                |-- 否 --> 有射频经验?
                            |-- 否 --> 模组(避免踩坑)
                            |-- 是 --> SoC(长期更省)
```

模组推荐: E73-2G4M08S1C(nRF52832, 15x11mm)、CC2640R2F模组(7x7mm, 超低功耗)、CH582F(国产方案)。

## 9. 认证考量

| 认证 | 地区 | 关键测试项 |
|------|------|------------|
| FCC Part 15 | 美国 | 传导发射、辐射发射、杂散 |
| CE RED | 欧洲 | 发射功率、杂散发射、频谱占用 |
| SRRC | 中国 | 等效全向辐射功率、杂散 |

常见失败项: 发射功率超标、杂散发射(晶振谐波/PA非线性)、频偏超限(负载电容不正确)。

降低认证风险: 使用已认证模组、走线严格按参考设计、预留pi型匹配网络位置(0402封装)、发射功率预留3dB余量。

## 10. 参考设计

### 10.1 nRF52832参考电路

```
DEC1 ---[1nF]--- GND
DEC2 ---[1nF]--- GND
ANT ---[C1 1.5pF]---+---[L1 3.3nH]---+--- 天线
                     |                  |
                   [C2 1.0pF]          GND
                     |
                    GND
注: C1/C2/L1需根据PCB和天线调整
```

### 10.2 CC2640R2F参考设计

CC2640R2F集成巴伦(balun)，差分射频输出，外部仅需1个匹配电感+天线匹配网络+32MHz晶振。超低功耗: RX 5.9mA。

## 11. 生产测试

### 11.1 传导与辐射测试

传导测试: 同轴线缆连接模组射频输出到频谱仪，连续发射模式下测发射功率、频偏、杂散，用信号源测接收灵敏度。

辐射测试: 在屏蔽暗室测EIRP、天线方向图、不同方向通信距离。产线简化方案: 耦合板+频谱仪。

### 11.2 产线测试代码

```c
void production_test(void)
{
    if (!ble_chip_verify_id()) { set_result(FAIL); return; }

    // 晶振频率校准
    int32_t freq_offset = ble_measure_freq_offset();
    if (abs(freq_offset) > 20) { // ppm
        ble_write_freq_compensation(freq_offset);
    }

    // 发射功率测试
    ble_start_cw_tx(2440); // ch20, 2.44GHz
    delay_ms(50);
    ble_stop_tx();

    // 接收测试
    ble_start_rx(2440);
    delay_ms(50);
    ble_stop_rx();

    if (freq_ok && power_ok) set_result(PASS);
}
```

## 12. 常见RF问题与调试

| 症状 | 可能原因 | 排查方法 |
|------|----------|----------|
| 距离极短(<1m) | 天线未接/匹配严重失配 | 目检天线，测S11 |
| 距离偏短(3-5m) | 匹配不够好/地平面不完整 | 微调匹配元件 |
| 频偏大 | 晶振负载电容不对 | 测频偏，调电容 |
| 杂散超标 | PA偏置不对/电源滤波差 | 查电源纹波，加去耦 |
| 接收灵敏度差 | LNA输入匹配差/干扰 | 检查接收通路匹配 |

```
调试流程:
  能否扫描到设备? --否--> 检查天线焊接/匹配网络/晶振起振/供电
      |
     是
      |
  连接是否稳定? --否--> 检查频偏/发射功率/干扰源
      |
     是
      |
  距离是否够? --否--> 微调匹配，检查地平面
```

关键测试设备: NanoVNA(~500元, 测S11/S21)、频谱仪(~3000元二手, 测功率杂散)、nRF52840DK(~300元, 协议分析)。

## 总结

1. **天线是关键**: 选对天线类型，做好匹配，通信距离翻倍
2. **匹配不可省略**: 即使0.5dB的损耗也意味着约10%的距离损失
3. **Smith圆图是工具**: 不是玄学，学会用它做匹配设计
4. **地平面是基础**: 没有好的地平面，一切匹配都白费
5. **晶振决定频偏**: 精度不达标，协议都跑不通
6. **电源要干净**: 纹波会直接污染射频信号
7. **模组降低门槛**: 没有射频经验时优先选择模组方案
8. **认证要前置**: 设计之初就考虑认证要求，后期改版成本很高
9. **生产测试必不可少**: 产线频偏校准和功率测试保证一致性
