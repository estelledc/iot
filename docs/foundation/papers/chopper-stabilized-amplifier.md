# 斩波稳定放大器消除低频噪声与失调
> **难度**：🔴 高级 | **领域**：精密放大器技术 | **阅读时间**：约 22 分钟

## 引言

普通运放就像一个略有近视的测量员,低频时总有一个固定的"视差"(失调电压),还伴随着越来越粗的"手抖"(1/f噪声)。斩波稳定技术就像给测量员装了一个高频振动的镜架:让信号在"手抖"最弱的频段被放大,再把信号搬回原位,而失调和低频噪声被抖到了远处,用滤波器轻松滤掉。

## 1 问题: 为什么低频精度受限

### 1.1 失调电压

运放输入端存在的固有电压失调:

```
Vout = Aol * (Vin+ - Vin- + Vos)

Vos: 输入失调电压
  通用运放: 0.1 - 5 mV
  精密运放: 10 - 100 uV
  斩波运放: 0.05 - 5 uV
```

失调电压的影响:
- 高增益电路中失调被放大: 1000倍增益 + 100uV失调 = 100mV输出误差
- 失调随温度漂移: 典型0.1-10 uV/度C

### 1.2 1/f噪声(闪烁噪声)

低频段噪声功率谱密度随频率降低而增大:

```
噪声功率谱密度: S(f) = K / f^alpha  (alpha ~ 1)

电压噪声密度: en(f) = en_corner * sqrt(f_corner / f)

例: en_white = 10 nV/sqrt(Hz), f_corner = 1 kHz
  在 0.1 Hz: en = 10nV * sqrt(1000/0.1) = 1 uV/sqrt(Hz)
  在 10 Hz:  en = 10nV * sqrt(1000/10) = 100 nV/sqrt(Hz)
  在 1 kHz:  en = 10 nV/sqrt(Hz) (等于白噪声)
```

### 1.3 对传感器测量的影响

| 传感器 | 信号幅度 | 频率范围 | 问题 |
|--------|---------|---------|------|
| 热电偶 | 几十uV/度C | DC-1Hz | 失调漂移 > 信号 |
| 应变片 | mV级 | DC-100Hz | 高增益放大,失调被放大 |
| 电流分流 | uV级 | DC-1kHz | 微弱信号被1/f噪声淹没 |
| pH电极 | 59mV/pH | DC-0.1Hz | 长期漂移影响测量 |

## 2 斩波稳定原理

### 2.1 核心思想

将输入信号调制到斩波频率(远高于1/f拐角频率),在高频处放大(没有1/f噪声),再解调回基带:

```
时域过程:

输入信号:      ___/^^^\___        DC低频信号
输入斩波调制:  _|^|_|^|_|^|_      (乘以方波,频谱搬移到fchop)
放大器放大:    _|^|_|^|_|^|_      (在fchop处无1/f噪声)
输出斩波解调:  ___/^^^\___        (乘以同频方波,搬回基带)
失调和1/f:     _|^|_|^|_|^|_     (被斩波调制到fchop)
低通滤波:      ___/^^^\___       (滤除fchop处噪声)
```

### 2.2 频域理解

```
频谱搬移过程:

原始:  信号在DC, 1/f噪声低频大, 失调在DC
调制后: 信号在fchop(此处1/f噪声小), 失调+1/f仍在低频
放大后: 信号在fchop被放大, 失调+1/f也被放大
解调后: 信号回DC, 失调+1/f搬到fchop
滤波后: 保留信号, 滤除fchop处噪声+失调
```

## 3 斩波放大器架构

### 3.1 基本结构

```
           +--CH1--+     +---------+     +--CH2--+     +--LPF--+
  Vin -----| 斩波  |-----| 放大器  |-----| 斩波  |-----| 低通  |----> Vout
           | 开关  |     | A_open  |     | 开关  |     | 滤波  |
           +-------+     +---------+     +-------+     +-------+
               |                              |
               +---------- f_chop ------------+
                (同步时钟, 几kHz到几MHz)
```

CH1(输入斩波): 将信号从基带搬移到f_chop
CH2(输出斩波): 将信号搬回基带,同时将失调和1/f搬移到f_chop

### 3.2 斩波开关实现

```
CMOS开关对:

  Phase 1 (phi1闭合, phi2断开):
    Vin+ --> SW1 --> AMP+     信号直通
    Vin- --> SW2 --> AMP-

  Phase 2 (phi2闭合, phi1断开):
    Vin+ --> SW2 --> AMP-     信号交叉(等效反相)
    Vin- --> SW1 --> AMP+

  效果: 等效于乘以+1/-1方波 = 调制
```

### 3.3 残余失调

理想情况下斩波完全消除失调,但实际存在残余:

**电荷注入**: 开关通断时沟道电荷注入到输入端,差分不匹配 -> 残余失调(优化后nV级)

**时钟馈通**: 栅极耦合时钟馈入输入端,减小方法:
1. 互补开关(PMOS+NMOS): 电荷注入部分抵消
2. Dummy switch: 吸收电荷
3. 全差分结构: 抑制共模电荷注入

### 3.4 斩波纹波

斩波频率处的残余信号出现在输出端:

```
纹波来源:
  1. 放大器自身失调被输出斩波搬移到fchop
  2. 电荷注入不匹配
  3. 热失配引起的低频扰动

典型纹波: 几uV到几百uV
必须加低通滤波器或使用内置纹波消除回路
```

## 4 自动调零技术

### 4.1 原理

自动调零(Auto-zero)是另一种消除失调的方法:

```
Phase A (采样失调):
  输入短接,放大器输出 = A * Vos
  将Vos存储在电容Caz上

Phase B (放大信号):
  接入输入信号,从输出减去存储的Vos
  Vout = A*(Vin + Vos) - A*Vos = A*Vin
```

### 4.2 自动调零的宽频噪声折叠

```
问题: Phase A中采样了瞬时宽带噪声
  采样使宽带噪声折叠到基带
  基带白噪声增大约sqrt(2)倍

影响: 消除了1/f噪声,但白噪声略有增加
```

## 5 斩波与自动调零对比

| 特性 | 斩波 | 自动调零 |
|------|------|---------|
| 工作方式 | 连续时间 | 离散时间(两相切换) |
| 1/f噪声 | 消除(搬移到fchop) | 消除(采样相不含信号) |
| 白噪声 | 不增加 | 增加(噪声折叠~sqrt(2)) |
| 失调消除 | 优(nV级) | 良(uV级) |
| 斩波纹波 | 有(需LPF) | 无(但有过渡毛刺) |
| 带宽 | 受LPF限制 | 受采样率限制 |
| 瞬态响应 | 连续,无切换伪影 | 有切换瞬态 |

混合架构: 斩波+自动调零协同(如OPA388),达到极低噪声和失调。

关键选型参数优先级:
1. 失调电压和漂移(决定DC精度)

## 7 传感器读出应用

### 7.1 热电偶放大

```
K型热电偶: 41 uV/度C
  测量0.1度C -> 需分辨4.1uV
  普通运放漂移1uV/度C -> 1度C变化=1度C误差
  ADA4522漂移12nV/度C -> 10度C变化=0.003度C误差
```

### 7.2 应变片电桥放大

```
桥路输出: 0-10 mV (满量程)
放大增益: 1000x -> 输出0-10V

普通运放: Vos=100uV -> 输出误差100mV (1%FS)
斩波运放: Vos=0.25uV -> 输出误差0.25mV (0.0025%FS)
```

### 7.3 电流分流测量

```
分流电阻: 1 mohm, 电流0-10A, 电压0-10mV
需分辨1mA -> 分辨1uV
斩波运放: Vos=0.25uV -> 可分辨0.25mA
普通运放: Vos=100uV -> 无法使用
```

## 8 实例: 热电偶冷端补偿放大器

### 8.1 系统框图

```
[热电偶] --> [斩波运放ADA4522] --> [24bit ADC] --> [MCU]
                 |
          [冷端温度传感器]
          (NTC或数字温度IC)

MCU: T_hot = T_cold + V_tc / S_tc
```

### 8.2 电路设计

```
冷端补偿电路:

  热电偶(+) ----[R1]----+-------(+) ADA4522
                         |                |
                   [R_ntc]          输出 -> ADC
                         |
  热电偶(-) ----[R2]----+-------(-)

R_ntc变化产生冷端补偿电压
斩波运放确保微伏级信号无漂移放大
```

### 8.3 设计要点

```python
# 热电偶测量系统
class ThermocoupleAmplifier:
    def __init__(self):
        self.vos_max = 0.25e-6     # ADA4522最大失调
        self.vos_drift = 12e-9     # nV/度C漂移
        self.tc_sensitivity = 41e-6 # K型, 41uV/度C

    def calc_error(self, temp_change):
        """计算环境温度变化引入的误差"""
        vos_drift = self.vos_drift * temp_change
        return vos_drift / self.tc_sensitivity

    # 例: 环境变化20度C
    # error = 12nV/度C * 20度C / 41uV/度C = 0.006度C
```

## 9 设计注意事项

### 9.1 输入阻抗调制

斩波开关导致输入阻抗在fchop处周期性变化:

```
高源阻抗信号可能引入误差
解决:
  1. 源阻抗尽量低(<10kohm)
  2. 输入端加RC滤波(截止<fchop)
  3. 选择高输入阻抗型号(CMOS输入)
```

### 9.2 EMI敏感性

斩波频率处的电磁干扰可能被解调回基带:

```
防护:
  1. 输入端加RC低通(截止频率 < fchop)
  2. PCB布局远离RF源
  3. 金属罩屏蔽输入走线
  4. OPA388等内置EMI滤波
```

### 9.3 交调失真

信号频率接近fchop时,|fchop-fsig|可能落在基带内:

```
预防:
  f_signal_max < fchop / 10
  例: fchop=100kHz -> 信号带宽<10kHz
```

### 9.4 何时使用斩波放大器

```
使用场景:
  1. DC到低频精密测量(<10kHz)
  2. 高增益应用(>100x)
  3. 温度变化大的环境
  4. uV级信号,1/f噪声主导

不使用场景:
  1. 高频(>100kHz): 白噪声主导
  2. 交流耦合: DC已阻断
  3. 低增益: 失调影响小
  4. 对纹波/EMI极敏感
```

## 10 与其他低失调技术对比

| 技术 | 最低Vos | 漂移 | 白噪声影响 | 复杂度 |
|------|--------|------|----------|--------|
| 精密双极型 | 10uV | 0.1uV/度C | 无 | 低 |
| 自动调零 | 1uV | 10nV/度C | 增加~sqrt(2) | 中 |
| 斩波稳定 | 0.1uV | 5nV/度C | 无增加 | 中 |
| 斩波+AZ混合 | 0.05uV | 3nV/度C | 轻微增加 | 高 |
| 数字校准 | 1uV(校准后) | 漂移仍存在 | 无 | 低(需MCU) |

数字校准可消除初始Vos,但无法消除温度和时间漂移。斩波稳定是唯一能连续实时消除漂移的技术。

## 总结

斩波稳定技术通过将输入信号调制到高频、在低1/f噪声区域放大、再解调回基带,从根本上消除了运放的失调电压和1/f噪声。这使得uV甚至nV级的DC精密测量成为可能。设计时需注意斩波纹波的滤除、EMI防护和输入阻抗调制等问题。对于热电偶、应变片、电流分流等需要DC精度的传感器读出,斩波放大器是不可或缺的关键器件。
