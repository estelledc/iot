# SAW/BAW射频滤波器在IoT频段选择中的应用
> **难度**：🔴 高级 | **领域**：射频滤波器 | **阅读时间**：约 22 分钟

## 引言

想象你在一个嘈杂的音乐节现场打电话。你只想听到对方说话(目标频段)，但周围有人在唱歌、有人在喊叫(干扰信号)。你的耳朵和大脑会自动过滤掉无关声音，专注于对方的频率特征。射频滤波器做的正是这件事——在众多无线电信号中，只让目标频段通过，把其他频段的干扰挡在门外。但"挡住"这件事的难度取决于干扰离目标有多近：隔壁频道的干扰就像旁边有人低声说话，需要非常精密的"耳朵"(高Q值滤波器)才能分离。SAW和BAW滤波器就是这种精密的频率选择器——前者像水面上的涟漪(声表面波)，后者像深井中的回声(体声波)，用声学谐振实现电信号无法企及的频率选择性。对IoT设备来说，WiFi、BLE、LTE可能挤在同一块电路板上，没有好的滤波器，它们会互相干扰到谁也工作不了。

## 1. 为什么需要射频滤波

### 1.1 共存问题

当多个无线标准工作在同一设备中：

```
[IoT网关内部频谱环境]
2.4GHz:  WiFi(20/40MHz) + BLE(1MHz信道)
  - WiFi发射带外噪声可能落入BLE信道
  - BLE跳频可能跳到WiFi信道边缘

1.8/2.1GHz:  LTE(5-20MHz)
  - LTE谐波可能落入2.4GHz
  - WiFi互调产物可能落入LTE频段
```

没有滤波器，这些信号会在射频前端互相"串门"，轻则灵敏度下降，重则接收机阻塞。

### 1.2 带外抑制与法规合规

接收机需抑制带外强信号防止LNA过载。例如IoT网关中LTE B1下行(-50dBm)与WiFi接收(-80dBm)仅差240MHz，无滤波时WiFi灵敏度恶化20-30dB。

法规也强制要求：FCC Part 15、ETSI EN 300 328、3GPP TS 36.101均对带外和杂散发射有限制，滤波器是合规的必要手段。

## 2. 滤波器类型概览

### 2.1 四类滤波器对比

| 类型 | 频率范围 | Q值 | 插入损耗 | 尺寸 | 成本 |
|------|----------|-----|----------|------|------|
| LC | DC-3GHz | 10-50 | 1-3dB | 大 | 最低 |
| 陶瓷 | 0.5-6GHz | 50-200 | 1-2dB | 中(3x3mm) | 低 |
| SAW | 0.1-2.7GHz | 200-2000 | 1-3dB | 小(2x2mm) | 中 |
| BAW/FBAR | 1.5-6GHz | 500-3000 | 1-2.5dB | 小(1.5x1.5mm) | 高 |

### 2.2 选型决策

```
频率 < 100MHz --> LC滤波器
频率 100MHz-2.5GHz + 成本敏感 --> SAW
频率 1.5-6GHz + 高选择性 --> BAW/FBAR
频率 0.5-3GHz + 宽带 + 低成本 --> 陶瓷
```

## 3. SAW滤波器原理

### 3.1 基本结构

SAW(Surface Acoustic Wave，声表面波)滤波器利用压电基片上传播的声表面波：

```
输入IDT        延迟路径        输出IDT
  |||||                        |||||
  |||||  --> 声表面波 -->      |||||
--|||||------------------------|||||--
  压电基片(LiNbO3 / LiTaO3)

IDT: 叉指换能器
  电信号 --> 压电效应 --> 声表面波 --> 逆压电效应 --> 电信号
```

### 3.2 频率选择机制

IDT指条间距d决定中心频率：f0 = v_SAW / (2 * d)。

例：LiTaO3基片(v=3488m/s)，f0=2.45GHz时d=0.712um。指条对数越多，带宽越窄、选择性越好，但插损越高。

### 3.3 SAW滤波器特性

优势：0.1-2.7GHz选择性极佳；尺寸小(1.4x1.2mm)；成本低(0.2-1元/片)。
劣势：频率上限约2.7GHz；功率承受<+24dBm；温度系数-30~-50ppm/度。

## 4. BAW/FBAR滤波器原理

### 4.1 基本结构

BAW(Bulk Acoustic Wave，体声波)滤波器利用压电薄膜厚度方向振动：

```
FBAR截面:
  顶部电极
  --------- 
  压电层(AlN, 约1-2um)
  ---------
  底部电极
     |
  [空气桥/空腔]  <-- 声波全反射

声波在上下电极间来回反射形成驻波
频率由压电层厚度决定: f0 = v_BAW / (2*t)
```

### 4.2 SMR-BAW vs FBAR

- **FBAR**：底部用空气桥(悬空膜)，Q值极高(>2000)，工艺稍复杂
- **SMR-BAW**：底部用Bragg反射器，结构更牢固，Q值略低但更稳定

### 4.3 BAW滤波器特性

优势：频率1.5-6GHz；Q值极高(500-3000)；插损低(1-2.5dB)；功率承受达+33dBm；温度系数仅-17~-25ppm/度。
劣势：成本约为SAW的3-5倍；低频(<1.5GHz)不适用；厂商集中(Qorvo/Broadcom)。

BAW的压电层厚度(约1-2um)远大于SAW的指条间距(约0.7um)，因此在更高频率下更容易制造。

## 5. SAW vs BAW对比

| 参数 | SAW | BAW/FBAR |
|------|-----|-----------|
| 频率范围 | 0.1-2.7GHz | 1.5-6GHz |
| Q值 | 200-2000 | 500-3000 |
| 插入损耗 | 1.5-3.0dB | 1.0-2.5dB |
| 带外抑制 | 20-45dB | 30-55dB |
| 功率承受 | <+24dBm | <+33dBm |
| 温度系数 | -30~-50ppm/度 | -17~-25ppm/度 |
| 成本 | 低(0.2-1元) | 高(1-5元) |
| 主要厂商 | Murata, TDK/Epcos | Qorvo, Broadcom, TDK |

选择依据：频率<1.5GHz选SAW；1.5-2.7GHz视成本/性能取舍；>2.7GHz选BAW；功率>+24dBm或温度>85度选BAW。

## 6. 滤波器关键指标

### 6.1 插入损耗(IL)与带宽

IL(dB) = Pin - Pout(通带内)。接收链路中IL直接恶化NF；发射链路中IL直接降低输出功率。

形状因子 = BW20dB / BW3dB，越小选择性越好。SAW约1.5-3.0，BAW约1.2-2.0。

### 6.2 抑制与群时延

抑制(dB) = 带外信号衰减量。近端抑制最难做：SAW约20-35dB，BAW约30-45dB。

群时延波动大导致信号失真，WiFi/LTE要求通带内<20-50ns。SAW约10-30ns，BAW约5-15ns。

## 7. IoT频段的滤波需求

### 7.1 2.4GHz WiFi + BLE

```
WiFi信道1-11: 2412-2472MHz
BLE: 2402-2480MHz (40个2MHz信道)

共存挑战: WiFi发射+20dBm vs BLE接收-90dBm, 差110dB
滤波方案: WiFi TX端加带通抑制谐波，BLE前端加带通抑制WiFi泄漏
实际多依赖时分复用(WiFi和BLE不同时收发)
```

### 7.2 Sub-GHz LoRa

```
中国470-510MHz, 欧洲863-870MHz, 美国902-928MHz

SAW滤波器: Murata SAFEA470/868/915系列
典型指标: IL=2.0dB, BW=2MHz, 抑制>30dB
发射端抑制谐波，接收端抑制其他Sub-GHz信号
```

### 7.3 LTE频段

FDD频段(B1/B3/B5/B8)需要双工器；TDD频段(B38/B41)需要带通滤波器。

B41(2496-2690MHz)与WiFi 2.4GHz紧邻，间隔仅24MHz！必须用BAW滤波器才能提供足够近端抑制。

## 8. 双工器与多工器

### 8.1 双工器

FDD系统中发射和接收同时工作但频率不同，双工器 = TX滤波器 + RX滤波器 + 匹配网络，TX-RX隔离度通常>45dB。

### 8.2 多工器

多频段设备将多个滤波器组合，每个滤波器在其他通带内呈现高阻抗，实现各频段互不干扰。

| 类型 | 厂商 | 频段组合 | 技术 | 参考价 |
|------|------|----------|------|--------|
| 双工器 | Murata | LTE B1 | SAW | 1-2元 |
| 双工器 | Qorvo | LTE B41 | BAW | 3-5元 |
| 三工器 | Skyworks | WiFi 2.4G+5G | BAW | 5-8元 |
| 四工器 | Qorvo | WiFi+LTE B41 | BAW | 8-15元 |

## 9. 滤波器在射频链路中的位置

### 9.1 接收链路

```
方案A: 天线 --> [BPF] --> LNA --> 混频器
  滤波器保护LNA，但IL恶化NF (适合干扰强的共存场景)

方案B: 天线 --> LNA --> [BPF] --> 混频器
  LNA NF不受IL影响 (适合干扰弱、NF优先)

IoT通常选方案A，干扰不可控，LNA过载后果更严重
```

### 9.2 发射链路

```
方案A: PA --> [BPF] --> 天线  (最常见，抑制谐波和带外)
方案B: PA --> [LPF] --> 天线  (只抑制谐波，插损更低)
方案C: 调制器 --> [BPF] --> PA --> 天线  (PA前滤波，但PA还会产生新谐波)
```

## 11. PCB布局设计

### 11.1 布局原则与接地

```
1. 靠近天线端(接收)或PA输出端(发射)
2. 输入输出走线最短，不平行走线(避免直通)
3. 滤波器下方地平面完整
4. GND引脚直接连地平面(不要走线连接)
5. 至少4个GND过孔分布在滤波器周围

接地不良后果: IL增加0.5-2dB, 抑制恶化10-20dB
```

### 11.2 常见布局错误

```
错误1: 输入输出走线靠太近 --> 信号直通耦合
  修复: 分两侧引出，中间有地屏蔽

错误2: 缺少GND过孔 --> 地回路电感大
  修复: 每个GND焊盘至少2个过孔

错误3: 放在PCB边缘 --> 边缘辐射效应改变特性
  修复: 距板边至少3mm

错误4: 电源走线穿过滤波器下方 --> 噪声耦合
  修复: 电源走线绕行
```

## 12. 共存滤波设计

### 12.1 WiFi + BLE共存

```
方案一: 时分复用(最常用)
  WiFi和BLE不同时工作，不需要专门共存滤波器
  代价: BLE延迟增大

方案二: 共存滤波器(理论方案)
  WiFi SAW通带2412-2472MHz, BLE SAW通带2402-2480MHz
  问题: WiFi和BLE频段重叠! SAW无法区分!
  结论: WiFi+BLE共存只能靠时分，不能用滤波器分离
```

### 12.2 WiFi + LTE共存

```
关键冲突: LTE B41(2496-2690MHz) vs WiFi(2412-2472MHz)
频率间隔仅24MHz! 必须用BAW滤波器

WiFi端: BAW带通, 2496MHz处抑制>35dB (SAW做不到)
LTE B41端: BAW双工器, 2484MHz处抑制>40dB

设计方案: [天线] --> [BAW三工器] --> WiFi 2.4GHz / LTE B41
```

### 12.3 三共存(WiFi + BLE + LTE)与检查清单

```
最复杂IoT场景(工业网关):
  LTE FDD: SAW双工器(B1/B3)
  LTE TDD B41: BAW带通
  WiFi 2.4G: BAW带通(抑制LTE B41)
  BLE: 复用WiFi通道，时分工作

共存设计检查清单:
1. 列出所有无线技术的频段和带宽
2. 频率间隔>100MHz: SAW可能满足; <100MHz: 需BAW; <30MHz: 需定制
3. 频段重叠(如WiFi+BLE): 只能时分复用
4. 验证接收端灵敏度恶化<3dB
5. 用网络分析仪实测在板性能
6. 做系统级共存测试(同时开启所有无线)
```

## 总结

1. 射频滤波器的核心任务是在拥挤的频谱环境中保护接收机和满足法规要求，共存、带外抑制和合规是三大驱动力
2. 四类滤波器各有定位：LC低频宽带、陶瓷中等性能、SAW低成本中频、BAW高频高性能
3. SAW利用压电基片表面的声波传播，频率由叉指间距决定，适合2.7GHz以下频段
4. BAW/FBAR利用压电薄膜厚度方向振动，Q值更高、频率更高、温漂更小，但成本也更高
5. 滤波器关键指标包括插入损耗(直接影响NF和输出功率)、带宽、抑制和群时延
6. WiFi与BLE频段重叠无法用滤波器分离，只能时分复用；WiFi与LTE B41需BAW滤波器
7. PCB布局对滤波器性能影响巨大，接地设计和输入输出隔离是重点
8. 多无线共存IoT设备的前端设计需要系统级考量，滤波器选择、布局和共存策略必须同步规划
