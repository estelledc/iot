# 继电器与固态继电器在IoT控制中的选型
> **难度**：🟢 初级 | **领域**：执行器控制 | **阅读时间**：约 18 分钟

## 引言

想象你要用一根细绳拉动一扇沉重的铁门。你不会直接用细绳去拉,而是在门上装一个滑轮机构,细绳只需轻轻一拽,滑轮的杠杆作用就能把门打开。继电器就是这个滑轮:MCU给出一个微弱的3.3V信号,继电器就能接通220V的大电流电路。而固态继电器(SSR)则是没有活动部件的升级版滑轮,没有磨损,开关更快,但有自己的权衡。

在IoT设备中,继电器和SSR是最常见的强电控制元件。本文介绍两者原理、特性对比、驱动方法以及在IoT场景中的选型策略。

## 1 电磁继电器基础

### 1.1 结构与工作原理

电磁继电器的核心部件:

- **线圈(Coil)**:通电产生磁场
- **衔铁(Armature)**:被磁场吸引的铁片
- **触点(Contacts)**:随衔铁动作的电气连接

工作流程:

```
MCU输出高电平 -> 驱动管导通 -> 线圈通电 ->
  磁场吸引衔铁 -> 触点状态改变 -> 负载电路接通/断开
```

### 1.2 触点类型

| 触点类型 | 英文缩写 | 常态 | 动作后 | 典型用途 |
|----------|----------|------|--------|----------|
| 常开 | NO | 断开 | 闭合 | 启动负载 |
| 常闭 | NC | 闭合 | 断开 | 安全断开 |
| 转换 | CO(或SPDT) | NC闭合,NO断开 | NC断开,NO闭合 | 切换电路 |

一个SPDT(单刀双掷)继电器同时具有NO和NC触点,通电时从NC切换到NO。

## 2 继电器关键规格

### 2.1 线圈参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| 线圈电压 | 额定工作电压 | 3V/5V/9V/12V/24V |
| 线圈电流 | 额定电压下的电流 | 15-80mA |
| 线圈电阻 | 决定功耗 | 70-1000Ohm |
| 吸合电压 | 保证动作的最低电压 | 额定电压的70-80% |
| 释放电压 | 保证复位的最小电压 | 额定电压的10-40% |

### 2.2 触点参数

| 参数 | 说明 | 选型注意 |
|------|------|----------|
| 接触电阻 | 触点闭合时的电阻 | 通常<100mOhm,影响发热 |
| 最大切换电压 | 触点能承受的最高电压 | 区分AC和DC |
| 最大切换电流 | 触点能通过的最大电流 | 区分阻性负载和感性负载 |
| 最大切换功率 | 电压和电流的乘积上限 | 不可单独取电压或电流上限 |
| 机械寿命 | 无负载时的动作次数 | 通常1千万到1亿次 |
| 电气寿命 | 额定负载下的动作次数 | 通常10万到50万次 |

### 2.3 时间参数

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 动作时间 | 3-10ms | 线圈通电到触点完全闭合 |
| 释放时间 | 2-5ms | 线圈断电到触点完全断开 |
| 弹跳时间 | 1-5ms | 触点闭合瞬间的机械弹跳 |

弹跳(Bounce)是继电器的固有问题:触点闭合时会弹跳几次才稳定,在信号电路中可能被误判为多次触发。

## 3 固态继电器基础

### 3.1 工作原理

SSR内部结构:

```
输入侧:          输出侧:
MCU信号 -> LED -> 光耦隔离 -> TRIAC/SCR(AC) 或 MOSFET(DC)
                                  |
                              负载电路
```

核心机制:

1. 输入端LED通电发光
2. 光信号穿过隔离层到达检测侧
3. 检测侧光敏元件触发输出开关(TRIAC或MOSFET)
4. 负载电路导通

整个过程没有机械运动,因此没有磨损、弹跳和电弧。

### 3.2 交流SSR(AC输出)

输出开关为TRIAC或反向并联SCR:

- 过零触发型(Zero-cross):在交流电过零点时开通,EMI最小
- 随机触发型(Random-fire):任意时刻开通,可实现相位控制(调光/调功)
- 典型控制:220V AC负载,如加热器、灯具

过零触发的限制:开通延迟最长达半个周期(10ms at 50Hz)。

### 3.3 直流SSR(DC输出)

输出开关为MOSFET:

- 单向或双向MOSFET
- 开关速度快(微秒级)
- 典型控制:太阳能切换、DC负载通断

## 4 继电器与SSR对比

| 对比项 | 电磁继电器 | 固态继电器(SSR) |
|--------|-------------|-----------------|
| 开关速度 | 5-15ms | 1us-10ms(取决于类型) |
| 机械寿命 | 1千万-1亿次 | 无限(无活动部件) |
| 电气寿命 | 10万-50万次 | 无限 |
| 开关噪音 | "嗒"声 | 无声 |
| 电气隔离 | 优秀(物理隔离) | 优秀(光耦隔离) |
| 接触电阻 | <100mOhm | 无(导通压降0.7-1.5V for TRIAC) |
| 导通压降 | 接近0 | TRIAC约1.5V,MOSFET约0.1V |
| 浪涌电流承受 | 好(触点短时过载能力强) | 差(半导体过载能力有限) |
| 体积 | 较大 | 中等(需散热片时更大) |
| 成本 | 低 | 中到高 |
| 漏电流 | 无 | 有(off状态微小漏电流) |
| 输出波形 | 真正通/断 | TRIAC有压降,MOSFET有体二极管 |
| 抗振动 | 差(触点可能抖动) | 好(无活动部件) |

### 选型核心判断

- 需要极低接触电阻或零漏电流 -> 电磁继电器
- 高频开关(>1次/分钟)或长寿命需求 -> SSR
- 强浪涌负载(电机启动等) -> 电磁继电器
- 需要安静运行 -> SSR

## 5 MCU驱动继电器

### 5.1 晶体管/MOSFET驱动

MCU的GPIO无法直接驱动继电器线圈(电流需求15-80mA,超GPIO输出能力):

```
MCU GPIO ---[1kOhm]--- Base(NPN)
                        Collector --- 继电器线圈 --- VCC
                        Emitter --- GND
```

NPN三极管(如S8050)或N-MOSFET(如SI2302)作为开关管:

```c
// 继电器控制
#define RELAY_PIN  GPIO_PIN_0
#define RELAY_PORT GPIOA

void Relay_On(void) {
    HAL_GPIO_WritePin(RELAY_PORT, RELAY_PIN, GPIO_PIN_SET);
}

void Relay_Off(void) {
    HAL_GPIO_WritePin(RELAY_PORT, RELAY_PIN, GPIO_PIN_RESET);
}
```

### 5.2 续流二极管

继电器线圈是感性负载,断电时会产生反向电动势,可能损坏驱动管:

```
              继电器线圈
VCC ----+---- [===] ----+---- Collector(NPN)
        |               |
      [1N4148]         |
        |               |
        +---------------+---- GND
        续流二极管(阴极接VCC)
```

- 二极管必须紧靠线圈安装,走线尽量短
- 选用快恢复二极管(1N4148适合小继电器,1N4007适合大继电器)
- 忽略续流二极管是初学者最常见的硬件错误之一

### 5.3 光耦隔离驱动

当继电器控制强电(220V AC)时,MCU侧应与强电侧隔离:

```
MCU GPIO ---[330Ohm]--- LED(光耦输入)
                        |
                    光耦隔离
                        |
                    光敏三极管 --- 驱动管 --- 继电器线圈
```

推荐光耦:PC817(普通隔离),6N137(高速隔离)。

## 6 MCU驱动SSR

### 6.1 输入电流计算

SSR输入侧是LED,需要限制电流:

```
I_LED = (V_MCU - V_LED) / R_limit
```

典型值:V_LED约1.2V(红外LED),目标电流10-20mA:

```
R_limit = (3.3V - 1.2V) / 15mA = 140Ohm, 取标准值150Ohm
```

### 6.2 过零触发与随机触发选择

| 应用 | 选择 | 原因 |
|------|------|------|
| 加热器通断 | 过零触发 | EMI最小,对电网友好 |
| 电阻性负载简单控制 | 过零触发 | 开关损耗小 |
| 调光/调功(相位控制) | 随机触发 | 可控制导通角 |
| 需要快速响应 | 随机触发 | 过零等待最长10ms |

## 7 吸收电路

### 7.1 RC吸收器

继电器触点切换感性负载(电机、电磁阀)时会产生电弧,加速触点烧蚀:

```
继电器触点 ---+--- 负载(感性)
               |
          [R]---[C]  (RC吸收器)
               |
              GND
```

参数选择(经验值):

- C: 0.1-0.47uF(耐压>2倍峰值电压)
- R: 47-100Ohm(功率1-2W)

### 7.2 压敏电阻(MOV)

用于吸收浪涌电压:

- 并联在触点两端或负载两端
- 标称电压选择略高于正常工作电压峰值
- 例如:220V AC选用470V MOV

## 8 IoT实际应用

### 8.1 智能插座

```
MCU -> 光耦 -> 继电器(5V线圈,10A/250VAC触点)
               |
          220V AC 输出
```

选型要点:

- 继电器触点额定电流>=负载最大电流的1.5倍
- 线圈电压5V(与MCU供电一致)
- 电气寿命>10万次(家庭使用频率低,足够)
- 模块化设计:使用现成继电器模块(含驱动电路和续流二极管)

### 8.2 灌溉阀门

```
MCU -> MOSFET驱动 -> 磁保持继电器(双线圈)
                       |
                   12V DC 电磁阀
```

磁保持(双稳态)继电器的优势:

- 只在切换瞬间消耗电能,保持状态无需电流
- 适合太阳能供电的灌溉系统
- 双线圈型:一个线圈吸合,另一个线圈释放
- 单线圈型:正脉冲吸合,反脉冲释放

### 8.3 加热器控制

```
MCU -> SSR(过零触发) -> 220V AC 加热丝
                          |
                     温度传感器反馈
```

选用SSR而非继电器的原因:

- 加热控制需要频繁通断(PID输出)
- 继电器电气寿命仅10万次,高频开关数月即耗尽
- SSR无磨损,可承受每秒数次开关

SSR散热:

- TRIAC导通压降约1.5V,10A负载时功耗=15W
- 必须加装散热片
- 热阻计算:Rth < (T_jmax - T_amb) / P = (125 - 50) / 15 = 5C/W

## 9 选型指南

### 9.1 根据负载类型

| 负载类型 | 浪涌倍数 | 推荐方案 |
|----------|----------|----------|
| 纯阻性(加热器) | 1x | SSR(频繁)或继电器(偶尔) |
| 钨丝灯 | 10-15x | 继电器(触点耐浪涌) |
| 电机 | 5-8x | 继电器(浪涌能力强) |
| 电磁阀/接触器 | 5-10x | 继电器 + RC吸收器 |
| 变压器 | 15-20x | 继电器(额定电流3倍以上) |
| 电容输入电源 | 20-50x | 继电器(额定电流5倍以上)或限流电阻+继电器 |

### 9.2 根据开关频率

| 频率 | 继电器寿命(10万次) | 推荐 |
|------|---------------------|------|
| 1次/天 | 约274年 | 继电器 |
| 1次/小时 | 约11年 | 继电器或SSR |
| 1次/分钟 | 约69天 | SSR |
| 1次/秒 | 约28小时 | 必须用SSR |

### 9.3 根据功耗预算

- 电磁继电器:线圈持续功耗约0.1-0.5W(非磁保持型)
- 磁保持继电器:保持状态零功耗
- SSR:输入LED功耗约0.02-0.05W,输出导通功耗0.5-15W(取决于电流)

电池供电IoT设备优先选磁保持继电器,降低静态功耗。

## 10 安全考虑

### 10.1 爬电距离与电气间隙

强电(220V AC)与弱电(MCU侧)之间的安全距离:

| 参数 | 最小值(250V AC) |
|------|-----------------|
| 电气间隙 | 3mm(基本绝缘),6mm(加强绝缘) |
| 爬电距离 | 4mm(基本绝缘),8mm(加强绝缘) |

PCB设计要点:

- 强电和弱电区域用开槽隔离(槽宽>=1.5mm)
- 跨越强电弱电的光耦或继电器,其隔离距离必须满足要求
- 不允许强电走线与弱电走线平行走线

### 10.2 熔丝保护

继电器/SSR输出侧必须安装熔丝:

- 熔丝额定电流 >= 最大正常工作电流
- 熔丝额定电流 < 触点/SSR最大电流
- 熔丝分断能力 >= 最大短路电流(220V AC通常选1000A以上)
- 推荐使用慢熔型(容性负载浪涌)

### 10.3 安规认证

IoT产品连接市电时需通过:

- **UL**:北美市场,UL 60730(家电控制)或UL 60950(信息技术设备)
- **CE/LVD**:欧洲市场,EN 60730或EN 62368
- **CCC**:中国市场,GB 14536(家用控制)

关键测试项:耐压测试(1500V AC或3000V AC,1分钟)、接地连续性、温升测试。

## 总结

继电器和SSR在IoT强电控制中各有定位,选型要点:

1. 电磁继电器:低接触电阻、强浪涌承受、低成本;但寿命有限、有噪音
2. SSR:无磨损、高速开关、静音;但有导通压降、漏电流、散热需求
3. MCU驱动继电器必须加续流二极管,这是最容易被遗漏的元件
4. 交流SSR有过零触发和随机触发两种,根据EMI和响应速度需求选择
5. 强电PCB设计必须满足爬电距离和电气间隙要求
6. 熔丝和安规认证是市电连接IoT设备的法律和技术底线
