# 传导EMI滤波器设计：共模/差模抑制
> **难度**：🟡 中级 | **领域**：EMI滤波设计 | **阅读时间**：约 20 分钟

## 引言

想象住在铁路旁的公寓。楼下货运火车经过时,低沉的轰鸣穿透墙壁(差模噪声,沿电源线传播);同时,整栋楼的结构振动让所有房间的窗户一起嗡嗡作响(共模噪声,通过地回路耦合到所有线路)。EMI滤波器的任务就是给公寓装上双层隔音窗和减震地基:差模滤波器隔离线间噪声,共模滤波器消除对地耦合噪声。本文讲解传导EMI的机理、共模与差模噪声的区分、滤波器设计与元件选型,以及PCB布局和迭代调试的完整流程。

## 1 IoT设备中的传导EMI源

### 1.1 开关电源

开关电源(SMPS)是IoT设备中最主要的传导EMI源。Buck/Boost/Flyback拓扑在开关管切换瞬间产生高di/dt和dv/dt:

- 开关频率:100kHz - 2MHz,谐波延伸到30MHz以上
- 开关管漏极的dv/dt通过寄生电容耦合到地,产生共模噪声
- 输入侧脉动电流产生差模噪声

### 1.2 电机驱动与高速数字电路

电机驱动(步进/BLDC,PWM 20-100kHz)和高速数字IC(MCU时钟/DDR/以太网PHY)也是重要噪声源。电机绕组与机壳间10-100pF的寄生电容是共模噪声的主要耦合路径;数字IC的同步开关噪声(SSN)通过电源和信号线向外传导。

## 2 传导发射测量基础

### 2.1 LISN与测量原理

LISN(线路阻抗稳定网络)提供稳定的50Ohm阻抗并将EUT的传导噪声耦合到频谱分析仪:

```
LISN基本结构:

  电网 ---[50uH电感]---+---[1uF]--- EUT电源输入
                         |
                    [1kOhm]---[50Ohm]--- 频谱仪
                         |
                        GND
```

### 2.2 CISPR限值

IoT设备通常按CISPR 32 Class B(住宅环境)认证:

| 频段 | Quasi-peak限值 | Average限值 |
|------|---------------|-------------|
| 150kHz - 500kHz | 66-56 dBuV | 56-46 dBuV |
| 500kHz - 5MHz | 56 dBuV | 46 dBuV |
| 5MHz - 30MHz | 60 dBuV | 50 dBuV |

> Class B比Class A(工业环境)严格约10dB。大多数消费级IoT产品必须满足Class B。

## 3 差模噪声与共模噪声

### 3.1 差模(DM)噪声

差模噪声电流在L和N之间流动,与正常工作电流方向相同:

```
  L (相线) --->  [EUT]  ---> N (中线)
       I_DM -->      <-- I_DM

  噪声在L和N之间形成回路
```

来源:开关电源输入脉动电流、大电容充电峰值、负载突变电流。

### 3.2 共模(CM)噪声

共模噪声电流在L和N上同向流动,通过地线返回:

```
  L (相线) --->  [EUT]  ---> N (中线)
       I_CM -->              I_CM -->
         |                      |
         +-------- GND ---------+
```

来源:开关管漏极与散热器间寄生电容、变压器初/次级间寄生电容、PCB与机壳间寄生电容。

### 3.3 DM与CM的区分方法

| 方法 | 原理 | 适用场景 |
|------|------|----------|
| 电流探头法 | L和N电流相加得CM,相减得DM | 调试阶段 |
| LISN分离法 | 使用DM/CM分离器(如LISN MATE) | 认证测试 |
| 拔Y电容法 | 去掉Y电容后CM噪声明显增大 | 快速判断 |

> 经验法则:5MHz以下差模通常占主导;5MHz以上共模通常占主导。但每个设计必须实测确认。

## 4 差模滤波器设计

### 4.1 差模滤波器元件

- **差模电感**:单独绕制,仅对差模电流呈现阻抗,与X电容构成低通滤波器
- **X电容**(跨接L-N):安全等级X2(250VAC)最常用,容值0.1-1uF,为差模噪声提供低阻抗旁路

### 4.2 差模滤波器拓扑与截止频率

```
单级L型(20dB/decade):          单级Pi型(40dB/decade):

  L ---[L_DM]---+--- N           L ---[X1]---[L_DM]---[X2]--- N
                 |                     |               |
               [X_Cap]                GND             GND
                 |
                GND

截止频率: f_c = 1 / (2 * pi * sqrt(L_DM * C_X))
```

## 5 共模滤波器设计

### 5.1 共模电感(共模扼流圈,CMC)

CMC在同一个磁芯上绕制两组线圈,L线和N线各绕一组:

```
     磁芯
    +-----+
   |||   |||   L线绕组
   |||   |||
   |||   |||   N线绕组
    +-----+

差模电流:磁通抵消,电感极低(仅漏感)
共模电流:磁通叠加,电感量高
```

关键参数:

| 参数 | 含义 | 典型值 |
|------|------|--------|
| 共模阻抗 | 对共模噪声的阻抗 | 100-2000Ohm @100MHz |
| 漏感 | 对差模电流的残余电感 | 1-5%标称电感 |
| 额定电流 | 最大工作电流 | 0.5-10A |
| 寄生电容 | 绕组间电容,限制高频性能 | 1-10pF |

### 5.2 Y电容与漏电流限制

Y电容连接在线-地和中线-地之间,为共模噪声提供低阻抗通路:

```
  L ---+---[CMC]---+--- EUT
       |           |
     [Y1]        [Y2]
       |           |
       +---GND----+
```

Y电容直接连地,工频电压产生漏电流,存在触电风险:

| 设备类型 | 最大漏电流 |
|----------|-----------|
| I类(接地)便携式 | 0.75mA |
| I类(接地)固定式 | 3.5mA |
| II类(双重绝缘) | 0.25mA |

漏电流: I_leak = 2 * pi * f * V * C_Y_total。例如220V/50Hz,总Y电容4700pF时I_leak = 0.325mA,满足I类但不满足II类。Y电容容值的上限由漏电流决定,这是共模滤波设计的主要约束。

## 6 滤波器拓扑与级数

### 6.1 单级与多级

```
单级Pi型(40-50dB衰减):

  L ---[Y]---[CMC]---[Y]--- EUT
       |               |
      [X_Cap]        [X_Cap]
       |               |
      GND             GND

两级级联(60-80dB衰减):

  L --[Y1]-[CMC1]-[Y2]-[X1]-[CMC2]-[Y3]-[X2]-- EUT
```

### 6.2 拓扑选择指南

| 需求 | 拓扑 | 适用场景 |
|------|------|----------|
| 衰减 < 40dB | 单级L型 | 低成本IoT |
| 衰减 40-60dB | 单级Pi/T型 | 消费级IoT |
| 衰减 > 60dB | 两级Pi型 | 工业设备 |
| DM为主 | L型 + X电容 | 开关电源输入 |
| CM为主 | CMC + Y电容 | 高速数字/电机驱动 |

## 7 关键元件选型

### 7.1 共模电感选型

选型优先级:额定电流 > 共模阻抗-频率曲线 > 漏感 > 寄生电容 > DCR

```
共模阻抗-频率典型曲线:

  |Z| (Ohm)
  10000 |          ___
  1000  |       /       \___  寄生电容旁路
  100   |     /
  10    |   /
  1     |_/
         +-----------------------> f
        100kHz  1MHz  10MHz  100MHz
```

> 关注目标频段的阻抗值,而非标称电感量。饱和后电感量急剧下降,额定电流必须留30%余量。

### 7.2 X/Y电容选型

- **X电容**:X2类金属化薄膜电容(0.1-1uF),自愈性好,必须安全认证
- **Y电容**:Y2类陶瓷电容(1-4.7nF),X7R温度稳定性,II类设备总容值不超过2.2nF

## 8 插入损耗与仿真

插入损耗: IL = 20 * log10(V_without / V_with)

对2阶LC滤波器: IL = 10 * log10(1 + (f/f_c)^4)

例如:截止10kHz的2阶滤波器,150kHz处: IL = 10 * log10(1 + 15^4) = 47dB

仿真方法:SPICE(含寄生参数的AC扫频) -> S参数测量(VNA) -> 全波仿真(CST/HFSS,含PCB和机壳)。

## 9 实际设计案例:24V DC输入IoT网关

### 9.1 需求与裸机测量

- 输入:24V DC, 最大2A;噪声源:Buck(400kHz) + 以太网PHY(125MHz)
- 认证:CISPR 32 Class B;约束:II类绝缘,漏电流 < 0.25mA

| 频段 | 超标量 | 主导类型 |
|------|--------|----------|
| 400kHz - 2MHz | +15dB | 差模 |
| 5MHz - 30MHz | +20dB | 共模 |

### 9.2 滤波器设计

```
+24V --+--[CMC 1mH]--+--[L_DM 10uH]--+-- 负载
       |              |                |
     [C_X]          [C_Y]           [C_bulk]
     0.47uF         1nF             100uF
       |              |                |
GND ---+--------------+----------------+

差模截止频率: f_c = 1/(2*pi*sqrt(10uH*0.47uF)) = 73.4kHz
400kHz处衰减: 10*log10(1+(400/73.4)^4) = 29.5dB (需15dB,余量充足)
漏电流: 2*pi*50*24V*1nF = 7.5uA << 0.25mA (合格)
```

### 9.3 优化后结果

| 频段 | 裸机 | 加滤波器 | CISPR限值 | 余量 |
|------|------|----------|-----------|------|
| 400kHz-2MHz | 71dBuV | 45dBuV | 56dBuV | -11dB |
| 5MHz-30MHz | 80dBuV | 42dBuV | 50dBuV | -8dB |

设计通过,余量充足。

## 10 PCB布局要点

### 10.1 输入输出隔离

滤波器输入侧与输出侧必须物理隔离,防止噪声通过PCB耦合绕过滤波器:

- 滤波器放在PCB边缘,靠近电源连接器
- 输入和输出走线不平行走线
- 输入输出间放置接地隔离带

### 10.2 地平面处理

- 输入侧Y电容接"脏地"(噪声地),输出侧电路接"净地"(信号地)
- 两个地通过0Ohm电阻或磁珠单点连接,避免噪声地电流流入净地

```
  +---脏地(噪声地)---+---单点---+---净地(信号地)---+
  | [Y电容] [X电容]  |  0Ohm   |      [IC]       |
  | CMC输入侧        |  磁珠    | CMC输出侧        |
  +------------------+---------+------------------+
```

### 10.3 电容焊盘与过孔

X/Y电容接地焊盘直接打过孔到地平面,至少2个过孔并联,走线尽可能短。

## 11 安全考虑

### 11.1 Y电容放电与X电容等级

断电后Y电容可能残留危险电压,需并联放电电阻。例如C_Y=4.7nF,R=1MOhm,tau=4.7ms,1秒后电压安全。

| X电容等级 | 峰值脉冲电压 | 应用 |
|-----------|-------------|------|
| X1 | 4kV | 工业设备 |
| X2 | 2.5kV | 消费类(最常用) |

X2电容必须通过UL/ENEC/CQC安全认证,不可用普通电容替代。

### 11.2 保险丝与压敏电阻

```
完整AC输入保护链:

  L ---[保险丝]---[MOV]---[EMI滤波器]--- EUT
  N ---+----------[MOV]---[EMI滤波器]--- EUT

  保险丝:额定电流为最大工作电流的1.5-2倍
  MOV:压敏电压高于最大工作电压的1.2-1.5倍
```

## 12 迭代设计流程

传导EMI滤波器设计本质上是迭代过程:

```
 [初始设计] --> [测量] --> [DM/CM分析] --> [针对性改进] --> [再测量]
     ^                                                          |
     |                                                          v
     +---[不通过]<--------[对比CISPR限值]<-------[通过]-->[锁定设计]
```

调试技巧:

1. **先测裸机**:确定噪声基线和主导类型
2. **加Y电容看CM变化**:CM噪声明显下降说明共模判断正确
3. **加X电容看DM变化**:同理验证差模判断
4. **逐个增加元件**:理解每个元件的贡献,不要一次全加
5. **检查布局耦合**:效果远低于理论值时,检查输入输出是否有耦合

常见失误:

- 试图一个元件解决所有问题 -> DM和CM应分别处理
- 忽略高频段 -> >10MHz是难点,CMC寄生电容在高频旁路电感
- 滤波器放板中间 -> 应放电源入口
- 过度设计 -> 6dB余量已足够

## 总结

传导EMI滤波器设计关键要点:

1. **噪声分型**:先区分CM和DM,再针对性设计
2. **滤波元件**:CMC + Y电容抑制共模;差模电感 + X电容抑制差模
3. **安全约束**:Y电容容值受漏电流限制,X/Y电容必须安全认证
4. **布局关键**:输入输出隔离,地平面合理分割,元件短路径接地
5. **迭代闭环**:测量 -> 分析 -> 改进 -> 再测量

EMI滤波器不是"加个电容就行了",而是需要理解噪声机理、安全约束和布局影响的系统工程。掌握DM/CM分离方法是高效设计的前提,也是避免"盲目试错"的关键。
