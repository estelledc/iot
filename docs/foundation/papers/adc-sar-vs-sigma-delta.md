# SAR ADC与Sigma-Delta ADC架构对比与选型

> **难度**：🟡 中级 | **领域**：数据转换器 | **阅读时间**：约 20 分钟

## 引言

想象你去菜市场买菜。SAR ADC 就像一个精明的买家，拿着秤一个一个试:放砝码、偏重了拿掉、偏轻了加一块，N 次尝试后精准称出重量--速度快，但精度受限于砝码的精细程度。Sigma-Delta ADC 则像一个耐心的统计员，不停地快速采样然后取平均，用"量多取胜"把噪声压下去--慢工出细活，但最终结果极其精确。

在 IoT 传感器系统中，ADC 是连接物理世界和数字世界的桥梁。选错架构，轻则精度不够、数据白采，重则功耗超标、电池一月一换。本文对比两大主流架构，帮你建立清晰的选型判断框架。

## 1. ADC 架构全景

### 1.1 四大主流架构

按速度-精度谱系排列:

```
速度高 <------------------------------------------------> 精度高

  Flash          Pipeline          SAR          Sigma-Delta
  (并行比较)      (流水线)         (逐次逼近)      (过采样+噪声整形)
  ~10-bit        ~14-bit          ~18-bit        ~24-bit+
  ~1 GSPS        ~500 MSPS        ~10 MSPS       ~1 MSPS (等效)
  2^N 比较器     多级子ADC        N次比较         过采样+数字滤波
  功耗极高       功耗较高         功耗适中         功耗低(低速时)
```

### 1.2 架构适用场景

| 架构 | 速度 | 精度 | 功耗 | 典型场景 |
|------|------|------|------|----------|
| Flash | 极高 (GSPS级) | 低 (6-10 bit) | 极高 | 示波器、雷达 |
| Pipeline | 高 (100+ MSPS) | 中 (10-14 bit) | 高 | 通信、视频 |
| SAR | 中 (1-10 MSPS) | 中高 (12-18 bit) | 低 | 通用采集、音频 |
| Sigma-Delta | 低 (SPS-KSPS) | 极高 (24+ bit) | 极低 | 称重、温度、DC测量 |

IoT 传感器系统中，99% 的场景在 SAR 和 Sigma-Delta 之间选择。下文聚焦这两者。

## 2. SAR ADC 工作原理

### 2.1 二分搜索:逐次逼近

SAR (Successive Approximation Register) 的核心思想是二分搜索。就像猜数字游戏:目标是 0-255 之间的数，先猜 128，偏大往小找，偏小往大找，8 次必出结果。

```
输入电压 Vin ──> 采样保持 ──> 比较器 ──> SAR逻辑 ──> 数字输出
                              ^           |
                              |           v
                          电容DAC <── 反馈码字
```

转换步骤 (以 4-bit 为例):

1. 采样阶段: 采样开关闭合，电容阵列跟踪输入电压
2. 转换阶段: 开关断开，逐位判定
   - MSB 先试: DAC 输出 Vref/2，比较器判断大小
   - Vin > Vref/2 则 MSB=1，保留 Vref/2 再加 Vref/4
   - Vin < Vref/2 则 MSB=0，减到 Vref/4 再比较
   - 依此类推直到 LSB
3. N 位 ADC 需要 N 个时钟周期完成一次转换

### 2.2 电容 DAC 与关键指标

现代 SAR ADC 几乎都采用电容式 DAC (CDAC):

```
          比较器
            |
    ┌───────+───────┐
    |               |
   C/2             C/2       <-- LSB
   2C              2C        <-- bit[n-1]
   ...             ...
   2^(N-2)C        2^(N-2)C  <-- MSB
    |               |
   Vref           GND (开关选择)
```

电容 DAC 优势: 零静态功耗、天然采样保持、面积随位数线性增长。

| 指标 | 典型值 | 说明 |
|------|--------|------|
| 转换时间 | N 个 SAR 时钟 | 12-bit 需要 12 个时钟 |
| 吞吐率 | 1 ~ 10 MSPS | 受限于比较器建立时间 |
| 分辨率 | 12 ~ 18 bit | 更高位数需校准 |
| 功耗 | 数 uW ~ 数 mW | 与采样率近似线性 |

## 3. Sigma-Delta ADC 工作原理

### 3.1 三大核心机制

Sigma-Delta (也叫 Delta-Sigma) 用三个机制协作实现超高精度:

1. **过采样 (Oversampling)**: 以远高于奈奎斯特率的频率采样，将量化噪声分散到更宽频带
2. **噪声整形 (Noise Shaping)**: 用反馈环路把低频噪声推到高频
3. **数字抽取滤波 (Decimation Filter)**: 低通滤波 + 降采样，滤除高频噪声

三者的关系: 过采样是"稀释"噪声，噪声整形是"驱赶"噪声到高频，抽取滤波是"清除"高频噪声。

### 3.2 一阶调制器结构

```
             +----+      +----+      +----+
  Vin --+--->|积分器|--->|量化器|--->|DAC |--+---> 数字输出 (1-bit流)
        |    +----+      +----+      +----+  |
        |         ^                         |
        |         |       反馈路径           |
        +---------+-------------------------+
```

关键直觉: 当输入为直流时，1-bit 输出流的平均值精确等于输入电压。噪声被积分器反馈推到高频。

### 3.3 过采样与信噪比

过采样率定义: OSR = f_sampling / (2 * f_signal)

| 每翻倍OSR | SNR提升 | 等效位数提升 |
|-----------|---------|-------------|
| 普通过采样 | 3 dB | 0.5 bit |
| 一阶 SD | 9 dB | 1.5 bit |
| 二阶 SD | 15 dB | 2.5 bit |
| M 阶 SD | (6M+3) dB | (M+0.5) bit |

这就是 Sigma-Delta 能用 1-bit 量化器实现 24-bit 精度的原因。

### 3.4 数字抽取滤波器

```
1-bit高速流 --> CIC滤波 --> FIR补偿 --> 降采样 --> N-bit低速数据
(OSR=256)                                    (输出数据率)
```

1. **CIC 滤波器**: 无乘法器，适合高速降采样
2. **FIR 补偿滤波器**: 修正 CIC 通带衰减
3. **降采样**: 从调制器频率降到最终输出数据率

## 4. 分辨率与速度的权衡

```
有效位数 (ENOB)
    ^
    |  Sigma-Delta
    |    *  *  *
    |          *  *
    |              *
    |------------------- SAR
    |                     *  *  *
    |                           *  *
    |                                *
    |  Pipeline
    |    *  *  *  *
    +-----------------------------------> 采样率
```

| 维度 | SAR ADC | Sigma-Delta ADC |
|------|---------|-----------------|
| 有效分辨率 | 12-18 bit | 20-24 bit |
| 最大采样率 | ~10 MSPS | ~4 MSPS (低分辨率) |
| 24-bit 可达速率 | 不适用 | 10 SPS ~ 100 KSPS |
| 16-bit 可达速率 | ~1 MSPS | ~1 MSPS |
| 分辨率提升成本 | 指数级 (电容面积) | 对数级 (增加OSR或阶数) |

## 5. 噪声性能对比

### 5.1 SNR 与 SFDR

理论 SNR = 6.02 * N + 1.76 dB (N = 有效位数)

| 分辨率 | 理论SNR | SAR实际 | SD实际 |
|--------|---------|---------|--------|
| 12-bit | 74 dB | 70-72 dB | - |
| 16-bit | 98 dB | 88-92 dB | 90-95 dB |
| 24-bit | 146 dB | - | 105-120 dB |

注意: 24-bit Sigma-Delta 实际 ENOB 通常为 19-21 bit。

SFDR 对比:
- SAR: 80-100 dB (受电容失配和非线性影响)
- Sigma-Delta: 90-120 dB (1-bit 量化器天然线性，不存在 INL)

### 5.2 噪声频谱特征

```
SAR ADC:           Sigma-Delta ADC:
  |                  |          \
  | 平坦             |           \  (整形后噪声)
  |_______           |____________\_______
  0    fs/2          0          fc    fs/2
                     ^信号带宽^
```

SAR: 量化噪声在奈奎斯特带内均匀分布; SD: 噪声被推向高频，信号带内噪声极低。

## 6. 多路复用能力

### 6.1 SAR: 天然适合多路切换

每次转换独立完成，切换通道只需重新采样:

```
通道1 --> 采样 --> 转换 --> 输出 (1 us)
通道2 --> 采样 --> 转换 --> 输出 (1 us)
通道3 --> 采样 --> 转换 --> 输出 (1 us)
```

切换延迟仅采样保持的建立时间 (数百 ns ~ 数 us)，无通道间串扰。

### 6.2 Sigma-Delta: 多路切换的痛点

切换通道后，数字滤波器需要重新建立:

```
通道1 --> 调制器输出 --> 滤波器建立中... --> 稳定输出
                          ^^^^^^^^^^^^^^^^^
                          这段时间数据无效!
```

100 SPS 输出率切换通道，可能要等 50-100 ms 才稳定。

| 方案 | SAR | Sigma-Delta |
|------|-----|-------------|
| 外部MUX+单ADC | 适合，延迟低 | 不推荐，建立慢 |
| 内置MUX多通道 | 常见，延迟可控 | 可用但需等建立 |
| 多ADC并行 | 不经济 | 高精度场景推荐 |
| 同步采样 | 需多颗ADC | ADS131M04等内置多通道 |

## 7. 延迟对比

**SAR ADC -- 几乎即时**:

```
12-bit SAR @ 1 MSPS: 采样~200ns + 转换1us = 总延迟~1.2us
```

**Sigma-Delta ADC -- 显著群延迟**:

```
24-bit SD @ 60 SPS (OSR=256):
  CIC+FIR群延迟: 40-80 个调制器周期
  总延迟: 2.6 ~ 5.2 ms (正常工作)
  首次有效输出: 50-100 ms (滤波器需填满)
```

延迟影响: SD 的大延迟不适合快速闭环控制; 间歇采样时每次唤醒都需等滤波器建立，增加有效工作时间。

## 8. 功耗对比

### 8.1 功耗构成

- **SAR**: 比较器动态功耗 + CDAC零静态功耗 + 少量逻辑功耗; 总功耗正比采样率
- **SD**: 调制器运放偏置(持续) + 数字滤波器动态 + 参考缓冲器(持续); 含静态功耗分量

### 8.2 功耗数值对比

| 条件 | SAR ADC | Sigma-Delta ADC |
|------|---------|-----------------|
| 1 KSPS @ 16-bit | 1-10 uW | 50-200 uW |
| 100 KSPS @ 16-bit | 0.1-1 mW | 0.5-2 mW |
| 1 MSPS @ 12-bit | 1-10 mW | 不适用 |
| 10 SPS @ 24-bit | 不适用 | 0.5-5 mW |

关键发现: 低速(SPS级)SD 功耗偏高(调制器持续运行); 超低功耗间歇采样 SAR 更优(可完全关断)。

### 8.3 功耗优化策略

- **SAR**: 降采样率(线性降功耗)、采样间关断(零静态)、降低基准电压
- **SD**: 斩波稳定(允许低偏置电流)、间歇模式保持调制器运行(快速恢复)、可编程OSR

## 9. 典型应用场景

### 9.1 SAR ADC 的主场

**振动监测**: 10-50 KSPS/通道、4-8通道同步、16-18 bit。代表: ADS8866, MAX11131

**多通道音频**: 48-96 KSPS、多路切换、16-24 bit。代表: AD7606

**电池监测**: 快速采样多节串联电池、低功耗间歇唤醒。代表: ADS1115

### 9.2 Sigma-Delta ADC 的主场

**温度测量**: 信号< 1 Hz、20+ bit 精度。代表: ADS1256

**称重/力传感器**: 桥式输出uV级、内置PGA、24 bit。代表: HX711, ADS1256

**直流精密测量**: 分流电阻电流采样、电量计。代表: ADS131M04

### 9.3 应用场景速查表

| 应用 | 推荐 | 理由 |
|------|------|------|
| 振动/加速度 | SAR | 高采样率+多通道 |
| 音频采集 | SAR/SD | 看是否需多通道切换 |
| 温度/湿度 | SD | 慢信号+高精度 |
| 称重/压力 | SD | uV级信号+PGA |
| 电流采样 | SD | 直流精密+高分辨率 |
| 电池电压 | SAR | 快速切换+低功耗 |
| ECG/EEG | SD | 微弱信号+高CMRR |
| 电机控制 | SAR | 低延迟闭环 |
| 环境监测 | SD | 慢信号+高精度 |
| 多路扫描 | SAR | 快速通道切换 |

## 11. IoT 传感器选型决策框架

### 11.1 决策流程

```
开始选型
    |
    v
信号带宽 > 10 KHz? ──是──> SAR ADC
    |
    否
    v
多路切换速率 > 100 Hz? ──是──> SAR ADC
    |
    否
    v
需要 20+ bit 精度? ──是──> Sigma-Delta ADC
    |
    否
    v
需要 PGA (uV级信号)? ──是──> Sigma-Delta ADC
    |
    否
    v
超低功耗间歇采样? ──是──> SAR ADC
    |
    否
    v
两种均可，看成本/供应/习惯
```

### 11.2 决策检查清单

- [ ] **输入信号范围**: 差分/单端? 信号幅度在量程内?
- [ ] **基准电压源**: 精度要求? 需要外部基准?
- [ ] **抗混叠滤波**: SAR 需 AA 滤波器; SD 天然过采样要求低
- [ ] **通道数**: 同步采样 vs 顺序扫描?
- [ ] **接口**: I2C (省引脚) vs SPI (高速)? DMA?
- [ ] **封装**: IoT 常选 QFN/WLCSP
- [ ] **软件驱动**: 现成 HAL/驱动? 自研成本?
- [ ] **供货稳定性**: IoT 产品 5-10 年生命周期，避免 EOL

### 11.3 常见选型误区

1. **误区: 24-bit 一定比 16-bit 好**
   - 实际: 24-bit SD 在 100 SPS 时有效精度可能仅 18-20 bit
   - 正确: 看 ENOB 而非标称分辨率

2. **误区: SD 更准所以什么场景都选它**
   - 实际: 多通道切换场景 SD 延迟让系统设计复杂
   - 正确: 按信号特征选，不盲目追高精度

3. **误区: SAR 不需要抗混叠滤波**
   - 实际: SAR 在奈奎斯特率附近采样，AA 滤波器必需
   - 正确: 设计截止频率 < fs/2 的低通滤波器

4. **误区: 采样率越高越好**
   - 实际: 过高采样率浪费功耗和数据处理带宽
   - 正确: 采样率 = 信号最高频率的 5-10 倍即可

## 总结

SAR ADC 和 Sigma-Delta ADC 是两种不同哲学的产物:

- **SAR**: 用最少时钟周期完成转换，快而灵活，适合多通道、高速、低延迟场景
- **Sigma-Delta**: 用时间和计算换精度，慢而精确，适合直流/低速、超高精度场景

| 判断维度 | 选 SAR | 选 Sigma-Delta |
|----------|--------|----------------|
| 信号带宽 | > 10 KHz | < 1 KHz |
| 有效精度需求 | <= 18 bit | >= 20 bit |
| 多路切换频率 | > 100 Hz | < 10 Hz |
| 延迟要求 | < 10 us | 无严格要求 |
| 信号幅度 | mV级以上 | uV级 (需PGA) |
| 间歇采样 | 频繁唤醒关断 | 持续运行 |

记住一句话: **快速多变选SAR，慢而求精选SD**。先用决策流程确定架构，再用检查清单细化参数，最后结合供货和成本做最终选择。
