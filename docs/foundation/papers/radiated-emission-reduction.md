# 辐射发射降低：走线/接地/屏蔽综合策略
> **难度**：🔴 高级 | **领域**：EMC设计实践 | **阅读时间**：约 22 分钟

## 引言

把PCB上的电流回路想象成一把隐形的弓: 电流越大弓弦越紧, 回路面积越大弓身越长, 信号频率越高拉弓的速度越快 -- 这把弓向外辐射的能量就越强。辐射发射的本质就是电流回路充当了天线, 将电能量以电磁波的形式"射"出去。降低辐射发射不是靠一招一式, 而是需要从走线、接地、屏蔽三个维度同时入手, 就像给弓同时减张力、缩短弓身、加隔音罩, 缺一不可。

## 1 辐射发射机理

### 1.1 电流回路即天线

任何载流回路都可以等效为一个小型环形天线。根据电磁场理论, 远场辐射场强 E 与三个因素成正比:

```
E = k * I * A * f^2
```

其中:
- I -- 回路中的电流幅值 (A)
- A -- 回路包围的面积 (m^2)
- f -- 信号频率 (Hz)
- k -- 常数, 与测量距离和媒质有关

关键结论: **频率的平方关系**意味着高频谐波的辐射效率远高于基频。一个50 MHz时钟的5次谐波(250 MHz)辐射强度是基频的25倍。

### 1.2 差模辐射与共模辐射

| 类型 | 机制 | 场强公式特点 | 典型来源 |
|------|------|-------------|----------|
| 差模辐射 | 信号电流与回流构成的环路 | E 正比于 I * A * f^2 | 时钟走线、高速数据线 |
| 共模辐射 | 电缆或结构上的共模电流 | E 正比于 I_cm * L * f | USB线缆、传感器引线 |

共模辐射往往比差模更难处理, 因为共模电流的回流路径不明确, 等效天线长度可能远大于PCB上的小回路。

### 1.3 识别发射源

IoT设备中常见的辐射源:

1. **时钟谐波**: MCU外部晶振(8-48 MHz)的高次谐波可延伸到GHz
2. **开关边沿**: 数字信号快速上升/下降沿产生宽带频谱
3. **电缆辐射**: 外接线缆上的共模电流是最主要的辐射天线
4. **DC-DC开关噪声**: 开关电源的开关频率及谐波
5. **射频谐波**: 无线模块发射频率的2次/3次谐波

## 2 PCB走线策略

### 2.1 最小化回路面积

回路面积是差模辐射的核心变量。实际操作:

- 信号走线紧邻其回流路径(地平面)布线
- 电源走线与地线并行走线, 间距越小回路面积越小
- 避免信号走线绕远路导致回流被迫绕大圈

```
不良走线:                    良好走线:
  Signal ─────┐              Signal ────────────
              │                                  │
  GND    ─────┘              GND ────────────────
  (回路面积大)               (回路面积小, 上下紧贴)
```

### 2.2 信号走在完整地平面上方

高速信号必须走在完整的地平面(或电源平面)相邻层上, 这样回流路径就在正下方, 回路面积最小。

核心规则:
- **不要跨分割平面走线** -- 信号跨越地平面缝隙时, 回流被迫绕行, 回路面积暴增
- 如果必须跨区, 在跨越处放缝合过孔(stitching via)桥接

### 2.3 避免回流路径中断

回流路径中断的典型情况:

1. 地平面开槽(slot) -- 元件安装孔或过孔阵列在地平面上形成缝隙
2. 连接器下方地平面被挖空 -- 导致信号回流路径中断
3. 模拟/数字地分割不当 -- 信号跨越分割线时产生大回路

### 2.4 差分对长度匹配

差分信号对(USB、HDMI、SPI时钟/数据)的长度不匹配会:

- 将差模信号转化为共模信号, 产生共模辐射
- 长度差越大, 共模电流越强

规则: 差分对长度差控制在5 mil以内(高速信号)或50 mil以内(低速信号)。长度补偿应靠近不匹配源头。

## 3 地平面设计

### 3.1 完整地平面

完整的地平面是EMC设计的基础:

- 提供最低阻抗的回流路径
- 屏蔽层间串扰
- 作为参考平面控制阻抗

在IoT四层板中, 典型做法是Layer 2整层作为地平面, 不做任何分割。

### 3.2 高速走线下方不开槽

地平面上的开槽对高速信号的影响:

- 开槽迫使回流绕行, 增加回路面积
- 开槽两侧形成缝隙天线, 向外辐射
- 开槽长度等于1/4波长时谐振, 辐射最强

经验法则: 高速走线下方的地平面至少保持走线宽度3倍范围内无任何开槽。

### 3.3 缝合过孔

缝合过孔(stitching via)用于连接不同层的地平面, 缩短回流路径:

| 位置 | 间距建议 | 目的 |
|------|---------|------|
| 地平面边缘 | 每隔1/20波长 | 防止边缘辐射 |
| 高速走线换层处 | 换层过孔旁边2个 | 为回流提供短路 |
| 连接器焊盘旁 | 每个信号引脚1个 | 缩短信号回流路径 |
| 屏蔽罩四周 | 每隔3-5 mm | 确保屏蔽罩良好接地 |

### 3.4 分割地重新连接

当设计中必须分割模拟地和数字地时:

- 在ADC/DAC下方用窄桥连接两地
- 桥宽约20-50 mil, 仅够回流通过
- 桥的位置确保所有跨越分割的信号都从桥上方经过
- 绝对禁止信号跨越分割而不经过连接桥

## 4 信号完整性对EMI的影响

### 4.1 上升时间与带宽

信号的频谱成分不仅取决于时钟频率, 更取决于边沿的上升时间:

```
信号带宽 f_bw 约等于 0.35 / t_rise
```

其中 t_rise 为上升时间(10%-90%)。

示例: 一个10 MHz时钟, 上升时间1 ns时带宽约350 MHz, 远高于基频。这意味着350 MHz以下的谐波都有显著能量。

### 4.2 串联终端减缓边沿

在信号源端串联一个小电阻(通常22-33 欧姆), 可以:

- 减缓上升时间, 降低高频能量
- 匹配传输线阻抗, 减少反射
- 反射引起的振铃也是辐射源, 匹配后振铃消失

```
MCU输出 ──[R_s 33R]──┬───── 走线 ────── 接收端
                      │
                  走线阻抗 Z_0=50R

选择 R_s 使 R_s + R_out 约等于 Z_0
```

### 4.3 展频时钟 (SSCG)

展频时钟发生器(Spread Spectrum Clock Generator)通过低频调制时钟频率, 将时钟能量分散到一定频带内:

- 中心频率在 +/- delta_f 范围内缓慢摆动
- 频谱峰值降低6-12 dB, 但总能量不变
- 调制方式: 三角波或Hershey Kiss形调制
- 典型展频比: +/- 0.5% 到 +/- 2%

注意事项:
- SSCG不适用于需要精确时序的通信接口(UART波特率、CAN)
- 对USB等有频率容差要求的接口需确认容差范围
- 对MCU内核时钟使用SSCG通常没有问题

## 5 电缆作为天线

### 5.1 共模电流与电缆辐射

外接电缆是IoT设备最高效的辐射天线:

- 1 m长的USB线在150 MHz处, 5 uA共模电流即可导致辐射超标
- 共模电流来源: PCB上地电位不均匀, 连接器两端地电位差驱动共模电流流过线缆

### 5.2 铁氧体磁环

在电缆上套铁氧体磁环是降低共模辐射最快捷的方法:

- 磁环对共模电流呈现高阻抗(50-200 欧姆), 对差模信号几乎无影响
- 阻抗公式: Z = 2 * pi * f * L_cm * mu_r
- 选择磁环: 在目标频率处阻抗 > 100 欧姆

安装方式:
- **整体绕**: 电缆穿过磁环 -- 阻抗最低, 适合空间充裕时多绕几圈
- **多圈绕**: 电缆在磁环上绕N圈 -- 阻抗增大N^2倍
- **板载磁珠**: 在PCB连接器处加铁氧体磁珠, 成本更低

### 5.3 电缆屏蔽与接地

屏蔽电缆的接地方式对EMI至关重要:

| 接地方式 | 优点 | 缺点 | 适用场景 |
|----------|------|------|----------|
| 单端接地 | 低频效果好 | 高频时屏蔽层成为天线 | 音频、低频传感器 |
| 双端接地 | 高频屏蔽好 | 可能形成地环路 | 高速数字信号 |
| 360度连接 | 完整屏蔽, 无缝隙 | 工艺复杂 | 射频电缆、EMI严苛场合 |

对于IoT设备, USB线缆建议在设备端360度接地, 主机端通过磁环接地。

## 6 元件布局策略

### 6.1 噪声元件远离I/O连接器

I/O连接器是辐射的出口, 噪声源越靠近出口, 辐射越强:

- DC-DC芯片、时钟振荡器、高速MCU应尽量远离板边连接器
- 将安静元件(传感器接口、ADC)放在连接器附近
- 将噪声元件放在板中央, 利用PCB地平面的屏蔽效应

### 6.2 时钟源靠近负载

时钟信号是最强的辐射源之一:

- 晶振应尽量靠近MCU时钟引脚
- 缩短时钟走线长度, 减少走线充当辐射天线的面积
- 时钟走线避免经过连接器附近

### 6.3 分区布局

将PCB按功能分区:

```
+-----------------------------------+
|  电源区   |   数字区   |  射频区  |
|  DC-DC    |   MCU      |  BLE模块 |
|  LDO      |   Flash    |  天线    |
|-----------|------------|---------|
|           模拟/传感器区           |
|           ADC, 传感器接口         |
+-----------------------------------+
```

每个区域有独立的去耦和地平面, 信号通过明确的通道跨区。

## 7 层叠设计优化

### 7.1 四层板典型叠层

IoT设备最常用的四层板叠层:

```
Layer 1 (Top):    信号 + 元件
Layer 2 (Inner1): 完整地平面 (GND)
Layer 3 (Inner2): 电源平面 (VCC)
Layer 4 (Bottom): 信号 + 少量元件
```

关键要点:
- Layer 1与Layer 2紧密耦合, 为顶层信号提供紧邻的回流路径
- Layer 2(GND)完整不分割
- Layer 3电源平面提供低阻抗电源分配, 同时作为底层信号的回流参考

### 7.2 六层板叠层

对EMI要求更高的设计:

```
Layer 1: 信号
Layer 2: 地平面
Layer 3: 信号 (高速信号放在内层, 被地平面屏蔽)
Layer 4: 电源平面
Layer 5: 地平面
Layer 6: 信号
```

内层信号(S-G-S-P-G-S)被地平面夹击, 辐射大幅降低。

## 8 实战案例: IoT网关辐射发射整改

### 8.1 问题描述

某IoT网关(基于ESP32 + USB接口 + 以太网)在CE认证辐射发射测试中, 144 MHz和192 MHz频点超标6 dB。

### 8.2 定位过程

1. **近场探头扫描**: 发现MCU附近48 MHz晶振区域辐射最强
2. **频率分析**: 144 MHz = 48 MHz x 3, 192 MHz = 48 MHz x 4, 确认是晶振谐波
3. **电缆排查**: 拔掉USB线后辐射下降约10 dB, 确认USB线是主要辐射天线
4. **共模电流测量**: 电流探头测USB线共模电流约15 uA, 远超5 uA安全线

### 8.3 整改措施

按成本从低到高逐步实施:

| 步骤 | 措施 | 改善量 | 成本 |
|------|------|--------|------|
| 1 | 晶振下方补铺地铜皮, 缩短地线 | -2 dB | 0元 |
| 2 | USB差分对走线加地平面缝合过孔 | -3 dB | 0元 |
| 3 | USB连接器处加共模扼流圈 | -5 dB | 0.3元 |
| 4 | USB线缆加铁氧体磁环 | -4 dB | 0.5元 |

合计改善约14 dB, 余量8 dB, 顺利通过认证。

### 8.4 经验总结

- 48 MHz晶振的谐波是IoT网关的常见问题, 设计之初就应在晶振布局上预防
- 电缆共模辐射是导致超标的首要原因, PCB连接器处加共模扼流圈应成为标配
- 地平面缝合过孔成本为零但效果显著, 应在布局阶段就规划

## 9 测量与迭代

### 9.1 近场探头定位

近场探头是EMI诊断的"听诊器":

- **H场探头**(环形): 检测电流环路(磁场), 靠近走线或过孔时有强响应
- **E场探头**(针形): 检测电压节点(电场), 靠近高dv/dt节点时有强响应

使用方法:
1. 将探头连接到频谱分析仪
2. 设置频段到问题频率附近
3. 在PCB上移动探头, 找到信号最强的位置
4. 记录频率和幅度, 对比整改前后

### 9.2 远场确认

近场探头只能定位源, 不能替代远场测试:

- 近场强度与远场发射之间没有简单换算关系
- 整改后必须用远场测试(暗室或开阔场)确认是否达标
- 预测试留6 dB余量, 抵消场地和线缆差异

### 9.3 迭代流程

```
近场扫描 --> 定位源 --> 分析机制 --> 实施措施
    ^                                         |
    |                                         v
    +------ 验证效果 <--- 远场确认 <----------+
```

每次只改一项措施, 量化每项改善量, 避免引入新问题。

## 总结

辐射发射降低是一个系统性工程, 需要从三个维度同时发力: 走线(最小化回路面积, 信号走在完整地平面上, 差分对长度匹配)、接地(完整地平面, 高速走线不开槽, 缝合过孔, 分割地谨慎连接)、屏蔽(电缆磁环, 连接器共模扼流圈, 屏蔽罩)。信号完整性措施(串联终端减缓边沿、展频时钟)能从源头减少高频能量。实战中, 近场探头定位 + 逐项整改验证是最有效的工作流程。最核心的原则: EMC不是事后补救, 而是从第一版PCB布局就要贯彻的设计理念。
