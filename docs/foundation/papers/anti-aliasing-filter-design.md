# 抗混叠滤波器设计与截止频率选择

> **难度**：🟡 中级 | **领域**：采样系统设计 | **阅读时间**：约 20 分钟

## 引言

想象你用手机拍一个快速旋转的风扇。如果快门速度不够快，照片里风扇叶片看起来是静止的，甚至像在反方向转——这就是"频闪效应"。混叠(aliasing)的本质和它一样：采样速度跟不上信号变化，高频成分在采样后"伪装"成低频，让你无法分辨真假。

抗混叠滤波器就是放在 ADC 前面的一道"安检门"：只有信号真正包含的频率才能通过，高于采样率能处理的高频成分被拦在外面。没有这道门，ADC 采回来的数据就是"掺了假"的，再怎么用数字滤波也救不回来。

本文从奈奎斯特采样定理出发，逐步讲清混叠的成因、抗混叠滤波器的选型与设计，最后给出一个振动传感器的完整设计实例。

## 1. 奈奎斯特-香农采样定理回顾

### 1.1 定理表述

奈奎斯特-香农采样定理(Nyquist-Shannon Sampling Theorem)是所有采样系统的理论基础：

> 若连续信号 x(t) 的最高频率分量为 f_max，则当采样频率 f_s > 2 * f_max 时，采样后的离散序列可以无失真地重建原信号。

其中 2 * f_max 称为奈奎斯特频率 f_Nyquist = 2 * f_max。必须满足 f_s > f_Nyquist（严格大于），才能保证信息不丢失。

### 1.2 从频域理解

时域上的"采样"等于信号乘以冲激串，对应频域上是信号频谱的周期性搬移。当 f_s 足够大时，各次搬移的频谱副本不会重叠，可用低通滤波器取出基带频谱，完美重建原信号。当 f_s 不够大时，频谱副本发生重叠——这就是混叠。

### 1.3 常见误解澄清

| 错误认知 | 正确理解 |
|---------|---------|
| "采样率等于 2 倍最高频率就够了" | 需要**严格大于** 2 倍，且工程上需要远大于 2 倍 |
| "只要信号频率低于 f_s/2 就安全" | 还要确保没有高于 f_s/2 的噪声或干扰混入 |
| "数字滤波能替代抗混叠滤波" | 数字滤波发生在采样之后，对已经混叠的信号无能为力 |

## 2. 混叠现象详解

### 2.1 什么是混叠

混叠是采样过程中高频信号"折叠"到低频区域的现象。一个频率为 f 的正弦信号，以 f_s 采样后，其表观频率为：

```
f_alias = |f - n * f_s|    (n 为整数，取使 f_alias 最小者)
```

任何高于 f_s/2 的频率成分，采样后都会"映射"到 0 ~ f_s/2 区间内，与真实低频信号混在一起，无法区分。

### 2.2 数字实例

假设 ADC 采样率 f_s = 1000 Hz，奈奎斯特频率 f_s/2 = 500 Hz：

| 原始频率 f | 混叠后表观频率 | 说明 |
|-----------|--------------|------|
| 300 Hz | 300 Hz | 低于 f_s/2，无混叠 |
| 600 Hz | 400 Hz | 混叠! 600 Hz 伪装成 400 Hz |
| 900 Hz | 100 Hz | 混叠! 900 Hz 伪装成 100 Hz |
| 1100 Hz | 100 Hz | 与 900 Hz 混叠到同一频率 |

注意 900 Hz 和 1100 Hz 都混叠到 100 Hz——一旦混叠发生，你根本不知道这个 100 Hz 是"真的"还是"假的"。

### 2.3 实际危害

- **振动监测**：电机轴承故障特征频率被高频振动噪声"覆盖"
- **音频采集**：超声波焊接噪声混入可听频段，产生虚假信号
- **电力测量**：开关电源高次谐波混叠到基波附近，功率因数计算错误
- **生物电信号**：肌电高频分量混叠到心电频段，干扰诊断

## 3. 为什么必须加抗混叠滤波器

### 3.1 核心原因：混叠不可逆

一旦高频成分通过采样"折叠"到低频区间，它和真实低频信号在数学上完全等价——没有任何数字算法能把它们分开。

类比：两瓶墨水倒进同一杯水里，你没法再分出哪滴来自哪瓶。唯一办法是在倒之前就分开——这就是抗混叠滤波器的作用。

### 3.2 信号链位置

```
传感器 --> 抗混叠滤波器 --> ADC --> 数字后处理
              ^                    
              |              
         必须在采样之前（模拟域），不可替代
```

### 3.3 反面案例

用 1kSPS 采样率直接采集含 800 Hz 成分的信号：800 Hz > 500 Hz (f_s/2)，必然混叠；混叠频率 = |800 - 1000| = 200 Hz；采样数据中出现一个"200 Hz 信号"，而实际信号中根本没有 200 Hz。任何下游算法都无法识别这个 200 Hz 是伪造的。

## 4. 滤波器阶数与奈奎斯特频率处的衰减

### 4.1 阶数与衰减的关系

n 阶低通滤波器在阻带的衰减斜率为 -20n dB/decade（或 -6n dB/octave）。阶数越高，过渡带越陡，但高阶意味着更多运放、更多元件、更容易振荡。选阶数就是在"够用"和"可控"之间找平衡。

### 4.2 不同阶数衰减对比

假设 f_c = 1 kHz，f_s = 10 kHz，计算 f_s/2 = 5 kHz 处的衰减：

| 阶数 | 衰减斜率 | f_s/2 处衰减 | 所需运放数 |
|------|---------|-------------|-----------|
| 1 阶 | -20 dB/dec | 约 -14 dB | 0 (RC) |
| 2 阶 | -40 dB/dec | 约 -28 dB | 1 |
| 4 阶 | -80 dB/dec | 约 -56 dB | 2 |
| 6 阶 | -120 dB/dec | 约 -84 dB | 3 |
| 8 阶 | -160 dB/dec | 约 -112 dB | 4 |

注：以上为巴特沃斯型近似值。

### 4.3 衰减量需求确定

n 位 ADC 的动态范围约为 6.02n + 1.76 dB。12 位约 74 dB，16 位约 98 dB。若要求混叠残余低于 LSB：

```
所需衰减 >= ADC 动态范围 (dB)
```

12 位 ADC 至少需约 74 dB 衰减。1 阶 RC 只提供 14 dB——远远不够。

## 5. 截止频率的选择

### 5.1 基本原则

截止频率 f_c 的选择需要同时满足两个约束：

1. **信号保真**：f_c >= f_signal（信号最高有效频率），否则有用信号被衰减
2. **混叠抑制**：f_c << f_s/2，给阻带留出足够的过渡带做衰减

两者之间的矛盾是抗混叠设计的核心张力。

### 5.2 过渡带分析

```
过渡带宽度 = f_s/2 - f_c
```

过渡带越宽，滤波器需要做的"工作"越少，阶数可以越低。这就是过采样能放松滤波器要求的原因。

### 5.3 实用选择规则

```
f_c = f_signal (信号最高有效频率)
f_s >= K * f_c    (K = 过采样比)
```

| K 值 | 过渡带宽度 | 所需阶数 | 适用场景 |
|------|-----------|---------|---------|
| 2~3 | 很窄 | 6-8 阶 | 采样率受限、功耗敏感 |
| 4~5 | 适中 | 2-4 阶 | 通用嵌入式系统 |
| 8~16 | 很宽 | 1-2 阶 | Sigma-Delta ADC |
| >32 | 极宽 | 纯 RC | 音频、慢速工业信号 |

### 5.4 通带余量

巴特沃斯滤波器在 f_c 处已有 -3 dB 衰减。实用建议：f_c 取 f_signal 的 1.2~1.5 倍，给通带边缘留余量。

## 6. 过采样降低滤波器要求

### 6.1 原理

过采样(oversampling)是指用远高于 2 倍 f_signal 的采样率采样。采样率越高，f_s/2 离信号频带越远，过渡带越宽，对滤波器陡峭度要求越低。

```
标准采样:  f_s = 2 * f_signal  --> 过渡带 = 0 (不可实现)
适度过采样: f_s = 5 * f_signal --> 过渡带 = 1.5 * f_signal
强过采样:   f_s = 16 * f_signal --> 过渡带 = 7 * f_signal
```

### 6.2 代价与额外收益

代价：ADC 功耗与采样率成正比；数据量增大；需要抽取滤波器。

额外收益：每 4 倍过采样可提升 1 位有效分辨率。

```
ENOB_gain = log2(OSR) / 2
OSR = 4 --> +1 位; OSR = 16 --> +2 位; OSR = 64 --> +3 位
```

用"多一点采样率"换"少几阶模拟滤波器"在很多场景下是合算的：模拟元件更少、PCB 更简单、系统更可靠。

## 7. 无源 RC 滤波器（一阶）

### 7.1 电路与公式

```
Vin ---[R]---+--- Vout (to ADC)
             |
            [C]
             |
            GND

f_c = 1 / (2 * pi * R * C)
|H(f)| = 1 / sqrt(1 + (f / f_c)^2)
```

f = f_c 处增益 -3 dB，超过 f_c 后以 -20 dB/decade 滚降。

### 7.2 优点与局限

优点：2 个元件、无需供电、无噪声、绝对稳定。

局限：一阶滚降太慢；带负载时 f_c 漂移；无增益无法做阻抗变换；12 位及以上 ADC 单独使用几乎不满足要求。

### 7.3 典型参数

以 f_c = 1 kHz 为例：选 C = 100 nF，R = 1.59 kOmega，取标称值 R = 1.6 kOmega，实际 f_c = 995 Hz。

## 8. 有源滤波器（二阶/四阶 Sallen-Key）

### 8.1 为什么需要有源滤波器

级联多个一阶 RC 会因级间负载效应导致截止频率偏移。有源滤波器用运放的缓冲作用隔离各级，确保每级独立工作。

### 8.2 二阶 Sallen-Key 拓扑

等增益（增益 = 1）Sallen-Key 传递函数：

```
H(s) = omega_c^2 / (s^2 + (omega_c / Q) * s + omega_c^2)
```

Q 值决定通带边缘形状：

| Q 值 | 滤波器类型 | 通带特征 |
|------|-----------|---------|
| 0.707 | 巴特沃斯 | 最大平坦，无纹波 |
| 0.577 | 贝塞尔 | 线性相位，阶跃响应好 |
| 1.0~1.3 | 切比雪夫 | 陡峭过渡带，通带有纹波 |

### 8.3 二阶设计公式（等增益巴特沃斯）

```
R1 = R2 = R, C1 = 2 * C2
f_c = 1 / (2 * pi * R * sqrt(C1 * C2))
```

设计步骤：选定 f_c -> 选定 C2 -> C1 = 2*C2 -> 计算 R。

### 8.4 四阶：级联两个二阶

四阶巴特沃斯需要两级不同的 Q 值：第 1 级 Q1 = 0.5412，第 2 级 Q2 = 1.3066。级联后总衰减 -80 dB/decade。

### 8.5 设计注意事项

- 运放 GBW 至少是 f_c 的 100 倍
- 输入偏置电流通过 R 产生直流偏移
- PCB 布局：反相输入端走线尽量短
- 电源去耦电容(100 nF + 10 uF)靠近供电引脚
- Sallen-Key 增益不能超过 3，否则不稳定

## 9. 滤波器类型选择

| 特性 | 巴特沃斯 | 切比雪夫 | 贝塞尔 |
|------|---------|---------|--------|
| 通带平坦度 | 最好 | 有纹波 | 好 |
| 过渡带陡度 | 中等 | 最陡 | 最缓 |
| 相位线性度 | 中等 | 差 | 最好 |
| 阶跃响应 | 中等 | 有振铃 | 无过冲 |
| 适用场景 | 通用采样系统 | 频谱分析 | 时域波形保持 |

抗混叠通常选巴特沃斯：通带内幅度响应均匀，不引入频率相关的增益误差，配合适当过采样比衰减通常足够。信号含瞬态脉冲时考虑贝塞尔（群延迟恒定，脉冲不变形）。

## 10. 元件容差的影响

### 10.1 容差与截止频率偏移

| 元件 | 典型容差 | 成本 |
|------|---------|------|
| 碳膜电阻 | 5% | 低 |
| 金属膜电阻 | 1% | 中 |
| 陶瓷电容 X7R | 10% | 低 |
| 陶瓷电容 C0G/NP0 | 5% | 中 |

截止频率偏差：用 5% 电阻 + 10% 陶瓷电容，四阶滤波器 f_c 偏差可达 15%~20%。

### 10.2 应对策略

1. 选 C0G/NP0 陶瓷电容（温度系数极低，容差 5%）
2. 用 1% 金属膜电阻
3. f_c 设为 f_signal 的 1.3~1.5 倍，即使偏移也不影响有用信号
4. 生产时用已知频率信号源实测 f_c，软件补偿增益偏差
5. 四阶以内容差可控，六阶以上建议用集成滤波器芯片

## 11. 实际设计示例

### 11.1 需求定义

- 传感器：MEMS 加速度计，模拟输出
- 信号带宽：0 ~ 1000 Hz
- ADC：12 位 SAR ADC，采样率可配置
- 目标：混叠残差低于 ADC 的 1 LSB

### 11.2 方案比选

**方案 A**：f_s = 5 kSPS，过渡带 = 1.5 kHz，需 6~8 阶，电路复杂。

**方案 B**：f_s = 10 kSPS，f_c = 1.2 kHz，四阶在 5 kHz 处衰减约 -56 dB，略低于 74 dB 目标。

**方案 C（选定）**：f_s = 12 kSPS，f_c = 1.2 kHz，f_s/2 = 6 kHz。四阶在 6 kHz 处约 -64 dB，加一阶 RC 前置（-12 dB），总计 -76 dB > -74 dB。五阶电路，12 kSPS 在大多数 MCU 上轻松实现。

### 11.3 电路设计

**第一级：RC 前置低通**

```
f_c1 = 2 kHz, C = 100 nF (C0G)
R = 800 Omega (1% 金属膜), 实际 f_c1 = 1990 Hz
此级在 6 kHz 处衰减约 -12 dB
```

**第二级：二阶 Sallen-Key 巴特沃斯 (Q = 0.541)**

```
f_c2 = 1.2 kHz, C2 = 10 nF (C0G)
C1 = 25.2 nF (22 nF + 3.3 nF 并联)
R = 8.2 kOmega (1% 标称值)
```

**第三级：二阶 Sallen-Key 巴特沃斯 (Q = 1.307)**

```
f_c3 = 1.2 kHz, C2 = 10 nF (C0G)
C1 = 54.1 nF (47 nF + 6.8 nF 并联)
R = 5.6 kOmega (1% 标称值)
```

运放：MCP6002（双通道，GBW = 1 MHz，低功耗，轨到轨）。GBW/f_c = 833，远大于 100 倍要求。

### 11.4 验证

f = 6 kHz (f_s/2) 处衰减：RC -12 dB + 二阶#1 -26 dB + 二阶#2 -38 dB = -76 dB > -74 dB。设计满足要求。

### 11.5 信号链汇总

```
MEMS加速度计 --> [RC前置 2kHz] --> [Sallen-Key #1 1.2kHz Q=0.541]
  --> [Sallen-Key #2 1.2kHz Q=1.307] --> [12-bit SAR ADC 12kSPS] --> MCU
```

## 12. 数字后滤波的补充作用

### 12.1 角色定位

抗混叠滤波器保证采样数据不含混叠分量，但模拟滤波器不可能做到"砖墙"特性。数字后滤波进一步清理残余高频。

常用类型：移动平均（最简单，去白噪声）、IIR 低通（计算量小，可能相位失真）、FIR 低通（线性相位，阶数高）。

### 12.2 各级各司其职

```
模拟滤波器: 抑制 f_s/2 以上 --> 防止混叠 (必须)
数字滤波器: 抑制 f_signal ~ f_s/2 --> 提高信噪比 (推荐)
抽取滤波器: 降低有效采样率 --> 减少数据量 (过采样必需)
```

三者不互相替代。数字滤波器截止频率设为 f_signal，比模拟 f_c 更紧。

## 13. Sigma-Delta ADC 的内置滤波

### 13.1 为什么 Sigma-Delta 不需要外置 AA 滤波器

Sigma-Delta ADC 以极高频率（64~256 倍输出数据率）做 1 位量化，反馈环路本身就是噪声整形滤波器，内置数字抽取滤波器同时完成降采样和低通滤波。

调制器采样率极高（如 2.4576 MHz 用于 48 kHz 音频），奈奎斯特频率远在信号带宽之外。一阶 RC 就足以防止极高频噪声混入，内部数字 FIR 提供等效 100+ dB 阻带衰减。

### 13.2 内置滤波特性

| 参数 | 典型值 |
|------|--------|
| 调制器采样率 | 64~256 * f_out |
| 数字滤波器类型 | sinc3/sinc5 或 FIR |
| 阻带衰减 | 80~120 dB |
| 通带纹波 | <0.005 dB |

### 13.3 Sigma-Delta vs SAR + 外置 AA

| 选择因素 | Sigma-Delta | SAR + 外置 AA |
|---------|------------|--------------|
| 信号带宽 | 低频 DC~数kHz | 宽带 数kHz~数MHz |
| 分辨率 | 16~24 位 | 12~16 位 |
| 建立时间 | 长(ms级) | 短(us级) |
| 多路切换 | 不适合 | 适合 |
| 外置 AA | 简单 RC 即可 | 需精心设计 |

对于振动传感器(1 kHz 带宽)这种低频、高精度、不需要多路切换的场景，Sigma-Delta 是更省心的选择——但本示例选择 SAR 是为了完整展示抗混叠设计流程。

## 总结

1. **混叠不可逆**：必须在 ADC 之前用模拟滤波器阻止高频成分，数字域无法补救
2. **截止频率选择**：f_c = (1.2~1.5) * f_signal，给通带边缘留余量
3. **采样率与阶数**：过采样比 K = 8~12 配合四阶滤波器是常见折中
4. **滤波器类型**：巴特沃斯适合大多数抗混叠场景，贝塞尔适合波形保真
5. **元件选择**：C0G 电容 + 1% 金属膜电阻是基本要求
6. **数字后滤波**：是对模拟 AA 滤波器的补充，不是替代
7. **Sigma-Delta ADC**：内置高阶数字滤波，大多数场景只需外置简单 RC

设计口诀：**先定带宽，再选采样率，然后算过渡带够不够做衰减，最后选拓扑画电路。**
