# ADC基准电压源设计与温漂控制

> **难度**：🟡 中级 | **领域**：精密基准设计 | **阅读时间**：约 20 分钟

## 引言

想象你用一把尺子量桌子长度。尺子的刻度准不准,直接决定你量出来的数字有没有意义。如果这把尺子还会随温度热胀冷缩,那冬天量和夏天量就不一样--不是桌子变了,而是尺子变了。ADC的基准电压源就是这把"尺子"。无论你的ADC有多少位分辨率,如果基准在飘,多出来的位数全是噪声。

在IoT传感器系统中,基准电压的精度和稳定性往往比ADC本身的分辨率更关键。一颗24位Sigma-Delta ADC配上漂移50ppm/C的基准,实际有效精度可能还不如16位ADC配3ppm/C的精密基准。本文从基准源原理出发,覆盖选型、设计、布局和温漂控制全链路,帮你在精度和成本之间找到最优解。

## 1 基准电压为何决定ADC精度

### 1.1 ADC的量化基准

ADC将模拟输入电压转换为数字码,转换公式为:

```
D = (Vin / Vref) x (2^N - 1)
```

Vref就是基准电压。如果Vref偏了1%,所有转换结果就偏1%,这是系统级的增益误差,无法通过增加分辨率弥补。

### 1.2 基准误差如何映射到转换结果

以12位ADC为例,LSB大小为Vref/4096。假设Vref=2.5V:

| 基准误差 | 电压偏差 | 等效LSB误差 | 占满量程百分比 |
|---------|---------|------------|-------------|
| 0.1% | 2.5mV | 4.1 LSB | 0.1% |
| 0.5% | 12.5mV | 20.5 LSB | 0.5% |
| 1% | 25mV | 41 LSB | 1% |
| 2% | 50mV | 82 LSB | 2% |

一个初始精度2%的基准,在12位ADC上直接吃掉82个LSB,等效精度只剩不到10位。

### 1.3 温漂的累积效应

温度系数(TC)表示基准电压随温度变化的速率,单位ppm/C。在-40C到+85C的工业温域内,温差125C:

- 50ppm/C: 125 x 50 = 6250ppm = 0.625%
- 10ppm/C: 125 x 10 = 1250ppm = 0.125%
- 3ppm/C: 125 x 3 = 375ppm = 0.0375%

初始精度可校准,温漂只能靠选型或补偿控制。这是基准选型中最关键的指标。

## 2 带隙基准原理

### 2.1 带隙电压的由来

硅的带隙电压约1.12V。带隙基准的核心思想:将负温度系数(CTAT)的Vbe与正温度系数(PTAT)的Vt按恰当比例相加,使总温度系数趋近于零:

```
Vbg = Vbe(CTAT) + K x Vt(PTAT)
```

Vbe约-2mV/C,Vt(=kT/q)约+0.086mV/C。调节K使两者抵消,最终得到与温度几乎无关的电压,约1.2V。

### 2.2 Brokaw带隙单元

经典Brokaw结构由两个发射极面积不同的双极型晶体管和运放反馈环路组成。Q1发射极面积为Q2的n倍,运放维持两路电流相等:

```
delta_Vbe = Vt x ln(n)            (PTAT电压)
Vbg = Vbe + 2 x (R3/R1) x delta_Vbe
```

调节R3/R1的比值即可调整K值,实现零温漂。

### 2.3 不同架构对比

| 架构 | 特点 | 噪声 | 适用场景 |
|------|------|------|---------|
| Brokaw | 经典双极型,结构简单 | 中等 | 通用基准 |
| Widlar | 低压兼容,CMOS友好 | 较高 | 低压MCU内部基准 |
| XFET | 结型场效应管 | 极低 | 精密测量 |
| 掩埋齐纳 | 超低噪声,高压供电 | 最低 | 实验室级仪器 |

## 3 基准源类型:并联与串联

### 3.1 并联(Shunt)基准

并联基准像稳压二极管,与负载并联,需串联限流电阻供电:

```
Vcc ---[Rbias]---+--- Vref
                 |
                [Ref]  (并联基准器件)
                 |
                GND
```

基准器件吸收多余电流维持节点电压恒定。优点:可从任意高于Vref的电源供电,可做负基准,适合电流环。缺点:偏置电阻需精心计算,PSRR较差,空载也耗电。

### 3.2 串联(Series)基准

串联基准像线性稳压器,输入输出串联,内部运放调节调整管维持输出恒定。

优点:静态电流低(可至uA级),PSRR高,负载调整率好,无需计算偏置电阻。缺点:输入电压须高于Vref+dropout,不方便做负基准。

### 3.3 选型判断

| 条件 | 推荐 |
|------|------|
| 低压差、低功耗 | 串联基准 |
| 电源电压变化大 | 串联基准(PSRR好) |
| 需要负基准或浮地 | 并联基准 |
| 4-20mA电流环 | 并联基准(如LM4040) |

通用IoT传感器采集优先串联基准,电流环/隔离侧优先并联基准。

## 4 关键性能指标详解

### 4.1 初始精度

初始精度指25C下输出电压与标称值的偏差。可通过出厂校准、单点校准或自动校准消除。因此初始精度不是最关键指标--温漂才是。

### 4.2 温度系数(TC)

TC是基准选型的核心指标,单位ppm/C:

```
TC = (Vmax - Vmin) / (Tmax - Tmin) / Vnominal x 10^6  (ppm/C)
```

两种计算方式:
- 方框法(Box Method):取全温范围最大/最小值算斜率,数据手册常用,偏保守
- 蝴蝶图法(Butterfly Curve):每个温度点取多器件统计分布,更精确但数据难获取

选型以数据手册TC值为准,留出余量。

### 4.3 长期漂移

长期漂移指基准电压随时间的缓慢变化,单位ppm/kHr(ppm/1000小时):
- 普通基准: 50-100 ppm/kHr
- 精密基准: 10-25 ppm/kHr
- 超精密基准: < 5 ppm/kHr

长期漂移呈对数衰减:前1000小时变化最大,之后趋稳。精密设备出厂前会做老化(Burn-in)处理。

### 4.4 噪声

基准噪声直接叠加在ADC结果上。两种指标形式:
- 宽带噪声: 10Hz-10kHz RMS噪声(uVrms)
- 1/f噪声: 0.1Hz-10Hz峰峰值噪声(uVpp)

| 基准型号 | 0.1-10Hz噪声 | 宽带噪声 |
|---------|-------------|---------|
| REF5025 | 3 uVpp | 12 uVrms |
| ADR4525 | 1.0 uVpp | 4.8 uVrms |
| LM4040 | 23 uVpp | 100 uVrms |
| MCP1501 | 8 uVpp | 30 uVrms |

### 4.5 指标优先级

- 工业测量(宽温): TC > 初始精度 > 长期漂移 > 噪声
- 实验室仪器(恒温): 噪声 > 长期漂移 > TC > 初始精度
- 消费级IoT(成本敏感): 初始精度 > TC > 噪声 > 长期漂移

## 6 基准缓冲器设计

### 6.1 为什么需要缓冲器

精密基准输出驱动能力有限(通常仅几mA),而ADC采样会在VREF引脚产生瞬态电流。若基准直接驱动ADC,瞬态电流导致基准跌落,引入转换误差。

判断方法:若ADC的Iref峰值超过基准的Iout能力,则需要缓冲器。

### 6.2 缓冲器电路

用低噪声精密运放做单位增益跟随器:

```
Vref_in ---[+]----[R1]---+--- Vref_out (to ADC)
             [OPA]-------+
          [--]
           |
          GND
```

R1为隔离电阻(10-100欧),与去耦电容构成RC滤波,同时隔离运放与容性负载防振荡。

运放选型要求:输入偏置电流<1nA,失调电压<50uV,噪声密度<10nV/sqrt(Hz),单位增益稳定。推荐OPA320、ADA4522。

### 6.3 缓冲器噪声贡献

总噪声为基准噪声与运放噪声的RMS和:

```
Vnoise_total = sqrt(Vref_noise^2 + Vopa_noise^2)
```

若基准3uVpp、运放0.5uVpp,则总噪声3.04uVpp,运放贡献可忽略。但若用普通运放(10uVpp以上),运放反而成为噪声主力,得不偿失。

## 7 去耦与滤波

### 7.1 基准引脚去耦

ADC的VREF引脚必须就近放置去耦电容:1uF(X7R) + 0.1uF(C0G)并联。避免Y5V/Z5U介质(温漂和偏压特性差)。

### 7.2 RC低通滤波

基准输出与ADC之间加RC低通,截止频率fc = 1/(2pi x R x C)。R取10-100欧(过大则采样电流产生直流压降),C取1-10uF。例如R=22欧,C=4.7uF,fc=1.5kHz。

### 7.3 多级滤波

24位超低噪声系统可用多级RC:

```
Vref -[R1]-+-[R2]-+-[R3]-+-- ADC VREF
           |      |      |
          C1     C2     C3
          GND    GND    GND

R1=10,C1=10uF(fc1=1.6kHz) -> R2=10,C2=4.7uF -> R3=22,C3=1uF
```

多级滤波显著降低宽带噪声,但每级电阻都引入直流压降,需权衡。

## 8 PCB布局要点

### 8.1 布局原则

1. **基准紧邻ADC**: 缩短Vref走线,减少拾取噪声
2. **独立模拟地平面**: 基准电路地不与数字地混用
3. **远离开关电源**: 至少5mm间距,避免磁场耦合
4. **Vref走线不穿越数字区**: 避免高速数字信号干扰

### 8.2 接地策略

基准的去耦电容接模拟地平面。模拟地和数字地单点连接(星形地)。基准GND引脚直接连模拟地平面,不走细走线。

### 8.3 热布局

- 基准远离LDO、功率MOSFET、DC-DC等发热器件
- 基准下方不要铺铜走大电流
- 高精度系统可在基准周围做热隔离沟槽
- 基准与ADC放在同一热区,温度变化一致可部分抵消温漂

## 9 比例测量:用VDD做基准的取舍

### 9.1 比例测量原理

比例测量指ADC基准与传感器激励共用同一电源,Vdd同时出现在分子和分母中相消,电源波动不影响结果:

```
ADC输出 = (R1/(R1+R2)) x Vdd / Vdd x 2^N = R1/(R1+R2) x 2^N
```

### 9.2 适用与不适用场景

适用: 电阻分压式传感器(热敏电阻、电位器)、惠斯通电桥(激励与基准共用电源)。

不适用: 绝对电压输出传感器(热电偶、PH探头)、需跨系统校准、Vdd噪声大(开关电源)、多通道不同基准需求。

### 9.3 VDD基准精度

- 普通LDO供电: VDD精度1-2%
- 精密LDO: 0.5-1%
- 开关负载时纹波可达50mV以上

结论: 12位及以下ADC在比例测量场景可用VDD;16位以上必须用外部精密基准。

## 10 内部基准与外部基准权衡

### 10.1 内部基准特点

| MCU系列 | 内部基准电压 | 初始精度 | TC |
|---------|------------|---------|-----|
| STM32F4 | 1.21V | 3% | 50 ppm/C |
| STM32L4 | 1.212V | 1.5% | 30 ppm/C |
| ESP32 | 1.1V | 5% | -- |
| nRF52832 | 0.6V | 2% | 40 ppm/C |

内部基准优点: 零BOM成本、零PCB面积、无外部布线噪声风险。缺点: 精度差(1-5%)、温漂大(30-50ppm/C)、受MCU发热影响、无法驱动外部电路。

### 10.2 决策流程

```
ADC <= 10位 + 精度 <= 2%  --> 内部基准即可
ADC = 12位 + 比例测量     --> VDD基准
ADC = 12位 + 绝对测量     --> MCP1501级外部基准
ADC >= 16位              --> 必须外部精密基准(REF5025级)
ADC >= 24位              --> 超精密基准(ADR4525级)
```

### 10.3 混合策略

实际系统可混合使用: 高精度通道用外部精密基准,低精度通道用内部基准,比例测量通道用VDD。通过模拟开关(如TMUX1111)在不同基准间切换复用。

## 11 温度补偿技术

### 11.1 软件补偿原理

若温漂曲线已知,通过测量温度来补偿:

```
Vref_compensated = Vref_measured x (1 - TC x (T - 25) / 10^6)
```

实现步骤: (1)多温度点标定,拟合温漂曲线;(2)系统中加温度传感器;(3)运行时查表或计算补偿值。

### 11.2 两点校准

最常见的补偿方式,在两个温度点标定后线性插值:

```
T1=-10C, Vref1=2.4990V; T2=+60C, Vref2=2.5015V
TC_calc = (2.5015-2.4990)/(60-(-10))/2.5 x 10^6 = 14.3 ppm/C
```

两点校准消除一阶温漂,残留二阶误差。

### 11.3 高阶补偿与硬件补偿

高阶补偿用多项式拟合Vref(T) = a0 + a1*T + a2*T^2 + a3*T^3,需3-4个标定温度点,可将ADR4525有效温漂降到1ppm/C以下。

硬件补偿方式:
- 恒温槽: 加热维持恒定温度,经典但功耗大
- 对称布局: 两侧对称放置相同热源,使温度梯度均匀
- 差分基准: 两颗同型号基准做差分,温漂共模抑制

### 11.4 补偿方法选择

| 方法 | 精度提升 | 成本 | 适用场景 |
|------|---------|------|---------|
| 两点校准 | 3-5倍 | 低(软件) | 工业测量 |
| 多项式补偿 | 5-10倍 | 中(软件+标定) | 精密仪器 |
| 恒温槽 | 10倍以上 | 高(功耗+面积) | 实验室基准 |
| 直接选低温漂基准 | -- | 中(器件成本) | 最推荐 |

工程实践: 优先选低温漂基准。3ppm/C的REF5025配两点校准,效果远好于50ppm/C的LM4040配多项式补偿。

## 12 不同ADC分辨率的基准选型指南

### 12.1 12位ADC基准要求

1LSB = 610uV(2.5V基准)。要求: 初始精度0.1-0.5%(可校准),TC<50ppm/C(工业温域约3LSB),噪声<100uVpp。推荐MCP1501、LM4040、内部基准(比例测量)。

### 12.2 16位ADC基准要求

1LSB = 38uV。要求: 初始精度0.05%(建议校准),TC<10ppm/C(工业温域约8LSB),噪声<10uVpp。推荐REF5025。16位是外部精密基准的起步点。

### 12.3 24位ADC基准要求

1LSB = 0.15uV。要求: 初始精度0.02%,TC<3ppm/C,噪声<1.5uVpp,长期漂移<10ppm/kHr。推荐ADR4525或REF5025配合滤波。24位系统还必须:加缓冲器、多级RC滤波、严格模拟地布局、温度补偿。

### 12.4 选型速查表

| ADC位数 | 基准TC要求 | 基准噪声要求 | 推荐型号 | 大致成本 |
|---------|-----------|------------|---------|---------|
| 10-12位 | < 100 ppm/C | < 100 uVpp | LM4040 / 内部 | <$0.5 |
| 12-14位 | < 30 ppm/C | < 30 uVpp | MCP1501 | ~$0.9 |
| 14-16位 | < 10 ppm/C | < 10 uVpp | REF5025 | ~$1.8 |
| 16-18位 | < 5 ppm/C | < 5 uVpp | REF5025 + 缓冲 | ~$3 |
| 24位+ | < 3 ppm/C | < 1.5 uVpp | ADR4525 + 缓冲 | ~$6 |

以上为单器件参考价,不含运放和阻容成本。

## 总结

基准电压是ADC精度体系的基石。回到开头的类比:无论尺子刻度多密(分辨率多高),尺子本身不准(基准漂移),量出来的长度就没意义。核心要点:

1. **温漂是头号敌人**: 初始精度可校准,温漂只能选型或补偿。工业温域内50ppm/C吃掉12位5个LSB、16位82个LSB。
2. **选型先看TC和噪声**: 按ADC分辨率匹配基准等级,24位ADC配50ppm/C基准是大材小用,12位ADC配ADR4525是过度设计。
3. **串联优先并联补充**: IoT采集用串联基准更简单可靠,电流环和负基准才用并联。
4. **缓冲器+滤波不可省**: 16位以上必须加基准缓冲器和RC滤波。
5. **PCB布局定生死**: 基准远离热源、紧邻ADC、模拟地单点接地。
6. **补偿不如选型**: 优先选低温漂基准,软件补偿是锦上添花不是雪中送炭。

精密测量系统的设计,基准是起点也是终点。选对了基准,后续的信号调理和数字处理才有意义。
