# 射频能量采集整流天线设计与效率分析
> **难度**: 红色 高级 | **领域**: RF能量采集 | **阅读时间**: 约 22 分钟

## 引言

想象一下:你的手机没电了,周围没有充电器,但空气中到处都是WiFi信号、基站信号、电视广播信号。如果能把这些"看不见的无线电波"变成电,是不是像在沙漠里从空气中挤出水来?

射频(RF)能量采集做的就是这件事。整流天线(rectenna)负责"接收+整流"两步,把高频交流信号变成直流电,给低功耗IoT设备供电。不需要电池,设备靠"喝"周围的电磁波就能活。这已有商用芯片和产品。本文从可用功率密度出发,逐步分析架构、核心器件、效率瓶颈与设计权衡。

## 1. 射频环境中的可用能量

### 1.1 常见RF源的功率密度

| 信号源 | 频率 | 典型距离 | 功率密度 (uW/cm2) | 备注 |
|--------|------|----------|-------------------|------|
| FM广播 | 88-108 MHz | 1-5 km | 0.01 - 1 | 覆盖广,密度低 |
| TV广播 | 470-790 MHz | 1-10 km | 0.05 - 5 | 功率较高 |
| GSM 900 | 900 MHz | 25-100 m | 0.1 - 10 | 基站附近丰富 |
| WiFi 2.4G | 2.4 GHz | 1-10 m | 0.01 - 0.5 | 室内主要来源 |
| WiFi 5G | 5 GHz | 1-5 m | 0.001 - 0.1 | 衰减严重 |
| 5G NR | 3.5 GHz | 10-50 m | 0.01 - 1 | 密集部署时可观 |

关键观察:室外以TV/FM和蜂窝基站为主;室内WiFi是主要来源;总体功率极低,在uW量级。

### 1.2 Friis传输方程

接收功率的计算基础:

```
Pr = Pt * Gt * Gr * (lambda / 4*pi*d)^2
```

dB形式: `Pr(dBm) = Pt(dBm) + Gt(dBi) + Gr(dBi) - 20*log10(4*pi*d/lambda)`

计算示例:WiFi路由器(20dBm, 2dBi)在5米处,接收天线0dBi:

```python
import math
Pt, Gt, Gr = 20, 2, 0     # dBm, dBi, dBi
freq, d, c = 2.4e9, 5, 3e8
wavelength = c / freq
fsl = 20 * math.log10(4 * math.pi * d / wavelength)
Pr = Pt + Gt + Gr - fsl
print(f"接收功率: {Pr:.1f} dBm = {10**(Pr/10)/1000:.4f} uW")
# 输出: 接收功率: -32.0 dBm = 0.6310 uW
```

5米处WiFi仅提供约0.63 uW,即使整流效率50%,也只有约0.3 uW直流功率。

## 2. 整流天线(Rectenna)架构

### 2.1 系统组成

```
RF信号 --> [天线] --> [阻抗匹配网络] --> [整流电路] --> [储能/负载]
              |              |                |              |
         接收电磁波     最大功率传输      AC转DC       平滑+供电
```

四个模块各司其职:天线转换电磁波为交流信号;匹配网络最小化反射;整流电路将AC转DC;储能模块滤波并供电。

### 2.2 关键指标

| 指标 | 定义 | 典型范围 |
|------|------|----------|
| PCE | 直流输出功率 / RF输入功率 | 10% - 80% |
| 灵敏度 | 产生有效直流输出的最低RF功率 | -30 to -10 dBm |
| 带宽 | PCE不低于阈值的频率范围 | 窄带<5%; 宽带>20% |
| 输出电压 | 负载条件下的直流电压 | 0.1 - 5 V |

PCE是核心指标,它随输入功率、频率、负载阻抗变化,不是一个固定值。

## 3. Schottky二极管整流器

### 3.1 为什么选Schottky二极管

三个核心原因:

1. **低开启电压**: 0.15-0.3V,远低于PN结的0.6-0.7V,微功率场景下至关重要
2. **高速开关**: 多数载流子导电,无少数载流子存储,GHz频率下工作
3. **低结电容**: 减少RF信号泄漏

常用型号:

| 型号 | 开启电压(V) | 结电容(pF) | 适用频率 |
|------|------------|-----------|----------|
| HSMS-285x | 0.15 | 0.18 | DC-1.5 GHz |
| HSMS-286x | 0.25 | 0.30 | DC-4 GHz |
| SMS7630 | 0.09 | 0.06 | DC-10 GHz |

### 3.2 整流拓扑对比

**单级半波整流器**(最简单):

```
RF输入 --+--> [D1] -->+--> DC输出
         |            |
         +---[C1]-----+--- GND
```

正半周D1导通C1充电;负半周D1截止C1维持。仅利用正半周,效率低。

**倍压整流器**(最常用):

```
RF输入 --+--> [C1] -->+--> [D2] -->+--> DC输出
         |            |            |
         +---[D1]-----+---[C2]-----+--- GND
```

正半周D1导通C1充电;负半周C1电压与RF叠加,通过D2给C2充电至约2*(Vpeak - Vd)。

**Dickson电荷泵**(多级级联):

N级理论输出: `Vout = N * (Vpeak - Vd)`

| 级数 | 适用场景 | 实际效果 |
|------|---------|---------|
| 1 | 高输入功率 | 接近理论值 |
| 2-3 | WiFi距离<3m | 略低于理论 |
| 4-7 | 远距离弱信号 | 明显低于理论 |
| >7 | 不推荐 | 寄生效应严重 |

级数trade-off:级数多能在低功率下产生足够电压,但每个二极管都有压降和漏电,总体效率下降,阻抗匹配也更困难。

## 4. 功率转换效率(PCE)分析

### 4.1 PCE随输入功率的变化

PCE是输入功率的强函数,呈钟形曲线:

- **低功率区**(Pin < -20dBm):PCE极低,信号不足以有效开启二极管
- **峰值区**(Pin约-5到5dBm):二极管充分导通,PCE达最大值
- **高功率区**(Pin > 10dBm):PCE下降,正向压降和寄生效应加剧

不同拓扑PCE对比(2.4GHz, 10kOhm负载):

| 拓扑 | 最优Pin(dBm) | 峰值PCE | -20dBm时PCE | -10dBm时PCE |
|------|-------------|---------|------------|------------|
| 单级半波 | 0 | 45% | 2% | 15% |
| 倍压整流 | -5 | 55% | 5% | 22% |
| 4级Dickson | -10 | 40% | 8% | 25% |
| 7级Dickson | -15 | 30% | 10% | 20% |

### 4.2 影响PCE的关键因素

1. **二极管参数**:开启电压越低,低功率PCE越高;结电容越小,高频损耗越低
2. **阻抗匹配**:输入阻抗随功率变化,固定匹配只在窄范围最优
3. **负载阻抗**:存在最优值,偏离则PCE快速下降
4. **频率与温度**:结电容限制高频性能;温度影响二极管特性稳定性

## 5. 能量采集天线设计

### 5.1 与通信天线的区别

| 参数 | 通信天线 | 能量采集天线 |
|------|---------|-------------|
| 优化目标 | 增益方向性、信噪比 | 接收总功率、带宽 |
| 频率 | 窄带/单频 | 宽带/多频 |
| 极化 | 固定极化 | 圆极化/双极化 |
| 方向性 | 定向 | 全向(未知源方向) |

### 5.2 宽带与多频天线

宽带天线同时覆盖多个频段采集不同RF源能量。常见设计:
- **平面单极天线**:简单,带宽达几十百分比
- **分形天线**:自相似结构实现多频谐振
- **Vivaldi天线**:超宽带渐变缝隙,带宽达数个倍频程

多频天线在特定频段谐振,各有好匹配:
- 嵌套贴片:大小贴片分别谐振不同频率
- 开槽天线:不同长度槽产生多个谐振点
- 多分支单极:不同长度辐射分支

代价:增益低于窄带天线,匹配难度更大。

### 5.3 圆极化天线

环境RF极化方向未知,线极化天线极化失配最多损失50%功率(3dB)。圆极化天线接收任意极化信号,损耗固定3dB。常用结构:微带贴片切角、十字振子、螺旋天线。

## 6. 阻抗匹配挑战

### 6.1 非线性阻抗问题

整流二极管的非线性导致输入阻抗随功率剧烈变化(2.4GHz倍压整流器):

| Pin(dBm) | Re(Zin)(Ohm) | Im(Zin)(Ohm) |
|----------|-------------|-------------|
| -30 | 2500 | -800 |
| -20 | 800 | -500 |
| -10 | 300 | -200 |
| 0 | 80 | -50 |
| 10 | 30 | -10 |

从-30到10dBm,实部变化80倍。任何固定匹配网络都无法覆盖如此大的范围。

### 6.2 匹配网络设计方法

1. **固定L型匹配**:在目标功率点共轭匹配,简单但窄功率范围有效
2. **多频匹配**:多级LC网络,增带宽但增损耗
3. **自适应匹配**:变容二极管或开关电容阵列动态调整,效果好但控制电路耗能
4. **电阻性匹配**:牺牲效率换带宽,适合宽功率范围

```python
# 固定L型匹配网络设计示例(Pin=-10dBm处匹配)
f = 2.4e9; Z_ant = 50; Z_rect = 300 - 200j  # Ohm
Z_target = 300 + 200j  # 共轭
Q = math.sqrt(abs(Z_target.real / Z_ant - 1))
X_series = Z_ant * Q       # 串联电抗
X_shunt = Z_target.real / Q # 并联电抗
```

### 6.3 回波损耗对PCE的影响

```
PCE_effective = PCE_ideal * (1 - |Gamma|^2)
```

S11=-10dB时10%功率被反射;S11=-3dB时50%被反射。微功率场景下损失致命。

## 7. 商用解决方案

### 7.1 Powercast P2110B

| 参数 | 规格 |
|------|------|
| 工作频率 | 850/900/1800/1900/2100 MHz |
| 输入功率范围 | -12 to +10 dBm |
| 峰值PCE | 约58% (915MHz, -2dBm) |
| 输出电压 | 可调,最高5.25V |
| 冷启动最低功率 | -12 dBm |

工作流程:RF整流 -> 电容充电至阈值 -> DC-DC升压启动 -> 稳压供电 -> 电压下降后关断回到充电。

### 7.2 其他商用方案

| NTAG I2C Plus | NXP | NFC能量采集+通信 |

### 7.3 参考设计

```
[915MHz天线] --> [P2110B] --> [储能电容 1mF] --> [LDO 1.8V] --> [传感器+MCU]
     |              |                                    |
  增益6dBi     冷启动-12dBm                          功耗约10uA平均
```

工作周期:充电(数十秒到分钟) -> 工作(数十毫秒) -> 睡眠(等待再充满)。

## 8. 与其他能量采集方式的比较

| 采集方式 | 功率密度 | 优势 | 劣势 |
|----------|---------|------|------|
| 太阳能(室内) | 1-100 uW/cm2 | 功率较高 | 依赖光照 |
| 热电(TEG) | 10-50 uW/cm2 | 持续稳定 | 需要温差 |
| 振动(压电) | 50-500 uW/cm2 | 功率可观 | 依赖振动源 |
| RF采集 | 0.01-1 uW/cm2 | 无处不在 | 功率极低 |

RF采集功率密度最低,但有两个不可替代优势:

1. **可控性**:专用发射源(RFID读写器)可按需供能
2. **无处不在**:城市RF信号持续存在,不依赖光照/温差/振动

实际中常互补使用:室内RF+太阳能,工业RF+热电,可穿戴RF+振动。

## 9. 法规与监管考量

RF采集本身是接收,不受发射限制。但部署专用发射源需遵守法规:

| 地区 | 频段 | 最大EIRP | 法规 |
|------|------|---------|------|
| 美国 | 902-928 MHz | 4W (ISM) | FCC Part 15 |
| 欧洲 | 868 MHz | 0.5W | ETSI EN 300 220 |
| 中国 | 920-925 MHz | 2W | 微功率短距离无线电 |

主动发射RF充电可能被视为"有意辐射",需更严格认证。大功率无线充电需特别许可。RF采集接收功率远低于ICNIRP安全限值,无健康风险。

## 10. 实际限制与未来展望

### 10.1 核心限制

1. **功率密度太低**:室内仅uW级,充电需秒到分钟
2. **PCE随功率急剧下降**:-20dBm以下PCE<5%
3. **阻抗匹配困难**:非线性使固定匹配低效,自适应又耗能
4. **频谱依赖**:信号弱处无法工作
5. **天线尺寸**:900MHz半波偶极子约16cm,小型化困难

### 10.2 技术发展方向

**新型整流器件**:隧道二极管(理论PCE 100%,工艺一致性是瓶颈);GaN Schottky(高击穿电压);二维材料二极管(MoS2超低开启电压)。

**智能匹配技术**:开关电容阵列动态切换;变容二极管连续调谐;机器学习在线优化。

**协同采集架构**:

```
[多频天线阵列] --> [并行整流通道] --> [功率合并] --> [超级电容] --> [负载]
  900MHz 1.8GHz 2.4GHz   每路独立匹配    各路叠加
```

**反向散射通信+能量采集**:设备通过backscatter通信(功耗极低),同时采集RF能量维持工作。

### 10.3 应用前景

RF能量采集不会取代电池,但在以下场景不可替代:

1. **免维护传感器**:墙体内部、桥梁、地下管道等难以换电池的位置
2. **有源RFID/电子货架标签**:大规模部署,零电池维护
3. **生物医学植入物**:体内无线供电,避免手术更换电池
4. **大规模IoT网络**:数万节点,换电池不现实

## 总结

射频能量采集整流天线是一个"听起来很美,做起来很难"的技术:

- 核心矛盾:环境RF功率极低(uW级),整流效率在低功率下急剧下降
- 关键器件:Schottky二极管是目前唯一实用选择,非线性阻抗是根本挑战
- 设计权衡:级数vs效率、带宽vs增益、匹配精度vs功率范围,处处取舍
- 商用进展:Powercast等芯片已证明可行性,但应用仍受限于功率预算
- 未来方向:新型器件(隧道二极管、2D材料)、智能匹配、多频协同采集

理解整流天线设计,本质上是在理解一个极端约束下的优化问题:如何在微瓦级输入下,榨取每一丝可用能量。这既是工程挑战,也是对物理极限的探索。
