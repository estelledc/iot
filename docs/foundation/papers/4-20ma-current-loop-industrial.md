# 4-20mA电流环在工业IoT中的信号传输

> **难度**：🟢 初级 | **领域**：工业信号传输 | **阅读时间**：约 18 分钟

## 引言

想象你在一个嘈杂的工厂车间里,需要把100米外温度传感器的读数传回控制室。如果你用电压信号传输,就像在嘈杂的人群中用轻声耳语交流——沿途的电磁干扰很容易把有用信号淹没。但如果你改用电流信号传输,不管周围多吵,你始终用同样的力气说话,信号不会因线缆长短或噪声干扰而走样。这就是4-20mA电流环的核心思想——用电流而非电压携带信息,天然抗干扰。

再打个比方:电压信号像水管里的水压,管子越长终端水压越低;而电流信号像水管里的流量,只要管子不破,流进去多少就流出多少——基尔霍夫电流定律保证了这一点。4-20mA正是利用电流在闭合回路中处处相等的物理特性,让信号在长距离传输中不衰减。

最后,4mA的下限不是0mA,这也不是随便选的。如果信号范围是0-20mA,当读数为0mA时,你分不清是"测量值为零"还是"线断了"。用4mA作为起点,0mA意味着断线故障,4mA代表量程零点,20mA代表量程满度——这就是4-20mA的巧妙之处。

## 1. 为什么是4-20mA

### 1.1 噪声免疫的物理基础

工业现场充斥着电磁干扰:变频器开关噪声、电机启停浪涌、继电器触点火花。这些干扰在信号线上感应出的电压噪声可达数伏量级。

电压信号传输时,感应噪声直接叠加:

```
V_received = V_signal + V_noise
```

而电流信号传输时,感应电压噪声只改变回路两端的电位分布,不改变回路中的电流:

```
I_received = I_signal  (串联回路电流处处相等)
```

这是基尔霍夫电流定律的直接推论:串联回路中电流处处相等,与线缆电阻和沿线感应电压无关(在电源驱动能力范围内)。

### 1.2 长距离传输优势

电压信号在长线缆上因线缆电阻产生压降:

```
0-10V信号, 线缆电阻50欧, 接收端输入阻抗250欧
V_received = V_signal * (250 / (250 + 50)) = V_signal * 0.833
误差达16.7%
```

电流信号则不存在此问题——只要电源能驱动回路总电阻上的压降不超过其电压容量:

```
V_supply > I_max * R_total
典型: 24V电源驱动20mA电流, 可承受回路总电阻1200欧, 对应2公里以上传输
```

### 1.3 断线检测

4-20mA与0-20mA的关键区别在于"活零点"(Live Zero):

| 电流值 | 0-20mA含义 | 4-20mA含义 |
|--------|-----------|-----------|
| 0 mA | 量程零点(正常) | 断线/断电(故障) |
| 4 mA | 量程的20%(正常) | 量程零点(正常) |
| 20 mA | 量程满度(正常) | 量程满度(正常) |

### 1.4 与电压信号传输的对比

| 特性 | 0-10V电压信号 | 4-20mA电流环 |
|------|-------------|-------------|
| 抗干扰能力 | 差,噪声直接叠加 | 强,电流不受感应电压影响 |
| 传输距离 | 通常<30米 | 可达2公里以上 |
| 线缆电阻影响 | 大,分压产生误差 | 无,电流处处相等 |
| 断线检测 | 不能(0V可能是零点) | 能(0mA=断线,4mA=零点) |
| 功耗 | 随距离变化 | 恒定,最大0.48W(24Vx20mA) |

## 2. 电流环的基本组成

### 2.1 四个核心元件

```
+--[Transmitter]--+--[R_wire]--+--[R_receiver]--+--[R_wire]--+
|                  |             |                  |           |
|              传感器输入        |            采样电阻          |
|                  |             |                  |           |
+------[Power Supply (24V)]-----+------------------+-----------+
```

1. **电源**:通常24V DC,为整个回路提供驱动电压
2. **变送器**:将传感器信号转换为4-20mA电流
3. **接收端**:通过精密采样电阻将电流转换为电压,再由ADC读取
4. **线缆**:连接各部件的导线,电阻不影响电流值

### 2.2 回路电阻预算

电源必须驱动最大电流(20mA)流过回路总电阻:

```
V_supply >= I_max * R_total
R_total = R_transmitter + R_wire + R_receiver + R_wire_return

典型: V_supply=24V, R_wire=200欧(往返2km), R_receiver=250欧
R_total = 450欧, V_required = 20mA x 450 = 9V, 裕量 = 15V(充足)
R_max = 24V / 20mA = 1200欧 (超过则信号削顶失真)
```

### 2.3 采样电阻选择

```
V_out = I_loop * R_sense
R_sense=250欧: 4mA->1.0V, 20mA->5.0V (工业标准)
R_sense=500欧: 4mA->2.0V, 20mA->10.0V (兼容0-10V)
R_sense=165欧: 4mA->0.66V, 20mA->3.3V (直接接3.3V MCU)
```

电阻精度直接影响测量精度,应选用0.1%或更好的金属箔电阻。

## 3. 变送器的接线方式

### 3.1 两线制变送器(Loop-Powered)

变送器与信号共用同一对线缆,工作电源直接从电流环路上获取:

```
       +--------[Transmitter]--------+
       |      (从环路取电)           |
24V ---+                             +---[R_sense]--- GND
       +----------- Signal -----------+
```

- 变送器串联在回路中,自身消耗电流包含在4-20mA信号内
- 最低工作电流必须小于4mA(通常<3.5mA)
- 功率极低(~12mW),只能驱动低功耗传感器
- 优势:只需2根线,省线缆成本,占使用量~70%

### 3.2 三线制变送器

使用独立电源线,信号和电源共地:

```
       +--[24V+]--[Transmitter]--+
       +---------- Signal --------+---[R_sense]--- GND
       +---------- GND -----------+
```

变送器有独立供电不受环路电流限制,适合带HART通信的智能变送器。

### 3.3 四线制变送器

电源和信号完全隔离,抗干扰最优,但需4根线缆,成本最高。适用于高精度分析仪表和需要电气隔离的场合。

| 特性 | 两线制 | 三线制 | 四线制 |
|------|--------|--------|--------|
| 线缆数 | 2 | 3 | 4 |
| 变送器功耗 | <12mW | 不限 | 不限 |
| 电气隔离 | 无 | 无 | 有 |
| 使用占比 | ~70% | ~20% | ~10% |

## 4. 环路供电变送器的设计要点

### 4.1 功率预算

以24V电源、250欧采样电阻为例:

```
I_loop=4mA时: V_rx=1V, V_available=23V
  变送器内部需留裕量(>7V): P_usable = 4mA x 16V = 64mW
I_loop=20mA时: V_rx=5V, V_available=19V
  P_usable = 20mA x 12V = 240mW

设计约束: 以4mA时为最严条件, 可用功率约64mW
```

### 4.2 电压裕量

```
V_tx_min 通常为 7-12V
R_external_max = (V_supply - V_tx_min) / I_loop
V_tx_min=10V时: 20mA下 R_external_max = (24-10)/0.020 = 700欧 (瓶颈)
```

## 5. DAC转电流:集成驱动器

### 5.1 XTR111(TI)

精密电压-电流转换器,专为4-20mA设计:

```
                    XTR111
                  +---------+
  V_in (0-5V) --->| VIN     |
  R_set ----------| RSET    |     I_out = 10 * V_in / R_set
                  | Iout  -->---> 4-20mA
  V_ref (5V) ---->| VREF    |
                  +---------+
R_set=1250欧: V_in=0.5V->4mA, V_in=2.5V->20mA
精度:0.015%非线性, 内置5V/3.3V参考, 开环/短路保护
```

### 5.2 AD5420(ADI)

16位低功耗4-20mA DAC,适合环路供电:

```
                    AD5420
  SPI(MCU) ------->| SCLK/SDIN |
                  | I_out   -->---> 4-20mA
  R_set ----------| R_set     |
                  +-----------+
16位分辨率: LSB = 16mA/65535 = 0.244uA
静态电流仅200uA, 从环路取电无需额外电源
```

| 特性 | XTR111 | AD5420 |
|------|--------|--------|
| 输入方式 | 模拟电压 | SPI数字 |
| 分辨率 | 由输入DAC决定 | 16位 |
| 环路供电 | 否 | 是 |
| 适用场景 | 模拟传感器直连 | MCU/PLC数字控制 |

## 6. 接收端:电流转电压与ADC采集

### 6.1 基本采样电路

```
4-20mA -->---[R_sense 250R 0.1%]---+---> ADC输入
                                    |
                                   GND
12位ADC, V_ref=5V: 有效码数3276, 分辨率4.88uA/LSB
16位ADC, V_ref=5V: 有效码数52429, 分辨率0.305uA/LSB
```

### 6.2 提高精度的方法

1. **增大采样电阻**:500欧使输出2-10V,但需注意回路电阻预算
2. **差分采集+仪表放大器**:消除多通道共地时的共模干扰
3. **高位数ADC**:16位或24位ADC可达到0.004C温度分辨率

### 6.3 IoT网关中的4-20mA采集

```
[4-20mA ch1] -->-[MUX]-->-[ADC 16bit]-->-[MCU]-->-[MQTT/OPC-UA]--> Cloud
[4-20mA ch2] -->/      \
[4-20mA ch3] -->/        -->[R_sense x4]
[4-20mA ch4] -->/
```

软件处理: ADC码值->电压->电流->工程量PV=(I-4)/16*(URV-LRV)+LRV, 同时判断断线(I<3.6mA)和超量程(I>21mA)。

## 7. HART协议:数字信号叠加在4-20mA上

### 7.1 HART基本原理

HART在4-20mA模拟信号上同时传输数字信号:

```
模拟: 4-20mA DC (缓慢变化,代表过程变量)
数字: +/-0.5mA AC, 1200bps FSK叠加在DC上
  Bell 202调制: 1200Hz=Mark(1), 2200Hz=Space(0)
  FSK平均值为零, 不影响4-20mA的DC分量
```

### 7.2 HART在IoT中的价值

| 能力 | 模拟4-20mA | 4-20mA+HART |
|------|-----------|-------------|
| 读取过程变量 | 是 | 是 |
| 设备诊断 | 否 | 是 |
| 远程组态 | 否 | 是 |
| 多变量传输 | 否 | 是(最多4变量) |
| 设备标识 | 否 | 是 |

HART使传统4-20mA变送器升级为"智能"设备,是工业IoT过渡期的重要桥梁。

## 8. NAMUR NE43故障信令

NAMUR NE43定义了4-20mA范围外的故障电流值:

```
< 3.6 mA       -> 断线/变送器失电
3.6 - 4.0 mA   -> 下溢(低于量程下限)
4.0 - 20.0 mA  -> 正常测量范围
20.0 - 21.0 mA -> 上溢(高于量程上限)
> 21.0 mA      -> 变送器内部故障
```

变送器检测到故障时驱动电流到预设值(如3.2mA或22mA),接收端据此区分故障类型。NAMUR NE43是功能安全系统的重要支撑。

## 9. 接线实践与接地

### 9.1 典型接线图

**两线制标准接线:**

```
  +24V DC
    |
    +-------+--------------------+
            |                    |
        [PSU 24V]            [AI卡件]
            |                [R_sense 250R]
            +---[+] Transmitter [-]---+
            +---[+]   线缆      [-]---+
                                      |
                                     GND
```

**带HART通信:**

在AI卡件端并联HART调制解调器,其余接线相同。

### 9.2 接地原则

**规则一:避免地环路**——只在接收端单点接地,变送器端浮地:

```
错误: 两端接地 -> 地电位差V_gnd导致干扰电流 V_gnd/R_wire
正确: 接收端单点接地
```

**规则二:屏蔽层单端接地**——屏蔽层只在接收端接机壳地,变送器端悬空。

**规则三:使用双绞屏蔽线**——双绞抵消磁场干扰,屏蔽阻隔电场干扰。推荐RVVP 2x0.75或AWG18双绞屏蔽线。

### 9.3 常见接线错误

| 错误 | 现象 | 原因 |
|------|------|------|
| 两端接地 | 读数漂移,串入50Hz纹波 | 地环路引入干扰电流 |
| 屏蔽层两端接地 | 读数跳动 | 屏蔽层成为第二信号通路 |
| 正负接反 | 读数为0或负值 | 电流方向错误 |
| 采样电阻精度不够 | 系统性偏差 | 电阻误差直接转为测量误差 |

## 10. 4-20mA与数字总线的对比

| 特性 | 4-20mA | HART | Modbus RTU | PROFIBUS | IO-Link |
|------|---------|------|------------|----------|---------|
| 信号类型 | 模拟电流 | 模拟+数字 | 数字RS485 | 数字 | 数字 |
| 传输距离 | 2km | 2km | 1.2km | 100m | 20m |
| 每线设备数 | 1 | 1(多drop时15) | 247 | 126 | 1 |
| 诊断能力 | 断线 | 丰富 | 有 | 丰富 | 丰富 |
| 协议复杂度 | 无 | 中 | 中 | 高 | 低 |
| 成本 | 最低 | 低 | 中 | 高 | 低 |

工业IoT选择策略:

```
需要设备诊断和远程组态?
  否 --> 仅需过程变量? --> 4-20mA (最简单最可靠)
  是 --> 设备密度和距离?
    高密度+短距离 --> IO-Link
    中密度+中距离 --> Modbus RTU
    高可靠性+长距离 --> HART 或 4-20mA+独立诊断
```

工业现场大量存量设备仍是4-20mA接口,IoT网关的首要任务是将这些模拟信号数字化。

## 11. 从4-20mA到云端:完整信号链路

### 11.1 信号链路

```
[传感器]->[4-20mA变送器]->[线缆]->[采样电阻]->[ADC]->[MCU]->[协议栈]->[云端]
  物理量    电流环传输     模拟    V/I转换   数字化  工程量换算  通信    数据库
```

### 11.2 软件处理示例

```c
float adc_to_engineering(uint16_t adc_code,
                         float v_ref, float r_sense,
                         float lrv, float urv) {
    float voltage = (float)adc_code * v_ref / 65536.0f;
    float current_ma = (voltage / r_sense) * 1000.0f;
    float pv = ((current_ma - 4.0f) / 16.0f) * (urv - lrv) + lrv;
    if (current_ma < 3.6f || current_ma > 21.0f)
        pv = NAN;  // NAMUR NE43故障标记
    return pv;
}
```

### 11.3 误差链分析

| 环节 | 典型误差 | 累积方式 |
|------|---------|---------|
| 变送器精度 | 0.075% of span | RSS |
| 线缆(电流环) | 0% (理想) | 不累积 |
| 采样电阻 | 0.1% | RSS |
| ADC积分非线性 | 0.01% (16位) | RSS |
| ADC参考电压 | 0.05% | RSS |

总误差(RSS合成): sqrt(0.075^2 + 0.1^2 + 0.01^2 + 0.05^2) = 0.135% of span

## 总结

4-20mA电流环是工业自动化领域最经典、最可靠的模拟信号传输标准。它的核心优势来自三个物理事实:

1. **电流在串联回路中处处相等**——信号不因线缆电阻衰减,天然抗干扰
2. **4mA活零点**——0mA明确表示断线故障,实现故障自诊断
3. **恒流驱动**——电源只需克服回路压降,传输距离可达数公里

在工业IoT时代,4-20mA不会退出舞台,而是与数字技术融合:

- **HART协议**在模拟信号上叠加数字通信,实现"模拟+数字"双通道
- **NAMUR NE43**利用信号范围外的电流值传递故障状态
- **IoT网关**将4-20mA模拟信号数字化,通过MQTT/OPC-UA上报云端
- **集成驱动器**(XTR111/AD5420)简化了变送器设计

理解4-20mA,是理解整个工业信号传输体系的起点。从电压信号到电流环,从模拟到HART,从HART到纯数字总线,每一步演进都保留了对可靠性和长距离传输的执着追求。在可见的未来,4-20mA仍将是传感器到网关这"最后一公里"的标准语言。
