# 万用表高级测量：真有效值/温度/频率

> **难度**：🟢 入门 | **领域**：测量与调试 | **关键词**：True RMS, 频率, 热电偶, Min/Max | **阅读时间**：约 14 分钟

## 日常类比

只会测干电池电压的表，像只会量身高的尺子。IoT 现场常遇开关电源纹波、PWM（Pulse-Width Modulation）方波、MOSFET 发热——需要 **真有效值**（True RMS）、频率/占空比、温度与 Min/Max 等“高级档”，否则平均值档会把非正弦波读错一截[1][2]。

## 摘要

说明 True RMS 与平均响应差异，以及电容、频率、温度、二极管/通断、Min/Max/REL 在嵌入式调试中的用法。误差百分比为仪表精度等级常见量级，以具体型号手册为准[3]。

## 1. 超越 V/I/R

| 功能 | IoT 用途 |
|------|----------|
| True RMS 电压/电流 | 开关电源、变频器旁非正弦 |
| 频率/占空比 | 校验 PWM、时钟 |
| 温度 | 热点、热设计 |
| 电容 | 去耦是否虚焊/衰减 |
| Min/Max | 捕捉跌落与尖峰（受采样限制） |
| REL | 去表笔电阻测小 R |

## 2. 真有效值

平均响应表按正弦有效值刻度；方波、三角、畸变波会显著偏差。True RMS 对交流分量按均方根计算（注意带宽与波峰因数限制）[1]。

| 波形 | 平均响应风险 | 建议 |
|------|--------------|------|
| 工频正弦 | 通常可接受 | 普通表 |
| PWM/开关噪声 | 易误读 | True RMS + 够带宽 |
| 窄脉冲 | 波峰因数超限 | 示波器 |

## 3. 频率、温度与其他

频率档可粗验 MCU 时钟与风扇 PWM；注意触发电平与输入耦合。温度多接 K 型热电偶，测壳温/模块温——接触与热惯性影响读数。二极管档看正向压降；通断档查线束。Min/Max 不能替代示波器抓 ns 级毛刺[2][4]。

| 注意 | 原因 |
|------|------|
| 电容档先放电 | 安全与保护输入 |
| 在线测电容不准 | 并联路径 |
| 电流档勿并电压 | 保险与表内分流器 |

## 4. 局限、挑战与可改进方向

### 1. 把万用表当示波器

**局限**：Min/Max 更新慢，漏短毛刺。
**改进**：电源完整性与数字线用示波器/逻辑分析仪[4]。

### 2. 带宽不足的 True RMS

**局限**：高频开关基波外的能量测不到。
**改进**：查仪表带宽；关键节点用差分探头+真有效值功率分析[3]。

### 3. 温度接触误差

**局限**：热电偶虚贴导致读数偏低。
**改进**：固定贴合、等待稳定；关键用热像仪扫热点[5]。

### 4. 安全量程误用

**局限**：CAT 等级不足或电流孔测电压。
**改进**：按配电环境选 CAT；养成看孔位与量程习惯[1]。

## 总结

IoT 调试里，True RMS、频率与温度档能快速定位电源与热问题；认清带宽与采样极限，该上示波器时不要硬撑。选表先看安全等级与 True RMS 带宽，再看附加功能。
