# EMI屏蔽与机壳设计减少辐射发射
> **难度**：🟡 中级 | **领域**：EMI屏蔽设计 | **阅读时间**：约 20 分钟

## 引言

想象一个隔音录音棚: 墙壁用厚实的隔音材料包裹, 门缝用密封条压紧, 连空调出风口都做了消声处理, 外面再吵也传不进来。EMI 屏蔽就是电子设备的"隔音棚" -- 用导电壳体把设备内部的高频噪声关在里面, 同时挡住外部的电磁骚扰。但屏蔽不是简单套个铁壳就行, 缝隙、孔洞、电缆穿过都会成为"漏声的门缝", 需要精心设计。

## 1 屏蔽理论

### 1.1 法拉第笼原理

法拉第笼 (Faraday Cage) 是屏蔽的理论基础: 用导电材料包围的空间, 外部电场在导体表面感应出电荷, 使内部电场为零。同理, 内部电磁波在导电壳体内表面感应出电流, 能量被反射和吸收, 泄漏到外部的场强大幅衰减。

关键条件:
- 壳体必须是良好的导体
- 壳体必须尽可能封闭 (无大缝隙)
- 壳体必须良好接地 (低频屏蔽时)

### 1.2 屏蔽机理: 三重损耗

电磁波穿过屏蔽体时经历三重衰减:

1. **反射损耗 (Reflection Loss, R)**: 电磁波在屏蔽体表面被反射, 与波阻抗和屏蔽体表面阻抗的失配程度相关。电场反射损耗远大于磁场反射损耗。

2. **吸收损耗 (Absorption Loss, A)**: 电磁波进入屏蔽体后被涡流衰减, 与屏蔽体厚度和趋肤深度相关。频率越高, 趋肤深度越小, 吸收损耗越大。

3. **多次反射修正 (Multiple Reflection Correction, B)**: 电磁波在屏蔽体内两个界面之间来回反射。当吸收损耗 A > 10 dB 时, B 可忽略。

总屏蔽效能:

```
SE = R + A + B  (单位: dB)
```

### 1.3 屏蔽效能 (SE)

屏蔽效能 (Shielding Effectiveness) 定义为无屏蔽时场强与有屏蔽时场强之比, 以 dB 表示:

```
SE = 20 * log10(E_unshielded / E_shielded)
```

| SE 值 | 含义 | 应用 |
|-------|------|------|
| 10-30 dB | 低屏蔽 | 塑料壳内导电涂层 |
| 30-60 dB | 中等屏蔽 | 铝壳 + 接缝处理 |
| 60-90 dB | 高屏蔽 | 钢壳 + 密封垫片 |
| > 90 dB | 极高屏蔽 | 军工级屏蔽室 |

## 2 屏蔽效能的频率依赖性

### 2.1 低频磁场屏蔽

低频 (kHz 级) 磁场很难屏蔽: 趋肤深度大, 反射损耗小。需要:
- 使用高磁导率材料 (坡莫合金、硅钢)
- 增加屏蔽体厚度
- 多层屏蔽 (不同磁导率组合)

### 2.2 高频电场屏蔽

高频 (MHz-GHz) 电场屏蔽相对容易: 趋肤深度小, 反射损耗大。薄铝箔就能提供 60 dB 以上屏蔽效能。

### 2.3 谐振效应

当屏蔽体尺寸接近波长的一半时, 会产生腔体谐振, 屏蔽效能急剧下降。1 GHz 时半波长为 15 cm, 小型 IoT 设备机壳通常不会产生低次谐振, 但较大机箱需要注意。

## 3 机壳材料选择

### 3.1 金属板材

| 材料 | 厚度 | SE (100 MHz) | 特点 |
|------|------|-------------|------|
| 镀锌钢板 | 0.8-1.5 mm | > 100 dB | 高屏蔽, 低成本, 笨重 |
| 铝板 | 0.5-2.0 mm | > 80 dB | 轻量, 导热好, 中等成本 |
| 不锈钢板 | 0.5-1.5 mm | > 90 dB | 耐腐蚀, 硬度高, 难加工 |
| 铜板 | 0.3-1.0 mm | > 100 dB | 最佳导电性, 昂贵 |

铝板是 IoT 设备最常用的屏蔽材料, 在重量、成本和屏蔽效能之间取得平衡。

### 3.2 导电塑料

在塑料中掺入导电填料 (碳纤维、金属纤维、碳纳米管):
- 塑料基材提供结构强度和轻量性
- 导电填料提供屏蔽能力
- SE 通常 20-40 dB, 取决于填料浓度
- 适合消费级 IoT 设备的外观要求

### 3.3 金属化塑料

在塑料壳内表面喷涂或沉积金属层:
- 导电漆 (银/铜/镍): 喷涂施工, SE 约 30-60 dB
- 真空镀铝: 均匀薄层, SE 约 40-70 dB
- 电镀: 附着力好, SE 约 50-80 dB

优点: 外壳保持塑料的轻量和设计自由度, 内表面获得金属的屏蔽性能。

### 3.4 压铸壳体

铝合金或锌合金压铸一体成型:
- 无接缝, 屏蔽连续性最佳
- 适合批量生产
- 模具成本高, 适合量产产品

## 4 孔洞与缝隙效应

### 4.1 缝隙作为缝隙天线

机壳上的缝隙 (seam) 和孔洞 (aperture) 是屏蔽的最大弱点。缝隙等效为缝隙天线, 当缝隙长度接近波长的一半时, 辐射效率最高。

### 4.2 孔洞尺寸规则

经验法则: **最大孔洞尺寸应小于最短波长的 1/20**

```
max_aperture = lambda / 20
```

| 频率 | 波长 | 最大孔洞 |
|------|------|----------|
| 100 MHz | 3 m | 150 mm |
| 300 MHz | 1 m | 50 mm |
| 1 GHz | 300 mm | 15 mm |
| 3 GHz | 100 mm | 5 mm |

1 GHz 以上, 普通拼缝和散热孔就需要认真处理。

### 4.3 多孔洞的叠加效应

N 个相同尺寸的孔洞, 屏蔽效能下降约:

```
SE_total = SE_single - 20 * log10(N)
```

100 个 5 mm 孔洞比单个孔洞屏蔽效能低约 40 dB。通风孔阵列需要特别关注。

### 4.4 减小缝隙泄漏的方法

- **增加搭接面积**: 两块金属板的搭接面越大, 接触电阻越低
- **增加紧固点密度**: 螺丝间距小于缝隙允许长度
- **焊接**: 连续焊缝实现零缝隙, 但不可拆卸
- **导电衬垫**: 填充缝隙, 保证电气连续性

## 5 衬垫与接触设计

### 5.1 指形簧片 (Finger Stock)

铍铜 (BeCu) 指形簧片是常用的缝隙屏蔽材料:
- 安装在机壳盖板和主体之间的槽内
- 靠弹性变形保证接触压力
- 屏蔽频段: DC - 10+ GHz
- 耐反复开合 (数千次)

选型要点:
- 压缩量: 通常为自由高度的 20-30%
- 接触电阻: < 10 mOhm
- 安装方式: 卡入槽、粘接或铆接

### 5.2 导电橡胶 (Conductive Elastomer)

在硅橡胶或氟橡胶中掺入导电颗粒 (银、镍、碳):
- 同时提供电磁屏蔽和环境密封 (防水防尘)
- 压缩永久变形率较低, 适合长期密封
- SE 约 40-80 dB (银填充最高)
- 成本较高, 银填充型最贵

### 5.3 金属丝网 (Wire Mesh)

铝或铜编织丝网衬垫:
- 高频屏蔽性能好 (> 1 GHz)
- 压缩回弹性好
- 不提供环境密封
- 成本较低

### 5.4 铍铜簧片选型建议

| 应用场景 | 推荐衬垫 | 原因 |
|----------|----------|------|
| 频繁拆卸的面板 | 指形簧片 | 弹性好, 耐反复压缩 |
| 防水密封要求 | 导电橡胶 | 双功能: 屏蔽 + 密封 |
| 大面积缝隙 | 金属丝网 | 成本低, 易裁剪 |
| 高频精细缝隙 | 铍铜簧片 + 导电胶 | 高频接触阻抗低 |

## 6 电缆穿透

### 6.1 滤波连接器

电缆穿过机壳时, 如果不处理, 会成为 EMI 的"高速公路":
- 内部噪声沿电缆辐射到外部
- 外部干扰沿电缆传导到内部

滤波连接器内置穿心电容或 π 型滤波器, 衰减高频信号, 仅通过有用信号频段。

### 6.2 铁氧体磁环

在电缆上套铁氧体磁环 (Ferrite Core):
- 增加共模电感, 抑制共模电流
- 安装位置: 尽量靠近机壳穿出点
- 选择磁导率: 高频用 NiZn (100-1000 MHz), 低频用 MnZn (1-100 MHz)
- 多绕几匝可增加阻抗

### 6.3 屏蔽电缆

使用屏蔽电缆并正确接地:
- 同轴电缆: 外层屏蔽编织网在穿壳处 360 度搭接
- 双绞线 + 独立屏蔽: 每对双绞线独立屏蔽, 总屏蔽 360 度搭接
- 关键: 屏蔽层必须在机壳穿出点做 360 度环形接地, 不能用"猪尾巴"引线接地

### 6.4 屏蔽层的接地

| 频率 | 推荐接地方式 | 原因 |
|------|-------------|------|
| < 1 MHz | 单端接地 | 避免地环路电流 |
| > 10 MHz | 双端接地 | 提供低阻抗回流路径 |
| 混合 | 一端 360 度接地, 另端电容接地 | 兼顾低频和高频 |

## 7 PCB 级屏蔽

### 7.1 板级屏蔽罩 (Board-Level Shield)

在 PCB 上安装金属屏蔽罩:
- 将噪声源 (RF 功放、DC-DC) 或敏感电路 (接收前端) 罩住
- 屏蔽罩焊接在 PCB 接地焊盘上
- 高度通常 2-5 mm, 占用面积随电路而定

### 7.2 分区隔离 (Compartmentalization)

将 PCB 划分为不同功能区, 用接地隔离:

```
+------------------+--------+------------------+
|   无线模块区     | 隔离带  |   数字电路区     |
|  (屏蔽罩覆盖)    | (地沟)  |                  |
+------------------+--------+------------------+
```

隔离带 (Moat) 做法:
- 在地平面上切出一条无铜沟槽
- 信号跨区时用磁珠或共模扼流圈
- 各区电源独立滤波

### 7.3 屏蔽罩设计要点

- 屏蔽罩底部必须有连续接地焊盘
- 接地过孔间距 < lambda/20 (目标最高频率)
- 屏蔽罩与内部元件保持高度间隙, 避免短路
- 可开小孔散热, 但孔径需满足屏蔽要求

## 8 通风与屏蔽的权衡

### 8.1 冲孔 vs 屏蔽

散热通风需要开孔, 开孔降低屏蔽效能。解决冲突: **蜂窝波导-截止通风板 (Honeycomb Waveguide-Below-Cutoff Vent)**

### 8.2 波导-截止原理

当波导 (导管) 的截面尺寸小于截止波长时, 电磁波无法传播:

- 圆形波导截止频率: fc = 1.841 * c / (2 * pi * d), d 为直径
- 矩形波导截止频率: fc = c / (2 * a), a 为宽边

在截止频率以下, 电磁波在波导中指数衰减。蜂窝板就是大量小孔径波导并联, 空气可流通但电磁波被衰减。

### 8.3 蜂窝通风板参数

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 孔径 | 1.5-3 mm | 决定截止频率 |
| 厚度 | 6-12 mm | 决定衰减量 |
| SE | 40-80 dB | 厚度越大, SE 越高 |
| 压降 | < 50 Pa | 对散热风扇的影响 |

## 9 显示窗口屏蔽

### 9.1 ITO 导电膜

在显示窗口的玻璃或亚克力上沉积 ITO (氧化铟锡) 导电膜:
- 透光率 70-85%
- 表面方阻 5-20 Ohm/sq
- SE 约 20-40 dB
- ITO 膜的四边必须与机壳导电连接

### 9.2 金属丝网嵌入

在两层玻璃之间嵌入细密金属丝网:
- 丝径 0.02-0.05 mm
- 网格间距 0.1-0.5 mm
- SE 约 40-60 dB (优于 ITO)
- 透光率 60-80%
- 可能在某些角度可见网格纹

### 9.3 选择依据

| 需求 | 推荐 | 原因 |
|------|------|------|
| SE > 40 dB | 金属丝网 | 屏蔽效能高 |
| 高透光 | ITO | 透光率更高 |
| 触摸屏 | ITO | 丝网干扰触摸 |
| 成本敏感 | ITO | 工艺简单 |

## 10 实际设计示例: IoT 网关

### 10.1 设备描述

一款 IoT 网关, 包含:
- Wi-Fi + BLE 无线模块
- 以太网接口
- DC-DC 电源 (5V 输入, 3.3V 输出)
- STM32 MCU

辐射发射问题: 200-500 MHz 频段超标 6 dB。

### 10.2 屏蔽方案

1. **机壳**: 铝合金挤出型材, 上下盖板
2. **盖板接缝**: 上下盖与主体之间加装铍铜指形簧片, 螺丝间距 25 mm
3. **I/O 接口**: 以太网口使用带金属壳的 RJ45 连接器, 壳体与机壳 360 度搭接
4. **电源入口**: DC-DC 输入端加共模扼流圈 + Y 电容, 电源线套铁氧体磁环
5. **无线模块**: 单独加 PCB 屏蔽罩, 天线从屏蔽罩侧面引出
6. **通风**: 侧壁开蜂窝状散热孔 (孔径 2 mm, 壁厚 8 mm)

### 10.3 改进效果

| 措施 | 改善量 |
|------|--------|
| 加铍铜簧片 | -8 dB (200-500 MHz) |
| RJ45 金属壳搭接 | -5 dB (100-300 MHz) |
| DC-DC 输入滤波 | -10 dB (1-30 MHz 传导) |
| 无线模块屏蔽罩 | -12 dB (1-3 GHz) |
| 总改善 | 从超标 6 dB 降至余量 10 dB |

## 11 消费级 IoT 的成本优化

### 11.1 选择性屏蔽

不需要全机壳屏蔽, 只屏蔽最关键的噪声源:
- DC-DC 变换器: 用 PCB 屏蔽罩 (成本约 0.5 元)
- Wi-Fi 模块: 自带金属屏蔽罩 (已包含在模块成本中)
- 其他低频数字电路: 靠 PCB 布局和滤波解决, 不加屏蔽

### 11.2 塑料壳 + 局部导电漆

- 塑料壳保持外观和成本优势
- 只在 DC-DC 和射频模块对应的内壁区域喷涂导电漆
- 喷涂面积小, 成本增加有限 (约 1-2 元)
- SE 约 30-40 dB, 对消费级设备通常足够

### 11.3 PCB 设计代替机械屏蔽

很多时候, PCB 设计优化可以减少对机械屏蔽的依赖:
- 四层板: 完整地平面天然提供 20-30 dB 屏蔽
- 优化布局: 噪声源远离板边和连接器
- 滤波: 接口处加 RC 滤波和 TVS
- 这些措施成本极低, 应优先采用

## 12 屏蔽效能测试

### 12.1 同轴法

将屏蔽材料样品夹在同轴传输线中:
- 测量插入损耗, 即为屏蔽效能
- 频率范围: 30 MHz - 1.5 GHz
- 适用于平面材料 (板材、涂层)

### 12.2 自由空间法

在暗室中用天线照射屏蔽体, 测量透射场强:
- 适用于完整机壳或大型组件
- 测量结果反映实际使用效果
- 成本较高

### 12.3 近场扫描法

用近场探头在屏蔽体内外扫描:
- 定位泄漏点
- 快速验证设计改进效果
- 适合研发阶段迭代

### 12.4 测试注意事项

- 测试夹具和样品的接触质量直接影响结果
- 衬垫压缩量需与实际安装一致
- 电缆连接和接地方式需与量产状态一致

## 总结

EMI 屏蔽设计是辐射发射达标的最后一道防线。屏蔽效能由反射损耗、吸收损耗和多次反射修正三部分组成, 随频率和材料特性变化。机壳材料选择需平衡屏蔽效能、重量和成本; 缝隙和孔洞是屏蔽的薄弱环节, 最大孔洞尺寸应小于最短波长的 1/20; 衬垫 (簧片、导电橡胶、丝网) 保证接缝处的电气连续性; 电缆穿透必须用滤波连接器、铁氧体磁环或屏蔽电缆处理; PCB 级屏蔽罩是局部隔离的有效手段; 通风与屏蔽的矛盾用蜂窝波导截止板解决; 显示窗口用 ITO 或金属丝网兼顾透光和屏蔽。消费级 IoT 设备可通过选择性屏蔽和 PCB 设计优化降低成本。
