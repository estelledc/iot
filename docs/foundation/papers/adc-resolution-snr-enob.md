# ADC分辨率/SNR/ENOB有效位数深度解析

> **难度**：🟡 中级 | **领域**：数据转换器性能 | **阅读时间**：约 20 分钟

## 引言

想象你用一把尺子量木头。分辨率是尺子上刻了多少格——毫米尺比厘米尺刻度更密，能读出更小的数。但精度是另一回事：如果尺子本身受热膨胀了 0.5%，即使刻到微米也没用，读出来的数照样偏。ADC 也是同理：标称 24 位不是说你有 24 位可靠数据，实际有效位数 (ENOB) 可能只有 18-20 位——就像一把刻了 2400 格但只有 1800 格能信的尺子。

本文系统梳理分辨率、SNR、ENOB 三者的关系，拆解噪声来源，解释为什么"标称位数"不等于"有效位数"，并给出 IoT 传感器选型和实践测量的具体方法。

## 1. 分辨率与精度：两个常被混淆的概念

### 1.1 定义对比

- **分辨率**：ADC 能区分的最小输入变化，由位数 N 决定，将满量程 (FSR) 分成 2^N 个离散等级。只回答"能分多少层"，不回答"每层准不准"
- **精度**：输出码与真实输入的偏差，包含偏移误差、增益误差、积分非线性 (INL) 和微分非线性 (DNL)

### 1.2 关键区别

| 属性 | 分辨率 | 精度 |
|------|--------|------|
| 含义 | 能分多少层 | 每层离真实值多远 |
| 决定因素 | 位数 N | INL、DNL、偏移、增益误差 |
| 单独提高 | 增加位数即可 | 需改善模拟电路质量 |
| 关系 | 精度 <= 分辨率 | 可以远低于分辨率 |

一个 16 位 ADC 如果 INL = 8 LSB，实际无失真精度约 13 位 (16 - log2(8))。分辨率是天花板，精度是实际高度。

### 1.3 LSB 大小计算

LSB (Least Significant Bit) 是 ADC 能区分的最小模拟量：`LSB = FSR / 2^N`

| 满量程 (VREF) | 位数 N | LSB 大小 |
|---------------|--------|----------|
| 5 V | 8 | 19.5 mV |
| 5 V | 12 | 1.22 mV |
| 5 V | 16 | 76.3 uV |
| 2.5 V | 24 | 149 nV |

24 位 ADC 的 LSB 在纳伏量级——任何微小的噪声都会淹没好几个 LSB，这就是为什么 24 位不等于 24 位有效数据。

## 2. 理想 ADC 的信噪比

### 2.1 量化噪声与信号功率

理想 ADC 唯一误差是量化。量化误差在 -LSB/2 到 +LSB/2 均匀分布：

```
e_rms^2 = LSB^2 / 12
```

满量程正弦波均方值：`S_rms = FSR^2 / 8`

### 2.2 理想 SNR 公式推导

将 FSR = LSB * 2^N 代入信噪比定义：

```
SNR = 10 * log10((FSR^2/8) / (LSB^2/12))
    = 10 * log10(1.5 * 2^(2N))
    = 6.02N + 1.76  [dB]
```

这是 ADC 领域最核心的公式：**SNR_ideal = 6.02N + 1.76 dB**

### 2.3 各位数的理想 SNR

| 位数 N | 理想 SNR (dB) | LSB/FSR |
|--------|---------------|---------|
| 8 | 49.92 | 1/256 |
| 12 | 74.00 | 1/4096 |
| 16 | 98.08 | 1/65536 |
| 20 | 122.16 | 1/1M |
| 24 | 146.24 | 1/16M |

每增加 1 位，SNR 提升约 6 dB——这是极其有用的工程直觉。

## 3. 噪声来源全景

### 3.1 四大噪声源

| 噪声源 | 机制 | 关键公式/数值 |
|--------|------|---------------|
| 量化噪声 | 有限位数截断，均匀分布在 0~fS/2 | 功率 = LSB^2/12，不可消除 |
| 热噪声 | 采样保持开关的 kT/C 噪声 | C=1pF 时 v_n=64.5uV; C=10pF 时 v_n=20.4uV |
| 参考噪声 | VREF 抖动等比传递到输出 | 10ppm 噪声 = FSR*10ppm 等效输入噪声 |
| 时钟抖动 | 孔径不确定度 | V_err = 2*pi*f_in*V_peak*t_j |

### 3.2 其他噪声源

- **电源纹波**：PSRR 有限 (通常 60-80 dB)，纹波耦合到模拟通路
- **数字馈通**：开关动作耦合到模拟输入，布局隔离可缓解
- **1/f 噪声**：MOSFET 闪烁噪声，低频段主导，斩波技术可消除

### 3.3 噪声叠加原则

不相关噪声源按功率叠加：`N_total = N_quant + N_thermal + N_ref + N_jitter + N_other`

任何一项超过量化噪声，都会使实际 SNR 低于理想值，ENOB 随之下降。

## 4. ENOB：有效位数的定义与计算

### 4.1 为什么需要 ENOB

实际 ADC 的噪声不仅是量化噪声，还有热噪声、参考噪声等。实际 SNR 低于理想值 6.02N+1.76 dB。ENOB 回答："这个实际 ADC，相当于几位理想 ADC？"

### 4.2 从 SINAD 计算 ENOB

SINAD (Signal-to-Noise-And-Distortion) 是信号功率与全部误差之比：

```
SINAD = 10 * log10(P_signal / (P_noise + P_distortion))
ENOB = (SINAD - 1.76) / 6.02
```

注意：用 SINAD 而非 SNR，因为谐波失真也吃掉有效位。

### 4.3 关系链

```
SNR = 信号/噪声 (不含失真)
THD = 谐波失真/信号 (不含噪声)
SINAD = 信号/(噪声+失真)  ==>  SINAD <= SNR
ENOB = (SINAD - 1.76) / 6.02  ==>  ENOB <= N
```

### 4.4 计算实例

标称 16 位 ADC，实测 SINAD = 86 dB：`ENOB = (86 - 1.76) / 6.02 = 14.0 位`——2 位被噪声和失真"吃掉"。

标称 24 位 Sigma-Delta，实测 SINAD = 110 dB：`ENOB = (110 - 1.76) / 6.02 = 18.0 位`——6 位是"名义上的"。

## 5. THD 对有效分辨率的影响

### 5.1 谐波失真来源

传输函数非线性、采样保持电荷注入、比较器亚稳态、输入缓冲器失真。

### 5.2 THD 如何降低 SINAD

假设 SNR = 90 dB、THD = -80 dB (失真功率 = S*10^(-8))：

```
SINAD = 10*log10(S / (S*10^-9 + S*10^-8)) = 79.6 dB
```

THD = -80 dB 时，SINAD 从 90 dB 降到 79.6 dB——失真主导了性能。

### 5.3 THD 对 ENOB 的影响量

| THD (dB) | SNR (dB) | SINAD (dB) | ENOB 损失 (位) |
|-----------|----------|------------|-----------------|
| -100 | 90 | 89.6 | 0.07 |
| -90 | 90 | 87.8 | 0.37 |
| -80 | 90 | 79.6 | 1.73 |
| -70 | 90 | 69.9 | 3.34 |
| -60 | 90 | 60.0 | 4.99 |

THD 每恶化 10 dB（接近 SNR 范围），ENOB 大约损失 1.5-2 位。

## 6. 无噪声分辨率 vs ENOB

### 6.1 定义与关系

无噪声分辨率 (Noise-Free Resolution) 指输出码在统计上不跳动的位数。高斯噪声下峰峰值与 rms 的关系：`V_pp = 6.6 * V_rms`（覆盖 99.9% 概率），因此：

```
N_noise_free = ENOB - log2(6.6) = ENOB - 2.72
```

### 6.2 对比

| 指标 | 含义 | 适用场景 |
|------|------|----------|
| ENOB | 含噪声的 rms 等效位数 | 动态信号、频域分析 |
| 无噪声分辨率 | 码不跳动的位数 (ENOB-2.72) | 静态/直流测量、称重 |

称重秤等直流应用，无噪声分辨率更有实际价值：20 位无噪声分辨率意味着稳定输入下连续读数不会因噪声跳动。

## 7. 如何读懂 ADC 数据手册

### 7.1 关键参数

| 参数 | 符号 | 关注点 |
|------|------|--------|
| 分辨率 | N | 标称位数，不等于有效位数 |
| SINAD/SNR | - | 频域性能，直接算 ENOB |
| THD | - | 谐波失真，大信号时关注 |
| INL/DNL | - | 静态线性度，精密测量关注 |
| 输入噪声 | V_n | 峰峰值或 rms，直流应用关注 |

### 7.2 常见陷阱

**陷阱 1：用分辨率代替精度。** 某 24 位 ADC 首页大写"24-Bit"，内页 SINAD=108 dB，ENOB=17.7 位，差 6.3 位。

**陷阱 2：最佳条件数据。** SINAD 常在 f_in=1 kHz 测量，高频时性能下降。必须查看典型特性曲线确认你的工作点。

**陷阱 3：混淆 SNR 和 SINAD。** 厂商只标 SNR 不标 SINAD 时，按 SNR - 3 dB 保守估算 SINAD。

**陷阱 4：忽略输入范围。** 实际输入只有满量程 10% 时，信号功率降 20 dB，ENOB 损失约 3.3 位。

### 7.3 标准读法

1. 找 SINAD 在目标输入频率和采样率下的典型值
2. 算 ENOB = (SINAD - 1.76) / 6.02
3. 对比 ENOB 与标称 N，判断"水分"
4. 直流应用额外查 input noise 峰峰值，算无噪声分辨率
5. 确认测试条件 (温度、供电、参考电压) 是否与你的应用匹配

## 8. 实际测量 ENOB 的方法

### 8.1 频域法 (FFT)

1. 给 ADC 施加精确正弦波，频率选相干频率避免频谱泄漏
2. 采集 M 个样本 (M 为 2 的幂，通常 >= 8192)
3. 做 FFT，识别信号 bin、谐波 bin、噪声 bin
4. `SINAD = P_signal - P_(noise+harmonics) [dB]`，然后算 ENOB

相干采样条件：`f_in / f_s = M_exact / N_fft`（M_exact 为整数且与 N_fft 互质）

### 8.2 时域法 (直方图法)

对直流应用更实用：

1. 施加稳定直流输入，连续采集 >= 10,000 个样本
2. 统计输出码分布，计算标准差 sigma (单位: LSB)
3. `Effective_Resolution = N - log2(sigma)`
4. `Noise_Free_Resolution = N - log2(6.6 * sigma)`

### 8.3 测量注意事项

| 注意项 | 处理方法 |
|--------|----------|
| 信号源纯度 | 使用低失真信号源或滤波器 |
| 时钟质量 | 使用低抖动时钟源 |
| 样本数量 | FFT 点数 >= 8192 |
| 板级布局 | 独立 LDO 供电、充分去耦 |
| 温度 | 在目标温度下测量 |

## 9. 为什么 24 位 ADC 不会给你 24 位有效数据

### 9.1 现实数据

| 型号 | 标称位数 | 典型 ENOB | 无噪声分辨率 | 差距 |
|------|----------|-----------|--------------|------|
| ADS124S08 | 24 | ~21.5 | ~18.8 | 2.5-5.2 位 |
| AD7176-2 | 24 | ~22 | ~19.3 | 2-4.7 位 |
| ADS1256 | 24 | ~23 | ~20.3 | 1-3.7 位 |
| HX711 | 24 | ~17-19 | ~14-16 | 5-10 位 |

### 9.2 六大原因

1. **热噪声**：kT/C 噪声，室温下无法消除
2. **参考电压噪声**：VREF 的 ppm 级波动消耗 LSB
3. **1/f 噪声**：低频闪烁噪声，斩波可缓解但增加功耗
4. **增益级噪声**：PGA 引入额外噪声，高增益时更严重
5. **电源抑制有限**：PSRR 通常 60-80 dB，纹波泄漏
6. **温度漂移**：偏移和增益随温度漂移，长期稳定性受限

### 9.3 24 位何时有用

- **过采样增益**：低速应用中 ENOB 越接近 24 位
- **动态范围**：144 dB 动态范围，允许大/小信号共存
- **数字滤波**：后端滤波器进一步压低带内噪声
- **系统校准**：校准消除偏移和增益误差后可用位数增加

关键认知：24 位是数字输出格式宽度，不是性能承诺。

## 10. 过采样提升有效位数

### 10.1 原理

以 OSR 倍过采样后，噪声分布在 0~f_s，但信号带宽只到 f_s/(2*OSR)。数字滤波器滤除带外噪声：

```
SNR_gain = 10*log10(OSR) [dB]
额外位数 = 0.5 * log2(OSR)
```

| OSR | SNR 增益 (dB) | 额外位数 | 代价 |
|-----|---------------|----------|------|
| 4 | 6.02 | 1.0 | 采样率 x4 |
| 16 | 12.04 | 2.0 | 采样率 x16 |
| 64 | 18.06 | 3.0 | 采样率 x64 |
| 256 | 24.08 | 4.0 | 采样率 x256 |

每 4 倍过采样率 = 1 位额外精度。

### 10.2 Sigma-Delta 的天然优势

同时使用过采样和噪声整形，一阶 Sigma-Delta：`SNR = 6.02N + 1.76 + 30*log10(OSR) [dB]`

对比简单过采样的 10*log10(OSR)，噪声整形带来 3 倍 OSR 收益（二阶为 5 倍）。

### 10.3 过采样的限制

- 不改善失真 (THD 由非线性决定)
- 不消除 1/f 噪声
- 增加功耗和延迟

## 11. 系统级噪声预算

### 11.1 方法

把所有噪声源折算到输入端，找到瓶颈：

```
V_n_total = sqrt(V_n_therm^2 + V_n_ref^2 + V_n_pga^2
                 + V_n_source^2 + V_n_psrr^2 + V_n_emc^2)
ENOB_system = N - log2(V_n_total / LSB)
```

### 11.2 典型 24 位称重系统噪声预算

| 噪声源 | 折算到输入 (nV rms) | 占比 |
|--------|---------------------|------|
| ADC 内部热噪声 | 150 | 38% |
| 参考电压噪声 | 100 | 17% |
| PGA 噪声 | 120 | 24% |
| 传感器/源噪声 | 80 | 11% |
| 电源耦合 (PSRR) | 50 | 7% |
| EMI/串扰 | 30 | 3% |
| **总计** | **~238** | **100%** |

### 11.3 瓶颈定位策略

1. 短路 ADC 输入，测本底噪声 → ADC 自身贡献
2. 接入传感器但不激励 → 与步骤 1 差值即传感器噪声
3. 激励传感器 → 与步骤 2 差值即信号源噪声
4. 逐项打开电源/时钟 → 定位耦合路径

找到最大贡献者后集中优化，而不是盲目加位数。

## 12. IoT 传感器分辨率需求映射

### 12.1 常见传感器需求

| 传感器类型 | 量程 | 所需精度 | 所需 ENOB | 推荐 ADC |
|------------|------|----------|-----------|----------|
| 温度 (NTC) | -40~125 C | 0.1 C | ~11 位 | MCU 内置 12 位 |
| 温度 (RTD) | -200~850 C | 0.01 C | ~16 位 | 独立 16-24 位 |
| 湿度 | 0~100 %RH | 0.1 %RH | ~10 位 | MCU 内置 12 位 |
| 压力 (绝压) | 300~1100 hPa | 0.1 hPa | ~13 位 | 独立 16 位 |
| 称重 (低精度) | 0~5 kg | 1 g | ~13 位 | HX711 |
| 称重 (高精度) | 0~5 kg | 0.01 g | ~19 位 | ADS1256 |
| 气体 (CO2) | 0~5000 ppm | 10 ppm | ~9 位 | MCU 内置 12 位 |
| 光照度 | 0~100 klux | 1 lux | ~17 位 | 独立 16-24 位 |

### 12.2 所需位数计算

`所需ENOB = log2(量程 / 所需精度)`

示例：称重 5 kg，精度 0.01 g：`ENOB = log2(5/0.00001) = log2(500000) = 18.9`，需至少 19 位有效位数。

### 12.3 选型决策树

```
所需 ENOB <= 10?
  |-- 是 --> MCU 内置 12 位 ADC (省成本、省空间)
  |-- 否 --> 所需 ENOB <= 14?
        |-- 是 --> 独立 16 位 SAR ADC (速度精度平衡)
        |-- 否 --> 所需 ENOB <= 20?
              |-- 是 --> 24 位 Sigma-Delta (主流选择)
              |-- 否 --> 多次采样平均 + 24 位 Sigma-Delta
```

### 12.4 IoT 系统额外考量

- **功耗**：高分辨率 ADC 功耗更高，需平衡电池寿命
- **采样率**：Sigma-Delta 低速高精度，SAR 高速中等精度
- **校准**：IoT 现场温度变化大，需定期或上电校准
- **成本**：MCU 内置 ADC 基本免费，独立 ADC 增加物料成本

## 总结

1. **分辨率 != 精度**：N 位只说明能分 2^N 层，实际精度取决于 INL、DNL 和噪声
2. **6.02N+1.76** 是理想 ADC 理论极限，实际 ADC 永远达不到
3. **ENOB 是唯一真实指标**：从实测 SINAD 计算，代表实际可信赖的数据位数
4. **THD 大信号时是 ENOB 主要杀手**：谐波失真每恶化 10 dB 可吃掉 1.5-2 位
5. **无噪声分辨率 = ENOB - 2.72**：直流/静态应用应关注此指标
6. **24 位 ADC 有效位通常只有 17-23 位**：其余是数字格式宽度，不是性能
7. **过采样每 4 倍增加 1 位**：简单有效但受限失真和 1/f 噪声
8. **噪声预算找瓶颈再优化**：盲目加位数不如找最大噪声源集中改善
9. **从传感器需求倒推 ADC 需求**：用 log2(量程/精度) 算出所需 ENOB 再选型

数据手册首页的位数是营销数字，SINAD 表格里的数字才是工程数字。
