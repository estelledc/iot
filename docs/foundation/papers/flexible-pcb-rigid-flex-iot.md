# 柔性PCB与刚柔结合板在IoT可穿戴中的应用

> **难度**: 🟡 中级 | **领域**: 柔性电路设计 | **阅读时间**: 约 20 分钟

## 引言

想象一下: 你穿上一件运动衣,胸口处有一片薄如蝉翼的传感器贴片,它随着你的呼吸起伏,测量心率并实时上传数据。没有任何硬邦邦的"电路板"硌着你,你甚至感觉不到它的存在。这就是柔性PCB(Flexible Printed Circuit Board)在IoT可穿戴设备中扮演的角色 -- 像衣服的衬里一样,电路不再是"贴在身上的一块板子",而是"织入衣物的一层膜"。

传统刚性PCB像一块木板 -- 坚硬、平整、不可弯折;柔性PCB则像一张宣纸 -- 可以卷曲、折叠、贴合曲面;刚柔结合板(Rigid-Flex PCB)则像一本精装书 -- 书页是柔软的,书脊是坚硬的,两者合为一体,各司其职。

本文将从材料、结构、设计规则、制造工艺、可靠性测试等维度,系统梳理柔性PCB与刚柔结合板在IoT可穿戴场景中的应用要点。

## 1. 柔性PCB基础材料

### 1.1 聚酰亚胺(Polyimide, PI)

聚酰亚胺是柔性PCB最主流的基材,常用厚度12.5-125um。其核心特性:

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 玻璃化转变温度(Tg) | > 300°C | 远高于FR-4的130-180°C |
| 抗拉强度 | 200-400 MPa | 弯曲寿命长 |
| 介电常数(Dk) | 3.2-3.5 | 高频信号适用 |
| 吸水率 | 0.8-3.0% | 需注意潮湿环境变形 |
| 颜色 | 琥珀色/棕色 | 行业俗称"黄膜" |

PI膜的优势在于耐高温焊接(可承受280°C短时回流焊)和优异的弯折寿命,缺点是成本较高(约为PET的3-5倍)且吸水后尺寸稳定性下降。

### 1.2 聚酯薄膜(PET)

PET(Polyethylene Terephthalate)是另一种常见柔性基材,成本低、透明度好,但热稳定性远不如PI:

| 参数 | PI | PET |
|------|------|------|
| 最高工作温度 | > 250°C | 70-105°C |
| 弯曲寿命 | 极佳 | 中等 |
| 成本 | 高 | 低 |
| 典型应用 | 焊接电路 | 热熔连接/印刷电子 |

PET的核心限制是耐温不足,无法通过标准回流焊工艺,因此通常用于热熔连接(Hot Bar Soldering)或导电银浆印刷的简单电路,如一次性医疗贴片、电子标签等低成本场景。

### 1.3 覆盖层与粘合层

柔性PCB的"三明治"结构中,覆盖层(Coverlay)相当于刚性板阻焊层的角色,通常使用与基材同材质的PI膜+环氧胶粘合。粘合层(Adhesive)的厚度和类型直接影响弯折性能:

- 丙烯酸胶(Acrylic): 粘接力强,但厚度偏厚(25-50um),弯折寿命一般
- 环氧胶(Epoxy): 耐温好,厚度可控
- 无胶(Adhesive-less): 铜箔直接溅射/电镀在PI上,弯折寿命最优,成本最高

## 2. 柔性PCB的结构类型

### 2.1 单面柔性板

最简单的柔性PCB,一层铜箔在一面。适用于:

- LED灯条
- 简单传感器走线
- 电池连接器

布线全部在一面,过孔不可用,信号密度低,但成本最低、最薄(总厚度可控制在0.05-0.1mm)。

### 2.2 双面柔性板

铜箔在PI基材两面,通过过孔导通。相比单面板:

- 走线密度翻倍
- 支持电源/地平面
- 可实现阻抗控制
- 厚度增加到0.1-0.2mm

在IoT可穿戴中,双面板是最常见的性价比之选。典型应用: 智能手环主板、TWS耳机排线。

### 2.3 多层柔性板

三层及以上铜层的柔性板。层数越多,弯折性能越差(中性层应力分布不均),因此在可穿戴领域通常限制在4层以内。

### 2.4 刚柔结合板

刚柔结合板将刚性PCB区域(元器件焊接)与柔性PCB区域(弯折连接)一体化制造,无需板间连接器。

结构示意:

```
[刚性区]          [柔性区]          [刚性区]
+-----------+    +-----------+    +-----------+
| FR-4/PI   |    | PI only   |    | FR-4/PI   |
| 元器件     |====| 走线带    |====| 元器件     |
| BGA/QFN   |    | 无器件    |    | BGA/QFN   |
+-----------+    +-----------+    +-----------+
     |                弯折区               |
     +--- 连接器省略 ---+--- 连接器省略 ---+
```

核心优势: 消除连接器,减少组装点,降低故障率,缩小体积。核心代价: 制造复杂度高,良率偏低,单板成本显著上升。

## 3. 弯曲半径与应力设计

### 3.1 弯曲半径规则

弯曲半径(Bend Radius)是柔性PCB设计中最关键的几何参数。规则非常明确:

| 场景 | 最小弯曲半径 |
|------|-------------|
| 静态弯曲(安装时弯一次) | 板厚的6-10倍 |
| 动态弯曲(反复弯折) | 板厚的20-50倍 |
| 单面柔性板 | 板厚的6倍(静态) |
| 双面柔性板 | 板厚的10倍(静态) |
| 多层柔性板 | 板厚的20倍(静态) |

举例: 0.1mm厚的双面PI柔性板,静态弯曲半径至少1mm;动态弯曲半径至少2-5mm。

### 3.2 动态弯曲 vs 静态弯曲

- **静态弯曲**: 产品组装时弯折到位,之后不再运动。例如: 将柔性板弯折后塞入手表壳体内部。设计目标是"弯过去不断",弯折次数要求通常< 100次。

- **动态弯曲**: 产品使用过程中持续弯折运动。例如: 智能手表表带与表体之间的连接排线,随腕部运动反复弯折。设计目标是"弯N次不断",通常要求10万-100万次弯折循环。

动态弯曲的设计要求远严于静态弯曲:
1. 只允许单层铜在弯曲区(中性轴对齐)
2. 弯曲区不允许有焊盘和过孔
3. 走线方向垂直于弯曲轴线
4. 使用无胶(adhesive-less)基材延长寿命

### 3.3 中性轴理论

弯曲时,柔性板的"中性轴"(既不受拉也不受压的层面)应力为零。铜箔越靠近中性轴,弯折寿命越长。

设计技巧: 双面板如果两面铜厚不同,较厚的铜层应放在中性轴外侧(受拉侧),因为铜的抗拉强度远高于粘合层,而粘合层在压缩下更容易分层。

## 4. 补强板与无连接器设计

### 4.1 补强板(Stiffener)的使用

补强板是在柔性板局部区域增加硬度的辅助材料,常见用途:

| 补强材料 | 厚度范围 | 用途 |
|---------|---------|------|
| PI膜 | 25-125um | 薄区补强,折弯引导 |
| 不锈钢片 | 0.1-0.3mm | 连接器区,重元器件支撑 |
| FR-4 | 0.2-1.0mm | 插接金手指区 |

补强板的设计原则:
- 补强区与弯折区之间留0.5-1mm过渡区,避免应力集中
- 补强板边缘倒圆角,防止割伤柔性区
- 补强板与元器件不在同一面,否则焊盘平整度受影响

在IoT可穿戴中的典型应用: 智能戒指的电池焊接区加补强,防止焊接时PI膜变形。

### 4.2 无连接器设计优势

刚柔结合板的最大价值之一是消除连接器。连接器是可穿戴设备中最常见的故障点:

**传统方案**(两块刚性板+连接器):
- 连接器占体积: 高度2-5mm
- 接触电阻: 20-50mOhm(随使用劣化)
- 振动脱落风险: 中-高
- 组装工序: 插接+锁扣

**刚柔结合方案**(一块板,刚性区+柔性区):
- 柔性区厚度: 0.1-0.2mm
- 接触电阻: < 5mOhm(铜直连)
- 振动脱落风险: 无(一体结构)
- 组装工序: 折弯到位即可

在空间极度受限的可穿戴设备中,省掉连接器往往意味着整机厚度可以减薄0.5-1mm,这对于智能戒指、医疗贴片等产品是决定性的。

## 5. IoT可穿戴形态与柔性PCB

### 5.1 常见可穿戴形态因子

| 形态 | 典型尺寸 | 柔性PCB角色 | 弯曲类型 |
|------|---------|------------|---------|
| 智能手表 | 40-50mm圆/方 | 主板+表带排线 | 静态+动态 |
| TWS耳机 | 15-25mm | 电池-主板排线 | 静态 |
| 智能戒指 | 内径16-22mm | 环形柔性板 | 静态(曲率固定) |
| 医疗贴片 | 30-80mm | 传感器柔性载体 | 皮肤贴合(静态) |
| 智能服装 | 大面积 | 织物集成柔性条 | 动态(随身体运动) |
| AR眼镜 | 镜腿120-150mm | 镜腿-镜片连接带 | 动态(开合) |

### 5.2 医疗贴片中的柔性PCB

医疗贴片是柔性PCB最极致的应用场景之一。要求:

1. **极致薄**: 总厚度< 0.5mm(含元器件),贴在皮肤上无异物感
2. **生物相容**: 外壳材料需通过ISO 10993测试,PI膜本身惰性但焊接残留需清洗
3. **一次性/短期使用**: 多数贴片使用7-14天,寿命要求不高,可采用PET基材降本
4. **天线集成**: 柔性板边缘走环形天线,频率通常为2.4GHz(BLE)或UWB

典型架构:

```
+---------------------------------------------+
|  医疗贴片柔性板 (PI, 0.1mm)                  |
|                                             |
|  [MCU]---[传感器]---[柔性天线]              |
|    |                      |                 |
|  [电池]              [BLE射频]              |
|                                             |
|  背面: 医用压敏胶,贴合皮肤                   |
+---------------------------------------------+
```

### 5.3 智能服装集成

智能服装将柔性电路"织入"或"贴合"到织物上,面临独特挑战:

- **缝合 vs 热压**: 柔性板可缝合固定(缝线孔需预留),也可热压粘合(温度受限于织物耐温)
- **洗涤耐受**: 要求10-50次机洗后功能正常,柔性板需IP67级密封
- **拉伸补偿**: 织物有弹性,柔性板走线需留蛇形冗余段吸收拉伸
- **连接方式**: 柔性板与织物导电线的连接常用铆接(crimp)或导电胶

蛇形走线吸收拉伸的示意:

```
  普通走线:  ------------------  (拉伸即断)
  蛇形走线:  ∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿  (拉伸可展开)
```

## 6. 制造工艺与成本对比

### 6.1 柔性PCB制造流程

柔性PCB与刚性PCB的制造流程在核心步骤上相似,但细节差异显著:

| 工序 | 刚性PCB(FR-4) | 柔性PCB(PI) |
|------|---------------|-------------|
| 开料 | 玻纤布+环氧树脂 | PI膜+铜箔(有胶/无胶) |
| 钻孔 | CNC钻孔 | 激光钻孔(更小孔径) |
| 孔金属化 | 化学沉铜+电镀 | 同左,但需柔性适配 |
| 线路制作 | 曝光/蚀刻 | 同左,但需真空贴合(防气泡) |
| 覆盖层 | 液态阻焊油墨 | PI覆盖膜+冲切+热压 |
| 表面处理 | HASL/ENIG/OSP | ENIG为主(金手指区需镀厚金) |
| 外形加工 | 铣边(milling) | 钢模冲切/激光切割 |
| 检验 | AOI+电测 | AOI+电测+弯折测试 |

关键差异点:
1. 覆盖层: 刚性板用液态阻焊(可精密对位),柔性板用PI膜(需机械冲切开窗,精度略低)
2. 外形: 刚性板铣边可做复杂轮廓,柔性板小批量用激光切割(慢),大批量用钢模冲切(快但模具费高)
3. 载带: 柔性板太软无法在产线上自动传输,必须贴在刚性载带(carrier)上加工

### 6.2 刚柔结合板制造差异

刚柔结合板的制造流程更长,核心额外步骤:

1. 柔性区预成型: PI膜裁切出柔性区轮廓
2. 刚柔界面处理: 柔性层与刚性层在压制前需精确对位
3. 多次压合: 刚性区与柔性区分步压合
4. 空腔铣削: 刚性区多余部分铣掉,暴露柔性区

这些额外步骤导致:
- 制造周期: 比纯刚性板长30-50%
- 良率: 比纯刚性板低10-20%(刚柔界面分层是首要缺陷)
- 最小订单量: 通常50-100片起(刚性板可10片起)

### 6.3 成本对比

以一块30mm x 50mm的4层板为例:

| 项目 | 刚性PCB | 柔性PCB | 刚柔结合板 |
|------|---------|---------|-----------|
| 单板成本 | 1x (基准) | 3-5x | 5-10x |
| NRE费用 | 低 | 中(模具费) | 高(模具+工艺调试) |
| 组装成本 | 含连接器焊接 | 低 | 最低(无连接器) |
| 系统总成本 | 基准 | 1.2-1.5x | 1.5-2.5x |
| 体积 | 基准 | 减小30-50% | 减小40-60% |

注意: 单看PCB成本,柔性/刚柔结合方案远贵于刚性方案;但从系统总成本(含组装、连接器、外壳)看,差距缩小。如果体积缩小能带来外壳简化或功能增加,刚柔结合方案可能总体更优。

## 7. 可靠性测试

### 7.1 弯折循环测试

弯折循环(Bend Cycling)是柔性PCB最核心的可靠性测试:

| 测试标准 | IPC-6013 | JIS C 5016 |
|---------|----------|------------|
| 静态弯折 | 1-100次 | 1-50次 |
| 动态弯折 | 10k-1000k次 | 10k-500k次 |
| 弯曲半径 | 按设计值 | 按设计值 |
| 判据 | 电阻变化< 10% | 导通/绝缘合格 |

测试方法:
1. 取样: 从生产批中随机抽5-10片
2. 预处理: 85°C/85%RH, 24小时
3. 弯折: 以规定半径和频率反复弯折
4. 中间检测: 每10%进度测一次电阻
5. 终检: 金相切片检查铜箔裂纹

### 7.2 其他可靠性测试

| 测试项目 | 条件 | 目的 |
|---------|------|------|
| 高温高湿 | 85°C/85%RH, 240h | 考察粘合层吸湿分层 |
| 热冲击 | -40°C ~ +125°C, 100次循环 | 考察不同CTE材料界面 |
| 振动 | 10-2000Hz, 3轴各4h | 考察焊点与过孔 |
| 拉伸剥离 | 180度剥离, 速度50mm/min | 考察覆盖层附着力 |
| 盐雾 | 5% NaCl, 35°C, 48h | 考察镀层耐腐蚀(穿戴汗液) |

在IoT可穿戴设备中,汗液腐蚀是一个常被低估的风险。人体汗液含NaCl、乳酸等,长期接触可能导致金手指区镀层腐蚀,因此贴肤产品需要额外的密封设计(parylene涂层或硅胶包封)。

## 8. 柔性PCB设计规则

### 8.1 弯折区走线规则

弯折区的走线设计直接决定弯折寿命,核心规则:

1. **走线方向垂直于弯折轴**: 弯折时走线受拉/压应力最小
2. **避免直角走线**: 弯折区走线用圆弧过渡,禁止90度拐角
3. **走线宽度均匀**: 弯折区不允许走线宽度突变(宽窄过渡在弯折区外完成)
4. **间距加大**: 弯折区走线间距比平面区加大20-30%,防止弯折时短路
5. **避免过孔**: 弯折区内禁止放置过孔,过孔应距弯折区至少1mm

弯折区与非弯折区走线对比:

```
弯折区走线(理想):
  ===----====----====----====----====
       圆弧过渡    均匀宽度    垂直弯折轴

弯折区走线(错误):
  ===|===L====L====|====L====
      直角   窄变宽  过孔     直角
```

### 8.2 泪滴焊盘(Teardrop Pads)

泪滴焊盘是柔性PCB设计的标志性特征。与刚性板不同,柔性板在弯折时焊盘与走线的连接处应力集中,普通圆形/方形焊盘容易从连接处撕裂。

泪滴焊盘的形状: 焊盘在走线连接方向拉出一个"水滴尾",使铜面积从焊盘到走线平滑过渡,避免应力突变。

设计参数:
- 泪滴长度: 通常为焊盘直径的0.5-1.0倍
- 泥滴宽度: 从焊盘边缘线性收窄到走线宽度
- 适用范围: 弯折区及弯折区附近2mm内的所有焊盘

```
普通焊盘:          泪滴焊盘:
  (O)---           <(O)---
  突变,应力集中      渐变,应力分散
```

### 8.3 其他关键设计规则

| 规则 | 刚性PCB | 柔性PCB |
|------|---------|---------|
| 最小线宽/线距 | 3/3 mil | 4/4 mil(弯折区5/5) |
| 焊盘到走线过渡 | 可选泪滴 | 必须泪滴 |
| 覆铜 | 大面积铺铜 | 弯折区禁止铺铜 |
| 过孔到弯折区距离 | 不适用 | >= 1mm |
| 铜皮厚度 | 1 oz(35um)为主 | 1/3 oz(12um)或1/2 oz(18um) |
| 外形公差 | +/- 0.1mm | +/- 0.05mm(冲切) |
| 覆盖层开窗 | 阻焊开窗 | 冲切开窗,公差+/- 0.1mm |

弯折区禁止铺铜的原因: 大面积铜在弯折时产生极大应力,是分层和铜裂的元凶。

## 9. EDA工具对柔性设计的支持

### 9.1 主流EDA工具对比

| 功能 | Altium Designer | KiCad | Cadence Allegro |
|------|----------------|-------|-----------------|
| 柔性板层定义 | 支持 | 支持(7.0+) | 支持 |
| 弯折区定义 | 图形标注 | 图形标注 | Zone定义+3D显示 |
| 泪滴焊盘 | 自动生成 | 自动生成(8.0+) | 自动生成 |
| 刚柔结合3D预览 | 支持 | 支持(7.0+) | 支持(最佳) |
| 弯曲半径检查 | DRC规则 | 手动检查 | DRC规则 |
| 覆盖层设计 | 支持 | 支持覆盖层 Gerber | 支持 |
| 刚柔层堆叠管理 | 较完善 | 基础 | 最完善 |

### 9.2 柔性PCB设计在EDA中的特殊设置

1. **层堆叠定义**: 柔性层的材料属性需单独指定(PI vs FR-4),铜厚通常选1/3 oz
2. **弯折区域标注**: 用机械层绘制弯折区域轮廓,标注弯曲半径和方向
3. **覆盖层开窗**: 覆盖层(Coverlay)作为独立层,在焊盘位置开窗
4. **补强板定义**: 补强板作为机械层,不影响电气连接但需在Gerber中输出
5. **刚柔结合板叠层**: 不同区域定义不同层堆叠,这是最复杂的部分

Cadence Allegro在刚柔结合板设计方面最为成熟,支持在同一个设计中定义多个叠层区域(stackup zone),并在3D视图中模拟弯曲效果。Altium Designer也有类似功能但操作稍繁琐。KiCad从7.0版本开始支持柔性板层定义,适合预算有限的团队。

### 9.3 Gerber输出注意事项

柔性PCB的Gerber输出比刚性板多以下文件:

- 覆盖层(Coverlay)正面/反面
- 补强板轮廓
- 弯折线(Bend Line)
- 载带轮廓(Carrier Tape)

建议在Gerber中明确标注弯折方向(向上/向下)和弯曲半径,减少制造端理解偏差。

## 10. 设计检查清单

以下是在IoT可穿戴项目中使用柔性PCB时的核心检查项:

1. 弯曲半径是否满足静态/动态要求
2. 弯折区是否有过孔或焊盘(不应有)
3. 弯折区走线是否垂直于弯折轴
4. 焊盘是否全部使用泪滴形状
5. 弯折区是否铺铜(不应铺)
6. 补强板与弯折区之间是否有过渡区
7. 覆盖层开窗是否覆盖所有焊盘
8. 走线宽度在弯折区是否均匀
9. 刚柔界面是否有应力释放设计(如圆锥形过渡)
10. Gerber是否包含覆盖层和补强板文件

## 总结

柔性PCB和刚柔结合板是IoT可穿戴设备实现"小、轻、软"三大目标的关键使能技术。核心要点回顾:

1. **材料选择**: PI耐温好、弯折寿命长,是主力;PET成本低但只能低温工艺,适合一次性场景
2. **结构决策**: 单面/双面柔性板覆盖大多数IoT场景;刚柔结合板在需要消除连接器时才引入
3. **弯曲设计**: 弯曲半径是生命线,动态弯曲要求半径为板厚的20-50倍;弯折区走线垂直于弯折轴,禁止过孔和铺铜
4. **成本权衡**: 柔性板单板成本是刚性板的3-5倍,刚柔结合板是5-10倍;但系统总成本差距缩小,体积收益可能抵消PCB溢价
5. **可靠性**: 弯折循环测试是最关键的验证手段,汗液腐蚀是穿戴产品独有的可靠性挑战
6. **设计工具**: 主流EDA均已支持柔性板设计,刚柔结合板的层堆叠管理是工具选型的关键差异点

对于IoT可穿戴产品的硬件工程师,建议按以下路径逐步掌握柔性PCB设计:
- 第一阶段: 从纯柔性板(双面PI)入手,完成一个简单传感器排线设计
- 第二阶段: 尝试刚柔结合板,理解层堆叠区域定义和3D弯折预览
- 第三阶段: 深入动态弯曲设计,积累弯折寿命预估和验证经验

柔性电路设计是一门"弯"的艺术 -- 每一次弯折都需要精确计算,每一个过渡都需要平滑处理。掌握这些规则,才能让电路真正"贴"在人体上,成为可穿戴设备不可或缺的骨架。
