# PCB地平面分割与混合信号设计
> **难度**: 🔴 高级 | **领域**: 混合信号PCB设计 | **阅读时间**: 约 22 分钟

## 引言

想象一座城市,一侧是图书馆(模拟电路),需要绝对安静;另一侧是工地(数字电路),重型机械轰鸣不断。如果两区共用同一条下水道,工地冲刷的泥沙迟早会倒灌进图书馆的地下室。PCB混合信号设计面对的就是这个问题:模拟信号对噪声极度敏感,数字开关又天生吵闹,而它们共享同一块地平面。

地平面不是简单的"零电位参考" -- 它是所有信号的回流路径。回流电流怎么走,决定了噪声怎么耦合。理解回流路径,就理解了地平面分割的所有争论。

本文从回流路径的基本物理出发,分析分割地与统一地的争论,给出ADC/DAC接地策略,最后落到IoT场景的实际设计指南。

## 1 混合信号设计的核心矛盾

### 1.1 模拟电路的噪声敏感性

模拟信号处理对噪声的容忍度极低:

- 动态范围约束: 24位ADC理论动态范围约146dB,电源轨噪声仅1mV就可能吞掉最后4位有效精度
- 增益链放大: 前级放大器的微伏级噪声,经多级放大后在ADC输入端变成毫伏级
- 频率混叠: 带外噪声经采样后折叠进基带,无法后期滤除

模拟电路追求的核心目标:信号路径上任何两点之间的电位差,必须远小于最小可分辨信号。

### 1.2 数字电路的开关噪声

- 地弹(Ground Bounce): 输出驱动器同时翻转时,地回路电感上的瞬态压降可达数百毫伏
- 电源纹波: CMOS开关产生di/dt尖峰,频率分量延伸到数百MHz
- 串扰: 快速边沿通过互容和互感耦合到邻近走线

数字信号典型边沿速率:1-5V/ns。1ns上升时间的有效频率分量约350MHz(0.35/1ns)。

### 1.3 矛盾的本质

两类电路共享地平面时,数字开关噪声通过地平面阻抗耦合到模拟参考点。这不是"要不要隔离"的问题,而是"如何管理回流路径"的问题。

```
+------------------+     +------------------+
|  数字侧          |     |  模拟侧          |
|  MCU / RF / DSP  |     |  传感器AFE / 滤波 |
|       v          |     |       v          |
|  [数字地]--------+-----+------[模拟地]     |
|    噪声电流 -->        <-- 期望干净参考    |
+------------------+     +------------------+
         共享阻抗 = 噪声耦合通道
```

## 2 地平面理论与回流路径

### 2.1 回流电流的基本物理

回流电流遵循最小阻抗路径:

- 低频(< 1MHz): 走最小电阻路径,即最短几何距离
- 高频(> 1MHz): 走最小电感路径,即紧贴信号走线下方

高频回流紧贴信号走线,因为回流与信号形成的回路面积最小,电感最小。这是物理定律,不是设计选择。

```
信号走线  =========================================>
                    |  回流路径
低频回流   <--------+-------------------------------  (最短距离)
高频回流   <========+===============================  (紧贴走线下方)
                    |
              地平面截面
```

### 2.2 地平面阻抗

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 直流电阻 | 约0.7mΩ/sq (1oz铜) | 低频主导 |
| 趋肤效应电阻 | 随频率增加 | 高频时电流集中在表面 |
| 转移阻抗 | 与过孔密度、分割方式相关 | 决定跨区耦合 |

关键洞察:完整地平面的转移阻抗远低于任何分割方案。分割总是增加阻抗。

### 2.3 为什么回流路径如此重要

回流路径偏离信号走线时:

1. 回路面积增大 --> 辐射发射增加(EMI)
2. 回路电感增大 --> 信号完整性下降(振铃、过冲)
3. 回流绕行其他区域 --> 跨域噪声耦合

经验法则:任何迫使回流绕行的地平面不连续,都是潜在EMI源和噪声耦合源。

## 3 分割地 vs 统一地:一场持续30年的争论

### 3.1 分割地策略(Split Ground Plane)

做法:将地平面分割为AGND和DGND,在一点(通常在ADC下方)用短而宽的连接桥连通。

主张逻辑:数字噪声电流被限制在DGND内;AGND保持"干净";单点连接防止地环路。

```
+-------------------+     +-------------------+
|     DGND          |     |     AGND          |
|   数字IC          |     |   运放 / 传感器    |
|                   | === |                   |  <-- 单点连接桥
+-------------------+     +-------------------+
```

### 3.2 统一地策略(Unified Ground Plane)

做法:保持地平面完整不分,通过物理分区和走线约束隔离模拟与数字区域。

主张逻辑:回流路径永远紧贴信号走线,不会绕行;完整地平面提供最低阻抗;分割迫使跨区信号回流绕行,反而增加耦合。

Henry Ott:"不要分割地平面。通过分区和布局来实现隔离。"

### 3.3 争论的根源

| 维度 | 分割地支持者 | 统一地支持者 |
|------|-------------|-------------|
| 关注频率 | 音频至低频(<1MHz) | 高速数字(>10MHz) |
| 主要矛盾 | 地线电阻压降 | 地平面电感与EMI |
| 典型应用 | 音频设备、低速仪表 | 高速通信、处理器系统 |
| 设计时代 | 单面板/双面板 | 多层板 |

Eric Bogatin:"在多层板设计中,只要遵守分区规则,统一地永远优于分割地。唯一例外是安全隔离(如医疗设备患者连接部分)。"

### 3.4 折中:分区而不分割

目前主流推荐做法:

1. 板子上明确划分模拟区和数字区
2. 地平面保持完整,不切割
3. 模拟走线绝不进入数字区,反之亦然
4. ADC/DAC跨区放置,下方是两个区域的交界
5. 电源分别滤波,但地平面统一

这就是"分区(Partition)而不分割(Split)"的原则。

## 4 星形接地与单点连接

### 4.1 星形接地原理

星形接地:所有地连接汇聚到一个物理点,形如星形。各分支之间没有共享地阻抗,理论上零串扰。

```
                    运放地
                     |
  传感器地 ------ 星形中心点 ------ ADC地
                     |
                   电源地
```

### 4.2 星形接地的局限

- 高频失效: 高频回流不走星形,而是走最小电感路径直接穿透地平面
- 物理实现困难: 多层板上走线长且电感大
- 不适合复杂系统: 50个器件不可能做50条星形地线

星形接地适合低频精密模拟电路(称重传感器、热电偶前端),不适合包含高速数字的混合信号系统。

### 4.3 单点连接的正确实现

当确实需要AGND和DGND单点连接时:

1. 连接点选在ADC/DAC正下方
2. 连接桥短而宽(长宽比 < 5:1),最小化桥上电感
3. 桥上不放任何元件或过孔(ADC接地引脚除外)
4. 确保跨区信号回流只通过此桥,不绕行板边

```
          连接桥(短而宽)
     DGND ======== AGND
              |
           ADC芯片
        (跨区放置,骑在边界上)
```

## 5 ADC/DAC接地策略

### 5.1 ADC的接地困境与正确做法

ADC是混合信号器件:模拟输入需干净参考,数字输出产生开关噪声。ADC的AGND和DGND引脚都应连接到统一地平面(在分割方案中则都接AGND平面)。

原因:
- ADC内部模拟电路参考就是AGND引脚
- 数字输出驱动器的地弹是芯片内部事件,不影响模拟精度(前提是外部回流路径正确)
- 将DGND引脚接远处DGND平面会迫使数字输出回流绕行,地弹更严重

### 5.2 DAC的对称策略

- DAC模拟输出参考AGND,数字输入端回流也回到AGND(统一地)
- 模拟输出走线立即进入模拟区,远离数字开关走线

### 5.3 ADC/DAC布局要点

```
+-- 数字区 ----------+--- 边界 ---+-- 模拟区 ---------+
|                     |            |                    |
|  MCU数据总线        |  ADC芯片   |  模拟输入滤波      |
|  ----走线----> |AIN AGND DGND D0-D11| ----走线--->  运放  |
+---------------------+---+---+----+--------------------+
                          |   |
                    统一地平面(不分)
```

关键约束:
- 数字输出走线往数字区走,模拟输入走线往模拟区走
- 绝不让数字走线穿越模拟区
- ADC下方和周围不放数字过孔
- 去耦电容紧贴ADC电源引脚放置

## 6 地平面开槽与不连续

### 6.1 开槽的形成与影响

地平面开槽(Slot)常由以下原因造成:人为分割AGND/DGND、密集过孔形成的"护城河"、连接器占位切断地铜皮、电源平面侵入。

当信号走线跨越开槽时,回流被迫绕行:

```
信号走线  ==================|==================>
                          |开槽|
回流路径  <===============+    +===============  (绕行!)
                    ------+    +------
                    |     开槽区域     |
```

后果:回路面积增大致辐射增加(可达20dB+);回路电感增大致信号退化;绕行回流经过其他区域致串扰。

### 6.2 跨开槽走线的定量影响

估算:1英寸长、0.05英寸宽的开槽,信号频率200MHz时:

- 完整地平面回流电感: 约1nH
- 绕行回流电感: 约10-20nH (取决于绕行距离)
- 增量噪声: V = L * di/dt, 若di/dt = 50mA/ns,则约0.5-1.0V

这个量级足以破坏大多数模拟信号。

### 6.3 必须开槽时的缓解措施

1. 在开槽上跨接0欧姆电阻或小电感,为回流提供桥接
2. 开槽方向与信号走线平行(不横切信号路径)
3. 开槽两端各放去耦电容,为高频回流提供交流通路
4. 不让任何高速信号跨越开槽

```
                0R          0R
回流路径  <----[R]----[R]---->  (桥接)
          ------+   开槽   +------
```

## 7 地平面不连续与EMI

### 7.1 EMI产生机制

地平面不连续导致EMI的三步链:

1. 信号电流被迫沿较长路径回流 --> 回路面积A增大
2. 差模辐射场强正比于回路面积: E_dm ~ f^2 * A * I
3. 地平面缝隙充当缝隙天线,进一步放大辐射

### 7.2 常见不连续源与修复

| 不连续源 | EMI风险 | 修复方法 |
|----------|---------|----------|
| 地平面分割 | 高 | 改为分区不分割 |
| 过孔围栏 | 中 | 增加地回流过孔 |
| 连接器区 | 高 | 在连接器下铺完整地 |
| 电源岛入侵 | 中 | 在电源岛边缘放去耦电容 |
| 板边未处理 | 低 | 板边加地线护圈 |

连接器区域特殊要求:信号引脚旁必须有地引脚(至少1:3信号地比);连接器下方保持完整地平面;差分对地回流过孔紧贴差分过孔对放置。

## 8 实用设计指南

### 8.1 Henry Ott的建议

1. 永远不要在多层板上分割地平面(安全隔离除外)
2. 正确分区:模拟器件放模拟区,数字器件放数字区
3. 绝不让数字走线穿越模拟区,反之亦然
4. 电源可用铁珠或电感隔离,但地平面保持统一
5. ADC/DAC跨区放置,骑在分区边界上

### 8.2 Eric Bogatin的建议

1. 地平面唯一任务是提供低阻抗回流路径,不要增加其他职责
2. "安静地"和"吵闹地"的区别是错误的,地只是参考节点
3. 分割地迫使回流绕行,是最常见的EMI错误
4. 去耦电容位置比地平面分割方式重要10倍
5. 频率高于1MHz时,地平面电感比电阻重要100倍

### 8.3 分步设计流程

1. **器件分区**: 原理图阶段划分模拟和数字区域
2. **布局**: 模拟/数字器件分置两侧,ADC/DAC放中间
3. **地平面**: 内层2做完整地平面,不分割
4. **电源层**: 模拟/数字电源用铁珠隔离,各自紧贴去耦
5. **走线约束**: 模拟走线只在模拟区,数字走线只在数字区
6. **过孔管理**: 数字区过孔不打在模拟区正下方
7. **验证**: 用回流路径分析工具检查所有高速信号回流

### 8.4 检查清单

- [ ] 地平面完整,无分割?
- [ ] 模拟/数字器件物理分区?
- [ ] 无数字走线穿越模拟区?
- [ ] ADC/DAC放在分区边界?
- [ ] 去耦电容紧贴IC引脚?
- [ ] 电源隔离用铁珠而非地分割?
- [ ] 连接器下方地平面完整?
- [ ] 高速差分对有紧邻的地回流过孔?

## 9 IoT混合信号设计实例

### 9.1 典型架构:传感器AFE + 无线MCU

IoT设备最典型的混合信号架构:

```
传感器 --> AFE(运放+滤波) --> ADC --> MCU --> 无线模块
              模拟区              |      数字区
                                 |
                            分区边界
```

具体案例:温湿度传感器 + BLE无线传输

- 传感器输出:0-1V模拟信号(热敏电阻分压)
- AFE:仪表放大器(INA333),增益100倍
- ADC:24位Sigma-Delta(ADS1220)
- MCU/无线:nRF52832(BLE 5.0)
- 采样率:10SPS(低频,但数字无线部分时钟64MHz)

### 9.2 接地方案

四层板叠层:

```
Layer 1 (Top):    信号走线 + 器件
Layer 2 (GND):    完整地平面,不分割
Layer 3 (Power):  VDD_ANA / VDD_DIG,用铁珠隔离
Layer 4 (Bot):    低速信号 + 辅助地填充
```

关键决策:
1. ADS1220的AGND和DGND都接Layer 2统一地平面
2. nRF52832数字地也接Layer 2
3. VDD_ANA通过铁珠从主电源分出,紧贴去耦
4. 天线下方Layer 2地平面保持完整

### 9.3 常见IoT设计陷阱

**陷阱1**: 天线正下方分割地平面。有人为"隔离RF噪声"在天线下方切割地平面,结果天线效率下降10-15dB(地平面是单极天线镜像面)。正确:天线区地平面完整,RF匹配网络放天线附近。

**陷阱2**: 传感器走线紧挨RF走线。RF功放输出可达+10dBm,若与传感器输入走线平行,耦合噪声远大于传感器信号。正确:RF走线在数字区侧,传感器走线在模拟区侧,中间用地平面隔离。

**陷阱3**: 铁珠选型不当。选型要看阻抗曲线:100MHz处阻抗>600Ω(抑制高频);直流电阻<0.5Ω(不影响供电);额定电流>2倍工作电流(避免饱和)。

### 9.4 电池供电的特殊考量

IoT设备多为电池供电:

- 休眠时地回路电流仅微安级,地平面压降可忽略
- RF发射瞬间电流达数十毫安,di/dt极大
- 电池内阻(数十到数百毫欧)是数字噪声进入模拟参考的潜在通道

缓解:电池输出端并联大容量(10uF以上)低ESR电容,为RF脉冲提供本地能量,降低电池内阻上的瞬态压降。

## 10 常见错误与修正

### 10.1 盲目分割地平面

错误认知: 把模拟地和数字地分开就能隔离噪声。
正确理解: 分割迫使回流绕行,绕行路径耦合面积远大于紧贴回流面积,噪声不减反增。
修正: 统一地平面 + 物理分区隔离。

### 10.2 ADC的DGND引脚接DGND平面

错误认知: DGND引脚名字里有"Digital",应接数字地平面。
正确理解: ADC内部地弹是芯片内部事件。DGND接远处DGND平面,数字输出回流绕行,地弹更严重。
修正: ADC所有地引脚都接统一地平面,在ADC正下方。

### 10.3 铁珠两端都加去耦电容

错误认知: 铁珠两侧各放一个去耦电容增加滤波效果。
正确理解: 铁珠+两侧电容构成LC谐振电路,谐振频率(1-10MHz)处阻抗更低,噪声直接穿透。
修正: 负载侧(模拟侧)加大容量电容(10uF),源侧不额外加电容,或加阻尼电阻降低Q值。

### 10.4 模拟区过孔打到地平面下方

错误认知: 过孔只是层间连接通道,不影响模拟区。
正确理解: 数字信号过孔穿过地平面时,回流在地平面上产生局部扰动,若在敏感模拟走线下方,直接耦合到模拟信号。
修正: 数字区过孔远离模拟走线下方地平面区域,用回流分析工具验证。

### 10.5 地平面开窄缝"隔离"

错误认知: 开一条细缝就能阻止噪声传播。
正确理解: 窄缝对高频电流几乎无阻挡(通过缝间电容耦合穿过);缝隙只对低频有效,而低频是电阻性耦合,铁珠更有效。
修正: 不开缝,需要隔离时用铁珠或0欧姆电阻在电源路径上隔离,地平面保持完整。

## 总结

PCB地平面分割与混合信号设计的核心要义,浓缩为一条原则:

**分区而不分割,管理回流而非隔离地。**

关键要点回顾:

1. 回流电流走最小阻抗路径 -- 低频走电阻最小,高频走电感最小(紧贴信号走线)
2. 地平面首要职责是提供低阻抗回流路径,不是"隔离噪声"
3. 分割地平面迫使回流绕行,增大回路面积和电感,是EMI和噪声耦合的主要来源
4. 统一地平面+物理分区是目前多层板混合信号设计的主流最佳实践
5. ADC/DAC地引脚都接统一地平面,芯片骑在模拟/数字区边界
6. 电源隔离用铁珠,地平面不分割
7. 开槽和不连续是回流的天敌,任何迫使回流绕行的结构都是风险点
8. IoT设备中,天线区和传感器区的地平面完整性尤其重要

最后的忠告:当你想要分割地平面时,先画出所有信号的回流路径。如果回流路径面积比你预期的更大,分割就是错误的。
