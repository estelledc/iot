# IP防护等级与IoT设备外壳设计

> **难度**：🟢 初级 | **领域**：结构防护设计 | **阅读时间**：约 18 分钟

## 引言

你出门前会看天气决定穿什么——小雨穿冲锋衣，大雨穿雨衣，台风天就不出门了。IoT设备没法选天气，它必须"穿对衣服"站在野外日晒雨淋。IP防护等级就是这身"衣服"的规格标签，告诉你它能挡住多大的风雨。本篇从IP等级解读入手，讲清楚如何设计一个合格的IoT设备外壳。

## 1. IP防护等级体系

### 1.1 IEC 60529标准

IP防护等级由国际标准IEC 60529定义，格式为"IP"后跟两位数字：

```
IP 6 7
   |  |
   |  +-- 第二位数字: 防水等级 (0-9K)
   +----- 第一位数字: 防尘/防固体等级 (0-6)
```

### 1.2 第一位数字：防固体异物

| 等级 | 防护程度 | 典型测试 |
|------|----------|----------|
| 0 | 无防护 | — |
| 1 | 防大于50mm固体(手背) | 50mm直径探棒 |
| 2 | 防大于12.5mm固体(手指) | 12.5mm探棒 |
| 3 | 防大于2.5mm固体(工具) | 2.5mm探棒 |
| 4 | 防大于1mm固体(细线) | 1mm探线 |
| 5 | 防尘(允许少量灰尘进入，不影响运行) | 粉尘箱8小时 |
| 6 | 尘密(完全阻止灰尘进入) | 粉尘箱8小时 |

### 1.3 第二位数字：防水

| 等级 | 防护程度 | 典型测试 |
|------|----------|----------|
| 0 | 无防护 | — |
| 1 | 防垂直滴水 | 1mm/min垂直滴水 |
| 2 | 防15度倾斜滴水 | 四个固定角度滴水 |
| 3 | 防淋水(60度角喷淋) | 摆管喷淋 |
| 4 | 防溅水(任意方向) | 摆管喷淋 |
| 5 | 防低压水柱喷射(6.3mm喷嘴) | 30kPa, 12.5L/min |
| 6 | 防强力水柱喷射(12.5mm喷嘴) | 100kPa, 100L/min |
| 7 | 防短时浸水(1m, 30min) | 水箱浸没 |
| 8 | 防持续浸水(深度与时间由厂商定义) | 水箱长期浸没 |
| 9K | 防高温高压水柱喷射 | 80C, 100bar |

注意：IP67和IP68并非递进关系——IP7是短时浸水，IP8是持续浸水，但IP8不一定能承受高压水柱(IP6的测试)。

## 2. IoT设备常见IP等级选择

### 2.1 典型场景与IP等级

| 部署场景 | 推荐IP等级 | 理由 |
|----------|------------|------|
| 室内有遮挡 | IP20-IP40 | 仅需防触碰，无防水要求 |
| 室内有空调机房 | IP54 | 防尘+防溅水(空调冷凝水) |
| 户外屋檐下 | IP54-IP65 | 防雨飘+防尘 |
| 户外露天 | IP65-IP67 | 防暴雨/短时浸水 |
| 地下/易积水 | IP67-IP68 | 可能被水淹没 |
| 需高压冲洗(食品/化工) | IP69K | 高温高压冲洗 |

### 2.2 过度设计的代价

IP等级越高，外壳成本、密封难度、散热限制越大。IP67以上的设计需要全密封，对散热和气压平衡都带来挑战。选择IP等级的原则是**够用就好**。

## 3. 外壳材料

### 3.1 常见材料对比

| 材料 | 优点 | 缺点 | 适用IP等级 | 成本 |
|------|------|------|------------|------|
| ABS塑料 | 易成型、低成本 | 耐候差(UV老化)、强度一般 | IP54以下 | 低 |
| 聚碳酸酯(PC) | 透明、抗冲击、耐温 | 耐化学性一般 | IP65-IP67 | 中 |
| 铝合金压铸 | 高强度、散热好 | 重量大、需表面防腐 | IP67以上 | 中高 |
| 不锈钢(316) | 极耐腐蚀 | 重、贵、加工难 | IP68/69K | 高 |
| 玻纤增强PA | 强度高、耐化学 | 不透明、吸水率需关注 | IP67 | 中 |

### 3.2 塑料外壳的耐候问题

ABS在户外紫外线照射下会发黄变脆，使用寿命通常3-5年。解决方案：
- 添加UV稳定剂(使寿命延长到7-10年)
- 表面喷涂抗UV涂层
- 改用耐候性更好的ASA或PC/ABS合金

## 4. 密封方法

### 4.1 密封方式对比

**O型圈密封**
- 原理：弹性O型圈被压缩后填满间隙
- 优点：标准化件、可拆卸、密封可靠
- 缺点：需要精确的沟槽设计
- 适用：上下盖配合、观察窗

**平面垫圈(Flat Gasket)**
- 原理：弹性材料片被螺栓压力压紧
- 优点：加工简单、适应不规则形状
- 缺点：需要较大压缩力，螺栓间距要求高
- 适用：方形外壳的盖板密封

**超声波焊接**
- 原理：高频振动使塑料接合面熔化并融合
- 优点：密封可靠、生产效率高
- 缺点：不可拆卸、需要专用设备
- 适用：一次性密封的塑料外壳

**胶水粘接**
- 原理：密封胶填满接合面间隙
- 优点：适应复杂形状、成本低
- 缺点：固化时间、返修困难、长期老化
- 适用：小批量、非拆卸场景

**灌封(Potting)**
- 原理：用环氧或聚氨酯树脂将整个PCB包裹
- 优点：最高防护等级、同时提供机械减振
- 缺点：不可返修、散热受限、增重
- 适用：极端环境、一次性产品

### 4.2 O型圈沟槽设计要点

O型圈的密封效果取决于压缩率(通常15-30%)：

```
压缩率 = (O圈截面直径 - 沟槽深度) / O圈截面直径 x 100%

例: O圈截面直径2mm, 沟槽深度1.7mm
压缩率 = (2.0 - 1.7) / 2.0 x 100% = 15%
```

关键参数：
- 沟槽宽度应大于O圈截面直径(留膨胀空间)
- 沟槽表面粗糙度Ra < 1.6um(防止O圈被划伤)
- 避免尖角，所有转角加圆角R > 0.2mm

## 5. 电缆入口方案

### 5.1 电缆接头(Cable Gland)

电缆接头是穿墙密封的核心元件，常见规格：

| 规格 | 螺纹类型 | 适用电缆直径 | 典型IP等级 |
|------|----------|--------------|------------|
| PG7 | PG螺纹 | 3-6.5mm | IP68 |
| PG9 | PG螺纹 | 4-8mm | IP68 |
| M12 | 公制螺纹 | 3-6.5mm | IP68 |
| M16 | 公制螺纹 | 5-10mm | IP68 |
| M20 | 公制螺纹 | 7-13mm | IP68 |

安装步骤：
1. 在外壳上加工安装孔(配合螺纹或通孔)
2. 穿入电缆接头并拧紧螺母
3. 穿入电缆，拧紧接头尾部密封爪
4. 确认密封爪均匀包裹电缆

### 5.2 IP级连接器

对于需要频繁插拔的场景，使用IP67/IP68级连接器：
- M12连接器：圆形，4-8芯，IP67，工业标准
- M8连接器：小型圆形，3-4芯，IP67
- USB/IP67组合：带密封盖的USB接口

### 5.3 膜式透气阀

全密封外壳有一个容易被忽视的问题——气压变化。

当环境温度变化时，密封壳内空气热胀冷缩：
- 温升时内部气压升高，可能撑开密封面
- 温降时内部气压降低，外部水汽被"吸"入微隙

膜式透气阀(如Gore-Tex材质)解决了这个问题：
- 允许空气分子自由通过(ePTFE膜孔径约0.2um)
- 阻止水分子通过(水滴远大于膜孔)
- 实现壳内外气压平衡，防止密封失效
- 同时允许壳内水汽扩散出去，减少凝露

典型安装：在外壳壁上打M12螺纹孔，拧入透气阀，IP67/IP68等级不变。

## 6. 设计注意事项

### 6.1 垫圈压缩均匀性

密封的关键是垫圈各处压缩量一致。设计要点：

- **螺栓间距**：建议间距不超过垫圈厚度的10倍，或不超过50mm
- **加强筋(Boss)**：在螺栓位置设置凸台，保证局部刚性
- **盖板厚度**：盖板太薄会在螺栓间产生翘曲，建议金属盖板厚度 > 1.5mm，塑料盖板 > 3mm

### 6.2 凝露排水设计

即使是IP67的外壳，内部仍可能因温度变化产生凝露。设计建议：

- 在壳体最低点设置排水孔(带透气膜或密封塞)
- 内壁涂防凝露涂层或贴吸湿片
- 将PCB安装在壳体上方，避免水滴直接落在电路上

### 6.3 常见设计错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 螺栓扭矩不足 | 垫圈压缩量不够，漏水 | 用扭力扳手，按规范扭矩拧紧 |
| 垫圈过度压缩 | 垫圈永久变形，密封失效 | 控制压缩率在15-30% |
| 壳体尖角处垫圈撕裂 | 局部泄漏 | 转角处加大圆角R |
| 忽略电缆入口密封 | 水从电缆入口渗入 | 使用IP级电缆接头 |
| 未考虑气压平衡 | 温差导致密封失效 | 安装膜式透气阀 |

## 7. 实战设计示例：IP67户外传感器外壳

### 7.1 需求

- 户外露天部署，需防暴雨和短时浸水
- 内部PCB尺寸80mm x 60mm
- 需要一根4芯电缆出口(供电+RS485)
- 工作温度-20C到+60C

### 7.2 设计方案

```
外壳结构：
+----------------------------------+
|  PC聚碳酸酯上下盖                |
|  上盖(带透明观察窗PC)            |
|  +----------------------------+  |
|  |  [透气阀M12]              |  |
|  |                           |  |
|  |     [PCB 80x60mm]        |  |
|  |       (铜柱支撑)          |  |
|  |                           |  |
|  |         [电缆接头M12]     |  |
|  +---[硅胶O型圈沟槽]---------+  |
|  下盖                            |
+----------------------------------+

BOM:
- 外壳: PC注塑上下盖, 各3mm厚
- 密封: 硅胶O型圈, 截面直径2mm
- 电缆入口: M12电缆接头 x1, IP68
- 透气: M12 ePTFE透气阀 x1, IP67
- 紧固: M3不锈钢螺丝 x6, 间距<40mm
```

### 7.3 设计验证

1. O型圈沟槽深度1.7mm，压缩率15%，合格
2. 螺栓间距35mm，小于50mm，合格
3. PCB安装高度距底面15mm，凝露水不会浸泡PCB
4. 透气阀安装在最高点，兼顾排气和防水
5. 电缆从底部进入，避免雨水沿电缆流入

## 8. 测试方法

### 8.1 IEC 60529测试要点

**防尘测试(IP5X/IP6X)**
- 设备放入粉尘箱(滑石粉，2kg/m3)
- 抽真空使壳内负压(不超过2kPa)，持续8小时
- IP5X：灰尘可以进入但不影响运行
- IP6X：完全无灰尘进入

**防水测试**
- IPX5/IPX6：用标准喷嘴在3m距离喷水
- IPX7：浸入1m水深30分钟
- IPX8：按厂商声明的深度和时间浸水

### 8.2 实用简化测试

量产阶段不必每次做完整IEC测试，可采用简化方案：

```
IP67简化测试:
1. 将设备放入水桶, 水面没过设备顶部 > 0.1m
2. 在底部放置重物使设备完全浸没
3. 等待30分钟
4. 取出后打开外壳, 检查内部是否有水迹
```

注意：简化测试仅用于产线抽检，正式认证仍需按IEC 60529进行。

## 9. 成本与防护等级权衡

### 9.1 成本递增趋势

| IP等级 | 典型外壳方案 | 相对成本 |
|--------|-------------|----------|
| IP40 | 开孔塑料壳 | 1x (基准) |
| IP54 | 加垫圈的塑料壳 | 1.5-2x |
| IP65 | 密封塑料壳+O圈+电缆接头 | 2-3x |
| IP67 | 全密封壳+透气阀+IP级接头 | 3-5x |
| IP69K | 不锈钢壳+特殊密封 | 8-15x |

### 9.2 设计建议

1. 先确定最低防护需求，不要"一步到位"上IP68
2. 内部PCB做三防漆处理，外壳可以适当降低IP等级(双保险)
3. 量大的产品值得投入模具做一体化密封设计
4. 量小的产品用现成IP67铝壳改装更经济

## 总结

IP防护等级是IoT户外设备外壳设计的核心依据。理解IEC 60529两位数字的含义是选型基础。外壳材料、密封方式、电缆入口和气压平衡是设计的四大要素。O型圈沟槽的精确设计和膜式透气阀的使用是提升密封可靠性的关键。设计时避免常见错误(螺栓扭矩不足、垫圈过度压缩、忽略气压平衡)，并通过IEC 60529标准测试验证。最终，在防护等级和成本之间找到适合项目的平衡点。
