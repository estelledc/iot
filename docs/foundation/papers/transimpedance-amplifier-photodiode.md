# 跨阻放大器TIA在光电二极管信号读出中的应用
> **难度**：🔴 高级 | **领域**：光电信号处理 | **阅读时间**：约 22 分钟

## 引言

光电二极管像一扇对光敏感的小窗,光越强,流过的电流越大。但这个电流极其微弱(从pA到mA),直接用万用表量,表的内阻就把信号吃掉了。跨阻放大器(Transimpedance Amplifier, TIA)就像一个精密的电流秤,能把极微弱的光电流忠实地转换成电压,而且速度和精度都远超简单的电阻取样。

## 1 光电二极管工作原理

### 1.1 光电效应与光电二极管

光子射入半导体PN结,能量大于带隙时产生电子-空穴对,在反向偏压下被收集形成光电流:

```
光电流: Iph = R * P

R: 响应度(A/W), 典型值 0.4-0.9 A/W (硅光电二极管)
P: 光功率(W)

例: R = 0.5 A/W, P = 1 uW -> Iph = 0.5 uA
    R = 0.5 A/W, P = 1 nW -> Iph = 0.5 nA
```

### 1.2 光电二极管等效电路

```
          Iph(C)        Cd       Rs
    +------|>------||------/\/\------+
    |                                |
    |  Rsh                           |
    +----------/\/\/\----------------+
    |                                |
    +------------------------------+-+

Iph: 光电流源(与光功率成正比)
Cd:  结电容(几pF到几百pF,反偏时减小)
Rs:  串联电阻(通常<100 ohm)
Rsh: 分流电阻(通常>100 Mohm,好的>1 Gohm)
```

### 1.3 光电二极管的两种工作模式

| 参数 | 光伏模式(零偏置) | 光导模式(反偏置) |
|------|-----------------|-----------------|
| 偏置电压 | 0V | 反向几V到几十V |
| 暗电流 | 极低(pA以下) | 较高(nA到uA) |
| 结电容 | 大 | 小(耗尽层加宽) |
| 响应速度 | 慢(MHz级) | 快(GHz级) |
| 线性度 | 差(低光强) | 好 |
| 典型应用 | 环境光,精密测量 | 光通信,高速检测 |

## 2 TIA基本原理

### 2.1 电路结构

```
                   Rf
            +------/\/\/\------+
            |       Cf          |
            |      ----         |
            |                  |
  Iph -->---+--------(-)       |
  (光电二极管)       |  OPAMP--+---> Vout
                    (+)
                     |
                    GND(或反偏电压)

  Vout = -Iph * Rf
```

虚地: 反相端电位=同相端,光电二极管两端压降恒定。

### 2.2 增益选择

| 光电流范围 | Rf | 跨阻增益 | 应用 |
|-----------|-----|---------|------|
| pA - nA | 10M - 1G ohm | 10^7 - 10^9 V/A | 环境光,荧光检测 |
| nA - uA | 100k - 10M ohm | 10^5 - 10^7 V/A | 一般光检测 |
| uA - mA | 1k - 100k ohm | 10^3 - 10^5 V/A | 光通信,强光检测 |

## 3 带宽与稳定性

### 3.1 稳定性问题的根源

TIA是最容易振荡的运放电路之一:

```
反馈回路中的极点:
  Cd(光电二极管电容) + Rf --> 主极点
  环路增益 T = Aol * beta
  当T=1时,相位裕量不足 -> 振荡
```

### 3.2 补偿电容Cf

在Rf两端并联小电容Cf,提供零点补偿:

```
补偿条件(45度相位裕量):
  Cf >= sqrt(Cd_total / (2*pi*Rf*GBW))

  Cd_total = Cd + Ccm + C_stray

例: Rf=1Mohm, Cd=50pF, GBW=10MHz
  Cf >= 0.89 pF -> 选1 pF
```

### 3.3 带宽计算

加入Cf后,TIA的-3dB带宽:

```
f_3dB = sqrt(GBW / (2*pi*Rf*Cd_total))

例: GBW=10MHz, Rf=1Mohm, Cd=50pF
  f_3dB = sqrt(10e6 / (2*pi*1e6*50e-12)) = 178 kHz

Rf越大(增益越高)或Cd越大 -> 带宽越低
```

## 4 噪声分析

### 4.1 主要噪声源

```
1. Rf热噪声:
   en_Rf = sqrt(4kT * BW / Rf)
   Rf越大,热噪声电流越小 -- TIA优势

2. 运放电压噪声(增益peaking):
   增益peaking = 1 + Cd/Cf (在高频)

3. 运放电流噪声:
   in_op * Rf -> 输出噪声电压
   FET输入运放电流噪声极低

4. 散粒噪声:
   Is = sqrt(2*q*Iph*BW), 不可消除
```

### 4.2 总输出噪声

```python
# TIA噪声估算(简化模型)
import math

def tia_output_noise(Rf, Cf, Cd, en_op, in_op, BW):
    kT = 4e-21  # 4kT at 300K
    # Rf热噪声
    Vn_Rf = math.sqrt(kT * BW / Rf) * Rf
    # 运放电压噪声(含增益peaking)
    noise_gain = 1 + Cd / Cf
    Vn_en = en_op * noise_gain * math.sqrt(BW)
    # 运放电流噪声
    Vn_in = in_op * Rf * math.sqrt(BW)
    # RSS合成
    return math.sqrt(Vn_Rf**2 + Vn_en**2 + Vn_in**2)
```

### 4.3 最小可检测信号

```
最小可检测光电流 = Vn_total / Rf

例: Vn_total = 300 uV rms, Rf = 1 Mohm
  Imin = 300 pA rms
  Pmin = 300pA / 0.5A/W = 600 pW
```

## 5 设计步骤

### 5.1 系统化设计流程

```
1. 确定需求: Imin~Imax, f_signal, Vout范围
2. 选Rf = Vout_max / Imax, 验证f_3dB >= f_signal
3. 选运放: GBW满足带宽, en低, Ib远小于Imin
4. 计算Cf = sqrt(Cd_total/(2*pi*Rf*GBW))
5. 验证: 带宽/稳定性/噪声/输出摆幅
```

### 5.2 运放选型指南

| 应用 | 推荐运放 | GBW | en | Ib | 特点 |
|------|---------|-----|----|----|------|
| 低速高灵敏度 | LMP7721 | 6MHz | 6nV | 3fA | 超低偏置电流 |
| 通用 | OPA380 | 90MHz | 7nV | 50pA | 宽带低噪声 |
| 高速光通信 | OPA847 | 3.9GHz | 5nV | 5pA | 超宽带 |
| 低功耗IoT | TLV333 | 350kHz | 55nV | 5pA | 超低功耗 |
| 精密 | ADA4530-1 | 2MHz | 7nV | 20fA | 静电计级 |

## 6 对数TIA

### 6.1 动态范围问题

标准TIA的动态范围受Rf限制:

```
Rf = 1 Mohm: 动态范围 ~ 80 dB
需要6个数量级? -> 对数TIA
```

### 6.2 对数TIA原理

用二极管替代Rf:

```
  Vout = -Vt * ln(Iph/Is)
  输出与光电流的对数成正比
  动态范围可达6-9个数量级
```

注意: 温度敏感需补偿,精度约1-2%,需已知光强校准。

## 7 实例设计

### 7.1 环境光传感器(IoT)

```
检测0.01 lux - 10000 lux, 6个数量级
光电二极管: TEMD6200FX01 (R=0.55A/W)
光电流: 0.01lux~10pA, 10000lux~10uA

方案: 双增益TIA, MCU控制模拟开关切换
  低增益: Rf=10kohm (强光)
  高增益: Rf=10Mohm (弱光)
```

### 7.2 光通信接收器

```
1Mbps, SFH2400(Cd=3pF@5V反偏), Iph~1uA
Rf=10kohm, OPA380(GBW=90MHz)
f_3dB = sqrt(90e6/(2*pi*10e3*3e-12)) = 21.8MHz >> 1MHz
Cf = 0.073pF -> 寄生电容已足够
```

### 7.3 脉冲血氧仪

```
双波长LED(红660nm+红外940nm), 同步检测

工作流程:
  1. 红光LED点亮 -> TIA采样 -> 存储
  2. 红外LED点亮 -> TIA采样 -> 存储
  3. LED关断 -> 采样暗电流 -> 减去
  4. SpO2 = f( (AC_red/DC_red) / (AC_ir/DC_ir) )

TIA: Rf=100kohm, Cf=10pF, BW~10kHz
```

## 8 PCB布局要点

### 8.1 输入节点保护

```
TIA反相输入端是最敏感的节点:

  1. 走线最短: PD到运放输入
  2. 保护环: 环绕输入走线,接同相端
  3. 去除输入端周围地铜皮(减寄生电容)
  4. 焊盘最小化

布局示意:
  +--Guard Ring(GND)--+
  |                    |
  |   PD   Rf  Cf     |
  |   |    |    |     |
  |   +----+----+--(-)|--OPAMP--Vout
  |        |          |
  |       GND        (+)--GND
  +--------------------+
```

### 8.2 减少寄生电容

- 输入走线不在地平面上方
- Rf和Cf焊盘尽量小
- 过孔远离输入节点
- 多层板内层挖空输入节点下方

### 8.3 屏蔽

- PD遮蔽杂散光
- TIA电路用金属罩屏蔽射频干扰
- 电源去耦: 0.1uF + 10uF紧贴运放电源引脚

## 9 常见设计错误

### 9.1 相位裕量不足导致振荡

```
症状: 输出有高频振荡
原因: Cf太小或没有

阶跃响应判断:
  无过冲: 裕量>60度(保守)
  小过冲(<25%): 裕量~45度(合适)
  持续振荡: 裕量<0(必须修复)
```

### 9.2 输入节点杂散电容

症状: 带宽比计算值低。解决: 缩短走线,低电容连接器。

### 9.3 电源去耦不足

症状: 输出有数字同频噪声。解决: 0.1uF紧贴运放,模拟数字电源分开。

## 10 与其他读出方式对比

| 方式 | 带宽 | 灵敏度 | 动态范围 | 复杂度 |
|------|------|--------|---------|--------|
| 电阻负载 | 高 | 低(R限制) | 窄 | 最低 |
| TIA | 中-高 | 高 | 中 | 中 |
| 对数TIA | 低-中 | 高 | 极宽 | 高 |
| 积分放大器 | 低 | 极高 | 中 | 中 |

积分放大器适用于脉冲光: LED脉冲期间积分,噪声带宽极窄,常用于荧光测量、LiDAR。

## 总结

TIA是光电二极管信号读出的核心电路,将微弱光电流转换为可用电压。设计的关键挑战在于带宽与稳定性的矛盾:光电二极管电容与反馈电阻形成的极点需要用补偿电容Cf来维持相位裕量。噪声分析需要综合考虑Rf热噪声、运放电压噪声(含增益peaking)和电流噪声。根据应用场景选择合适的工作模式(光伏vs光导)和增益范围,配合精心的PCB布局,可以实现从pA到mA跨越6个数量级的光电流精确测量。
