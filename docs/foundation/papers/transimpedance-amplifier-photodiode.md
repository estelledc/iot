# 跨阻放大器与光电二极管读出设计

> **难度**：🔴 高级 | **领域**：光电模拟前端 | **关键词**：TIA, 光电二极管, 补偿电容 | **阅读时间**：约 16 分钟

## 日常类比

水龙头滴水用杯子接再称重——把微弱电流“接住”变成电压。跨阻放大器（Transimpedance Amplifier, TIA）把光电二极管（Photodiode）的光电流转成电压，是烟雾、血氧、激光接收等物联网（IoT）光学前端核心[1][2]。

## 摘要

讲解光伏/光导模式、增益电阻与反馈电容稳定性、噪声与带宽权衡。振荡与噪声常因忽略二极管结电容与板级寄生[2][3]。

## 1. 基本配置

光电二极管 → 运放虚地输入 → 反馈电阻 \(R_f\) 得 \(V_{out}= -I_{pd} R_f\)。并反馈电容 \(C_f\) 保证环路稳定[1]。

| 模式 | 偏置 | 特点 |
|------|------|------|
| 光伏 | 近零偏 | 暗电流小，偏线性 |
| 光导 | 反偏 | 更快，暗电流↑ |

## 2. 设计表

| 目标 | 手段 |
|------|------|
| 高增益 | 大 \(R_f\)，注意带宽与噪声 |
| 稳不振 | 选 \(C_f\) 满足噪声增益稳定 |
| 低噪声 | 低电流噪声运放、屏蔽 |
| 快响应 | 降 \(R_f\)、反偏、快运放 |

| 寄生 | 影响 |
|------|------|
| 结电容 Cd | 与 Rf 形成极点 |
| 输入电容 | 加剧不稳 |
| 板级漏电 | 暗电流误差 |
| 灯光干扰 | 工频纹波 |

## 3. 局限、挑战与可改进方向

### 1. 稳定性被忽视

**局限**：示波器见振荡当信号。
**改进**：按手册算 \(C_f\)；时域/频域检查[2]。

### 2. 漏电与清洁度

**局限**：潮湿/助焊剂造成偏置。
**改进**：防护涂覆、高阻布局、卫护环[3]。

### 3. 环境光饱和

**局限**：强光顶满量程。
**改进**：光学滤波、自动增益、调制光源同步检测[4]。

### 4. 运放选型不当

**局限**：电压噪声优但电流噪声毁高阻。
**改进**：按源阻抗选电流/电压噪声平衡器件[1]。

## 总结

TIA 设计是稳定性、噪声与带宽的三角。先把 \(C_f\) 与布局做对，再追求极限增益。
