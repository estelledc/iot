# 电流采样电阻与分流器精密电流测量

> **难度**：🟡 中级 | **领域**：电流测量技术 | **阅读时间**：约 20 分钟

## 引言

想象你要知道一条河的水流量。你可以在河里放一块石头,测量石头前后的水位差——水位差越大,水流越急。电流采样电阻就是电路里的那块"石头":串联在电流路径中,通过测量它两端的电压差来推算电流大小。石头越小(阻值越低)对河流影响越小,但水位差也越小越难测;石头越大(阻值越高)水位差明显好测,但对河流的阻碍也越大。这就是电流采样设计的核心权衡。

在IoT设备中,电流测量是功耗分析、电池管理、电机保护和故障检测的基础。选择合适的采样电阻和测量方案,直接决定系统能否在功耗、精度和成本之间找到平衡。本文从采样电阻原理、选型、放大电路到PCB布局,全面覆盖精密电流测量的设计要点。

## 1 电流测量方法概述

### 1.1 四种主流方法

| 方法 | 原理 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| 采样电阻 | V=IR测压降 | 精度高、线性好、成本低 | 有插入损耗、接触电阻影响 | 0.1mA-100A |
| 霍尔效应 | 磁场检测 | 隔离、无插入损耗 | 精度中等、受温度影响 | 大电流、隔离需求 |
| 电流互感器 | 电磁感应 | 隔离、大电流 | 只测交流、体积大 | 工频大电流 |
| 罗氏线圈 | 微分磁场 | 柔性、大电流 | 只测交流、需积分电路 | 瞬态大电流 |

采样电阻法精度最高、线性最好、成本最低,是IoT中小电流测量的首选。本文重点讨论此方法。

### 1.2 选择依据

- 电流范围 < 100A: 采样电阻
- 需要隔离: 霍尔效应
- 工频交流大电流: 电流互感器
- 瞬态脉冲大电流: 罗氏线圈

## 2 采样电阻原理

### 2.1 欧姆定律

采样电阻法基于欧姆定律:

```
I = V_sense / R_sense
```

在电流路径中串联一个已知阻值的电阻R_sense,测量其两端电压V_sense,即可算出电流I。

### 2.2 电压-电流-阻值的关系

以R_sense = 0.1欧为例:

| 电流 | 压降 | 功耗 |
|------|------|------|
| 10 mA | 1 mV | 0.01 mW |
| 100 mA | 10 mV | 1 mW |
| 1 A | 100 mV | 100 mW |
| 5 A | 500 mV | 2.5 W |

阻值越高,信号越强越容易测量,但功耗和压降也越大。这是采样电阻设计的核心权衡。

### 2.3 插入损耗的影响

采样电阻上的压降会减少负载端的可用电压。例如3.3V系统用0.1欧采样,1A电流时负载端只剩3.2V。如果MCU最低工作电压3.0V,那最大允许压降为0.3V,对应R_sense < 0.3欧。

## 3 采样电阻选型

### 3.1 阻值选择

阻值选择需平衡信号电平与功率损耗:

```
信号需求: V_sense = I_min x R_sense >= ADC的分辨率要求
功耗约束: P = I_max^2 x R_sense <= 电阻额定功率
压降约束: V_drop = I_max x R_sense <= 系统允许压降
```

经验值:
- 1mA-100mA量程: 1-10欧
- 100mA-1A量程: 0.1-1欧
- 1A-10A量程: 0.01-0.1欧
- 10A以上: 0.001-0.01欧(需用分流器)

### 3.2 精度与容差

| 容差 | 成本 | 适用场景 |
|------|------|---------|
| 1% | 低 | 过流保护、粗略估算 |
| 0.5% | 中 | 电池管理、功耗监测 |
| 0.1% | 高 | 精密校准、电量计 |

容差直接产生增益误差。0.1%的电阻在1A量程产生1mA的误差,对精密电量计可能不够,需要系统校准。

### 3.3 温度系数(TCR)

电阻阻值随温度变化:

```
R(T) = R25 x (1 + TCR x (T - 25))
```

TCR单位ppm/C。自发热和环温变化都会导致阻值偏移:

| TCR(ppm/C) | 温度变化50C时阻值偏移 |
|-------------|---------------------|
| 50 | 0.25% |
| 25 | 0.125% |
| 10 | 0.05% |
| 5 | 0.025% |

精密测量推荐TCR <= 25ppm/C的合金电阻(如锰铜、镍铬合金)。

### 3.4 功率额定

电阻功耗P = I^2 x R,自发热导致温升:

```
T_rise = P x R_th(ja)
```

R_th(ja)为热阻,0805封装约100C/W,2512封装约50C/W。建议降额使用,实际功耗不超过额定功率的50%。

### 3.5 四端子开尔文电阻

精密测量必须用四端子(开尔文)电阻。普通两端子电阻的焊锡和引脚电阻串联在测量路径中,引入误差。四端子电阻将电流端和电压检测端分开,消除接触电阻影响:

```
I+ ---[R_sense]--- I-
      |         |
    S+         S-   (Sense端,接到放大器输入)
```

推荐型号: WSBM8518(0.001-1欧,0.1%,50ppm/TC)、CSS2H-3920(0.001-0.005欧)。

## 4 高侧与低侧采样

### 4.1 低侧采样

采样电阻放在负载与地之间:

```
Vcc ---[Load]---+--- R_sense --- GND
                 |
              V_sense
```

优点: 检测电压以地为参考,可用普通运放或MCU内置ADC直接测量,电路简单。

缺点: 负载不直接接地(浮地),可能引入噪声;无法检测对地短路故障;多路采样时各路地不共。

### 4.2 高侧采样

采样电阻放在电源与负载之间:

```
Vcc --- R_sense ---+---[Load]--- GND
                   |
                V_sense
```

优点: 负载直接接地,系统稳定性好;可检测对地短路故障;多路采样地共参考。

缺点: V_sense叠加在共模电压上,需差分放大器或专用电流检测放大器;共模电压高时器件选择受限。

### 4.3 选择建议

| 条件 | 推荐 |
|------|------|
| 成本敏感、精度要求一般 | 低侧 |
| 负载必须接地 | 高侧 |
| 需检测对地短路 | 高侧 |
| 电池供电(单路) | 低侧也可 |
| 工业多路采集 | 高侧 |

IoT电池管理推荐高侧,可检测对地漏电;成本敏感的简单场景可用低侧。

## 5 电流检测放大器

### 5.1 INA219(TI)

集成增益放大器 + ADC + I2C接口的完整电流测量方案:

- 内置12位ADC,可测电流和母线电压
- 可编程增益: +/-40mV到 +/-320mV量程
- I2C数字输出,MCU直接读取
- 精度0.5%(校准后0.1%)

```
Vcc --- R_sense ---+---[Load]--- GND
       IN+    IN-  |
                    INA219 -- I2C --> MCU
```

优势: 无需外部ADC,节省BOM;数字输出抗干扰;可编程量程。

### 5.2 INA180(TI)

模拟输出电流检测放大器:

- 固定增益: 20/50/100/200V/V可选
- 带宽260kHz
- 输出电压正比于采样电压,送MCU ADC

适用场景: 需要快速过流检测(模拟比较)、多通道同步采样、已有ADC的系统。

### 5.3 MAX9938(Maxim/ADI)

纳米功耗电流检测放大器:

- 供电电流仅1uA
- 增益50/100/200V/V
- 适合电池电流长时间监测

### 5.4 选型对比

| 参数 | INA219 | INA180 | MAX9938 |
|------|--------|--------|---------|
| 输出类型 | I2C数字 | 模拟电压 | 模拟电压 |
| ADC位数 | 12位 | 无(外部) | 无(外部) |
| 增益 | 可编程 | 固定(4档) | 固定(3档) |
| 带宽 | 1.6kHz | 260kHz | 10kHz |
| 供电电流 | 1 mA | 260 uA | 1 uA |
| 参考价格 | ~$1.5 | ~$0.5 | ~$1.0 |
| 典型场景 | IoT功耗分析 | 过流保护 | 电池长期监测 |

## 6 增益与偏移误差分析

### 6.1 误差来源

电流测量的总误差由多个来源叠加:

| 误差源 | 典型值 | 说明 |
|--------|--------|------|
| 电阻容差 | 0.1-1% | 增益误差,可校准 |
| 电阻TCR | 10-100ppm/C | 温度相关增益误差 |
| 放大器偏移电压 | 10-500uV | 零点偏移 |
| 放大器增益误差 | 0.1-1% | 增益不精确 |
| ADC量化误差 | 0.5LSB | 分辨率限制 |
| PCB走线电阻 | 1-10m欧 | 低阻值时显著 |

### 6.2 误差计算示例

假设0.1欧/0.1%电阻 + INA219,1A电流:

```
V_sense = 1A x 0.1欧 = 100mV
电阻容差误差: 0.1% x 1A = 1mA
INA219偏移: 100uV / 0.1欧 = 1mA
增益误差: 0.5% x 1A = 5mA
总误差(RSS): sqrt(1^2 + 1^2 + 5^2) = 5.2mA (约0.52%)
```

校准可消除偏移和增益误差,残差主要由TCR和长期漂移决定。

## 7 PCB布局要点

### 7.1 开尔文连接

采样电阻的电压检测走线必须直接从电阻焊盘引出,不与电流路径共享:

```
正确:
    I+ ----[焊盘]---[电阻]---[焊盘]---- I-
              |                  |
           Sense+            Sense-   (细线,直接连到放大器)

错误:
    I+ ----[走线]---[电阻]---[走线]---- I-
              |                  |
           Sense+            Sense-   (Sense接在走线上,含走线电阻压降)
```

Sense走线从电阻焊盘内侧(靠近电阻体的一侧)引出,可最大限度消除焊锡和走线电阻的影响。

### 7.2 走线宽度

电流路径走线宽度按载流能力设计,1oz铜厚经验值:

| 电流 | 最小线宽(1oz) |
|------|--------------|
| 0.5A | 0.2mm |
| 1A | 0.4mm |
| 2A | 0.8mm |
| 5A | 2.0mm |

Sense走线可以很细(0.1-0.2mm),因为几乎无电流流过。

### 7.3 热管理

采样电阻是大功率器件,布局时需注意:
- 电阻下方铺铜散热
- 远离温度敏感器件(基准源、晶振)
- 避免将电阻放在MCU正上方
- 多层板可在电阻正下方内层铺散热铜

### 7.4 地线处理

低侧采样时,R_sense的接地端必须直接连到地平面,不与其他回路共享地线,避免地弹干扰。

高侧采样时,Sense走线要远离高速数字信号,差分走线平行走线,减少共模噪声拾取。

## 8 实际应用示例

### 8.1 电池电流监测(IoT功耗分析)

需求: 监测IoT设备的电池电流,包含uA级休眠和mA级工作状态。

方案: INA219 + 0.1欧采样电阻,高侧采样:

```
Vbat(3.6V) ---[0.1欧]---+---[MCU+传感器]--- GND
                         |
              INA219 IN+  IN-
                         |
                   INA219 -- I2C --> MCU
```

INA219量程 +/-320mV对应 +/-3.2A,1LSB约0.8mA。休眠电流需更大量程或更换更小阻值电阻。

### 8.2 电机过流保护

需求: 直流电机驱动,过流阈值2A,快速关断。

方案: INA180(增益20V/V) + 比较器:

```
Vcc ---[0.05欧]---+---[H桥+电机]--- GND
                   |
               INA180(20x) ---> 模拟输出
                   |
              比较器阈值=2V --> MCU中断/驱动关断
```

2A x 0.05欧 = 100mV,INA180放大20倍 = 2V。超过2V即过流,响应时间us级。

### 8.3 太阳能MPPT电流反馈

需求: 太阳能板电流采样,用于MPPT算法,量程0-5A。

方案: INA180(增益50V/V) + MCU ADC + 0.01欧锰铜分流器:

```
Solar+ ---[0.01欧]---+---[DC-DC]--- 电池
                      |
                  INA180(50x) ---> MCU ADC
```

5A x 0.01欧 = 50mV,放大50倍 = 2.5V,适合MCU 3.3V ADC量程。0.01欧锰铜TCR<10ppm/C,温度稳定性好。

## 9 动态范围考量

### 9.1 宽动态范围的挑战

IoT设备的工作电流范围极大:
- 深度休眠: 1-10uA
- 无线发射: 20-100mA
- 传感器采样: 0.5-5mA

单量程难以同时覆盖uA级和mA级。如果量程设为100mA/12位,1LSB=24uA,休眠电流完全无法分辨。

### 9.2 多量程方案

用模拟开关切换不同阻值的采样电阻:

```
Vcc ---[R1=10欧]---+---[R2=0.1欧]---+---[Load]--- GND
      |             |               |
     SW1           SW2              |
      |             |               |
    ADC1          ADC2              |
```

- uA级: 接通R1=10欧,10uA产生100uV信号
- mA级: 接通R2=0.1欧,100mA产生10mV信号

切换由MCU根据工作状态自动完成,休眠时用高阻值,唤醒后切低阻值。

### 9.3 对数放大方案

用对数放大器压缩动态范围,适合不需要精确幅值、只需检测电流等级的场景。成本较高,精度一般,工程中使用较少。

### 9.4 方案对比

| 方案 | 动态范围 | 精度 | 成本 | 复杂度 |
|------|---------|------|------|--------|
| 单量程 | 100:1 | 高 | 低 | 低 |
| 多量程切换 | 10000:1 | 高 | 中 | 中 |
| 对数放大 | 100000:1 | 中 | 高 | 中 |

IoT功耗分析推荐多量程切换方案,兼顾精度和动态范围。

## 10 常见错误与避免

### 10.1 Sense走线布在电流路径上

错误: Sense走线从电流走线上分叉引出,包含了走线电阻压降。

正确: Sense走线直接从电阻焊盘引出,不与电流走线共享任何铜皮。

### 10.2 采样电阻功率额定不足

电阻在过流时严重发热,阻值偏移甚至烧毁。必须按最大电流计算功耗,并至少2倍降额:

```
P = I_max^2 x R_sense
R_rated >= 2 x P
```

### 10.3 高侧采样的地环路

高侧采样时,放大器的地必须与负载地相连,不能与MCU数字地共享,否则形成地环路,引入共模噪声。使用差分走线或隔离放大器可解决。

### 10.4 忽略自发热导致的TCR偏移

大电流时采样电阻自发热显著。0.1欧/1W电阻通5A,功耗2.5W,温升可能超过100C。TCR=50ppm/C时阻值偏移0.5%,这比初始容差还大。

解决方案: 选择低TCR合金电阻(如锰铜<10ppm/C),或做温度补偿。

### 10.5 忽略PCB铜皮电阻

低阻值采样电阻(毫欧级)时,PCB走线和焊盘的电阻已经与采样电阻同量级。必须用开尔文连接,从焊盘内侧引出Sense走线。

## 总结

电流采样电阻法是最精确、最经济的电流测量方式,核心要点:

1. **阻值是核心权衡**: 太小信号弱难测量,太大功耗高压降大。按量程和精度需求选择。
2. **四端子电阻不可省**: 精密测量必须用开尔文型,消除接触电阻误差。
3. **高侧与低侧按需选择**: 需要接地检测和短路保护用高侧,成本敏感用低侧。
4. **TCR决定温漂性能**: 合金电阻(锰铜、镍铬)TCR远优于厚膜电阻,精密场合必选。
5. **PCB开尔文连接是关键**: Sense走线必须从焊盘引出,不与电流路径共享。
6. **宽动态范围用多量程**: IoT休眠uA和工作mA的跨度,单量程无法覆盖,需自动切换。

记住开头的类比:石头大小要刚刚好——既能看清水位差,又不影响河水流动。采样电阻的选型就是这个平衡的艺术。
