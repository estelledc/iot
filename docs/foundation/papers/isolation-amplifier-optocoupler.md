# 隔离放大器与光耦隔离在高压测量中的对比
> **难度**：🟡 中级 | **领域**：电气隔离技术 | **阅读时间**：约 20 分钟

## 引言

想象你在测量一栋 30 层大楼的水压。大楼的供水系统有自己的压力基准，而你的压力表也有自己的零点。如果直接把表接上去，两个"零点"打架，读数就乱套了。电气隔离也是同一个道理：高压侧和低压侧各守各的"地"，中间靠非电信号(光、磁、电容)传信息，两边永远不直接连通。

在 IoT 高压测量场景(智能电表、电机驱动、光伏逆变器)中，选择哪种隔离方式直接决定了系统的带宽、精度、寿命和成本。本文对比光耦隔离与隔离放大器两条技术路线，帮你在设计选型时做出有据可依的判断。

## 1 为什么需要电气隔离

### 1.1 三大核心动机

1. **人身安全**：高压侧故障时，隔离屏障阻止危险电压窜入低压侧(人可触及的接口)。IEC 61010 要求测量设备在额定电压下保持隔离完整性。
2. **地环路消除**：不同设备的地电位可能相差数伏甚至数十伏。直接连接会在地线中产生环流，叠加在信号上形成误差。隔离断开地环路，让各子系统独立参考。
3. **高压直接测量**：当被测电压超出 ADC 量程时，需要先衰减再隔离，或者用隔离放大器/隔离 ADC 在高压侧直接采样，再以数字或隔离模拟信号传回低压侧。

### 1.2 IoT 场景的隔离需求特点

- **体积受限**：智能电表和电机驱动器留给隔离器件的 PCB 面积有限，DIP-8 已嫌大，SOIC-8 甚至更小封装成为主流。
- **长期运行**：电表要求 10 年以上免维护，光耦的 CTR 衰减必须纳入寿命预算。
- **成本敏感**：消费级 IoT 产品 BOM 以美分为单位计较，隔离方案的选择常由单价决定。

## 2 隔离方法概览

### 2.1 三种物理介质

| 隔离介质 | 工作原理 | 代表器件 | 典型隔离电压 |
|----------|----------|----------|--------------|
| 光学(光耦) | LED 发光 -> 光电管接收 | PC817, IL300, Si8641 | 2.5-5 kVrms |
| 磁性(变压器) | 调制信号 -> 脉冲变压器 -> 解调 | AMC1301, ISO224 | 3-7 kVrms |
| 电容(容耦) | 高频调制信号经 SiO2 电容耦合 | ADuM1201, ISO7721 | 3-5 kVrms |

### 2.2 介质特性对设计的影响

- **光耦**：LED 是电流驱动器件，需要偏置电流；光传输单向，反馈需双通道；长期光输出衰减影响精度。
- **磁耦**：线圈需要调制/解调电路，通常芯片内集成；变压器尺寸决定隔离电压，微型变压器靠 CMOS 工艺实现。
- **容耦**：电容极板间距决定隔离电压；高频差分信号通过，低频需调制；对共模瞬态抗扰度(CMTI)优秀。

## 3 光耦隔离基础

### 3.1 基本结构：LED + 光电管

最简单的光耦由一个红外 LED 和一个光敏三极管组成，封装在不透光树脂中：

```
         LED侧              光电管侧
    +------|>|------+    +------[C
    |   Anode  Cathode|   |   Collector
    |                 |   |
    |   If (正向电流)  |   |   CTR x If = Ic
    |                 |   |
    +-----------------+   +------[E
                              Emitter
```

电流传输比(CTR)定义：

```
CTR = Ic / If x 100%
```

典型值：PC817 的 CTR 在 50%-300% 之间(分档)，随温度和老化下降。

### 3.2 CTR 衰减：光耦的"阿喀琉斯之踵"

CTR 衰减是光耦最严重的长期可靠性问题，原因有三：

1. **LED 光输出衰减**：红外 LED 的量子效率随工作时间和结温持续下降，典型速率 0.5%-2%/千小时。
2. **光电管灵敏度漂移**：光敏三极管的 hFE 随温度变化，高温下增益下降。
3. **封装透光率退化**：环氧树脂在高温高湿下黄变，光路效率降低。

**实用估算**：若光耦工作在 85 C 环境温度、If = 10 mA，5 年后 CTR 可能降至初始值的 50%-70%。设计时必须预留 CTR 降额裕量。

### 3.3 非线性问题

普通光耦的 CTR 随 If 变化很大(小电流区急剧下降，大电流区趋于平坦)，导致：

- 模拟信号传输产生严重非线性失真(THD 可达 5%-10%)
- 不同器件 CTR 分散性大，无法互换校准
- 不适合直接传输精密模拟信号

结论：普通光耦(如 PC817)只适合数字信号隔离或粗略模拟信号传输。

## 4 线性光耦：IL300 与伺服反馈

### 4.1 工作原理

线性光耦(以 IL300 为代表)采用伺服反馈结构消除 CTR 衰减和非线性的影响：

```
            +---[LED]---+
            |           |
    Vin --[R1]--A       |
                  |OPAMP|  <-- 伺服放大器
                  |     |
            +-----B     |
            |           |
            |   +--[D1光电管]---> Vout (输出侧)
            |   |
            |   +--[D2光电管]---> 反馈侧
            |           |
            +--[R2]-----+
```

关键设计思想：D1 和 D2 由同一个 LED 照射，具有高度匹配的光电响应。伺服放大器调整 LED 电流 If，使得 D2 的反馈电流精确跟踪输入电压。由于 D1 和 D2 匹配，D1 的输出电流也精确正比于输入。

### 4.2 优缺点

| 指标 | 数值/说明 |
|------|-----------|
| 带宽 | 约 200 kHz (小信号) |
| 非线性误差 | 0.01%-0.05% (匹配模式下) |
| 隔离电压 | 7.5 kVrms (IL300) |
| 温度系数 | 约 50 ppm/ C |
| CTR 衰减影响 | 伺服反馈补偿后极小，但 D1/D2 失配随老化增大 |
| 外围元件 | 需要精密运放和匹配电阻，BOM 增加 |

### 4.3 设计要点

- D1 和 D2 的光电匹配度是精度关键，选型时关注 "ip(inverse photo) match" 参数。
- 反馈电阻 R1 和 R2 的温漂必须匹配，否则引入额外温度误差。
- LED 驱动运放的带宽决定了整体闭环带宽，需选 GBW > 1 MHz 的运放。

## 5 数字隔离器

### 5.1 从光耦到数字隔离器

数字隔离器保留了光耦的"隔离"功能，但用更可靠的介质(电容或磁)替代 LED-光电管结构，专用于数字信号传输：

| 器件 | 隔离介质 | 通道数 | 速率 | CMTI | 隔离电压 |
|------|----------|--------|------|------|----------|
| Si8641 | 容耦 | 4 | 150 Mbps | 25 kV/us | 5 kVrms |
| ADuM1201 | 磁耦(iCoupler) | 2 | 1 Mbps | 25 kV/us | 5 kVrms |
| ISO7721 | 容耦 | 2 | 100 Mbps | 50 kV/us | 5 kVrms |

### 5.2 数字隔离器的优势

- **无 LED 衰减**：没有光发射器件，寿命由绝缘材料决定，远优于光耦。
- **低功耗**：每通道静态电流约 0.5-2 mA，远低于光耦的 LED 驱动电流(5-20 mA)。
- **高 CMTI**：容耦和磁耦结构对共模瞬态抑制能力远强于光耦。
- **高速**：支持 Mbps 级数据率，适合 SPI、UART、I2C 隔离。

### 5.3 局限性

- 只能传输数字信号，模拟信号需先经 ADC 转换。
- 双电源供电(两侧各需 VDD)，BOM 比单侧供电的光耦多一路电源。
- 高频调制信号可能产生 EMI，需注意 PCB 布局和去耦。

## 6 隔离放大器

### 6.1 AMC1301：电容隔离 + 高精度

AMC1301 是 TI 的一款隔离放大器，内部结构：

```
    高压侧                       低压侧
    +--------+                  +--------+
    | 输入   |   高频调制信号    | 解调   |
    | 放大器 |--[电容隔离屏障]-->| + 滤波 |
    |        |                  |        |
    | Vdd1   |                  | Vdd2   |
    +--------+                  +--------+
         |                           |
        GND1                        GND2
```

核心参数：

| 参数 | AMC1301 | ISO224 (TI) |
|------|---------|-------------|
| 增益 | 20 V/V (固定) | 1 V/V (可配置) |
| 带宽 | 250 kHz | 85 kHz |
| 非线性 | 0.03% | 0.01% |
| 隔离电压 | 7 kVPEAK | 6 kVPEAK |
| CMTI | 50 kV/us | 70 kV/us |
| 供电(高压侧) | 5 V | 5 V |
| 封装 | SOIC-8 | SOIC-16 |

### 6.2 ISO224：高精度工业级

ISO224 适用于更高精度要求场景：

- 增益 1 V/V，适合多种传感器前端配置
- 0.01% 非线性，满足 0.1 级电表精度要求
- 85 kHz 带宽覆盖工频及低次谐波测量
- 工作电压范围 4.5-5.5 V，兼容 5 V 系统

### 6.3 隔离放大器 vs 线性光耦

| 对比项 | 隔离放大器(AMC1301) | 线性光耦(IL300) |
|--------|---------------------|------------------|
| 集成度 | 高(片内调制/解调) | 低(需外部运放) |
| 精度 | 0.03%(出厂校准) | 0.01%-0.05%(需匹配) |
| 长期稳定性 | 无 LED 衰减 | CTR 匹配漂移 |
| 带宽 | 250 kHz | 200 kHz |
| 供电复杂度 | 双电源(简单 LDO) | 双电源 + 运放供电 |
| BOM 成本 | 器件贵，外围少 | 器件便宜，外围多 |

## 7 隔离 ADC 方案

### 7.1 ADS1220 + 数字隔离器组合

另一种思路是：在高压侧完成 ADC 转换，再用数字隔离器传输数字结果：

```
    高压侧                          低压侧
    +----------+    SPI 数据线     +----------+
    | ADS1220  |---[Si8641等]----->| MCU      |
    | (24-bit  |    (数字隔离)      | (接收端) |
    |  Delta-  |                   |          |
    |  Sigma)  |                   |          |
    +----------+                   +----------+
         |                              |
        GND1                           GND2
```

### 7.2 隔离 ADC 集成器件

更进一步的集成方案将 ADC 和数字隔离器合二为一：

| 器件 | 分辨率 | 采样率 | 隔离电压 | 特点 |
|------|--------|--------|----------|------|
| AMC1311 | 14-bit | 250 kSPS | 7 kVPEAK | 差分输入，低延迟 |
| ADS1202 + 隔离 | 16-bit | 高 | 取决于隔离器 | 灵活组合 |
| ISO ADC 系列 | 多种 | 多种 | 5-7 kV | 一体化方案 |

### 7.3 隔离 ADC 的优势

- **数字信号天然抗扰**：数字隔离器传输的是 0/1 电平，对模拟噪声免疫。
- **高分辨率**：24-bit Delta-Sigma ADC 可达到微伏级测量精度。
- **校准数据可存在高压侧**：增益/失调校准在数字域完成，无模拟漂移。
- **灵活性**：ADC 采样率、滤波参数均可通过 SPI 配置。

## 8 隔离额定值与安全标准

### 8.1 关键参数解读

| 参数 | 定义 | 设计意义 |
|------|------|----------|
| 工作电压(Viorm) | 可长期承受的隔离电压 | 决定器件在额定条件下的安全裕量 |
| 耐压(Viso) | 1 分钟耐压测试值(型式试验) | 通常为工频耐压，选型参考 |
| 爬电距离(Creepage) | 沿绝缘表面的最短距离 | 污染等级和材料组别决定最小要求 |
| 电气间隙(Clearance) | 两导电部件间最短空间距离 | 决定瞬态过压下的击穿风险 |
| CMTI | 共模瞬态抗扰度 | 衡量器件对快速 dv/dt 的抑制能力 |

### 8.2 爬电距离与电气间隙的工程计算

以 400V 系统为例，根据 IEC 60664-1：

- 污染等级 2、材料组 IIIa、过压类别 III：
  - 爬电距离 >= 4.0 mm
  - 电气间隙 >= 3.0 mm

光耦 DIP-8 封装的爬电距离约 7 mm，满足要求；但 SOIC-8 封装仅约 4 mm，处于临界，需确认具体器件规格。

### 8.3 安全标准体系

| 标准 | 覆盖内容 | 对隔离器件的要求 |
|------|----------|------------------|
| IEC 60747-5-5 | 光耦安全相关标准 | 定义光耦的隔离认证要求和测试方法 |
| UL 1577 | 光隔离器安全标准 | 规定耐压测试和温升测试 |
| IEC 61010-1 | 测量设备安全 | 要求测量类别与隔离等级匹配 |
| IEC 61800-5-1 | 电机驱动器安全 | 规定功率驱动隔离的具体要求 |
| IEC 60664-1 | 绝缘配合 | 爬电/间隙的通用计算方法 |

**关键提醒**：光耦用于安全功能(如过流保护)时，必须选通过 IEC 60747-5-5 认证的"增强隔离"等级器件(如 HCPL-2631)，普通光耦(仅 UL 1577)不满足功能安全要求。

### 9.2 选型决策流程

```
开始
  |
  +--> 传输模拟还是数字信号?
  |      |
  |      +--> 数字 --> 精度要求?
  |      |              |
  |      |              +--> 普通 --> 普通光耦(低成本)
  |      |              +--> 高速/高可靠 --> 数字隔离器
  |      |
  |      +--> 模拟 --> 精度要求?
  |                     |
  |                     +--> < 1% --> 线性光耦(成本适中)
  |                     +--> < 0.1% --> 隔离放大器
  |                     +--> 微伏级 --> 隔离ADC
  |
  +--> 是否涉及安全功能?
         |
         +--> 是 --> 必须选 IEC 60747-5-5 认证器件
         +--> 否 --> UL 1577 或基本绝缘即可
```

### 9.3 成本-性能权衡

- **极低成本场景**(智能家居插座、简易电量计)：普通光耦 + 分立 ADC，总 BOM < $1。
- **中等精度场景**(智能电表 1.0 级)：隔离放大器(AMC1301)，总 BOM 约 $3-5。
- **高精度场景**(电表 0.1 级、精密测量)：隔离放大器(ISO224) 或隔离 ADC，总 BOM 约 $5-10。
- **高速场景**(电机相电流 FOC 控制)：隔离放大器(AMC1301, 250 kHz 带宽)，总 BOM 约 $3-6。

## 10 IoT 应用实例

### 10.1 交流电力监测

智能电表或智能插座需测量 220V/380V 交流电压和电流：

- **电压通道**：电阻分压网络将 220V 衰减至 +/-250 mV，送入隔离放大器(AMC1301)。
- **电流通道**：分流器(0.1-1 mOhm)将电流转换为电压，直接送入隔离放大器。
- **数字接口**：隔离放大器差分输出接低压侧 ADC，或直接用隔离 ADC。

设计注意事项：电阻分压网络的精度和温漂直接影响测量精度，须选 0.1% / 25 ppm 精密电阻。

### 10.2 电机电流传感

电机驱动器(变频器、伺服)需实时检测相电流用于 FOC 控制：

- **带宽要求**：至少 10 倍基频(电机 500 Hz 运行 -> 5 kHz 带宽)，AMC1301 的 250 kHz 远超需求。
- **延迟要求**：FOC 闭环周期通常 10-50 us，隔离放大器 3 us 延迟可接受。
- **CMTI 要求**：IGBT 开关产生的 dv/dt 可达 10-50 kV/us，必须选 CMTI > 50 kV/us 的器件。

推荐方案：AMC1301 或 ISO224 + 分流器，避免使用霍尔传感器(磁饱和和偏移问题)。

### 10.3 光伏逆变器

光伏逆变器中隔离测量用于：

- **MPPT 电压/电流检测**：在 DC 侧(最高 1000V)测量组串电压和电流。
- **并网电流检测**：在 AC 侧测量注入电网的电流波形和 THD。
- **绝缘检测**：检测光伏阵列对地绝缘电阻，需高隔离电压器件。

设计注意：光伏 DC 侧可达 1000V DC，爬电距离要求 >= 8 mm(污染等级 2)，部分 SOIC 封装不满足，需选宽体 SOIC 或 DIP 封装。

## 11 常见设计误区

### 11.1 "光耦隔离就是安全隔离"

错误认知：只要用了光耦，高压侧就安全了。

正确理解：光耦的隔离等级取决于器件的认证等级和实际爬电/间隙距离。普通光耦(PC817)仅提供基本绝缘(1.5 kVrms 耐压)，在 400V 系统中不足以满足双重绝缘或加强绝缘要求。安全隔离需选特定认证等级的器件。

### 11.2 "隔离放大器不需要考虑 PCB 布局"

错误认知：隔离放大器内部已有隔离，PCB 布局随意即可。

正确理解：PCB 布局直接影响实际隔离性能：
- 高压侧和低压侧的走线不能交叉，否则沿走线形成爬电路径。
- 隔离沟槽(slot)应无覆铜，防止沿铜箔形成低阻抗通路。
- 两侧的去耦电容应各自放在对应侧，不能跨越隔离带。

### 11.3 "数字隔离器可以替代所有光耦"

错误认知：数字隔离器各方面都优于光耦，可以直接替换。

正确理解：数字隔离器确实在寿命和 CMTI 上优于光耦，但有以下限制：
- 某些安全认证(IEC 60747-5-5)目前仍主要针对光耦，数字隔离器需额外认证。
- 数字隔离器需要两侧独立供电，在仅有单侧电源的应用中需增加隔离电源。
- 在极低成本(< $0.5)场景下，普通光耦仍有价格优势。

## 12 总结

| 维度 | 光耦方案 | 隔离放大器方案 |
|------|----------|----------------|
| 精度上限 | 0.01%(线性光耦) | 0.01%(ISO224) |
| 长期稳定性 | 受 CTR 衰减限制 | 无 LED 衰减，更稳定 |
| 带宽 | 200 kHz(线性光耦) | 85-250 kHz |
| 安全认证 | IEC 60747-5-5 体系成熟 | 需查具体器件认证状态 |
| 成本(含外围) | 线性光耦 + 运放: $5-8 | AMC1301: $3-6 |
| 功耗 | LED 驱动 5-20 mA | 静态 5-12 mA(双侧) |
| 体积 | DIP-8/SOIC-8 + 运放 | SOIC-8(单芯片) |
| 设计难度 | 中(需匹配运放和电阻) | 低(片内集成) |

**选型建议**：

1. 新设计优先考虑隔离放大器或数字隔离器，避免光耦的 CTR 衰减风险。
2. 安全功能应用必须选 IEC 60747-5-5 认证器件，不可用普通光耦替代。
3. 高精度测量(0.1 级以上)选隔离放大器或隔离 ADC；低成本场景普通光耦仍可胜任。
4. PCB 布局严格遵守隔离间距要求，器件的额定值只是起点，实际隔离性能取决于 PCB 实现。
