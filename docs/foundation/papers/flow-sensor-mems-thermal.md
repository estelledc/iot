# MEMS 热式流量传感器微结构与响应特性

> **难度**：🔴 高级 | **领域**：流量测量、MEMS 热传导、微流控 | **阅读时间**：约 40 分钟

## 日常类比

想象你在冬天开了一台暖风机。没风时，热量均匀扩散，左右感受差不多。但如果你打开窗户，冷风从左边吹进来，热量被吹向右边——左边变凉，右边更热。风越大，左右温差越大。

MEMS 热式流量传感器原理完全一样：芯片上有个微型"暖风机"（加热器），两边各有一个"温度计"（热电堆）。流体流过时把热量带走，造成上下游温差。测出温差，就能算出流量。整个结构比一粒沙子还小，却能感知每秒几毫米的微弱气流。

---

## 1. 热式流量传感原理

热式流量传感根据检测方式分为三种基本模式。

### 1.1 量热式（Calorimetric）

芯片中央放加热器，上下游对称放温度传感器。无流时两侧温度相等；有流时上游被冷却、下游被加热，产生温差 ΔT。

```
ΔT = T_downstream - T_upstream ∝ ρ · v · c_p · P_heater
```

特点：可同时测流向和流速；低流速灵敏度极高（mm/s 级）；输出呈 S 形曲线，高速段饱和；适合微流控、呼吸机。

### 1.2 风速式（Anemometric）

只关心加热器散热速率。维持温度恒定（CT 模式）或功率恒定（CP 模式），测量维持条件所需的功率或温度变化。

King 定律（CT 模式）：`P = (A + B · √(ρ · v)) · (T_sensor - T_fluid)`

特点：结构最简单；量程宽；无法直接判流向；适合 HVAC、工业管道。

### 1.3 飞行时间式（ToF）

加热器发出短热脉冲，下游检测脉冲到达时间 Δt。`v = L / Δt`。

特点：直接测流速，与热物性关系较小；适合多相流；微尺度下热扩散显著，信噪比挑战大。

### 1.4 三种模式对比

| 特性 | 量热式 | 风速式 | 飞行时间式 |
|------|--------|--------|------------|
| 测量量 | 上下游温差 ΔT | 加热功率/温度偏移 | 热脉冲飞行时间 |
| 流向判别 | 天然支持 | 不支持 | 支持 |
| 低流速灵敏度 | 极高 | 中等 | 较低 |
| 量程 | 中等（S 曲线饱和） | 宽 | 宽 |
| 物性依赖 | 中等 | 较强 | 较弱 |
| 结构复杂度 | 中等 | 最简单 | 较高 |
| 典型应用 | 微流控、呼吸机 | HVAC、管道流量 | 多相流、特殊流体 |

## 2. MEMS 加热器与热电堆微结构

### 2.1 加热器微结构

薄膜加热器：多晶硅（poly-Si）/ 铂（Pt）/ 氮化钛（TiN），厚度 100-300 nm，蛇形或折线形图形。铂的 TCR 约 0.00385 /K，多晶硅约 0.001 /K。

悬浮膜结构是关键创新：加热器制作在极薄介质膜（SiO₂/SiNₓ，1-2 μm）上，膜下方是背腔（KOH 或 DRIE 刻蚀）。就像在暖风机下垫隔热板——减少衬底传导损耗，把更多热量留给流体。

```
Q_total = Q_convection（传给流体，信号）
        + Q_conduction（传给衬底，损耗）
        + Q_radiation（辐射，可忽略）
设计目标：最大化 Q_convection / Q_total
```

关键参数：膜面积 500×500 μm² ~ 2×2 mm²；膜厚 1-2 μm（越薄隔热越好但强度下降）；背腔深 50-200 μm。

### 2.2 热电堆微结构

基于塞贝克效应，两种不同导体组成热电偶，冷热端温差产生热电动势：`V_out = N · α · ΔT`。N 为串联对数，α 为塞贝克系数（μV/K）。

| 材料对 | 塞贝克系数 (μV/K) | CMOS 兼容 | 备注 |
|--------|-------------------|-----------|------|
| n-p 多晶硅 | 100-200 | 是 | 最常用 |
| Al/p-Si | 50-100 | 是 | 铝做冷端互连 |
| Bi₂Te₃/Sb₂Te₃ | 200-400 | 否 | 性能最优，需后端工艺 |
| Cr/Au | 10-30 | 是 | 系数低，结构简单 |

布局：上下游热电堆关于加热器对称，各 10-40 对串联。热端距加热器 50-200 μm，冷端延伸至衬底等温区。

### 2.3 三层结构剖面

```
┌──────────────────────────────────────┐
│  流体流动方向 →                        │
├──────────────────────────────────────┤
│  热电堆(上) │  加热器  │  热电堆(下)   │  传感层
├──────────────────────────────────────┤
│       SiO₂ / SiNₓ 介质膜 (1-2 μm)    │  隔热层
├──────────────────────────────────────┤
│            背腔 (空气/真空)            │  隔热腔
├──────────────────────────────────────┤
│            Si 衬底                    │  机械支撑
└──────────────────────────────────────┘
```

背腔让加热器像"悬索桥"，两侧锚定、中间悬空。热量被迫向上方流体传递——这是高灵敏度的物理基础。

## 3. 热传递建模

### 3.1 传导（Conduction）

傅里叶定律：`q_cond = -k · ∇T`

MEMS 中的传导路径：(1) 沿介质膜向衬底锚点——主要损耗；(2) 沿热电偶向冷端——正常工作所需；(3) 通过背腔残余气体——需真空封装抑制。

悬浮膜传导热阻：`R_cond = L_membrane / (k_SiO2 · A_cross_section)`。膜越薄越长，隔热越好。

### 3.2 对流（Convection）

牛顿冷却定律：`q_conv = h · (T_surface - T_fluid)`

这是工作信号来源。MEMS 尺度下 Re < 100，层流为主。对流换热系数 h 与流速非线性——低速时自然对流叠加强制对流。努塞尔数关联（层流平板）：`Nu_x = 0.332 · Re_x^(1/2) · Pr^(1/3)`。

### 3.3 辐射（Radiation）

`q_rad = ε · σ · (T_s⁴ - T_∞⁴)`。MEMS 尺度下可忽略——加热器温度仅比环境高 10-50 K，面积 mm² 级，辐射比传导对流小 3-4 个数量级。

### 3.4 稳态热平衡

```
P_heater = Q_cond + Q_conv + Q_rad ≈ Q_cond + Q_conv
Q_conv = f(ρ, v, c_p, μ, k_fluid, geometry)  ← 包含流量信息
```

核心设计目标：让 Q_conv 占比尽可能大（高灵敏度），Q_cond 尽可能小且稳定（低噪声低漂移）。

## 4. 流量传感器类型对比

| 对比维度 | 热式 MEMS | 科里奥利式 | 超声波式 | 差压式 |
|----------|-----------|-----------|----------|--------|
| 测量原理 | 热传导/对流 | 科里奥利力 | 声波传播时间 | 伯努利方程 |
| 直接测量量 | 质量流量 | 质量流量 | 体积流量 | 体积流量 |
| 量程比 | 1:50~1:100 | 1:20~1:50 | 1:100~1:200 | 1:3~1:10 |
| 精度 | ±1%~±3% FS | ±0.1%~±0.5% | ±0.5%~±2% | ±1%~±3% FS |
| 低流速灵敏度 | 极高 | 较低 | 低 | 低 |
| 压损 | 极小 | 中等 | 极小 | 较大 |
| 体积 | 极小（芯片级） | 大 | 中等 | 中等 |
| 成本 | 低 | 高 | 中等 | 低 |
| 响应时间 | 1-100 ms | 50-200 ms | 10-100 ms | 10-50 ms |
| 典型 IoT 应用 | 呼吸机、燃气表 | 工业配比 | 大管径水/气 | HVAC 风量 |

选择建议：微流量+快响应+低成本→热式；高精度质量流量→科里奥利；大管径非侵入→超声波；简单场景→差压式。

## 5. 动态响应与时间常数

### 5.1 一阶热系统模型

`τ · dT/dt + T = T_ss(v)`，τ = R_th · C_th。阶跃输入后输出达稳态 63.2% 所需时间即为 τ。

### 5.2 影响 τ 的因素

| 因素 | 影响 | 典型值 |
|------|------|--------|
| 介质膜厚度 | 越厚→C_th 越大→τ 越大 | 1-2 μm |
| 加热器面积 | 越大→C_th 越大→τ 越大 | 0.01~1 mm² |
| 流体速度 | 越快→强制对流增强→τ 减小 | 空气 0-10 m/s |
| 封装热容 | 增加系统 C_th | 环氧/陶瓷 |

### 5.3 提升响应速度策略

1. 减小加热器质量（降低 C_th）
2. 悬浮膜结构（减少衬底热耦合）
3. CT 模式代替 CP 模式（反馈环路加速响应）
4. 脉冲驱动（仅测量时加热，降低功耗）

典型值：体硅结构 τ ≈ 50-200 ms；悬浮膜 τ ≈ 1-20 ms；脉冲+CT 模式有效 τ ≈ 0.5-5 ms。带宽 `f_3dB = 1/(2π·τ)`，τ=5ms 时约 32 Hz，呼吸机应用（0.2-1 Hz）绰绰有余。

## 6. 封装与介质兼容性

### 6.1 封装类型

| 封装方式 | 优点 | 缺点 |
|----------|------|------|
| DIP/SOP + 旁通管 | 标准化、易替换 | 死体积大、响应慢 |
| 裸片 + 微通道 | 死体积小、响应快 | 工艺复杂 |
| 陶瓷基板 + 倒装 | 耐高温腐蚀 | 成本较高 |
| 3D 打印集成 | 设计灵活 | 一致性挑战 |

### 6.2 介质兼容

气体测量相对简单；腐蚀性气体需 SiNₓ/Parylene 钝化；湿度影响热物性需补偿。液体测量挑战更大：热导率高导致信噪比下降；需电隔离；长期浸泡需惰性保护；气泡会剧烈干扰温度场。

### 6.3 防护涂层

| 涂层 | 厚度 | 耐化学性 | 热阻影响 |
|------|------|----------|----------|
| Parylene C | 1-5 μm | 优良 | 小 |
| Si₃N₄ | 0.5-1 μm | 极优 | 极小 |
| 环氧树脂 | 50-200 μm | 中等 | 大 |

Parylene C 最常用——1-2 μm 即可电隔离，又不显著增加热阻。

## 7. 不同气体/液体的标定

### 7.1 为何需要标定

热式传感器测的是热传递速率而非质量流量：`Q_conv ∝ f(ρ, c_p, μ, k_fluid, v)`。同一传感器测空气和水，质量流量相同时信号可差 10 倍以上。

### 7.2 气体标定

直接标定：用标准流量计提供已知流量，拟合标定曲线。低流速段密集取点（S 曲线非线性），中流速段可减少，高流速段适当加密。

气体转换系数法（以空气为基准）：`Q_gas = Q_air · K_gas`，`K_gas = √(ρ_air · c_p,air) / √(ρ_gas · c_p,gas)`

| 气体 | K_gas | 说明 |
|------|-------|------|
| 空气 | 1.00 | 基准 |
| N₂ | 1.00 | 与空气接近 |
| CO₂ | 0.70 | 密度大、比热容高 |
| He | 1.38 | 密度小、热导率极高 |
| Ar | 0.86 | 密度大、热导率低 |
| CH₄ | 0.78 | 天然气主要成分 |

> 转换系数法精度约 ±5%，高精度应用仍需实测标定。

### 7.3 液体标定与温度补偿

液体标定更难：流量更小（μL/min~mL/min）、标准设备贵、温度对粘度影响剧烈、气泡干扰。

温度补偿策略：芯片上集成参考温度传感器，多项式拟合 `Flow = f(V_sensor, T_fluid)`，或二维查表法。

## 8. 代码示例：带补偿的流量传感器读取

### 8.1 基础读取与温度补偿

```python
from dataclasses import dataclass
from typing import Optional

@dataclass
class ThermalFlowSensor:
    """MEMS 热式流量传感器模型（量热式）"""
    name: str
    offset: float = 0.0          # 零流量偏置 (mV)
    sensitivity: float = 12.5    # 灵敏度 (mV / (m/s))
    tco: float = -0.002          # 零点温度系数 (/K)
    tcs: float = -0.001          # 灵敏度温度系数 (/K)
    ref_temp: float = 25.0       # 标定参考温度 (°C)
    gas_factor: float = 1.0      # 气体转换系数
    max_flow: float = 10.0       # 最大流速 (m/s)

    GAS_FACTORS = {'air': 1.0, 'N2': 1.0, 'O2': 0.98,
                   'CO2': 0.70, 'He': 1.38, 'Ar': 0.86, 'CH4': 1.28}

    def read_flow(self, voltage_mv: float, fluid_temp: float,
                  gas: Optional[str] = None) -> dict:
        """从传感器电压计算补偿后流速"""
        if gas and gas in self.GAS_FACTORS:
            self.gas_factor = self.GAS_FACTORS[gas]
        dt = fluid_temp - self.ref_temp
        offset_comp = self.offset * (1 + self.tco * dt)
        sens_comp = self.sensitivity * (1 + self.tcs * dt)
        raw_flow = (voltage_mv - offset_comp) / sens_comp
        comp_flow = raw_flow * self.gas_factor
        saturated = abs(comp_flow) > self.max_flow * 0.95
        return {"flow_mps": round(comp_flow, 4),
                "raw_flow_mps": round(raw_flow, 4),
                "gas_factor": self.gas_factor,
                "status": "saturated" if saturated else "ok"}

# 使用示例
sensor = ThermalFlowSensor(name="SFM3000", offset=0.5)
result = sensor.read_flow(voltage_mv=63.0, fluid_temp=30.0, gas="CO2")
print(f"补偿后流速: {result['flow_mps']} m/s, 状态: {result['status']}")
```

### 8.2 动态响应模拟

```python
import numpy as np

class FlowSensorDynamics:
    """热式流量传感器动态响应模型（一阶低通）"""

    def __init__(self, tau_ms: float = 5.0, dt_ms: float = 0.1):
        self.tau = tau_ms
        self.dt = dt_ms
        self.alpha = dt_ms / (tau_ms + dt_ms)
        self.output = 0.0

    def step_response(self, step_value: float = 1.0,
                      n_steps: int = 500) -> list:
        """模拟阶跃响应"""
        response = []
        self.output = 0.0
        for _ in range(n_steps):
            self.output += self.alpha * (step_value - self.output)
            response.append(self.output)
        return response

    def bandwidth_hz(self) -> float:
        """-3dB 带宽"""
        return 1.0 / (2 * np.pi * self.tau * 1e-3)

# 不同时间常数的稳态时间
for tau in [1, 5, 20]:
    dyn = FlowSensorDynamics(tau_ms=tau)
    resp = dyn.step_response()
    # 找到达到 63.2% 的时间
    idx63 = next(i for i, v in enumerate(resp) if v >= 0.632)
    t63 = idx63 * dyn.dt
    print(f"τ={tau}ms: 63.2%时间={t63:.1f}ms, "
          f"带宽={dyn.bandwidth_hz():.1f}Hz")
```

## 9. IoT 应用场景

### 9.1 医疗呼吸机

MEMS 热式流量传感器是现代呼吸机核心部件。需求：量程 0-120 L/min（成人）、分辨率 < 0.1 L/min、响应 < 10 ms、双向测量、医疗级电隔离。

选热式 MEMS 的理由：极低死体积不影响呼吸功；快响应实时追踪呼吸波形；小型化可集成到管路；低压损。

代表产品：Sensirion SFM3019（FDA 510(k) 认证）。

### 9.2 HVAC 智能风量控制

需求：0-10 m/s、±3% FS、10 年免维护、低成本（< $5）。热式优势：可测 0.01 m/s 极低风速，无活动部件，同时输出温度。

```
MEMS 流量传感器 → MCU (ADC + 补偿) → Modbus/BLE → 楼宇自控 → VAV 风门
```

### 9.3 工业过程控制

| 行业 | 应用 | 流体 | 特殊要求 |
|------|------|------|----------|
| 半导体 | CVD 气体控制 | SiH₄, NF₃ | 耐腐蚀、超高纯度 |
| 食品饮料 | 配比灌装 | 液体/糖浆 | 卫生级、CIP 清洗 |
| 化工 | 反应釜进气 | 多种气体 | 防爆认证 |
| 能源 | 燃气计量 | 天然气/沼气 | 防爆、宽量程 |

工业级需 ATEX/IECEx 防爆、316L 不锈钢隔膜隔离、HART/Modbus/IO-Link 通信。

### 9.4 新兴应用

- 氢能：燃料电池氢气监测（热导率极高，需专用标定）
- 电池热管理：液冷回路微流量（液体介质、低流速）
- 实验室仪器：GC 载气控制、微反应器（极高精度）
- 可穿戴：呼吸监测贴片（超低功耗、柔性封装）

## 10. 总结与展望

### 核心要点

1. **三种模式各有所长**：量热式→低流速双向，风速式→大量程，ToF→特殊流体
2. **悬浮膜是结构灵魂**——减少衬底热漏、提高灵敏度
3. **热电堆串联放大微弱温差**——塞贝克系数和对数是关键参数
4. **对流是信号，传导是损耗，辐射可忽略**——设计即最大化信号/损耗比
5. **热物性依赖是双刃剑**——能测质量流量，但换介质必须标定
6. **τ = R_th · C_th**——减小 τ 靠减热容和减热耦合

### 技术趋势

短期（1-3 年）：多气体自动识别（AI 切换标定曲线）；更低功耗（脉冲驱动，电池 > 5 年）；边缘智能补偿。

中期（3-5 年）：柔性衬底贴合不规则管道；3D 打印微流控集成；流量+温度+湿度+压力单芯片四合一。

长期（5-10 年）：基于物理模型的自标定；量子增强传感突破灵敏度极限；可降解植入式流量传感器。

### 研究挑战

1. 多组分混合气体（天然气、沼气）成分变化大，单一转换系数不够
2. 两相流热传递模型尚不成熟
3. 纳米级污染累积导致长期漂移，需自诊断
4. 极端环境（>200°C、强腐蚀）封装瓶颈
5. 缺乏统一测试标准，厂商数据难以对比
