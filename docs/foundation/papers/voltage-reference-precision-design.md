# 精密电压基准源设计与噪声分析

> **难度**：🔴 高级 | **领域**：精密模拟 | **关键词**：Bandgap, 温漂, 1/f 噪声 | **阅读时间**：约 16 分钟

## 日常类比

天平要有标准砝码。模数转换器（ADC）与传感器激励需要电压基准（Voltage Reference）；基准的温漂与噪声会直接变成测量误差[1][2]。

## 摘要

对比带隙、埋齐纳等技术，讨论初始精度、温漂、长期稳定性、噪声密度与负载调整。手册 ppm 数字依赖测试条件，系统布局可毁掉好基准[2][3]。

## 1. 关键指标

| 指标 | 含义 |
|------|------|
| 初始误差 | 未校准偏差 |
| 温漂 | ppm/°C |
| 长期漂移 | ppm/√kh 量级表述常见 |
| 噪声 | μVpp（0.1–10 Hz）等 |
| 负载调整 | 电流变化影响 |

| 技术 | 特点 |
|------|------|
| 带隙 | 低电压、常见、成本友好 |
| 埋齐纳 | 低噪声低漂，偏高压/成本 |
| XFET 等 | 厂商工艺特性 |

## 2. 应用注意

为基准提供干净电源与去耦；Kelvin 连接到 ADC 参考脚；避免大电流地回流共享。缓冲驱动开关电容参考输入[3]。

## 3. 局限、挑战与可改进方向

### 1. 自热

**局限**：负载电流导致温升改变输出。
**改进**：降负载；热隔离布局[2]。

### 2. 焊接与应力迟滞

**局限**：PCB 弯曲改变输出。
**改进**：机械应力释放；回流后老化再校准[4]。

### 3. 低频噪声主导直流测量

**局限**：1/f 限制有效分辨率。
**改进**：选低噪声基准；数字滤波与斩波前端[1]。

### 4. 过度规格浪费

**局限**：0.01% 基准配 10 位 ADC。
**改进**：按误差预算选型[3]。

## 总结

精密基准是系统误差预算的锚点。选型看温漂与噪声，落地看布局与应力，最后用实测闭环。
