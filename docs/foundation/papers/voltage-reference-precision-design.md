# 精密电压基准源设计与温度系数分析

> **难度**：🟡 中级 | **领域**：精密模拟设计 | **阅读时间**：约 20 分钟

## 引言

想象你在菜市场买菜,摊主用的秤准不准,直接决定你买到的菜够不够分量。如果这把秤早上凉的时候和中午热的时候读数不一样,那你就永远不知道自己到底买了多少。ADC和DAC的电压基准源就是电子系统里的"秤"——它提供一个已知的稳定电压作为参考,所有测量和输出都以此为基准。基准不准,后面再多的位数都是空谈。

在IoT精密测量系统中,电压基准源的质量往往比ADC本身更关键。一颗16位ADC配上温漂100ppm/C的基准,实际有效精度可能还不如12位ADC配3ppm/C的精密基准。本文从基准源类型、关键指标、温度系数分析方法到实际选型与PCB布局,系统梳理精密电压基准设计的全链路。

## 1 电压基准源的作用与原理

### 1.1 为什么需要电压基准

ADC将模拟电压转换为数字码,需要一个已知的参考电压;DAC将数字码转换为模拟电压,同样需要基准;比较器判断阈值也需要参考。基准源的核心任务就是提供一个不随温度、时间、负载和电源变化的稳定已知电压。

转换公式:

```
ADC输出码 = (Vin / Vref) x (2^N - 1)
DAC输出电压 = (Dcode / 2^N) x Vref
```

Vref偏了1%,所有结果都偏1%。这不是分辨率能弥补的——它是系统级增益误差。

### 1.2 基准源的核心要求

| 要求 | 含义 | 为什么重要 |
|------|------|-----------|
| 精确 | 输出电压接近标称值 | 决定系统初始精度 |
| 稳定 | 随温度/时间/负载变化小 | 决定长期可用精度 |
| 低噪声 | 输出电压波动小 | 影响ADC有效位数 |
| 低功耗 | 静态电流小 | IoT电池寿命 |

精确可校准,稳定才是基准源真正的技术门槛。

## 2 基准源类型

### 2.1 带隙基准(Bandgap)

带隙基准基于硅的带隙电压(约1.12V)原理:将负温度系数的Vbe(约-2mV/C)与正温度系数的PTAT电压按比例相加,使总温度系数趋近零:

```
Vbg = Vbe(CTAT) + K x Vt(PTAT) = 约1.2V
```

特点:输出约1.2V,可通过电阻分压产生2.048V/2.5V/3.0V/4.096V等标准值;工艺成熟;成本低;适用于大多数IoT场景。

### 2.2 齐纳基准(Zener)

利用硅的雪崩击穿效应,击穿电压通常在2.5V-10V之间。击穿电压在5.5V以上时噪声较低,5V以下噪声显著增大。

特点:输出电压较高;需要供电电压高于击穿电压;噪声中等;适合高压系统。

### 2.3 掩埋齐纳基准(Buried Zener)

在硅片表面以下制造齐纳二极管,避免表面缺陷导致的噪声和漂移。

特点:噪声最低;长期漂移最小;需要大于6V的供电;价格高;适用于实验室级仪器和校准设备。

### 2.4 类型对比

| 类型 | 输出电压 | 噪声 | 温漂 | 成本 | 典型应用 |
|------|---------|------|------|------|---------|
| 带隙 | 1.2-5V | 中 | 中 | 低 | IoT通用采集 |
| 齐纳 | 2.5-10V | 中高 | 中高 | 中 | 高压系统 |
| 掩埋齐纳 | 6-10V | 极低 | 极低 | 高 | 实验室仪器 |
| XFET | 2-5V | 低 | 低 | 中高 | 精密测量 |

## 3 关键性能指标

### 3.1 初始精度

25C下输出电压与标称值的最大偏差,以百分比或mV表示。初始精度可通过校准消除,因此不是最关键指标——但校准需要额外工序和成本。

典型值:
- 通用基准: 0.5-2%
- 精密基准: 0.05-0.1%
- 超精密基准: 0.01-0.02%

### 3.2 温度系数(TC)

温度系数是基准选型中最重要的指标,单位ppm/C,表示温度每变化1度基准电压的相对变化量:

```
TC = (Vmax - Vmin) / (Tmax - Tmin) / Vnom x 10^6   (ppm/C)
```

以2.5V基准在-40C到+85C范围内为例:

| TC(ppm/C) | 温度范围 | 最大偏差 | 等效LSB(16位) |
|-----------|---------|---------|--------------|
| 100 | 125C | 31.25mV | 82 LSB |
| 50 | 125C | 15.6mV | 41 LSB |
| 10 | 125C | 3.125mV | 8 LSB |
| 3 | 125C | 0.94mV | 2.5 LSB |
| 0.5 | 125C | 0.156mV | 0.4 LSB |

TC是基准的"身份证等级",直接决定系统可用精度。

### 3.3 长期漂移

基准电压随时间的缓慢变化,单位ppm/kHr(ppm每1000小时):

- 通用基准: 50-100 ppm/kHr
- 精密基准: 10-25 ppm/kHr
- 超精密基准: < 5 ppm/kHr

长期漂移呈对数衰减:前1000小时变化最大,之后逐渐趋稳。精密设备出厂前会做老化(Burn-in)处理加速稳定。

### 3.4 噪声

基准噪声直接叠加在ADC结果上,两种指标形式:

- 宽带噪声: 10Hz-10kHz,用uVrms表示
- 1/f低频噪声: 0.1Hz-10Hz,用uVpp表示

对高分辨率ADC,低频噪声更关键,因为它无法被数字滤波消除。

### 3.5 电源与负载调整率

- 电源调整率(Line Regulation): 输入电压变化1V时输出变化量(uV/V)
- 负载调整率(Load Regulation): 负载电流变化1mA时输出变化量(uV/mA)

串联基准通常两项都优于并联基准。

### 3.6 静态电流

对电池供电的IoT设备很重要:
- 串联基准: 0.5uA-2mA
- 并联基准: 需要最小工作电流(如50-100uA)加上负载电流

## 4 温度系数深度分析

### 4.1 方框法(Box Method)

在全温度范围内取最大值和最小值,计算:

```
TC_box = (Vmax - Vmin) / (Tmax - Tmin) / Vnom x 10^6
```

数据手册常用此方法,偏保守——实际大部分温度范围内温漂可能远小于此值。

### 4.2 蝴蝶图法(Butterfly Curve)

每个温度点取多器件统计分布,画出类似蝴蝶翅膀的包络线。更精确地反映实际性能,但数据获取困难,通常只有精密基准才提供。

### 4.3 TC等级划分

| 等级 | TC范围 | 适用ADC位数 | 典型器件 |
|------|--------|-----------|---------|
| C级 | 50-100 ppm/C | 10-12位 | LM4040C |
| B级 | 10-50 ppm/C | 12-14位 | REF3030 |
| A级 | 3-10 ppm/C | 14-16位 | REF5025 |
| 超A级 | < 3 ppm/C | 16-24位 | ADR4540 |

### 4.4 温漂不可校准的本质

初始精度可通过单点校准消除,但温漂是曲线特性——除非你为每个温度点都做校准,否则温漂始终存在。选择低温漂器件是最经济有效的方式,软件补偿只是补充手段。

## 5 主流基准芯片对比

### 5.1 REF3030(TI)

串联带隙基准,3.0V输出,0.2%精度,25ppm/C TC,42uA IQ,SOT-23封装。

适用场景: 电池供电IoT的12-14位ADC基准,3.0V方便配合单节锂电池系统。

### 5.2 LM4040(TI/NSC)

并联带隙基准,2.048/2.5/4.096V多档,0.1%精度,50-100ppm/C,SOT-23封装。

适用场景: 4-20mA电流环、低成本12位ADC基准、需要负基准的场合。

### 5.3 ADR4540(ADI)

串联XFET基准,4.096V输出,0.02%精度,0.5ppm/C TC,1.0uVpp噪声。

适用场景: 24位精密测量、医疗仪器、实验室校准。温漂和噪声均为业界顶级。

### 5.4 MAX6070(ADI/Maxim)

串联带隙基准,2.5/3.0/3.3V可选,0.04%精度,3ppm/C TC,1.5uVpp低噪声。

适用场景: 高精度低噪声IoT采集、16-18位ADC基准。

### 5.5 全面对比表

| 参数 | REF3030 | LM4040 | ADR4540 | MAX6070 |
|------|---------|--------|---------|---------|
| 拓扑 | 串联 | 并联 | 串联(XFET) | 串联 |
| 输出电压 | 3.0V | 2.048/2.5/4.096V | 4.096V | 2.5/3.0/3.3V |
| 初始精度 | 0.2% | 0.1% | 0.02% | 0.04% |
| TC(max) | 25 ppm/C | 50 ppm/C | 0.5 ppm/C | 3 ppm/C |
| 噪声(0.1-10Hz) | 12 uVpp | 23 uVpp | 1.0 uVpp | 1.5 uVpp |
| IQ | 42 uA | 60 uA(最小) | 950 uA | 1.2 mA |
| 参考价格(1K) | ~$0.6 | ~$0.5 | ~$6 | ~$3 |

## 6 并联与串联拓扑详解

### 6.1 并联基准(Shunt)

并联基准像稳压二极管,与负载并联工作:

```
Vcc ---[Rbias]---+--- Vref
                 |
               [Ref]  (并联基准器件)
                 |
                GND
```

偏置电阻Rbias需满足:在任何负载和电源条件下,流过基准的电流都在规格范围内。计算:

```
Rbias = (Vcc_min - Vref) / (Iref_min + Iload_max)
```

优点: 可从任意高于Vref的电源供电;可做负基准;适合电流环。

缺点: 空载也耗电(始终流过Iref);PSRR差;负载调整率不如串联。

### 6.2 串联基准(Series)

串联基准像LDO,输入输出串联:

```
Vcc ---[Ref IN]---[Ref OUT]--- Vref
```

优点: 静态电流低;PSRR高;负载调整率好;无需计算偏置电阻。

缺点: 输入电压须高于Vref+dropout;不方便做负基准。

### 6.3 IoT系统选型建议

| 条件 | 推荐 |
|------|------|
| 电池供电、低功耗 | 串联基准(如REF3030) |
| 电源波动大 | 串联基准(PSRR好) |
| 4-20mA电流环 | 并联基准(如LM4040) |
| 需要负基准或浮地 | 并联基准 |
| 通用采集 | 串联基准(更简单) |

## 7 开尔文连接(Kelvin Connection)

### 7.1 为什么需要开尔文连接

普通的两线制连接中,PCB走线和焊盘的电阻与基准串联,产生电压降:

```
Vref_actual = Vref_output - Iload x Rtrace
```

如果走线电阻10m欧,负载电流10mA,就产生100uV压降——对16位ADC这已经是2.6个LSB。

### 7.2 四线制原理

开尔文连接用两根线提供电流(Force),另外两根线检测电压(Sense):

```
Force+ ---[Rtrace1]---+--- 基准负载端
                      |
Sense+ ---------------+  (Sense直接连在负载端)

Force- ---[Rtrace2]---+--- 基准地端
                      |
Sense- ---------------+  (Sense直接连在负载地端)
```

Sense线流过的电流极小(运放输入偏置电流),走线电阻上的压降可忽略。测量的是负载端的真实电压,消除了Force走线电阻的误差。

### 7.3 实际PCB实现

在基准源输出焊盘处引出两对走线:一对走大电流给负载供电(Force),一对仅连到ADC的VREF引脚做检测(Sense)。Sense走线要细,Force走线要宽。

## 8 去耦与滤波设计

### 8.1 输出去耦电容

基准输出端就近放置去耦电容,提供瞬态电流并滤除高频噪声:
- 1uF X7R陶瓷电容 + 0.1uF C0G陶瓷电容并联
- 避免Y5V/Z5U介质(偏压特性和温漂差)
- 电容ESR必须低于基准规格允许的最大值(否则可能振荡)

### 8.2 低通滤波

基准输出与ADC之间加RC低通滤波,降低宽带噪声:

```
Vref ---[R]---+--- ADC VREF
               |
              [C]
               |
              GND

fc = 1 / (2 x pi x R x C)
```

R取10-100欧,C取1-10uF。例如R=22欧,C=4.7uF,fc=1.5kHz。

注意:R过大时ADC采样瞬态电流会在R上产生直流压降,需确保压降在可接受范围内。

### 8.3 电容稳定性

某些基准对输出电容的ESR范围有要求。ESR过低可能导致零点移位引发振荡;ESR过高则滤波效果差。必须查阅数据手册的电容稳定性曲线。

## 9 实际应用示例

### 9.1 IoT精密温度测量

系统需求: PT100 RTD温度测量,精度0.1C,温度范围-50C到+200C。

方案: ADS1118(16位ADC) + REF3030(3.0V基准) + PT100三线制接法。

```
REF3030(3.0V) ---[RC滤波]--- ADS1118 VREF
                              |
PT100 ---[INA] ------------ ADS1118 AIN0/AIN1
```

REF3030提供3.0V基准,25ppm/C温漂在工业温域内约0.3%误差,配合两点校准可满足0.1C精度要求。REF3030仅42uA静态电流适合电池供电。

### 9.2 16位DAC输出校准

DAC8562(16位双通道DAC)输出需要校准,基准精度决定输出精度:

```
ADR4540(4.096V) ---[R=10欧]---+--- DAC8562 VREF
                                |
                              [4.7uF]
                                |
                               GND
```

ADR4540的0.5ppm/C温漂确保DAC输出在-40C到+85C范围内变化不超过0.006%,对工业控制足够。

## 10 PCB布局要点

### 10.1 布局原则

1. 基准紧邻ADC/DAC,缩短Vref走线
2. 基准电路使用独立模拟地平面
3. 远离开关电源和功率器件,至少5mm间距
4. Vref走线不穿越数字信号区

### 10.2 保护环(Guard Ring)

在基准输出走线周围布置接地保护环,屏蔽漏电流和外部干扰:

```
          GND GND GND
          GND       GND
          GND Vref  GND
          GND       GND
          GND GND GND
```

保护环接低阻抗地,可显著降低PCB表面漏电流对高阻抗基准节点的影响。

### 10.3 热对称布局

- 基准与ADC放在同一热区,温度变化一致可部分抵消温漂
- 基准两侧对称放置相同阻值的电阻,使温度梯度均匀
- 基准下方不要铺铜走大电流
- 高精度系统可在基准周围做热隔离沟槽

### 10.4 远离热源

必须远离的发热器件:DC-DC转换器、LDO、功率MOSFET、大电流走线。热源的温度波动会直接调制基准输出电压。

## 11 基准选型指南

### 11.1 选型决策流程

```
ADC <= 10位,精度 <= 2%     --> MCU内部基准即可
ADC = 12位,比例测量        --> VDD基准
ADC = 12位,绝对测量        --> REF3030级
ADC = 14-16位              --> REF5025 / MAX6070级
ADC >= 18位                --> ADR4540级
```

### 11.2 关键选型参数

| 优先级 | 参数 | 原因 |
|--------|------|------|
| 1 | TC | 温漂不可校准,决定可用精度 |
| 2 | 噪声 | 影响ADC有效位数 |
| 3 | 长期漂移 | 决定校准周期 |
| 4 | 初始精度 | 可通过校准消除 |
| 5 | 静态电流 | IoT系统功耗约束 |
| 6 | 输出电压 | 匹配ADC量程 |

### 11.3 不同应用场景推荐

| 场景 | 推荐基准 | 原因 |
|------|---------|------|
| IoT传感器(12位) | REF3030 | 低功耗,25ppm/C够用 |
| 工业采集(16位) | MAX6070 | 3ppm/C,低噪声 |
| 精密秤重(24位) | ADR4540 | 0.5ppm/C,超低噪声 |
| 电流环(4-20mA) | LM4040 | 并联型,适合环路供电 |
| 便携设备 | REF3030 | 42uA超低功耗 |

### 11.4 封装热阻考量

基准芯片自身发热也会影响精度。封装热阻越大,芯片结温越高,温漂越大:

| 封装 | 热阻(JA) | 说明 |
|------|---------|------|
| SOT-23 | 200-300 C/W | 小封装热阻大 |
| SOIC-8 | 150-180 C/W | 中等 |
| MSOP-8 | 100-150 C/W | 较好 |
| DIP-8 | 80-120 C/W | 散热好但体积大 |

功耗=P=VxI,温升=P x Rja。如REF3030功耗3.0V x 42uA = 0.126mW,SOT-23封装温升仅0.03C,可忽略。

## 总结

电压基准源是精密模拟系统的基石,回到开头的类比:秤不准,菜就称不对。核心要点:

1. **温漂是头号敌人**: 初始精度可校准,温漂只能选型或补偿。选型时TC指标优先级最高。
2. **基准类型按需选择**: IoT通用用带隙串联型,超精密用XFET或掩埋齐纳,电流环用并联型。
3. **开尔文连接消除走线误差**: 16位以上系统必须考虑四线制,避免走线电阻吃掉精度。
4. **去耦滤波不可省**: RC低通降噪、电容稳定性检查、避免ESR引发的振荡。
5. **PCB布局定生死**: 远离热源、保护环、热对称、模拟地独立——好基准配烂布局等于白搭。
6. **选型优于补偿**: 优先选低温漂基准,软件补偿是锦上添花不是雪中送炭。

精密测量系统的设计,基准是起点也是终点。选对了基准并做好布局,后续的信号调理和数字处理才有意义。
