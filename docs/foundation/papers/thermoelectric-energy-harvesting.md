# 温差发电热电能量采集器件与电路

> **难度**：🔴 高级 | **领域**：热电能量采集 | **阅读时间**：约 22 分钟

## 引言

你有没有摸过夏天被太阳晒过的铁栏杆？栏杆烫手，旁边的空气却不烫。栏杆和空气之间存在温度差，而这个温差里就藏着能量。

温差发电的原理可以比作一个"热的水车"。传统水车靠水流从高处落下的势能驱动，温差发电则靠热量从高温端"流"向低温端的过程中驱动载流子运动，从而产生电能。不同的是，水车的效率可以很高，而温差发电的效率通常只有个位数百分比——但对功耗极低的 IoT 传感器来说，已经够用了。

本文深入分析温差发电的核心物理机制、器件结构、电路设计以及 IoT 应用中的实际考量。

## 1. 塞贝克效应与热电物理基础

### 1.1 塞贝克效应

1821 年塞贝克发现：当两种不同导体（或半导体）组成闭合回路，且两个接点处于不同温度时，回路中会产生电动势。开路电压与温差成正比：

```
V_oc = S * delta_T
```

其中 S 是塞贝克系数（uV/K），delta_T 是热端与冷端的温度差。单对热电偶的 S 量级在几十到几百 uV/K，为获得实用电压需将多对热电偶串联。

### 1.2 三大热电效应

| 效应 | 描述 | 方向关系 |
|------|------|----------|
| 塞贝克效应 | 温差产生电压 | 热 -> 电 |
| 帕尔帖效应 | 电流产生温差 | 电 -> 热 |
| 汤姆逊效应 | 电流在温度梯度中吸/放热 | 电 + 热梯度 -> 热 |

帕尔帖效应是塞贝克的逆效应，在温差发电中表现为额外热流，会降低器件效率。汤姆逊效应影响较小，简化分析中通常忽略。

### 1.3 微观解释与 PN 对结构

在半导体中，载流子在热端动能高，倾向向冷端扩散。N 型半导体冷端积累负电荷，P 型冷端积累正电荷。P 型和 N 型塞贝克系数符号相反，串联后电压叠加——这正是 TEG 采用 PN 对结构的原因。

## 2. 热电发生器（TEG）结构

### 2.1 基本结构

TEG 由多对 P-N 热电偶电串联、热并联，夹在两块陶瓷基板之间：

```
        热端 (T_h)
  ========================
  | P |  | N |  | P |  | N |   <-- 热电偶对（电串联）
  ========================
        冷端 (T_c)
```

陶瓷基板（通常 Al2O3）提供电绝缘、热导通和机械支撑。

### 2.2 商用 TEG 参数与分类

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 热电偶对数 | 31-127 对 | 决定开路电压 |
| 内阻 | 1-10 Ohm | 匹配负载关键 |
| 最大温差 | 200-250 K | 超过会损坏 |
| 尺寸 | 15x15 到 62x62 mm | 受安装空间约束 |

按温度范围分类：

| 类型 | 温度范围 | 典型材料 | 典型应用 |
|------|----------|----------|----------|
| 低温 TEG | < 450K | Bi2Te3 | 可穿戴、工业管道 |
| 中温 TEG | 450-850K | PbTe, Skutterudite | 汽车尾气余热 |
| 高温 TEG | > 850K | SiGe | 航天 RTG |

IoT 领域几乎全部使用低温 Bi2Te3 基 TEG，因为 IoT 温差通常在 1-30K。

## 3. 热电优值 ZT 与材料体系

### 3.1 ZT 定义

```
ZT = (S^2 * sigma * T) / kappa
```

- S：塞贝克系数（越大越好）
- sigma：电导率（越大越好）
- kappa：热导率（越小越好）

商用 Bi2Te3 的 ZT 约 1.0，实验室材料可达 1.5-2.0。

### 3.2 ZT 中的根本矛盾

根据 Wiedemann-Franz 定律，电子热导率与电导率成正比：

```
kappa_e = L * sigma * T
```

提高电导率的同时电子热导率也增大，不利于维持温差。突破关键在于降低晶格热导率 kappa_l 而不降低电导率——"声子玻璃-电子晶体"（PGEC）概念：让声子像在玻璃中一样散射，载流子像在晶体中一样自由运动。

### 3.3 主流热电材料

| 材料体系 | 最佳 ZT | 工作温区 | 优势 | 劣势 |
|----------|---------|----------|------|------|
| Bi2Te3 | ~1.0 | 室温附近 | ZT 最高、技术成熟 | 含 Te 昂贵、机械脆 |
| PbTe | ~1.2 | 500-800K | 中温 ZT 高 | 含 Pb 有毒 |
| SiGe | ~0.9 | >1000K | 高温稳定 | 成本高 |
| Mg3Sb2 | ~1.2 | 300-700K | 无毒、元素丰富 | 稳定性待验证 |

Bi2Te3 是 IoT 温差发电核心材料：菱方晶系层状结构，各向异性显著，P 型掺 Sb（Bi0.5Sb1.5Te3），N 型掺 Se（Bi2Te2.7Se0.3），纳米析出相可将 ZT 从 1.0 提升至 1.3-1.5。

## 4. TEG 电气模型与最大功率传输

### 4.1 戴维南等效模型

```
          V_oc = S * N * delta_T
           |
           +--[ R_teg ]--+-----> V_load
           |              |
          (V_oc)        [R_load]
           |              |
           +--------------+
```

- V_oc = S * N * delta_T：开路电压
- R_teg：TEG 内阻（所有热电偶电阻 + 接触电阻）

### 4.2 阻抗匹配与最大功率

```
V_load = V_oc * R_load / (R_teg + R_load)
P_load = V_oc^2 * R_load / (R_teg + R_load)^2

当 R_load = R_teg 时，P_load 最大：
P_max = V_oc^2 / (4 * R_teg)
```

最大功率传输与最高效率条件略有不同（最大效率在 R_load 略大于 R_teg 时），实际设计按阻抗匹配近似。

### 4.3 数值示例

| 参数 | 值 |
|------|-----|
| 塞贝克系数 S | 200 uV/K |
| 对数 N | 127 |
| delta_T | 10 K |
| R_teg | 3 Ohm |

```
V_oc = 200e-6 * 127 * 10 = 0.254 V
P_max = 0.254^2 / (4 * 3) = 5.38 mW
```

5.38 mW 可驱动低功耗节点间歇工作，但 0.254 V 远低于 MCU 最低工作电压，必须升压。

## 5. 升压变换器与电源管理

### 5.1 低电压启动挑战

TEG 小温差输出仅几十到几百 mV，而 IoT 负载需 1.8-3.6V。升压变换器核心挑战：启动电压（20-300mV 启动）、变换效率、冷启动（储能耗尽时从零开始）。

### 5.2 LTC3108 方案

Analog Devices 专用热电升压芯片，IoT 温差发电最经典方案之一。

- 最低启动电压：20 mV（配 1:100 升压变压器）
- 升压拓扑：谐振升压变压器 + 整流
- 输出：可编程 2.35/3.3/4.1/5.0V，双路 LDO
- 效率：低输入功率下仅 20-40%（变压器损耗 + 整流压降）

启动原理：升压变压器匝数比将 TEG 低压提升约 100 倍，整流后为控制电路供电，启动后切换到正常模式管理输出和储能。

### 5.3 BQ25504 方案

TI 超低功耗升压充电管理芯片，设计思路不同：

- 最低启动：330 mV（冷启动），80 mV（热启动）
- 升压拓扑：同步升压变换器（无变压器）
- 最大效率：80-90%
- MPPT：分数开路电压法，默认工作点为 V_oc 的 50%
- 电池管理：过充/过放保护

### 5.4 方案对比与选型

| 特性 | LTC3108 | BQ25504 |
|------|---------|---------|
| 最低启动电压 | 20 mV | 330 mV |
| 升压方式 | 变压器 | 同步升压 |
| 最高效率 | 40% | 90% |
| MPPT | 无 | 有 |
| 适用场景 | 极小 delta_T | delta_T > 5K |

delta_T 极小（1-3K）时 LTC3108 几乎唯一选择；TEG 输出 300mV 以上时 BQ25504 效率和成本均更优。

### 5.5 MPPT 参考实现

```c
// 伪代码：MPPT 扰动观察法
void mppt_perturb_observe(uint16_t v_teg, uint16_t i_teg) {
    static uint16_t prev_power = 0;
    static int8_t duty_step = 1;

    uint16_t current_power = v_teg * i_teg;

    if (current_power > prev_power) {
        duty_cycle += duty_step;       // 功率增加，维持方向
    } else {
        duty_step = -duty_step;        // 功率减少，反转方向
        duty_cycle += duty_step;
    }

    prev_power = current_power;
    set_pwm_duty(duty_cycle);
}
```

实际产品多用更精确的分数开路电压法（BQ25504 的做法）。

## 6. 热设计：散热器与热界面材料

### 6.1 热路模型与热阻分压

P_max 正比于 delta_T^2，维持足够温差至关重要。等效热路：

```
T_hot ---> [R_hsink] ---> [R_TIM_hot] ---> [TEG] ---> [R_TIM_cold] ---> [R_csink] ---> T_cold
```

TEG 实际分得温差由热阻分压决定：

```
delta_T_teg = delta_T_total * R_teg / (R_hsink + R_TIM_hot + R_teg + R_TIM_cold + R_csink)
```

散热器热阻过大会导致大部分温差降落在散热器而非 TEG 上，这是系统设计中最常见错误。

### 6.2 散热器与热界面材料

| 散热器类型 | 热阻范围 | 适用场景 |
|-----------|----------|----------|
| 被动翅片散热器 | 1-10 K/W | 工业管道表面 |
| 小型铝挤压散热 | 5-20 K/W | 可穿戴设备 |
| 热管 + 翅片 | 0.2-1 K/W | 高功率工业采集 |

IoT 通常无法主动散热（风扇功耗可能超过 TEG 产出），被动散热是唯一选择。散热器尺寸往往远大于 TEG 本体。

| TIM 材料 | 热导率 W/(m*K) | 特点 |
|----------|----------------|------|
| 导热硅脂 | 1-5 | 易涂覆、易老化 |
| 导热垫 | 1-6 | 可压缩、可返修 |
| 液态金属 | 20-80 | 热阻极低、导电风险 |
| 石墨片 | 5-15（面内） | 各向异性、薄 |

## 7. IoT 场景中的可用温差与功率估算

### 7.1 典型温差场景

| 温差源 | delta_T | 可用性 |
|--------|---------|--------|
| 人体皮肤 vs 环境 | 5-15K | 持续但小 |
| 工业蒸汽管道 | 70-170K | 大且稳定 |
| 工业热水管 | 25-55K | 中等稳定 |
| HVAC 送风管道 | 15-35K | 周期性 |
| 数据中心热通道 | 13-27K | 持续 |

### 7.2 人体热能采集特殊考量

可穿戴 IoT 面临独特挑战：皮肤温度非恒定、散热受限（不能装大型散热器）、热舒适度约束（可接受热流约 20 mW/cm2）、运动导致功率波动。

### 7.3 功率估算完整链路

工业蒸汽管道场景示例：

```
管道 150C / 环境 25C -> delta_T_total = 125K
热阻分压后 TEG 分得 delta_T_teg = 31.25K

V_oc = 200e-6 * 127 * 31.25 = 0.794V
P_max = 0.794^2 / (4*3) = 52.5mW
考虑变换器效率 80%: P_available = 42mW
```

### 7.4 功率密度与 IoT 功耗匹配

| 场景 | delta_T | 功率密度 mW/cm2 |
|------|---------|----------------|
| 人体可穿戴 | 5K | 0.01-0.05 |
| 人体可穿戴 | 10K | 0.05-0.2 |
| 暖气片表面 | 30K | 0.5-2 |
| 蒸汽管道 | 100K | 10-50 |

| IoT 节点类型 | 平均功耗 | 可行温差源 |
|-------------|----------|-----------|
| BLE 温度计 | 10-50 uW | 人体（勉强）、管道（充足） |
| LoRa 压力传感器 | 50-200 uW | 暖气、管道 |
| NB-IoT 燃气表 | 100-500 uW | 管道 |

## 8. 应用案例

### 8.1 体温供电智能手表

```
[皮肤] --[TEG 5-10K]-- [LTC3108] -- [超级电容] -- [BLE SoC]
```

实测数据：室内静坐约 10-30 uW，室外行走约 50-200 uW。剧烈运动时输出反而可能下降（冷端散热增强导致 delta_T 减小）。uW 级功率不足以持续驱动 BLE 通信，但配合能量缓冲可实现间歇传输。

### 8.2 工业管道无线监测

```
[管道] --[TEG 30-80K]-- [BQ25504] -- [Li-ion] -- [STM32L0 + 433MHz]
```

典型输出 5-50 mW，传感器采样间隔 1-10 秒，无线传输间隔 1-60 分钟。电池初始充电后可长期满充。这是目前温差发电 IoT 最成功的商业应用。

### 8.3 HVAC 智能温控节点

利用空调送风管道温差为分布式传感器供电。温差中等（15-35K）但具周期性，需 MPPT 适应变化，夜间 HVAC 关闭时依赖储能元件。

## 9. 核心挑战与前沿方向

### 9.1 小温差低效率

卡诺效率上限 eta_carnot = delta_T / T_hot。delta_T = 10K 时仅 3.3%，实际 TEG 效率约为卡诺效率的 10-20%，即 0.3-0.7%。这不是器件设计问题，而是热力学基本约束。

### 9.2 热短路与寄生热流

TEG 热导率较高时，大量热流直接"穿过" TEG 不做功。优化方向：增加 TEG 面积、使用低热导材料、引入热集中器。

### 9.3 柔性 TEG

可穿戴需要 TEG 贴合曲面。研究方向包括聚合物基板、丝网印刷热电材料、微纳结构热电腿、液态金属互连。目前柔性 TEG 功率密度仅为刚性 TEG 的 1/5-1/3。

### 9.4 新材料与系统级优化

高 ZT 追求：拓扑绝缘体、超晶格结构、半赫斯勒合金。系统级方向是热-电-信息三域联合优化：热域优化散热路径，电域 MPPT + 间歇工作 + 能量缓冲，信息域根据可用能量自适应调整采样率和传输频率。

## 总结

温差发电是 IoT 能量收集中技术成熟度较高的方向。核心优势在于温差源普遍性和输出稳定性；核心挑战同样明显：

1. 小温差下效率极低（< 1%），可穿戴仅能提供 uW 级功率
2. 热路设计与电路设计同等重要，散热器尺寸远大于 TEG 本体
3. 升压变换器极低输入电压下的效率和冷启动是关键瓶颈
4. 材料突破（高 ZT）是根本解决方案，但商用化进展缓慢

实际建议：优先评估场景是否存在 20K 以上稳定温差，若存在则温差发电可行；若仅 5-10K，需仔细计算功率预算并考虑混合能量源（温差 + 光伏）。
