# 三防漆在IoT户外设备中的防护应用

> **难度**：🟢 初级 | **领域**：PCB防护工艺 | **阅读时间**：约 18 分钟

## 引言

想象你住在一栋没有雨棚的房子里——风雨来时，墙面渗水、家具受潮、金属生锈。PCB板就是IoT设备的"房子"，而三防漆就是那层防雨涂层。涂上之后，水汽、盐雾、灰尘都进不来，电路板上的元器件就能安心工作很多年。

## 1. 什么是三防漆

### 1.1 定义与作用

三防漆(Conformal Coating)是一种涂覆在PCB表面的薄型保护膜，厚度通常在25-75um之间。所谓"三防"，指的是防潮湿、防盐雾、防霉菌，实际上它还防灰尘、防化学腐蚀和防漏电。

"Conformal"一词说明这层膜是"共形的"——它均匀地贴合在元器件和走线的轮廓上，像一层薄手套包住整个电路板。

### 1.2 为什么IoT设备尤其需要三防漆

IoT设备经常部署在户外或工业环境中，面临多重威胁：

- **湿度与凝露**：昼夜温差导致PCB表面凝露，水膜引发漏电或腐蚀
- **盐雾**：沿海地区的盐分会加速铜走线和焊点的电化学腐蚀
- **灰尘堆积**：灰尘吸湿后变成导电通道，导致走线间短路
- **化学暴露**：农业区的农药、工厂区的酸碱气体都会侵蚀PCB

室内消费电子通常不需要三防漆，但任何暴露在非受控环境中的IoT设备都应该考虑。

### 1.3 有无三防漆的对比

| 对比项 | 无三防漆 | 有三防漆 |
|--------|----------|----------|
| 潮湿环境(85%RH) | PCB表面水膜 -> 漏电 | 漆膜阻隔水汽，绝缘电阻保持 |
| 盐雾(沿海) | 铜走线3-6个月腐蚀开路 | 漆膜保护，寿命可达5年以上 |
| 灰尘+湿度 | 灰尘吸湿形成导电通道 | 漆膜覆盖走线，灰尘不直接接触 |
| 化学气体 | 焊点/走线被侵蚀 | 漆膜阻隔化学物质接触 |
| 机械磨损 | 走线易被刮伤 | 漆膜提供表面保护 |

## 2. 三防漆的类型与特性

### 2.1 五大类型对比

| 类型 | 代号 | 优点 | 缺点 | 典型厚度 | 适用场景 |
|------|------|------|------|----------|----------|
| 丙烯酸 | AR | 易返修(溶剂溶解)、快干 | 耐化学性差 | 25-75um | 原型验证、需要频繁返修 |
| 有机硅 | SR | 耐温范围宽(-55C到+200C)、柔韧 | 返修困难、附着力一般 | 50-125um | 高温环境、汽车电子 |
| 聚氨酯 | UR | 耐化学腐蚀优异、耐磨损 | 返修困难、固化慢 | 25-75um | 化工区、恶劣环境 |
| 环氧 | ER | 硬度高、耐化学性好 | 脆性大、难返修、应力大 | 25-75um | 一次性产品、不需要返修 |
| 派瑞林 | XY | 气相沉积超薄(5-50um)、均匀无针孔 | 成本高、需要真空设备 | 5-50um | 医疗植入、高可靠性 |

### 2.2 派瑞林的特殊工艺

派瑞林(Parylene)采用气相沉积工艺：固体原料在真空室中气化，裂解为单体，然后在室温PCB表面聚合形成薄膜。整个过程无溶剂、无液体流动，因此涂层极其均匀，即使是元器件底面的缝隙也能覆盖到。代价是必须使用专用真空设备，批量生产成本较高。

派瑞林有多个型号，IoT领域常用：

| 型号 | 介电常数 | 介电强度 | 特点 |
|------|----------|----------|------|
| N型 | 2.65 | 7000 V/mil | 最低介电常数，天线附近首选 |
| C型 | 3.15 | 5600 V/mil | 综合性能最好，最常用 |
| D型 | 2.84 | 5500 V/mil | 耐温性最好 |

## 3. 选型依据

### 3.1 关键选型因素

选三防漆不是"越贵越好"，而是根据实际工况匹配：

1. **工作温度范围**：高温环境优先SR，常温可选AR或UR
2. **化学暴露**：有酸碱蒸汽选UR或ER，无化学暴露选AR即可
3. **返修需求**：量产阶段是否需要现场维修？需要则选AR
4. **涂层厚度**：空间受限时选派瑞林实现超薄防护
5. **介电强度**：高压电路需要更高介电强度的涂层

### 3.2 选型决策流程

```
需要返修吗？
  |-- 是 --> 丙烯酸(AR)
  |-- 否 --> 工作温度 > 125C？
              |-- 是 --> 有机硅(SR)
              |-- 否 --> 有化学腐蚀？
                          |-- 是 --> 聚氨酯(UR)
                          |-- 否 --> 丙烯酸(AR)即可
```

## 4. 涂覆工艺

### 4.1 四种涂覆方法

**刷涂(Brush)**
- 方法：用毛刷蘸漆手工涂覆
- 优点：零设备成本，适合原型阶段
- 缺点：厚度不均匀，生产效率极低
- 适用：样机试制、小批量修补

**喷涂(Spray)**
- 方法：手持或半自动喷枪喷涂
- 优点：效率较高，设备投入低
- 缺点：过喷浪费，需遮蔽非涂覆区
- 适用：中小批量生产

**选择性涂覆(Selective Coating)**
- 方法：机器手臂带喷阀，按程序精确涂覆
- 优点：无需遮蔽、厚度均匀、一致性好
- 缺点：设备投资大(10-50万元)
- 适用：中大批量生产

**浸涂(Dip)**
- 方法：将PCB整体浸入漆液再提起
- 优点：覆盖率最好，包括元器件底面
- 缺点：所有非涂覆区必须遮蔽，漆液污染
- 适用：全板涂覆且无精密遮蔽需求的场景

### 4.2 遮蔽区域

以下区域在涂覆前必须遮蔽，否则会导致功能故障：

| 遮蔽区域 | 原因 |
|----------|------|
| 连接器触点 | 漆膜增加接触电阻，导致信号异常 |
| 测试点 | 影响ICT/FCT探针接触 |
| 开关/按键 | 漆膜阻碍机械动作 |
| 散热器表面 | 漆膜降低散热效率 |
| 天线区域 | 漆膜改变天线介电常数，影响射频性能 |
| 传感器感测面 | 漆膜阻挡气体/光线进入 |

### 4.3 遮蔽方法详解

遮蔽方法包括：高温胶带、可剥离遮蔽胶、专用遮蔽工装。

**高温胶带遮蔽**
- 适用：连接器、测试点等规则形状区域
- 使用方法：裁切后贴覆，涂覆后趁漆膜未全固化时撕除
- 注意：胶带边缘必须压紧，防止漆液渗入

**可剥离遮蔽胶**
- 适用：不规则形状的遮蔽区域
- 使用方法：点涂或刷涂在非涂覆区，固化后形成弹性遮蔽层
- 注意：必须在漆膜表干前撕除，否则与三防漆粘连

**专用遮蔽工装**
- 适用：大批量生产中固定位置的遮蔽
- 使用方法：硅胶或金属工装扣合在PCB上，遮蔽特定区域
- 优点：重复使用，一致性高
- 缺点：初始制作成本较高(500-3000元/套)

## 5. 固化与检验

### 5.1 固化方式

- **室温固化**：最简单，丙烯酸类20-30分钟表干，24小时全固化
- **UV固化**：紫外线照射下数十秒固化，适合流水线生产
- **热固化**：60-80C烘烤加速固化，适用于UR和ER类型

### 5.1.1 固化方式选型建议

| 固化方式 | 适用漆类型 | 固化时间 | 设备需求 | 注意事项 |
|----------|------------|----------|----------|----------|
| 室温 | AR | 24h全固化 | 无 | 环境湿度 < 60%RH |
| UV | UV型丙烯酸 | 30-60s | UV固化炉 | 阴影区需二次热固化 |
| 热固化 | UR, ER | 1-4h | 烘箱 | 温度不超过PCB耐受上限 |
| 湿气固化 | SR | 4-24h | 无 | 需要环境湿气参与 |

UV固化的特殊注意：UV光只能照射到暴露面，元器件底面和阴影区域无法被UV光覆盖。因此UV型三防漆通常设计为"双重固化"——UV光照射区域快速固化，阴影区域通过湿气或热完成二次固化。

### 5.2 检验方法

**UV荧光检验**：大多数三防漆添加了荧光剂，在UV灯(365nm)下发蓝白色光。未涂覆区域呈暗色，一目了然。

检验要点：
- 覆盖完整性：所有规定区域是否都有荧光
- 边缘覆盖：元器件引脚根部和边角是否涂到位
- 针孔/气泡：漆膜中是否有漏涂的针孔
- 桥接：不该涂覆的区域是否被漆膜桥接

**目视检查**：在正常光照下检查颜色均匀性、流挂、橘皮等外观缺陷。

### 5.3 IPC-CC-830标准

IPC-CC-830是三防漆的鉴定标准，规定了以下测试项目：

- 介电强度(耐压测试)
- 绝缘电阻(潮湿条件下)
- 附着力(划格法)
- 柔韧性
- 易燃性
- 热冲击
- 耐霉菌性

产品选型时应确认涂层材料通过IPC-CC-830鉴定。

## 6. IoT设备特殊考虑

### 6.1 天线性能影响

三防漆会在天线辐射面形成一层介质层，改变天线的等效介电常数，导致：
- 谐振频率偏移(通常偏低)
- 阻抗失配
- 辐射效率下降

应对措施：
- 在天线区域设计"禁涂区"(keep-out)
- 涂覆后重新校准天线匹配网络
- 选用介电常数低的三防漆(如派瑞林)

### 6.2 热影响

三防漆的导热系数远低于金属和PCB基材(通常0.1-0.3 W/mK)，涂覆在大功率器件表面会增加热阻。但对于IoT设备中常见的低功耗器件，热影响可忽略。

### 6.3 连接器密封与涂覆的选择

连接器防水有两种思路：
- **密封连接器**：选用IP67级连接器，本身防水，不需要涂覆
- **普通连接器+涂覆**：连接器根部涂覆，但触点区域必须遮蔽

实际项目中两者常结合使用：户外连接器选IP67密封型，内部板间连接器根部涂覆三防漆。

## 7. 返修流程

### 7.1 丙烯酸(AR)返修

丙烯酸可用溶剂(如异丙醇、专用剥离液)溶解去除：

```
1. 用棉签蘸取溶剂
2. 反复擦拭需返修区域的漆膜
3. 漆膜溶解后露出焊盘
4. 用烙铁更换元器件
5. 清洁后重新涂覆三防漆
```

### 7.2 其他类型返修

有机硅、聚氨酯、环氧漆膜无法用溶剂简单去除：
- **机械去除**：用手术刀或专用工具小心刮除漆膜
- **热切割**：用烙铁直接烫穿漆膜焊接(有损坏焊盘风险)
- **激光去膜**：精确去除小面积漆膜(需专用设备)

返修难度：AR < SR < UR < ER，这也是AR在需要维修的场景中广泛使用的原因。

## 8. 成本分析

### 8.1 成本构成

| 成本项目 | 刷涂 | 手工喷涂 | 选择性涂覆 | 派瑞林 |
|----------|------|----------|------------|--------|
| 材料费 | 低 | 中 | 中 | 高 |
| 人工费 | 高 | 中 | 低 | 低 |
| 设备折旧 | 无 | 低 | 高 | 很高 |
| 遮蔽成本 | 低 | 高 | 无 | 无 |
| 返修成本 | 低 | 低 | 中 | 高 |

### 8.2 典型成本参考

以100mm x 80mm的IoT传感器PCB为例：

- 手工喷涂丙烯酸：每片约2-5元(含材料+人工)
- 选择性涂覆聚氨酯：设备摊销后每片约1-3元
- 派瑞林沉积：每片约15-40元(取决于批量)

量产1万片以上时，选择性涂覆的综合成本最优。

### 8.3 隐性成本

除了直接的材料和人工成本，还需考虑以下隐性成本：

- **不良率成本**：遮蔽不当导致返工的比例(手工喷涂约3-8%)
- **库存成本**：三防漆开封后保质期有限(丙烯酸约6个月，聚氨酯更短)
- **环境成本**：VOC排放处理(丙烯酸和聚氨酯含有机溶剂)
- **培训成本**：操作人员需要培训才能保证涂覆一致性

隐性成本通常占直接成本的15-30%，在预算时应纳入考量。

## 9. 常见失效模式

### 9.1 漆膜失效类型

| 失效模式 | 根因 | 预防措施 |
|----------|------|----------|
| 漆膜脱层(剥离) | PCB表面清洁不良，油污/助焊剂残留 | 涂覆前严格清洗，用等离子处理 |
| 漆膜开裂 | 涂层太厚或热冲击过猛 | 控制厚度，选用柔性好的涂层 |
| 边缘覆盖不足 | 漆液在元器件边缘回缩 | 检查漆液粘度，多次薄涂 |
| 白化(起雾) | 高湿环境下涂覆，水汽被包裹 | 控制环境湿度低于60%RH |
| 气泡 | 涂覆速度过快或PCB有盲孔 | 减压脱泡，控制浸涂速度 |

### 9.2 关键预防

最重要的预防措施是**涂覆前的清洁**。PCB上的助焊剂残留、指纹油脂、灰尘是导致脱层的首要原因。推荐流程：

```
焊接完成 --> 水洗或溶剂清洗 --> 烘干 --> 等离子处理(可选) --> 涂覆
```

## 总结

三防漆是IoT户外设备防护的"第一道防线"，它能有效阻挡水汽、盐雾、灰尘和化学物质对PCB的侵蚀。选型时根据环境条件、返修需求和成本综合考量。涂覆工艺从手工刷涂到选择性涂覆，适合不同批量阶段。IoT设备需特别注意天线区域的禁涂设计和连接器的密封策略。无论选择哪种漆，涂覆前的清洁和涂覆后的UV荧光检验都是质量保证的关键环节。
