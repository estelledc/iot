# 涡流传感器接近检测与金属识别
> **难度**：🟡 中级 | **领域**：接近传感技术 | **阅读时间**：约 20 分钟

## 引言

想象你在机场安检门下走过，门框里没有任何东西碰到你，但它知道你身上有金属。这个"隔空感知"的魔法，就是涡流效应。安检门的线圈发出交变磁场，你身上的钥匙或皮带扣感应出涡流，涡流反过来改变了线圈本身的电气特性，于是系统就"看到"了金属。IoT 设备里的涡流传感器用的是完全相同的原理，只是做得更小、更精准，不仅能检测"有没有金属"，还能判断"是什么金属"和"离多远"。

## 1 涡流效应的物理基础

### 1.1 电磁感应与涡流

当交流电流通过线圈时，线圈周围产生交变磁场。如果附近有导电体（金属目标），交变磁场会在导体表面感应出环形电流——这就是涡流（eddy current）。涡流的名字来源于它在导体内形成一圈圈类似水涡的闭合回路。

涡流的关键特性：

- 只在导电材料中产生（金属、石墨等）
- 集中在导体表面（趋肤效应）
- 频率越高，趋肤深度越浅
- 涡流强度与目标电导率和磁场强度成正比

### 1.2 涡流对线圈的反作用

涡流本身也是一个电流，它会产生自己的磁场。根据楞次定律，涡流磁场总是阻碍原来的磁场变化。这个反作用磁场叠加到激励磁场上，等效结果就是线圈的有效阻抗发生了变化。

具体变化规律：

- 导电目标靠近时，涡流增大，线圈等效电阻增大
- 磁导率高的铁磁目标会额外增加线圈电感
- 非铁磁导电目标（铝、铜）会减小线圈电感
- 目标距离越近，阻抗变化越大

这就是涡流传感的核心信号：通过测量线圈阻抗的变化来推断目标的距离和材料属性。

### 1.3 趋肤深度

交流磁场在导体中衰减的程度用趋肤深度描述：

```
delta = sqrt(2 / (omega * mu * sigma))
```

其中 omega 是角频率，mu 是磁导率，sigma 是电导率。

实际影响：

- 100kHz 下铝的趋肤深度约 0.3mm，铜约 0.2mm
- 高频检测只"看到"目标表面
- 低频可以穿透更厚的导体层
- 传感器设计时需要根据目标厚度选择频率

## 2 测量模式：接近检测与金属识别

### 2.1 电感式接近检测

最常见的模式：只关心"有没有金属目标"以及"离多远"。

工作方式：

- 线圈作为 LC 振荡器的一部分
- 金属目标靠近时，振荡幅度下降（阻尼增大）
- 振荡幅度低于阈值时输出开关信号
- 典型检测距离：线圈直径的 0.1-1 倍

这就是工业中"电感式接近开关"的工作原理，产量巨大、价格低廉。

### 2.2 阻抗分析：金属类型识别

更高级的模式：不仅检测存在，还识别金属种类。

原理：不同金属的电导率和磁导率不同，对线圈阻抗的影响不同：

| 金属类型 | 电导率影响 | 磁导率影响 | 净效应 |
|----------|-----------|-----------|--------|
| 钢（铁磁） | 电阻增大 | 电感增大 | 阻抗增大（幅值和相位都变） |
| 铝（非铁磁） | 电阻增大 | 电感减小 | 阻抗增大但相位相反 |
| 铜（非铁磁） | 电阻增大 | 电感减小更多 | 与铝类似但幅度更大 |

通过同时测量阻抗的幅值和相位（或等价的电阻和电感分量），可以区分铁磁和非铁磁金属，甚至在同类金属中粗略区分不同合金。

### 2.3 多频涡流检测

用多个激励频率分别测量，每个频率对不同深度和不同材料特性的灵敏度不同。多频数据组合后可以更精确地识别材料或检测多层结构。

## 3 传感器构造与电路

### 3.1 传感器探头

典型涡流传感器探头包含：

- 线圈：绕制在铁氧体磁芯上，集中磁场、提高灵敏度
- 线圈直径：从 3mm 到 50mm 不等，决定检测范围
- 保护壳：不锈钢外壳，保护线圈
- 电缆：屏蔽线缆连接到电路

### 3.2 激励与检测电路

常见电路架构：

**振荡器 + 解调器方案：**

```
         +---[LC振荡器]---+
         |                |
    [线圈]           [振幅检测]
         |                |
         +---[比较器]---[输出]
```

线圈参与 LC 振荡，目标靠近改变阻尼，振幅变化经峰值检波后输出。

**电感-数字转换器方案：**

```
    [线圈]---[LDC芯片]---[SPI/I2C]---[MCU]
```

TI 的 LDC 系列直接将线圈等效电感转换为数字值，MCU 读取后做距离或材料判断。

### 3.3 激励频率选择

| 频率范围 | 适用场景 |
|---------|---------|
| 100kHz - 1MHz | 大距离接近检测，铁磁目标 |
| 1MHz - 5MHz | 通用接近检测和金属识别 |
| 5MHz - 10MHz | 高分辨力小距离检测，薄金属 |

频率越高：

- 趋肤深度越浅，对表面特性越敏感
- 分辨力提高，但检测距离缩短
- PCB 走线和寄生参数影响增大

## 4 检测范围与目标材料

### 4.1 检测距离

检测距离主要取决于线圈直径：

- 典型范围：线圈直径的 0.1 到 1 倍
- 标称检测距离（Sn）：厂家给出的额定值
- 有效检测距离（Su）：实际可用值，受温度和电压影响
- 可靠检测距离（Sa）：Su 的 0-81%，保证可靠触发

大线圈 = 远距离但低空间分辨力，小线圈 = 近距离但精确定位。

### 4.2 目标材料修正系数

电感式接近开关的标准目标是一块 1mm 厚的正方形低碳钢，边长等于线圈直径。其他材料需要修正：

| 材料 | 修正系数 |
|------|---------|
| 低碳钢 | 1.0 |
| 不锈钢（铁磁类） | 0.85 |
| 铝 | 0.4 |
| 黄铜 | 0.35 |
| 铜 | 0.3 |

含义：铝目标的有效检测距离只有钢目标的 40%。这在选型时非常关键——如果你的 IoT 设备要检测铝制外壳的零件，必须选更大检测距离的传感器。

### 4.3 目标尺寸影响

目标尺寸小于线圈直径时，有效检测距离下降。一般要求目标面积至少等于线圈端面面积才能达到额定检测距离。

## 5 IoT应用场景

### 5.1 位置检测

线性或旋转位置的非接触测量：

- 气缸活塞位置：铝活塞在气缸内，外置线圈检测到位信号
- 旋转角度：齿轮 + 涡流线圈，齿/槽交替触发计数
- 阀门开闭：金属阀板靠近/远离传感器

### 5.2 转速测量

齿轮测速的典型方案：

```
[旋转齿轮] ---齿/槽交替---> [涡流传感器] ---脉冲序列---> [MCU计时器]
                                                            |
                                                    频率 = 转速 * 齿数
```

优势：不怕油污、灰尘、水，比光电传感器更可靠。

### 5.3 金属分拣

利用阻抗分析能力：

- 废金属回收分拣线：识别铁/铝/铜
- 生产线零件确认：区分正确材料和错误材料
- 硬币识别：自动售货机判别硬币面值

### 5.4 涂层/镀层厚度测量

涡流对基体和涂层电导率差异敏感：

- 钢铁上的油漆层厚度
- 铝材上的阳极氧化层
- 线路板上的铜箔厚度

## 6 与其他接近传感器的对比

| 特性 | 涡流（电感式） | 电容式 | 光电式 | 超声波 |
|------|-------------|--------|--------|--------|
| 检测对象 | 金属 | 所有材料 | 所有材料 | 所有材料 |
| 检测距离 | 中（mm级） | 短 | 长（m级） | 长（m级） |
| 环境耐受 | 极好（油/水/尘） | 一般 | 差（怕脏） | 一般 |
| 响应速度 | 快（kHz） | 快 | 极快（MHz） | 慢（ms） |
| 成本 | 低 | 低 | 中 | 高 |

涡流传感器在"只需要检测金属"且"环境恶劣"的场景下几乎没有对手。

## 7 常用传感器IC

### 7.1 TI LDC 系列

| 型号 | 通道数 | 分辨率 | 接口 | 特点 |
|------|--------|--------|------|------|
| LDC1000 | 1 | 24bit | SPI | 经典入门款 |
| LDC1612 | 2 | 28bit | I2C | 高分辨率双通道 |
| LDC1614 | 4 | 28bit | I2C | 四通道 |
| LDC3114 | 4 | - | I2C | 按钮/滑条专用 |

LDC1612 是 IoT 设计中最常用的选择，28bit 分辨率可以检测亚微米级的距离变化。

### 7.2 其他方案

- FDC2214：TI 电容数字转换器，也能测电感
- TDC1000：TI 超声模拟前端，配合不同探头
- 分立方案：振荡器 + MCU 测频

## 8 实战示例：IoT旋转位置传感器

### 8.1 需求

工业阀门需要非接触式角度测量：

- 测量范围：0-360度
- 分辨力：优于 1 度
- 环境：户外，有灰尘和湿气
- 输出：无线传输到监控平台

### 8.2 方案设计

```
[金属目标齿轮] ---涡流---> [PCB线圈] ---[LDC1612]---[STM32L0]---[LoRa]---> [网关]
```

关键设计点：

- 目标齿轮：铝制，36 齿，每齿对应 10 度
- PCB 线圈：双层外径 20mm，激励频率 3MHz
- LDC1612：28bit 分辨率，I2C 接口
- STM32L0：低功耗 MCU，从 LDC 读取数据并计算角度
- LoRa：长距离低功耗无线

### 8.3 角度计算

```c
// 从 LDC1612 读取线圈电感值
uint32_t raw = LDC1612_ReadChannel(0);

// 基准值（无目标时的读数）
uint32_t baseline = LDC1612_ReadChannel(1);

// 计算归一化变化量
float delta = (float)(raw - baseline) / (float)baseline;

// 齿轮通过时 delta 周期性变化
// 通过计数过零点确定粗角度
// 通过插值确定精角度
float angle = calculate_angle_from_delta(delta);
```

## 9 设计要点

### 9.1 线圈设计

- 匝数：影响电感值和品质因数 Q，一般 20-50 匝
- 直径：决定检测距离和空间分辨力
- 形状：圆形最常见，矩形用于特定目标
- PCB 线圈：成本低，一致性好，适合批量

### 9.2 频率选择

- 高电导率目标（铜、铝）：可用较高频率
- 铁磁目标：中低频率效果更好
- 厚涂层目标：低频穿透力强

### 9.3 温度补偿

线圈电阻和目标电导率都随温度变化：

- 使用参考线圈做差分测量
- MCU 内置温度传感器做软件补偿
- LDC1612 的双通道天然支持差分

### 9.4 EMI 防护

传感器线圈既是发射天线也是接收天线：

- 远离强电磁干扰源（电机、变频器）
- 使用屏蔽电缆连接探头
- PCB 线圈下方铺地平面（但保持距离避免降低 Q 值）

## 10 恶劣环境下的优势

涡流传感器在工业 IoT 中广受欢迎的核心原因：

- 无机械磨损：没有触点、没有弹簧
- 穿透非金属障碍：可以透过塑料外壳、油漆、玻璃检测
- 免疫污染：灰尘、油污、水都不影响（因为这些都不导电）
- 长寿命：没有运动部件，MTBF 极高
- 本质安全：低功耗激励，不会产生火花

## 总结

涡流传感器是工业 IoT 中最可靠的接近检测手段之一。它的工作原理——交流磁场感应涡流、涡流反作用于线圈阻抗——虽然听起来抽象，但本质上就是"电磁隔空感知"。只要目标是金属，不管环境多脏多湿，涡流传感器都能稳定工作。

从简单的"有没有金属"到高级的"是什么金属、离多远"，涡流传感器的信息提取深度取决于电路和算法。TI LDC 系列芯片让 IoT 开发者无需设计模拟前端就能用上涡流检测，大幅降低了门槛。

关键记住三点：(1) 检测距离取决于线圈直径和目标材料；(2) 金属类型识别靠阻抗的幅相分析；(3) 恶劣环境下涡流传感器几乎是唯一可靠的选择。
