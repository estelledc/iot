# 3D打印传感器外壳设计与快速原型
> **难度**：🟢 初级 | **领域**：快速原型制造 | **阅读时间**：约 18 分钟

## 引言

想象你烤了一个蛋糕，发现奶油太甜、裱花歪了。如果这个蛋糕是3D打印出来的外壳，你只需要改一下设计文件，半小时后就能拿到改进版 -- 不用重新买模具、不用等工厂排产。这就是3D打印在IoT原型开发中的核心价值：快速迭代、低成本验证、从想法到实物只需几小时。

IoT传感器项目经常需要定制外壳 -- 不同传感器的尺寸各异，安装方式千奇百怪，市售外壳很难完美匹配。3D打印让你从"将就着用"变成"量身定做"，而且全程在自己办公桌上完成。

## 1. 3D打印技术概述

### 1.1 为什么IoT项目需要3D打印

传统外壳开发流程：

```
设计 -> 开模具 -> 试产 -> 修改模具 -> 再试产 -> 量产
 1周     4-8周     2周      1-2周       2周      持续
 费用:  模具费$5000-$50000
```

3D打印外壳开发流程：

```
设计 -> 打印 -> 测试 -> 修改 -> 再打印 -> 确认设计
 1天    2-8小时  1小时   30分钟   2-8小时    1天
 费用:  材料费$1-$10
```

对比关键指标：

| 维度 | 传统开模 | 3D打印 |
|------|----------|--------|
| 首次样品时间 | 4-8周 | 2-8小时 |
| 迭代周期 | 1-2周 | 数小时 |
| 首次成本 | $5000+ | $1-10 |
| 适合产量 | 1000+ | 1-100 |
| 材料选择 | 极多 | 有限但够用 |
| 表面质量 | 高 | 中等(取决于工艺) |

### 1.2 打印技术对比

IoT外壳常用的三种3D打印技术：

**FDM/FFF(熔融沉积)**

```
原理: 塑料丝材加热熔化，逐层挤出堆积
     _______________
    /               \     <-- 每层是挤出的塑料线条
   /  =============  \    <-- 填充图案(网格/蜂窝)
  /___________________\

优点: 最低成本($200起)、材料种类多、操作简单
缺点: 层纹明显、悬空需要支撑、精度一般(0.1-0.3mm)
适合: 结构性外壳、安装支架、大尺寸部件
```

**SLA/DLP(光固化)**

```
原理: 紫外光逐层照射液态树脂，使其固化
     _______________
    |               |     <-- 每层是整面固化，非线条
    |               |     <-- 表面光滑如注塑件
    |_______________|

优点: 表面光滑、细节精细(0.025-0.05mm)、适合小件
缺点: 树脂有气味、后处理需清洗、材料偏脆
适合: 精密传感器支架、小型外壳、外观验证
```

**SLS(选择性激光烧结)**

- 激光烧结尼龙粉末，逐层成型
- 优点: 强度高(尼龙)、无需支撑、可做复杂内腔
- 缺点: 设备昂贵(服务打印)、表面粗糙(粉末感)、成本高
- 适合: 功能验证件、需要强度的受力部件

技术选型速查：FDM最低成本、SLA最精细、SLS最强，大尺寸选FDM，精密卡扣选SLA。

## 2. 材料选择

### 2.1 常用FDM材料特性

| 材料 | 打印难度 | 强度 | 耐温 | 耐候 | 成本 | 适用场景 |
|------|----------|------|------|------|------|----------|
| PLA | 极易 | 中 | 低(55度变形) | 差(UV老化) | 最低 | 室内原型、验证外形 |
| PETG | 易 | 中高 | 中(80度) | 中 | 低 | 通用外壳、电气绝缘 |
| ABS | 中(翘曲) | 高 | 高(100度) | 中 | 低 | 汽车内饰、受热环境 |
| ASA | 中 | 高 | 高(95度) | 好(UV稳定) | 中 | 户外设备、长期日照 |
| Nylon | 难(吸湿) | 极高 | 高(120度) | 好 | 高 | 齿轮、铰链、受力件 |
| TPU | 中(柔性) | 柔性 | 中 | 中 | 中 | 减震垫、密封圈、缓冲套 |
| PC | 难 | 极高 | 极高(115度) | 中 | 高 | 高温/高冲击场景 |

### 2.2 材料选择与IoT适用性

决策思路：室内无高温选PLA，受热选PETG/ABS，户外选ASA，受力选Nylon/PC。

| IoT场景 | 推荐材料 | 原因 |
|----------|----------|------|
| 室内温湿度传感器 | PLA | 室内无UV、无高温、最便宜 |
| 户外气象站 | ASA | 抗UV、耐温、耐候 |
| 工业设备监测 | PETG/ABS | 耐温、电气绝缘 |
| 电池仓/充电座 | PETG | 耐温、韧性好 |
| 传感器安装支架 | Nylon/PETG | 承重、耐冲击 |
| 防水外壳原型 | PETG+后处理 | 涂环氧密封 |

## 3. 外壳设计要点

### 3.1 基本设计参数

3D打印外壳与注塑件的设计规则不同，无需考虑脱模斜度，但有其他约束：

- **壁厚**：FDM最小1.2mm(推荐2mm)，SLA最小0.8mm(推荐1.5mm)；过薄强度不足，过厚内应力大
- **圆角**：内圆角最小半径1mm(减少应力集中)，底角建议加倒角防止翘边
- **层纹方向**：沿层纹方向抗拉，垂直层纹方向容易分层，受力方向应沿层纹

### 3.2 3D打印特有的设计技巧

**无需脱模斜度**：3D打印完全垂直的壁面也没问题，内腔和卡扣可以做成完美直角。

**悬空与支撑**：超过45度的悬空需要支撑材料，设计时应尽量减少悬空。

**公差与配合**：FDM典型公差0.2-0.5mm，SLA典型0.1-0.2mm，两个配合件之间留0.3-0.5mm间隙。

### 3.3 传感器安装特征设计

**PCB安装柱(Standoff)**：

```
       ___
      |   |  <-- 顶部导向锥(方便PCB对准)
      |   |  <-- 柱体直径: 螺丝孔+1mm
      | O |  <-- 螺丝孔(热熔螺母或自攻螺丝)
      |___|
      /   \  <-- 底部加强筋
     /     \
```

- 柱体直径：螺丝直径+2mm(最小)
- 高度：PCB底部到外壳底部+0.5mm间隙
- 底部加三角加强筋增加稳定性

**传感器开窗**：

```
+---------------------------+
|                           |
|    +---+         +---+   |
|    |   |  传感器  |   |   |  <-- 开窗位置精确对准传感器
|    | O |  开窗口  | O |   |  <-- O = 定位柱
|    +---+         +---+   |
|                           |
+---------------------------+
```

- 光学传感器：透明窗口(SLA透明树脂或预留亚克力板安装位)
- 气体传感器：大量通风孔(蜂窝状排列)
- 温湿度传感器：百叶窗式通风(防直射雨滴)

**电缆通道**：

```
侧面视图:
    |        |
    |   __   |  <-- 顶部卡扣(电缆压紧)
    |  /  \  |
    | |    | |  <-- 电缆通道(直径比电缆大0.5mm)
    |  \__/  |
    |        |
```

- 通道内壁光滑，避免损伤电缆绝缘层
- 底部开口方便电缆放入
- 顶部卡扣防止电缆滑出

### 3.4 快拆与紧固设计

**卡扣(Snap-fit)**：

```c
// 卡扣设计参数(参考值)
snap_thickness = 1.0;   // 卡扣臂厚度(mm)
snap_length    = 8.0;   // 卡扣臂长度(mm)
snap_deflection = 0.8;  // 卡扣偏移量(mm), 需小于材料允许变形
insert_depth   = 1.5;   // 插入深度(mm)
```

- 悬臂式卡扣最简单：一个悬臂+末端的钩
- 卡扣臂长度至少8倍厚度(否则应力过大)
- PLA卡扣容易断裂，PETG/ABS/Nylon更适合
- 卡扣配合倒角便于对准和插入

**螺丝柱**：
- 热熔螺母：用烙铁将铜螺母压入塑料柱(最牢固)
- 自攻螺丝：直接拧入塑料孔(适合原型，反复拆卸会滑丝)
- 飞机孔：上盖孔径略大，下盖攻丝孔

## 4. 防水与耐候处理

### 4.1 密封设计

3D打印外壳要实现防水，比注塑件更难(因为层纹间的微间隙)：

**密封圈槽设计**：

```
截面图:
    |            |
    | +------+  |
    | | O圈 |  |  <-- O型圈槽(深度=线径的75%, 宽度=线径的120%)
    | +------+  |
    |____________|  <-- 配合面(平整、无层纹)
```

- 配合面必须平整：打印时配合面朝向热床
- O型圈规格选择：常用1.5mm或2mm线径
- 槽深 = 线径 x 0.75(压缩25%确保密封)
- 槽宽 = 线径 x 1.2(容纳膨胀)

**迷宫密封**：

对于不需要完全防水但需防溅的场景：

```
截面(放大):
上盖: ||  ||
      ||  ||      <-- 交错凹凸
下盖: ||  ||      <-- 多道弯折路径
```

- 多道弯折延长水路
- 不需要密封圈
- 适合IP54以下防护

### 4.2 后处理方法

| 方法 | 材料 | 效果 | 操作 |
|------|------|------|------|
| 环氧涂层 | 所有 | 封闭层纹、防水 | 刷涂/浸渍2-3层 |
| 丙酮蒸气平滑 | ABS | 表面光滑、层纹消失 | 密闭容器中蒸熏10-20分钟 |
| 底漆+喷漆 | 所有 | 美观、部分防水 | 喷涂2-3层 |
| 氰基丙烯酸酯 | 所有 | 快速封孔 | 喷涂后迅速固化 |
| 硅胶填充 | 所有 | 接缝密封 | 沿缝隙涂布 |

### 4.3 户外部署注意事项

PLA户外使用的问题：

- UV照射导致变脆(3-6个月明显劣化)
- 温度超过55度开始软化(暴晒车内可达70度)
- 吸湿后层间结合力下降

户外建议：

- 材料选ASA或PETG(加UV涂层)
- 关键结构加厚20%
- 避免直射：加装遮阳板或百叶窗
- 定期检查外壳完整性

## 5. CAD工具选择

### 5.1 工具对比

| 工具 | 类型 | 成本 | 上手难度 | 适合 |
|------|------|------|----------|------|
| Fusion 360 | 参数化建模 | 免费(个人版) | 中 | 大多数IoT外壳设计 |
| FreeCAD | 参数化建模 | 免费开源 | 中高 | 开源爱好者 |
| TinkerCAD | 浏览器拖拽 | 免费 | 极低 | 完全新手、简单外壳 |
| OpenSCAD | 代码建模 | 免费开源 | 中(需编程) | 参数化批量设计 |
| SolidWorks | 参数化建模 | 付费 | 中 | 专业机械设计 |

### 5.2 OpenSCAD代码示例

OpenSCAD用代码描述3D模型，适合参数化设计：

```openscad
// 温湿度传感器外壳参数
wall = 2.0;         // 壁厚
pcb_x = 35;         // PCB长度
pcb_y = 25;         // PCB宽度
pcb_h = 8;          // PCB高度(含元件)
standoff_h = 3;     // 安装柱高度
standoff_d = 6;     // 安装柱直径
screw_d = 2.5;      // 螺丝孔直径

// 外壳主体
difference() {
    // 外壁
    cube([pcb_x + 2*wall, pcb_y + 2*wall,
          pcb_h + standoff_h + wall]);
    // 内腔
    translate([wall, wall, wall])
        cube([pcb_x, pcb_y, pcb_h + standoff_h]);
    // 传感器通风孔(底部)
    for (i = [0:3])
        translate([wall + 5 + i*7, -1, wall])
            cube([4, pcb_y + 2*wall + 2, 3]);
}

// PCB安装柱(四个角)
for (x = [wall + 3, wall + pcb_x - 3])
    for (y = [wall + 3, wall + pcb_y - 3])
        translate([x, y, wall]) {
            difference() {
                cylinder(d=standoff_d, h=standoff_h, $fn=24);
                cylinder(d=screw_d, h=standoff_h+1, $fn=24);
            }
        }
```

修改参数即可生成不同尺寸的外壳，非常适合IoT项目中多种传感器的快速适配。

## 6. 实战案例：户外温湿度传感器

### 6.1 需求分析

设计一个户外温湿度传感器的外壳：

- 传感器：BME280(温湿度气压)
- MCU：ESP32-C3(低功耗WiFi)
- 供电：18650锂电池
- 安装：墙壁安装或悬挂
- 防护：防雨、通风(温度准确)、防晒

### 6.2 百叶窗式设计(Stevenson Screen原理)

借鉴气象站的百叶窗(Stevenson Screen)设计：

```
侧面截面:
    /===========\        <-- 顶盖(遮雨)
   / ventilation \       <-- 通风缝隙(空气流通)
  |   +-------+  |      <-- 传感器区域
  |   |BME280 |  |      <-- 传感器居中，远离壁面
  |   +-------+  |      <-- 通风缝隙(空气流通)
   \ ventilation /       <-- 底部通风
    \===========/        <-- 底盖(电池仓)
```

关键设计特征：

- 双层壁：外层遮雨、内层通风
- 顶盖有15度倾斜(排水)和2cm檐口(遮雨)
- 通风缝宽度5mm(空气流通但雨水不能直入)
- 传感器居中放置(减少壁面热辐射影响)
- 底部电池仓独立(方便更换)

### 6.3 打印设置

| 参数 | 设置 | 原因 |
|------|------|------|
| 材料 | ASA | 户外抗UV |
| 层高 | 0.2mm | 强度与速度平衡 |
| 壁厚 | 3层(约1.5mm) | 防水密闭 |
| 填充 | 20%网格 | 节省材料、强度够 |
| 支撑 | 是(顶盖悬空) | 仅顶盖需要 |
| 打印时间 | 约4-5小时 | 全部部件 |

### 6.4 组装与密封

- 上盖与主体：卡扣+密封圈(1.5mm线径O型圈)
- 电缆出口：PG7防水电缆接头
- 电池仓盖：滑动卡扣+硅胶条密封
- 壁挂孔：背部2个钥匙孔(挂墙安装)

## 7. 从原型到量产的过渡

### 7.1 原型与量产的设计差异

3D打印外壳和注塑量产外壳在设计上有本质区别：

| 设计特征 | 3D打印原型 | 注塑量产 |
|----------|-----------|----------|
| 壁厚 | 可以不均匀 | 必须均匀(2-3mm) |
| 脱模斜度 | 不需要 | 必须1-2度 |
| 倒扣 | 随便做 | 需要侧向抽芯(增加模具成本) |
| 卡扣 | 悬臂式即可 | 需要考虑注塑方向 |
| 公差 | 0.3-0.5mm | 0.05-0.1mm |
| 最小孔径 | 无限制 | 需要型芯(太深会断) |
| 表面纹理 | 层纹 | 可以做皮纹 |

### 7.2 过渡建议

从3D打印原型过渡到注塑量产的步骤：

1. 用3D打印验证功能和装配(1-3个月)
2. 确认外形和结构后，按注塑规则重新设计(DFM审查)
3. 3D打印验证注塑版设计(检查脱模斜度和壁厚)
4. 开模前用SLA打印1:1模型确认所有配合
5. 开模、试产、量产

关键原则：**3D打印是验证手段，不是最终产品**。设计时就要考虑量产可行性，避免原型和量产两套完全不同的设计。

### 7.3 低量产量产替代方案

如果产量不大(100-1000个)，不必急于开注塑模。硅胶复模(真空注型)适合10-500件、单件$5-15且表面质量高；钣金/铝型材适合10-1000件；注塑适合1000件以上。

## 总结

3D打印在IoT传感器外壳开发中扮演"快速验证器"的角色：

1. **技术选择**：FDM最通用、SLA最精细、SLS最强，按需求选
2. **材料选择**：室内PLA、户外ASA、通用PETG、受力Nylon
3. **设计要点**：壁厚2mm、通风孔对气体/温湿度传感器很重要、卡扣选对材料
4. **防水处理**：O型圈槽+后处理(环氧/丙酮蒸气)实现基本防护
5. **工具选择**：Fusion 360最平衡、OpenSCAD适合参数化、TinkerCAD最易上手
6. **量产过渡**：原型阶段就要考虑DFM，3D打印验证后按注塑规则重新设计

记住：3D打印让你一天迭代三次外壳设计，但不要在原型阶段就追求完美 -- 先能用，再好用，最后好看。
