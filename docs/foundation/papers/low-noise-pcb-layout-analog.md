# 低噪声模拟信号PCB布局设计规则
> **难度**：🔴 高级 | **领域**：模拟PCB设计 | **阅读时间**：约 22 分钟

## 引言

想象你在嘈杂工厂里接电话：背景噪音(数字开关噪声)不断干扰通话(模拟信号)，你需要隔音房间(布局隔离)和降噪耳机(滤波)才能听清。PCB上模拟信号面临相同处境 -- 数字电路的开关噪声、电源纹波和外部EMI都在污染微弱模拟信号。对于16位ADC(分辨率0.1mV级)，几毫伏耦合噪声就能让最低几位数据完全无效。低噪声模拟PCB布局就是在物理层面构建"隔音房间"，从源头抑制噪声耦合。

## 1. PCB上的噪声源

### 1.1 噪声分类

| 噪声类型 | 来源 | 频率范围 | 幅度 |
|----------|------|----------|------|
| 热噪声 | 电阻载流子热运动 | 宽带(白噪声) | 极低(uV级) |
| 开关噪声 | 数字翻转di/dt | MHz-GHz | 较高(mV-V级) |
| 电源纹波 | 开关电源斩波 | 开关频率及谐波 | 中等(mV级) |
| 地弹 | 共享地路径di/dt压降 | 与数字同步 | 中高(mV级) |
| EMI接收 | 外部射频场耦合 | MHz-GHz | 不确定 |

### 1.2 耦合机制

**传导耦合**：通过共享导体(地线、电源线)直接传导。最常见也最易被忽视。

**电容性耦合**：走线间寄生电容，高频噪声通过位移电流耦合。越近、平行越长耦合越强。

**电感性耦合**：环路间互感，di/dt在另一环路感应电压。大电流环路对高阻抗模拟输入影响最大。

**辐射耦合**：远场电磁波被PCB走线接收。>30MHz时显著。

### 1.3 噪声对ADC的影响

16位ADC、3.3V参考：LSB = 50.4uV，1mV噪声 = 约20 LSB抖动。10mV噪声让低6位无效，ENOB降到10-12位。

## 2. 模拟地与数字地

### 2.1 核心争议：分地还是统一地平面

**分地派**：模拟地和数字地分开，只在一点(ADC下方)连接。

**统一地派**：统一地平面，通过物理分区(而非分割)隔离，避免返回路径绕行和辐射。

### 2.2 何时分割地平面

| 条件 | 建议 | 原因 |
|------|------|------|
| 含ADC/DAC混合信号板 | 通常不分割 | 分割致返回路径绕行 |
| 大功率数字+敏感模拟 | 可分割 | 数字地电流太大 |
| 高速数字(>100MHz)+模拟 | 不分割 | 分割引起严重EMI |

### 2.3 统一地平面 + 物理分区

业界推荐方案：

1. 完整统一地平面(不分割)
2. 布局严格分区：模拟区、数字区、电源区
3. 信号不跨越分区边界
4. ADC/DAC跨在模拟区和数字区边界上
5. 混合信号IC的AGND和DGND在IC下方连接

**关键原则**：信号不跨越地平面分割线。地平面有缝时返回电流绕行，形成大环路增加辐射和串扰。

### 2.4 星形地

所有地连接汇聚到一点(电源地输入点)。仅适用于纯低频模拟(如音频前置放大)，高频电路连线长阻抗高不适用。

## 3. 元件布局策略

### 3.1 分区原则

```
+------------------------------------------+
| 电源区       | 数字区      | 模拟区      |
| (DCDC/LDO)  | (MCU/FPGA) | (ADC/OPA)   |
|             |            |             |
| 模拟LDO ---+--+--ADC ---+             |
|                  (跨边界)              |
+------------------------------------------+
```

### 3.2 模拟区布局要点

1. **敏感模拟IC靠近ADC输入**：信号路径最短
2. **高阻抗节点最短走线**：走线越长天线效应越强
3. **VREF紧挨ADC**：走线越短拾取噪声越少
4. **去耦电容紧贴电源引脚**：距离每增1cm高频去耦降约10dB

### 3.3 禁止事项

- DCDC绝不能放在模拟区
- 数字走线不能穿过模拟区
- 晶振不能靠近模拟输入(强辐射源)
- 继电器/电机驱动远离模拟前端

## 4. 走线布线规则

### 4.1 模拟走线规则

| 规则 | 原因 | 违反后果 |
|------|------|----------|
| 走线尽量短 | 减少天线效应 | 拾取更多EMI |
| 高阻抗节点最短 | 对位移电流最敏感 | 噪声耦合显著 |
| 不与数字走线平行 | 减少电容性耦合 | 数字噪声串入 |
| 不跨越地平面分割 | 返回路径完整 | 大环路辐射 |
| 差分对紧密耦合 | 共模抑制 | 差分信号退化 |

### 4.2 保护环(Guard Ring)

对极高阻抗输入(>1G)，用保护环：

```
        +---保护环(接同电位)---+
        |                      |
  ------+-- 高阻抗输入走线 --+------
        +----------------------+
```

保护环屏蔽漏电流和电容性耦合。接至与输入同电位的低阻抗驱动源(如运放输出)。

### 4.3 差分走线规则

1. 等长：长度差 < 上升时间/10
2. 等距：间距恒定保证差分阻抗一致
3. 紧密耦合：间距 < 2倍线宽
4. 同层布线：必须换层时两根线同时打过孔并加地过孔

### 4.4 3W规则

平行走线间距至少3倍线宽，串扰耦合约降70%(相对1W)。5W间距降88%。

## 5. 地平面完整性

### 5.1 地平面的作用

完整地平面提供：最低阻抗返回路径；屏蔽层；散热面；阻抗控制参考平面。

### 5.2 地平面破坏的后果

| 破坏类型 | 后果 | 严重程度 |
|----------|------|----------|
| 缝隙(Slot) | 返回路径绕行 | 严重 |
| 切痕(Split) | 返回电流无法直接回流 | 严重 |
| 孤洞群 | 高频返回路径阻断 | 中等 |

### 5.3 保持完整的原则

1. 模拟区地平面不切割：物理分区 != 切割地平面
2. 信号过孔换层时附近打缝合过孔(Stitching Via)
3. 避免在地平面上布长走线
4. 内层地平面优先完整保留

缝合过孔间距 < lambda/20。1GHz信号(FR4)：间距 < 7mm，实践每隔3-5mm放一个。

## 6. 电源滤波

### 6.1 模拟电源独立供电

**方法一：铁氧体磁珠 + 电容**

```
VDD_digital --> [磁珠] --> [100uF] --> VDD_analog
                            |
                        [100nF陶瓷]
                            |
                           GND
```

磁珠100MHz时阻抗约100-600欧，对高频噪声有良好衰减。

**方法二：独立LDO**：PSRR低频段可达60-80dB，高频下降需配合磁珠。

### 6.2 磁珠选择

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 阻抗@100MHz | 100-600 | 越高衰减越强 |
| DCR | <1欧 | 影响压降和功耗 |
| 额定电流 | >负载电流 | 过流饱和失效 |

### 6.3 局部去耦

100nF电容距电源引脚<5mm(理想<2mm)，GND过孔直接连地平面不经走线。

## 7. 屏蔽技术

### 7.1 地铜皮(Ground Pour)

模拟区信号层铺地铜皮并通过缝合过孔连地平面。提供屏蔽和返回路径。**注意**：孤立铜皮是天线，必须连地平面!

### 7.2 过孔栅栏(Via Fence)

敏感走线两侧排列接地过孔(间距 < lambda/20)，相当于法拉第笼侧壁，有效隔离串扰。

```
  GND_via   GND_via   GND_via
     |         |         |
  ----+-- 模拟走线 --+----
     |         |         |
  GND_via   GND_via   GND_via
```

### 7.3 屏蔽罩

极敏感电路(射频前端、高阻抗放大器)用五面金属罩覆盖，第六面由PCB地平面充当。衰减>40dB(1GHz以下)。

## 8. 参考平面选择

### 8.1 信号参考哪一层

高速信号返回电流在最近参考平面流动。

| 层安排 | 参考 | 风险 |
|--------|------|------|
| L1:模拟, L2:GND, L3:Power | L2 GND | 好 |
| L1:模拟, L2:Power, L3:GND | L2 Power | 差 |

**原则**：模拟信号层必须紧邻完整地平面(非电源面)。

### 8.2 层转换问题

信号换层时参考平面变了。**规则**：模拟信号尽量不换层；必须换层时在信号过孔旁打缝合过孔。

## 9. 16位ADC布局实例

### 9.1 系统描述

16位SAR ADC(如ADS8866)，4通道复用输入，3.3V参考，100kSPS。目标ENOB > 14位，耦合噪声 < 150uV。

### 9.2 参考电压布局

```
VREF源 --> [1uF] -- VREF引脚 (<2mm)
         |              |
         +-- [100nF] --+
                        |
                      [10nF]  (紧贴引脚)
                        |
                       GND (直接过孔)
```

规则：VREF走线短且宽；不与数字信号交叉/平行；去耦用C0G/NP0介质；周围铺地铜皮。

### 9.3 输入通道隔离

通道间寄生电容串扰约0.1-1pF。隔离：各通道间距>3W；通道间布地走线；复用器靠近ADC；未用通道接地。

### 9.4 去耦电容布局

每个电源引脚独立去耦，100nF < 2mm，10uF < 5mm，电容过孔直接到地平面，回路面积最小化。

## 10. 热效应考虑

### 10.1 热电偶效应

PCB上不同金属交接处形成热电偶：

| 交接类型 | 热电势(uV/C) | 16位ADC影响 |
|----------|-------------|------------|
| 铜-焊锡 | 1-5 | 0.02-0.1 LSB/C |
| 铜-金(ENIG) | 0.5-2 | 0.01-0.04 LSB/C |

>16位精度系统须考虑焊锡和表面处理热电势。

### 10.2 对称布局

差分电路布局必须对称：几何相同(长度/宽度/过孔数)；热环境相同(距热源相同)；电气环境相同(周围走线/铜皮对称)。

不对称导致：热梯度偏移；共模抑制退化；应力梯度压电效应。

## 11. 设计审查清单

1. **地平面完整**：模拟区无切割/缝隙，信号不跨分割线
2. **分区隔离**：模拟/数字/电源物理分开，信号不跨界
3. **电源滤波**：模拟VDD经磁珠/LDO独立，每IC去耦到位
4. **参考电压**：VREF走线短且隔离，C0G去耦紧贴引脚
5. **走线规则**：高阻抗最短，模拟不与数字平行，差分等长等距
6. **保护措施**：极高阻抗有保护环，敏感走线有地铜皮/栅栏
7. **层叠设计**：模拟层紧邻地平面，尽量不换层
8. **过孔缝合**：换层处有地过孔，地铜皮通过缝合过孔连地
9. **对称布局**：差分电路几何和热环境对称
10. **热电偶**：高精度确认焊锡/表面处理热电势可接受

## 12. 不同精度等级的布局要求

| 精度等级 | ENOB | 允许噪声 | 布局要求 |
|----------|------|---------|----------|
| 8-10位 | 8-10 | >1mV | 基本分区+去耦 |
| 12位 | 10-12 | 0.1-1mV | 分区+独立电源+走线规则 |
| 14-16位 | 12-14 | 10-100uV | 严格分区+VREF隔离+保护环 |
| 18-24位 | 16-20+ | <10uV | 全部规则+屏蔽罩+热对称 |

12位是"认真布局即可达到"的边界，16位以上必须系统性应用所有规则。

## 总结

低噪声模拟PCB布局的核心是"隔离与完整性"：

1. **物理分区**：模拟、数字、电源分开，信号不跨界
2. **地平面完整**：统一地平面 + 物理分区，优于分割
3. **电源独立**：模拟电源经磁珠/LDO独立，去耦紧贴引脚
4. **走线最短**：尤其高阻抗节点
5. **返回路径**：信号必有完整地平面返回路径，换层加缝合过孔
6. **对称设计**：差分电路布局对称，避免热电偶和共模退化

以上规则是工程实践的总结。实际项目应结合仿真(IBIS串扰)和测量(近场探针、频谱仪)验证布局效果。
