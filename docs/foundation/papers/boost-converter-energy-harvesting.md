# Boost升压转换器在能量采集中的应用
> **难度**: 黄 中级 | **领域**: 升压电源设计 | **阅读时间**: 约 20 分钟

## 引言

想象你在荒野中用一面小镜子聚光生火。镜面只有巴掌大,收集的阳光很微弱,但通过持续聚焦,最终能点燃火绒。能量采集(energy harvesting)也是这个道理: 环境中的能量(光、热、振动)微弱而分散,但通过Boost升压转换器逐步积累,最终能为MCU提供足够的启动和工作电压。

能量采集面临的根本挑战是: 能源输出电压太低,远不足以直接驱动电子系统。一片微型太阳能电池在室内光照下输出仅0.3-0.6V,热电发电机(TEG)可能只产生20-100mV,而MCU最低需要1.8V才能工作。Boost转换器必须从这些极低电压启动,并在微瓦级功率下保持可接受的效率,这是完全不同于传统电源设计的挑战领域。

本文从Boost原理出发,深入讲解能量采集中的冷启动、MPPT集成、超低压升压IC选型,以及从太阳能到超电容到MCU的完整设计流程。

## 1. Boost升压转换器工作原理

### 1.1 基本拓扑

Boost转换器由开关管、电感、二极管(或同步开关)和输出电容组成:

```
       Q1(开关管)        D1(续流)
Vin ---+---[ MOSFET ]---+---[ DIODE ]---+--- Vout
       |                                 |
      [L]电感                           Cout
       |                                 |
      GND                               GND
```

两个工作阶段:
1. Q1导通: 电感与Vin并联,电感电流线性上升,电感储能;输出电容单独向负载供电
2. Q1关断: 电感电流通过二极管向输出电容和负载供电,电感释放能量,输出电压高于输入

### 1.2 伏秒平衡与增益

稳态伏秒平衡:

```
Vin x ton = (Vout - Vin) x toff

增益 M = Vout / Vin = 1 / (1 - D)
```

占空比D = 1 - Vin/Vout

例如: Vin=0.5V, Vout=3.3V, D = 1 - 0.5/3.3 = 84.8%

注意: 占空比不能无限接近100%,实际最大D约90-95%,否则电感没有足够的关断时间释放能量。

### 1.3 与Buck的关键区别

| 特性 | Buck | Boost |
|------|------|-------|
| 电压关系 | Vin > Vout | Vin < Vout |
| 输入电流 | 断续(脉冲) | 连续(电感串联) |
| 输出电流 | 连续(电感串联) | 断续(二极管续流) |
| 输出纹波 | 较小 | 较大 |
| 右半平面零点 | 无 | 有(环路补偿更困难) |

Boost的输出电流是断续的,意味着输出纹波通常比Buck更大,需要更大的输出电容。

## 2. 能量采集为何需要Boost

### 2.1 能源输出电压低

各种能量采集源的典型输出电压:

| 能源类型 | 典型输出电压 | 典型输出功率 |
|----------|-------------|-------------|
| 室内太阳能(单节) | 0.3-0.6V | 10-500uW/cm2 |
| 室外太阳能(单节) | 0.5-0.7V | 10-100mW/cm2 |
| 热电发电机(TEG) | 20-100mV | 10-500uW |
| 压电振动 | 1-5V(AC) | 50-500uW |
| RF采集 | 0.1-0.5V | 1-50uW |

MCU最低工作电压通常1.8V,储能元件(超电容/电池)需要2.0-3.6V充电。这意味着必须将源电压升高3-100倍。

### 2.2 能量微弱且波动

能量采集源不仅电压低,功率也在微瓦级,且随环境变化剧烈:
- 室内光照: 人走过来可能遮挡,光照骤降50%
- TEG: 环境温度变化导致温差波动
- 振动: 设备启停导致振动能量间歇

Boost转换器必须在极宽的输入功率范围内保持可接受的效率,且自身功耗(静态电流Iq)必须远低于采集到的能量。

### 2.3 冷启动问题

冷启动(Cold Start)是能量采集Boost面临的最严峻挑战: 系统完全没电时,Boost IC自身需要一定的最低输入电压才能启动。一旦启动,可以通过电荷泵等辅助电路在更低电压下维持工作,但从零电压到最低启动电压这一步,必须有特殊机制。

典型冷启动电压:
- BQ25570: 330mV冷启动(启动后可降至100mV)
- LTC3108: 20mV冷启动(使用变压器升压)
- SPV1050: 100mV冷启动

## 3. 电荷泵与电感式Boost对比

### 3.1 电荷泵(Charge Pump)

电荷泵通过电容的交替充放电实现升压,无需电感:

```
Phase 1(充电): Vin --[C_fly]--> C_fly充电到Vin
Phase 2(转移): C_fly叠加Vin --> Vout = 2xVin(倍压)
```

优点:
- 无需电感,体积小
- 低压差时效率高
- 适合小电流(uA级)辅助启动

缺点:
- 升压比固定(2x, 3x等),不够灵活
- 大电流时效率急剧下降
- 输出纹波大

### 3.2 电感式Boost

传统电感式Boost,通过电感储能实现连续可调升压。

优点:
- 升压比灵活可调
- 大电流效率高
- 输出电压可精确控制

缺点:
- 需要电感,体积大
- 极低输入电压时启动困难
- 超低功率时效率下降

### 3.3 混合方案

能量采集IC通常采用混合方案:
- 冷启动阶段: 电荷泵辅助,从极低电压建立初始输出
- 正常运行阶段: 切换到电感式Boost,效率更高
- 这类IC内部包含两套转换电路

## 4. 超低压Boost IC详解

### 4.1 BQ25570 (Texas Instruments)

BQ25570是能量采集领域最成熟的IC之一:

```
冷启动电压: 330mV
启动后最低输入: 100mV
输入范围: 0.1-5.5V
输出范围: 2.0-5.5V(可编程)
MPPT: 有(可编程采样间隔)
电池管理: 过充/过放保护
封装: QFN-20 (3.5x4.0mm)
```

核心特性:
- 内置冷启动电荷泵: 从330mV开始建立Vout
- 可编程MPPT: 支持多种能源的最优匹配
- 电池管理: 可充锂电池或超电容,带过充过放保护
- 独立的Vout调节: 存储电压和负载电压可分别设置

典型应用: 太阳能/TEG供电的无线传感器节点

### 4.2 LTC3108 (Analog Devices)

LTC3108使用外部变压器实现极低电压启动:

```
冷启动电压: 20mV(变压器升压)
输入范围: 20mV-500mV
输出选项: 2.35V/3.3V/4.1V/5.0V(可选)
MPPT: 无(固定变压器比)
电池管理: LDO输出(2.2V)
封装: SSOP-24 (5.3x6.2mm)
```

核心特性:
- 外部1:100变压器: 将20mV升至2V以上,直接驱动内部电路
- 极低电压启动: 业界最低冷启动电压之一
- 多路输出: 主输出+LDO+VS2,为不同负载供电
- 适合TEG: 低电压大内阻源

局限性: 外部变压器体积大(约10x10mm),不适合超小型化设计

### 4.3 SPV1050 (STMicroelectronics)

```
冷启动电压: 100mV(内置电荷泵)
输入范围: 0.15-5.5V
输出范围: 1.8-5.25V(可编程)
MPPT: 有(开路电压法)
电池管理: 超电容/电池充电
封装: QFN-20 (4.0x4.0mm)
```

核心特性:
- 内置电荷泵冷启动: 无需外部变压器
- 超低Iq: 800nA(典型值)
- MPPT开路电压法: 定期采样开路电压,工作点设为75-90%
- 适合太阳能和TEG

### 4.4 IC对比总结

| 参数 | BQ25570 | LTC3108 | SPV1050 |
|------|---------|---------|---------|
| 冷启动电压 | 330mV | 20mV | 100mV |
| 最低工作输入 | 100mV | 20mV | 150mV |
| MPPT | 有(可编程) | 无 | 有(开路电压) |
| 外部变压器 | 不需要 | 需要 | 不需要 |
| Iq | 325nA | 变压器驱动 | 800nA |
| 电池管理 | 完善 | 基础 | 完善 |
| 典型应用 | 太阳能/TEG | TEG | 太阳能/TEG |

## 5. MPPT在Boost中的集成

### 5.1 为什么需要MPPT

能量采集源(如太阳能电池)有非线性I-V特性,存在一个最大功率点(MPP)。如果Boost的输入阻抗不匹配,则无法从源提取最大功率。

```
功率
 ^
 |    *
 |   / \
 |  /   \
 | /     \
 |/       \
 +---------+--> 电压
        Vmp
```

MPPT(Maximum Power Point Tracking)的作用就是持续调节Boost的输入阻抗,使工作点始终在MPP附近。

### 5.2 开路电压法(OCV法)

最常用的能量采集MPPT方法:

1. 周期性断开Boost输入,测量开路电压Voc
2. 设定工作电压 = k x Voc (k通常取0.75-0.85)
3. Boost在工作阶段保持输入电压接近k x Voc
4. 间隔: 通常1-16秒采样一次

优点: 简单可靠,无需电流采样
缺点: 采样期间有短暂中断,不适用于快速变化的环境

### 5.3 扰动观察法(P&O)

更精确但更复杂的MPPT方法:
1. 微调占空比,观察输入功率变化
2. 功率增大则继续同方向微调
3. 功率减小则反向微调
4. 持续振荡在MPP附近

能量采集IC很少使用P&O,因为:
- 需要电流采样,增加功耗和成本
- 微瓦级功率下,采样误差容易超过信号本身
- OCV法已足够精确

## 6. 输出调节与储能管理

### 6.1 储能元件选择

能量采集是间歇性的,负载是脉冲式的,储能元件是两者之间的缓冲:

| 储能类型 | 能量密度 | 充放电次数 | 自放电 | 适用场景 |
|----------|---------|-----------|--------|----------|
| 超电容 | 低 | >50万次 | 高(天级) | 短期缓冲,频繁充放 |
| 可充电电池 | 高 | 300-1000次 | 低(月级) | 长期储能 |
| 薄膜电池 | 中 | >5000次 | 极低 | 微型IoT |

### 6.2 超电容充电管理

超电容充电需要恒流-恒压(CC-CV)控制:
- 电容电压低时: 恒流充电,限制电流不超过源的最大功率
- 电容电压接近目标: 恒压充电,逐渐减小电流
- 充满后: 维持涓流充电

BQ25570内置超电容/电池充放电管理:
- 过充保护: 电压达到上限时停止充电
- 过放保护: 电压低于下限时断开负载
- 可编程的充放电阈值

### 6.3 输出电压调节

能量采集Boost通常有两级输出:
1. 存储输出(VSTOR): 直接连储能元件,电压随充放电变化
2. 调节输出(VOUT): 经LDO稳压后给负载供电

这种两级结构确保: 即使储能电压波动(如超电容2.5-5.0V),负载电压始终稳定(如3.3V)。

## 7. 微瓦级功率下的效率考量

### 7.1 效率曲线的特殊性

传统Boost在数瓦级功率下效率可达90%以上,但在微瓦级(1-100uW)效率急剧下降:

| 输入功率 | 典型效率 | 原因 |
|----------|---------|------|
| 1W | 85-92% | 正常范围 |
| 10mW | 75-88% | Iq开始影响 |
| 100uW | 40-60% | Iq和驱动损耗占比大 |
| 10uW | 10-30% | 几乎无法工作 |

### 7.2 提升微瓦级效率的策略

1. **超低Iq**: IC自身Iq必须远低于采集功率
   - 目标: Iq < Pin的5%
   - 如Pin=100uW, Vin=0.5V, 则Iq < 10uA

2. **间歇工作**: 积攒足够能量后一次性工作,而非持续运行
   - 超电容充电到阈值 --> MCU唤醒工作 --> 超电容放电到下限 --> MCU睡眠
   - 工作占空比: 0.01-1%

3. **最小化控制开销**: 简化MPPT频率,减少采样次数

4. **选择合适的电感**: 低DCR,高饱和电流,但体积与效率权衡

## 8. 关键元器件选型

### 8.1 电感选择

能量采集Boost的电感选择比传统Boost更苛刻:
- DCR < 1Ohm(微安级电流下,DCR损耗占比大)
- 饱和电流 > 峰值电感电流(含冷启动浪涌)
- 电感值: 通常10-100uH(低开关频率下储能更多)

### 8.2 二极管 vs 同步整流

| 参数 | 肖特基二极管 | 同步整流(MOSFET) |
|------|-------------|-----------------|
| 正向压降 | 0.2-0.4V | Rds_on x I, 极低 |
| 效率(大电流) | 较低 | 较高 |
| 效率(微电流) | 较高(无驱动损耗) | 较低(驱动损耗占比大) |
| 复杂度 | 简单 | 需要驱动电路 |

微瓦级能量采集: 肖特基二极管往往更高效,因为MOSFET驱动损耗在微安级电流下占比太大。

中功率能量采集(>1mW): 同步整流效率更高。

### 8.3 输入电容

能量采集源内阻高,输入电容用于:
- 在开关导通时提供瞬态电流
- 稳定MPPT采样时的输入电压
- 典型值: 10-100uF(低ESR陶瓷+钽电容并联)

## 9. 实战设计: 太阳能到超电容到MCU

### 9.1 系统需求

- 能源: 室内太阳能电池,5cm x 5cm
- 光照条件: 200-1000 lux办公室照明
- 储能: 0.1F超电容
- 负载: nRF52832 MCU + 温湿度传感器
- 工作模式: 每5分钟采样一次,每次工作200ms
- 工作电流: 10mA(采样+BLE广播)
- 睡眠电流: 2uA

### 9.2 能量预算计算

```
// 太阳能电池输出(室内200lux)
P_solar = 5cm x 5cm x 20uW/cm2 = 500uW
V_solar_oc = 0.5V (开路电压)
V_solar_mpp = 0.4V (MPP电压)
I_solar_mpp = 500uW / 0.4V = 1.25mA

// Boost效率(100uW-500uW): 约50%
P_boost_out = 500uW x 50% = 250uW

// 超电容充电时间
C = 0.1F
V_charge = 3.3V (目标)
E_cap = 0.5 x C x V^2 = 0.5 x 0.1 x 3.3^2 = 544.5mJ

// 充电时间
T_charge = E_cap / P_boost_out = 544.5mJ / 250uW = 2178s ~ 36min

// 每次工作消耗
E_work = 3.3V x 10mA x 0.2s = 6.6mJ
E_sleep_5min = 3.3V x 2uA x 300s = 1.98mJ
E_cycle = 6.6 + 1.98 = 8.58mJ

// 等效连续功耗
P_avg = 8.58mJ / 300s = 28.6uW

// 净充电功率
P_net = 250uW - 28.6uW = 221.4uW(充电时)
// 系统可持续运行,且每36分钟可充满一次超电容
```

### 9.3 电路设计

IC选择: BQ25570(太阳能MPPT + 超电容管理)

```
太阳能电池(0.4V) --> BQ25570 Boost --> 超电容(0.1F, 3.3V)
                                            |
                                        LDO(3.0V)
                                            |
                                      nRF52832 MCU
                                            |
                                      SHT30 温湿度传感器
```

BOM:

| 元件 | 型号 | 参数 |
|------|------|------|
| U1 | BQ25570 | 能量采集Boost |
| L1 | 74406610 | 10uH, DCR=0.1Ohm |
| Cstor | EESXJ101 | 0.1F/3.6V超电容 |
| Cin | GRM188R60J475M | 4.7uF陶瓷 |
| Cout | GRM188R60J106M | 10uF陶瓷 |
| D1 | RB521ZS30 | 肖特基二极管 |

### 9.4 关键设计考量

1. MPPT设置: k=0.8, 采样间隔16秒(室内光照变化慢)
2. 超电容过压保护: 3.6V(防止超电容过充损坏)
3. 超电容欠压保护: 2.0V(保证LDO输出稳定)
4. 冷启动: 首次启动需330mV输入,之后可降至100mV维持

## 总结

Boost升压转换器在能量采集中的角色远不止"升压": 它是连接微弱环境能源和电子系统的桥梁。冷启动能力决定了系统能否从零开始工作,MPPT确保从源提取最大功率,超低Iq保证在微瓦级功率下不会"入不敷出",储能管理则平衡了间歇供能和脉冲负载之间的矛盾。设计能量采集Boost时,不能只看峰值效率,更要关注微瓦级功率下的效率、冷启动电压、以及IC自身功耗与采集功率的比例。能量采集不是"免费午餐",每一微瓦都来之不易。
