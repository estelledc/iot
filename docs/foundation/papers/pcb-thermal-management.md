# PCB热设计：铜面积/过孔散热/热仿真
> **难度**：🟡 中级 | **领域**：PCB热管理 | **阅读时间**：约 20 分钟

## 引言

想象你在厨房煎牛排。锅底面积越大，热量分布越均匀，牛排就不容易局部焦糊。PCB上的芯片发热也是一样——热量从芯片结(junction)出发，经过封装外壳、焊盘、铜皮、PCB基材，最终散到空气里。如果任何一段"路"太窄，热量就堵在芯片内部，温度飙升，轻则性能降级，重则烧毁。

IoT设备空间紧凑、外壳密闭、无风扇，热设计比消费电子更苛刻。本文从热源识别、热阻模型、铜皮散热、过孔阵列、热仿真到实测验证，给出面向IoT PCB的完整热设计指南。

## 1. IoT PCB上的热源

### 1.1 主要发热器件

| 器件类型 | 典型功耗 | 发热特征 |
|----------|----------|----------|
| DC-DC降压转换器 | 0.5-3 W | 效率损耗集中在开关管和电感 |
| LDO线性稳压器 | 0.2-2 W | 压差乘电流全部变热 |
| 功率放大器(PA) | 1-5 W | 发射时脉冲发热，占空比影响平均温度 |
| MCU/SOC | 0.1-1 W | 全速运行时发热，休眠可忽略 |
| Wi-Fi/BLE模块 | 0.3-2 W | 发射峰值高，平均低 |
| 电池充电IC | 0.5-3 W | 快充时持续发热 |

### 1.2 功耗估算

对LDO，发热最容易算：`P_LDO = (V_in - V_out) * I_load`

例如3.6V锂电池经LDO降至1.8V，负载300mA：P = (3.6-1.8)*0.3 = 0.54W

对DC-DC：`P_DCDC = P_out * (1 - efficiency) / efficiency`

例如输出3.3V/1A，效率90%：P = 3.3*(1-0.9)/0.9 = 0.37W

### 1.3 热源布局原则

- 大功率器件分散放置，避免热聚集
- 发热器件远离温度敏感器件(温敏晶振、NTC热敏电阻)
- 同一热流路径上不要堆叠多个大功率器件
- 充电IC靠近电池，缩短大电流路径

## 2. 热阻模型：从结到环境

### 2.1 热阻链路

热量从芯片结到环境空气，经过串联热阻：

```
T_j = T_amb + P * (R_jc + R_cs + R_sa)
```

- T_j：结温，T_amb：环境温度，P：器件功耗(W)
- R_jc：结到外壳热阻(junction-to-case)，数据手册给出
- R_cs：外壳到散热器/PCB热阻，取决于接触质量
- R_sa：散热器/PCB到环境热阻，取决于面积和风况

### 2.2 典型热阻值

| 封装 | R_jc (C/W) | R_ja (C/W) | 备注 |
|------|-----------|-----------|------|
| QFN-48 (7x7mm) | 8-15 | 30-60 | 取决于暴露焊盘 |
| BGA-256 | 5-10 | 20-40 | 焊球导热好 |
| SOT-23 | 80-150 | 200-400 | 小封装，散热差 |
| DPAK/D2PAK | 2-5 | 40-80 | 功率器件常用 |
| WLCSP | 10-20 | 50-100 | 无封装，PCB直接散热 |

注意：R_ja是"自然对流，标准测试板"条件下的值，实际PCB上可能差2-3倍。

### 2.3 目标结温设计

大多数硅器件最高结温125C或150C。工程上留25C余量：

```
T_j_target = T_j_max - 25C
```

对125C器件设计到100C以下。原因：温度每降10C寿命约翻倍(Arrhenius模型)；高温下电参数漂移(时序、漏电流)；环境温度波动未计入。

## 3. 铜面积作为散热器

### 3.1 铜皮的导热原理

铜的导热系数约385 W/(m*K)，是FR4(0.3 W/(m*K))的1000倍。顶层/底层铜皮相当于横向导热板，把芯片热点扩散到更大面积，再由对流和辐射传给空气。1oz铜厚约35um，2oz约70um，铜越厚散热越好。

### 3.2 铜面积与热阻的关系

PCB铜皮到环境的热阻近似：`R_sa = 1 / (h * A)`，h为对流换热系数(自然对流约5-10 W/(m^2*K))，A为铜面积(m^2)。

1oz铜皮在自然对流下的经验估算：

| 铜面积 (mm^2) | 近似 R_sa (C/W) | 可散功耗 (W) at 40C温升 |
|---------------|----------------|------------------------|
| 100 | 50-70 | 0.6-0.8 |
| 400 | 25-35 | 1.1-1.6 |
| 900 | 15-22 | 1.8-2.7 |
| 1600 | 10-16 | 2.5-4.0 |
| 2500 | 8-12 | 3.3-5.0 |

面积增大收益递减——面积翻倍，热阻降不到一半，因为热量还没传到边缘就已散掉。

### 3.3 铜皮设计规则

1. **铺铜到器件周围**：发热器件下方及周围铺实心铜皮(不是网格铜)
2. **同层连通**：宽铜皮连接相邻发热器件散热区，共享散热板
3. **避免割裂**：信号走线不要在散热铜皮上开出长窄槽
4. **多面铺铜**：顶层和底层都铺，通过过孔热耦合
5. **2oz铜**：对1W以上器件优先选2oz铜厚

### 3.4 铜面积计算实例

某QFN MCU功耗0.8W，R_jc = 10C/W，要求T_j < 100C，环境温度60C(密闭外壳)。

```
允许温升 = 100 - 60 = 40C
允许 R_ja = 40 / 0.8 = 50 C/W
R_sa = R_ja - R_jc = 50 - 10 = 40 C/W
```

查上表，约400mm^2铜面积可满足，即20mm x 20mm铺铜区。

## 4. 热过孔阵列

### 4.1 过孔为什么能散热

过孔是贯穿PCB各层的金属化孔。铜过孔壁将热量从顶层传到底层或内层，大幅增加有效散热面积。对QFN等底部散热的封装，过孔阵列是必须的。

### 4.2 过孔热阻模型

单个过孔热阻近似：`R_via = L / (k_Cu * A_Cu)`

L为PCB厚度(1.6mm典型)，k_Cu为铜导热系数385 W/(m*K)，A_Cu为过孔铜壁截面积。

对通孔直径0.3mm，铜壁厚25um：

```
A_Cu = pi * d * t = 3.14 * 0.3e-3 * 25e-6 = 2.36e-8 m^2
R_via = 1.6e-3 / (385 * 2.36e-8) = 176 C/W
```

单个过孔热阻很高，需阵列并联：`R_via_array = R_via / N`。9个过孔R=176/9=20 C/W，25个过孔R=7 C/W。

### 4.3 热过孔设计规则

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 过孔直径 | 0.3-0.6 mm | 太小镀不上铜，太大占空间 |
| 铜壁厚 | >= 25 um | 1oz镀铜 |
| 间距(pitch) | 1.0-1.5 mm | 过密挤占布线空间，过疏散热不足 |
| 放置区域 | 器件暴露焊盘正下方 | 最短热流路径 |
| 阵列规模 | 4-25个 | 视功耗和可用面积而定 |
| 过孔填铜 | 建议填铜或导电环氧 | 空气导热差，填铜降阻30-50% |

### 4.4 过孔阵列实例

某QFN-48封装，中央暴露焊盘5x5mm，功耗1.5W。按1.2mm间距排列最多4x4=16个过孔。

```
R_via_array = 176 / 16 = 11 C/W
与顶层铜皮(R_sa=15 C/W)并联: R_total = 1/(1/15+1/11) = 6.3 C/W
R_ja = 10 + 6.3 = 16.3 C/W
T_j = 60 + 1.5 * 16.3 = 84.5 C (可接受)
```

## 5. 暴露焊盘(Exposed Pad)焊接

### 5.1 为什么暴露焊盘是关键

QFN、QFP-EP、BGA等封装底部有金属暴露焊盘，直接连到芯片内部散热片。这是芯片到PCB最短的热通路。暴露焊盘没焊好，R_jc可能翻2-3倍。

### 5.2 焊盘设计与缺陷影响

焊盘设计要点：
- PCB焊盘与封装暴露焊盘等大或略大(每侧加0.25mm)
- 分为多个子焊盘(window-pane)，中间留过孔位
- 锡膏覆盖50-80%面积，太多桥接，太少空洞

| 缺陷 | 热阻影响 | 检测方法 |
|------|----------|----------|
| 空洞(void) > 30% | R_jc增加20-50% | X-ray检查 |
| 未润湿 | R_jc增加100-300% | X-ray + 电测试 |
| 锡膏不足 | R_jc增加50-100% | X-ray |
| 焊盘无过孔 | R_ja增加30-60% | 设计审查 |

对IoT产品，建议X-ray 100%检查暴露焊盘焊接质量，特别是功耗>0.5W的器件。

## 6. 热释放(Thermal Relief)图案

### 6.1 用途与矛盾

热释放图案在大铜皮和器件焊盘之间放2-4条窄铜辐条，减少焊盘吸热改善可焊性。但辐条越窄热阻越大，对暴露焊盘恰恰相反——需要最小热阻。

| 场景 | 推荐 | 原因 |
|------|------|------|
| 普通信号焊盘连大铜皮 | 热释放，2-4条辐条 | 保证可焊性 |
| 暴露散热焊盘 | 直连，不加热释放 | 热阻最小化 |
| 大电流电源焊盘 | 直连或宽辐条 | 降低电阻和电感 |
| 小信号IC电源脚 | 热释放可接受 | 功耗极低 |

辐条尺寸：宽度0.25-0.4mm(或焊盘宽度的50-60%)，数量2-4条，等分360度。暴露焊盘则直接铺铜连接，过孔在焊盘内。

## 7. PCB材料与热导率

### 7.1 常见PCB基材

| 材料 | 面内导热 (W/mK) | 垂直导热 (W/mK) | 典型应用 |
|------|-----------------|-----------------|----------|
| FR4 (标准) | 0.3-0.4 | 0.2-0.3 | 通用 |
| FR4 (高Tg) | 0.4-0.6 | 0.3-0.4 | 无铅焊接 |
| 金属基板(铝基) | 1.0-2.0 | 1.0-2.0 | LED照明 |
| 陶瓷(Al2O3) | 20-30 | 20-30 | 高功率射频 |
| Rogers RO4350B | 0.6 | 0.4 | 射频/高速 |

FR4垂直方向导热极差，这是过孔散热的核心原因——热量无法靠FR4垂直传导，必须靠铜过孔。

### 7.2 铜层对有效导热的影响

4层板叠层Top/GND/VCC/Bottom：

| 铜厚 | 有效面内导热 (W/mK) | 说明 |
|------|---------------------|------|
| 1oz all | 8-12 | 内层铜贡献大 |
| 2oz inner, 1oz outer | 15-25 | 内层2oz有效 |
| 2oz all | 20-35 | 最佳但成本高 |

内层大面积铺铜(GND和VCC平面)显著提高PCB整体面内导热，这是4层板比2层板散热好的重要原因。

## 8. 热仿真工具

### 8.1 手工计算 vs 仿真

| 方法 | 精度 | 耗时 | 适用场景 |
|------|------|------|----------|
| 热阻链手算 | +/-30% | 10分钟 | 早期估算，简单拓扑 |
| PCB估算工具 | +/-20% | 1小时 | 布局前评估 |
| ANSYS Icepak | +/-10% | 1-3天 | 关键器件，密闭壳体 |
| Flotherm | +/-10% | 1-3天 | 电子散热专业工具 |

### 8.2 手工计算流程

1. 列出所有发热器件及其功耗
2. 查数据手册获取R_jc
3. 估算R_sa(铜面积 + 过孔阵列)
4. 串联相加得R_ja
5. 计算T_j = T_amb + P * R_ja
6. 检查是否低于T_j_target

### 8.3 ANSYS Icepak工作流

1. 导入PCB布局(ODB++或IDF格式)
2. 定义材料属性(铜、FR4、器件封装)
3. 设置边界条件(外壳壁面、开孔、风扇)
4. 添加热源(器件功耗)，划分网格(关键区域加密)
5. 求解稳态/瞬态温度场，后处理温度云图、热流矢量图

IoT设备特别注意：密闭壳体自然对流模型；电池作为热源和热沉的双重角色；外壳壁面辐射换热；安装表面接触热阻。

### 8.4 简化仿真示例

对单层板+QFN器件，一维热阻模型快速估算：

```
R_board = t_board / (k_Cu_eff * A_copper)
k_Cu_eff = 铜覆盖率 * 385
```

假设铜覆盖率10%，散热面积900mm^2：k_Cu_eff = 38.5 W/(m*K)，R_board = 1.6e-3/(38.5*900e-6) = 0.046 C/W。板内热阻很小，瓶颈在板面到空气的对流热阻。

## 9. 器件温度降额

### 9.1 温度降额曲线

数据手册给出功耗-温度降额曲线：`P_allowable = P_max * (T_j_max - T_amb) / (T_j_max - 25C)`，当T_amb > 25C。

例如某LDO，P_max = 3W，T_j_max = 150C，环境85C：P_allowable = 3*(150-85)/(150-25) = 1.56W，不到标称一半。

### 9.2 IoT常见高温场景

| 场景 | 环境温度 | 降额比 (T_j_max=125C) |
|------|----------|----------------------|
| 室内常温 | 25C | 100% |
| 工业车间 | 40-50C | 80-88% |
| 阳光直射外壳 | 60-70C | 60-72% |
| 密闭金属壳+日照 | 70-85C | 48-60% |
| 汽车引擎舱 | 85-105C | 20-40% |

IoT户外设备必须按60-70C环境温度设计，保守取75C。

### 9.3 关键参数的温度漂移

| 参数 | 温度系数 | 影响 |
|------|----------|------|
| 硅器件漏电流 | 约每10C翻倍 | 高温静态功耗增加 |
| 电阻温漂 | 50-200 ppm/C | 采样精度 |
| MOSFET导通电阻 | 约0.7%/C | 开关损耗增加 |
| 电解电容ESR | 每C降约2% | 高温寿命缩短 |

## 10. 密闭壳体的热设计

### 10.1 IoT密闭设备的特殊性

- IP67/IP68防水：不能开散热孔
- 美观/小型化：无外露散热片
- 墙壁/管道安装：一面贴着绝热体

### 10.2 密闭壳内热模型

壳内空气近似均匀温度T_int，壳外环境T_amb：`P_total = (T_int - T_amb) / R_shell`

典型塑料壳：`R_shell = 1 / (h_out * A_shell)`

对100x60x20mm塑料壳，外表面积0.0128m^2：R_shell = 1/(8*0.0128) = 9.8 C/W。若总功耗2W：T_int = 25 + 2*9.8 = 44.6C

### 10.3 密闭壳体散热优化

1. **导热垫/导热胶**：芯片到壳体壁面，建立导热短路
2. **金属内衬**：塑料壳内贴铝箔或金属板
3. **利用安装面**：金属安装支架兼作散热器
4. **铜皮延展**：PCB铜皮延伸到板边缘，接触壳体壁
5. **降低总功耗**：Duty cycling、降压运行、低功耗模式
6. **导热塑料**：k=1-3 W/mK替代普通塑料k=0.2 W/mK

壳内布局：发热器件放PCB边缘靠近壳壁；电池放中部偏上(怕热)；导热垫面积最大化厚度最小化(0.5-1mm)；避免发热器件上下堆叠。

## 11. 实测验证方法

### 11.1 红外热像仪(IR Camera)

优点：非接触，全视场温度分布，直观。缺点：只能测表面，受发射率影响。

使用要点：设置正确发射率(PCB绿油约0.95，裸铜约0.1-0.3)；裸铜区涂黑漆提高发射率；环境反射补偿。

### 11.2 热电偶(Thermocouple)

优点：直接接触，精度高(+/-1C)。缺点：单点测量，布线影响对流。

使用要点：K型最常用；用导热胶固定测点(不用胶带)；测点靠近器件封装表面。

### 11.3 结温间接测量

利用硅器件温度敏感电参数(TSP)间接测结温：

**二极管法**：施加大电流(<1mA)，测正向压降V_F，查V_F-T校准曲线。`T_j = (V_F - V_F0) / K + T_0`，K约-2 mV/C(硅二极管)。

**R_ds-on法**：对MOSFET，低电流下测导通电阻，查温度系数算结温。

### 11.4 测量与仿真对比

| 项目 | 仿真 | 实测 | 偏差 |
|------|------|------|------|
| MCU表面温度 | 72C | 68C | +4C |
| LDO表面温度 | 85C | 82C | +3C |
| PCB最高温度 | 78C | 75C | +3C |

仿真偏保守(高)是正常的，模型未计入某些散热路径(安装面、连接器等)。

## 12. 设计指南总结

### 12.1 检查清单

**布局阶段**：
- [ ] 识别所有>0.25W的发热器件
- [ ] 计算每个器件的R_ja需求
- [ ] 发热器件分散放置，远离温度敏感器件

**布线阶段**：
- [ ] 发热器件下方铺实心铜皮
- [ ] 暴露焊盘直连铜皮，不加热释放
- [ ] 热过孔阵列：直径0.3-0.6mm，间距1.0-1.5mm
- [ ] 内层铺铜平面完整，无热流瓶颈

**生产阶段**：
- [ ] X-ray检查暴露焊盘空洞率<30%
- [ ] 过孔镀铜厚度>=25um，导热垫正确安装

**验证阶段**：
- [ ] 红外热像仪扫描，关键器件温度在降额范围内
- [ ] 高温环境(60-70C)下长时间运行测试

### 12.2 功耗与对策速查

| 总板功耗 | 散热策略 |
|----------|----------|
| < 0.5W | 标准FR4 1oz铜，基本铺铜即可 |
| 0.5-2W | 铜面积扩展 + 热过孔 + 布局优化 |
| 2-5W | 2oz铜 + 密集过孔阵列 + 导热垫到壳体 |
| 5-10W | 金属基板或内层2oz + 主动散热考虑 |
| > 10W | 需要风扇或外部散热器 |

### 12.3 常见错误

1. **忽视LDO发热**：压差大时LDO发热远超DC-DC，尽量用DC-DC替代
2. **暴露焊盘未焊**：QFN底部虚焊，热阻翻倍，X-ray必查
3. **网格铜散热**：网格铜有效面积仅50-70%，散热用实心铜
4. **过孔填空气**：空气导热0.026 W/mK，填铜降阻3-5倍
5. **只看R_ja**：数据手册R_ja在标准测试板上测得，实际PCB可能差2-3倍
6. **忽略壳体效应**：密闭壳内环境温度比标称高10-30C

## 总结

PCB热设计是IoT产品可靠性的底层保障。核心思路只有一条：**让热量从芯片结到环境空气的每一段路径都足够宽**。

关键要点回顾：

1. **热阻链**：T_j = T_amb + P * (R_jc + R_cs + R_sa)，每段热阻都要算
2. **铜面积**：最廉价高效的散热手段，实心铺铜，面积越大越好，但有递减效应
3. **热过孔**：穿层导热的唯一有效手段，阵列设计，填铜优先
4. **暴露焊盘**：芯片到PCB的第一道关卡，焊接质量决定成败
5. **密闭壳体**：IoT的常态，导热短路到壳壁是核心策略
6. **温度降额**：高温环境下器件额定功耗大幅缩水，必须校核
7. **仿真+实测**：先算后仿再测，三者交叉验证

热设计不是事后补救，而是布局阶段就要介入的约束条件。功耗0.5W以上的器件，在放第一个footprint之前就该算好热阻预算。
