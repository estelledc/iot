# 仪表放大器INA在微弱信号测量中的应用
> **难度**：🟡 中级 | **领域**：精密模拟电路 | **阅读时间**：约 20 分钟

## 引言

想象你在一个嘈杂的工地上,试图听清远处朋友说的一句话.朋友的声音就是你要测量的微弱差分信号,而工地上机器的轰鸣就是共模噪声.普通的耳朵(普通运算放大器)会把轰鸣声一起放大,你什么也听不清;但如果你戴上一副特殊的降噪耳机,它能精确抵消左右两侧同时传来的轰鸣(共模),只保留两侧不一致的声音(差分),你就能清楚地听到朋友的话了.

这副"降噪耳机"就是仪表放大器(Instrumentation Amplifier,简称INA).在传感器信号采集领域,温度桥、压力传感器、心电电极等输出的有用信号往往只有微伏到毫伏量级,而同时叠加在两条信号线上的共模干扰却高达数伏.INA的核心使命就是:在极高共模环境下,精准地提取微弱差分信号.

## 1 仪表放大器 vs 普通运算放大器

### 1.1 为什么普通运放不够用

普通运算放大器在实际差分采集中有三个致命问题:

1. **输入阻抗不对称**:反相端和同相端阻抗不同,信号源内阻不可忽略时引入失调和增益误差.
2. **共模抑制能力有限**:单运放CMRR通常只有60-80dB,且增益越低抑制越差.
3. **增益设定困难**:差分电路需要四个精密电阻精确匹配才能保证CMRR,电阻容差直接破坏共模抑制.

### 1.2 INA的核心优势

| 特性 | 普通运放差分电路 | 仪表放大器INA |
|------|-----------------|--------------|
| 输入阻抗 | 低且不对称 | 极高且对称(GΩ级) |
| CMRR | 60-80dB,依赖电阻匹配 | 80-120dB,内部匹配 |
| 增益设定 | 需多个精密电阻 | 单电阻设定 |
| 带宽-增益权衡 | 增益越高带宽越窄 | 可灵活配置 |

普通运放是"通用工具",而INA是专为"微弱差分信号采集"打磨的"精密手术刀".

## 2 三运放INA拓扑详解

### 2.1 电路结构

三运放拓扑是绝大多数集成INA的内部结构,由两级组成:

```
                +---[R1]---+---[R1]---+
                |          |          |
     Vin+ --+->[A1]--+    Rg    +--[A2]<---+-- Vin-
             |      |          |       |
             |      +---[R2]---+       |
             |            |            |
             |     +------+-------+    |
             |     |      R3      |    |
             |     |   +--[R3]--+ |    |
             |     |   |       |  |    |
             |     +---|->[A3]--|--+    |
             |         |   |    |      |
             |         |   v    |      |
             |         |  Vout  |      |
             |         +--------+      |
             +---------------------------(Ref)
```

- **第一级**(A1, A2):两个同相放大器,提供高输入阻抗和前级增益
- **第二级**(A3):差分放大器,将双端信号转为单端,提供最终CMRR

### 2.2 增益公式推导

Rg上的电流为 I_Rg = (V1 - V2) / Rg,同一电流流过R1,故:

```
Vout1 - Vout2 = (V1 - V2) x (1 + 2xR1 / Rg)
```

第二级差分增益为1(R2=R3时),总增益:

```
G = 1 + 2 x R1 / Rg
```

关键洞察:增益仅由Rg一个外部电阻决定,内部R1已在芯片中精密匹配.

### 2.3 为什么三运放拓扑CMRR高

CMRR在两级中逐级累积:第一级对共模信号增益为1(不被放大),对差分信号增益为(1 + 2xR1/Rg).若G1=100,则第一级贡献40dB的CMRR,加上第二级差分级的80-100dB,总CMRR可达120dB以上.这就是"先放大差分、不放大共模"的设计精髓.

## 3 CMRR: 共模抑制比

### 3.1 定义与意义

```
CMRR(dB) = 20 x log10(Ad / Acm)
```

实际场景:称重传感器输出10mV差分信号,叠加5V共模干扰.CMRR=100dB时,共模输出仅50uV,而差分输出1000mV(G=100),信噪比20000:1.CMRR每降低20dB,共模噪声贡献增大10倍,直接淹没微弱信号.

### 3.2 CMRR随频率衰减

数据手册给出的CMRR通常是DC值,随频率升高,内部寄生电容导致对称性下降:

| 频率 | 典型INA CMRR |
|------|-------------|
| DC | 100-120dB |
| 1kHz | 80-100dB |
| 10kHz | 60-80dB |
| 100kHz | 40-60dB |

工程启示:kHz以上干扰,不能只看DC的CMRR值,必须查CMRR-vs-Frequency曲线.

### 3.3 影响CMRR的外部因素

1. **源阻抗不平衡**:两条输入线阻抗不对称,等效注入差分误差.对策:保证阻抗对称.
2. **PCB漏电流**:基板漏电阻在高压共模下形成不对称泄漏.对策:保护环布线.
3. **参考引脚噪声**:Ref引脚噪声以增益1叠加到输出.对策:低阻抗驱动,通常用运放缓冲.

## 4 增益设定: 单电阻的智慧

### 4.1 外部电阻选型

增益公式: Rg = 2 x R1 / (G - 1).以INA128为例(R1=25kΩ):

| 目标增益G | Rg值 |
|----------|------|
| 1 | 不接(开路) |
| 10 | 5.56kΩ |
| 100 | 505Ω |
| 1000 | 50.1Ω |

### 4.2 电阻精度与温度系数

- G=100时,Rg误差1%引起约1%增益误差;精密测量用0.1%或更好的金属膜电阻
- 温度系数:普通电阻50-100ppm/°C,精密电阻<10ppm/°C
- 高增益(G>100)时Rg阻值很小,PCB走线电阻和焊锡电阻也引入误差,需考虑开尔文连接

### 4.3 增益与带宽的权衡

INA增益带宽积近似为常数:BW = GBW / G.以INA128(GBW约1.3MHz):

| 增益G | -3dB带宽 |
|-------|---------|
| 1 | 1.3MHz |
| 10 | 130kHz |
| 100 | 13kHz |
| 1000 | 1.3kHz |

设计时需确认:在所需增益下,信号带宽是否落在INA的可用带宽内.

## 5 输入阻抗优势

### 5.1 为什么高输入阻抗至关重要

传感器等效为高内阻电压源.测量电路输入阻抗与信号源内阻形成分压器:

```
V_measured = V_signal x Rin / (Rsource + Rin)
```

Rsource=1MΩ时,Rin=10MΩ损失约9%,Rin=1GΩ仅损失0.1%.pH电极(100MΩ)、压电传感器(GΩ级)等必须搭配极高输入阻抗.

### 5.2 INA的输入阻抗

三运放拓扑第一级为同相输入,输入阻抗由运放共模输入阻抗决定:

- 双极型输入INA(如INA128):约10GΩ
- FET输入INA(如INA111):约10^12Ω
- 自动归零INA(如INA333):约10^13Ω

该阻抗与Rg无关,因为Rg连接在两运放输出之间,不在信号输入通路上.

## 6 典型器件与应用选型

### 6.1 电流检测: INA219 / INA226

| 参数 | INA219 | INA226 |
|------|--------|--------|
| 共模范围 | 0-26V | 0-36V |
| ADC分辨率 | 12位 | 16位 |
| 精度 | 1% | 0.1% |
| 接口 | I2C | I2C |

高侧电流检测典型接法:

```
  V+ ────────[负载]──────── GND
               |
          [Rsense]
               |
     IN+ ─────┘    ┌───────┐
     IN- ──────────│INA226 │──> I2C to MCU
                   └───────┘
```

一般电源监测选INA219,精密电量计量选INA226.

### 6.2 桥式传感器: INA128 / INA333

| 参数 | INA128 | INA333 |
|------|--------|--------|
| 供电 | 双电源 +/-2.25V~+/-18V | 单电源 1.8V~5.5V |
| 输入类型 | 双极型 | 自动归零 |
| 失调电压(最大) | 50uV | 10uV |
| 失调漂移 | 0.5uV/°C | 0.05uV/°C |
| 带宽(G=100) | 13kHz | 550Hz |
| 静态电流 | 700uA | 17uA |

选型决策:需宽带宽选INA128,需极低失调漂移或低功耗选INA333.

### 6.3 选型速查表

| 应用场景 | 推荐器件 | 核心理由 |
|---------|---------|---------|
| 电池电量计 | INA226 | 16位精度,宽共模 |
| 称重传感器 | INA333 | 超低失调漂移 |
| ECG前端 | INA333 | 低功耗,高CMRR |
| 工业过程控制 | INA128 | 宽带宽,高电压 |
| 高温环境 | AD8422 | -40~125°C |

## 7 自动归零与斩波稳零INA

### 7.1 失调电压的困境

普通INA失调电压在几十uV量级且随温度漂移.对于uV级信号,失调漂移可能比信号本身还大.类比:用一把刻度随温度变化的尺子量头发丝粗细,尺子本身的不确定性比待测尺寸还大.

### 7.2 自动归零(Auto-Zero)原理

通过片内交替执行"校准"和"放大"两个阶段消除失调:

- 阶段A(校准):输入短接,测量并存储失调电压Vos
- 阶段B(放大):正常放大,从输出减去存储的Vos

kHz级频率交替,实现近零失调:INA333的Vos=10uV(max),漂移=0.05uV/°C.

### 7.3 斩波稳零(Chopper)原理

将输入信号调制到高频放大后再解调回基带,失调保持在DC不被调制, 解调时被滤除.

| 特性 | 自动归零 | 斩波稳零 |
|------|---------|---------|
| 失调电压 | <1uV | <1uV |
| 失调漂移 | <0.05uV/°C | <0.01uV/°C |
| 1/f噪声 | 中等 | 极低 |
| 带宽 | 较宽 | 较窄 |

### 7.4 选用建议

- DC/极低频信号(称重、温度桥):优先斩波稳零
- 兼顾带宽和低失调:选自动归零
- 对开关纹波敏感(如音频前端):慎重考虑,需加滤波

## 8 噪声分析

### 8.1 INA噪声的三个来源

1. **电压噪声(e_n)**:等效输入电压噪声密度,单位nV/sqrt(Hz)
2. **电流噪声(i_n)**:等效输入电流噪声密度,单位pA/sqrt(Hz)
3. **电阻热噪声**:增益设定电阻和源电阻的Johnson噪声

总等效输入噪声: Vn_total = sqrt( e_n^2 + (i_n x Rs)^2 + 4kTRs )

### 8.2 1/f噪声(闪烁噪声)

低频段噪声功率谱密度随频率降低而增大: e_n(f) = e_n_white x sqrt( 1 + f_corner / f ),f_corner通常在1-100Hz.

| 噪声类型 | 频率特征 | 影响场景 |
|---------|---------|---------|
| 白噪声 | 平坦 | 宽带测量 |
| 1/f噪声 | 低频增大 | DC/低频精密测量 |
| 采样噪声 | 自动归零/斩波特有 | 自动归零INA |

对策:DC/低频测量选斩波稳零抑制1/f噪声,宽带测量关注白噪声密度值.

### 8.3 噪声带宽积分

白噪声段RMS噪声: Vn_rms = e_n_white x sqrt(BW).

示例:INA128在G=100,e_n=10nV/sqrt(Hz),带宽13kHz:

```
Vn_rms = 10nV x sqrt(13000) = 1140nV = 1.14uV (折合输入)
输出端: 1.14uV x 100 = 114uV, 信号10mV时SNR约83dB
```

## 9 实际应用电路

### 9.1 惠斯通电桥读出

```
        Vexc
         |
    [R1]   [Rx]
     |       |
     +---V+--+--- V- ---+
     |       |          |
    [R2]   [R3]    [INA差分输入]
     |       |          |
     +-------+----------+--- GND
```

设计要点:
- 桥路输出: Vdiff = Vexc x dR / (4R)(小变化近似)
- INA增益设定使满量程输出接近ADC输入范围
- Ref引脚接Vexc/2(单电源),输出偏置在中间

称重传感器示例:桥路灵敏度2mV/V,激励5V,满量程输出10mV,ADC范围0-3.3V,所需增益330,INA128的Rg=152Ω.

### 9.2 ECG前端电路

ECG信号:0.5-5mV,0.05-150Hz,共模干扰高达300mV.

```
  右臂 ──[RA]──┐              ┌──────────┐
                │    ┌───────│  INA333  │──> 后级滤波/ADC
  左臂 ──[LA]──┼────┘       │  G=100   │
                │            │  Ref<─── │<── 右腿驱动
  右腿 ──[RL]──┘            └──────────┘
```

关键决策:
1. **INA选型**:INA333,低功耗(17uA),高CMRR(100dB),适合可穿戴
2. **右腿驱动(RLD)**:共模信号反相放大后驱动右腿电极,额外提升CMRR 20-40dB,但增益不宜过大(防振荡)
3. **高通滤波**:输入端加0.05Hz RC高通,去除电极偏移电压

### 9.3 称重传感器(Load Cell)接口

商用称重传感器通常120Ω或350Ω电桥,灵敏度2mV/V:

```
  +5V ──────────────────────────
         |                  |
     [E+]              [S+] [S-]
  Load Cell(6线)     远程采样端
     [E-]                  |
  GND ──┘            ┌─────┴─────┐
                      │  INA128   │──> 24bit ADC(HX711)
                      │  G=330    │
                      └───────────┘
```

6线制优势:S+/S-远程采样端补偿长导线电阻引起的激励电压衰减.

调试步骤:先不接传感器查零点 -> 接传感器无负载查偏移 -> 加载已知砝码校准增益 -> 不同温度查温漂.

## 10 PCB布局要点

### 10.1 保护环(Guard Ring)

INA输入端阻抗极高,PCB漏电流会引入误差.保护环将输入焊盘周围用等电位导电环包围,使漏电流流向保护环而非输入端:

```
  ┌──────────────────────────┐
  │        Guard Ring        │
  │   ┌────────────────┐    │
  │   │   IN+ pad      │    │
  │   │   IN- pad      │    │
  │   └────────────────┘    │
  │   (Guard接IN-同电位)     │
  └──────────────────────────┘
```

保护环应覆盖所有层(通孔处也要围住),用运放缓冲器驱动以确保低阻抗.

### 10.2 布局规则

| 规则 | 原因 |
|------|------|
| INA下方不走其他信号线 | 避免耦合噪声到高阻抗输入 |
| Rg尽量靠近Rg引脚 | 减少寄生电容和拾取噪声 |
| 输入走线平行等长 | 保证阻抗匹配,维持CMRR |
| 信号地与功率地单点连接 | 避免地线环路 |
| Ref引脚低阻抗驱动 | Ref噪声直接叠加输出 |
| 远离开关电源和数字信号 | 减少电磁耦合 |

### 10.3 退耦与接地

- 每个电源引脚旁放0.1uF陶瓷电容,尽量靠近引脚
- 增加1-10uF电容处理低频纹波
- 双电源时正负都要退耦
- 退耦电容接地端直连地平面,不经长走线

## 11 常见错误与排查

### 11.1 参考引脚(Ref)处理不当

Vout = G x (Vin+ - Vin-) + Vref.Ref不是简单"接地":

| 错误做法 | 后果 | 正确做法 |
|---------|------|---------|
| Ref直接接PCB地平面 | 地噪声耦合到输出 | 通过0.1uF旁路+电阻接地 |
| Ref悬空 | 输出不确定 | 必须接低阻抗驱动源 |
| Ref通过长走线连接 | 拾取噪声 | 就近用运放缓冲器驱动 |
| 单电源Ref接地 | 负信号被裁剪 | Ref接Vcc/2,输出偏置到中间 |

单电源正确接法:R1=R2分压到Vcc/2 -> 0.1uF旁路 -> 运放跟随器缓冲 -> INA Ref.

### 11.2 输入端过压

INA输入电压范围受供电限制,超出可能导致相位反转、保护二极管导通甚至损坏.排查:示波器检查输入瞬态,串联1-10kΩ限流电阻,加TVS二极管保护.

### 11.3 其他常见问题

1. **增益公式用错**:不同型号R1不同,INA128的R1=25kΩ,INA129的R1=12.5kΩ,必须查数据手册.
2. **忽略偏置电流回路**:FET输入INA仍需DC回路,输入完全浮空(如接变压器次级)时输出会漂移到电源轨.解决:两输入端之间接10MΩ高阻电阻.
3. **电阻匹配影响CMRR**:0.1%匹配CMRR约54dB,0.01%约74dB.要更高CMRR必须用集成INA.
4. **屏蔽电缆驱动**:长电缆电容降低CMRR,用INA共模电压驱动电缆屏蔽层消除共模-差模转换.

## 总结

仪表放大器的核心价值在于三个不可替代的特性:极高且对称的输入阻抗、单电阻设定增益的便利性、远超普通运放的共模抑制能力.

核心设计要点:

1. **选型**:根据信号特征(带宽、幅度、源阻抗)和应用约束(功耗、电压、温度)选择型号
2. **增益设定**:Rg精度决定增益精度,高增益时注意PCB走线电阻
3. **CMRR保障**:源阻抗对称、保护环布线、Ref低阻抗驱动三者缺一不可
4. **噪声控制**:DC/低频选斩波稳零,宽带选低e_n器件,按实际带宽计算总噪声
5. **PCB布局**:输入走线平行等长,Rg靠近引脚,保护环环绕高阻抗节点

掌握这些要点后,面对微弱信号测量任务时,就能从容地选型、设计和调试INA电路,而不是在噪声和漂移中反复挣扎.
