# 光纤传感器在工业IoT温度/应变测量中的应用
> **难度**：🔴 高级 | **领域**：光纤传感技术 | **阅读时间**：约 22 分钟

## 引言

想象一条高速公路,它不仅是交通通道,还能通过路面微小的振动感知每一辆车的位置和速度。光纤传感器就像这样的"感知高速公路":一根光纤不仅能传输数据,还能沿途感知温度和应变的变化。更妙的是,它不怕电磁干扰,不会产生电火花,可以在高压变压器旁、油井深处、化工厂里安全工作。

## 1 光纤传感的基本原理

### 1.1 光作为传感介质

光在光纤中传播时,以下属性会受外部物理量影响:

| 光的属性 | 受影响的物理量 | 传感类型 |
|----------|---------------|----------|
| 强度 | 微弯、位移 | 强度调制型 |
| 相位 | 应变、温度 | 干涉型 |
| 波长 | 应变、温度 | FBG 型 |
| 偏振 | 磁场、电流 | 偏振型 |
| 散射 | 温度、应变 | 分布式型 |

### 1.2 光纤传感的分类

按测量方式分为两大类:

```
光纤传感器
  |
  +-- 点式传感器: 在特定位置测量
  |     +-- FBG (Fiber Bragg Grating)
  |     +-- 法布里-珀罗干涉仪
  |     +-- 光纤光微弯
  |
  +-- 分布式传感器: 沿光纤全长连续测量
        +-- 拉曼散射 (温度)
        +-- 布里渊散射 (温度 + 应变)
        +-- 瑞利散射 (高分辨率应变)
```

## 2 FBG 光纤光栅传感器

### 2.1 FBG 的结构

FBG 是在光纤芯中写入周期性的折射率变化:

```
光纤芯:  ||||  ||||  ||||  ||||  ||||
         周期 Lambda_B (约 0.5 um)

入射光:  宽谱光 ---------->
反射光:  <---- 波长 = Lambda_B 的窄带光
透射光:  ---------> 去掉 Lambda_B 的宽谱光
```

满足布拉格条件的波长被反射:

```
Lambda_B = 2 * n_eff * Lambda

n_eff: 光纤有效折射率 (~1.46)
Lambda: 光栅周期 (~0.5 um)
Lambda_B: 布拉格波长 (~1550 nm 常见)
```

### 2.2 FBG 的传感原理

当温度或应变变化时,布拉格波长发生偏移:

```
delta_Lambda_B / Lambda_B = (1 - Pe) * epsilon + (alpha + xi) * delta_T

Pe: 光弹系数 (~0.22 for silica)
epsilon: 应变
alpha: 热膨胀系数 (~0.55e-6 /C for silica)
xi: 热光系数 (~6.67e-6 /C for Ge-doped silica)
delta_T: 温度变化
```

典型灵敏度:

- 应变: ~1.2 pm/micro-strain (1550 nm 处)
- 温度: ~10 pm/C (1550 nm 处)

### 2.3 FBG 的复用

FBG 的最大优势之一是可以在同一根光纤上写入多个光栅:

```
光源 ---> [FBG1: 1540nm] --- [FBG2: 1545nm] --- [FBG3: 1550nm] --->

每个 FBG 反射不同波长,互不干扰
一根光纤可复用 10-100+ 个 FBG 传感器
```

波分复用(WDM)是在同一根光纤上放置不同中心波长的 FBG。还可以结合空分复用(SDM)扩展到数百个传感器。

### 2.4 FBG 解调(Interrogation)

解调的核心任务是精确测量布拉格波长的偏移:

| 方法 | 原理 | 精度 | 成本 |
|------|------|------|------|
| 宽谱光源+光谱仪 | ASE光源,OSA测反射峰 | ~1 pm | 中 |
| 可调谐激光扫描 | 扫描波长,记反射峰值 | ~0.1 pm | 高 |
| 边缘滤波法 | 双探测器比值测偏移 | ~5 pm | 低 |

## 3 分布式光纤传感

### 3.1 原理: OTDR 技术

光时域反射仪(OTDR)是分布式传感的基础:

```
脉冲光注入光纤 --->
        |
        v
沿途每一点都有后向散射光 <---
        |
        v
测量散射光的时间和强度:
  时间 -> 距离 (d = c*t / 2n)
  强度 -> 物理量 (温度/应变)
```

### 3.2 拉曼散射: 温度测量

拉曼散射光的反斯托克斯分量强度对温度敏感:

```
拉曼反斯托克斯/斯托克斯强度比:
R(T) = (I_AS / I_S) = C * exp(-hc*delta_v / kT)

delta_v: 拉曼频移 (~13 THz for silica)
T: 绝对温度
```

特点:

- 只测温度,不受应变影响
- 空间分辨率: 0.5-2 m
- 测温精度: +/- 0.5-1 C
- 测量距离: 30-50 km(单模光纤)
- 测量时间: 几秒到几分钟

### 3.3 布里渊散射: 温度 + 应变

布里渊频移与温度和应变都相关:

```
delta_v_B = C_epsilon * epsilon + C_T * delta_T

C_epsilon: ~0.05 MHz/micro-strain
C_T: ~1.1 MHz/C
```

布里渊系统的特点:

- 同时测量温度和应变(需要解耦)
- 空间分辨率: 0.5-5 m
- 应变精度: +/- 20 micro-strain
- 温度精度: +/- 0.5 C
- 测量距离: 50-100 km

温度-应变交叉敏感的解决:

- 使用两根光纤: 一根自由(测温度+应变), 一根松套管内(只测温度)
- 或使用不同光纤参数的布里渊频移差异

### 3.4 三种分布式技术对比

| 参数 | 拉曼 | 布里渊 | 瑞利(OFDR) |
|------|------|--------|-----------|
| 测量量 | 温度 | 温度+应变 | 应变+温度 |
| 空间分辨率 | 0.5-2 m | 0.5-5 m | 1-10 mm |
| 测量距离 | 30-50 km | 50-100 km | < 100 m |
| 测量时间 | 秒-分 | 秒-分 | 秒 |
| 典型应用 | 电缆/管道温度 | 管道/隧道 | 复合材料 |

## 4 光纤传感器在工业IoT中的优势

### 4.1 抗电磁干扰

光纤由石英玻璃制成,不导电、不产生电磁场,可在变电站、电解铝厂、雷达站等强电磁环境正常工作。

### 4.2 本质安全

光纤中没有电流,不会产生电火花,可直接用于油气储罐、煤矿井下、化工厂等易燃易爆环境。

### 4.3 多点复用与长距离

一根光纤可挂 10-100+ 个 FBG 传感器,分布式传感沿光纤每米一个测点。光纤衰减仅 0.2 dB/km,分布式传感可达 50-100 km,无需中继。

### 4.4 恶劣环境

特种光纤可耐 700C+ 高温,石英玻璃耐腐蚀,抗辐射光纤可用于核电。

## 5 应用场景

### 5.1 结构健康监测

桥梁、建筑、大坝的应变监测: FBG 传感器贴在关键截面,测量应变变化,实现荷载监测、疲劳累计和损伤预警。一个桥梁可能需要 50-200 个 FBG。

### 5.2 油气行业

分布式温度传感(DTS)沿井筒连续测温,用于发现气水界面变化、监测蒸汽驱进程(SAGD)和获取井筒温度剖面。

### 5.3 电力行业

FBG 埋入变压器绕组实时监测热点温度;DTS 沿电缆沟敷设发现局部过热,实现载流量动态评估。

### 5.4 复合材料

FBG 嵌入碳纤维复合材料,实现固化过程监测(温度+应变)、使用中载荷监测和损伤检测(冲击、脱层)。

## 6 IoT 集成架构

### 6.1 系统层级

```
光纤传感器(FBG/分布式)
    |
    v
解调仪/OTDR(光电转换 + 波长/频率分析)
    |
    v
嵌入式处理器(数据预处理、特征提取)
    |
    v
通信网关(Ethernet/4G/NB-IoT/LoRa)
    |
    v
云平台(数据存储、趋势分析、报警)
    |
    v
用户界面(Web/App/告警推送)
```

### 6.2 数据量考量

不同类型传感器的数据量:

| 传感器类型 | 数据速率 | 传输方式 |
|-----------|----------|----------|
| 单个 FBG | ~10 S/s | 任何方式 |
| 100 个 FBG | ~1 KB/s | Ethernet |
| 分布式(DTS) | ~100 KB-1 MB/次 | Ethernet/4G |
| 高速动态 FBG | ~100 KB/s | Ethernet |

大多数结构健康监测场景(分钟级采样),LoRa 或 NB-IoT 即可满足传输需求。

## 7 实战案例: 风电叶片监测

### 7.1 需求

- 风电叶片疲劳损伤早期检测
- 叶片长度: 50-80 m
- 监测: 根部、中部、尖部的应变
- 环境: 雷击高发区,电磁干扰强
- 通信: 舱内 Ethernet 到塔基, 4G 上云

### 7.2 系统设计

```
叶片内嵌入 FBG 应变传感器
  - 每个叶片: 10-20 个 FBG
  - 3 个叶片: 30-60 个 FBG
  - 分 3 根光纤(每叶片 1 根)
  
机舱内安装 FBG 解调仪
  - 波长范围: 1520-1570 nm
  - 采样率: 100 S/s(动态应变)
  - 精度: 1 micro-strain
  
通过 Ethernet 传到塔基控制柜
  - 特征提取: 峰值应变、频率、阻尼比
  
4G 上传到云平台
  - 趋势分析: 应变幅值变化 -> 疲劳累计
  - 异常检测: 突变大应变 -> 可能损伤
  - 报警: 超阈值推送运维人员
```

### 7.3 监测效果

- 正常运行: 应变幅值稳定,频率与转速一致
- 前缘开裂: 应变幅值突增,出现额外频率分量
- 根部疲劳: 应变均值缓慢偏移,阻尼比变化

## 8 挑战

### 8.1 解调仪成本

FBG 解调仪价格:

| 类型 | 精度 | 价格 |
|------|------|------|
| 基于光谱仪 | ~1 pm | $5K-$15K |
| 可调谐激光 | ~0.1 pm | $15K-$50K |
| 边缘滤波 | ~5 pm | $2K-$5K |

分布式 OTDR 更贵: $20K-$80K。

这是光纤传感推广的主要障碍。

### 8.2 光纤脆弱性与交叉敏感

裸光纤极易折断(断裂应变 ~1%),需保护套管或铠装。FBG 波长偏移同时受温度和应变影响,解决方案:双 FBG(一个自由,一个松套管只测温度)或温度补偿封装。

### 8.3 连接器损耗

FC/APC ~0.3 dB, SC/APC ~0.2 dB, 熔接 ~0.05 dB。多个连接器累积损耗会降低信噪比,户外安装需防水防尘。

## 9 成本趋势

随着光通信组件规模化生产,可调谐激光器从 $5K 降至 $500,光电探测器阵列从 $1K 降至 $100。硅光子学(SiPh)集成解调仪正在兴起,目标 $100-$500 级别的解调模块。预计 5 年内入门级 FBG 解调仪可降至 $1K-$2K,推动光纤传感从专业检测走向常规 IoT 监测。

## 总结

光纤传感器为工业 IoT 提供了电传感器无法替代的能力:

1. FBG 适合多点离散测量,可在一根光纤上复用数十到上百个传感器
2. 分布式传感(DTS/DSS)沿光纤全长连续测量,覆盖数公里
3. 抗 EMI、本质安全、耐高温等特性适合恶劣工业环境
4. 解调仪成本是推广的主要障碍,但正在快速下降
5. 芯片级解调技术将推动光纤传感进入更广泛的 IoT 应用

从桥梁健康到油井温度,从变压器监测到风电叶片,光纤传感正在成为工业 IoT 的重要感知手段。
