# 灌封材料在传感器防护中的选型与工艺
> **难度**：🟡 中级 | **领域**：封装防护工艺 | **阅读时间**：约 20 分钟

## 引言

想象你把一部手机丢进水里——屏幕进水、电路短路、几天就报废。但如果把手机整个封进一块透明的树脂里，水就再也进不去了。灌封(Encapsulation/Potting)就是给电子元器件穿上这样一层"密封铠甲"，把整个电路板或传感器模块包裹在固态材料内部。和三防漆只在表面涂一层薄膜不同，灌封是彻底填满整个腔体，防护等级更高，代价也更重。

## 1. 灌封与敷形涂层的区别

### 1.1 防护原理对比

| 对比项 | 灌封(Potting) | 敷形涂层(Conformal Coating) |
|--------|---------------|---------------------------|
| 覆盖方式 | 填满整个外壳腔体 | 仅在PCB表面涂覆薄膜 |
| 典型厚度 | 数毫米至数十毫米 | 25-75um |
| 防水等级 | 可达IP67-IP68 | 通常IP54-IP65 |
| 减震能力 | 优秀(材料吸收振动能量) | 几乎没有 |
| 抗冲击 | 优秀(整体包裹) | 一般 |
| 返修难度 | 极高(需破坏性去除) | 可返修(溶剂溶解) |
| 热管理 | 需考虑散热路径 | 影响很小 |
| 成本 | 较高(材料+工艺) | 较低 |

### 1.2 何时选择灌封

满足以下任一条件应优先考虑灌封：

- 设备需要长期浸水或处于极高湿度环境
- 存在强烈振动或冲击(如电机安装面、车载场景)
- 需要防止物理篡改或逆向工程
- 化学腐蚀环境(酸碱气体、溶剂蒸气)

敷形涂层适合中等防护需求，灌封适合极端环境。

## 2. 灌封材料类型与特性

### 2.1 三大主流材料

| 特性 | 环氧树脂(Epoxy) | 聚氨酯(PU) | 有机硅(Silicone) |
|------|-----------------|------------|-----------------|
| 硬度 | 硬(HV80-90) | 中等(HV40-60) | 软(邵A20-50) |
| 耐化学性 | 优异 | 中等 | 良好 |
| 附着力 | 极高 | 良好 | 一般 |
| 温度范围 | -40C至+130C | -50C至+125C | -60C至+200C |
| 柔韧性 | 差(脆性) | 良好 | 优异 |
| 返修性 | 几乎不可能 | 困难 | 容易(可切割) |
| 介电强度 | >15 kV/mm | >15 kV/mm | >20 kV/mm |
| 固化收缩率 | 1-3% | 0.5-2% | <0.5% |
| 成本 | 低 | 中 | 高 |

### 2.2 环氧树脂详解

环氧是最"硬核"的选择，固化后形成刚性固体：

- **优点**：耐化学腐蚀一流，对金属和PCB附着力极强，介电性能好
- **缺点**：脆性大，温度冲击下容易开裂；固化放热量大，可能损伤热敏感元件；一旦灌入几乎无法返修
- **适用**：水下传感器、化学暴露环境、一次性产品(无需维修)

### 2.3 聚氨酯详解

聚氨酯在刚性和柔性之间取得平衡：

- **优点**：柔韧性好，能吸收振动；低温性能优于环氧；耐磨损
- **缺点**：对某些溶剂和酸碱的耐受性不如环氧；固化受湿度影响(湿气固化型)；长期紫外线下可能黄变
- **适用**：电机振动传感器、车载设备、中等化学暴露场景

### 2.4 有机硅详解

有机硅是"温度之王"：

- **优点**：-60C到+200C的工作范围无出其右；极柔软，对元器件应力最小；固化收缩率极低；透明品种可目视检查内部
- **缺点**：价格是环氧的2-3倍；附着力一般，可能需要底涂(Primer)；某些品种表面有硅油渗出
- **适用**：极端温度环境、需要返修的产品、精密传感器(应力敏感)

## 3. 选型准则

### 3.1 选型决策矩阵

按以下优先级逐一判断：

1. **工作温度范围**：超出材料范围直接淘汰
2. **柔韧性需求**：有振动/冲击/热循环必须选柔性材料
3. **化学暴露类型**：匹配耐化学性数据表
4. **导热需求**：大功率器件需选导热型灌封料
5. **返修可能性**：需返修则排除环氧
6. **成本约束**：批量产品考虑材料单价和工艺成本

### 3.2 关键参数解读

**导热系数**：普通灌封料0.2-0.4 W/mK，导热型0.5-3 W/mK。填充氧化铝(Al2O3)可达1-2 W/mK，氮化硼(BN)可达3 W/mK以上。注意：导热填料会提高粘度，影响工艺性。

**介电强度**：所有三类材料都满足常规绝缘需求(>15 kV/mm)，高压场合需核实具体数值。

**固化方式**：
- 室温固化(RTV)：操作简单，固化时间长(数小时至24小时)
- 加热固化：速度快(30-120分钟)，但需烤箱，热敏感元件需注意
- UV固化：秒级固化，但仅适用于透明外壳或浅层灌封

## 4. 热管理考量

### 4.1 灌封对散热的影响

灌封材料的热导率远低于金属和陶瓷，可能成为散热瓶颈：

```
芯片结温 = 环境温度 + 功耗 x (Rth_灌封 + Rth_外壳 + Rth_对流)
```

普通灌封料的热阻可能比空气间隙还高，因为灌封料消除了空气对流。

### 4.2 导热灌封方案

| 方案 | 导热系数 | 成本增量 | 适用功率等级 |
|------|----------|----------|-------------|
| 普通灌封(无填料) | 0.2-0.4 W/mK | 基准 | <0.5W |
| 氧化铝填料 | 0.8-2.0 W/mK | +30-50% | 0.5-3W |
| 氮化硼填料 | 2.0-3.5 W/mK | +100-200% | 3-10W |
| 金属基灌封(特殊) | 3-5 W/mK | +200%以上 | >10W |

### 4.3 热设计建议

- 计算灌封层热阻：Rth = 灌封厚度 / (导热系数 x 散热面积)
- 将高发热器件尽量靠近外壳壁，缩短热传导路径
- 在灌封料中预埋金属散热片或导热铜管
- 热敏感传感器远离高发热器件

## 5. 灌封工艺流程

### 5.1 完整工艺步骤

```
PCB清洗/干燥 --> 遮蔽(保护不灌封区域) --> 称量配比
    --> 混合搅拌 --> 真空脱泡 --> 灌注 --> 固化 --> 去遮蔽 --> 检验
```

### 5.2 双组份混合要点

环氧和聚氨酯通常是双组份(A组分+B组分)：

- **混合比例**：必须严格按供应商规定的重量比，偏差不超过2%
- **搅拌**：沿同一方向缓慢搅拌2-3分钟，避免卷入气泡
- **适用期(Pot Life)**：混合后可操作的时间窗口，环氧通常20-60分钟，需在此时间内完成灌注

### 5.3 真空脱泡

气泡是灌封的大敌，会导致：

- 局部绝缘强度下降
- 水汽渗透通道
- 热阻增大(空气是热的不良导体)

脱泡参数参考：

| 材料粘度 | 真空度 | 脱泡时间 |
|----------|--------|----------|
| 低粘度(<1000 cP) | -0.08 MPa | 5-10 min |
| 中粘度(1000-5000 cP) | -0.09 MPa | 10-15 min |
| 高粘度(>5000 cP) | -0.095 MPa | 15-20 min |

### 5.4 固化参数

| 固化方式 | 典型条件 | 注意事项 |
|----------|----------|----------|
| 室温固化 | 25C/24-48h | 环境湿度影响PU固化速度 |
| 中温固化 | 60-80C/2-4h | 避免升温过快导致气泡膨胀 |
| 高温固化 | 100-120C/30-60min | 确认元器件耐温等级 |
| 阶梯固化 | 25C/2h + 60C/2h + 80C/1h | 减少内应力，推荐用于大型灌封体 |

## 6. 灌封设计要点

### 6.1 Dam-and-Fill技术

对于只需要局部灌封的场景，使用"围坝-填充"法：

1. 先用高粘度材料在目标区域周围筑一圈"坝"
2. 待坝固化后，向围合区域内注入低粘度灌封料
3. 优点：节省材料，不影响其他区域

### 6.2 必须暴露的区域

灌封时需要遮蔽保护的部位：

- **连接器插针**：灌封料不可进入插接面
- **传感器敏感面**：如MEMS麦克声孔、温湿度传感器窗口
- **天线区域**：灌封料介电常数改变天线特性
- **LED出光面**：灌封料可能影响透光率
- **散热器表面**：需要与空气或冷板直接接触

### 6.3 应力释放设计

线缆入口是灌封失效的高发区：

- 在线缆入口处使用柔性灌封料或硅胶密封圈
- 灌封料与线缆绝缘层之间需有良好附着力
- 线缆在灌封体内应有弯曲余量，避免直角拐弯
- 多芯线缆应分散进入，避免集中一点

### 6.4 避免气阱

气阱(Air Trap)是灌封体内部的空洞：

- 灌注口设在最低点，让灌封料从下往上填充
- 在最高点设置排气孔
- 复杂形状模具需分析填充流动路径
- 分层灌注可减少气阱风险

## 7. 实际案例

### 7.1 水下IoT传感器——环氧灌封实现IP68

**需求**：水深10米长期工作，检测水温和溶解氧

**方案**：
- 外壳：不锈钢密封壳体
- 灌封：双组份环氧，导热系数0.8 W/mK
- 工艺：真空脱泡后灌注，60C固化4小时
- 关键：传感器探头透过壳体密封引出，灌封料包裹所有PCB和焊点
- 检验：0.5 MPa气密性测试通过

### 7.2 电机振动传感器——聚氨酯灌封抗振

**需求**：安装在电机端盖，持续振动20-2000Hz，温升最高85C

**方案**：
- 灌封：柔性聚氨酯，邵A硬度45
- 工艺：Dam-and-Fill局部灌封，保留MEMS传感器顶部暴露
- 关键：选择低模量PU以减少振动传递到MEMS芯片
- 验证：正弦扫频振动测试，共振点处无脱层

### 7.3 户外接线盒——有机硅灌封耐温度循环

**需求**：户外屋顶安装，夏季壳体温度可达95C，冬季-30C，日温差大

**方案**：
- 灌封：有机硅，工作温度-60C至+200C
- 工艺：整板灌封，连接器使用硅胶密封圈
- 关键：有机硅低模量吸收温度循环热应力，CTE匹配性好
- 验证：-40C至+125C温度循环100次无开裂

## 8. 质量控制

### 8.1 检验方法

| 检验项 | 方法 | 判据 |
|--------|------|------|
| 内部空洞 | X射线检测 | 单个空洞面积<总面积2% |
| 附着力 | 拉拔试验(参照IPC) | 附着力>1.5 MPa |
| 耐温循环 | -40C至+125C循环 | 100次循环无开裂脱层 |
| 介电强度 | 耐压测试 | >规定电压1分钟无击穿 |
| 防水等级 | IP测试(浸水/喷淋) | 符合目标IP等级 |

### 8.2 常见缺陷与对策

| 缺陷 | 可能原因 | 对策 |
|------|----------|------|
| 内部气泡 | 脱泡不充分/灌注过快 | 延长脱泡时间，降低灌注速度 |
| 表面裂纹 | 固化过快/温度冲击 | 阶梯固化，缓慢升降温 |
| 与PCB脱层 | 表面清洁不良 | 灌封前清洗+底涂处理 |
| 灌封料变色 | 紫外线老化/热老化 | 加UV稳定剂，确认耐温等级 |
| 连接器堵塞 | 遮蔽不良 | 改进遮蔽工装，灌注后立即检查 |

## 9. 常见失效与设计陷阱

### 9.1 CTE失配应力

热膨胀系数(CTE)失配是灌封失效的首要原因：

- 环氧CTE：40-60 ppm/C
- PCB(FR4)CTE：14-17 ppm/C
- 陶瓷芯片CTE：5-7 ppm/C

温度变化时，灌封料的膨胀/收缩量远大于PCB和元器件，产生的应力可能导致焊点断裂或芯片开裂。

**对策**：选择低CTE灌封料(添加硅微粉填料)，或选用柔性材料吸收应力。

### 9.2 嵌入式传感器释气

某些灌封料在固化过程中或长期使用中会释放小分子(如硅氧烷、胺类)：

- 硅氧烷可能毒化催化式气体传感器
- 胺类可能腐蚀敏感电极
- 挥发物可能在光学窗口上冷凝

**对策**：选择低释气等级材料，对关键传感器设置隔离腔体，或使用金属密封将传感器与灌封体隔开。

### 9.3 返修困难

灌封后的返修几乎是破坏性的：

- 环氧：只能机械铣削去除，PCB通常报废
- 聚氨酯：可用专用剥离剂，但耗时且可能损伤元器件
- 有机硅：可手工切割，相对容易

**设计建议**：对于可能需要维修的产品，将核心控制板设计为可插拔模块，仅对传感器前端做灌封。

## 总结

灌封与敷形涂层不是二选一的关系——在极端环境中，两者可以叠加使用(先涂三防漆再灌封)，实现双重防护。但对于大多数IoT传感器，选择一种即可，关键是匹配环境需求。

灌封是IoT传感器极端环境防护的终极手段，比敷形涂层提供更全面的防水、抗振和化学防护。三大材料各有定位：环氧适合化学腐蚀和水下场景，聚氨酯适合振动环境，有机硅适合宽温度范围和需要返修的场景。选型时优先匹配温度和柔韧性需求，工艺上严格控制配比、脱泡和固化参数，设计中注意CTE失配、释气和应力释放，才能实现可靠的灌封防护。
