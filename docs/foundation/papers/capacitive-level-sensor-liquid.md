# 电容式液位传感器原理与防腐设计
> **难度**：🟢 初级 | **领域**：液位测量 | **阅读时间**：约 18 分钟

## 引言

你有没有试过往玻璃杯里倒水，不用看，只听声音就能判断快满了？声音变调是因为杯子上方空气柱变短了。电容式液位传感器用的不是声音，而是"电容"——两个极板之间的"蓄电能力"。液体和空气的"蓄电能力"不同（介电常数不同），当液面上升，液体逐渐替代极板间的空气，电容就跟着变。测量这个电容变化，就知道液位多高了。

## 1 电容式液位测量原理

### 1.1 基本电容公式

电容的基本公式：

```
C = epsilon_r * epsilon_0 * A / d
```

其中：

- C：电容值（法拉）
- epsilon_r：极板间介质的相对介电常数
- epsilon_0：真空介电常数，约 8.854 x 10^(-12) F/m
- A：极板正对面积（平方米）
- d：极板间距（米）

液位测量的核心思路：当液面从 0 上升到满，极板间的介质从"全是空气"（epsilon_r 约 1）逐渐变成"全是液体"（水的 epsilon_r 约 80），电容值随之线性增大。

### 1.2 液位与电容的关系

设极板总长度为 L，液位高度为 h，液体介电常数为 epsilon_L：

```
C_total = C_air_part + C_liquid_part
        = epsilon_0 * A * (L - h) / (d * L) + epsilon_0 * epsilon_L * A * h / (d * L)
        = epsilon_0 * A / d * (1 + (epsilon_L - 1) * h / L)
```

h 从 0 到 L，C_total 线性增长。测量 C_total 就能算出 h/L，即液位百分比。

### 1.3 介质的影响

常见液体的相对介电常数：

| 液体 | epsilon_r | 说明 |
|------|-----------|------|
| 空气 | 1.0 | 基准 |
| 汽油 | 2.0-2.2 | 变化小，需要高分辨率 |
| 柴油 | 2.1 | 与汽油接近 |
| 乙醇 | 24 | 变化明显 |
| 甲醇 | 33 | 容易检测 |
| 水 | 80 | 变化最大，最易检测 |

水基液体的 epsilon_r 大，电容变化大，检测容易。油类 epsilon_r 小，需要更高分辨率的测量电路。

## 2 电极结构

### 2.1 棒式电极

最简单的结构：一根金属棒插入储罐中，储罐壁（如果是金属）作为另一个极板。

```
    ┌───────────┐
    │  液体      │ ← 金属罐壁（第二极板）
    │   ┌─┐     │
    │   │ │     │
    │   │ │ ←───│── 金属棒（第一极板）
    │   └─┘     │
    └───────────┘
```

适用场景：金属罐体，液体不导电，安装方便。

### 2.2 板式电极

两块平行板安装在储罐外壁，透过非金属罐壁检测液位。

```
    ┌───────────┐
    │  液体      │ ← 非金属罐壁（塑料/玻璃）
    ├═══════════┤ ← 外壁安装电极板
    │           │
    └───────────┘
```

核心优势：完全不接触液体，没有腐蚀和污染问题。

### 2.3 同轴式电极

中心金属棒 + 外层金属管，两者之间是测量区域。

```
    ┌───┐
    │ ┌─│─── 外管（第二极板）
    │ │ │
    │ │ │←─ 液体进入环形空间
    │ │ │
    │ └─│─── 中心棒（第一极板）
    └───┘
```

优势：电场集中在内外电极之间，不受外界干扰，适合小口径安装。

## 3 非接触式测量

### 3.1 透过罐壁检测

当储罐是非金属材料（HDPE、PP、玻璃）时，电极可以完全安装在罐外：

- 罐壁材料成为电容介质的一部分
- 罐壁厚度增加相当于增大了极板间距
- 需要校准补偿罐壁带来的固定电容

### 3.2 非接触式的优势

- 无湿件（wetted parts）：液体不接触传感器
- 不受液体腐蚀影响
- 安装维护不需要清空储罐
- 食品/制药行业首选方案

### 3.3 局限

- 仅适用于非金属罐壁
- 罐壁厚度变化会引入误差
- 罐壁上的冷凝水或结垢影响测量
- 需要比接触式更高的测量分辨率

## 4 信号调理电路

### 4.1 电容-数字转换器（CDC）

专用 IC 直接将电容转换为数字值：

| 型号 | 通道数 | 分辨率 | 接口 | 特点 |
|------|--------|--------|------|------|
| FDC1004 | 4 | 24bit | I2C | TI，内置屏蔽驱动 |
| AD7746 | 2 | 24bit | I2C | ADI，高精度 |
| PCAL8523 | 1 | 12bit | I2C | NXP，低成本 |
| CY8CMBR3110 | 10 | - | I2C | Infineon，CapSense |

FDC1004 是 IoT 液位设计的热门选择，内置屏蔽驱动器特别适合液位应用中的寄生电容抑制。

### 4.2 振荡器频率偏移法

将传感器电容作为 LC 振荡器的 C，液位变化改变振荡频率，MCU 用定时器测频：

```c
// 频率测量
uint32_t freq = MeasureFrequency();  // MCU定时器计数

// 查表法：频率->液位
float level = LookupTable(freq);

// 或公式法（线性关系时）
// f = 1 / (2*PI*sqrt(L*C))
// C 变化 -> f 变化 -> 液位
```

优势：电路简单，无需专用 CDC IC。劣势：非线性、受温度影响大。

### 4.3 屏蔽驱动

长电缆连接传感器时，电缆电容会严重影响测量。使用屏蔽驱动技术：

- 电缆屏蔽层不是接地，而是驱动到与中心导体相同的电位
- 屏蔽层和中心导体之间电位差为零
- 电缆电容不产生充放电电流，等效于"消失"
- FDC1004 内置此功能

## 5 材料选择与防腐设计

### 5.1 电极材料

| 材料 | 耐腐蚀性 | 成本 | 适用场景 |
|------|---------|------|---------|
| 不锈钢 316L | 良好 | 中 | 食品、一般化学 |
| 哈氏合金 C276 | 极好 | 高 | 强酸、强氧化 |
| PTFE 包覆 | 极好 | 中 | 几乎所有化学介质 |
| 陶瓷氧化铝 | 极好 | 高 | 高温强腐蚀 |

### 5.2 PTFE 包覆设计

PTFE（聚四氟乙烯，特氟龙）几乎不与任何化学物质反应：

- 将金属电极用 PTFE 管或涂层完全包裹
- PTFE 同时作为绝缘层和防腐层
- 对于导电液体，PTFE 绝缘层防止极间短路
- PTFE 介电常数约 2.1，对测量影响可预测

### 5.3 密封设计

传感器顶部（电气连接处）是最脆弱的部位：

- 环氧树脂灌封：低成本，但不耐强溶剂
- 玻璃烧结密封：高可靠性，用于苛刻环境
- O 型圈密封：可拆卸维护，选对材质（Viton/NBR/EPDM）

## 6 常见挑战与解决方案

### 6.1 泡沫问题

泡沫的介电常数介于空气和液体之间，可能导致：

- 测量值偏高（泡沫被当作液体）
- 读数抖动（泡沫时有时无）

解决方案：

- 延时滤波：取一段时间内的稳定值
- 导波管：将测量区域与主液面隔开，泡沫进不来
- 结合温度和流量数据做智能判断

### 6.2 导电液体

水、酸碱溶液等导电液体会"短路"裸露电极：

- 使用绝缘电极（PTFE 包覆）
- 绝缘层厚度均匀，电容变化仍然反映液位
- 注意绝缘层破损会导致测量失效

### 6.3 乳化液/混合介质

油水混合时介电常数不确定：

- 分层时可以分别检测油位和水位
- 乳化时读数不稳定
- 需要结合密度或其他辅助测量

### 6.4 温度影响

液体介电常数随温度变化：

- 水的 epsilon_r 在 0 度约 88，100 度约 55
- 需要温度补偿
- 在传感器附近安装温度探头，MCU 查表修正

### 6.5 结垢与沉积

电极表面结垢等效于改变介电常数：

- 定期清洗（如果可接触）
- 选用光滑表面材料（PTFE、抛光不锈钢）
- 自清洁设计：超声波振动或加热除垢

## 7 实战示例

### 7.1 农业灌溉水箱液位

需求：HDPE 水箱，500L，需要远程监控水量。

方案：

```
[水箱外壁] --- [FDC1004 + 板式电极] --- [ESP32] --- [WiFi] --- [云平台]
```

- 板式电极贴在水箱外壁
- FDC1004 检测电容变化
- ESP32 读取数据并 WiFi 上传
- 云平台显示水量，低于 20% 报警

校准步骤：

1. 空箱时记录 FDC1004 读数作为零点
2. 注满水后记录读数作为满点
3. 线性插值计算中间液位

### 7.2 化工储罐液位

需求：盐酸储罐，316L 不锈钢罐体，需要防腐蚀。

方案：

- PTFE 包覆同轴式传感器
- 插入罐内，不与液体直接金属接触
- AD7746 做电容数字转换
- 4-20mA 输出连接现有 PLC

### 7.3 柴油油箱液位

需求：车辆柴油箱，塑料油箱，需要低成本。

方案：

- 同轴电容探棒插入油箱
- 振荡器频率偏移法
- MCU 测频并查表转换
- epsilon_r 低（约 2.1），需要高分辨率

## 9 IoT集成

### 9.1 系统架构

```
[电容传感器] → [CDC芯片] → [低功耗MCU] → [无线模块] → [网关] → [云平台]
```

关键参数：

- 采样率：液位变化慢，1-10 次/秒足够
- 功耗：间歇测量，MCU 休眠，电池可用 1-2 年
- 通信：LoRa（远距离低功耗）或 WiFi（近距离高带宽）

### 9.2 报警策略

- 高液位报警：防止溢出
- 低液位报警：防止泵干转
- 变化率报警：异常快速变化可能泄漏
- 数据记录：趋势分析，预测补液时间

## 10 校准方法

### 10.1 两点校准

最常用的方法：

1. 空罐时记录读数 C_empty
2. 满罐时记录读数 C_full
3. 任意读数 C_x 对应液位 h = (C_x - C_empty) / (C_full - C_empty) * H

### 10.2 线性化

非圆柱罐体（如卧式圆柱、球罐）液位和体积不是线性关系：

- 将罐体形状参数输入 MCU
- 查表或公式计算液位到体积的转换
- 某些厂商提供罐表（strapping table）

### 10.3 定期校验

- 液位传感器漂移通常很小
- 建议每 6-12 个月人工对比一次
- 导电液体中 PTFE 绝缘层破损是最大风险

## 总结

电容式液位传感器用最简单的物理原理——液体和空气的介电常数不同——实现了从家用到工业级全场景覆盖的液位测量。它的魅力在于：可以做非接触测量（电极贴在罐外），可以防腐蚀（PTFE 包覆），可以低成本（振荡器 + MCU），还可以高精度（FDC1004/AD7746）。

设计电容液位系统时，优先想清楚三件事：(1) 罐体材质决定接触还是非接触；(2) 液体导电性决定是否需要绝缘电极；(3) 液体的 epsilon_r 决定需要的测量分辨率。剩下的事情——校准、补偿、无线传输——都是标准 IoT 工程问题。
