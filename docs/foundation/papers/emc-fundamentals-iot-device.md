# IoT设备EMC电磁兼容性基础与标准

> **难度**：🟢 初级 | **领域**：电磁兼容基础 | **关键词**：EMC, EMI, EMS, CISPR, LISN | **阅读时间**：约 20 分钟

## 日常类比

嘈杂会议室里，人人既在“说话”（发射电磁波），又要听清同事发言（接收有用信号）。**电磁兼容性**（Electromagnetic Compatibility, EMC）就是：设备既不当噪音源，也不被噪音打垮。物联网（Internet of Things, IoT）产品若过不了 EMC 认证，往往无法合法上市[1][5]。

## 摘要

本文梳理 EMC 的发射（Electromagnetic Interference, EMI）与抗扰度（Electromagnetic Susceptibility / Immunity, EMS）两翼，对照 CISPR 32 / IEC 61000-4-x 等常用限值与试验，并给出接地、屏蔽、滤波、布局与预合规（pre-compliance）要点。文中限值与费用为标准/行业常见量级，正式认证以目标市场现行版本与实验室报告为准[1][2][6]。

## 1. EMC 定义与 IoT 意义

EMC 两层含义：

1. **不干扰别人**：骚扰发射不超过限值（EMI）
2. **不怕别人干扰**：在规定电磁环境中功能可接受（EMS）

| 维度 | EMI（发射） | EMS（抗扰度） |
|------|-------------|---------------|
| 方向 | 设备向外 | 外部向设备 |
| 关注 | 电平是否超标 | 功能是否降级/失效 |
| 测试 | 测发射 | 施加干扰观察响应 |
| 不通过后果 | 干扰他机、认证失败 | 复位、丢包、误动作 |

IoT 侧额外压力：CE / FCC / CCC 等市场准入；Wi-Fi / BLE / Zigbee / 蜂窝多无线共存；工业现场电机与变频器环境恶劣[5][7]。

## 2. 发射（EMI）

### 2.1 辐射与传导

**辐射发射**：高速时钟/数据线、电缆、机壳缝隙充当天线。多媒体设备常用测量频段约 30 MHz–数 GHz，场地多为开阔场或半电波暗室[1][6]。

**传导发射**：开关电源、数字开关电流、电机 PWM 噪声沿电源/信号线传播。常用测量约 150 kHz–30 MHz，经 **线性阻抗稳定网络**（Line Impedance Stabilization Network, LISN）在电源端口取样[1][3]。

### 2.2 限值量级（示意）

以 CISPR 32 Class B（住宅类）常见表述为例（准峰值 QP 等检波方式以标准正文为准）：

| 频率范围 | 传导限值（QP，示意） | 辐射限值（QP，示意） |
|----------|----------------------|----------------------|
| 150 kHz–500 kHz | 约 56→46 dBμV（斜线） | — |
| 500 kHz–5 MHz | 约 46 dBμV | — |
| 5–30 MHz | 约 50 dBμV | — |
| 30–230 MHz | — | 约 30 dBμV/m @ 10 m |
| 230 MHz–1 GHz | — | 约 37 dBμV/m @ 10 m |

Class B 通常比 Class A（工业）更严约一个数量级量级（约 10 dB 量级），具体以现行标准表为准[1]。

## 3. 抗扰度（EMS）

| 项目 | 标准族 | 机制示意 | IoT 场景 |
|------|--------|----------|----------|
| 辐射抗扰度 RS | IEC 61000-4-3 | 射频场照射 | 近旁手机/对讲机 |
| 静电放电 ESD | IEC 61000-4-2 | 接触/空气放电 | 人手摸外壳/按键 |
| 电快速瞬变 EFT | IEC 61000-4-4 | 继电器分断脉冲群 | 同回路大功率开关 |
| 浪涌 Surge | IEC 61000-4-5 | 雷击/大功率切换 | 户外电源入口 |
| 传导抗扰度 CS | IEC 61000-4-6 | 线缆注入射频 | 长线缆拾取噪声 |

场强、注入电压与等级（如 3 V/m 与 10 V/m、接触数 kV）由产品标准与目标市场选定，勿把“常见等级”当成唯一合规点[2][8]。

## 4. 关键标准与标志

### 4.1 发射与抗扰度

| 标准 | 范围 | 说明 |
|------|------|------|
| CISPR 32 / EN 55032 | 多媒体设备发射 | 覆盖 IT + 音视频等 |
| CISPR 35 / EN 55035 | 多媒体抗扰度 | 与发射配套 |
| FCC Part 15 | 美国无意/有意辐射体 | 与 CE 路径不同 |
| GB 9254 / GB/T 17618 等 | 中国市场 | 常与 CISPR 体系对应[5] |

| IEC 61000-4-x | 项目 | 常见等级量级（示意） |
|---------------|------|----------------------|
| 4-2 | ESD | 接触数 kV / 空气更高 |
| 4-3 | RS | 数 V/m 量级 |
| 4-4 | EFT | 信号/电源不同等级 |
| 4-5 | Surge | 线-线 / 线-地 |
| 4-6 | CS | 数 V 注入量级 |

### 4.2 认证标志（概览）

| 标志 | 地区 | EMC 相关要求（概览） |
|------|------|----------------------|
| CE | 欧盟 | EMC 指令等 |
| FCC | 美国 | Part 15 等 |
| CCC | 中国 | 适用国标 |
| VCCI / MIC 等 | 日本等 | 地区规则 |

## 5. 测试与预合规

半电波暗室、LISN、ESD 枪、EFT 发生器是正式试验常见装备[2][6]。研发阶段用近场探头 + 频谱仪定位热点，再用电流探头估共模电流；经验换算式仅作量级判断，不能替代暗室数据[3][9]。

| 项目 | 预测试 | 正式测试 |
|------|--------|----------|
| 场地 | 开放空间/简易暗室 | 标准半电波暗室等 |
| 精度 | 常有数 dB 量级偏差 | 按 CISPR 16 等场地要求 |
| 费用 | 设备一次性投入为主 | 按次计费，量级因实验室而异 |
| 用途 | 定位与迭代 | 报告与符合性声明 |

预测试宜留数 dB 余量覆盖场地差异；样品须与量产一致（外壳、线缆、软件）[7]。

## 6. 设计原则与 IoT 常见源

四大原则：**接地、屏蔽、滤波、布局**[3][4]。

- 低频倾向单点接地，高频多点/平面；模拟/数字地在合适点汇合，避免随意分割地平面
- 屏蔽注意缝隙与电缆穿透（详见屏蔽专文）
- 电源共模扼流圈 + X/Y 电容；信号 RC/磁珠；IC 旁路电容就近
- 时钟内层、远离板边与连接器；天线远离 DC-DC

IoT 高频问题源：DC-DC 开关边沿、SPI/USB/SDIO、射频谐波、晶振谐波、外部电缆共模辐射。整改成本通常“加电容/磁珠 ≪ 改板 ≪ 改壳开模”，故应在布局阶段介入[3][10]。

## 7. 常见失败与整改

| 测试项 | 常见失败频段/现象 | 典型原因 |
|--------|-------------------|----------|
| 辐射发射 | 百 MHz 量级超标 | 电缆、缝隙 |
| 传导发射 | 数 MHz 量级超标 | 开关电源无滤波 |
| ESD | 复位/死机 | 接地差、按键无 TVS |
| EFT | 通信中断 | 信号无滤波/隔离不足 |
| 浪涌 | 器件损坏 | 无浪涌器件 |
| RS | 丢包 | 前端/走线拾取噪声 |

流程：定位 → 分清共模/差模与频率 → 单变量改一项 → 预测试验证 → 固化 BOM/Gerber。

## 8. 局限、挑战与可改进方向

### 1. 预测试与正式暗室不可互换

**局限**：近场/简易场地与标准半电波暗室可差数 dB，易“预测试过、正式挂”。
**改进**：关键频点留余量；问题频段用电流探头+电缆处理交叉验证；预算允许时做半正式预扫[6][9]。

### 2. 无线共存与无意发射纠缠

**局限**：有意发射（Wi-Fi/BLE）谐波与无意数字噪声同屏，整改易误伤射频性能。
**改进**：分区屏蔽与滤波；PA 后低通；天线匹配与谐波抑制分开验证[7][10]。

### 3. 软件状态影响发射谱

**局限**：不同固件（全速外设 vs 休眠）发射谱差异大，单一样本模式可能漏测。
**改进**：定义最坏工作模式矩阵；认证样品锁定与量产同固件配置[1][7]。

### 4. 标准版本与市场路径碎片化

**局限**：CE/FCC/CCC 路径与引用标准版本不同，一套整改难全球通用。
**改进**：按目标市场列适用标准矩阵；模块化电源/无线子板降低重复认证成本[5][8]。

## 总结

EMC 是 IoT 量产门槛：先分清 EMI/EMS，再按市场选 CISPR/IEC/FCC/GB 路径，设计阶段落实接地、屏蔽、滤波、布局，并用预合规降低一次通过风险。数值与费用以现行标准与实验室为准。
