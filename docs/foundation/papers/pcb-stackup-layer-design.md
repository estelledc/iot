# 多层PCB叠层设计与阻抗控制

> **难度**：🟡 中级 | **领域**：PCB叠层设计 | **阅读时间**：约 20 分钟

## 引言

想象你在盖一栋楼。单层板就像平房 -- 所有人挤在一层，吵闹声(干扰)无处隔离。多层PCB就像多层楼房 -- 信号走各层楼，地板(地平面)和天花板(电源平面)把噪声隔开，每层各司其职。叠层设计就是决定"几层楼、每层做什么、楼层间距多少"的建筑方案。阻抗控制则是确保信号在"走廊"里跑的时候不会因为走廊宽窄不一而反射回来 -- 就像水管粗细必须匹配，否则水流会在变径处产生回冲。

本文从IoT设备角度出发，系统讲解多层PCB叠层设计原理与阻抗控制方法。

## 1. 为什么需要多层PCB

### 1.1 单层/双层板的局限

双层板在简单电路中够用，但面对以下场景捉襟见肘:

- 信号速率提高时，返回电流路径过长，产生较大环路电感
- 模拟与数字混合时，缺乏完整地平面隔离，噪声耦合严重
- 射频走线缺少参考平面，阻抗无法控制，驻波比恶化
- 电源分配网络(PDN)缺少专属电源平面，电压纹波增大

### 1.2 多层板的核心优势

| 优势 | 机制 | 实际效果 |
|------|------|----------|
| 信号完整性 | 信号层紧邻参考平面，返回路径短 | 减少反射和振铃 |
| EMI抑制 | 完整地平面提供屏蔽和镜像回流 | 辐射降低10-20dB |
| 阻抗可控 | 介电层厚度和材质精确管理 | 特性阻抗误差<5% |
| 电源完整性 | 专属电源平面降低PDN阻抗 | 纹波和噪声显著降低 |
| 布线密度 | 内层走线释放表面积 | 元件更密集，板子更小 |

### 1.3 何时从双层升级到多层

判断标准:

- 数字信号上升沿 < 1ns (约对应 > 200MHz 时钟频率)
- 存在模拟/射频电路需要隔离
- 板面积受限，双层无法完成布线
- 需要阻抗控制的走线(如USB差分对、射频微带线)

## 2. 常见叠层方案

### 2.1 四层板叠层

四层板是最常见的入门多层方案，性价比高。

**方案A: 标准SIG-GND-PWR-SIG**

| 层号 | 分配 | 说明 |
|------|------|------|
| L1 | 信号(顶层) | 主要走信号和放置元件 |
| L2 | 地(GND) | 完整地平面 |
| L3 | 电源(PWR) | 主电源平面 |
| L4 | 信号(底层) | 辅助信号走线 |

优点: L1信号紧邻L2地平面，返回路径最优。缺点: L3电源和L2地之间间距较大，平面间去耦电容效果弱。

**方案B: SIG-GND-SIG-PWR(嵌入式信号)**

| 层号 | 分配 | 说明 |
|------|------|------|
| L1 | 信号(顶层) | 元件面 |
| L2 | 地(GND) | 完整地平面 |
| L3 | 信号(内层) | 带状线走线，EMI更好 |
| L4 | 电源(PWR) | 电源平面 |

优点: L3内层信号被L2和L4夹在中间，天然屏蔽。缺点: L1与L2间距被挤占，顶层阻抗控制需额外注意。

### 2.2 六层板叠层

六层板在IoT网关、BLE+WiFi复合设备中常见。

**方案A: SIG-GND-SIG-PWR-GND-SIG**

| 层号 | 分配 | 说明 |
|------|------|------|
| L1 | 信号(顶层) | 关键信号、射频走线 |
| L2 | 地(GND) | 主地平面 |
| L3 | 信号(内层) | 非关键信号 |
| L4 | 电源(PWR) | 主电源平面 |
| L5 | 地(GND) | 辅助地平面 |
| L6 | 信号(底层) | 辅助信号 |

优点: 每个信号层都有紧邻参考平面。L1微带线紧贴L2地平面，L3带状线被L2/L4夹住。

**方案B: SIG-GND-GND-SIG-PWR-SIG(双地平面)**

| 层号 | 分配 | 说明 |
|------|------|------|
| L1 | 信号(顶层) | 射频走线 |
| L2 | 地(GND) | 地平面1 |
| L3 | 地(GND) | 地平面2(与L2通过过孔缝合) |
| L4 | 信号(内层) | 高速差分对 |
| L5 | 电源(PWR) | 电源平面 |
| L6 | 信号(底层) | 一般信号 |

适用: 射频敏感场景，L2+L3构成极低阻抗地参考。

### 2.3 叠层选择决策表

| 因素 | 四层板 | 六层板 | 八层及以上 |
|------|--------|--------|------------|
| 成本(相对) | 1x | 1.8-2.5x | 3-5x |
| 最大信号完整性 | 中等 | 好 | 优秀 |
| 射频走线支持 | 基本满足 | 良好 | 优秀 |
| 适用场景 | 简单IoT终端 | BLE+WiFi网关 | 复杂通信设备 |
| 层数与成本权衡 | 最优 | 中等 | 仅必要时 |

## 3. 层分配原则

### 3.1 信号层

信号层分配的核心原则 -- 每个信号层必须有紧邻的参考平面:

- 高速信号优先分配在紧邻地平面的层
- 差分对走同一层，避免换层带来的不对称
- 关键时钟信号走内层(带状线)，减少EMI辐射
- 换层时必须在过孔旁放置地过孔，保证返回路径连续

### 3.2 地平面层

地平面是叠层设计的"锚":

- 尽量保持完整，不分割
- 如果必须分割(如模拟地和数字地)，只在单点连接
- 多个地平面通过密集过孔缝合，降低平面间阻抗
- 地平面与信号层间距越小，返回路径越短，EMI越低

### 3.3 电源平面层

电源平面的设计要点:

- 主电源(如3.3V)分配完整平面
- 多电压系统可用分割平面，但需注意分割沟道不要切断信号返回路径
- 电源平面与地平面间距越小，平面间寄生电容越大，高频去耦效果越好
- 模拟电源和数字电源平面分开，避免噪声耦合

### 3.4 层间距设计

层间距直接影响阻抗和耦合:

```
叠层截面示意(六层板):

L1 信号  ----  0.035mm 铜厚
    介电    ....  0.10mm  (L1-L2: 信号-地，薄=低阻抗)
L2 地     ----  0.035mm 铜厚
    介电    ....  0.40mm  (L2-L3: 地-信号，厚=低耦合)
L3 信号  ----  0.035mm 铜厚
    芯板    ....  0.50mm  (L3-L4: 信号-电源，芯板刚性)
L4 电源  ----  0.035mm 铜厚
    介电    ....  0.10mm  (L4-L5: 电源-地，薄=高去耦)
L5 地     ----  0.035mm 铜厚
    介电    ....  0.10mm  (L5-L6: 地-信号，薄=低阻抗)
L6 信号  ----  0.035mm 铜厚
```

关键规则: 信号-参考间距小(0.1mm级)，参考-参考间距可大(0.4mm级)，电源-地间距小(增强去耦)。

## 4. 微带线与带状线

### 4.1 微带线(Microstrip)

微带线走在外层，只有一个参考平面。

```
截面示意:

    铜走线 (宽度W, 厚度T)
  ========================
  介电层 (厚度H, 介电常数Er)
  ========================
  参考平面 (地或电源)
```

特性:

- 信号速率较快(有效介电常数低于体材料)
- 制造后可目视检查和修切
- 暴露在表面，EMI辐射较高
- 阻抗受阻焊层影响(约2-5%偏移)

### 4.2 带状线(Stripline)

带状线走在内层，被两个参考平面夹住。

```
截面示意:

  参考平面 (上方)
  ========================
  介电层 (厚度H1)
  ========================
    铜走线 (宽度W, 厚度T)
  ========================
  介电层 (厚度H2)
  ========================
  参考平面 (下方)
```

特性:

- 天然屏蔽，EMI极低
- 阻抗稳定，不受阻焊层和空气影响
- 信号速率较慢(完全被介电材料包围)
- 制造后不可目视检查

### 4.3 微带线与带状线对比

| 特性 | 微带线 | 带状线 |
|------|--------|--------|
| EMI辐射 | 较高 | 极低 |
| 阻抗稳定性 | 中等 | 高 |
| 信号速率 | 较快(约0.5c) | 较慢(约0.35c) |
| 可检查性 | 好 | 差 |
| 过孔换层 | 不需要 | 需要 |
| 适用走线 | 射频天线馈线、顶层高速 | 内层高速差分对、时钟 |

## 5. 特性阻抗计算

### 5.1 特性阻抗的本质

特性阻抗不是直流电阻，而是信号沿传输线传播时电压与电流的比值。就像高速公路的车流密度和车速的比值 -- 不是路面的摩擦力，而是流动状态决定的特征量。

公式关系:

```
Z0 = sqrt(L/C)

其中:
  Z0 = 特性阻抗(欧姆)
  L  = 单位长度电感(H/m)
  C  = 单位长度电容(F/m)
```

### 5.2 微带线阻抗计算

常用简化公式(Hammerstad-Jensen):

```
Z0_microstrip = (87 / sqrt(Er + 1.41)) * ln(5.98 * H / (0.8 * W + T))

参数:
  Er = 介电常数
  H  = 介电层厚度(mil)
  W  = 走线宽度(mil)
  T  = 铜厚(mil)

常见1.6mm四层板, Er=4.4, H=3.94mil, T=1.4mil:
  50欧姆 => W 约 6.5mil
```

### 5.3 带状线阻抗计算

对称带状线(Cohn公式):

```
Z0_stripline = (60 / sqrt(Er)) * ln(4 * B / (0.67 * pi * (0.8 * W + T)))

参数:
  Er = 介电常数
  B  = 两参考平面总间距(mil)
  W  = 走线宽度(mil)
  T  = 铜厚(mil)

常见六层板, Er=4.2, B=20mil, T=1.4mil:
  50欧姆 => W 约 4.2mil
```

### 5.4 差分阻抗

差分对两根线之间的等效阻抗:

```
Zdiff = 2 * Z0 * (1 - k)

其中:
  Z0 = 单端特性阻抗
  k  = 耦合系数(与线间距S有关)

典型值:
  USB 2.0: Zdiff = 90 ohm (单端45 ohm)
  USB 3.0: Zdiff = 85 ohm
  HDMI:    Zdiff = 100 ohm
  PCIe:    Zdiff = 85 ohm
```

### 5.5 实际计算注意事项

- 以上公式为近似值，生产前必须用2D场求解器校验
- 阻焊层会使微带线阻抗降低2-5%，计算时需考虑
- 铜厚蚀刻后不是矩形而是梯形，影响有效宽度
- 表面处理(喷锡/沉金/OSP)对阻抗有微小影响

## 6. 介电常数与厚度的影响

### 6.1 介电常数(Er)的影响

介电常数决定信号传播速率和阻抗:

| 材料 | Er(1MHz) | Er(1GHz) | 损耗因子 | 典型应用 |
|------|----------|----------|----------|----------|
| FR-4 标准 | 4.5 | 4.2 | 0.02 | 通用数字板 |
| FR-4 高Tg | 4.4 | 4.1 | 0.015 | 无铅工艺板 |
| Rogers RO4350B | 3.48 | 3.48 | 0.0037 | 射频/微波 |
| Rogers RO4003C | 3.38 | 3.38 | 0.0027 | 高频射频 |
| PTFE(聚四氟乙烯) | 2.1 | 2.1 | 0.0004 | 微波/毫米波 |
| IS410 | 4.3 | 4.0 | 0.012 | 中速数字板 |

Er对阻抗的影响规律:

- Er增大 -> 阻抗降低(电容增大)
- Er增大 -> 信号速率降低
- Er变化1% -> 阻抗变化约0.5%

### 6.2 介质厚度的影响

介质厚度是调节阻抗最直接的参数:

- 厚度增大 -> 阻抗增大(电容减小)
- 厚度增大 -> 返回路径变长，EMI恶化
- 厚度变化1mil -> 阻抗变化约3-5 ohm(典型4层板)

实际板厂公差:

| 参数 | 标准公差 | 紧密公差 | 超紧密 |
|------|----------|----------|--------|
| 介质厚度 | +/-10% | +/-5% | +/-3% |
| 铜厚 | +/-20% | +/-10% | +/-5% |
| 线宽 | +/-20% | +/-10% | +/-5% |
| 阻抗 | +/-10% | +/-5% | +/-3% |

### 6.3 Er与厚度的联合调控

实际设计中需要联合考虑:

- 优先调节线宽(最灵活)
- 其次调节介质厚度(影响叠层结构)
- 最后选择材料(影响成本最大)

调节优先级: 线宽 > 介质厚度 > 材料Er

## 7. 射频走线阻抗控制

### 7.1 50欧姆标准

为什么射频走线普遍用50欧姆:

- 这是功率容量和信号损耗的折中最优点
- 50欧姆在同轴线中衰减接近最小值
- 行业生态(连接器、天线、测试设备)均以50欧姆为标准
- IoT设备中: WiFi天线馈线、BLE匹配网络、LoRa射频走线均需50欧姆

### 7.2 射频微带线设计步骤

1. 确定PCB材料和介电常数(如FR-4 Er=4.2)
2. 确定介质厚度(如3.94mil)
3. 用计算工具或公式求初始线宽
4. 用2D场求解器精确计算
5. 通知板厂做阻抗控制，给出目标值和公差
6. 板厂反馈调整后的线宽和叠层参数
7. 确认板厂阻抗测试coupon结果

### 7.3 射频走线设计规则

- 走线最短化，避免直角转弯(用45度或圆弧)
- 射频走线不换层(避免过孔引入不连续性)
- 射频走线下方地平面完整，不跨越分割
- 射频走线两侧包地，间距>3倍线宽
- 去耦电容尽量靠近射频IC引脚放置

## 8. 板厂能力与公差

### 8.1 阻抗控制流程

```
设计方                       板厂
  |                           |
  |-- 提交Gerber + 阻抗要求 -->|
  |                           |-- 计算调整线宽/间距
  |<-- 反馈确认叠层参数 --------|
  |                           |-- 生产 + 阻抗测试coupon
  |<-- 阻抗测试报告 -----------|
  |                           |
  确认/退货                    出货
```

### 8.2 阻抗测试Coupon

板厂会在拼板边缘制作测试条:

- 每种阻抗类型一条测试线
- 用TDR(时域反射计)测量实际阻抗
- 测量结果记录在阻抗测试报告中
- 公差通常为+/-10%(标准)或+/-5%(加价)

### 8.3 板厂能力分级

| 能力项 | 常规厂 | 中端厂 | 高端厂 |
|--------|--------|--------|--------|
| 最小线宽/间距 | 4/4mil | 3/3mil | 2/2mil |
| 阻抗公差 | +/-10% | +/-7% | +/-5% |
| 层数 | 2-8层 | 2-14层 | 2-32层 |
| 盲埋孔 | 不支持 | 支持简单 | 全面支持 |
| 射频材料混压 | 不支持 | 支持 | 支持 |
| HDI | 不支持 | 1阶 | 任意阶 |

### 8.4 与板厂沟通要点

提交Gerber时必须说明:

- 需要阻抗控制的走线网络列表
- 目标阻抗值和公差(如50ohm +/-10%)
- 参考层是地还是电源
- 是否有差分阻抗要求
- 特殊材料要求(如Rogers混压)

### 9.3 2D场求解器

为什么公式不够用:

- 公式假设走线截面为矩形，实际蚀刻后为梯形
- 公式未考虑阻焊层、表面处理的影响
- 差分对耦合效应在简化公式中精度不足
- 场求解器基于数值方法，精度可达1%以内

## 10. 成本与性能权衡

### 10.1 叠层成本构成

| 成本项 | 占比(约) | 节省策略 |
|--------|----------|----------|
| 基材 | 25-35% | 通用设计用FR-4 |
| 铜箔 | 5-10% | 标准铜厚1oz/0.5oz |
| 压合 | 15-20% | 减少压合次数 |
| 钻孔 | 10-15% | 避免盲埋孔 |
| 表面处理 | 5-10% | OSP最便宜 |
| 阻抗控制 | 5-10% | 只对必要走线控制 |
| 测试 | 10-15% | 合理的测试要求 |

### 10.2 典型IoT设备成本决策

**场景: BLE温湿度传感器(简单IoT终端)**

- 2层板足够，不需要阻抗控制
- 成本最优: 约$0.5-1/pcs(1000pcs)
- 射频部分用Pi型匹配网络补偿

**场景: WiFi+BLE网关(中等复杂度)**

- 4层板，天线馈线做50ohm阻抗控制
- 成本: 约$2-4/pcs(1000pcs)
- 标准FR-4材料，0.8mm或1.0mm板厚

**场景: LoRa+WiFi+BLE复合设备(高复杂度)**

- 6层板，多段射频走线阻抗控制
- 可能需要Rogers混压(射频层)
- 成本: 约$5-10/pcs(1000pcs)

### 10.3 成本优化原则

- 能4层不6层: 每加2层成本增加50-80%
- 阻抗控制只做必要的: 射频和高速差分对
- 避免HDI: 盲埋孔大幅增加成本
- 标准板厚(0.8/1.0/1.2/1.6mm): 非标板厚加价
- 拼板优化: 合理拼板提高板材利用率

## 11. IoT设备的特殊考量

### 11.1 天线层设计

IoT设备的天线布局对叠层有特殊要求:

- 天线区域下方地平面必须挖空(cutout)，避免金属平面吸收辐射
- 天线馈线从RF IC到天线必须最短，且做50欧姆阻抗控制
- 天线周围至少留出lambda/4的净空区(2.4GHz约31mm)
- 天线放在板边或板角，避免被元件遮挡

天线区域叠层处理:

```
正常区域                    天线净空区
  L1 信号/元件              L1 天线
  L2 GND (完整)             L2 GND (挖空)
  L3 PWR                    L3 PWR (可保留)
  L4 信号                    L4 信号
```

### 11.2 屏蔽罩(Shield Can)放置

屏蔽罩用于隔离射频模块或敏感模拟电路:

- 屏蔽罩焊盘放在地平面上，确保低阻抗接地
- 屏蔽罩内走线通过馈通(feed-through)与外部连接
- 屏蔽罩下方对应层的地平面保持完整
- 屏蔽罩高度需兼容元件高度，通常2-5mm

### 11.3 IoT小板叠层技巧

IoT设备常因尺寸受限使用小板(如20x30mm):

- 0.8mm板厚是IoT小板常用规格
- 0.8mm四层板层间距更小，阻抗更容易偏低，需调整线宽
- 小板天线设计更受限，优先选择现成天线模块
- 小板连接器多在板边，注意连接器参考地连续性

### 11.4 多协议IoT设备叠层

WiFi+BLE+LoRa多协议设备的叠层建议:

- 射频走线分配在不同层或不同区域，避免交叉
- 各协议的天线馈线独立走线，不要共用参考平面区域
- 各RF IC的电源去耦独立，不要共用去耦电容
- 晶振远离射频走线，避免谐波干扰

## 总结

多层PCB叠层设计的核心逻辑:

1. **每个信号层必须有紧邻参考平面** -- 这是信号完整性的基础
2. **信号-参考间距要小** -- 控制阻抗、降低EMI
3. **电源-地间距要小** -- 增强平面间去耦
4. **阻抗控制 = 线宽 x 介电厚度 x 介电常数** -- 三者联合调节
5. **板厂是伙伴不是供应商** -- 提前沟通，确认叠层和阻抗

IoT设备设计要点速查:

| 关注点 | 要点 |
|--------|------|
| 叠层数 | 简单终端2层，无线设备4层，多协议6层 |
| 阻抗控制 | 射频馈线50ohm，高速差分按协议要求 |
| 天线 | 净空区、下方挖空、馈线最短 |
| 成本 | 能少不多、阻抗只控必要、避免HDI |
| 板厂 | 提前沟通叠层、确认阻抗测试coupon |

叠层设计不是孤立的技术决策，而是信号完整性、EMC、成本和可制造性的综合权衡。好的叠层方案从需求出发，用最经济的层数和材料满足性能要求。
