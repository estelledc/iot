# SMT贴片与通孔焊接工艺对比
> **难度**：🟢 初级 | **领域**：电子制造工艺 | **阅读时间**：约 18 分钟

## 引言

想象你在做一道菜。SMT(表面贴装)就像用平底锅煎蛋——食材平铺在锅面上,加热就熟;THT(通孔焊接)就像用竹签串烤肉——食材要穿过竹签固定,再上火烤。两种做法都能做出好菜,但适合的食材和场景不同。薄切培根用煎的最快,大块肉串用烤的更稳。

IoT设备制造也是这样:贴片电阻、小芯片用SMT又快又准;连接器、大电容用通孔更牢靠。大多数IoT产品其实是两种工艺混着用的。理解两者的差异,才能在设计阶段就做出正确的选择,避免后期返工。

## 1. SMT概述

### 1.1 什么是SMT

SMT(Surface Mount Technology,表面贴装技术)将电子元件直接焊在PCB表面的焊盘上,不需要钻通孔。元件体积小、重量轻,适合自动化大批量生产。

### 1.2 SMT的核心优势

- 体积小:同功能元件,SMT封装面积是通孔的1/5到1/10
- 自动化程度高:贴片机每秒可放置3-10个元件
- 双面可用:PCB正反两面都能贴元件
- 高频性能好:寄生电感电容小,适合射频电路
- 成本低(批量):自动化生产,人工少

### 1.3 SMT的局限

- 不能承受大的机械应力(插拔、振动)
- 大功率散热路径受限
- 返修困难(特别是BGA)
- 需要专用设备(贴片机、回流焊炉)

## 2. THT概述

### 2.1 什么是THT

THT(Through-Hole Technology,通孔技术)将元件引脚穿过PCB上的镀通孔,在另一面焊接。引脚与焊孔的配合提供了机械锚固力。

### 2.2 THT的核心优势

- 机械强度高:引脚穿过PCB,抗振动、抗拉拔
- 散热好:引脚本身就是散热通道
- 功率承载强:引脚截面积大,承载电流大
- 手焊方便:原型调试阶段随时更换元件
- 可靠性直观:焊点可见可检

### 2.3 THT的局限

- 占用面积大:焊盘+通孔占据大量PCB面积
- 只能单面有效利用(另一面被焊盘和引脚占据)
- 自动化成本高:元件成型和插装需要额外工序
- 寄生参数大:引脚长度带来额外电感

## 3. SMT工艺流程

### 3.1 焊膏印刷

用不锈钢漏板(stencil)将焊膏(锡粉+助焊剂)印刷到PCB焊盘上。焊膏的量和位置精度直接决定焊接质量。

```
焊膏印刷参数:
+----------+------------------+
| 参数     | 典型值           |
+----------+------------------+
| 漏板厚度 | 0.1-0.15mm       |
| 焊膏类型 | Sn96.5/Ag3.0/Cu0.5|
| 印刷速度 | 20-60mm/s        |
| 刮刀压力 | 3-8kg            |
| 脱模速度 | 0.5-2mm/s        |
+----------+------------------+
```

### 3.2 贴片

贴片机从料带或料盘中吸取元件,通过视觉对位后放到焊膏上。贴片精度通常在0.05mm以内。

### 3.3 回流焊

PCB经过回流焊炉,经历以下温度曲线:

```
回流焊温度曲线(无铅焊膏):
温度(C)
|         /---\
|        /     \       峰值: 245-255C
|       /       \     保持: 30-60s
|      /         \
|     /           \
|    / 保温区      \  冷却区
|   /  150-200C     \  下降<4C/s
|  /   60-120s       \
| /                    \
|/ 预热区                \
1-3C/s上升
```

四个阶段:
1. **预热**: 温度从室温升到150C左右,焊膏中溶剂挥发
2. **保温**: 150-200C保持60-120s,使板面温度均匀,助焊剂激活
3. **回流**: 升至峰值245-255C,焊膏熔化形成焊点
4. **冷却**: 降温速率控制在1-4C/s,太快会产生热应力

## 4. THT工艺流程

### 4.1 元件插装

将元件引脚穿过PCB上的镀通孔。可以手工插装,也可以用自动插装设备。引脚插入后需要修剪到合适长度(通常突出板面1-2mm)。

### 4.2 波峰焊

波峰焊是THT的主要自动化焊接方式:

```
波峰焊流程:
PCB进入 -> 预热(100-130C) -> 助焊剂喷涂 -> 
波峰接触(焊锡250-260C,接触时间2-6s) -> 冷却
```

波峰焊的关键参数:
- 预热温度: PCB表面100-130C
- 焊锡温度: 250-260C
- 接触时间: 2-6秒
- 传送角度: 5-8度(倾斜角度帮助脱锡)

### 4.3 手工焊接

原型和小批量最常用的方式。烙铁温度350-380C,每个焊点2-5秒。适合调试修改,但一致性不如自动化工艺。

## 5. 元件封装对比

### 5.1 SMT常用封装

```
SMT电阻电容封装尺寸:
+------+---------+-----------+----------+
| 封装 | 公制mm  | 功率W    | 典型用途 |
+------+---------+-----------+----------+
| 0201 | 0603    | 0.05     | 手机     |
| 0402 | 1005    | 0.063    | IoT常用  |
| 0603 | 1608    | 0.1      | 通用     |
| 0805 | 2012    | 0.125    | 初学推荐 |
| 1206 | 3216    | 0.25     | 电源     |
+------+---------+-----------+----------+
```

IC封装:
- **QFN**: 无引脚扁平封装,底部散热焊盘,4x4mm到12x12mm
- **BGA**: 球栅阵列,焊点在芯片底部,不可视检,需X光
- **TSSOP**: 薄型缩小外形封装,引脚间距0.65mm,可手焊

### 5.2 THT常用封装

- **DIP**: 双列直插,引脚间距2.54mm,最经典的IC封装
- **轴向元件**: 电阻、二极管,两端引脚水平安装
- **径向元件**: 电解电容、LED,引脚从同一侧垂直安装

## 6. SMT与THT全面对比

```
SMT vs THT 对比表:
+------------------+------------------+------------------+
| 维度             | SMT              | THT             |
+------------------+------------------+------------------+
| 体积             | 小(1/5-1/10)     | 大               |
| 批量成本         | 低(自动化)       | 高(工序多)      |
| 组装速度         | 快(数秒/板)      | 慢(分钟/板)     |
| 返修难度         | 高(需热风/BGA站) | 低(烙铁即可)    |
| 机械强度         | 低               | 高(引脚锚固)    |
| 热循环可靠性     | 中(焊点小)       | 高(焊点大)      |
| 原型制作         | 难(需钢网+回流)  | 易(烙铁直焊)    |
| 功率承载         | 低               | 高               |
| 高频性能         | 好(寄生小)       | 差(引脚电感)    |
| 双面安装         | 支持             | 不支持           |
| 检验方式         | AOI/X光          | 目视             |
+------------------+------------------+------------------+
```

## 7. IoT设备中何时选用THT

虽然SMT是主流,但以下场景THT仍然不可替代:

### 7.1 连接器

外部连接器承受插拔力和振动,必须用通孔安装。M12、USB、接线端子等连接器的通孔引脚提供机械锚固,防止焊盘撕裂。

### 7.2 大功率元件

大电流电感、大容量电解电容、功率MOSFET——引脚本身就是散热通道。SMT封装的散热路径需要通过PCB铜层,效率较低。

### 7.3 原型开发

调试阶段频繁更换元件,THT手焊3分钟搞定,SMT换一个0402电阻可能需要热风枪加放大镜。

### 7.4 需要穿过PCB的连接

PCB背面需要连接的面,比如正面传感器背面排针引出信号,必须用通孔。

## 8. 混合装配工艺

### 8.1 SMT+THT同板

大多数IoT产品都是混合板:MCU和阻容用SMT,连接器和大电容用THT。

### 8.2 工艺顺序

```
混合装配流程:
1. SMT面焊膏印刷
2. SMT元件贴装
3. 回流焊(SMT焊接完成)
4. 翻板
5. THT元件插装
6. 选择性波峰焊(或手焊THT元件)
7. 清洗、检验
```

关键原则: **先SMT后THT**。因为回流焊温度(245C)远高于波峰焊(250C但接触时间短),THT元件的塑料壳体可能受不了回流焊的长时间高温。

### 8.3 选择性波峰焊

不是整板过波峰,而是用喷嘴只焊THT焊点,避开SMT元件。这样可以保护已经焊好的SMT元件不被波峰焊二次加热。

## 9. 制造设计(DFM)要点

### 9.1 焊盘设计

```
SMT焊盘尺寸(0603电阻为例):
+------------------+----------+
| 参数             | 推荐值   |
+------------------+----------+
| 焊盘宽度         | 0.9mm    |
| 焊盘长度         | 1.1mm    |
| 焊盘间距         | 0.6mm    |
| 阻焊开窗         | 焊盘+0.05mm|
+------------------+----------+
```

### 9.2 钢网设计

- 厚度: 0.1-0.15mm(元件越小越薄)
- 开孔: 通常比焊盘小10%,防止焊膏溢出
- 特殊元件: QFN中心焊盘开孔率50-70%,防止焊锡过多导致芯片浮起

### 9.3 元件方向

- 同类元件方向一致,便于贴片机高效拾放
- 极性元件(电解电容、二极管)统一标记方向
- IC去耦电容尽量靠近电源引脚

### 9.4 基准点(Fiducial)

PCB上用于贴片机视觉定位的圆形铜点:
- 数量: 至少3个(两个对角+一个防旋转)
- 大小: 直径1-2mm裸铜,周围1mm无走线
- 位置: 对角分布,边缘优先

## 10. 回流焊曲线详解

### 10.1 预热区

温度从室温升至150C,速率1-3C/s。太快会导致焊膏飞溅(锡珠),太慢会使助焊剂过早失效。

### 10.2 保温区

150-200C保持60-120s。目的是让大小元件温度均匀,避免大小元件温差过大导致冷焊或过热。

### 10.3 回流区

升至峰值温度245-255C,焊点处于液相线以上时间30-60s。时间过短则焊点不饱满,过长则金属间化合物过厚,降低可靠性。

### 10.4 冷却区

降温速率1-4C/s。冷却过快会产生热应力导致焊点开裂,过慢则晶粒粗大降低强度。

```
常见回流焊问题与温度曲线关系:
+----------+---------------------------+------------------+
| 问题     | 可能的曲线原因            | 修正方法         |
+----------+---------------------------+------------------+
| 锡珠     | 预热太快                  | 降低预热速率     |
| 冷焊     | 峰值温度不够              | 提高峰值温度     |
| 立碑     | 保温区温差大              | 延长保温时间     |
| 焊点灰暗 | 回流时间过长              | 缩短回流时间     |
| 元件开裂 | 冷却太快                  | 降低冷却速率     |
+----------+---------------------------+------------------+
```

## 11. 常见焊接缺陷

### 11.1 立碑(Tombstoning)

贴片电阻或电容一端翘起,像墓碑一样竖立。原因:两端焊膏熔化时间不同,表面张力不平衡。多见于0201/0402等小元件。

预防:焊盘对称设计,保温区温度均匀,减小焊盘尺寸差异。

### 11.2 焊桥(Solder Bridge)

相邻焊点之间形成不需要的锡连接。原因:焊膏量过多,引脚间距过小,贴片偏移。

预防:钢网开孔缩小10%,减小焊膏量,提高贴片精度。

### 11.3 冷焊(Cold Solder Joint)

焊点表面粗糙灰暗,焊锡未完全润湿。原因:峰值温度不够,焊膏过期,助焊剂失效。

预防:定期校准回流焊温度曲线,焊膏冷藏保存(2-10C),开封后24小时内用完。

### 11.4 BGA空洞(Void)

BGA焊球内部出现气泡空洞。原因:焊膏中助焊剂挥发气体未能逸出。少量空洞(面积占比<25%)可接受,大面积空洞影响可靠性。

预防:优化回流曲线的保温区,使用低空洞焊膏,适当延长峰值时间。

```
缺陷可接受标准(IPC-A-610E):
+----------+------------+----------+----------+
| 缺陷     | 1级(电子)  | 2级(工业)| 3级(军工)|
+----------+------------+----------+----------+
| 立碑     | 不允许     | 不允许   | 不允许   |
| 焊桥     | 不允许     | 不允许   | 不允许   |
| 冷焊     | 不允许     | 不允许   | 不允许   |
| BGA空洞  | <30%       | <25%     | <20%     |
| 少锡     | 润湿>75%   | 润湿>75% | 润湿>75% |
+----------+------------+----------+----------+
```

## 12. IoT初创团队的实用路径

### 12.1 原型阶段:手焊THT为主

- 用DIP封装的MCU开发板(Arduino/ESP32-DevKit)
- 传感器模块用杜邦线连接
- 烙铁+吸锡器+助焊剂,预算200元以内
- 目标:验证功能,不追求体积和外观

### 12.2 小批量(10-50片):钢网+热台+手贴

- 设计PCB,定制不锈钢钢网(费用200-500元)
- 手工刮焊膏,镊子贴片,热台回流
- THT元件手焊补件
- 总设备投入:1000-2000元(热台+钢网+镊子+放大镜)

### 12.3 中批量(100-1000片):贴片机+回流炉

- 桌面型贴片机(Neoden TM系列,2-5万元)
- 小型回流炉或热风回流
- THT用选择性波峰焊或手焊
- 一人操作,日产50-100片

### 12.4 大批量(1000片以上):代工(EMS)

- 设计文件发给PCBA代工厂
- 代工厂完成SMT+THT全流程
- 关注NRE费用(钢网、工装)和最小起订量
- 每片加工费随批量下降显著

## 13. 成本分析

### 13.1 NRE(一次性工程费用)

```
NRE费用对比:
+------------------+----------+----------+
| 项目             | SMT      | THT      |
+------------------+----------+----------+
| 钢网             | 300-800元| 无       |
| THT工装夹具     | 无       | 200-1000元|
| 贴片程序调试     | 200-500元| 无       |
| 波峰焊夹具      | 无       | 300-800元|
| 合计NRE         | 500-1300 | 500-1800 |
+------------------+----------+----------+
```

### 13.2 单板加工成本

```
加工成本随批量变化(以100x80mm IoT板为例):
+----------+-----------+-----------+-----------+
| 批量     | 手焊      | 半自动    | EMS代工   |
+----------+-----------+-----------+-----------+
| 10片     | 50元/片   | 不适用    | 不接单    |
| 50片     | 50元/片   | 20元/片   | 15元/片   |
| 200片    | 不现实    | 12元/片   | 6元/片    |
| 1000片   | 不现实    | 8元/片    | 3元/片    |
| 5000片   | 不现实    | 不现实    | 1.5元/片  |
+----------+-----------+-----------+-----------+
```

注意:EMS代工的"不接单"是指最小起订量通常50-100片,低于此数量NRE摊销后单价极高。

### 13.3 元件成本差异

同功能元件,SMT封装通常比THT便宜10-30%,因为封装小、材料少、产量大。但SMT的小封装(如0201)调试困难,增加隐性成本。

## 总结

SMT和THT不是非此即彼的选择,而是各有所长的工具:

1. **SMT是量产的基础**: 小体积、高效率、低成本,占IoT板上元件数量的90%以上
2. **THT是可靠的保障**: 连接器、大功率、原型调试,这三种场景THT不可替代
3. **混合装配是常态**: 先SMT回流再THT波峰/手焊,是IoT产品最成熟的工艺路线
4. **DFM决定成本**: 好的DFM设计可以让小批量手焊和大批量代工无缝切换
5. **量级决定工艺**: 10片手焊,50片钢网热台,200片贴片机,1000片以上找EMS

选工艺就像选交通工具:出门买菜走路就行,跨城出差坐高铁,别上来就买飞机。
