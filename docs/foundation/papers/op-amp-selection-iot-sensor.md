# 运算放大器在IoT传感器前端的选型要点

> **难度**：🟢 初级 | **领域**：模拟前端设计 | **阅读时间**：约 18 分钟

## 引言

想象你用手机录远处的人说话——声音太小听不清。你需要的不是更大的手机，而是一个麦克风放大器。运算放大器(op-amp)在IoT传感器系统中就是这个角色：传感器输出的微弱信号(毫伏甚至微伏级)，需要运放放大到ADC能够识别的范围(通常0-3.3V)。选错运放，就像用劣质话筒放大器——要么信号淹没在噪声里，要么放大了但变形了，ADC采到的全是废数据。

## 1. 运放在传感器信号链中的角色

### 1.1 信号链全貌

一个典型的IoT传感器信号链：传感器 --> 信号调理(运放: 放大/阻抗变换/电平移位) --> 抗混叠滤波 --> ADC --> MCU。传感器输出通常是mV级、高阻抗的差分或单端信号。

### 1.2 为什么不能直接接ADC

- **信号太弱**：热电偶输出约40uV/C，100C温差仅4mV，3.3V 12bit ADC的LSB=0.8mV，信号不到1个LSB
- **阻抗不匹配**：pH传感器输出阻抗>100MO，ADC输入阻抗通常<100kO，直接接会拉偏信号
- **差分转单端**：应变桥输出差分信号，ADC通常需要单端输入
- **电平不匹配**：传感器输出可能以2.5V为中心摆动，ADC输入范围0-3.3V

## 2. IoT运放关键参数

### 2.1 输入失调电压(Vos)

运放输入端为零差分电压时，输出不为零的电压折算到输入端。

- 通用运放：1-5mV
- 精密运放：10-100uV
- 斩波稳零运放：1-5uV

对IoT的影响：Vos会被增益放大后叠加到信号上，引入直流偏移误差。

```
例: Vos = 1mV, 增益G = 100
输出偏移 = 1mV * 100 = 100mV (3.3V系统的3%)
```

### 2.2 输入偏置电流(Ib)

运放输入端需要的直流电流。

- 双极型运放：nA-uA级
- JFET输入：pA-nA级
- CMOS输入：fA-pA级

对IoT的影响：Ib流过高阻抗传感器会产生电压误差。

```
例: Ib = 1nA, 传感器阻抗 = 1MO
误差电压 = 1nA * 1MO = 1mV
```

### 2.3 增益带宽积(GBW)

增益与带宽的乘积为常数：

```
GBW = Gain * Bandwidth

例: GBW = 1MHz
增益100时: 带宽 = 1MHz/100 = 10kHz
增益10时: 带宽 = 1MHz/10 = 100kHz
```

选择原则：GBW至少为所需闭环增益和信号频率乘积的10倍以上，保证足够的相位裕度。

### 2.4 电源电流(Iq)

运放自身的静态电流，直接影响电池寿命：

| 类别 | 典型Iq | 代表型号 | 适用场景 |
|------|--------|---------|---------|
| 纳瓦级 | <1uA | TSU111(580nA) | 常开传感器 |
| 微瓦级 | 1-50uA | OPA333(17uA) | 周期采集 |
| 通用级 | 50uA-1mA | OPA2340(750uA) | 宽带信号 |
| 高速级 | >1mA | OPA2350(5.2mA) | 高速采集 |

### 2.5 噪声

运放噪声用电压噪声密度表示，单位nV/sqrt(Hz)：

- 低噪声运放：1-5 nV/sqrt(Hz)
- 通用运放：10-50 nV/sqrt(Hz)
- 低功耗运放：50-200 nV/sqrt(Hz)

总噪声计算：

```
总RMS噪声 = 噪声密度 * sqrt(BW)

例: 50nV/sqrt(Hz) * sqrt(10kHz) = 50 * 100 = 5000nV = 5uV(rms)
经100倍放大: 输出噪声 = 500uV(rms)
```

## 3. 轨到轨输入输出

### 3.1 为什么需要轨到轨

IoT设备通常用单节锂电池(3.0-4.2V)或3.3V供电，信号动态范围非常宝贵。轨到轨(Rail-to-Rail)运放的输入和输出可以接近电源轨：

```
非RR运放:
  输入范围: Vcc-1.5V (3.3V系统: 0-1.8V)
  输出范围: Vcc-1.0V (3.3V系统: 0.3-2.3V)

RR运放:
  输入范围: -50mV ~ Vcc+50mV
  输出范围: 10mV ~ Vcc-10mV (轻载)
```

### 3.2 注意事项

- "轨到轨"不是真的到轨，输出通常有5-50mV的裕量
- 输出摆幅与负载电流有关：重载时裕量增大
- 某些RRIO运放在输入穿越共模区时有失调跳变

## 4. 低功耗运放推荐

### 4.1 纳瓦级：TSU111

ST的TSU111是目前最低功耗的运放之一：

- Iq = 580nA(每通道)
- Vos = 1.5mV(典型)
- GBW = 8kHz
- 噪声 = 105 nV/sqrt(Hz)

适合：常开的环境传感器(温度、湿度)、电池监测。

### 4.2 微瓦级：OPA333

TI的OPA333是低功耗精密运放的标杆：

- Iq = 17uA
- Vos = 10uV(最大)
- GBW = 350kHz
- 噪声 = 55 nV/sqrt(Hz)
- 自动零技术(几乎无1/f噪声)

适合：称重传感器、热电偶、医疗传感器。

### 4.3 通用级：OPA2340

TI的OPA2340提供较好的速度-功耗平衡：

- Iq = 750uA
- Vos = 150uV(最大)
- GBW = 5.5MHz
- 噪声 = 15 nV/sqrt(Hz)
- RRIO

适合：音频采集、振动传感器、光电检测。

## 5. 常见放大器配置

### 5.1 同相放大器

```
        Rf
Vin --|+ \--+-- Vout
      |  /  |
      |-/   |
       |    +-- Rg -- GND
       |
      GND

增益 G = 1 + Rf/Rg
输入阻抗: 极高(>GO)
```

适合：高阻抗传感器(如pH电极)的信号缓冲和放大。

### 5.2 差分放大器

```
        Rf
V+ ---|+ \--+-- Vout
      |  /  |
V- ----|-/   |
        |    |
       Rg    +-- Rf -- Vref
        |
       GND

增益 G = Rf/Rg (当电阻匹配时)
共模抑制: 取决于电阻匹配精度
```

适合：桥式传感器(应变片、压力传感器)的差分输出。

### 5.3 仪表放大器

```
V+ --[缓冲]--+--[差分放大]-- Vout
              |
V- --[缓冲]--+

增益由单一外部电阻设定
CMRR: 80-120dB
输入阻抗: >GO
```

推荐芯片：INA333(低功耗)、AD8237(零漂移)、INA826(通用)。

### 5.4 跨阻放大器(TIA)

```
              Rf
              |   |
PD --|+ \--+--|---|-- Vout
     |  /  |  |
    GND  --+ Cf
            |
           GND

I-V转换: Vout = -Iin * Rf
Cf: 防止振荡的补偿电容
```

适合：光电二极管、电流输出型传感器。

## 6. 增益带宽积选择

### 6.1 选择原则

```
所需GBW >= 10 * Gain * f_signal

例: 传感器信号1kHz, 需要增益100
GBW >= 10 * 100 * 1kHz = 1MHz
选GBW >= 1MHz的运放
```

### 6.2 常见信号频率与GBW选择

| 传感器类型 | 信号频率 | 典型增益 | 所需GBW |
|-----------|---------|---------|---------|
| 温度(NTC/热电偶) | <1Hz | 100-1000 | 10kHz |
| 称重(应变桥) | <10Hz | 100-1000 | 100kHz |
| 振动(压电) | 1-10kHz | 10-100 | 10MHz |
| 音频(麦克风) | 20-20kHz | 10-100 | 20MHz |
| 光电(脉冲) | 1kHz-1MHz | 1-10 | 10MHz |

## 7. 单电源设计技巧

### 7.1 虚地偏置

单电源运放需要将信号偏置到电源中点(Vcc/2)，使双极性信号能在单电源下工作：

```c
// 生成Vcc/2虚地
// 方案1: 电阻分压 + 运放缓冲
// 方案2: 专用参考芯片(如TLV431)

          Vcc
           |
          10k
           |---- Vcc/2 (虚地)
          10k
           |
          GND

// 虚地用运放缓冲以降低阻抗
```

### 7.2 AC耦合

如果只关心交流信号(如振动、音频)，可用电容耦合去除直流偏移：

```
传感器 --||--+--|+ \-- Vout
        C1   | |  /
             | |-/
            R1 |
             | |
            Vcc/2(虚地)
```

### 7.3 电平移位

将传感器输出(如0-5V)移位到ADC范围(0-3.3V)：

```
Vout = Vin * (R2/(R1+R2)) * (1 + Rf/Rg)

选择R1, R2分压到3.3V以内，再用运放缓冲
```

## 8. 斩波稳零运放

### 8.1 工作原理

斩波稳零(chopper)运放通过不断校零来消除失调和漂移：

```
正常相位: 放大输入信号
校零相位: 短接输入，测量并存储失调电压
下一正常相位: 从输入中减去存储的失调

循环频率: 通常几kHz到几十kHz
```

### 8.2 优势与代价

优势：
- 失调电压低至1-5uV
- 几乎零温漂(0.01-0.05uV/C)
- 无1/f噪声(低频噪声极低)

代价：
- 斩波频率处有开关噪声(需后置滤波)
- GBW通常较低(350kHz-2.5MHz)
- 成本高于通用运放

### 8.3 IoT适用场景

- 称重传感器(长期零点稳定性)
- 热电偶(微伏级信号，温度漂移致命)
- pH值测量(高阻抗+需要零漂)

## 9. 实战案例

### 9.1 热敏电阻桥式放大器

```
         Vcc
          |
         10k(固定)
          |---- V+ --> |+ \
         NTC          |   /---> ADC
          |---- V- --> |-/
         10k(固定)
          |
         GND

NTC 10k@25C, 温度范围0-100C
桥输出约: 0-200mV差分
需要增益: 3.3V / 0.2V = 16.5倍
选用: OPA333(低功耗, 低失调)
```

### 9.2 压电振动传感器电荷放大器

```
压电传感器 --> TIA(Rf=10MO, Cf=10pF)
               |
               +--> 二阶带通滤波
                    |
                    +--> ADC

信号频率: 10Hz - 5kHz
选用: OPA2340(低噪声, 5.5MHz GBW)
```

### 9.3 称重传感器仪表放大

```
称重桥(Vexc=3.3V, 2mV/V)
  差分输出: 0-6.6mV
  需要增益: 3.3V / 6.6mV = 500倍
  选用: INA333(G=500, Iq=50uA)

  INA333增益设定:
  G = 1 + 100k/Rg
  Rg = 100k / (G-1) = 100k/499 = 200.4O
  选200O标准阻值: G = 1 + 100k/200 = 501
```

## 10. 选型决策树

```
IoT运放选型
  |
  +--> 功耗预算 < 5uA?
  |      +--> 是: TSU111(580nA) / MAX9914(1uA)
  |      +--> 否: 继续
  |
  +--> 需要极高DC精度(失调<50uV)?
  |      +--> 是: OPA333 / AD8629(斩波)
  |      +--> 否: 继续
  |
  +--> 信号频率 > 10kHz?
  |      +--> 是: OPA2340 / MCP6002(宽带)
  |      +--> 否: 继续
  |
  +--> 差分信号/高CMRR需求?
  |      +--> 是: 仪表放大器(INA333 / AD8237)
  |      +--> 否: 通用运放(MCP6001 / LMV358)
  |
  +--> 输入是电流(光电二极管)?
         +--> 是: TIA配置(OPA2340 / OPA350)
         +--> 否: 回到同相/反相放大器
```

## 11. 常见避坑

### 11.1 低功耗运放不等于低噪声

很多超低功耗运放的噪声密度高达100-200nV/sqrt(Hz)，在微弱信号场景下信噪比极差。选型时必须同时看Iq和噪声。

### 11.2 单电源忘记偏置

单电源运放处理双极性信号(如交流耦合的振动信号)时，必须将输入偏置到Vcc/2，否则负半周被截断。

### 11.3 反馈电阻过大引入噪声

高增益电路中反馈电阻可能用到MO级，电阻热噪声(4kTR)会显著增加。解决方案：

- 使用T型反馈网络(降低单个电阻值)
- 选择低噪声运放补偿电阻噪声
- 缩小带宽(增加滤波电容)

### 11.4 忽视PCB漏电

在pA-fA级偏置电流的应用中，PCB表面漏电可能比运放Ib还大。对策：

- 运放输入引脚周围做保护环(guard ring)
- 使用聚四氟乙烯绝缘子
- PCB清洁去除焊剂残留
- 用防潮漆涂覆

## 总结

IoT传感器前端运放选型，核心是在功耗、精度和带宽之间找到平衡点：

- **超低功耗场景**：TSU111(580nA)，适合常开环境传感器
- **精密测量场景**：OPA333(17uA, 10uV Vos)，适合称重和热电偶
- **宽带信号场景**：OPA2340(750uA, 5.5MHz)，适合振动和音频
- **差分信号场景**：INA333仪表放大器，适合桥式传感器

三个必须检查的陷阱：单电源偏置、反馈电阻噪声、PCB漏电。运放选对了，传感器数据才能"放大不失真"；选错了，后续的数字滤波再强大也救不回来。
