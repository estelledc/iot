# LED恒流驱动电路设计与调光方案
> **难度**：🟢 初级 | **领域**：LED驱动设计 | **阅读时间**：约 18 分钟

## 引言

想象你在家用水龙头控制水流：水压忽高忽低时，如果只靠阀门开度来控制，水量就会忽大忽小。LED也一样，它对电压非常敏感，电压稍微变一点，电流就可能翻倍。所以驱动LED不是"给个电压"就行，而是要"给个恒定的电流"，就像给水龙头装一个恒流阀，不管水压怎么变，出水量始终稳定。

## 1 LED电气特性基础

### 1.1 正向电压与正向电流

LED是二极管，具有非线性伏安特性。关键参数：

- **正向电压Vf**：LED导通时两端的压降。红光约1.8-2.2V，绿光约2.0-3.0V，蓝光/白光约2.8-3.6V
- **正向电流If**：流过LED的工作电流，指示灯5-20mA，照明级350-700mA甚至更高

### 1.2 V-I曲线的非线性

LED的V-I曲线呈指数关系，可以近似为：

```
If = Is * (exp(Vf / (n * Vt)) - 1)
```

其中Is是反向饱和电流，n是理想因子(1-2)，Vt是热电压(约26mV @ 25C)。

这意味着：Vf变化0.1V，If可能变化数倍。这不是线性器件，不能用简单"给电压"来控制亮度。

### 1.3 温度与热失控

LED的Vf具有负温度系数，约-2mV/C到-4mV/C。温度升高时：

1. Vf下降 -> 如果电压恒定，电流增大
2. 电流增大 -> 发热更多 -> 温度继续升高
3. 形成**热失控**循环，最终烧毁LED

| 温度变化 | Vf变化(典型) | 电流变化(恒压驱动) |
|----------|-------------|-------------------|
| +10C     | -30mV       | +30% 到 +50%      |
| +25C     | -75mV       | +100% 到 +200%    |
| +50C     | -150mV      | 可能失控           |

这就是为什么**恒流驱动**对LED至关重要。

## 2 为什么需要恒流驱动

### 2.1 电压驱动的三重困境

- **V-I非线性**：同一电压下，不同LED因Vf分散性导致电流差异大
- **温度漂移**：Vf随温度下降，恒压下电流持续增大
- **批量差异**：同型号LED的Vf可能有0.2-0.5V的分散性

### 2.2 恒流驱动的优势

恒流驱动确保电流不随Vf和温度变化：LED亮度稳定；多颗LED串联自动均流；不存在热失控风险；寿命可预测。

## 3 简单电阻限流

### 3.1 基本计算

最简单的LED驱动方式就是串联一个电阻：

```
R = (Vsupply - Vf) / If
```

例如：5V电源驱动白光LED，Vf=3.0V，If=20mA：

```
R = (5 - 3.0) / 0.02 = 100 ohm
```

### 3.2 功率损耗

电阻上的功耗：

```
P_R = (Vsupply - Vf) * If = 2.0 * 0.02 = 40mW
```

LED上的功耗：P_LED = Vf * If = 3.0 * 0.02 = 60mW

效率 = P_LED / (P_LED + P_R) = 60/100 = 60%

当电源电压远高于Vf时，效率急剧下降。

### 3.3 电阻限流的局限

- 电源电压波动直接导致电流波动
- 温度变化时Vf变化，电流跟着变
- 多颗LED无法保证均流
- 效率低，不适合大功率场景

适用场景：5-20mA指示灯，对亮度精度要求不高。

## 4 线性恒流源

### 4.1 晶体管+基准方案

用晶体管和基准电压构建简单恒流源：

```
         Vsupply
           |
         [LED]
           |
           C
    B -- NPN (Q1)
           E
           |
         [R_sense]
           |
          GND
```

工作原理：R_sense上的压降V_sense = If * R_sense。当V_sense超过晶体管Vbe(约0.6V)时，基极电流被限制，从而限制集电极电流If。

恒流值：If = Vbe / R_sense = 0.6V / R_sense

### 4.2 LM317作为恒流源

LM317是经典的线性稳压器，也可配置为恒流源：

```
         Vsupply
           |
     [LM317 IN]
           |
     [LM317 OUT] ---[R_set]---+
           |                   |
     [LM317 ADJ]              |
           |                   |
          [LED]               |
           |                   |
          GND ----------------+
```

恒流值：If = 1.25V / R_set

| 目标电流 | R_set    | 功耗(R_set) |
|---------|----------|------------|
| 20mA    | 62.5 ohm | 25mW       |
| 350mA   | 3.57 ohm | 437mW      |
| 700mA   | 1.78 ohm | 875mW      |

### 4.3 线性恒流的局限

- 效率 = V_LED / V_supply，压差大时效率极低
- 多余功率全部转化为热量
- 需要散热设计
- 适合低压差、小电流场景

## 5 开关型LED驱动器

### 5.1 Buck降压驱动(Vin > Vled)

当输入电压始终高于LED正向电压时使用Buck拓扑：

```
    Vin ----+----[SW]----+----[L]----+---- Vout(到LED)
            |            |           |
            |          [D]         [C_out]
            |            |           |
            +------------+-----------+---- GND
```

工作过程：SW导通时电流经L给LED供电，L储能；SW关断时L通过续流二极管D维持电流。效率90-95%。

典型应用：12V/24V驱动LED灯条。

### 5.2 Boost升压驱动(Vin < Vled)

当输入电压低于LED正向电压时使用Boost拓扑。典型应用：单节锂电池(3.0-4.2V)驱动串联白光LED(Vf_total约9-12V)。效率85-92%。

### 5.3 Buck-Boost升降压驱动

输入电压可能高于或低于LED电压时使用，覆盖全范围。适合电池供电的便携设备，输入电压变化范围大的场景。效率80-90%。

### 5.4 三种拓扑对比

| 拓扑        | 输入输出关系   | 效率     | 典型应用             |
|------------|--------------|---------|---------------------|
| Buck       | Vin > Vout   | 90-95%  | 12V/24V驱动LED灯条  |
| Boost      | Vin < Vout   | 85-92%  | 锂电池驱动串联白光LED |
| Buck-Boost | 任意关系      | 80-90%  | 宽输入电压范围应用    |

## 6 常见LED驱动IC

### 6.1 AP3019 - 升压白光LED驱动

- 拓扑：Boost升压
- 输入：2.5V - 16V，输出最高25V
- 可驱动2-6颗串联白光LED
- 内置1.5A开关管，开关频率1.2MHz
- 典型应用：手机/平板背光

### 6.2 TPS61165 - 大功率LED驱动

- 拓扑：Boost升压
- 输入：2.7V - 18V，输出最高38V
- 开关电流最高2A，内置1.6A开关管
- 支持PWM调光
- 典型应用：手电筒、投影仪光源

### 6.3 CAT4104 - 多通道LED驱动

- 拓扑：线性恒流(4通道)
- 每通道电流最高175mA，通道间匹配精度1%
- 支持PWM调光
- 典型应用：RGB照明、LCD背光

### 6.4 IC选型建议

| 应用场景        | 推荐IC      | 理由                     |
|---------------|-------------|--------------------------|
| 小尺寸背光     | AP3019      | 集成度高，外围少          |
| 大功率照明     | TPS61165    | 大电流，高效率            |
| RGB多通道      | CAT4104     | 4通道匹配，调光方便       |
| 电池供电低功耗  | TPS61042    | 低静态电流                |

## 7 调光方法

### 7.1 模拟调光(直流调光)

通过改变LED电流大小来调节亮度：

- 方法：调整恒流源的参考电压或电流设定电阻
- 范围：通常可从100%调到10%左右
- 优点：无闪烁，电路简单
- 缺点：低电流时色温偏移，亮度线性度差

### 7.2 PWM调光

通过高速开关LED来调节平均亮度：

```
亮度百分比 = 占空比 = Ton / (Ton + Toff) * 100%
```

PWM频率选择：

- 低于200Hz：肉眼可见闪烁
- 1kHz以上：无闪烁感
- 超过20kHz：开关损耗增加，EMI问题

PWM调光的关键优势：

1. **色温一致性**：每次导通都是满电流，光谱不变
2. **线性度好**：亮度与占空比严格线性关系
3. **精度高**：数字控制占空比，分辨率可达12bit(4096级)

### 7.3 两种调光对比

| 特性     | 模拟调光       | PWM调光        |
|---------|---------------|----------------|
| 色温稳定性 | 低电流偏移     | 始终一致       |
| 线性度   | 差             | 好             |
| 闪烁风险 | 无             | 频率低时有      |
| 电路复杂度| 低             | 中             |
| 精度     | 受模拟电路限制  | 数字控制精度高  |

## 8 调光接口

### 8.1 MCU PWM输出

最直接的调光方式，MCU的定时器输出PWM波形：

- 频率：1-5kHz常见
- 分辨率：8-16bit
- 只需一个GPIO
- 适合MCU直接控制的场景

### 8.2 0-10V模拟调光

工业照明标准接口：

- 0V：关闭
- 1-10V：线性调节亮度(10%-100%)
- 10V：最大亮度
- 传输距离可达数十米
- 需要专用的0-10V转PWM转换电路

### 8.3 DALI协议

数字可寻址照明接口(Digital Addressable Lighting Interface)：

- 两线制差分总线
- 每个设备有唯一地址(64个短地址/组地址)
- 调光命令：0-254级 + 关闭
- 支持场景、分组、渐变
- 波特率1200bps，适合楼宇照明控制

## 9 RGB LED驱动

### 9.1 三通道独立控制

RGB LED需要独立控制R/G/B三通道的电流，混合出各种颜色：

- 红：Vf约2.0-2.2V，If典型20mA
- 绿：Vf约3.0-3.2V，If典型20mA
- 蓝：Vf约3.0-3.4V，If典型20mA

三通道Vf不同，不能简单并联驱动。

### 9.2 共阳极 vs 共阴极

| 类型    | 连接方式        | 驱动方式          |
|--------|----------------|------------------|
| 共阳极  | 阳极接VCC       | 阴极灌电流驱动    |
| 共阴极  | 阴极接GND       | 阳极拉电流驱动    |

共阳极更常见，因为多数LED驱动IC采用灌电流(吸电流)架构。

### 9.3 颜色混合原理

通过调节三通道的PWM占空比实现颜色混合：

```
R_duty = (R_value / 255) * 100%
G_duty = (G_value / 255) * 100%
B_duty = (B_value / 255) * 100%
```

例如橙色：R=255, G=165, B=0

## 10 IoT实际应用

### 10.1 状态指示灯：GPIO + 电阻

最简单的IoT LED应用：

```
MCU GPIO ---[R=330]---[LED]--- GND
```

```c
// ESP32 GPIO控制LED示例
#define LED_PIN 2

void app_main() {
    gpio_set_direction(LED_PIN, GPIO_MODE_OUTPUT);

    // 简单闪烁指示
    while (1) {
        gpio_set_level(LED_PIN, 1);
        vTaskDelay(pdMS_TO_TICKS(500));
        gpio_set_level(LED_PIN, 0);
        vTaskDelay(pdMS_TO_TICKS(500));
    }
}
```

要点：
- GPIO输出电压通常3.3V，限流电阻计算：R = (3.3 - Vf) / If
- GPIO驱动能力有限，ESP32约12-40mA，STM32约8-25mA
- 多颗LED建议用三极管或MOSFET驱动

### 10.2 智能照明：Buck驱动 + PWM调光

大功率LED照明方案：

```
12V电源 -> Buck LED驱动IC -> LED灯珠
              |
         MCU PWM信号 -> EN/DIM引脚
```

```c
// ESP32 PWM调光示例
#define LEDC_CHANNEL 0
#define LEDC_GPIO    15
#define LEDC_FREQ    5000    // 5kHz
#define LEDC_RES     12      // 12bit分辨率

void set_brightness(uint8_t percent) {
    // percent: 0-100 -> duty: 0-4095
    uint32_t duty = (percent * 4095) / 100;
    ledc_set_duty(LEDC_LOW_SPEED_MODE, LEDC_CHANNEL, duty);
    ledc_update_duty(LEDC_LOW_SPEED_MODE, LEDC_CHANNEL);
}
```

## 11 功率LED的热管理

### 11.1 热量来源

LED的电光转换效率约20-40%，其余60-80%转化为热量。1W LED产生约0.6-0.8W热量。

### 11.2 散热设计要点

热阻计算：Tj = Ta + P * (R_jc + R_cs + R_sa)

- Tj：结温，不超过125C(典型)
- R_jc：结到壳热阻
- R_cs：壳到散热片热阻(需导热硅脂)
- R_sa：散热片到环境热阻

设计要点：

- 增加铜箔面积，热过孔，铝基板
- 根据热阻和成本选择适当散热片
- 大功率设计建议加NTC热敏电阻监测结温

### 11.3 降额使用

为延长寿命，建议：

- 工作电流不超过额定电流的80%
- 结温控制在100C以下
- 考虑最恶劣环境温度工况

## 12 效率考量

| 环节       | 效率范围    | 主要损耗         |
|-----------|-----------|-----------------|
| 开关驱动器  | 85-95%    | 开关损耗、导通损耗 |
| 线性驱动器  | 50-80%    | 压差上的热损耗    |
| 限流电阻    | 40-80%    | 电阻上的热损耗    |
| LED本身    | 20-40%    | 非辐射复合发热    |

提升效率的方法：

- 选择合适的拓扑(Buck效率最高)
- 同步整流替代续流二极管(提升3-5%)
- 减小开关频率(降低开关损耗，但增大电感体积)
- 优化PCB布局(减小寄生参数)

## 总结

LED驱动设计的核心是**恒流**，因为LED的非线性V-I特性和负温度系数使恒压驱动不可靠。从简单的电阻限流到高效的开关驱动，方案选择取决于功率大小和效率要求。调光方面，PWM调光因色温一致性好、线性度高而成为首选。在IoT应用中，小电流指示灯用GPIO加电阻即可，大功率照明则需要专用驱动IC配合PWM调光。热管理是大功率LED设计不可忽视的环节，直接影响寿命和可靠性。
